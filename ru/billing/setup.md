# Создание платёжного аккаунта

У нового пользователя платёжного аккаунта нет — его необходимо создать перед началом использования платных сервисов.

Перейдите в раздел **Биллинг → Платёжные аккаунты** и нажмите **Создать аккаунт**.

![img.png](img.png)

---

## Шаг 1 — Общая информация

Первый шаг одинаков для обоих типов аккаунта.

![img_1.png](img_1.png)

| Поле | Описание |
|---|---|
| **Имя аккаунта** | Отображаемое название платёжного аккаунта |
| **Организация** | Определяется автоматически по вашей текущей организации |
| **Страна** | Страна плательщика |
| **Тип аккаунта** | Физическое лицо или Юридическое лицо / ИП |

Выберите тип аккаунта и нажмите **Далее**.

> **Важно для юридических лиц:** регистрация под юридическим лицом необходима для выставления счетов, актов выполненных работ и официального документооборота. Для юридических лиц также предусмотрен расширенный набор возможностей и условий работы с платформой.

---

## Юридическое лицо или ИП

### Шаг 2 — Данные о компании

![img_2.png](img_2.png)

| Поле | Описание |
|---|---|
| **БИН/ИИН** | Бизнес-идентификационный номер компании |
| **Название организации** | Официальное название компании |
| **Город** | Город ведения бизнеса |
| **Код бенефициара (КБе)** | Код бенефициара для банковских операций |
| **Юридический адрес** | Официальный юридический адрес компании |
| **Полный адрес** | Фактический адрес компании |
| **Индекс** | Почтовый индекс (необязательно) |
| **Расчётный счёт (ИИК)** | Номер расчётного счёта компании в банке |
| **БИК** | Банковский идентификационный код |

Нажмите **Далее**.

### Шаг 3 — Контакты

![img_3.png](img_3.png)

| Поле | Описание |
|---|---|
| **Имя** | Имя контактного лица |
| **Фамилия** | Фамилия контактного лица |
| **Почтовый адрес** | Email для связи |
| **Номер телефона** | Используется исключительно для экстренной связи — при критических инцидентах инфраструктуры, безопасности или биллинга |

Нажмите **Создать**.

## Способ оплаты

После создания аккаунта добавьте способ оплаты:

- **Банковская карта** — доступна сразу. Привяжите карту в разделе **Биллинг → Информация об аккаунте**.
- **Банковский перевод** — оплата по счёту. Для подключения необходимо загрузить документы о регистрации компании. Эта возможность **находится в разработке** и будет доступна в ближайшее время. Требуется для подтверждения, что аккаунт действительно зарегистрирован от имени компании, а также для оформления счетов и актов выполненных работ.

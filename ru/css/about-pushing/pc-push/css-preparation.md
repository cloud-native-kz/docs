### Выбор сети для CSS

#### Опции сети

| Тип сети | Доступность | Стабильность |
| --- | --- | --- |
| Проводная сеть | Низкая | Высокая |
| Wi-Fi | Высокая | Низкая |
| По возможности рекомендуется использовать проводную сеть, которая более стабильна, чем Wi-Fi, чтобы избежать помех сигнала. Wi-Fi больше подходит для прямого видеотрансляции событий благодаря удобству. |  |  |

#### Тест пропускной способности восходящего канала

Требования к пропускной способности восходящего канала зависят от качества видео и разрешения. Как правило, лучшее качество видео с более высоким разрешением означает более высокие требования к пропускной способности восходящего канала. Пропускная способность восходящего канала должна быть не менее 1 Мбит/с. Чтобы проверить состояние пропускной способности восходящего канала, вы можете провести тест с помощью

speedtest

.

![](https://mc.qcloudimg.com/static/img/b5724af9873220c395e295894205e4ad/image.png)

### Установка программного обеспечения отправки

#### Установка OBS

Загрузите пакет установщика OBS с

официального веб-сайта OBS

и установите его с параметрами по умолчанию. Поддерживаются операционные системы Windows, Mac и Linux. Убедитесь, что загруженное программное обеспечение — это Open Broadcaster Software. OBS Studio также доступна, но не рассматривается в данном документе.

![](https://mc.qcloudimg.com/static/img/dcbb929e364b1d8e80c04e326a756a26/image.png)

#### Установка XSplit

При необходимости загрузите пакет установщика XSplit с официального веб-сайта XSplit и установите его с параметрами по умолчанию. XSplit — это платное программное обеспечение. В случае ограничения бюджета рекомендуется OBS (

**бесплатно**

). Для CSS игр доступен специальный пакет установщика XSplit. Для не-игровых CSS рекомендуется BroadCaster.

![](https://mc.qcloudimg.com/static/img/18c47cb7646e189acc168e6a5e8e4714/image.png)

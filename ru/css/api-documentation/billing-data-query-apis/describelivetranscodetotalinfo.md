# DescribeLiveTranscodeTotalInfo

## 1. Описание API

Доменное имя для запроса API: live.tencentcloudapi.com.

Этот API используется для запроса информации об использовании транскодирования. Вы можете использовать его для запроса данных за последние три месяца.
Примечания:
Если время начала и время окончания приходятся на один день, возвращаемые данные будут предоставляться на основе 5-минутного интервала.
Если время начала и время окончания приходятся на разные дни или если запрашивается информация по конкретным доменам, возвращаемые данные будут предоставляться на почасовой основе.

Максимум 200 запросов можно инициировать в секунду для этого API.

## 2. Входные параметры

Следующий список параметров запроса содержит только параметры запроса API и некоторые общие параметры. Для полного списка общих параметров см. [Общие параметры запроса](https://intl.cloud.tencent.com/document/api/267/30763).

| Имя параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://intl.cloud.tencent.com/document/api/267/30763). Значение, используемое для этого API: DescribeLiveTranscodeTotalInfo. |
| Version | Да | String | [Общие параметры](https://intl.cloud.tencent.com/document/api/267/30763). Значение, используемое для этого API: 2018-08-01. |
| Region | Нет | String | [Общие параметры](https://intl.cloud.tencent.com/document/api/267/30763). Этот параметр не требуется для этого API. |
| StartTime | Да | String | Время начала запроса поддерживает запрос данных за последние три месяца, разрыв между временем начала и временем окончания не может превышать три месяца. Интерфейс запроса поддерживает два формата времени: |
| EndTime | Да | String | Время окончания запроса поддерживает запрос данных за последние три месяца, разрыв между временем начала и временем окончания не может превышать три месяца. Интерфейс запроса поддерживает два формата времени: |
| PushDomains.N | Нет | Array of String | Список доменов отправки для запроса. Если этот параметр оставлен пустым, запрашиваются данные всех доменов. Если этот параметр указан, возвращаемые данные будут предоставляться на почасовой основе. |
| MainlandOrOversea | Нет | String | Допустимые значения: `Mainland`: запрашивает данные транскодирования в Китайском материке `Oversea`: запрашивает данные транскодирования вне Китайского материка По умолчанию запрашиваются данные как внутри, так и вне Китайского материка. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| DataInfoList | Array of [TranscodeTotalInfo](https://intl.cloud.tencent.com/document/api/267/30767#TranscodeTotalInfo) | Список данных транскодирования Примечание: это поле может возвращать `null`, что указывает на то, что не найдено действительного значения. |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый для каждого запроса. RequestId необходим для определения проблемы. |

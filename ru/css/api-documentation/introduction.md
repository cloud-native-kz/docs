# Введение

Используя мощные технологии Tencent и возможности его основных аудио- и видеосервисов, CSS (Cloud Streaming Services) предоставляет профессиональные, стабильные и быстрые услуги прямой трансляции и распространения контента с гарантией минимальной задержки и способностью обрабатывать большое количество одновременных запросов.

Перед использованием API CSS убедитесь, что вы прочитали и согласны с его [правилами выставления счетов](https://intl.cloud.tencent.com/document/product/267/2819).

> **Примечания:**
> 
> Все API в этом разделе имеют версию 3.0. Новые API CSS, добавленные в будущем, также будут версии 3.0. Мы настоятельно рекомендуем вам использовать API версии 3.0.
> Устаревшие API остаются доступными, но не будут обновляться. Дополнительную информацию об устаревших API см. в разделе "Обзор CSS API (Legacy)".

# CreateScreenshotTask

## 1. Описание API

Доменное имя для запроса API: live.intl.tencentcloudapi.com.

Этот API используется для создания задачи захвата скриншотов с определённым временем начала и окончания, которые выполняются в соответствии с настроенным шаблоном.

Примечание
Если поток прерывается, захват скриншотов останавливается. Однако задача остаётся действительной до указанного времени окончания, и захват скриншотов будет выполняться по требованию после возобновления потока.
Избегайте создания задач захвата скриншотов с перекрывающимися периодами времени. Система будет выполнять максимум три задачи захвата скриншотов на одном потоке одновременно.
Записи о задачах сохраняются только в течение трёх месяцев.
Новые API захвата скриншотов (CreateScreenshotTask/StopScreenshotTask/DeleteScreenshotTask) несовместимы с устаревшими (CreateLiveInstantSnapshot/StopLiveInstantSnapshot). Не смешивайте их при вызове API для управления задачами захвата скриншотов.
Если вы создаёте задачу захвата скриншотов и одновременно публикуете поток, задача может не выполниться в указанное время. После создания задачи захвата скриншотов рекомендуется подождать не менее трёх секунд перед публикацией потока.

Для этого API можно инициировать максимум 20 запросов в секунду.

## 2. Входные параметры

Следующий список параметров запроса содержит только параметры запроса API и некоторые обычные параметры. Полный список обычных параметров см. в разделе [Обычные параметры запроса](https://www.tencentcloud.com/document/api/267/30763).

| Имя параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Обычные параметры](https://www.tencentcloud.com/document/api/267/30763). Значение для этого API: CreateScreenshotTask. |
| Version | Да | String | [Обычные параметры](https://www.tencentcloud.com/document/api/267/30763). Значение для этого API: 2018-08-01. |
| Region | Нет | String | [Обычные параметры](https://www.tencentcloud.com/document/api/267/30763). Этот параметр не требуется для этого API. |
| StreamName | Да | String | Имя потока. |
| DomainName | Да | String | Домен трансляции. |
| AppName | Да | String | Путь трансляции. |
| EndTime | Да | Integer | Время окончания задачи, которое должно быть временной меткой Unix, позже `StartTime` и текущего времени. Разница между временем окончания и началом не может превышать 24 часа. |
| TemplateId | Да | Integer | ID шаблона захвата скриншотов, возвращаемый `CreateLiveSnapshotTemplate`. Если передан неправильный ID шаблона, задача захвата скриншотов будет неудачной. |
| StartTime | Нет | Integer | Время начала задачи, которое должно быть временной меткой Unix и не может быть позже чем через шесть дней от текущего времени. Если вы не указываете этот параметр, задача начнётся немедленно. |
| StreamType | Нет | Integer | Тип публикации. Допустимые значения: `0` (по умолчанию): Живой поток `1`: Смешанный поток |
| Extension | Нет | String | Дополнительное поле, которое в настоящее время не определено и по умолчанию пусто. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| TaskId | String | Уникальный ID задачи. Если этот параметр возвращается, задача захвата скриншотов создана успешно. |
| RequestId | String | Уникальный ID запроса, который возвращается для каждого запроса. RequestId требуется для определения проблемы. |

## 4. Пример

### Пример1 Создание задачи захвата скриншотов

#### Пример входных данных

```
https://live.intl.tencentcloudapi.com/?Action=CreateScreenshotTask
&AppName=live
&DomainName=5000.live.push.com
&StreamName=livetest
&StartTime=1589889600
&EndTime=1589904000
&TemplateId=0
&<Common request parameters>
```

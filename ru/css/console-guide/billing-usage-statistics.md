# Статистика использования выставления счётов

### Статистика использования выставления счётов

Консоль Tencent Cloud предоставляет функцию запроса статистики использования выставления счётов для Live Video Broadcasting (LVB) и Live Event Broadcasting (LEB). Если вы хотите узнать использование таких услуг, как входящий трафик/полоса пропускания, исходящий трафик/полоса пропускания, каналы трансляции, прямая трансляция с транскодированием, запись живого потока, сдвиг времени, снимок экрана при прямой трансляции, улучшение, модерация контента, продолжительность задач ретрансляции, ретрансляция третьей стороны, логи в реальном времени и шифрование DRM, вы можете просмотреть соответствующие данные в разделе **Statistics** > [Billing Usage Statistics](https://console.tencentcloud.com/live/analysis/bill) в консоли Cloud Streaming Services (CSS). Вы можете просматривать данные, связанные с использованием услуг CSS за последний месяц.

### Детали выставления счётов

| --- | --- | --- |
| Входящий трафик/полоса пропускания | Входящий трафик | Входящий трафик, использованный при трансляции в течение выбранного периода времени |
|  | Пиковая входящая полоса пропускания | Пиковая полоса пропускания, используемая при трансляции в течение выбранного периода времени |

|  | Детали | Почасовое потребление входящего трафика/полосы пропускания за последние 30 дней |
| Исходящий трафик/полоса пропускания | Исходящий трафик | Исходящий трафик, использованный при воспроизведении в течение выбранного периода времени |
|  | Пиковая исходящая полоса пропускания | Пиковая полоса пропускания, используемая при воспроизведении в течение выбранного периода времени |
|  | График исходящего трафика/полосы пропускания | Использование исходящего трафика/полосы пропускания на основе 5-минутных интервалов. Вы можете выбрать просмотр использования LVB или LEB. |
|  | Детали | Почасовое потребление исходящего трафика/полосы пропускания за последние 30 дней |

|  | График каналов трансляции | Количество каналов трансляции на основе 5-минутных интервалов |
|  | Пиковые каналы за последние 30 дней | Наибольшее количество каналов в день за последние 30 дней |

|  | График пика каналов записи/общей продолжительности отправки в COS | Наибольшее количество одновременных каналов записи на основе 5-минутных интервалов/накопленные данные по продолжительности записи, отправленной в COS |

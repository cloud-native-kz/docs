# Обзор консоли

Чтобы помочь вам быстро начать работу с консолью CSS, этот документ знакомит вас с некоторыми часто используемыми сервисами CSS. Они разделены на четыре модуля на основе потребностей пользователей: Базовые услуги, Услуги, специфичные для сценариев, Центр данных и CSS Toolkit.

## Базовые услуги

Этот модуль предоставляет базовые услуги CSS. Если вы хотите использовать только базовые услуги прямого вещания, это правильный модуль для вас.

| Функция | Описание |
| --- | --- |
| [Обзор](https://www.tencentcloud.com/document/product/267/31054?lang=en&pg=) | Вы можете просматривать услуги добавленной стоимости прямого вещания, их сценарии применения и функциональные характеристики. Вы можете просматривать соответствующие данные, такие как тренды пропускной способности / трафика выставления счетов, данные реального времени прямого вещания и количество одновременных соединений. Вы можете переключать режимы выставления счетов или изменять гранулярность времени по мере необходимости. |
| [Управление доменами](https://www.tencentcloud.com/document/product/267/35970) | Добавляйте и управляйте своими собственными доменами ускорения и настраивайте CNAME для них. Генерируйте URL-адреса прямого вещания. Вы можете вызывать созданные шаблоны записи, транскодирования, захвата экрана, сдвига времени, водяных знаков, модерации и обратного вызова для доменов прямого вещания. Вы можете настраивать домены прямого вещания с аутентификацией, протоколом HTTPS, регионами ускорения, ограничениями пропускной способности, отложенным воспроизведением, чёрными и белыми списками IP, заголовками HTTP-ответов, информацией о сервере-источнике и многим другим. |
| [Управление потоками](https://www.tencentcloud.com/document/product/267/31068) | Вы можете управлять потоками прямого вещания, основными и резервными потоками, историей потоков и отключёнными потоками. Вы также можете отключать и возобновлять потоки прямого вещания. |
| [Управление пакетами ресурсов](https://www.tencentcloud.com/document/product/267/52221) | Вы можете просматривать использование пакетов трафика и транскодирования. Поддержка автоматического продления пакетов трафика и транскодирования при их истечении или полном использовании. |

## Live +

Live+ собирает различные услуги добавленной стоимости Cloud Streaming Services, включая транскодирование, водяные знаки, захват экрана, модерацию, резервные потоки, субтитры и ретрансляцию. Если вам нужно использовать связанные услуги, вы можете выполнить соответствующие настройки в этом модуле.

| Функция | Описание |
| --- | --- |
| [Конфигурация функций](https://www.tencentcloud.com/document/product/267/31073) | Мы предоставляем услуги конфигурационных шаблонов для различных функций, необходимых при прямом вещании, таких как водяные знаки, запись, транскодирование, захват экрана, сдвиг времени, адаптивный битрейт, обратные вызовы и управление DRM. Чтобы снизить сложность навигации по страницам, мы добавили новый процесс для привязки шаблонов к доменным именам. |
| [LEB](https://console.tencentcloud.com/live/leb) | Описывает функции продукта LEB и шаги руководства по доступу. LEB можно оценить в три этапа и он плавно совместим с LVB. |
| [LVC](https://www.tencentcloud.com/document/product/267/58534) | Трансляция в прямом эфире может быть реализована через консоль CSS. Она поддерживает пользовательский макет экрана, синхронное переключение аудио-видео, список программ и автоматическую трансляцию. Эта функция устраняет необходимость в тяжёлом оборудовании, позволяя пользователям удобно и быстро использовать услуги трансляции. |
| [Pull и Relay](https://www.tencentcloud.com/document/product/267/42524) | Мы предоставляем возможность извлекать видео в прямом эфире или файлы по требованию с платформ третьих сторон и отправлять их в Cloud Streaming Services. Вы можете непосредственно выполнять операции, такие как смешивание, запись и многое другое с содержимым аудио и видео. Эта функция позволяет легко распределять контент на несколько платформ и преобразовывать содержимое по требованию в прямую трансляцию. |

## Центр данных

Анализ данных предоставляет пользователям профессиональные услуги анализа данных. Вы можете запрашивать потребление трафика / пропускной способности, транскодирования, водяных знаков, ретрансляции и захвата экрана в рамках определённой временной гранулярности. Кроме того, он предоставляет функциональность анализа логов, облегчая пользователям мониторинг ресурсов и получение полезных данных.

| Функция | Описание |
| --- | --- |
| [Статистика использования выставления счетов](https://www.tencentcloud.com/document/product/267/31076) | Вы можете запрашивать элементы выставления счетов, включая соответствующие данные, сгенерированные трафиком / пропускной способностью отправки и получения потока, записью, сдвигом времени, захватом экрана, транскодированием и ретрансляцией. |
| [Анализ операций](https://www.tencentcloud.com/document/product/267/31078?lang=en&pg=) | Вы можете просматривать анализ данных воспроизведения прямого вещания, распределение пользователей и данные возврата сервера-источника, среди прочей информации. |
| [Запрос данных потока](https://www.tencentcloud.com/document/product/267/31077) | Вы можете запрашивать детали данных одного видеопотока, такие как отправка, воспроизведение, записи прямого вещания, события обратного вызова и т. д., и экспортировать данные на локальный компьютер. |
| [Ошибки](https://www.tencentcloud.com/document/product/267/51202) | Вы можете запрашивать аномальные события, которые происходят при отправке прямого вещания. |
| [Записи прерываний потока](https://www.tencentcloud.com/document/product/267/31083) | Вы можете запрашивать записи и причины прерываний отправки прямого вещания. |
| [Анализ логов](https://www.tencentcloud.com/document/product/267/59877) | Вы можете выполнять сбор логов доступа прямого вещания в реальном времени, очистку, анализ и поиск для быстрого обнаружения ошибок доступа. |
| [Мониторинг качества SDK](https://www.tencentcloud.com/document/product/267/56123?lang=en&pg=) | Вы можете запрашивать информацию о данных отправки Live SDK за последние 3 дня. |

## CSS Toolkit

Live Toolbox в основном предоставляет некоторые вспомогательные функции для обеспечения процессов прямого вещания и использования и управления Live SDK.

| Функция | Описание |
| --- | --- |
| [Web Push](https://www.tencentcloud.com/document/product/267/35968?lang=en&pg=) | Быстро протестируйте функцию Web push с источниками ввода, включая захват с камеры, общий доступ к экрану и сбор локальных файлов. Он также поддерживает многопоточное смешивание, позволяя тестировать отправку в различных сценариях. |
| [Генератор адресов](https://www.tencentcloud.com/document/product/267/31084) | Предоставляет необходимую информацию для составления URL-адресов отправки / воспроизведения. |
| [Самодиагностика](https://www.tencentcloud.com/document/product/267/39467?lang=en&pg=) | Быстро диагностируйте распространённые проблемы отправки / воспроизведения прямого вещания. Результаты диагностики предоставляются только в справочных целях. |
| [MLVB SDK](https://www.tencentcloud.com/document/product/1071/38546?lang=en&pg=) | В сочетании с Live SDK вы можете добавлять и управлять официальными лицензиями и привязывать связанные пакеты ресурсов. |

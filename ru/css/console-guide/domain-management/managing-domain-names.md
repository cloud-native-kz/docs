# Управление доменными именами

На странице управления доменными именами консоли Cloud Streaming Services (CSS) вы можете выполнять различные операции с вашими доменными именами в соответствии с потребностями вашего бизнеса, включая включение, отключение и удаление доменных имен.

## Примечание

CSS по умолчанию предоставляет тестовое доменное имя `xxxx.livepush.myqcloud.com`. Вы можете использовать его для тестирования трансляции, но мы не рекомендуем использовать его в качестве доменного имени трансляции для вашего реального бизнеса. Это доменное имя не может быть удалено.

## Предварительные условия

1. Вы активировали сервис [CSS](https://intl.cloud.tencent.com/product/css).
2. Вы добавили [доменное имя](https://intl.cloud.tencent.com/document/product/267/35970).

## Просмотр доменного имени

Войдите в [консоль CSS](https://console.tencentcloud.com/live/domainmanage) и выберите **Управление доменом**. Вы можете просмотреть статус конфигурации CNAME, тип, статус и время добавления доменного имени. Для просмотра деталей нажмите на доменное имя или на кнопку "Управление" справа от него.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6fdaf0c7531e11f0adb15254005ef0f7.png)

## Конфигурация доменного имени

- Если вам нужно настроить доменное имя трансляции, см. [Конфигурация трансляции](https://intl.cloud.tencent.com/document/product/267/31059).
- Если вам нужно настроить доменное имя воспроизведения, см. [Конфигурация воспроизведения](https://intl.cloud.tencent.com/document/product/267/31058).

## Отключение доменного имени

Если вы временно не хотите использовать доменное имя для трансляции, вы можете его отключить. Вот шаги для отключения доменного имени:

> **Примечание:** После отключения доменного имени для трансляции информация о доменном имени будет сохранена в системе, но сервисы трансляции больше не будут обрабатывать запросы для этого доменного имени. Это означает, что пользователи больше не смогут инициировать трансляцию и воспроизведение через это доменное имя. Кроме того, текущие потоки трансляции или воспроизведения не будут прерваны.

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/domainmanage) и выберите **Управление доменом**. В списке доменных имен найдите доменное имя, которое вы хотите отключить, и нажмите **Отключить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bf78eeb3531e11f0bf84525400454e06.png)

2. В появившемся окне нажмите **Подтвердить**, чтобы отключить доменное имя для трансляции.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e9625b51531e11f095fc5254001c06ec.png)

3. В столбце "Статус" на странице "Управление доменом" вы можете видеть, что текущий статус доменного имени изменился на **Отключено**. Доменное имя успешно отключено и больше не может использоваться для трансляции и воспроизведения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/32f898d3531f11f0abce52540099c741.png)

## Включение доменного имени

Если вам нужно повторно включить отключенное доменное имя, выполните следующие шаги:

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/domainmanage) и выберите **Управление доменом**. В списке доменных имен найдите доменное имя, которое вы хотите включить, и нажмите **Включить**. Сервисы трансляции будут возобновлены для этого доменного имени.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8ac71986531f11f095fc5254001c06ec.png)

2. В столбце "Статус" на странице "Управление доменом" вы можете видеть, что текущий статус доменного имени изменился на **Включено**. Вы можете снова использовать это доменное имя для трансляции и воспроизведения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c6164289531f11f09a935254007c27c5.png)

## Удаление доменного имени

> **Примечание:** Удаление необратимо. При удалении доменного имени все его конфигурации будут окончательно удалены. Вы по-прежнему сможете просматривать данные об использовании удаленных доменных имен.

Если вам нужно удалить доменное имя, выполните следующие шаги:

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/domainmanage) и выберите **Управление доменом**. В списке доменных имен найдите доменное имя, которое вы хотите удалить, и нажмите **Удалить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f886ae3c532011f095fc5254001c06ec.png)

2. В появившемся окне нажмите **Подтвердить**, чтобы удалить доменное имя из консоли CSS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/43e7114c532111f09a935254007c27c5.png)

# Управление доступом по регионам

Управление доступом по регионам позволяет управлять чёрным списком или белым списком для регионов воспроизведения текущего домена, обеспечивая лучший контроль над распределением контента в определённых областях.

## Как это работает

- Если вы настраиваете чёрный список, запросы из выбранных регионов блокируются.
- Если вы настраиваете белый список, разрешены только запросы из выбранных регионов.

## Важная информация

- После завершения настройки чёрного списка и белого списка для региона воспроизведения вступление в силу занимает примерно 10 минут.
- Если вы одновременно включили функции [Конфигурация IP чёрного/белого списка](https://www.tencentcloud.com/document/product/267/46767) и управления регионом воспроизведения, учитывайте, что IP чёрный/белый список имеет приоритет над управлением регионом воспроизведения.

## Предварительные условия

- Вы активировали CSS и вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat).
- Вы добавили [домен воспроизведения](https://www.tencentcloud.com/document/product/267/35970).

## Настройка управления доступом по регионам

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели. Нажмите на **домен воспроизведения**, для которого вы хотите настроить управление регионом, или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c5c7cfd611c811ef9fa952540019e87e.png)

2. На вкладке **Управление доступом** найдите **Управление доступом по регионам**.
3. Нажмите ![](https://staticintl.cloudcachetci.com/cms/backend-cms/31558f75cc2211f08658525400454e06.png), чтобы включить управление доступом по регионам и выполните следующие настройки:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/006c97cbcc2211f0afdc52540044a08e.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1933afc3cc2211f084a45254005ef0f7.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Аутентификация по | Выберите, хотите ли вы настроить **Белый список** или **Чёрный список**. Белый список и чёрный список не могут быть активны одновременно. Если вы настроите **белый список**, доступ к контенту трансляции будет разрешён только для регионов, добавленных в белый список. Доступ будет запрещён для регионов, не добавленных в белый список. Если вы настроите **чёрный список**, доступ к контенту трансляции будет запрещён для регионов, добавленных в чёрный список. Доступ будет разрешён для всех регионов, не добавленных в чёрный список. |
| Настройка регионов | Вы можете просматривать и расширять регионы в разделах **Азия**, **Африка**, **Европа**, **Северная Америка**, **Южная Америка** и **Океания** и добавлять их в список выбранных регионов. Вы также можете выполнять поиск по регионам, чтобы найти их более быстро. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/570a8feecc2211f084a45254005ef0f7.png)

4. Нажмите **Сохранить**, чтобы сохранить конфигурацию (вступление в силу конфигурации займёт некоторое время).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/144265344b0811eeb917525400b31cf9.png)

## Изменение управления доступом по регионам

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели. Нажмите на **домен воспроизведения**, для которого вы хотите изменить параметры управления регионом, или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/da1ac17011c811ef9af4525400720cb5.png)

2. На вкладке **Управление доступом** найдите **Управление доступом по регионам** и нажмите **Редактировать**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2ebc422f4b0511eeb9595254003d2706.png)

3. Измените конфигурацию и нажмите **Сохранить**.

## Отключение управления доступом по регионам

Выполните следующие шаги, чтобы отключить управление доступом по регионам:

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели. Нажмите на **Домен воспроизведения**, для которого вы хотите отключить управление доступом по регионам, или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/de48632a11c811efa09b525400762795.png)

2. На вкладке **Управление доступом** найдите **Управление доступом по регионам**. Нажмите ![img](https://staticintl.cloudcachetci.com/cms/backend-cms/52abd0582c9111efa01d5254005235d8.png), чтобы отключить управление доступом по регионам.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9b7dc78f4b0711ee9f16525400a32c73.png)

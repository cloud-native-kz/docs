# Конфигурация списка блокировки/разрешения IP-адресов

Этот документ показывает, как настроить список разрешений/блокировки IP-адресов для фильтрации запросов и контроля доступа к потоковому контенту.

## Принцип работы

- **Список разрешений IP**: доступ к вашему потоковому контенту имеют только IP-адреса из списка.
- **Список блокировки IP**: IP-адреса из списка не могут получить доступ к вашему потоковому контенту.

## Важная информация

Список блокировки/разрешения IP-адресов вступает в силу примерно через десять минут после конфигурации.

## Предварительные требования

- Вы активировали CSS и вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat).
- Если [управление регионом воспроизведения](https://www.tencentcloud.com/document/product/267/56788) (региональная блокировка/разрешение) включены одновременно со списком блокировки/разрешения IP-адресов, логика системы следующая:
  1.1. **Сначала проверяется список блокировки/разрешения IP-адресов**:
  - Если IP-адрес находится в списке разрешений, доступ разрешается немедленно.
  - Если IP-адрес находится в списке блокировки, доступ немедленно запрещается.
  - Если IP-адрес не находится ни в списке блокировки, ни в списке разрешений, выполняется переход к следующему шагу.
  1.2. **Затем проверяется региональная блокировка/разрешение** (только если IP-адрес не находится в списке блокировки/разрешения IP-адресов):
  - Если регион находится в списке разрешений, доступ разрешается; в противном случае доступ запрещается.
  - Если регион находится в списке блокировки, доступ запрещается; в противном случае доступ разрешается.

## Конфигурация списка разрешений/блокировки IP-адресов

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели и нажмите на целевой **домен воспроизведения** или нажмите **Управлять** справа, чтобы открыть страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ede7fcc7013311efb881525400a7e516.png)

2. В разделе **Контроль доступа** > **Список разрешений/блокировки IP**, нажмите на ![](https://staticintl.cloudcachetci.com/cms/backend-cms/264adb0e013411efbba352540095b445.png), чтобы включить список разрешений/блокировки IP-адресов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/52650437cc0811f091ab5254007c27c5.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6742d4d5cc0811f091ab5254007c27c5.png)

3. После включения **списка разрешений/блокировки IP** перейдите на страницу конфигурации **списка разрешений/блокировки IP** и выполните следующую конфигурацию:

Список блокировки

Список разрешений

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bf3fa5e5d3e111f08f8f525400454e06.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cb104ab4d3e111f08206525400e889b2.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Аутентификация по | Список разрешений или блокировки: вы не можете выбрать оба одновременно. Если вы настроите список разрешений, доступ к вашему потоковому контенту смогут получить только IP-адреса из списка. Если вы настроите список блокировки, IP-адреса из списка не смогут получить доступ к вашему потоковому контенту. |
| Список IP | Список блокировки IP поддерживает максимум 200 правил, а список разрешений IP поддерживает до 500 правил. Пожалуйста, разделяйте записи символом новой строки. Список разрешений/блокировки поддерживает форматы IPv4, IPv6 и сегменты сети. |

4. Нажмите **Сохранить**, чтобы сохранить конфигурацию (применение конфигурации занимает некоторое время).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/dfe67ca5d3e111f09f555254007c27c5.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4f71c64a013411efbf03525400ae4d13.png)

## Изменение списка разрешений/блокировки IP-адресов

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели и нажмите на целевой **домен воспроизведения** или нажмите **Управлять** справа, чтобы открыть страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/56cd152e013411ef9125525400275b90.png)

2. Нажмите **Контроль доступа** и в разделе **Список разрешений/блокировки IP** нажмите **Изменить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5d389a19013411efbba352540095b445.png)

3. Измените конфигурацию и нажмите **Сохранить**.

## Отключение списка разрешений/блокировки IP-адресов

1. Выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели и нажмите на целевой **домен воспроизведения** или нажмите **Управлять** справа, чтобы открыть страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/671ed55c013411efbba352540095b445.png)

2. Выберите вкладку **Контроль доступа**. В разделе **Список разрешений/блокировки IP** нажмите ![img](https://staticintl.cloudcachetci.com/cms/backend-cms/94e49a114ee911ee84f2525400494e51.png), чтобы отключить список разрешений/блокировки IP-адресов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6c54345e013411efbf03525400ae4d13.png)

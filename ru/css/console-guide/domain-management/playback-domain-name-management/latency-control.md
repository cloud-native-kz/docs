# Управление задержкой

Установите задержку, соответствующую вашим потребностям. Обратите внимание, что установка слишком низкой задержки может привести к прерывистому воспроизведению.

## Примечание

Конфигурация задержки вступит в силу примерно через 10 минут после настройки.

## Предварительные требования

- Вы активировали CSS и вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat).
- Вы добавили [доменное имя воспроизведения](https://intl.cloud.tencent.com/document/product/267/35970).

## Управление задержкой

1. Выберите [Domain Management](https://console.tencentcloud.com/live/domainmanage) на левой боковой панели и нажмите на **доменное имя воспроизведения**, для которого вы хотите настроить задержку RTMP и FLV, или нажмите **Manage** справа, чтобы открыть страницу сведений.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7e68436f142611ef83585254007bbd8c.png)

2. В разделе **Advanced Configuration** > **Playback latency configuration** вы можете настроить задержку для RTMP и FLV.
3. При настройке параметров задержки прямой трансляции рекомендуется выбирать соответствующие параметры задержки в зависимости от ваших фактических потребностей бизнеса. Рекомендуется установить GOP для публикации на 1–2 секунды, так как чем больше значение GOP, тем больше задержка прямой трансляции. Обратите внимание, что установка слишком низкой задержки может привести к прерывистому воспроизведению.
4. Когда GOP установлен на две секунды, задержка выглядит следующим образом:

| Параметр | Высокая | Средняя | Низкая |
| --- | --- | --- | --- |
| Предполагаемая задержка | 7–9 с | 5–7 с | 4–5 с |

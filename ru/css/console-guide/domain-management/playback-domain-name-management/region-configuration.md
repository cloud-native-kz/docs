# Конфигурация региона

Для использования услуг доставки контента, ускорения и воспроизведения в другом регионе вы можете изменить регион ускорения для своего домена воспроизведения в консоли CSS.

## Предварительные требования

- Вы вошли в [консоль CSS](https://console.tencentcloud.com/live).
- Вы добавили **доменное имя воспроизведения**.

> **Примечание:** При добавлении нового домена и выборе домена воспроизведения выберите регион ускорения, необходимый для распространения прямой трансляции, например "Материковый Китай". Затем заполните информацию о доменном имени и теги (опционально). Нажмите **Добавить домен** и перейдите к следующему шагу.![](https://staticintl.cloudcachetci.com/cms/backend-cms/947a4776144811f0aaa3525400e889b2.png)

## Примечания

- **Цены CSS отличаются внутри и вне материкового Китая. Дополнительные сведения см. в**[Обзор выставления счетов](https://intl.cloud.tencent.com/document/product/267/2819)**.**
- Домен воспроизведения не может быть использован вне его региона ускорения.
- Если регион ускорения включает материковый Китай, вам необходимо подать заявку на регистрацию ICP для вашего домена воспроизведения.
- Изменение региона ускорения приведет к сбросу лимита пропускной способности. Необходимо будет настроить его снова.

## Инструкции

1. Перейдите в [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на имя вашего **домена воспроизведения** или **Управление** справа.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/aa2e7c3d3dd211efbec9525400a9236a.png)

2. Выберите вкладку **Дополнительная конфигурация** и найдите **Конфигурация региона**.
3. Нажмите **Редактировать**. В появившемся окне вы можете изменить регион ускорения на **Материковый Китай**, **Глобально** или **Вне материкового Китая**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/63d8c9d9cbf711f08e74525400bf7822.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e3bd293bcc0411f096d1525400e889b2.png)

4. Нажмите **Сохранить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2f18bb663dd311efb89a52540075b605.png)

| Регион ускорения | Требуется регистрация ICP | Описание |
| --- | --- | --- |
| Материковый Китай | Да | Не может обрабатывать запросы вне материкового Китая. |
| Глобально | Да | Поддерживается глобальное ускорение, но цены отличаются внутри и вне материкового Китая. |
| Вне материкового Китая | Нет | Не может обрабатывать запросы внутри материкового Китая. Цены отличаются внутри и вне материкового Китая. |

# Шаблон стирания аудио-видеоконтента

Функция интеллектуального стирания отключена по умолчанию. В этом документе приведены инструкции по связыванию шаблона стирания аудио-видеоконтента с определённым доменом трансляции для активации функции стирания аудио-видеоконтента и по разъединению шаблона для деактивации стирания аудио-видеоконтента.

## Примечания

- Шаблон вступает в силу примерно через 5–10 минут после связывания.
- После успешного связывания шаблона функция интеллектуального стирания будет включена для URL трансляции под указанным доменом трансляции.
- С доменом можно связать только один шаблон стирания аудио-видеоконтента. После связывания все потоки под этим доменом будут подвергаться стиранию аудио-видеоконтента в соответствии с связанным шаблоном.

## Предварительные требования

- Вы успешно вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat) и завершили [добавление собственного домена](https://www.tencentcloud.com/document/product/267/35970).
- Шаблон стирания аудио-видеоконтента успешно создан.

## Связывание шаблона стирания аудио-видеоконтента

1. Перейдите в [управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **имя домена**, который требуется настроить, или нажмите **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9991e87dc5b711f0a7ca5254001c06ec.png)

2. Выберите вкладку **Template Configuration** (Конфигурация шаблонов). Нажмите **Edit** (Редактировать) в правом верхнем углу раздела **Audio-Video Content Erasing Template** (Шаблон стирания аудио-видеоконтента).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/311727b23c3011f0b0ce5254007c27c5.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c91690b600b711f1b5c0525400380f7d.png)

3. Выберите шаблон конфигурации интеллектуального стирания в соответствии с требованиями вашего бизнеса и нажмите **Confirm** (Подтвердить), чтобы завершить конфигурацию.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/eeec32d500b711f1b5c0525400380f7d.png)

## Разъединение шаблона стирания аудио-видеоконтента

1. Перейдите в [управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **имя домена**, который требуется настроить, или нажмите **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a17e9bfdc5b711f086d9525400e889b2.png)

2. Выберите вкладку **Template Configuration** (Конфигурация шаблонов) и нажмите Edit (Редактировать) в правом верхнем углу раздела **Audio-Video Content Erasing Template** (Шаблон стирания аудио-видеоконтента).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8cdf3d553c3011f09bbe525400454e06.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/021e580500b811f198c8525400370dda.png)

3. В соответствии с требованиями вашего бизнеса снимите флажок с соответствующего шаблона и нажмите **Confirm** (Подтвердить) для продолжения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/15a1987300b811f1b5c0525400380f7d.png)

> **Примечание:** разъединение шаблона стирания аудио-видеоконтента не влияет на текущие трансляции.

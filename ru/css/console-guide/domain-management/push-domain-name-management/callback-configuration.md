# Конфигурация обратного вызова

Функция обратного вызова отключена по умолчанию для облачных потоковых служб (CSS) при отправке. После того как имя домена отправки привязано к конфигурации обратного вызова, функция обратного вызова будет включена для всех адресов отправки на этом имени домена. Если во время прямого эфира запускается событие обратного вызова, настроенное в шаблоне, Tencent Cloud отправит запрос на сервер клиента, который отвечает за ответ. После проверки клиент может получить пакет JSON, содержащий информацию об обратном вызове.
В этом документе описано, как привязать/отвязать имя домена отправки к/от шаблона обратного вызова для включения/отключения функции обратного вызова.

## Примечания

- Конфигурация шаблона вступит в силу примерно через 5–10 минут.
- Когда событие CSS запускается после включения функции обратного вызова, вы можете получить информацию о событии через [уведомление о сообщении события](https://intl.cloud.tencent.com/document/product/267/38080).
- Шаблоны обратного вызова управляются на уровне имени домена в консоли, и правила, созданные API, на данный момент невозможно отменить. Если вы привязали шаблон к определённому потоку через API обратного вызова и хотите его отвязать, вам нужно вызвать API [DeleteLiveCallbackTemplate](https://intl.cloud.tencent.com/document/product/267/30813).
- Одно имя домена можно привязать только к одному шаблону обратного вызова. После привязки все потоки в нём будут использовать обратный вызов согласно этому шаблону.

## Предварительные требования

- Вы вошли в [консоль CSS](https://console.tencentcloud.com/live) и добавили [имя домена отправки](https://intl.cloud.tencent.com/document/product/267/35970).
- Вы создали [шаблон обратного вызова](https://www.tencentcloud.com/document/product/267/31074#creating-a-callback-template.5B.5D(id.3Acallback)).

## Привязка шаблона обратного вызова

1. Перейдите в [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на целевое **имя домена отправки** или **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/678d56c6ce9911f084a45254005ef0f7.png)

2. Выберите **Конфигурация шаблона** и нажмите **Редактировать** в верхнем правом углу вкладки **Конфигурация обратного вызова**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5b8afc48ab0b11f096c2525400454e06.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/722517eeab0b11f0b08552540044a08e.png)

3. Выберите соответствующий шаблон обратного вызова и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/973549e8ce9911f0aecd525400454e06.png)

## Отвязка шаблона обратного вызова

1. Перейдите в [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на целевое **имя домена отправки** или **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6da32782ce9911f093295254001c06ec.png)

2. Выберите **Конфигурация шаблона** и нажмите **Редактировать** в верхнем правом углу вкладки **Конфигурация обратного вызова**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c7162a9dce9911f093295254001c06ec.png)

3. Снимите флажок с связанного шаблона и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e037a90dce9911f098d1525400bf7822.png)

# Конфигурация интеллектуального сжатия прямых трансляций

Функция интеллектуального сжатия отключена по умолчанию. В этом документе приведены инструкции по привязке шаблона интеллектуального сжатия к определенному домену отправки для активации функции интеллектуального сжатия и по отвязке шаблона для деактивации интеллектуального сжатия.

## Примечания

- Шаблон вступает в силу примерно через 5-10 минут после его привязки.
- После успешной привязки шаблона функция интеллектуального сжатия будет включена для URL отправки под указанным доменом отправки.
- С доменом можно привязать только один шаблон интеллектуального сжатия. После привязки все потоки под этим доменом будут обработаны с использованием соответствующего шаблона интеллектуального сжатия.

## Предварительные требования

- Вы успешно вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat) и завершили [Добавление собственного домена](https://www.tencentcloud.com/document/product/267/35970).
- Шаблон интеллектуального сжатия успешно создан.

## Привязка шаблона интеллектуального сжатия

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **Имя домена**, который вы хотите настроить, или на кнопку **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e521558bc5b711f0a7ca5254001c06ec.png)

2. Выберите вкладку **Конфигурация шаблона**. Нажмите **Редактировать** в верхнем правом углу раздела **Конфигурация интеллектуального сжатия прямых трансляций**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/325e67f5c5bb11f086d9525400e889b2.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/47f9f52ec5bb11f0a7ca5254001c06ec.png)

3. Выберите шаблон конфигурации интеллектуального сжатия в соответствии с требованиями вашего бизнеса и нажмите **Подтвердить**, чтобы завершить конфигурацию.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7cb13782c5bb11f0a2c65254005ef0f7.png)

## Отвязка шаблона интеллектуального сжатия

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **Имя домена**, который вы хотите настроить, или на кнопку **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ec5244b3c5b711f0a7ca5254001c06ec.png)

2. Выберите вкладку **Конфигурация шаблона** и нажмите **Редактировать** в верхнем правом углу раздела **Конфигурация интеллектуального сжатия прямых трансляций**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8a3f8d2cc5bb11f0a62b5254007c27c5.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9d47b855c5bb11f0b88b525400bf7822.png)

3. В зависимости от требований вашего бизнеса снимите флажок с соответствующего шаблона и нажмите **Подтвердить**, чтобы продолжить.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6d0e8170c5bb11f086d9525400e889b2.png)

> **Примечание:** Отвязка шаблона интеллектуального сжатия не влияет на текущие прямые трансляции.

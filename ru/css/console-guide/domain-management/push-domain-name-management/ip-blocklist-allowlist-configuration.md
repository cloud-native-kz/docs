# Конфигурация списка блокировки/разрешения IP-адресов

В этом документе показано, как настроить список разрешенных/запрещенных IP-адресов для фильтрации запросов и управления доступом к потоковому контенту.

## Принцип работы

- **Список разрешенных IP-адресов**: Потоки в Cloud Streaming Services могут отправлять только настроенные IP-адреса.
- **Список блокировки IP-адресов**: Только настроенные IP-адреса имеют ограничение на отправку потоков в Cloud Streaming Services.

## Напоминания

- Список разрешенных/запрещенных IP-адресов вступает в силу примерно через десять минут после конфигурации.
- Чтобы конфигурация списка разрешенных/запрещенных IP-адресов применялась к активным потокам, необходимо перезагрузить потоки.

## Предварительные условия

- Вы активировали CSS и вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat).
- Вы добавили [домен push](https://intl.cloud.tencent.com/document/product/267/35970).

## Настройка списка разрешенных/запрещенных IP-адресов

1. На левой боковой панели выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на имя целевого домена воспроизведения или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/31ce70fc013211efbba352540095b445.png)

2. В разделе **Расширенная конфигурация** > **Список разрешенных/запрещенных IP-адресов** нажмите ![](https://staticintl.cloudcachetci.com/cms/backend-cms/75e67e17013211efbf03525400ae4d13.png), чтобы включить список разрешенных/запрещенных IP-адресов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/eedcf94cd3dd11f09f555254007c27c5.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/705b12f7d3e111f0a78a52540044a08e.png)

3. После включения **списка разрешенных/запрещенных IP-адресов** перейдите на страницу конфигурации **списка разрешенных/запрещенных IP-адресов** и выполните следующую конфигурацию:

Список блокировки

Список разрешенных

![](https://staticintl.cloudcachetci.com/cms/backend-cms/86aae7cdd3e111f09975525400bf7822.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8f7a4006d3e111f09f555254007c27c5.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Аутентификация по | Список разрешенных или запрещенных IP-адресов: невозможно выбрать оба одновременно. Если вы настроите список разрешенных IP-адресов, доступ к вашему потоковому контенту смогут получить только IP-адреса из списка. Если вы настроите список блокировки IP-адресов, IP-адреса из списка не смогут получить доступ к вашему потоковому контенту. |
| Список IP-адресов | Список блокировки IP-адресов поддерживает максимум 200 правил, а список разрешенных IP-адресов поддерживает до 500 правил. Пожалуйста, разделяйте записи символом новой строки. Список разрешенных/запрещенных IP-адресов поддерживает форматы IPv4, IPv6 и сегменты сети. |

4. Нажмите **Сохранить**, чтобы сохранить конфигурацию (конфигурация вступит в силу через некоторое время).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a5af2f1cd3e111f0a73852540099c741.png)

## Изменение списка разрешенных/запрещенных IP-адресов

1. На левой боковой панели выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на имя целевого домена воспроизведения или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7177425d013311efbf03525400ae4d13.png)

2. Выберите вкладку **Управление доступом**. В области **Список разрешенных/запрещенных IP-адресов** нажмите **Редактировать**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/00b9079cd3de11f0a73852540099c741.png)

3. Измените конфигурацию и нажмите **Сохранить**.

## Отключение списка разрешенных/запрещенных IP-адресов

1. На левой боковой панели выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на имя целевого домена воспроизведения или нажмите **Управление** справа, чтобы перейти на страницу управления доменом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8680ac48013311efb881525400a7e516.png)

2. Выберите вкладку **Управление доступом**. В области **Список разрешенных/запрещенных IP-адресов** нажмите ![img](https://staticintl.cloudcachetci.com/cms/backend-cms/0d0592e04eec11ee84f2525400494e51.png), чтобы отключить список разрешенных/запрещенных IP-адресов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/15c87680d3de11f09975525400bf7822.png)

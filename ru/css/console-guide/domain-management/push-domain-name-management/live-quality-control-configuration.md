# Конфигурация контроля качества в реальном времени

Функция контроля качества и оценки для трансляции в реальном времени отключена по умолчанию. Этот документ содержит рекомендации по включению функции контроля качества и оценки для указанного домена вещания путем связывания его с шаблоном контроля качества и оценки, а также по отвязыванию шаблона для отключения функции после успешной связи.

## Примечания

- Шаблон вступает в силу примерно через 5–10 минут после его связывания с доменом.
- После успешной связи шаблона включите функцию контроля качества и оценки для адреса трансляции в указанном домене вещания.
- Один домен может быть связан только с одним шаблоном контроля качества и оценки. После связи все потоки в этом домене будут соответствовать задачам контроля качества и оценки, определенным шаблоном.

## Предварительные условия

- Вы вошли в [консоль CSS](https://console.tencentcloud.com/live) и добавили [домен вещания](https://intl.cloud.tencent.com/document/product/267/35970).
- Вы создали [шаблон контроля качества и оценки](https://www.tencentcloud.com/document/product/267/73200#.E5.88.9B.E5.BB.BA.E6.97.B6.E7.A7.BB.E6.A8.A1.E6.9D.BF).

## Связывание шаблона контроля качества и оценки

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на целевой **домен вещания** или нажмите **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/300751aa79af11f0914f52540099c741.png)

2. Откройте вкладку **Конфигурация шаблона** и нажмите **Редактировать** в разделе **Конфигурация контроля качества трансляции**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7f134ffcab0711f09710525400e889b2.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a0b0508fab0711f0a68e5254001c06ec.png)

3. Выберите шаблон контроля качества и оценки и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3200f331ab0811f0b08552540044a08e.png)

## Отвязывание шаблона контроля качества и оценки

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на целевой **домен вещания** или нажмите **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3693cbc479af11f08255525400e889b2.png)

2. Откройте вкладку **Конфигурация шаблона** и нажмите **Редактировать** в разделе **Конфигурация контроля качества трансляции**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5c6ad4e6ab0811f0b5345254005ef0f7.png)

3. Отмените выбор шаблона и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6e01b169ab0811f096c2525400454e06.png)

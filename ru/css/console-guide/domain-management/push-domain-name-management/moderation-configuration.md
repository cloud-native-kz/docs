# Конфигурация модерации

Функция модерации при трансляции потока по умолчанию отключена. В этом документе содержатся инструкции по связыванию шаблона модерации с определенным доменом push для активации функции модерации и по разъединению шаблона для деактивации модерации.

## Примечания

- Шаблон вступает в силу примерно через 5-10 минут после его связывания.
- После успешного связывания шаблона функция модерации будет активирована для URL-адресов push в указанном домене push.
- С доменом можно связать только один шаблон модерации. После связывания все потоки в этом домене будут модерироваться в соответствии со связанным шаблоном.

## Предварительные условия

- Вы успешно вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat) и завершили [добавление собственного домена](https://www.tencentcloud.com/document/product/267/35970).
- Создан шаблон модерации.

## Связывание шаблона модерации

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **имя домена**, который нужно настроить, или на кнопку **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/afea20b5c5b711f0b88b525400bf7822.png)

2. Выберите вкладку **Конфигурация шаблона**. Нажмите кнопку **Редактировать** в верхнем правом углу раздела **Конфигурация модерации**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ac69c94a3c6311f0912c52540044a08e.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1ecab9343c6411f0912c52540044a08e.png)

3. Выберите шаблон конфигурации модерации в соответствии с требованиями вашего бизнеса и нажмите кнопку **Подтвердить**, чтобы завершить конфигурацию.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3d8436853c6411f0be2052540099c741.png)

## Разъединение шаблона модерации

1. Перейдите в раздел [Управление доменами](https://console.tencentcloud.com/live/domainmanage). Нажмите на **имя домена**, который нужно настроить, или на кнопку **Управление**, чтобы перейти на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b6b9c273c5b711f0a7ca5254001c06ec.png)

2. Выберите вкладку **Конфигурация шаблона** и нажмите кнопку **Редактировать** в верхнем правом углу раздела **Конфигурация модерации**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/549df8073c6411f08806525400bf7822.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/aa87b3a63c6411f09bbe525400454e06.png)

3. В зависимости от требований вашего бизнеса снимите флажок для соответствующего шаблона и нажмите кнопку **Подтвердить**, чтобы продолжить.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c71f57be3c6411f0b0ce5254007c27c5.png)

> **Примечание:** разъединение шаблона модерации не влияет на текущие прямые трансляции.

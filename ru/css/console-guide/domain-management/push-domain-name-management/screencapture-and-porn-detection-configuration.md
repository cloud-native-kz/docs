# Конфигурация функции скриншотирования и обнаружения контента

По умолчанию функция скриншотирования при трансляции отключена. В этой статье описано, как включить функцию скриншотирования для указанного домена отправки, установить связь с шаблоном скриншотирования, а также как отсоединить шаблон и отключить функцию скриншотирования.

## Примечания

- Конфигурация шаблона вступает в силу примерно через **5–10 минут**.
- После завершения конфигурации шаблона скриншотирования необходимо также настроить шаблон обратного вызова для получения результатов скриншотирования. Информацию о конфигурации шаблона обратного вызова см. в [Конфигурация обратного вызова](https://www.tencentcloud.com/document/product/267/31065).
- Домен может быть связан только с одним шаблоном скриншотирования. После связи все потоки в этом домене будут выполнять задачи скриншотирования в соответствии с этим шаблоном.

## Предварительные требования

- Вы вошли в [консоль CSS](https://console.tencentcloud.com/live) и добавили [домен отправки](https://intl.cloud.tencent.com/document/product/267/35970).
- Вы уже создали [Шаблон скриншотирования](https://www.tencentcloud.com/document/product/267/31072).

## Связывание шаблона скриншотирования

1. Перейдите в [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на целевой домен отправки или нажмите кнопку «Управление» для входа на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/791df0f32a2d11ef9130525400bf8054.png)

2. Выберите вкладку **Конфигурация шаблона** и нажмите кнопку **Редактировать** в верхнем правом углу вкладки **Конфигурация скриншотирования**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e57b41deab0811f0a0a052540099c741.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/02edd301ab0911f09b75525400bf7822.png)

3. Выберите шаблон конфигурации скриншотирования в соответствии с вашими фактическими потребностями.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4c6f9d85ab0911f09710525400e889b2.png)

4. Нажмите **Подтвердить**, чтобы завершить конфигурацию.

## Отсоединение шаблона скриншотирования

1. Перейдите в [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите на домен отправки для конфигурации или **Управление** для входа на страницу сведений о домене.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b544fc342a2d11efb8c45254005a8b94.png)

2. Выберите вкладку **Конфигурация шаблона** и нажмите кнопку **Редактировать** в верхнем правом углу вкладки **Конфигурация скриншотирования**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6a035afeab0911f09b75525400bf7822.png)

3. В соответствии с вашими фактическими потребностями снимите флажок с соответствующего шаблона и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7cacc718ab0911f0b08552540044a08e.png)

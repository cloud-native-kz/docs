# Адаптивный битрейт

С функцией адаптивного битрейта скорость воспроизведения прямой трансляции может плавно изменяться в зависимости от условий сети. Это обеспечивает плавное воспроизведение при изменяющихся условиях сети.

## Примечания

- После создания шаблона его можно привязать к доменному имени воспроизведения. Привязка вступает в силу за 5-10 минут.
- Доменное имя воспроизведения может быть привязано к **нескольким шаблонам адаптивного битрейта**, а шаблон адаптивного битрейта может быть привязан к **нескольким доменным имёнам воспроизведения**.
- Каждый шаблон адаптивного битрейта может содержать не более 15 потоков.
- Для работы функции адаптивного битрейта плеер должен поддерживать адаптивный битрейт.
- GOP для потоков шаблона адаптивного битрейта должен быть одинаковым.
- Кодек для потоков шаблона адаптивного битрейта должен быть одинаковым.
- Адреса воспроизведения адаптивного битрейта поддерживают только протоколы воспроизведения HLS и WebRTC. Правила конкатенации адресов см. в разделе [Генератор адресов](https://www.tencentcloud.com/document/product/267/31084?lang=en&pg=).
- Шаблон конфигурации трансцирования для адаптивного потокового воспроизведения поддерживает конфигурацию функции размытия лиц, которая может достичь размытия лиц и определённых объектов. Для использования этой функции необходимо [отправить заявку](https://console.tencentcloud.com/workorder/category). В шаблоне адаптивного битрейта, даже если несколько подпотоковых шаблонов имеют включённую функцию размытия лиц, плата будет взиматься только один раз. Включение этого сервиса повлечёт за собой начисление платежей за [трансцирование видео в прямом эфире](https://www.tencentcloud.com/document/product/267/39604) и платежей за [интеллектуальное распознавание](https://www.tencentcloud.com/document/product/1041/49204) сервиса обработки медиа (MPS).

## Создание шаблона адаптивного битрейта

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Feature Configuration > Adaptive Bitrate** (Конфигурация функций > Адаптивный битрейт) в левой боковой панели.
2. Нажмите **Create template** (Создать шаблон) и выполните следующие настройки:
  - Основная информация: Имя шаблона и описание. Подробнее см. в разделе [Основная информация](#259b6eab-5b7d-44ca-95c5-9483cddea573).
  - Информация о потоке: см. раздел [Информация о потоке](#e5c271c5-129d-4271-a204-903606dccbe7).
3. Нажмите **Add stream** (Добавить поток), чтобы добавить новый поток в шаблон. Можно добавить до 15 потоков для одного шаблона.
4. Нажмите **Save** (Сохранить).

### Основная информация

| Основная информация | Обязательно | Описание |
| --- | --- | --- |
| Имя шаблона | Да | Введите 1-10 символов. Имя шаблона адаптивного битрейта. Поддерживает только буквы и буквенно-цифровые комбинации, не поддерживает чистые цифры. Имя шаблона не должно совпадать с названиями существующих шаблонов трансцирования, шаблонов адаптивного битрейта или имёнами подпотоков. |
| Описание шаблона | Нет | Описание шаблона адаптивного битрейта. Может содержать только китайские иероглифы, буквы, цифры, пробелы, подчёркивания (_) и дефисы (-). |
| Живые субтитры | Нет | Функция субтитров отключена по умолчанию, но может быть активирована вручную. Для активации этой функции необходимо привязать шаблон субтитров. Выберите шаблон субтитров для привязки в соответствии с требованиями вашего бизнеса. Просмотрите и оцените эффекты шаблона субтитров. Вы можете в любое время отрегулировать шаблон субтитров в соответствии с потребностями бизнеса. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/721b5ac1c46411f08c0e52540044a08e.png)

### Информация о потоке

| Информация о потоке | Обязательно | Описание |
| --- | --- | --- |
| Тип трансцирования | Да | Доступные типы трансцирования включают стандартное трансцирование и TSC трансцирование. |
| Имя потока | Да | Введите 1-10 символов. Имя шаблона подпотока поддерживает 1-10 символов, допускает только буквы и буквенно-цифровые комбинации, не поддерживает чистые цифры. Имя подпотока не должно совпадать с названиями существующих шаблонов трансцирования, шаблонов адаптивного битрейта и имёнами подпотоков. |
| Качество видео | Нет | Можно выбрать **Smooth** (Плавное), **SD** (Стандартное), **HD** (Высокое) или **FHD** (Полное HD). После выбора значения система автоматически введёт рекомендуемый видеобитрейт и высоту, которые можно изменить. |
| Видеобитрейт (Кбит/с) | Да | Выходной видеобитрейт. Диапазон значений: 101-8000. Если вы вводите значение не более 1000, оно должно быть кратно 100. Если вы вводите значение больше 1000, оно должно быть кратно 500. |
| Разрешение видео (пиксели) | Да | По умолчанию вы устанавливаете **высоту** выходного видео. Вы также можете **установить длину короткой стороны**. Значение должно быть в диапазоне 2-3000 и должно быть кратно двум. Другая сторона будет масштабирована пропорционально. |
| DRM шифрование | Да | Отключено по умолчанию и может быть включено вручную. Для включения DRM шифрования сначала необходимо получить информацию о ключе в разделе [DRM Management](https://console.intl.cloud.tencent.com/live/config/drm). Схемы DRM включая Widevine, FairPlay и NormalAES поддерживаются для воспроизведения HLS. Для шифрования FairPlay необходимо загрузить сертификат, полученный от Apple, в ваш плеер. Типы шифрования: По умолчанию **Widevine**, опционально Fairplay и NormalAES. |
| Видеокодирование | Нет | По умолчанию выбран исходный кодек. Можно изменить на H.264, H.265, H.266 или AV1. Кодек для всех потоков одного шаблона адаптивного битрейта должен быть одинаковым. |
| Размытие лиц | Нет | При необходимости можно [отправить заявку](https://console.tencentcloud.com/workorder/category), чтобы включить эту функцию и активировать [сервис обработки медиа (MPS)](https://www.tencentcloud.com/document/product/1041/33482). Функция отключена по умолчанию и может быть включена вручную. Функция может использоваться для размытия лиц и определённых объектов со следующим эффектом: ![](https://staticintl.cloudcachetci.com/cms/backend-cms/35a98b68b11e11efbfb3525400bdab9d.png) Включение сервиса повлечёт за собой начисление платежей за [трансцирование видео в прямом эфире](https://www.tencentcloud.com/document/product/267/39604) и платежей за [интеллектуальное распознавание](https://www.tencentcloud.com/document/product/1041/49204) MPS. |
| Частота кадров видео (fps) | Нет | Можно выбрать сохранение исходной частоты кадров или установить частоту кадров видео. По умолчанию сохраняется исходная частота кадров. Диапазон установки частоты кадров видео: 1 fps - 60 fps. |
| GOP (секунды) | Нет | GOP не указан по умолчанию. Диапазон значений: 1-6. Чем выше GOP, тем выше задержка. Меньший GOP может потенциально привести к заиканию. GOP для всех потоков одного шаблона адаптивного битрейта должен быть одинаковым. |
| Ограничение параметров | Нет | Ограничения параметров отключены по умолчанию. После включения ограничения, если вы введёте значение выше исходного, будет использовано исходное значение. Это поможет избежать проблем с качеством видео, вызванных трансцированием видео низкого качества с использованием высоких параметров качества видео. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/dae4b6fe24a911f09e67525400bf7822.png)

## Привязка доменного имени

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration > Adaptive Bitrate** (Конфигурация функций > Адаптивный битрейт) в левой боковой панели.
2. Привяжите доменное имя одним из двух способов:
  - **Привязать доменное имя к существующему шаблону**: Нажмите **Bind Domain Name** (Привязать доменное имя) в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/81c7b596c46511f0a6dd5254005ef0f7.png)

  - **Привязать доменное имя после создания шаблона**: После [создания шаблона адаптивного битрейта](https://www.tencentcloud.com/document/product/267/50271#creating-an-adaptive-bitrate-template) нажмите **Bind Domain Name** (Привязать доменное имя) во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/fd009ed0c46411f0b4a7525400454e06.png)

3. Выберите **шаблон адаптивного битрейта** и **доменное имя воспроизведения** (можно одновременно привязать несколько доменных имён воспроизведения) и нажмите **Confirm** (Подтвердить).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1698f442c46511f0b4a7525400454e06.png)

## Отвязка доменного имени

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration > Adaptive Bitrate** (Конфигурация функций > Адаптивный битрейт) в левой боковой панели.
2. Выберите целевой шаблон и нажмите **Unbind** (Отвязать).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bc1d0de124aa11f0b0b1525400e889b2.png)

3. Во всплывающем окне нажмите **Confirm** (Подтвердить).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ed46f369b8f311eeb395525400461a83.png)

## Изменение шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration > Adaptive Bitrate** (Конфигурация функций > Адаптивный битрейт) в левой боковой панели.
2. Выберите целевой шаблон и нажмите **Edit** (Редактировать) справа, чтобы изменить его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/dbc1e8c424aa11f0b44b5254007c27c5.png)

3. После изменения нажмите **Save** (Сохранить).

## Удаление шаблона

> **Примечание:** Если шаблон привязан к доменам, необходимо [отвязать](#unbinding-a-domain-name) их перед удалением шаблона. Невозможно удалить шаблон трансцирования, у которого включена функция Live subtitles (Живые субтитры). После удаления шаблона его нельзя восстановить. Действуйте осторожно.

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration > Adaptive Bitrate** (Конфигурация функций > Адаптивный битрейт) в левой боковой панели.
2. Выберите целевой шаблон (убедитесь, что он не привязан к доменному имени) и нажмите **Delete** (Удалить).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/fb8e48f424aa11f0a62e525400454e06.png)

3. Во всплывающем окне нажмите **Confirm** (Подтвердить).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1741fa1624ab11f0b44b5254007c27c5.png)

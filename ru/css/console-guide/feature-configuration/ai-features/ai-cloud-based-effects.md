# Облачные AI эффекты

Функция облачных AI эффектов в Cloud Streaming Services (CSS) интегрируется с технологией AI преобразования текста в видео, позволяя пользователям генерировать персонализированные видео со специальными эффектами в реальном времени путем ввода текстовых описаний (Prompts). Пользователи также могут управлять списком сгенерированных специальных эффектов через консоль или API, а также отправлять видео со специальными эффектами в указанный прямой эфир. Эта функция повышает интерактивность пользователей, создавая уникальные методы эмоционального выражения как для трансляторов, так и для аудитории. По сравнению с традиционными фиксированными специальными эффектами подарков, она обладает повышенной интерактивностью.

В этой статье показано, как отправлять эффекты через консоль.

## Важная информация

- Использование функции облачных AI эффектов CSS будет сопровождаться двумя платежами: один за генерацию видео со специальными эффектами и другой за его отправку. Конкретные комиссии рассчитываются на основе фактического использования. Подробные инструкции по выставлению счетов см. в [Документации](https://www.tencentcloud.com/document/product/267/70336).
- Видео со специальными эффектами генерируются большими моделями и имеют определенную степень случайности. Визуальные эффекты будут постепенно улучшаться с технологической эволюцией и итерациями версий.
- Использование Prompts, содержащих запрещенные чувствительные слова (такие как насилие и незаконные чувствительные слова), строго запрещено.

## Предварительные требования

Вы активировали сервисы потокового вещания Tencent Cloud.

## Отправка специального эффекта

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и перейдите в раздел AI Features > Cloud AI Special Effects, нажмите **Send Special Effect**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/debe8734ca7611f0aedb525400454e06.png)

2. На открывшейся странице проверьте Online Live Stream и нажмите **Confirm** для отправки специального эффекта.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5f98a88c4dc211f09bf25254005ef0f7.png)

3. Вы также можете нажать кнопку **Send Special Effect and Preview** для предварительного просмотра.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/75b2a64e4dc211f08a7252540099c741.png)

4. Нажмите **Cancel** для остановки отправки специального эффекта.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c925460d4fd511f0b5e7525400e889b2.png)

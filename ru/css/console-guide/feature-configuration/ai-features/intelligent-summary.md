# Интеллектуальное резюме

Интеллектуальное резюме прямой трансляции основано на технологии искусственного интеллекта. Эта функция выполняет мультимодальный глубокий анализ содержимого прямой трансляции для интеллектуального создания резюме и структур.

В этом документе описано, как создавать, изменять и удалять шаблоны интеллектуального резюме через консоль.

## Моменты, требующие внимания

После успешного создания и привязки шаблона его вступление в силу займет примерно 5-10 минут.

## Предусловия для использования

Служба CSS была активирована.

## Создание шаблона резюме

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** > **AI Features** > **Intelligent Summary**.

> **Примечание****：** Для использования **Intelligent Summary** в разделе **AI Features** при первом создании необходимо создать роль службы и авторизовать роль текущей учетной записи для использования продуктов **Media Processing Service** (MPS) и **Cloud Object Storage** (COS).

2. Нажмите кнопку **Authorize**, чтобы перейти на страницу управления ролями CAM.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/fbe634d0c5f111f0a62b5254007c27c5.png)

3. На странице управления ролями нажмите кнопку **Authorize**. После завершения проверки личности вы сможете авторизовать Media Processing Service и Cloud Object Storage (COS) и использовать эти службы в обычном режиме.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2e5b0db9c5f211f0a62b5254007c27c5.png)

4. После успешной авторизации выберите соглашение об обслуживании и нажмите кнопку **Start**. Система автоматически активирует продукт Media Processing Service и Cloud Object Storage (COS), а затем перенаправит вас на страницу управления Intelligent Summary.
5. Нажмите кнопку **Create Summary Template**, чтобы перейти на страницу создания, и выполните конфигурацию следующим образом:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2c1c61c100ca11f18080525400074c32.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Template Name | Настраивается (максимум 30 символов; поддерживает буквы, цифры, подчеркивания и дефисы). |
| Template Description | Настраивается (максимум 100 символов; поддерживает китайские иероглифы, буквы, цифры, пробелы и _-). |
| Summary Mode | В зависимости от требований вашего бизнеса вы можете выбрать между режимами Intelligent Summary или Scheduled Summary: Intelligent Summary: система автоматически анализирует содержимое прямой трансляции и создает интеллектуальные сегментированные резюме с гибкими интервалами времени. Scheduled summary: создание резюме через фиксированные интервалы времени. По умолчанию интервал составляет 5 минут, доступны варианты интервалов 5, 10, 20 или 30 минут. |
| Character Quantity of Summary | Количество символов в резюме должно быть между 20 и 200. |
| Input Language | По умолчанию поддерживается автоматическое определение (китайский/английский), но вы также можете вручную выбрать китайский, английский, японский или корейский язык. |
| Output Language | По умолчанию поддерживается исходный язык, но вы также можете вручную выбрать китайский, английский, японский или корейский язык. |
| Keyword Generation | Включено по умолчанию. Создание ключевых слов на основе содержимого резюме. |
| Title Generation | Включено по умолчанию. Автоматическое создание названий резюме. |

## Привязка шаблона резюме

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** > **AI Features** > **Intelligent Summary**.
2. Привяжите доменное имя одним из двух способов:
  - **Привязка доменного имени к существующему шаблону**: нажмите кнопку **Bind Domain Name** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/49e588e000ca11f18e41525400ecee81.png)

  - **Привязка доменного имени после создания шаблона резюме**: после успешного создания шаблона резюме нажмите кнопку **Bind Domain Name** в появившемся окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2e45d452c53211f0a62b5254007c27c5.png)

3. В окне привязки доменного имени выберите **summary template** и **push domain** (одновременно может быть привязано несколько доменных имен push) и нажмите кнопку **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/41d073a1c53211f0a1dc52540044a08e.png)

## Отмена привязки

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** > **AI Features** > **Intelligent Summary**.
2. Выберите шаблон резюме, связанный с доменным именем, и нажмите кнопку **Unbind**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/873a5211c91511f08942525400e889b2.png)

3. Подтвердите, требуется ли отмена привязки текущего связанного доменного имена. Нажмите кнопку **Confirm**, чтобы отменить привязку.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/701603adc53211f0a7ca5254001c06ec.png)

## Изменение шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** > **AI Features** > **Intelligent Summary**.
2. Выберите успешно созданный шаблон резюме и нажмите кнопку **Edit** справа, чтобы перейти на страницу изменения информации шаблона. Нажмите кнопку **Save**, чтобы завершить изменение.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9de66bb1c91511f09e745254007c27c5.png)

## Удаление шаблона

> **Примечание****：** Если шаблон уже был связан, вы должны сначала [отменить привязку](#f17f9f28-c1f5-4975-85ab-156766c34c00), а затем приступить к процессу удаления. Обратите внимание, что шаблон не может быть восстановлен после удаления. Будьте осторожны при выполнении операций.

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** > **AI Features** > **Intelligent Summary**.
2. Выберите целевой шаблон (убедитесь, что он не привязан к доменному имени) и нажмите кнопку **Delete**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b0c7dcbfc91511f09e745254007c27c5.png)

3. Подтвердите, требуется ли удаление текущего шаблона резюме, затем нажмите кнопку **Confirm**, чтобы успешно удалить его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/966ddf99c53211f0a1dc52540044a08e.png)

# Управление лексиконом

Лексикон состоит из двух отдельных категорий: Словарь терминов и Словарь горячих слов. Словарь терминов облегчает правила перевода для специфической лексики (такой как названия продуктов, неоднозначные термины и заимствованные слова) в сценариях кросс-лингвистического перевода, обеспечивая точность перевода. Тем временем Словарь горячих слов значительно повышает точность распознавания речи для собственных терминов (включая бренды и техническую терминологию) путем включения взвешенных записей. При столкновении с омофонами с идентичными тонами система приоритизирует совпадения с более высоко взвешенными горячими словами, что позволяет более точное и профессиональное создание трансляционных субтитров в реальном времени.

## Важные моменты

- Система будет определять и заменять горячие слова на основе их веса и частоты появления. Чем больше вес и ниже частота, тем выше вероятность совпадения.
- В настоящее время лексиконы поддерживают только конфигурацию китайского и английского языков. Содержание лексикона вступает в силу через 10 минут после каждого обновления.
- Словарь терминов может переводить терминологию клиента в конкретной области.

## Создание библиотеки

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и нажмите **Конфигурация функций** > **Функции ИИ** > **Субтитры и синхронный перевод**, затем нажмите **Управление лексиконом**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/811a3a3700c911f18e41525400ecee81.png)

2. Нажмите **Создать библиотеку**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f803b245b3c711f0a6a2525400e889b2.png)

3. В появившемся окне заполните информацию конфигурации. Категория: Словарь горячих слов по умолчанию (опционально Словарь терминов). Подробную конфигурацию см. в следующей таблице.

Словарь горячих слов

Словарь терминов

![](https://staticintl.cloudcachetci.com/cms/backend-cms/55fafa3eb3c811f0bd1d5254001c06ec.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Категория | Словарь горячих слов: для горячих слов с одинаковым произношением и тоном распознавать горячее слово с наибольшим весом. |
| Библиотека | Префикс имени библиотеки всегда "hotword". Поддерживает только английские буквы, цифры, подчеркивания (_) и дефисы (-), содержит до 30 символов. |
| Описание | Поддерживает китайские иероглифы, буквы, цифры, пробелы и _-. Описание не может превышать 100 символов. |
| Прямой импорт | Включите эту опцию, если вы хотите импортировать горячие слова из файла. Нажмите **Выбрать файл**, а затем выберите файл на вашем компьютере. Убедитесь, что файл соответствует следующим требованиям: Формат файла: TXT. Размер файла: в пределах 100 КБ. Кодировка файла: кодировка UTF-8 или GBK. |
| Ключевые слова | Добавьте горячие слова здесь. Поддерживаются только китайские и английские горячие слова. Каждое горячее слово может содержать не более 10 китайских иероглифов или 30 английских символов. Пунктуационные знаки и специальные символы не допускаются. Несколько горячих слов должны быть разделены запятыми, а количество горячих слов не может превышать 1000. Горячие слова и веса должны быть разделены "\|". Например, "Tencent Cloud\|10,speech recognition\|5,ASR\|10". Вес горячего слова варьируется от 1 до 10. Чем больше вес горячего слова, тем больше вероятность того, что горячее слово будет распознано. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8776d600b3c711f09e195254007c27c5.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Категория | Словарь терминов: переводить специальные термины для клиентов и исправлять результаты перевода. |
| Библиотека | Префикс библиотеки зафиксирован как "term". Поддерживает только английские буквы, цифры, подчеркивания (_) и дефисы (-), содержит до 30 символов. |
| Описание | Поддерживает китайские иероглифы, буквы, цифры, пробелы и _-. Описание не может превышать 100 символов. |
| Языковая пара | Направления языковой конвертации: поддерживает двусторонний перевод между китайским и английским языками, включая следующие варианты преобразования: Китайский → АнглийскийАнглийский → Китайский |
| Двусторонний | На основе ваших бизнес-требований вы можете выбрать, включить ли двусторонний перевод между китайским и английским языками. Параметры: Да/Нет (Значение по умолчанию: "Да") |
| Прямой импорт | В зависимости от ваших фактических бизнес-потребностей вы можете загрузить файлы с вашего компьютера или использовать функцию "Общий словарь". **Выбрать файл** Если вам нужно импортировать терминологию, вы можете вручную включить функцию прямого импорта терминов. Нажмите "Выбрать файл" и выберите файл для загрузки на вашем компьютере. Пожалуйста, убедитесь, что загруженный файл соответствует следующим требованиям: Формат файла: TXT. Размер файла: в пределах 100 КБ. Кодировка файла: кодировка UTF-8 или GBK. **Общий словарь** Вы можете напрямую выбрать предустановленный словарь. |
| Ключевые слова | Добавьте горячие слова здесь. Несколько терминов должны быть разделены запятыми, а количество терминов не может превышать 150. Разделяйте термины на исходном языке и те, которые на целевом языке с помощью "\|". Например, "Tencent Cloud\|腾讯云,ASR\|语音识别,P&L\|盈亏,API\|API". Термины чувствительны к регистру. |

4. Нажмите **Подтвердить**, и горячие слова будут добавлены.

## Просмотр библиотеки

На странице [Управление лексиконом](https://console.tencentcloud.com/live/config/subtitle/hot-word) нажмите на имя библиотеки, которую вы хотите просмотреть, с левой стороны и просмотрите её подробную информацию во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/46827bcab3ce11f0bb7b525400bf7822.png)

Информация включает имя библиотеки, ID таблицы лексикона, время последнего обновления, количество горячих слов и список горячих слов с их весами.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/68c7bd3cb3ce11f0b96752540044a08e.png)

## Изменение библиотеки

1. На странице [Управление лексиконом](https://console.tencentcloud.com/live/config/subtitle/hot-word) найдите библиотеку, которую вы хотите изменить, нажмите **Редактировать** с правой стороны, а затем измените информацию конфигурации библиотеки во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e66f5e3eb3cf11f099275254005ef0f7.png)

2. Нажмите **Подтвердить**, чтобы сохранить текущий шаблон и завершить изменение пользовательской библиотеки.

## Удаление библиотеки

1. На странице [Управление лексиконом](https://console.tencentcloud.com/live/config/subtitle/hot-word) найдите библиотеку, которую вы хотите удалить, а затем нажмите **Удалить** с правой стороны.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/14e9274db3d011f0a6a2525400e889b2.png)

2. Появится окно подтверждения. Нажмите **OK**, чтобы удалить пользовательскую библиотеку.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cbda10da6e5a11ef92db52540055f650.png)

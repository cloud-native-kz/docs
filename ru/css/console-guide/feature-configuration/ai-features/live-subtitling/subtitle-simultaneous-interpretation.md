# Субтитры и одновременный перевод

Функция «Субтитры и AI дубляж» может выполнять распознавание речи в реальном времени (ASR) при прямой трансляции, преобразовывать речь в субтитры и поддерживает перевод на целевые языки, а также AI дубляж. В настоящее время эта функция предоставляет услуги перевода речи для взаимного перевода между несколькими типами языков. В практических приложениях, пожалуйста, выбирайте соответствующую комбинацию исходного и целевого языков в соответствии с потребностями вашего бизнеса и профилем аудитории.

В этом документе описывается, как создавать, изменять и удалять шаблоны субтитров через консоль.

## Моменты внимания

- Шаблон вступает в силу примерно через 5–10 минут после успешного создания.
- Существует два способа использования функции субтитров и AI дубляжа:
  - Способ 1: привяжите шаблон субтитров и AI дубляжа к шаблону трансодирования, затем добавьте соответствующее имя шаблона трансодирования к URL-адресу трансляции, чтобы получить поток субтитров;
  - Способ 2: добавьте параметр трансляции `SubtitleTemplateName=SubtitleTemplateName` в конец URL-адреса трансляции. Пример URL-адреса трансляции приведен ниже:

`http://domain/AppName/StreamName.flv?txSecret=Md5(key+StreamName+hex(time))&txTime=hex(time)&SubtitleTemplateName=subtitletest`

- Функция вещания субтитров в прямом эфире — это **премиум-сервис**. Использование этой функции приведет к взиманию плат за [прямое трансодирование](https://www.tencentcloud.com/document/product/267/39604) и платы за ASR от MPS. Перевод между языками может привести к возникновению платы за голосовой перевод от MPS. Подробные правила выставления счетов см. в [документе по выставлению счетов](https://www.tencentcloud.com/document/product/1041/49204?lang=zh).

## Предварительные требования для использования

Сервис CSS был активирован.

## Создание шаблона субтитров

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** (Конфигурация функций) > **AI Features** (Функции AI) > **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод).

> **Примечание:** Для использования **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод) в разделе **AI Features** (Функции AI) необходимо при первом создании создать роль сервиса и авторизовать текущую роль учетной записи для использования продуктов **Media Processing Service** (MPS) и **Cloud Object Storage** (COS).

2. Нажмите **Authorize** (Авторизовать), чтобы перейти на страницу управления ролями CAM.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/aa5fc060c9da11f08942525400e889b2.png)

3. На странице управления ролями нажмите **Authorize** (Авторизовать). После завершения проверки личности вы сможете авторизировать Media Processing Service и Cloud Object Storage (COS) и нормально использовать сервисы Media Processing Service и Cloud Object Storage.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d77acb61c9da11f08942525400e889b2.png)

4. После успешной авторизации выберите соглашение об обслуживании и нажмите **Start** (Начать). Система автоматически активирует продукт Media Processing Service и Cloud Object Storage (COS) и перенаправит вас на страницу управления Smart Highlight.
5. Нажмите **Create template** (Создать шаблон), чтобы перейти на страницу создания шаблона «Субтитры и одновременный перевод» и настройте шаблон следующим образом:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7e953a9400a011f198c8525400370dda.png)

> **Примечание:** Выбор шрифта для субтитров будет варьироваться в зависимости от выбранного вами **Subtitle Translation** (Перевод субтитров) и **Subtitles** (Субтитры). Выберите соответствующий исходный тип языка, целевой язык и субтитры в соответствии с вашими реальными потребностями.

| Элемент конфигурации | | Описание |
| --- | --- | --- |
| Template Name (Имя шаблона) | | Префикс имени шаблона фиксирован как «subtitle». Имя шаблона не может превышать 30 символов и поддерживает только английские буквы и цифры. |
| Template Description (Описание шаблона) | | Описание шаблона субтитров. Содержит до 100 символов, поддерживает только буквы, цифры, подчеркивания и дефисы. |
| Mode (Режим) | | Режим по умолчанию — Subtitles (Субтитры); вы также можете выбрать Subtitles and Dubbing (Субтитры и дубляж). |
| Language (Язык) | | Исходный язык по умолчанию — китайский. Также поддерживаются английский, японский и корейский языки. По умолчанию установлено значение «No Selection» (Без выбора). Поддерживается переключение на любой из других невыбранных вариантов в исходном языке. Например, если выбран английский язык в качестве исходного, доступные варианты для целевого языка включают: No Selection (Без выбора), Chinese (Китайский), Japanese (Японский) и Korean (Корейский). Перевод между языками приведет к взиманию платы за голосовой перевод от MPS. Плата за ASR будет взиматься без перевода. Для получения подробной информации см. [Content II](https://www.tencentcloud.com/document/product/1041/49204). Если вам необходимо перевести на другие языки, пожалуйста, [отправьте запрос в службу поддержки](https://console.intl.cloud.tencent.com/workorder/category). |
| Preset styles (Предустановленные стили) | | При выборе предустановленных стилей система автоматически адаптирует **Font Color** (Цвет шрифта), **Subtitle Line Count** (Количество строк субтитров), **Characters per Line** (Количество символов в строке) и **Margins and Line Spacing** (Поля и межстрочный интервал). В настоящее время поддерживаются следующие предустановленные стили: вариант по умолчанию — Small text without bar (Маленький текст без полосы). Вы можете выбрать Small text without bar (Маленький текст без полосы), Large text without bar (Большой текст без полосы), Small text with bar (Маленький текст с полосой) или Large text with bar (Большой текст с полосой) для настройки параметров. После выбора система автоматически установит соответствующий стиль субтитров, который можно изменить. Изменение конфигурации приведет к очистке выбранных предустановленных стилей. |
| Style Adjustment (Настройка стиля) | Subtitles (Субтитры) | Source (Источник) поддерживается по умолчанию. Также поддерживаются Target (Целевой) и Bilingual Subtitle (Двуязычный субтитр). Выбирая три варианта Source language (Исходный язык), Target language (Целевой язык) и Subtitles (Субтитры), вы можете получить эффекты субтитров на разных языках. |
| | Subtitle Line Count (Количество строк субтитров) | Варианты: **1** или **2**. Когда строки превышают отображаемый диапазон, будет отображаться только последнее содержимое. |
| | Characters per line (Количество символов в строке) | Диапазон значений: 1 - 200. Если для предустановленных стилей установлено значение «Small text without bar» (Маленький текст без полосы) или «Small text with bar» (Маленький текст с полосой), количество символов в строке по умолчанию составляет 65. Если для предустановленных стилей установлено значение «Large text without bar» (Большой текст без полосы) или «Large text with bar» (Большой текст с полосой), количество символов в строке по умолчанию составляет 25. Один китайский иероглиф считается как 1, в то время как один английский символ или цифра считается как половина. Чем меньше символов в строке, тем больше размер шрифта. |
| | Target Font (Целевой шрифт) | Выбор шрифта для субтитров будет варьироваться в зависимости от выбранного вами **Subtitle Translation** (Перевод субтитров) и **Subtitles** (Субтитры). Вы можете выбрать различные шрифты, такие как DIN Alternate Bold, Helvetica или HelveticalnseratLTPro-Roman, или выбрать пользовательский шрифт в соответствии с потребностями вашего сервиса. Если вам требуется пользовательский шрифт, нажмите **Custom** (Пользовательский) для загрузки шрифта. Система отобразит диалоговое окно подтверждения. Перед загрузкой и использованием шрифта проверьте уведомление, а затем нажмите **OK** (ОК). Обратите внимание, что поддерживаются только файлы .ttf. Перед использованием пользовательского шрифта убедитесь, что вы получили надлежащую авторизацию, в противном случае Tencent Cloud не несет ответственность за любые судебные издержки, вытекающие из использования. Цвет шрифта по умолчанию — **white** (белый), и вы можете настроить цвет шрифта в соответствии со своими предпочтениями. Если для параметра «Subtitles» (Субтитры) установлено значение «Bilingual Subtitle» (Двуязычный субтитр), поддерживается настройка различных шрифтов и цветов для каждого языка отдельно. |
| | Subtitle Background (Фон субтитров) | Вы можете выбрать предустановленный стиль или загрузить пользовательский фон в соответствии с потребностями вашего бизнеса. После выбора вы можете просмотреть эффект. Инструкции по пользовательской загрузке: загрузите файл в формате PNG с размером файла, не превышающим 1 МБ. |
| | Dynamic/Steady State Effect (Динамический/стационарный эффект) | Параметр по умолчанию — **delayed steady-state sentence-by-sentence subtitles** (отложенные стационарные субтитры предложение за предложением). Вы можете переключиться на субтитры в реальном времени или отложенные стационарные субтитры слово за словом. При выборе режима «Delayed Steady State Subtitles» (Отложенные стационарные субтитры) необходимо установить задержку с значением по умолчанию 10 секунд. Доступные варианты задержки включают 20 секунд, 30 секунд и 60 секунд. **Примечание:** При выборе субтитров в реальном времени субтитры при прямой трансляции будут динамически исправляться слово за словом на основе изменений речи. При выборе отложенных стационарных субтитров слово за словом система будет задерживать отображение прямой трансляции на основе установленного времени, но режим полного предложения обеспечивает лучший опыт просмотра. |
| | Lexicon (Лексикон) | Функция Association with Lexicon (Связь с лексиконом) отключена по умолчанию. После включения функции пользовательских лексиконов горячих слов необходимо выбрать лексикон горячих слов, который вы хотите применить. Если вы еще не создали лексикон горячих слов, вы можете нажать Add (Добавить) для перехода на страницу Manage Lexicon (Управление лексиконом). Подробные инструкции см. в разделе [Manage Lexicon](https://www.tencentcloud.com/document/product/267/64435#3595e9e7-4a23-4836-b380-ba4edabd416d). |
| Real-Time Interpretation (Перевод в реальном времени) | | Функция Real-Time Interpretation (Перевод в реальном времени) отключена по умолчанию и будет отображаться только при одновременном выполнении следующих условий: Выбранный режим — «Subtitle and Dubbing» (Субтитры и дубляж). Эффект субтитров установлен на «Delayed steady-state sentence-by-sentence subtitles» (Отложенные стационарные субтитры предложение за предложением). Тип субтитров — «Target» (Целевой) или «Bilingual Subtitle» (Двуязычный субтитр). **Примечание:** Если эта функция включена, система будет интерпретировать исходный звук в целевой звук с выбранным эффектом голоса. Эта функция приведет к взиманию платы за распознавание [fees](https://www.tencentcloud.com/document/product/267/39604?lang=zh&pg=). Правила выставления счетов см. в [Billing Documentation](https://www.tencentcloud.com/document/product/1041/49204). В настоящее время эта функция не поддерживает корейский язык. Вы можете выбрать между Cloned timbre (Клонированный тембр) или Specified timbre (Указанный тембр) в зависимости от конкретных потребностей вашего бизнеса. Specified timbre (Указанный тембр): нажимая и перетаскивая мышь над соответствующим значком воспроизведения языка, вы можете прослушать эффекты перевода голоса в реальном времени. Если целевой язык установлен на английский, Real-Time Interpretation поддерживает следующий голос: WeRose. Нажимая и перетаскивая мышь над соответствующим значком воспроизведения языка, вы можете прослушать эффекты перевода голоса в реальном времени. |
| Preview (Предпросмотр) | | Вы можете отрегулировать эффект отображения субтитров, перетаскивая и масштабируя положение и размер типа содержимого субтитров и фона субтитров. Загрузка фоновых изображений, выбор разрешения окна предпросмотра и ввод тестового содержимого субтитров может помочь вам просмотреть стили субтитров. Эффект субтитров предназначен только для справки, и тестовое содержимое не будет переводиться. |

6. Нажмите **Save** (Сохранить), чтобы сохранить текущий шаблон.

## Привязка шаблона трансодирования

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** (Конфигурация функций) > **AI Feature** (Функция AI) > **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод).
2. Система позволяет привязывать шаблоны субтитров к шаблонам трансодирования или шаблонам адаптивного битрейта. И шаблоны трансодирования, и шаблоны адаптивного битрейта имеют аналогичные последующие процедуры использования. Шаблон трансодирования служит примером описания ниже:
  2.1. Прямая связь с шаблонами трансодирования:
    2.1.1. Свяжите шаблон трансодирования после успешного его создания: после успешного создания [шаблона трансодирования](https://www.tencentcloud.com/document/product/267/31071#creating-a-transcoding-template) нажмите **Bind transcoding template** (Привязать шаблон трансодирования) в левом верхнем углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d19a9bd900a011f18e41525400ecee81.png)

    2.1.2. Выберите **Transcoding template** (Шаблон трансодирования) или **Adaptive bitrate template** (Шаблон адаптивного битрейта) в соответствии с вашими реальными требованиями к бизнесу, затем нажмите **Confirm** (Подтвердить).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f397170c00a011f1b5c0525400380f7d.png)

  2.2. Для привязки [playback domain](https://www.tencentcloud.com/document/product/267/31071#binding-a-domain-name) к соответствующему шаблону трансодирования эффекты субтитров будут выводиться одновременно при последующем получении потока трансодирования (добавьте _ имя шаблона трансодирования после соответствующего имени потока прямой трансляции StreamName для создания адреса потока трансодирования).

> **Примечание:** После привязки шаблона трансодирования функция субтитров, настроенная в шаблоне трансодирования, будет синхронно активирована.

## Отмена привязки

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** (Конфигурация функций) > **AI Features** (Функции AI) > **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод).
2. Выберите шаблон трансодирования, связанный с доменом, и нажмите **Unbind** (Отмена привязки).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/13f1493c00a111f18080525400074c32.png)

3. Подтвердите, требуется ли отменить привязку текущего связанного домена. Нажмите **Confirm** (Подтвердить) для отмены привязки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cf79ee9df64711eea3bf525400fe11be.png)

## Изменение шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** (Конфигурация функций) > **AI Features** (Функции AI) > **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод).
2. Выберите успешно созданный шаблон субтитров и нажмите **Edit** (Редактировать) справа, чтобы перейти на страницу изменения информации о шаблоне. Нажмите **Save** (Сохранить), чтобы завершить изменение.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/24542a4f00a111f18080525400074c32.png)

## Удаление шаблона

> **Примечание:** Если шаблон уже был связан, необходимо сначала [отменить привязку](https://www.tencentcloud.com/document/product/267/31071#deleting-a-template), прежде чем приступить к процессу удаления. Обратите внимание, что шаблон невозможно восстановить после удаления. Будьте осторожны при выполнении операций.

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Feature Configuration** (Конфигурация функций) > **AI Features** (Функции AI) > **Subtitle & Simultaneous Interpretation** (Субтитры и одновременный перевод).
2. Выберите успешно созданный шаблон субтитров и нажмите **Delete** (Удалить) выше.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/39e54e4600a111f18e41525400ecee81.png)

3. Подтвердите, требуется ли удалить текущий шаблон субтитров, затем нажмите **Confirm** (Подтвердить), чтобы успешно удалить его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/46711b0100a111f183755254001d6acc.png)

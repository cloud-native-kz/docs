# Интеллектуальное распознавание ROI

Распознавание ROI (область интереса) может определять позиции важных визуальных элементов в видео в реальном времени, таких как лица, персонажи игры или ведущие потокового вещания, и отправлять эту информацию вместе с видео на устройство воспроизведения. Используя информацию ROI, плеер может выполнять такие операции, как размытие фона сцены и предотвращение перекрытия важных элементов видео текстовыми комментариями. В этом документе объясняется, как создавать, изменять и удалять шаблоны распознавания ROI в консоли.

## Примечания

- Шаблон вступает в силу примерно через 5-10 минут после его создания.
- Чтобы использовать функцию распознавания ROI, необходимо добавить параметр `roirecognition`= имя конфигурации ROI к URL трансляции. Это позволяет плееру получать доступ и обрабатывать данные ROI из прямой трансляции, что может включить такие функции, как размытие фона и предотвращение перекрытия важных частей видео текстовыми комментариями. Подробные инструкции см. в разделе [Практика функции интеллектуального распознавания ROI](https://www.tencentcloud.com/document/product/267/60146). Пример URL трансляции:

`rtmp://domain/AppName/StreamName?`

`txSecret=Md5(key+StreamName+hex(time))&txTime=hex(time)&roirecognition=Template Name`

- Функция распознавания ROI является **платной дополнительной услугой**. Использование этой функции влечет за собой сборы за транскодирование прямых трансляций и сборы за интеллектуальное распознавание содержимого Media Processing Service (MPS). Подробные правила выставления счетов см. в документации по выставлению счетов.

## Предварительные требования

Вы активировали Tencent Cloud Streaming Services.

## Создание шаблона конфигурации ROI

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и перейдите в раздел **AI Features >** **ROI Intelligent Recognition.**

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f9801263ca7511f0a7775254005ef0f7.png)

> **Примечание:** Чтобы использовать функцию интеллектуального распознавания ROI в Live Streaming Lab, **при первом создании** шаблона необходимо также создать роль сервиса и авторизовать текущий аккаунт для использования MPS. Нажмите **Authorize Now**, чтобы перейти в CAM для авторизации.

2. Нажмите **Authorize Now**, чтобы перейти на страницу управления ролями CAM.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/133b6c3efdf111ee9f79525400a7e516.png)

3. На странице управления ролями нажмите **Grant**, чтобы завершить проверку личности и завершить авторизацию Media Processing Service, обеспечивая нормальное использование Media Processing Service.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4c3a2e68fe0111ee83bb525400ae4d13.png)

4. После успешной авторизации проверьте соглашение об услугах и нажмите **Start**, и система автоматически активирует продукт MPS и откроет страницу управления Intelligent Streaming Media Processing.
5. Перейдите на страницу управления интеллектуальным распознаванием ROI и нажмите **Create ROI template**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1f0fa4ebca7611f09e745254007c27c5.png)

6. Перейдите на страницу конфигурации ROI и выполните следующую конфигурацию:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2ef6b341349711f0a927525400e889b2.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Name | К имени шаблона автоматически добавляется префикс "roi". Имя шаблона может содержать 1-10 символов (поддерживаются только комбинации букв и цифр). |
| Description | Поддерживает только китайский, английский языки, цифры, пробелы, подчеркивания (_), дефисы (-) и может быть до 100 символов в длину. |
| Training Model | По умолчанию — **General**. Поддерживаемые модели обучения включают: Honor of Kings, NBA2K Game и Live Shows. General: способна распознавать обычные области фокуса человеческого взгляда в различных окружениях. Honor of Kings: способна распознавать роли героев и зоны в различных окружениях в игре Honor of Kings. NBA2K Game: способна распознавать игроков, баскетбольные мячи, табло и другие зоны в различных окружениях в игре NBA2K. Live Shows: способна распознавать лицо ведущего. **Примечание:** Система может идентифицировать такие элементы, как лица и персонажи игр в видео. Выбор подходящей модели обучения для вашего сценария может значительно улучшить точность интеллектуального распознавания ROI. Если предоставленные модели не соответствуют потребностям вашего конкретного сценария, вы можете [отправить тикет](https://console.tencentcloud.com/workorder/category) для запроса модели. |

7. После заполнения элементов конфигурации нажмите **Confirm** для завершения.

## Изменение шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и перейдите в раздел **AI Features >** **ROI Intelligent Recognition**.
2. Выберите успешно созданный шаблон конфигурации ROI и нажмите **Edit** справа для изменения информации шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/472f5567ca7611f09e745254007c27c5.png)

3. Нажмите **Confirm** для завершения.

## Удаление шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и перейдите в раздел **AI Features > ROI Intelligent Recognition**.
2. Выберите успешно созданный шаблон конфигурации ROI и нажмите **Delete** справа.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/600ef85dca7611f0b011525400bf7822.png)

3. Нажмите **OK**, чтобы подтвердить удаление шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/74c1833d9aa411f0b8b9525400454e06.png)

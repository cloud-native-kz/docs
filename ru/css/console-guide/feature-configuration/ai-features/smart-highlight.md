# Интеллектуальное выделение контента

Интеллектуально определяйте замечательный контент и автоматически генерируйте выделения для повышения влияния вашего контента в 10 раз.

В этом документе описывается, как создавать, изменять и удалять шаблоны интеллектуального выделения контента через консоль.

## Важные замечания

- После успешного создания шаблона его можно связать с доменом доставки. Вступление в силу после успешной связи занимает 5-10 минут, и интеллектуальное выделение контента будет срабатывать сразу после успешной доставки данных связанным доменом.
- Изменения шаблонов, привязка и отвязка влияют только на трансляции, обновленные впоследствии. Текущие трансляции остаются неизменными; чтобы применить новые правила, текущую трансляцию необходимо прервать и переподключить.

## Предварительные условия использования

Сервис CSS активирован.

## Создание шаблона интеллектуального выделения контента

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Конфигурация функций** > **Функции искусственного интеллекта** > **Интеллектуальное выделение контента**.

> **Примечание****：** Для использования **Интеллектуального выделения контента** в разделе **Функции искусственного интеллекта** при первом создании необходимо создать роль сервиса и авторизовать текущую роль учетной записи для использования продуктов **Служба обработки мультимедиа** (MPS) и **Облачное хранилище объектов** (COS).

2. Нажмите **Авторизировать**, чтобы перейти на страницу управления ролями CAM.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0c309d13c5f211f0a1dc52540044a08e.png)

3. На странице управления ролями нажмите **Авторизировать**. После прохождения проверки личности вы сможете авторизировать Службу обработки мультимедиа и Облачное хранилище объектов (COS) и использовать эти сервисы в обычном режиме.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/26e648a0c5f211f085c7525400454e06.png)

4. После успешной авторизации выберите соглашение об обслуживании и нажмите **Начать**. Система автоматически активирует продукт Служба обработки мультимедиа и Облачное хранилище объектов (COS) и перенаправит вас на страницу управления интеллектуальным выделением контента.
5. Нажмите **Создать шаблон выделения** для входа на страницу создания и выполните следующую конфигурацию:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cafb69a800c911f1a616525400a31896.png)

| Элемент конфигурации |  | Описание |
| --- | --- | --- |
| Имя шаблона |  | Настраиваемое (макс. 30 символов; поддерживаются буквы, цифры, подчеркивания и дефисы). |
| Описание шаблона |  | Настраиваемое (макс. 100 символов; поддерживаются китайские иероглифы, буквы, цифры, пробелы и _-). |
| Тип сценария |  | Включает: Общее, Новости, Онлайн-образование, Электронная торговля, Финансы, Прямая трансляция, Спортивные события и Киберспорт. Выберите в соответствии с вашими фактическими требованиями бизнеса. |
| Дополнительное описание сценария |  | Настраиваемое (поддерживаются только буквы, цифры, пробелы, подчеркивания (_) и дефисы (-), не более 200 символов). Пример: Создавать выделения избранного контента, такие как "основные новости" и "информация" во время прямой трансляции новостей. |
| Часовой пояс |  | Вы можете выбрать UTC+8 или UTC. Когда параметр часового пояса выбран как UTC+8, файл интеллектуального выделения контента назначается на основе параметра времени UTC+8. Когда часовой пояс выбран как UTC, файл интеллектуального выделения контента назначается на основе параметра времени UTC. |
| Местоположение хранилища | Путь хранения | Выберите бакет COS, который вы создали и авторизовали в Облачном хранилище объектов. |
|  | Регион | Поле Регион указывает географическое расположение указанного выше бакета и не может быть изменено. |
|  | Папка | Содержимое папки по умолчанию: SmartHighlights/{Domain}/{Appname}/{StreamID}/{SessionID}/{Year}-{Month}-{Day}-{Hour}-{Minute}-{Second}-{Pts}.mp4 |

| Элемент дополнительной конфигурации | Описание |
| --- | --- |
| Включить сдвиг времени | В зависимости от потребностей вашего бизнеса вы можете выбрать включение функции сдвига времени. Интеллектуальное выделение контента может быть дополнительно подтверждено и точно отредактировано в полной трансляции, если также используется сдвиг времени. Вы можете перейти на страницу [Управление трансляциями](https://console.tencentcloud.com/live/streammanage), чтобы просмотреть результат работы интеллектуального выделения контента. |

## Привязка шаблона интеллектуального выделения контента

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Конфигурация функций** > **Функции искусственного интеллекта** > **Интеллектуальное выделение контента**.
2. Привяжите доменное имя одним из двух способов:
  - **Привязать домен к существующему шаблону**: Нажмите **Привязать доменное имя** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f7e2b27b00c911f18080525400074c32.png)

  - **Привязать доменное имя после создания шаблона интеллектуального выделения контента**: После успешного создания [шаблона интеллектуального выделения контента](#f7f41fc2-7fcd-4905-9ef9-45a5930871f6) нажмите **Привязать доменное имя** во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3674e0f7c53011f0a7ca5254001c06ec.png)

3. В окне привязки домена выберите **шаблон интеллектуального выделения контента** и **доменное имя доставки** (можно одновременно привязать несколько доменных имен доставки) и нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/46d0767fc53011f085c7525400454e06.png)

## Отвязка

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Конфигурация функций** > **Функции искусственного интеллекта** > **Интеллектуальное выделение контента**.
2. Выберите шаблон интеллектуального выделения контента, связанный с доменным именем, и нажмите **Отвязать**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a2641ea0c53011f099d952540099c741.png)

3. Подтвердите, хотите ли вы отвязать текущее связанное доменное имя. Нажмите **Подтвердить**, чтобы отвязать его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ada99689c53011f0a2c65254005ef0f7.png)

## Изменение шаблона

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Конфигурация функций** > **Функции искусственного интеллекта** > **Интеллектуальное выделение контента**.
2. Выберите успешно созданный шаблон интеллектуального выделения контента и нажмите **Редактировать** справа для перехода на страницу изменения информации шаблона. Нажмите **Сохранить**, чтобы завершить изменение.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c492b2b2c53011f085c7525400454e06.png)

## Удаление шаблона

> **Примечание****：** Если шаблон уже связан, вы должны сначала [отвязать](#6f6bd2ea-9e09-4e32-ac23-a8c4d6447b09) его перед тем, как приступать к процессу удаления. Обратите внимание, что шаблон не может быть восстановлен после удаления. Будьте осторожны при выполнении операций.

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat). Выберите **Конфигурация функций** > **Функции искусственного интеллекта** > **Интеллектуальное выделение контента**.
2. Выберите целевой шаблон (убедитесь, что он не привязан к доменному имени) и нажмите **Удалить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/db963087c53011f086d9525400e889b2.png)

3. Подтвердите, хотите ли вы удалить текущий шаблон интеллектуального выделения контента, затем нажмите **Подтвердить**, чтобы успешно удалить его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e66f625cc53011f0a7ca5254001c06ec.png)

## Дополнительно

Вы также можете отвязывать и привязывать доменные имена и шаблоны интеллектуального выделения контента на странице управления доменами. Подробную информацию см. в разделе [Конфигурация интеллектуального выделения контента](https://www.tencentcloud.com/document/product/267/74573).

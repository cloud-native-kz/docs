# Улучшение качества аудио и видео

Функция улучшения качества аудио и видео использует передовые алгоритмы искусственного интеллекта для восстановления и повышения качества аудиовизуального контента, достигая трансформационного улучшения визуального качества и значительного повышения субъективного качества аудио- и видеоконтента. В этом документе приведены инструкции по созданию, изменению и удалению шаблонов улучшения качества аудио и видео.

## Примечания

- Для использования функции улучшения качества аудио и видео она должна быть связана с трансcodированием Top Speed Codec (TSC). При получении потока включите параметр получения потока `txFeature=`, за которым следует имя шаблона улучшения качества аудио и видео, чтобы применить эффект улучшения к потоку в реальном времени. Пример URL-адреса получения потока выглядит следующим образом:

`http://domain/AppName/StreamName_Top Speed Codec transcoding template name.flv?txSecret=Md5(key+StreamName_Top Speed Codec transcoding template name+hex(time))&txTime=hex(time)&txFeature=enhancementtest`

- В настоящее время улучшение качества аудио и видео поддерживает разрешения трансcodирования ≤ 1080P по умолчанию. Чтобы включить поддержку других разрешений, вы можете [отправить заявку](https://console.tencentcloud.com/workorder/category) для настройки.
- Улучшение качества аудио и видео — это платная дополнительная услуга. Использование этой функции будет сопровождаться как платежами за трансcodирование TSC, так и платежами за улучшение качества аудио и видео. Подробные сведения о правилах выставления счетов см. в [документации по выставлению счетов](https://www.tencentcloud.com/document/product/267/57045).

## Предварительные требования

Облачные потоковые сервисы Tencent Cloud (CSS) активированы и добавлен [домен push](https://www.tencentcloud.com/document/product/267/35970).

## Создание шаблона улучшения качества аудио и видео

1. Войдите в консоль CSS и перейдите на **Feature Configuration** (Настройка функций) > [Audio/Video Enhancement](https://console.tencentcloud.com/live/config/enhance) (Улучшение качества аудио/видео).
2. Нажмите **Create Audio/Video Enhancement Template** (Создать шаблон улучшения качества аудио/видео) и настройте следующие параметры:

Enhancement Type - Video Enhancement (Тип улучшения — Улучшение видео)

Enhancement Type - Audio Enhancement (Тип улучшения — Улучшение аудио)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6e62aaeac4eb11f0a1dc52540044a08e.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/910a2ae3c4eb11f0a1dc52540044a08e.png)

| **Элемент конфигурации** |  | **Описание** |
| --- | --- | --- |
| Имя шаблона |  | Префикс имени шаблона фиксирован как enhancement. Имя не может превышать 10 символов и поддерживает только английские буквы и цифры. |
| Описание шаблона |  | Поддерживает китайские и английские символы, цифры, пробелы, подчеркивания (_) и дефисы (-), максимум 100 символов. |
| Тип улучшения |  | При выборе типа улучшения вы можете вручную выбрать **Video Enhancement** (Улучшение видео), **Audio Enhancement** (Улучшение аудио) или оба в зависимости от ваших фактических потребностей бизнеса. Если шаблон улучшения включает улучшение видео, улучшенный поток может быть получен только с использованием трансcodирования TSC. Если шаблон улучшения включает только улучшение аудио, улучшенный аудиопоток может быть получен с использованием либо стандартного трансcodирования, либо трансcodирования TSC. |
|  | Улучшение видео | **Конфигурация улучшения** По умолчанию задано общее улучшение — общий сценарий, которое можно переключить на общее улучшение — сценарий игры, цветовое улучшение, снижение шумов изображения или удаление артефактов. Значение интенсивности по умолчанию — 1, конфигурируемый диапазон от 0 до 1. При использовании функции улучшения видео выберите соответствующий тип улучшения в зависимости от ваших фактических потребностей. Ниже приведено описание доступных типов улучшения видео. **Общее улучшение — Общий сценарий**: Разработано для распространенных сценариев потокового вещания, таких как трансляции выступлений и электронная коммерция, это улучшение использует комплексные аналитические возможности искусственного интеллекта для автоматической балансировки содержимого текстур. Он удаляет артефакты сжатия и сбои при одновременном улучшении критических деталей, значительно повышая общее субъективное качество потока в реальном времени. **Общее улучшение — Сценарий игры**: Предназначено для сценариев игровых трансляций, это улучшение использует комплексные аналитические возможности искусственного интеллекта для автоматической балансировки содержимого текстур. Он удаляет артефакты сжатия и сбои при одновременном улучшении критических деталей, значительно повышая общее субъективное качество потока в реальном времени. **Цветовое улучшение**: Устранение искажений цвета или потребностей в улучшении, вызванных проблемами с устройствами захвата или хранилищем видео, эта функция корректирует цвета так, чтобы они лучше соответствовали реальным тонам при одновременном их улучшении в соответствии с предпочтениями человеческого зрения. **Снижение шумов изображения**: Во время потокового вещания в реальном времени камеры и факторы окружающей среды могут привносить случайные шумы. Эта функция обеспечивает снижение шумов при сохранении деталей, устраняя случайные шумы из видео. **Удаление артефактов**: Во время трансcodирования или нескольких раундов трансcodирования повторное сжатие может вносить блочные эффекты, эффекты звона, истечение цветности и комариный шум, вызывая визуальные искажения в видео. Эта функция эффективно восстанавливает искажения, вызванные сжатием, улучшая визуальное качество видео. |
|  | Улучшение аудио | **Конфигурация улучшения** Снижение шумов аудио выбрано по умолчанию. В зависимости от ваших фактических потребностей бизнеса вы можете вручную выбрать Volume Equalization (Выравнивание громкости) или включить обе функции. Снижение шумов аудио Значение интенсивности снижения шумов контролирует эффект снижения шумов аудио, низкие значения указывают на более слабое снижение шумов, а высокие значения указывают на более сильное снижение шумов. Значение интенсивности по умолчанию — 0,3, конфигурируемый диапазон — от 0 до 1. **Примечание:** Рекомендуется включать снижение шумов при отсутствии сложных фоновых шумов и когда основное внимание уделяется выделению вокала. Для источников **без фоновой музыки** **рекомендуется установить значение интенсивности ниже 0,6**. Более высокие значения могут снизить качество звука, приводя к проблемам, таким как приглушенная речь или чрезмерный реверберация. Для источников **с фоновой музыкой** **не рекомендуется включать снижение шумов. Если это необходимо, установите значение интенсивности ниже 0,3**. Более высокие значения могут подавить громкость вокала и негативно повлиять на качество аудио. Выравнивание громкости Значение громкости звука измеряет уровень громкости звукового сигнала. Рекомендуется корректировать этот параметр на основе фактической среды воспроизведения: более высокие значения для шумных сред и более низкие значения для тихих сред. Значение громкости по умолчанию — -20, конфигурируемый диапазон — от -40 до -10. |

## Изменение шаблона

1. Перейдите на **Feature Configuration** (Настройка функций) > [Audio/Video enhancement](https://console.tencentcloud.com/live/config/enhance) (Улучшение качества аудио/видео).
2. Выберите созданный шаблон улучшения качества аудио и видео и нажмите **Edit** (Изменить) справа, чтобы изменить информацию шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1e3b3508c4ec11f0a84e5254005ef0f7.png)

3. Нажмите **Save** (Сохранить).

## Удаление шаблона

1. Перейдите на **Feature Configuration** (Настройка функций) > [Audio/Video enhancement](https://console.tencentcloud.com/live/config/enhance) (Улучшение качества аудио/видео).
2. Выберите созданный шаблон улучшения качества аудио и видео и нажмите **Delete** (Удалить) в верхней части.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3b84bc2fc4ec11f0a7ca5254001c06ec.png)

3. Подтвердите, хотите ли вы удалить выбранный шаблон улучшения качества аудио и видео, и нажмите **Confirm** (Подтвердить), чтобы успешно удалить его.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e819e71eaae111efba3e5254002693fd.png)

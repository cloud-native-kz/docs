# Шаблон контроля качества в реальном времени и оценки вещания

## Обзор

CSS предлагает возможности **Контроля качества в реальном времени** и **Оценки вещания**, позволяющие вам комплексно выявлять проблемы качества в прямых трансляциях и оценивать показатели производительности, предоставляя соответствующие рекомендации.

#### Контроль качества в реальном времени

Функция **Контроля качества в реальном времени** включает **диагностику формата** и **проверку качества контента**:

- **Диагностика формата**: облачный сервис обработки аудио и видео, предназначенный для диагностики прямых трансляций в режиме реального времени, способный анализировать такие проблемы, как аномалии информации потока, нарушения синхронизации, отклонения состояния потока, ошибки инкапсуляции контейнера и отказы декодирования.
- **Проверка качества контента**
  - **Обнаружение типичных проблем изображения**: поддерживает обнаружение различных проблем качества прямого вещания, включая артефакты мозаики, сбои экрана, размытие изображения, дрожание экрана и визуальный шум.
  - **Обнаружение кодов распознавания**: поддерживает возможности обнаружения QR-кодов, штрих-кодов, кодов апплетов и других идентификационных кодов.
  - **Обнаружение однотонного экрана и яркости**: поддерживает обнаружение сценариев при прямом вещании, таких как чёрные/белые края, однотонные экраны, слабое освещение и переэкспозиция.
  - **Обнаружение типичных звуковых проблем**: поддерживает различные функции обнаружения качества звука в прямом вещании, включая тишину, тихий голос и громкий голос.

#### Оценка вещания

Функция **Оценки вещания** включает две отдельные возможности: **Оценка без эталона** и **Оценка с эталоном**.

- **Оценка без эталона**: алгоритм объективно оценивает качество прямых трансляций без опоры на сравнительные эталоны, непосредственно предоставляя оценки.
- **Оценка с эталоном**: обычно поток с перекодированием оценивается путём сравнения с исходным потоком или потоком с водяным знаком.
  - **PSNR (Peak Signal-to-Noise Ratio)**: выполняет оценку с эталоном качества поток перекодирования путём расчёта среднеквадратической ошибки между исходным/потоком с водяным знаком и потоком перекодирования для измерения качества сжатия или восстановления изображения в прямом вещании.
  - **SSIM (Structural Similarity Index)**: выполняет оценку с эталоном качества потока перекодирования путём сравнения яркости, контрастности и структурной информации между потоком перекодирования и исходным/потоком с водяным знаком и моделирования восприятия качества изображения человеческой визуальной системой.
  - **VMAF (Video Multimethod Assessment Fusion)**: выполняет объективную оценку с эталоном качества потока перекодирования на основе нескольких базовых метрик и моделей машинного обучения.

## Примечания

- После создания шаблона сдвига по времени необходимо привязать его к домену push.
- Как контроль качества, так и оценка без эталона могут выполнять обнаружение на исходных потоках, потоках с водяным знаком и перекодированных потоках. Оценка с эталоном, однако, применяется исключительно к перекодированным потокам — сравнивая их с потоками с водяным знаком при наличии водяного знака, и с исходными потоками при его отсутствии.

1. Войдите в консоль CSS. На левой боковой панели выберите **Конфигурация функций >** [Контроль качества в реальном времени и оценка вещания](https://console.tencentcloud.com/live/config/quality-inspection/template).
  - **Привязать домен к существующему шаблону**: нажмите **Привязать имя домена** в верхнем левом углу.

  - **Привязать домен после создания нового шаблона контроля качества в реальном времени и оценки вещания**: после успешного создания шаблона контроля качества в реальном времени и оценки вещания нажмите **Привязать имя домена** во всплывающем окне.

# Обратный вызов прямой трансляции

CSS поддерживает обратные вызовы. Для использования этой функции необходимо создать шаблон обратного вызова в консоли, настроить адрес для получения обратных вызовов для события и затем привязать шаблон к доменному имени для отправки. Если при прямой трансляции срабатывает обратный вызов события, Tencent Cloud отправит запрос на ваш сервер, который отвечает за ответ на запрос. После успешной проверки сервер получит пакет JSON обратного вызова через настроенный адрес.
В этом документе описывается, как создавать, изменять и удалять шаблон обратного вызова в консоли.

**Вы можете создать шаблон обратного вызова следующим образом:**

- Создать шаблон в консоли CSS. Дополнительные сведения см. в разделе [Создание шаблона обратного вызова](#Callback).
- Создать шаблон с помощью API. Для получения информации о параметрах и примерах запросов см. [CreateLiveCallbackTemplate](https://intl.cloud.tencent.com/document/product/267/30815).

## Важные замечания

- После создания шаблона необходимо [привязать его к доменному имени для отправки](https://intl.cloud.tencent.com/document/product/267/31065). Привязка вступает в силу примерно через 5–10 минут.
- Убедитесь, что используемый вами сервер HTTP или HTTPS для получения обратных вызовов способен получать запросы и нормально отвечать на них.
- В консоли вы можете привязывать и отвязывать шаблоны обратного вызова только на уровне домена. Для правил обратного вызова, привязанных к конкретным потокам через API, необходимо вызвать [DeleteLiveCallbackRule](https://intl.cloud.tencent.com/document/product/267/30814) для их отвязания.

## Привязка доменного имени

1. Войдите в консоль CSS и выберите **Feature Configuration** (Конфигурация функций) > [Live Stream Callback](https://console.tencentcloud.com/live/config/callback) (Обратный вызов прямой трансляции) на левой боковой панели.
2. Вы можете привязать домен к шаблону одним из двух способов:
  - **Привязать домен к существующему шаблону**: нажмите **Bind Domain Name** (Привязать доменное имя) в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/73b914e77d9611f0960452540044a08e.png)

  - **Привязать домен после создания шаблона**: после успешного [создания шаблона обратного вызова](https://www.tencentcloud.com/document/product/267/31074#creating-a-callback-template.5B.5D(id.3Acallback)) нажмите **Bind Domain Name** (Привязать доменное имя) в появившемся диалоговом окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d208a74c7d9611f0960452540044a08e.png)

3. В окне Bind Domain Name (Привязать доменное имя) выберите нужный **Live Callback template** (Шаблон обратного вызова прямой трансляции) и **Push Domain name** (Имя домена отправки) (одновременно можно привязать несколько имен доменов отправки), затем нажмите на Confirm (Подтвердить) для успешной привязки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b008a8907d9611f0bda35254007c27c5.png)

## Отвязание

1. Войдите в консоль CSS и выберите **Feature Configuration** (Конфигурация функций) > [Live Stream Callback](https://console.tencentcloud.com/live/config/callback) (Обратный вызов прямой трансляции) на левой боковой панели.
2. Выберите шаблон обратного вызова привязанного домена, который вы хотите отвязать, затем нажмите **Unbind** (Отвязать).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2d63b3e77d9f11f09cab525400bf7822.png)

3. Подтвердите, если вы хотите отвязать текущий связанный домен, и нажмите **Confirm** (Подтвердить) для отвязания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9710b1b58aca11ef80ff525400d5f8ef.png)

## Изменение шаблона обратного вызова

1. Войдите в консоль CSS и выберите **Feature Configuration** (Конфигурация функций) > [Live Stream Callback](https://console.tencentcloud.com/live/config/callback) (Обратный вызов прямой трансляции) на левой боковой панели.
2. Выберите целевой шаблон обратного вызова и нажмите **Edit** (Редактировать) для изменения его информации.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6cfd55407d9f11f0960452540044a08e.png)

3. После изменения нажмите **Save** (Сохранить).

## Удаление шаблона обратного вызова

> **Примечание:** Если шаблон был связан, необходимо сначала [отвязать его](https://www.tencentcloud.com/document/product/267/31074#fc56529b-3f7e-4bf8-90b5-8ee8eca29e5d) перед удалением. Обратите внимание, что удаленный шаблон нельзя восстановить. Действуйте с осторожностью.

1. Войдите в консоль CSS и выберите **Feature Configuration** (Конфигурация функций) > [Live Stream Callback](https://console.tencentcloud.com/live/config/callback) (Обратный вызов прямой трансляции) на левой боковой панели.
2. Выберите успешно созданный шаблон обратного вызова, нажмите **Delete** (Удалить) в верхней части.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8eff41047d9f11f09a9a5254001c06ec.png)

3. Подтвердите, если вы хотите удалить шаблон обратного вызова, нажмите **Confirm** (Подтвердить) для успешного удаления.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ac9a0c2c7d9f11f0bd33525400454e06.png)

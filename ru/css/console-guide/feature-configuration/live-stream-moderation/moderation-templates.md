# Шаблоны модерации

CSS предлагает функцию модерации трансляций. Вы можете настроить шаблоны модерации в консоли. После связи домена потоковой передачи с шаблоном этот сервис будет получать скриншоты в реальном времени и звук во время трансляции, сохраняя любые неcоответствующие скриншоты или аудиоданные в Tencent Cloud COS. Если домен потоковой передачи связан с конфигурацией обратного вызова, Tencent Cloud отправит запрос на сервер заказчика при срабатывании события обратного вызова во время прямой трансляции. Сервер заказчика несет ответственность за обработку запроса. После проверки можно получить пакет JSON, содержащий информацию об обратном вызове модерации. Этот документ содержит инструкции по созданию, привязке, отвязке, изменению и удалению шаблонов конфигурации модерации через консоль.

## Примечания

- Модерация трансляций — платная функция. Модерация изображений стоит 0,2294 USD за тысячу изображений, а модерация аудио — 0,0021 USD в минуту. Дополнительную информацию см. в разделе [Модерация трансляций](https://www.tencentcloud.com/document/product/267/58267). Дополнительную информацию см. в разделе «Дополнительные услуги — модерация трансляций».
- Скриншоты или аудио, помеченные во время модерации трансляции, хранятся в Tencent Cloud COS, что создает затраты на хранилище. Сведения о ценах см. в разделе [Ценообразование продукта COS](https://intl.cloud.tencent.com/pricing/cos).
- Если ваш бакет COS разрешает публичный доступ на чтение и содержит политически чувствительный, порнографический или другой неподходящий контент, чтобы избежать блокировки бакета, сначала удалите контент.
- После создания шаблона он может быть связан с доменом отправки. Дополнительные сведения см. в разделе «Конфигурация модерации трансляций». Связь шаблона обычно вступает в силу в течение 5-10 минут.
- Привязка, изменение или отвязка шаблона влияют только на новые трансляции и не влияют на текущие. Чтобы применить новые правила к текущим трансляциям, необходимо их остановить и снова отправить.
- Для использования пользовательской библиотеки ключевых слов для распознавания изображений убедитесь, что выбран параметр "OCR" раздела "Распознавание изображений". Когда OCR включен, система будет модерировать текст на скриншотах видео. Модерация на основе OCR обнаруживает весь неcоответствующий контент независимо от настроенной политики распознавания.
- Интеллектуальная модерация может выполняться на содержимом изображений, аудио и текста трансляции. Пользователи могут настраивать политики модерации и гибко настраивать размеры идентификации для обеспечения соответствия контента трансляции.

## Предварительные требования

- Вы активировали сервис Tencent CSS и добавили [домен отправки](https://www.tencentcloud.com/document/product/267/35970).
- Вы создали бакет COS. Дополнительную информацию см. в разделе [Создание бакета](https://www.tencentcloud.com/document/product/436/13309?lang=zh&pg=).

## Создание шаблонов модерации трансляций

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Конфигурация функций** > **Модерация контента** на левой панели навигации.

> **Примечание:** CSS хранит скриншоты видео и аудиоданные в бакетах COS. Поэтому, если вы впервые создаете шаблон модерации, необходимо создать роль сервиса и предоставить CSS доступ на чтение и запись к COS.

2. Нажмите **Создать шаблон**.
3. Появится всплывающее окно для предоставления CSS авторизации для доступа к ресурсам COS. Нажмите **Авторизовать сейчас**, чтобы перейти на страницу **Управление ролями**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4baf6c3194d111eeb6c6525400e46040.png)

4. На странице **Управление ролями** нажмите **Предоставить доступ**. После аутентификации CSS получит доступ к ресурсам COS, и вы сможете начать использовать функцию модерации трансляций.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4b81ca5294d111eeb6c6525400e46040.png)

5. После авторизации вы будете перенаправлены на страницу создания шаблона, где вы можете заполнить элементы конфигурации.

## Привязка доменов

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Конфигурация функций** > **Модерация контента** на левой панели навигации.
2. Вы можете привязать шаблон модерации к домену отправки следующими способами:
  - **Прямая привязка домена**: Нажмите **Привязать доменное имя** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8a5bdd57c51011f0a62b5254007c27c5.png)

  - **Привязка домена после создания нового шаблона модерации**: После успешного создания шаблона модерации нажмите **Привязать доменное имя** во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/56ef03cbc51011f099d952540099c741.png)

3. В окне «Привязать доменное имя» выберите **шаблон модерации** и **имя домена отправки** (одновременно может быть привязано несколько имен доменов отправки), которые вы хотите привязать вместе, и нажмите **Подтвердить**, чтобы привязать их.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/69e02cb1c51011f0a2c65254005ef0f7.png)

## Отвязка

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Конфигурация функций** > **Модерация контента** на левой панели навигации.
2. Выберите шаблон модерации трансляций, от которого вы хотите отвязать домен отправки, и нажмите **Отвязать**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/536bd68b3b7511f0be2052540099c741.png)

3. Во всплывающем окне нажмите **Подтвердить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4b72981394d111eebd87525400c40977.png)

## Изменение шаблона

1. На левой панели навигации выберите **Конфигурация функций** > **Модерация контента**.
2. Выберите успешно созданный шаблон модерации, нажмите **Редактировать** справа, чтобы перейти к изменению информации шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bc87536e3b7511f0b95f5254005ef0f7.png)

## Удаление шаблона

> **Примечание:** Если шаблон уже привязан к домену отправки, вы должны сначала [отвязать](https://www.tencentcloud.com/document/product/267/58265#fc56529b-3f7e-4bf8-90b5-8ee8eca29e5d) его, прежде чем шаблон можно будет удалить. Удаленный шаблон не может быть восстановлен. Будьте осторожны при удалении шаблонов.

1. На левой панели навигации выберите **Конфигурация функций** > **Модерация контента**.
2. Выберите ранее созданный шаблон модерации и нажмите **Удалить**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/efd4a1383b7511f0912c52540044a08e.png)

3. Нажмите **Подтвердить**, чтобы окончательно удалить шаблон.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4f822aa23c6611f0aa9f5254001c06ec.png)

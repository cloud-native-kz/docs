# Интеллектуальное стирание

Функция интеллектуального стирания позволяет пользователям настраивать шаблоны интеллектуального стирания через консоль. После связывания домена отправки с шаблоном система захватывает аудио во время потоковой передачи, определяет ненадлежащий контент и отключает его звук соответствующим образом, обеспечивая позитивную и хорошо контролируемую среду прямой трансляции.

В этом документе описано, как создавать, изменять и удалять шаблоны интеллектуального стирания через консоль.

## Примечания

- После создания шаблона он может быть привязан к домену отправки. Дополнительные сведения см. в разделе [Конфигурация интеллектуального стирания](https://www.tencentcloud.com/document/product/267/70285). Привязка шаблона обычно вступает в силу в течение 5-10 минут.
- Привязка, изменение или отмена привязки шаблона влияет только на новые прямые трансляции, а не на уже идущие. Чтобы применить новые правила к уже идущим трансляциям, необходимо остановить их и начать заново.
- Интеллектуальное стирание — Модерация прямой трансляции 2.0 может точно определять несоответствующее аудиосодержимое во время прямой трансляции и автоматически выполнять отключение звука в реальном времени. Это обеспечивает полностью автоматизированный контроль рисков контента и управление, а также гарантирует безопасность и соответствие контента платформ прямой трансляции.
- Функция интеллектуального стирания является платной дополнительной услугой. Использование этой функции во время прямой трансляции повлечет расходы на дополнительные услуги и плату за интеллектуальное определение для MPS. Сведения о правилах выставления счетов см. в разделе [Модерация прямой трансляции](https://www.tencentcloud.com/document/product/267/58267).
- При одновременном использовании интеллектуального стирания и функции субтитров они не могут быть активны одновременно (только стирание аудио может быть синхронизировано с субтитрами). Если пользовательский шаблон настроен как с стиранием изображения, так и со стиранием аудио, и включена функция субтитров, будут эффективны только субтитры и стирание аудио.

## Предварительные условия для использования

Сервис CSS активирован.

## Создание шаблона интеллектуального стирания

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/config/examine/erasure). Выберите **Feature Configuration** > **Content moderation** > **Smart Erasing**.

> **Примечание:** Из-за использования функции интеллектуального стирания при **первом** создании шаблона интеллектуального стирания требуется создать роль сервиса и авторизовать текущую роль учетной записи для использования служб продукта MPS.

2. Нажмите **Grant access**, чтобы перейти на страницу управления ролями CAM.
3. На странице управления ролями нажмите **Grant**. После завершения аутентификации личности для завершения авторизации MPS вы сможете использовать службу MPS в обычном режиме.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cdca529e3d0811f0912c52540044a08e.png)

4. После успешной авторизации примите соглашение об услуге и нажмите **Start**. Система автоматически активирует продукт MPS и откроет страницу управления интеллектуальным стиранием.
5. Нажмите **Create template**, чтобы перейти на страницу создания шаблона интеллектуального стирания, и настройте шаблон следующим образом:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c9f47fe7c51211f0a2c65254005ef0f7.png)

| Элемент конфигурации |  | Описание |
| --- | --- | --- |
| Template Name |  | Максимум 30 символов; поддерживаются буквы, цифры, подчеркивания и дефисы. |
| Template Description |  | Максимум 100 символов; поддерживаются китайские иероглифы, буквы, цифры, пробелы и `_-`. |
|   Recognition Scheme | Erasing Type | Несоответствующее аудио — определяет несоответствующее текстовое содержимое в аудио прямой трансляции и отключает звук соответствующих аудиосегментов для эффективной гарантии безопасности содержимого прямой трансляции. Разработка типов стирания для несоответствующих изображений, логотипов и содержимого защиты конфиденциальности в настоящее время ведется. Оставайтесь в курсе. |
|  | Non-compliant Audio Recognition Scheme | Привяжите шаблон в модуле модерации прямой трансляции с выбранным распознаванием аудиотекста для конфигурации содержимого модерации. Вы можете настроить несоответствующее текстовое содержимое, которое хотите стереть, в виде пользовательского словаря в [политике распознавания аудиотекста](https://www.tencentcloud.com/document/product/267/58265#084db3fe-6220-4f02-8a0e-702efbc95940). |
| Output Information |  | Выходное содержимое — это аудио после стирания |

## Привязка доменов

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Feature Configuration** > **Content moderation** > **Smart Erasing** из левой панели навигации.
2. Вы можете привязать шаблон модерации к домену отправки следующими методами:
  - **Прямая привязка домена**: Нажмите **Bind Domain Name** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/743a17c7c51311f0b88b525400bf7822.png)

  - **Привязка домена после создания нового шаблона интеллектуального стирания**: После успешного создания шаблона интеллектуального стирания нажмите **Bind Domain Name** в появившемся окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f4e46ca7c51211f0a62b5254007c27c5.png)

3. В окне "Bind Domain Name" выберите **шаблон интеллектуального стирания** и **имя домена отправки** (можно одновременно привязать несколько доменов отправки), которые вы хотите привязать вместе, и нажмите **Confirm**, чтобы их привязать.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5ba2cee2c51311f0a7ca5254001c06ec.png)

## Отмена привязки

1. Войдите в [консоль CSS](https://console.tencentcloud.com/live/livestat) и выберите **Feature Configuration** > **Content moderation** > **Smart Erasing** из левой панели навигации.
2. Выберите шаблон интеллектуального стирания, от которого вы хотите отменить привязку домена отправки, и нажмите **Unbind**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3580d1d13b6c11f08806525400bf7822.png)

3. В появившемся окне нажмите **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4318986e3b6c11f0be2052540099c741.png)

## Изменение шаблона

1. Выберите **Feature Configuration** > **Content moderation** > **Smart Erasing** из левой панели навигации.
2. Выберите успешно созданный шаблон интеллектуального стирания, нажмите **Edit** справа, чтобы перейти к изменению информации шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5699afc03b6c11f0be2052540099c741.png)

## Удаление шаблона

> **Примечание:** Если шаблон уже привязан к домену отправки, необходимо сначала [отменить его привязку](https://www.tencentcloud.com/document/product/267/70286#cee4a59a-cba1-40f9-a4f5-1efecb3f35fe), прежде чем можно будет удалить шаблон. Удаленный шаблон невозможно восстановить. Будьте внимательны при удалении шаблонов.

1. Выберите **Feature Configuration** > **Content moderation** > **Smart Erasing** из левой панели навигации.
2. Выберите ранее созданный шаблон интеллектуального стирания и нажмите **Delete**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6f542ca53b6c11f0912c52540044a08e.png)

3. Нажмите **Confirm**, чтобы окончательно удалить шаблон.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e164060d476711f089d45254001c06ec.png)

# Видимый водяной знак

Видимые водяные знаки достигаются путем наложения видимых обозначений (таких как изображения, текст или значки) на содержимое, такое как изображения, текст или видео. Они в основном используются для объявления авторских прав, определения источников или проведения брендовой промоции и могут интуитивно отображать информацию о владельце содержимого.

В видеоприложениях видимые водяные знаки могут эффективно повысить узнаваемость бренда и обеспечить некоторую функциональность предупреждения об авторских правах. Однако видимые водяные знаки имеют относительно низкую безопасность и в основном используются для идентификации и промоции, с ограниченными возможностями предотвращения утечек данных и отслеживания источников. Как видимый слой, видимые водяные знаки легко обрезаются, закрываются или удаляются с помощью обработки изображений сторонних инструментов. Для критических сценариев рекомендуется использовать их в сочетании с [невидимым водяным знаком (цифровой водяной знак)](https://www.tencentcloud.com/document/product/267/74854).

CSS поддерживает функцию водяного знака. Она добавляет водяные знаки на экран прямой трансляции для защиты видеоконтента от кражи. В этом документе описано, как создавать, изменять, привязывать, отвязывать и удалять шаблон водяного знака в консоли.

**Шаблон водяного знака можно создать следующими способами:**

- Создайте шаблон водяного знака в консоли CSS. Дополнительную информацию см. в разделе [Создание шаблона видимого водяного знака](#Watermark).
- Создайте шаблон водяного знака, вызвав API. Дополнительную информацию см. в разделе [AddLiveWatermark](https://www.tencentcloud.com/document/product/267/30826).

## Примечания

- После создания шаблона его можно привязать к домену отправки. Привязка вступит в силу через 5–10 минут.
- Привязка, отвязка и изменение шаблона влияют только на новые прямые трансляции после обновления, но не на текущие. Чтобы новое правило вступило в силу для текущих прямых трансляций, необходимо их прервать и отправить повторно.
- Функция видимого водяного знака является платной добавленной услугой. Использование услуги видимого водяного знака при отправке создает счет за транскодирование. Правила выставления счетов см. в [документации по выставлению счетов](https://www.tencentcloud.com/document/product/267/39604)

## Предварительные условия

Вы активировали услугу CSS и добавили [домен отправки](https://www.tencentcloud.com/document/product/267/35970).

## Создание шаблона видимого водяного знака

1. Войдите в консоль CSS и выберите **Feature Configuration** > [Live Watermarking](https://console.tencentcloud.com/live/config/watermark). Откройте страницу защиты авторских прав на прямую трансляцию.
2. Выберите **Visible Watermark** и нажмите **Create Template** для доступа к странице создания шаблона **Visible Watermark**, затем выполните следующие конфигурации:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b85e1b71d04711f093295254001c06ec.png)

| Элемент конфигурации | Описание конфигурации |
| --- | --- |
| Watermark Name | Введите имя водяного знака, настраиваемое, максимум 30 символов; поддерживает буквы, цифры, подчеркивания и дефисы |
| Image Upload | Нажмите **Select Image** для загрузки изображения водяного знака. Размер изображения водяного знака поддерживает растяжение до полного размера окна.**Примечание:**Для лучшего визуального впечатления рекомендуется использовать прозрачное PNG-изображение размером не более 2 МБ. Имя файла изображения может содержать только буквы, цифры и специальные символы - ! _ . * |
| Preview window size | Установите размер окна предварительного просмотра изображения водяного знака:Значения ширины и высоты по умолчанию: ширина 1920 пикс., высота 1080 пикс.Диапазон значений ширины и высоты: 360 пикс. - 4096 пикс.Нажатие **Update** справа автоматически проверит и синхронизирует обновление окна предварительного просмотра изображения водяного знака. |
| X-axis | Укажите местоположение водяного знака следующими способами:Перетащите изображение водяного знака в область конфигурации.Отрегулируйте координаты оси X и оси Y. |
| Y-axis |  |

3. Нажмите **Save**.

## Привязка имени домена

1. Войдите в консоль CSS и выберите **Feature Configuration** > **Live Watermarking** > [Visible Watermark](https://console.tencentcloud.com/live/config/watermark/light).
2. Откройте страницу привязки имени домена одним из следующих способов:
  - **Непосредственно привязать имя домена**: нажмите **Bind Domain Name** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a038408fd04a11f0b638525400e889b2.png)

  - **Привязать имя домена после создания шаблона видимого водяного знака**: после [создания шаблона видимого водяного знака](#Watermark) нажмите **Bind Domain Name** во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/280c4c7dd04a11f08823525400454e06.png)

3. Во всплывающем окне выберите **visible watermark template** и **push domain name** (можно одновременно привязать несколько доменов отправки) в окне привязки имени домена и нажмите **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4fd6b01dd04a11f093295254001c06ec.png)

## Отвязка

1. Войдите в консоль CSS и выберите **Feature Configuration** > **Live Watermarking**> [Visible Watermark](https://console.tencentcloud.com/live/config/watermark/light).
2. Выберите имена доменов, привязанные к шаблону видимого водяного знака, и нажмите **Unbind**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cfaef019d04c11f08e5c52540044a08e.png)

3. Подтвердите отвязку имени домена и нажмите **Confirm** для отвязки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1c762995d04d11f0a808525400bf7822.png)

## Изменение шаблона

1. Войдите в консоль CSS и выберите **Feature Configuration** > **Live Watermarking**> [Visible Watermark](https://console.tencentcloud.com/live/config/watermark/light).
2. Выберите целевой шаблон видимого водяного знака и нажмите **Edit** справа для изменения информации шаблона.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/76559202d0ec11f08e5c52540044a08e.png)

3. Нажмите **Save**.

> **Примечание:**Вы можете нажать **Preview** для предварительного просмотра того, как видимый водяной знак будет отображаться на экране.

## Удаление шаблона

> **Примечание:**Если шаблон уже связан, необходимо сначала [отвязать его](#unbinding), прежде чем можно будет выполнить операцию удаления.После удаления шаблона его невозможно восстановить. Пожалуйста, действуйте с осторожностью.

1. Войдите в консоль CSS и выберите **Feature Configuration** > **Live Watermarking**> [Visible Watermark](https://console.tencentcloud.com/live/config/watermark/light).
2. Выберите успешно созданный шаблон видимого водяного знака и нажмите **Delete** в верхнем правом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e4fd38fbd04d11f093295254001c06ec.png)

3. Во всплывающем диалоговом окне нажмите **Confirm** для подтверждения удаления.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f0af1823d04d11f093295254001c06ec.png)

# Шаблон

Временной сдвиг обеспечивается возможностью записи CSS. Он позволяет пользователям перематывать и воспроизводить более ранние части прямой трансляции. Обычно это используется для воспроизведения ярких моментов прямых трансляций спортивных событий.

## Примечания

- После создания шаблона временного сдвига необходимо привязать его к домену отправки. Конфигурация вступает в силу через 5-10 минут после привязки.
- При включении новой функции временного сдвига прямой трансляции биллинг будет основан на [Volume писания данных временного сдвига](https://www.tencentcloud.com/document/product/267/53262). Использование новой функции временного сдвига прямой трансляции также будет генерировать [Плата за трафик и пропускную способность прямой трансляции](https://www.tencentcloud.com/document/product/267/2819) и [Плата за транскодирование прямой трансляции](https://www.tencentcloud.com/document/product/267/39604).
- Чтобы выполнить временной сдвиг кодированного потока прямой трансляции, необходимо предварительно настроить задачу транскодирования для потока. Это будет влечь расходы на транскодирование. Убедитесь, что используемый шаблон транскодирования не удален.
- При записи данных кодированного потока временного сдвига сначала будет инициировано транскодирование, создавая [Плата за транскодирование прямой трансляции](https://www.tencentcloud.com/document/product/267/39604). Убедитесь, что выбранный шаблон транскодирования не был случайно удален, иначе случайно удаленный кодированный поток временного сдвига не будет воспроизводиться. При воспроизведении кодированного потока временного сдвига плата за транскодирование не взимается.

## Предварительные требования

Вы активировали CSS и добавили [имя домена отправки](https://intl.cloud.tencent.com/document/product/267/35970).

## Создание шаблона временного сдвига

1. Войдите в консоль CSS и выберите **Feature Configuration** > [Time Shifting](https://console.tencentcloud.com/live/config/time-shift/template) на левой боковой панели.
2. Нажмите **Create template**, чтобы установить информацию о шаблоне и настроить следующие параметры:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ebc52825c4f911f0a1dc52540044a08e.png)

| Элемент |  | Описание |
| --- | --- | --- |
| Template Name |  | Имя шаблона временного сдвига, которое может иметь длину 1-10 символов и может содержать китайские иероглифы, буквы, цифры и _- |
| Template Description |  | Введение и описание шаблона временного сдвига прямой трансляции можно настроить (поддерживаются только китайский язык, английский язык, цифры, пробелы, _, -). |
| Region |  | По умолчанию поддерживаются регионы вне материкового Китая, Гонконга, Макао и Тайваня, с возможностью выбора материкового Китая. Пожалуйста, привяжите правильный регион ускорения воспроизведения временного сдвига, так как кроссрегиональное воспроизведение временного сдвига может привести к задержкам или невозможности получить поток. |
| Stream type | Original stream | Если вы выберете эту конфигурацию, содержимое временного сдвига не будет иметь эффектов транскодирования, водяного знака или смешивания потоков. Для содержимого временного сдвига с WebRTC отправкой аудио может быть несовместимо с некоторыми плеерами. Рекомендуется выбрать «Watermarked Stream» или «Transcoded Stream». |
|  | Watermarked stream | Кодированный с водяным знаком поток (с водяным знаком, указанным в выбранном шаблоне водяного знака) получает временной сдвиг. |
|  | Transcoded stream | При выборе этой конфигурации содержимое видео временного сдвига будет содержимым после транскодирования согласно идентификатору шаблона транскодирования. Если шаблон транскодирования удален, содержимое воспроизведения временного сдвига станет недействительным. Кодированные потоки будут генерировать [Плату за транскодирование](https://www.tencentcloud.com/document/product/267/39604). |
| Time-shift days |  | По умолчанию 7 дней с возможностью выбора 1 дня, 3 дней, 15 дней и 30 дней. |
| TS segment length |  | Длина по умолчанию — пять секунд. Вы можете установить значение от 3 до 10. |

3. После завершения ввода нажмите **Save**, чтобы подтвердить.

## Привязка имени домена

1. Войдите в консоль CSS и выберите **Feature Configuration** > [Time Shifting](https://console.tencentcloud.com/live/config/time-shift/template) на левой боковой панели.
  - **Привязать домен к существующему шаблону**: Нажмите **Bind Domain Name** в верхнем левом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3ee6bcd6c4fa11f0a7ca5254001c06ec.png)

  - **Привязать домен после создания шаблона**: После [создания шаблона](https://www.tencentcloud.com/document/product/267/53312#creating-a-time-shifting-template) нажмите **Bind Domain Name** в появившемся диалоговом окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1239dc74c4fa11f0a7ca5254001c06ec.png)

2. В появившемся окне выберите **time shifting template** и **push domain** **name** (одновременно можно привязать несколько имен доменов отправки) и нажмите **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/24b6e5abc4fa11f085c7525400454e06.png)

## Отвязка имени домена

1. Войдите в консоль CSS и выберите **Feature Configuration** > [Time Shifting](https://console.tencentcloud.com/live/config/time-shift/template) на левой боковой панели.
2. Выберите шаблон временного сдвига, привязанный к доменам, найдите целевой домен и нажмите **Unbind**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/58cea842425111efa24b525400a9236a.png)

3. В появившемся окне нажмите **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c7f8c7fa594011ee9ff8525400d917da.png)

> **Примечание:** Отвязка шаблона временного сдвига не повлияет на текущие прямые трансляции.

## Изменение шаблона

2. Выберите целевой шаблон временного сдвига, нажмите **Edit** справа, измените параметры и нажмите **Save**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6dbec20b425111ef98fd52540075b605.png)

## Удаление шаблона

2. Выберите целевой шаблон временного сдвига и нажмите **Delete** в верхнем правом углу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7a3478d1425111efa24b525400a9236a.png)

3. В появившемся окне нажмите **Confirm**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/86f68b00425111efa24b525400a9236a.png)

> **Примечание:** Если к шаблону привязаны имена доменов, необходимо [отвязать](https://www.tencentcloud.com/document/product/267/53312#unbinding-a-domain-name) их перед удалением шаблона. В консоли шаблоны временного сдвига управляются на уровне домена. Вы не можете отвязать правила временного сдвига, привязанные к потокам через API.

## Дополнительно

Вы также можете **отвязать** и **привязать** домены и шаблоны временного сдвига на странице **Domain Management**. Подробные сведения см. в разделе [Time Shifting Configuration](https://www.tencentcloud.com/document/product/267/52829).

# Ошибки

Сервис потоковой трансляции Tencent Cloud (CSS) поддерживает функцию ошибок, позволяющую быстро проверять ошибки, которые возникают во время процесса прямой трансляции. Кроме того, вы можете понять статус этих потоков, просмотрев события основного и резервного потоков.

## Предварительные условия

Вы вошли в [консоль CSS](https://console.tencentcloud.com/live).

## Примечание

Вы можете настроить обратный вызов события ошибок отправки в параметрах [Live Callback](https://console.tencentcloud.com/live/config/callback). Когда в сервисе прямой трансляции происходит событие, сообщение будет уведомлено через унифицированный обратный вызов сообщения события.

## Запрос ошибок отправки

1. Выберите **Monitoring** > [Errors](https://console.tencentcloud.com/live/event/error-event) на левой боковой панели.
2. На странице ошибок поддерживается запрос по ID потока. Вы можете запрашивать ошибки отправки за последние 7 дней, а данные в периоде запроса менее 3 часов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2b5a1ec70e0411efa072525400b3b5af.png)

## Запрос основного/резервного потоков

1. Выберите **Monitoring** > [Errors](https://console.tencentcloud.com/live/event/error-event) на левой боковой панели, чтобы перейти на страницу основного/резервного потоков.
2. На странице основного/резервного потоков поддерживается запрос по ID потока. Вы можете запрашивать основные/резервные потоки за последние 7 дней, а данные в периоде запроса менее 3 часов.

## Типы ошибок

Ниже приведен список ошибок, которые могут возникнуть во время прямой трансляции.

| Номер | Тип ошибки |
| --- | --- |
| 1 | Временная метка видео движется в обратном направлении |
| 2 | Временная метка аудио движется в обратном направлении |
| 3 | Временная метка видео значительно увеличилась |
| 4 | Временная метка аудио значительно увеличилась |
| 5 | Размер блока слишком большой |
| 6 | Два последовательных видеокадра прибыли с опозданием |
| 7 | Два последовательных аудиокадра прибыли с опозданием |
| 8 | Видеокодек изменился |
| 9 | Аудиокодек изменился |
| 10 | Отсутствует заголовок кодека перед поступлением видеокадра |
| 11 | Отсутствует заголовок кодека перед поступлением аудиокадра |
| 12 | Ошибка анализа заголовка видео |
| 13 | Большой размер блока |
| 14 | Низкая частота кадров видео |
| 15 | Большой интервал временной метки аудиокадров |
| 16 | Большой размер GOP |
| 17 | Необычные форматы кодирования/декодирования аудио/видео |

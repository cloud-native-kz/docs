# Анализ операций

Консоль CSS предоставляет страницу [**Анализ операций**](https://console.tencentcloud.com/live/analysis/total), где вы можете просматривать данные, связанные с **Live Playback**, **User distribution**, **Top playbacks**, **Device Statistics** и **Origin Server**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cd3b7fcbfdb911f0b5c0525400380f7d.png)

> **Примечание:** Для вкладок **Live Playback** и **User distribution** данные основаны на IP-адресе пользователя. На вкладке **Origin Server** не поддерживается запрос данных до 18 февраля 2022 года. Данные за пределами китайской материка агрегируются вместо конкретных ISP.

## Live Playback

На вкладке **Live Playback** вы можете просматривать данные о пиковой пропускной способности, общем трафике, общем количестве запросов и пиковом количестве одновременных подключений, а также коэффициенте ошибок кода HTTP. Он поддерживает запрос данных на уровне детализации имени домена и позволяет выбирать регионы и ISP. Вы можете запрашивать данные за последние три месяца, максимальный период времени для каждого запроса составляет один месяц.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e8b00d08fdb911f08e41525400ecee81.png)

| Статистический элемент | Описание |
| --- | --- |
| Bandwidth Peak | Данные пиковой пропускной способности, генерируемые нисходящей трансляцией прямой трансляции. |
| Total Traffic | Данные общего трафика, генерируемые нисходящей трансляцией прямой трансляции. |
| Total Requests | Общее количество запросов между плеером и платформой CSS во время прямой трансляции. |
| Concurrent Connection Peak | Данные пикового одновременного соединения между плеерами и платформой CSS во время прямой трансляции. |
| HTTP Code Error Rate | Процентное соотношение и использование кодов статуса для входящих и исходящих запросов при прямой трансляции. Коэффициент ошибок относится к доле ответов 4xx (ошибки клиента) и 5xx (ошибки сервера) к общему количеству запросов. |

## User Distribution

На вкладке **User distribution** вы можете просматривать географическое местоположение, трафик и данные запросов пользователей прямой трансляции. Он поддерживает запрос данных на уровне детализации имени домена и позволяет выбирать регионы и ISP. Вы можете запрашивать данные за последние три месяца, максимальный период времени для каждого запроса составляет один месяц. Данные классифицируются по **Greater region** и **Region** и сортируются по отношению трафика этого региона к общему трафику всех регионов в списке.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b1740b6b288f11ef9dd5525400441de3.png)

- При запросе данных на территории китайского материка **Greater region** включает Северный Китай, Северо-Западный Китай, Северо-Восточный Китай, Восточный Китай, Центральный Китай, Юго-Западный Китай, Южный Китай и другие регионы. **Region** — это конкретная провинция под каждым **Greater region**.
- При запросе данных за пределами китайского материка **Greater region** включает Азиатско-Тихоокеанский регион 1, Азиатско-Тихоокеанский регион 2, Азиатско-Тихоокеанский регион 3, Северная Америка, Европа, Южная Америка, Ближний Восток, Африка и другие регионы. **Region** — это конкретная страна или регион под каждым **Greater region**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b91856cf288f11efa45a5254008fe934.png)

| Статистический элемент | Описание |
| --- | --- |
| Traffic (MB) | Общий трафик каждого региона. |
| Traffic/Current list | Отношение трафика этого региона к общему трафику всех регионов в списке. |
| Traffic/Total | Отношение трафика этого региона ко всем остальным регионам на территории или за пределами китайского материка. Например, когда пользователь запрашивает данные на территории китайского материка, этот статистический элемент указывает отношение трафика этого региона к общему трафику китайского материка. |
| Requests | Общее количество запросов для каждого региона. |
| Regional requests/Current list | Отношение запросов этого региона к общему количеству запросов всех регионов в списке. |
| Regional requests/Total | Отношение запросов этого региона ко всем остальным регионам на территории или за пределами китайского материка. Например, когда пользователь запрашивает данные на территории китайского материка, этот статистический элемент указывает отношение запросов этого региона к общему количеству запросов китайского материка. |

Для просмотра тенденций трафика и пропускной способности для одного или нескольких регионов вы можете выбрать нужные регионы и нажать **Traffic Trend** и **Bandwidth Trend**.

## Top Playbacks

Функция Playback Ranking предоставляет возможности отображения и запроса для **TOP 100 streams**, **TOP 100 client IPs**, **TOP 100 URL** и **TOP 100 Referer**, помогая вам понять популярность прямых трансляций и распределение зрителей. Кроме того, эта функция поддерживает запрос различных индикаторов данных, таких как рейтинг, трафик, коэффициент трафика, частота и коэффициент частоты. Она также поддерживает запрос данных на уровне домена (воспроизведение) и позволяет выбирать регионы и запрашивать данные за последние три месяца. Максимальный период времени, поддерживаемый для каждого запроса, составляет один день.

**Ключевые функции**

- **Top 100 streams:** Эта функция позволяет запрашивать и просматривать топ-100 прямых трансляций с наибольшей аудиторией, включая рейтинг, трафик и долю трафика.
- **Top 100 client IPs:** Эта функция позволяет запрашивать и просматривать топ-100 IP-адресов клиентов, которые смотрят больше всего прямых трансляций, помогая вам понять распределение аудитории.
- **TOP 100 URL:** Вы можете запрашивать и отображать топ-100 URL-адресов (путей запроса), отранжированных по частоте, включая такую информацию, как рейтинг, домен, URL-адрес, частота и коэффициент частоты. Это облегчает ваше понимание доступа аудитории к различным URL-адресам.
- **TOP 100 Referer:** Вы можете запрашивать и отображать топ-100 Referer (источников запроса), отранжированных по частоте, включая такую информацию, как рейтинг, домен, Referer и коэффициент частоты. Это облегчает ваше понимание ситуации с доступом аудитории из различных Referer.

> **Примечание:** Топ-рейтинговые данные потоков, IP, URL и Referer на текущей странице используются только для анализа операций, а конкретный результат анализа данных зависит от фактических данных журнала.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/dd4b1ca8288f11ef8d66525400e8fea1.png)

| Статистический элемент | Описание |
| --- | --- |
| Top 100 streams: Traffic/Current listTOP 100 Streams - Proportional Traffic Distribution | Отношение трафика этого потока к общему трафику всех потоков в списке. |
| Top 100 client IPs: Traffic/Current list | Отношение трафика этого IP-адреса к общему трафику всех IP-адресов в списке. |
| Top 100 URL - Frequency Ratio | Отношение трафика этого потока к общему трафику всех потоков в списке. |
| Top 100 Referer - Frequency Ratio | Отношение трафика этого потока к общему трафику всех потоков в списке. |

## Device Statistics

Функция Device Statistics предоставляет запросы и отображение для типа устройства, типа браузера и типа операционной системы. Она поддерживает запрос данных (воспроизведение) на уровне детализации имени домена и позволяет выбирать регионы. Она поддерживает запросы данных за последние 3 месяца.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/64e760f0288d11efae985254002ecb22.png)

| Статистический элемент | Описание |
| --- | --- |
| Device Type | **Типы устройств** включают: Tablet, Mobile, Desktop, TV и Other. |
| Browser Type | **Типы браузеров** включают: Empty, Chrome, Safari, Opera, QQBrowser, LBBrowser, MaxthonBrowser, SouGouBrowser, BIDUBrowser, TaoBrowser, UBrowser, IE, Microsoft Edge, Bot и Other. |
| Operating System Type | **Типы операционных систем** включают: Empty, Android, IOS, Mac OS, Windows, Linux, Chromium OS, NetBSD, Bot и Other. |

## Origin Server

Вкладка **Origin Server** предоставляет трафик и пиковую пропускную способность с сервера-источника. Она поддерживает запрос данных на уровне детализации имени домена и позволяет выбирать регионы и ISP. Вы можете запрашивать данные за последние три месяца, максимальный период времени для каждого запроса составляет один месяц.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e93618e3288f11ef9c87525400161bea.png)

| Статистический элемент | Описание |
| --- | --- |
| Total traffic | Данные трафика с сервера-источника. |
| Peak bandwidth | Данные пиковой пропускной способности с сервера-источника. |

# Обзор

В консоли CSS вы можете управлять доменными именами и потоками, настраивать трансферед, запись и ускорение, а также отправлять потоки (веб) и отслеживать ресурсы.

## Предварительные условия

- Вы активировали [CSS](https://intl.cloud.tencent.com/product/css).
- Вы вошли в [консоль CSS](https://console.tencentcloud.com/live/livestat).

## Обзор

Нажмите [Обзор](https://console.tencentcloud.com/live/livestat) в левой панели навигации.

- **Быстро создайте** свой персонализированный сервис потоковой трансляции. Вы можете обратиться к соответствующим руководствам, добавить домены, создать URL-адреса, изучить руководство по разработке потоковой трансляции и быстро перейти к функциям потоковой трансляции.
- Вы можете нажать карточку "**Отчет о состоянии здоровья**" в верхнем правом углу страницы "Обзор", чтобы получить доступ к отчету о состоянии здоровья Live. Дополнительные сведения см. в разделе [Отчет о состоянии здоровья.](https://www.tencentcloud.com/document/product/267/70558#)
- Вы можете просматривать соответствующие данные, включая пропускную способность нижестоящего потока в реальном времени, трафик нижестоящего потока на сегодня, текущее количество отправок, одновременные подключения и тренды для выставляемой по счетам пропускной способности, выставляемого по счетам трафика и количества отправок за последние 30 дней. Кроме того, вы можете просмотреть подробные сведения о потреблении **Отчета о состоянии здоровья** и **Планы**. Вы можете переключать **Режим выставления счетов** или изменять детализацию времени по мере необходимости. Для справки новичков вы можете нажать **Руководство** в верхнем правом углу, чтобы просмотреть инструкции по началу работы с CSS.

### Данные за сегодня

Этот раздел отображает пиковую пропускную способность нижестоящего потока, использование трафика нижестоящего потока, текущее количество каналов отправки и количество одновременных подключений за день.

| Элемент | Описание |
| --- | --- |
| Текущая пропускная способность нижестоящего потока | Пиковая пропускная способность, используемая всеми доменами воспроизведения для ускорения. |
| Трафик нижестоящего потока на сегодня | Общий трафик, используемый всеми доменами воспроизведения в текущий день для ускорения. |
| Текущие каналы отправки | Количество текущих каналов отправки. |
| Одновременные подключения | Если протокол воспроизведения — RTMP или FLV, **Одновременные подключения** указывают количество активных зрителей. Если протокол воспроизведения — HLS, **Одновременные подключения** нельзя использовать как индикатор количества активных зрителей. |

### Тренды использования

Этот раздел отображает тренды использования (**Тренд пропускной способности**, **Тренд трафика** и **Каналы отправки**) за сегодня, вчера, последние 7 дней и последние 30 дней.

| Элемент | Описание |
| --- | --- |
| Тренд пропускной способности | Сумма пиковой пропускной способности, используемой всеми доменами воспроизведения для ускорения в запрашиваемом периоде. |
| Тренд трафика | Общий трафик, используемый всеми доменами воспроизведения для ускорения в запрашиваемом периоде. |
| Каналы отправки | Количество каналов отправки под выбранными доменами в запрашиваемом периоде. |

### Пакет ресурсов

> **Примечание:** Оставшийся объем пакетов ресурсов, отображаемый на странице обзора, не является данными в реальном времени. Время обновления совпадает со временем выравнивания счета учетной записи.

#### Просмотр подробных сведений о потреблении пакета ресурсов

- Нажмите **Управление** справа, чтобы перейти на страницу [Управление пакетом ресурсов/плагином](https://console.tencentcloud.com/live/resources/package?type=traffic).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/82a35d4e533711f0989352540044a08e.png)

- Вы можете просмотреть **Использование**, **Время создания**, **Время истечения**, **Статус** и **Автоматическое обновление** пакетов ресурсов трафика/пакетов ресурсов трансферед.
- Автоматическое обновление (автоматическое обновление при исчерпании или истечении срока) поддерживается для пакетов ресурсов трафика потоковой трансляции и пакетов ресурсов трансферед потоковой трансляции (включая стандартную трансферед и трансферед с кодеком максимальной скорости). Подробные сведения см. в документации [Обновление](https://www.tencentcloud.com/document/product/267/53309).
- **Покупка пакета ресурсов трафика:** Нажмите "Купить" в статистике пакета трафика, и вы перейдете на [страницу покупки пакета ресурсов трафика CSS](https://buy.intl.cloud.tencent.com/live?type=transcode), чтобы приобрести соответствующие пакеты.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b0564898533711f090d5525400bf7822.png)

> **Примечание:** Информацию о ценах на выставление счетов см. в разделе [Обзор выставления счетов](https://intl.cloud.tencent.com/document/product/267/2819).

### Выставление счетов

- **Переключение режима выставления счетов**
В зависимости от ваших фактических бизнес-потребностей, если текущий режим выставления счетов — ежедневный счет по трафику или ежедневный счет по пропускной способности, вы можете нажать **Переключить**, чтобы изменить режим выставления счетов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/db3702cb119611ef9483525400720cb5.png)

- Чтобы просмотреть напоминания о переключении выставления счетов, нажмите **Подтвердить**, чтобы завершить переключение режима выставления счетов. Дополнительные сведения об изменениях выставления счетов см. в разделе [Изменение режимов выставления счетов](https://intl.cloud.tencent.com/document/product/267/30411).

# Управление пакетами

Вы можете просмотреть использование вашего плана [трафика](#traffic-package) и [транскодирования](#transcoding-package) в консоли CSS.

## План трафика

Выберите [план](https://console.tencentcloud.com/live/resources/package?type=traffic) на левой боковой панели. На вкладке **План трафика** вы можете выполнять следующие операции:

- Просматривать использование, время покупки, время истечения и статус приобретенного **плана трафика**.
  - Статус плана включает неиспользованный, используется, исчерпан, истекший и заморожен. Статус автоматического продления включает отключено, продление выполнено, продление не выполнено, автоматическое продление включено и неподдерживается. Если продление не выполнено, система выведет причину сбоя, например "недостаточно средств на счете".

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ab2b22c8594a11ef81cf525400d5f8ef.png)

- Нажмите **Детали использования**, чтобы просмотреть детали вычитания за использование приобретенного **плана трафика**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b73bc04d594a11efb927525400fdb830.png)

  - Детали использования включают ID плана, тип плана, использование, тип вычитания, коэффициент вычитания, размер вычитания и время вычитания.
  - **Использование плана трафика будет обновлено при создании счета на следующий день (точное время выставления счета может варьироваться).**
  - Правила вычитания для плана трафика см. в [Предоплаченный план](https://www.tencentcloud.com/document/product/267/52220#live_pag).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/be8cf5cb594a11ef9bf1525400a9236a.png)

- Просмотрите ваш **текущий режим выставления счетов** в верхнем правом углу. План трафика может использоваться для вычитания только в режиме **ежедневного выставления счетов по трафику**. Если вы используете другие режимы, ваш план будет заморожен и его действие не будет продлено.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c96d1375594a11ef8357525400bdab9d.png)

- Нажмите **Купить план**, чтобы перейти на страницу покупки для приобретения плана трафика.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cdb2b5d1594a11ef8357525400bdab9d.png)

> **Примечание:** Перед покупкой плана ознакомьтесь с [План трафика](https://www.tencentcloud.com/document/product/267/52220#.E6.B3.A8.E6.84.8F.E4.BA.8B.E9.A1.B9), чтобы узнать о **деталях выставления счетов** и **ограничениях**. Вы можете вернуть план в консоли, если он соответствует требованиям нашей [политики возврата](https://www.tencentcloud.com/document/product/267/52222).

## План транскодирования

Выберите [план](https://console.tencentcloud.com/live/resources/package?type=traffic) на левой боковой панели. На вкладке **План транскодирования** вы можете выполнять следующие операции:

- Просматривать использование, время покупки, время истечения и статус **плана стандартного транскодирования** и **плана быстрого транскодирования**.
![](https://staticintl.cloudcachetci.com/cms/backend-cms/ddec3d26594a11ef9bf1525400a9236a.png)
- Нажмите **Детали использования**, чтобы просмотреть детали вычитания за использование приобретенного **плана стандартного транскодирования** и **плана быстрого транскодирования**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e5672cc5594a11ef81cf525400d5f8ef.png)

  - Детали использования включают ID плана, тип плана, использование, тип вычитания, коэффициент вычитания, размер вычитания и время вычитания.
  - **Использование плана транскодирования будет обновлено при создании счета на следующий день (точное время выставления счета может варьироваться).**

![](https://staticintl.cloudcachetci.com/cms/backend-cms/eb86726d594a11efb66652540055f650.png)

- Просмотрите ваш **текущий режим выставления счетов** в верхнем правом углу. План транскодирования может использоваться для вычитания только в режиме **ежедневного выставления счетов по трафику**. Если вы используете другие режимы, ваш план будет заморожен и его действие не будет продлено.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f08e8a99594a11ef81cf525400d5f8ef.png)

- Нажмите **Купить план**, чтобы перейти на страницу покупки для приобретения плана транскодирования.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f4f5b659594a11ef998b525400f69702.png)

> **Примечание:** Перед покупкой плана ознакомьтесь с [План стандартного транскодирования](https://www.tencentcloud.com/document/product/267/52220#standard_pag) и [План быстрого транскодирования](https://www.tencentcloud.com/document/product/267/52220#topspeed_pag), чтобы узнать о **деталях выставления счетов** и **ограничениях**.

# Самодиагностика

CSS предоставляет инструмент самодиагностики, позволяющий быстро выявлять и устранять проблемы с трансляцией и воспроизведением, связанные с пользователями, URL-адресами, доменными именами, потоками и другими факторами. Эта функция в настоящее время проходит бета-тестирование. Результаты диагностики предоставляются только в справочных целях.

## Предварительные требования

- Имеется URL-адрес трансляции/воспроизведения, [составленный вами](https://intl.cloud.tencent.com/document/product/267/38393) или созданный [генератором адресов](https://intl.cloud.tencent.com/document/product/267/31084).
- URL-адрес трансляции был использован для [трансляции](https://intl.cloud.tencent.com/document/product/267/31558).

## Инструкции

Выполните следующие действия для диагностики проблемы трансляции/воспроизведения в прямом эфире:

1. Войдите в консоль CSS и выберите **Tools** > [Self-Diagnosis](https://console.tencentcloud.com/live/tools/selfcheck) в левой колонке.
2. Введите URL-адрес трансляции или воспроизведения, который требуется диагностировать.
3. Нажмите **Execute Diagnosis**.

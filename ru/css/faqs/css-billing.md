# Биллинг CSS

## Биллинг CSS

### Какие элементы CSS являются платными? Как определить, какие сборы необходимо оплатить?

- **Базовые сборы за услугу**: взимаются за исходящий трафик/полосу пропускания, генерируемые при подключении пользователей к серверам-источникам кэша. Другими словами, сборы за трафик/полосу пропускания будут взиматься каждый раз, когда зрители просматривают ваш контент прямой трансляции.

> **Примечание:** CSS предоставляет два режима выставления счетов на выбор: по трафику и по полосе пропускания. Подробную информацию о ценах см. в разделе «Выставление счетов по трафику/полосе пропускания» в документе [Обзор выставления счетов](https://intl.cloud.tencent.com/document/product/267/2818#bill-by-traffic.2Fbandwidth). Инструкции по смене режима выставления счетов см. в разделе [Изменение режимов выставления счетов](https://intl.cloud.tencent.com/document/product/267/30411).

- **Сборы за дополнительные услуги**: такие услуги включают трансодирование, запись, снимок экрана и обнаружение контента для взрослых, которые отключены по умолчанию. Если вы включите и используете их, будут взиматься сборы. Подробную информацию о ценах см. в разделе «Сборы за дополнительные услуги» в документе [Обзор ценообразования CSS](https://intl.cloud.tencent.com/document/product/267/2819#value-added-service-fees).
- **Сборы за расширенные услуги**: дополнительные услуги, предоставляемые совместно CSS и другими сервисами Tencent Cloud и рассчитываемые в соответствии с правилами выставления счетов соответствующих сервисов. Использование таких услуг влечет взимание сборов за расширенные услуги. Подробную информацию о ценах см. в разделе «Сборы за расширенные услуги» в документе [Обзор ценообразования CSS](https://intl.cloud.tencent.com/document/product/267/2819#extended-service-fees).

### Как я узнаю, есть ли просроченные платежи на моем аккаунте?

Вы можете войти в [консоль CSS](https://console.tencentcloud.com/live) и нажать **Fees** (Сборы) в правом верхнем углу, чтобы перейти на страницу обзора сборов. Если ваш доступный баланс ниже 0 USD, на вашем аккаунте есть просроченные платежи. Чтобы продолжить использование CSS и других услуг, вам необходимо своевременно оплатить просроченный баланс.

### Будут ли взиматься сборы за восходящую отправку?

- Восходящая отправка не взимается по умолчанию.
- Восходящее использование также будет взиматься, если соотношение исходящего трафика к входящему меньше 10:1 и максимальная используемая полоса пропускания восходящего потока в день превышает 100 Мбит/с. Метод выставления счетов, правила многоуровневого ценообразования и регионы выставления счетов для восходящего трафика внутри/вне материкового Китая **совпадают** с такими же параметрами для [исходящего трафика LVB](https://www.tencentcloud.com/document/product/267/2818#30ba7559-3d05-4e6f-90bc-f9f9fd86e56f). Восходящий трафик стал платежным элементом начиная с 00:00 (UTC+08:00), 1 июля 2021 года.

### Когда начинают взиматься сборы за дополнительные услуги?

После включения записи, снимков экрана, обнаружения контента для взрослых, водяных знаков и других дополнительных услуг, связанных с доменами отправки, сборы будут взиматься при начале отправки. После включения трансодирования и других дополнительных услуг, связанных с доменами воспроизведения, сборы будут взиматься при начале воспроизведения. Если вы создали шаблон трансодирования и связали его с доменом воспроизведения, сборы за трансодирование не будут взиматься при отсутствии воспроизведения. Включение водяного знака или смешивания потоков может потребовать сборы за стандартное трансодирование, которые будут рассчитываться по разрешению выходных потоков прямой трансляции.

### Есть ли две строки расходов для обнаружения контента для взрослых?

Поскольку CSS анализирует снимки экрана потоков прямой трансляции для обнаружения контента для взрослых, использование функции обнаружения контента для взрослых повлечет два вида расходов: **снимок экрана** и **обнаружение контента для взрослых**. Сборы за обнаружение контента для взрослых рассчитываются на основе количества проанализированных снимков экрана.

- CSS предоставляет бесплатную квоту из 1000 снимков экрана в месяц для функции снимков экрана.
- CSS предоставляет бесплатную квоту из 1000 снимков экрана в месяц для обнаружения контента для взрослых.

Вы можете просматривать использование пакетов в [консоли CSS](https://console.tencentcloud.com/live/resources/package?type=traffic), однако статистика использования в реальном времени в настоящее время не поддерживается.

### Могу ли я разрешить доступ к моему контенту только платящим пользователям?

CSS в настоящее время не может идентифицировать платящих пользователей. Если вы записываете свои потоки прямой трансляции и сохраняете их в VOD, вы можете разрешить доступ только платящим пользователям, зашифровав видео.

### Почему данные об использовании трафика, отображаемые в консоли, отличаются от данных журнала?

Исходящий трафик, записанный в журналы, представляет собой данные уровня приложения. Фактически потребляемый трафик на 5-15% больше, чем данные уровня приложения.

- Заголовки TCP/IP: каждый пакет запроса TCP/IP HTTP может быть размером до 1500 байт, включая 40-байтовые заголовки TCP и IP. Трафик, потребляемый заголовками (примерно 3% от общего объема), не учитывается в данных уровня приложения.
- Повторная передача TCP: примерно 3-10% пакетов теряются при передаче. Пакеты будут отправлены снова сервером. Такой трафик (примерно 3-7% от общего объема) также не учитывается уровнем приложения.

Как стандарт отрасли, плата взимается за трафик уровня приложения плюс дополнительный трафик, описанный выше. Tencent Cloud рассчитывает дополнительный трафик как 10% от общего объема, поэтому использование трафика, отображаемое в консоли, составляет 110% от записанного в журналах.

## Предоплаченные пакеты

### Я приобрел пакеты трафика. Почему сборы по-прежнему вычитаются с моего баланса счета?

Пакеты трафика могут только вычитать ваше использование трафика в ежедневном режиме выставления счетов по трафику.

- Если вы находитесь в ежедневном режиме выставления счетов по полосе пропускания, ваши пакеты трафика не могут быть использованы для вычета. Вы можете изменить режим выставления счетов в консоли. Подробнее см. в разделе [Изменение режимов выставления счетов](https://intl.cloud.tencent.com/document/product/267/30411).
- Если вы находитесь в ежедневном режиме выставления счетов по трафику и сборы по-прежнему вычитаются с баланса вашего счета, проверьте, используете ли вы [дополнительные услуги](https://intl.cloud.tencent.com/document/product/267/2819), такие как трансодирование прямой трансляции, запись прямой трансляции, снимки экрана и обнаружение контента для взрослых. Дополнительные услуги выставляются на основе использования. Ваше использование каждый день выставляется на следующий день. Вы также можете приобрести пакеты для дополнительных услуг. CSS предоставляет три типа пакетов: пакет трафика, пакет стандартного трансодирования и пакет трансодирования Top Speed Codec (TSC). Подробнее см. в разделе [Предоплаченные пакеты](https://www.tencentcloud.com/document/product/267/52220).
- Сборы также взимаются после того, как вы использовали все пакеты. Вы можете просмотреть детали вычитания в разделе [Bill Details](https://console.tencentcloud.com/expense/bill/summary). Подробные инструкции см. в разделе [Viewing Bills](https://intl.cloud.tencent.com/document/product/267/36278).

### Я приобрел только пакеты трафика. Приостановит ли CSS услуги для моего аккаунта после того, как я использую пакеты?

Нет. Пакеты трафика оплачиваются заранее, и использование вычитается из приобретенных пакетов.

- Если ваш режим выставления счетов для LVB является ежедневным выставлением счетов по трафику, исходящий трафик LVB сначала будет вычтен из ваших пакетов трафика, а дополнительное использование будет ежедневно выставляться по многоуровневым тарифам оплаты по мере использования.
- После использования пакетов убедитесь, что на вашем аккаунте достаточно средств для оплаты ежедневных сборов по мере использования. Если у вас есть просроченные платежи, пожалуйста, пополните счет в течение 24 часов. В противном случае услуги будут приостановлены для вашего аккаунта. Рекомендуется регулярно проверять свой [баланс счета](https://console.tencentcloud.com/expense/overview).

> **Примечание:** Пакеты трафика не могут быть использованы для вычета, если вы находитесь в ежемесячном режиме выставления счетов по трафику или ежемесячном/ежедневном режиме выставления счетов по полосе пропускания.

## Биллинг трансодирования

### Как взимается плата за трансодирование прямой трансляции? Как оценить сборы за трансодирование?

**Пример:** предположим, вы использовали трансодирование прямой трансляции и водяной знак прямой трансляции 1 января 2021 года, где поток прямой трансляции A был перекодирован в `H.264_720P` в течение 1 часа, а поток прямой трансляции B был отмечен водяным знаком при 480p в течение 30 минут.
Затем сборы за трансодирование, которые вам пришлось бы оплатить 2 января 2021 года, составили бы 0.0057 (USD/мин) × 60 (мин) + 0.0028 (USD/мин) × 30 (мин) = 0.426 USD.

### Почему был создан счет за трансодирование, если я не использовал трансодирование прямой трансляции?

Услуги трансодирования прямой трансляции включают трансодирование прямой трансляции в реальном времени, смешивание потоков и водяные знаки. Использование смешивания потоков или водяных знаков также повлечет взимание сборов за трансодирование.

### Будет ли смешивание потоков прямой трансляции обязательно влечь взимание сборов за трансодирование?

Да. Сборы за трансодирование будут выставляться в соответствии с выходным потоком после смешивания потоков. Поскольку ресурсы трансодирования будут потребляться после успешного смешивания потоков, продолжительность смешивания потоков будет использоваться для выставления счетов, независимо от того, воспроизводится ли поток. Это отличается от обычного трансодирования, которое выставляется по продолжительности воспроизведения.

### Запутаны в счете за трансодирование? Почему есть две единицы минут? Как определить, какой битрейт соответствует счету?

Платежи за услугу трансодирования различаются по установленным стратегиям ценообразования для разных методов кодирования и разрешений. Цена компонента за единицу (USD/мин/мин) может интерпретироваться как разрешение трансодирования, то есть (USD/разрешение/мин). Разные диапазоны цен трансодирования дифференцируются путем передачи разных объемов компонентов, поэтому объем компонента в счете должен соответствовать разрешению, а не фактической продолжительности трансодирования, которая включена в продолжительность использования счета. ## Биллинг записи прямой трансляции

### Как взимается плата за запись прямой трансляции?

Выставление счетов за функцию записи прямой трансляции основано на пиковом значении одновременных записей за месяц. Если запись хранится в COS, также будут взиматься платежи за общую продолжительность записи, доставленной в COS. Каждая запись прямой трансляции считается одним каналом записи; если вы записываете в два формата (MP4 и HLS) и запись длится один час, это будет считаться двумя каналами записи, а общая продолжительность записи, доставленной в COS, будет считаться двумя часами.

### Как рассчитывается пик канала записи прямой трансляции?

Когда один поток прямой трансляции (один ID потока) записывается в один формат файла, он будет считаться одним каналом записи прямой трансляции. Количество текущих каналов записи будет запрашиваться один раз каждые 5 минут, и максимальное значение точек выборки за текущий месяц будет использоваться как пик месячного канала записи для выставления счетов.
**Пример:**

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e76e735fe78211eebbb2525400564496.png)

> **Примечание:** Желтый: задачи записи под ID потока **A**. Зеленый: задачи записи под ID потока **B**. Синий: задачи записи под ID потока **C**.

### Почему мне выставили счет в размере 10.5882 USD после использования записи прямой трансляции?

Если два потока прямой трансляции записываются одновременно или один поток прямой трансляции записывается в два формата файла, будут созданы два канала записи. Запись прямой трансляции выставляется по пику канала записи по цене 5.2941 USD за канал в месяц, поэтому если в месяц приходится два канала, то будет взиматься плата в размере 10.5882 USD. Для получения дополнительной информации см. раздел [Live Recording Billing](https://intl.cloud.tencent.com/document/product/267/39605).

Рекомендуется перейти в **Bill Details** (Детали счета) > [**Bill by Instance**](https://console.tencentcloud.com/expense/bill/summary) в центре выставления счетов для проверки сведений о выставлении счетов за запись прямой трансляции. Вы можете нажать **Bill Details** (Детали счета) в столбце «Operation» (Операция) для просмотра фактического пика канала записи за последний месяц.

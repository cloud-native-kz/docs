# Различие между прерыванием потока и приостановкой потока

## 1. Прерывание потока

Если транслируемый поток прерывается, передача останавливается и зрители не могут смотреть прямую трансляцию. При прерывании потока передача может быть возобновлена с конца VJ для продолжения прямой трансляции.

## 2. Приостановка потока

Если транслируемый поток приостанавливается, передача останавливается и зрители не могут смотреть прямую трансляцию. При приостановке потока передача не может быть инициирована с конца VJ в течение определенного периода времени. Время окончания приостановки можно установить в консоли, и период приостановки не может превышать один месяц.
Например, если приостановка потока включена для потока test123 01.01.2018 12:00 и время окончания приостановки установлено на 05.01.2018 12:00, VJ этого потока не может передавать поток и зрители не могут смотреть прямую трансляцию в течение этого периода.

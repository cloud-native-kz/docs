# Регион обслуживания

### Какие регионы поддерживаются ускорением доменного имени CSS?

CSS имеет функцию глобального ускорения, и вы можете выбрать подходящий тип региона ускорения в зависимости от региона вашего бизнеса.

| Тип региона ускорения доменного имени воспроизведения | Доступный тип региона | Примечания |
| --- | --- | --- |
| Материковая часть Китая | Материковая часть Китая | Если в качестве региона ускорения выбрана "материковая часть Китая", ограничение пропускной способности будет поддерживаться только на материковой части Китая. |
| Гонконг/Макао/Тайвань (регион Китая) и другие регионы | Гонконг/Макао/Тайвань (регион Китая) и другие регионы | Если в качестве региона ускорения выбран "Гонконг/Макао/Тайвань (регион Китая) и другие регионы", ограничение пропускной способности будет поддерживаться только в Гонконге/Макао/Тайване (регион Китая) и других регионах. |
| Глобальное ускорение | Материковая часть Китая; Гонконг/Макао/Тайвань (регион Китая) и другие регионы | Если в качестве региона ускорения выбрано "глобальное ускорение", вы можете установить разные ограничения пропускной способности для материковой части Китая и Гонконга/Макао/Тайваня (регион Китая) и других регионов отдельно. Вы также можете установить ограничение пропускной способности для глобального ускорения. |

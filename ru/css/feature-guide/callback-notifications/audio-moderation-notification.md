# Уведомление о модерации аудио

Если вы настроили адрес обратного вызова модерации аудио, сервер вернёт результаты модерации в формате JSON на указанный адрес обратного вызова после завершения модерации аудио. Затем вы сможете выполнить последующие операции обработки файла на основе содержимого обратного вызова.

Данный документ описывает поля уведомления сообщения обратного вызова, отправляемого Tencent Cloud Streaming Services (CSS) пользователю после срабатывания события обратного вызова модерации аудио.

## Примечание

- Перед прочтением данного документа убедитесь, что вы понимаете, как настраивается функция обратного вызова и как получаются сообщения обратного вызова в Tencent CSS. Подробные инструкции см. в разделе [Как получать уведомления о событиях](https://www.tencentcloud.com/document/product/267/38080).
- По умолчанию возвращаются только потенциально несоответствующие результаты. Соответствующие результаты не возвращаются.

## Параметры обратного вызова модерации аудио

### Параметр типа события

| Тип события | Описание значения поля |
| --- | --- |
| Модерация аудио | event_type = 315 |

### Общие параметры обратного вызова

| Имя поля | Тип | Описание |
| --- | --- | --- |
| t | int64 | Время истечения. Это временная метка UNIX, обозначающая истечение подписи уведомления о событии. Время истечения по умолчанию для уведомлений сообщений от Tencent Cloud составляет 10 минут. Если время, указанное значением `t` в уведомлении о сообщении, истекло, уведомление считается недействительным, что защищает от атак воспроизведения по сети. Формат `t` — это десятичная временная метка UNIX, которая представляет секунды, прошедшие с полуночи 1 января 1970 года (UTC/GMT). |
| sign | string | Подпись безопасности для уведомления о событии: `sign` = MD5(`key` + `t`). Tencent Cloud объединяет зашифрованный **ключ** и t в строку, а затем использует вычисление MD5 для получения значения sign, которое затем помещается в сообщение уведомления. После получения уведомления ваш сервер обслуживания может определить, является ли sign правильным, используя тот же алгоритм, таким образом подтверждая, исходит ли сообщение от серверной части Tencent Cloud. |

> **Примечание:** Вы можете установить ключ обратного вызова в разделе **Event Center** > [Live Stream Callback](https://console.tencentcloud.com/live/config/callback), который используется для аутентификации. Рекомендуется указать это поле для обеспечения безопасности данных. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/5d15eadecb6711f0a93d52540044a08e.png)

### Параметры сообщения обратного вызова

| Параметр | Обязателен или нет | Тип данных | Описание |
| --- | --- | --- | --- |
| appid | Обязателен | Number | Бизнес-идентификатор. |
| bizid | Опционально | Number | Бизнес-идентификатор (устаревший). |
| stream_id | Обязателен | String | Название потока. |
| channel_id | Обязателен | String | Идентификатор канала. |
| domain | Обязателен | String | Домен отправки. |
| path | Опционально | String | Путь потока отправки. |
| HitFlag | Опционально | Number | Это поле используется для указания того, был ли контент под проверкой активирован моделью аудита. Значения: 0 (не активирован), 1 (активирован). |
| Score | Опционально | Number | Это поле используется для возврата уровня достоверности под текущий тег с диапазоном значений от 0 (самая низкая достоверность) до 100 (самая высокая достоверность). |
| SubTag | Опционально | String | Это поле служит вторичным подчинённым ярлыком. |
| task_id | Опционально | Number | Идентификатор задачи проверки аудио |
| status | Опционально | Number | Значение статуса обратного вызова: 2 (Нормальное) |
| asr_text | Опционально | String | Текстовая транскрипция. |
| cdn_url | Опционально | String | Адрес CDN. |
| duration | Опционально | Number | Продолжительность распознавания речи (секунды). |
| label | Опционально | String | Это поле используется для возврата ярлыка вредоносного контента с наивысшим приоритетом в результате обнаружения, чтобы указать рекомендуемый результат модерации модели. Рекомендуется обрабатывать различные типы нарушений и рекомендуемые значения в соответствии с вашими требованиями к бизнесу. |
| language_results | Опционально | Array of AudioResultDetailLanguageResult | Это поле возвращает результаты обнаружения для языка меньшинства. Конкретное содержимое результатов см. в подробных описаниях структуры данных AudioResultDetailLanguageResult. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| moan_results | Опционально | Array of MoanResult | Результат модерации вульгарного контента в аудио; Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| recognition_results | Опционально | Array of RecognitionResult | Активированные ярлыки распознавания. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| request_id | Опционально | String | Идентификатор запроса |
| seq | Опционально | Number | Последовательность аудио |
| speaker_results | Опционально | Array of AudioResultDetailSpeakerResult | Результат идентификации говорящего в аудио. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| sub_label | Опционально | String | Имя подярлыка. Если подярлык не совпадает, будет возвращена пустая строка. |
| suggestion | Опционально | string | Рекомендуемое значение. Допустимые значения: Block: фильтрация контента; Review: ожидается повторная модерация; Pass: нормально |
| text_results | Опционально | Array of TextResult | Результат модерации содержимого диалога в аудио. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| data | Опционально | Data | Результат распознавания речи. |

#### AudioResultDetailLanguageResult

Результат обнаружения языка меньшинства в аудио.

| Имя | Тип | Описание |
| --- | --- | --- |
| Label | String | Это поле используется для возврата соответствующей информации о типе языка. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| Score | Integer | Этот параметр используется для возврата достоверности текущего ярлыка. Диапазон значений: 0 (самая низкая достоверность) до 100 (самая высокая достоверность). Большее значение указывает на более высокую вероятность того, что аудио принадлежит текущему возвращаемому ярлыку языка. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| StartTime | Float | Этот параметр используется для возврата времени начала сегмента, соответствующего указанному ярлыку языка в файле аудио, в секундах. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| EndTime | Float | Этот параметр используется для возврата времени окончания сегмента, соответствующего указанному ярлыку языка в файле аудио, в секундах. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |

#### MoanResult

Результат модерации вульгарного контента.

| Имя | Тип | Описание |
| --- | --- | --- |
| Label | String | Значение этого поля фиксировано — Moan. Если в результате обратного вызова для аудио нет **MoanResult**, то в этом аудио нет соответствующих нарушений, связанных со стоном/тяжелым дыханием. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |
| Score | Integer | Уверенность, определённая машиной для текущей категории. Диапазон значений: 0–100. Более высокий балл указывает на более высокую вероятность принадлежности к текущей категории. (Пример: Moan 99 указывает на то, что образец имеет высокую вероятность принадлежности к категории стона/тяжелого дыхания) |
| Suggestion | String | Рекомендуемое действие после получения результатов. Рекомендуемое значение. Допустимые значения: Block: рекомендуется блокировка. Review: рекомендуется повторная модерация. Pass: рекомендуется пропуск. |
| StartTime | Float | Время начала события нарушения в секундах (s). |
| EndTime | Float | Время окончания события нарушения в секундах (s). |
| SubLabel | String | Это поле используется для возврата вторичного ярлыка под текущим ярлыком (Label). Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |

| Tags | Array of Label | Список ярлыков идентификации. Примечание: **Это поле может возвращать null, указывая на отсутствие доступного действительного значения.** |

Большее значение указывает на более высокую вероятность того, что аудио является голосовым отпечатком говорящего.

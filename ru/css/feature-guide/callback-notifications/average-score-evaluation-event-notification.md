# Уведомление о событии оценки среднего балла

Если вы настроили URL обратного вызова для оценки среднего балла, система автоматически отправит результаты оценки в формате JSON на указанный адрес обратного вызова во время процесса оценки. Затем вы можете использовать данные этого обратного вызова для последующей обработки.

Этот документ в основном описывает поля уведомления обратного вызова, которые Tencent Cloud Streaming Services (CSS) отправляет пользователям при срабатывании события оценки среднего балла.

## Примечание

В этом руководстве предполагается, что вы знаете, как настроить обратные вызовы и получать уведомления о событиях от CSS. Подробнее см. в разделе [Как получить уведомление о событии](https://intl.cloud.tencent.com/document/product/267/38080).

## Параметры события оценки среднего балла

### Тип события

| Тип события | Значение |
| --- | --- |
| Оценка среднего балла | event_type = 345 |

### Общие параметры обратного вызова

| Параметр | Тип | Описание |
| --- | --- | --- |
| t | int64 | Время истечения, представляющее собой UNIX-временную метку момента истечения подписи уведомления о событии. Период действия уведомления обратного вызова от Tencent Cloud по умолчанию составляет 10 минут. Если указанное значением `t` время истекло, то такое уведомление считается недействительным. Это предотвращает атаки воспроизведения в сети. Значение `t` является десятичной UNIX-временной меткой, то есть количеством секунд, прошедших с 00:00:00 (время UTC/GMT) 1 января 1970 г. |
| sign | string | Подпись безопасности. sign = MD5(key + t). Примечание: Tencent Cloud объединяет ключ шифрования **key** и `t`, генерирует хэш MD5 объединённой строки и встраивает его в сообщения обратного вызова. Ваш внутренний сервер может выполнить тот же расчёт при получении сообщения обратного вызова. Если подпись совпадает, это означает, что сообщение поступает от Tencent Cloud. |

> **Примечание:** Вы можете установить ключ обратного вызова в разделе **Конфигурация функций** > [Обратный вызов прямой трансляции](https://console.tencentcloud.com/live/config/callback), который используется для аутентификации. Рекомендуется установить это поле для обеспечения безопасности данных. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/72960661cb6711f09e745254007c27c5.png)

### Параметры обратного вызова

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| appid | int | ID бизнеса. |
| appname | string | Путь потока отправки. |
| stream_id | string | ID потока. |
| domain | string | Доменное имя отправки. |
| event_time | int64 | Время отправки запроса в UNIX-временной метке (в секундах). |
| transcode_template_id | int | ID шаблона перекодирования. |
| watermark_template_id | int | ID шаблона водяного знака. Оценка с опорой (PSNR/SSIM/VMAF) оценивает только перекодированные потоки. Идентификатор шаблона водяного знака не предоставляется. |
| frames | Array of [Frame](#labelresult) | Балл и время каждого кадра в период оценки. |
| average_score | float | Средний балл в период оценки. |
| max_score | float | Максимальный балл в период оценки. |
| min_score | float | Минимальный балл в период оценки. |
| function | string | Категория функции: оценка. |
| class | string | Конкретные типы оценки: PSNR, SSIM, VMAF, оценка без опоры (VQA) |

#### Frame

Балл оценки отдельного кадра и время.

| Название | Тип | Описание |
| --- | --- | --- |
| score | float | Балл. |
| time | int64 | UNIX-временная метка в миллисекундах. |


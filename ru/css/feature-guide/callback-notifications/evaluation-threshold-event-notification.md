# Уведомление о событии порога оценки

Если вы настроили URL-адрес обратного вызова порога оценки, во время процесса оценки бэкэнд автоматически отправляет результаты, соответствующие критериям порога, на ваш адрес обратного вызова в формате JSON. Вы можете использовать эти данные обратного вызова для последующих операций обработки.

Этот документ в основном раскрывает поля уведомления обратного вызова, передаваемые пользователям сервисом потокового видео Tencent Cloud (CSS) при срабатывании события порога оценки.

## Примечание

В этом руководстве предполагается, что вы знаете, как настраивать обратные вызовы и получать уведомления о событиях от CSS. Дополнительные сведения см. в разделе [Как получать уведомления о событиях](https://intl.cloud.tencent.com/document/product/267/38080).

## Параметры события порога оценки

### Тип события

| Тип события | Значение |
| --- | --- |
|  Порог оценки | event_type = 344 |

### Общие параметры обратного вызова

| Параметр | Тип | Описание |
| --- | --- | --- |
| t | int64 | Время истечения — это временная метка Unix, когда истекает срок действия подписи уведомления о событии. Период действия уведомления обратного вызова из Tencent Cloud по умолчанию составляет 10 минут. Если время, указанное значением `t` в уведомлении, истекло, то это уведомление считается недействительным. Это предотвращает атаки воспроизведения по сети. Значение `t` — это десятичная временная метка Unix, то есть количество секунд, прошедших с 00:00:00 (время UTC/GMT), 1 января 1970 г. |
| sign | string | Подпись безопасности. sign = MD5(key + t). Примечание: Tencent Cloud объединяет **ключ** шифрования и `t`, генерирует хеш MD5 объединённой строки и внедряет его в сообщения обратного вызова. Ваш серверный сервер может выполнить тот же расчёт при получении сообщения обратного вызова. Если подпись совпадает, это означает, что сообщение отправлено с Tencent Cloud. |

> **Примечание:** Вы можете установить ключ обратного вызова в **Конфигурация функций** > [Обратный вызов прямого потока](https://console.tencentcloud.com/live/config/callback), который используется для аутентификации. Мы рекомендуем установить это поле для обеспечения безопасности данных.![](https://staticintl.cloudcachetci.com/cms/backend-cms/6bf88bd0cb6711f0aedb525400454e06.png)![](https://staticintl.cloudcachetci.com/cms/backend-cms/2a581f76840a11f0b321525400e889b2.png)

### Параметры сообщения обратного вызова

| Параметр | Тип данных | Описание |
| --- | --- | --- |
| appid | int | ID бизнеса. |
| appname | string | Путь потока отправки. |
| stream_id | string | ID потока. |
| domain | string | Доменное имя отправки. |
| event_time | int64 | Время отправки запроса в формате временной метки UNIX (в секундах). |
| transcode_template_id | int | ID шаблона транскодирования. |
| watermark_template_id | int | ID шаблона водяного знака. Оценка с привязкой (PSNR/SSIM/VMAF) оценивает только транскодированные потоки. ID шаблона водяного знака не предоставляется. |
| frames | Array of [Frame](#labelresult) | Оценка и время ниже порога в период оценки. |
| function | string | Категория функции: Assessment (Оценка). |
| class | string | Конкретные типы оценок: PSNR, SSIM, VMAF, Scoring Without Reference(VQA) |

#### Frame

Оценка отдельного кадра и время.

| Имя | Тип | Описание |
| --- | --- | --- |
| score | float | Оценка. |
| time | int64 | Временная метка UNIX в миллисекундах. |

### Пример обратного вызова

```
{    "event_type": 344,    "appid": 1234576,    "appname": "live",    "domain": "abc.record.test.org",    "stream_id": "livestream1",    "transcode_template_id": 12345,    "event_time": 1752130787,    "frames": [        {            "score": 22.78,            "time": 1752130785280        },        {            "score": 32.75,            "time": 1752130786281        }    ],    "function": "Assessment",    "class": "PSNR"}
```

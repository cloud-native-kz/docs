# Уведомление о событии проверки изображений при прямой трансляции

Если вы настроили адрес обратного вызова проверки содержимого изображения, сервер вернёт результаты проверки в формате JSON на указанный адрес обратного вызова после завершения проверки. Затем вы можете выполнить последующие операции обработки файлов на основе содержимого обратного вызова.

В этом документе описаны поля уведомлений сообщений обратного вызова, отправляемых сервисом облачной потоковой передачи Tencent Cloud (CSS) пользователю после активации события обратного вызова проверки изображения.

## Примечание

- Перед прочтением этого документа вам необходимо понять, как настроить обратные вызовы и как вы будете получать сообщения через облачный CSS Tencent Cloud. Для получения дополнительной информации см. [Как получать уведомления о событиях](https://intl.cloud.tencent.com/document/product/267/38080).
- По умолчанию возвращаются только потенциально несоответствующие результаты. Соответствующие результаты не возвращаются.
- Рекомендуется использовать параметры type, score и suggestion из сообщения обратного вызова для проверки несоответствующих изображений. Система проверки не обеспечивает 100% точность, поэтому несколько изображений могут быть идентифицированы как потенциально несоответствующие или неправильно идентифицированы. Вы можете определить, требуется ли ручная проверка, на основе фактического сценария приложения.
- Сервис проверки изображений был обновлён, и некоторые параметры сообщений обратного вызова были скорректированы. Рекомендуется ознакомиться с последней версией и следующим описанием параметров. Для обеспечения совместимости со старыми версиями для пользователей мы по-прежнему будем вызывать некоторые параметры, которые в настоящее время не используются, на сервер приёма, включая: tid, abductionRisk, confidence, normalScore, hotScore, pornScore, terrorScore, polityScore, illegalScore, similarScore, abuseScore, teenagerScore, adScore и customScore.

## Параметры обратного вызова проверки изображения

### Параметр типа события

| Тип события | Значение параметра |
| --- | --- |
| Проверка изображения | event_type = 317 |

### Общие параметры обратного вызова

| Параметр | Тип | Описание |
| --- | --- | --- |
| t | int64 | Время истечения — это временная метка Unix, когда истекает подпись уведомления о событии. Период действия по умолчанию уведомления о сообщении от Tencent Cloud составляет 10 минут. Если время, указанное значением `t` в уведомлении о сообщении, истекло, такое уведомление считается недействительным, что предотвращает атаки повторного воспроизведения в сети. Формат `t` — это десятичная временная метка Unix, то есть количество секунд, прошедших с 00:00:00 (время UTC/GMT), 1 января 1970 года. |
| sign | string | Подпись безопасности уведомления о событии: sign = MD5(key + t). Примечание: Tencent Cloud объединяет ключ шифрования **key** и `t`, вычисляет значение `sign` через MD5 и помещает его в сообщение уведомления. Когда ваш серверный сервер получает сообщение уведомления, он может подтвердить правильность `sign` на основе того же алгоритма и затем определить, действительно ли сообщение поступило с серверной части Tencent Cloud. |

> **Примечание:** Вы можете установить ключ обратного вызова в **Event Center** > [**Live Stream Callback**](https://console.tencentcloud.com/live/config/callback), который используется для аутентификации. Рекомендуется установить это поле для обеспечения безопасности данных.![](https://staticintl.cloudcachetci.com/cms/backend-cms/5695bdeacb6711f0b011525400bf7822.png)

### Параметры сообщения обратного вызова

| Параметр | Обязательный или нет | Тип данных | Описание |
| --- | --- | --- | --- |
| streamId | Обязательный | String | Имя потока. |
| channelId | Обязательный | String | ID канала. |
| img | Обязательный | String | Ссылка на проверяемое изображение |
| type | Обязательный | Array | Категории негативных меток с наивысшим приоритетом в результате обнаружения. Подробнее см. описание `label`. |
| score | Обязательный | Array | Оценки `type` |
| ocrMsg | Дополнительный | String | Результат OCR (если имеется) |
| suggestion | Обязательный | String | Рекомендация. Допустимые значения: BlockReviewPass |
| label | Дополнительный | String | Негативная метка с наивысшим приоритетом в результате обнаружения (labelResults, objectResults, ocrResults). Это результат проверки, предложенный моделью. Рекомендуется обрабатывать различные типы нарушений и предложений на основе ваших потребностей в бизнесе. |
| subLabel | Обязательный | String | Подметка под негативной меткой с наивысшим приоритетом в результате обнаружения, такая как porn - sexual acts. Если содержимое не имеет подметки, этот параметр будет пустым. |
| labelResults | Дополнительный | [Array of LabelResult](#labelresult) | Детали попадания негативной метки модели категорий, включая обнаруженное содержимое для взрослых, объявления, контент о терроризме и политически чувствительный контент. Примечание: **Это поле может вернуть** `null` **, что указывает на то, что не удалось получить допустимые значения.** |
| objectResults | Дополнительный | [Array of ObjectResult](#objectresult) | Результат обнаружения модели объектов, включая имя метки, оценку попадания, координаты, сценарий и предлагаемую операцию в отношении объектов, логотипов рекламы, QR-кодов и т. д. Подробнее см. описание структуры данных `ObjectResults`. Примечание: **Это поле может вернуть** `null` **, что указывает на то, что не удалось получить допустимые значения.** |
| ocrResults | Дополнительный | [Array of OcrResult](#ocrresult) | Результат OCR, включая координаты текста, распознанный текст, предлагаемую операцию и т. д. Подробнее см. описание структуры данных `OcrResults`. Примечание: **Это поле может вернуть** `null` **, что указывает на то, что не удалось получить допустимые значения.** |
| libResults | Дополнительный | [Array of LibResult](#libresult) | Результат проверки по чёрному/белому списку |
| screenshotTime | Обязательный | Number | Время снимка экрана |
| sendTime | Обязательный | Number | Время отправки запроса в формате временной метки Unix |
| stream_param | Обязательный | String | Параметры трансляции |
| app | Обязательный | String | Домен трансляции. |
| appid | Обязательный | Number | ID приложения |
| appname | Обязательный | String | Путь трансляции |

#### LabelResult

Результат попадания модели категорий

| Параметр | Тип | Описание |
| --- | --- | --- |
| Scene | String | Сценарий, определённый моделью, такой как реклама, содержимое для взрослых и вредоносный контент |
| Suggestion | String | Операция, предлагаемая системой для текущей негативной метки. Рекомендуется обрабатывать различные типы нарушений и предложений на основе ваших потребностей в бизнесе. Возвращаемые значения: BlockReviewPass |
| label | String | Негативная метка в результате обнаружения |
| SubLabel | String | Имя подметки |
| Score | Integer | Оценка попадания модели метки |
| Details | [Array of LabelDetailItem](#labeldetailitem) | Детали попадания подметки модели категорий |

| Details | [Array of ObjectDetail](#objectdetail) | Детали обнаружения объекта |

#### ObjectDetail

Детали обнаружения объекта. Когда сценарий обнаружения — объект, логотип рекламы или QR-код, возвращаются имя метки, значение метки, оценка метки и информация о расположении кадра обнаружения.

| Параметр | Тип | Описание |
| --- | --- | --- |
| Id | Integer | ID обнаруженного объекта |
| Name | String | Идентифицированная метка объекта |
| Value | String | Значение или содержимое идентифицированной метки объекта. Например, если метка — QR-код ( `QrCode`), этот параметр — это URL QR-кода. |
| Score | Integer | Оценка попадания метки объекта. Диапазон значений: 0–100. Например, `QrCode 99` указывает на высокую вероятность того, что содержимое является QR-кодом. |
| Location | [Location](#location) | Координаты (верхнего левого угла), размеры и поворот кадра обнаружения объекта |

#### Location

Координаты и другая информация кадра обнаружения

| Параметр | Тип | Описание |
| --- | --- | --- |
| X | Float | Горизонтальная координата верхнего левого угла |
| Y | Float | Вертикальная координата верхнего левого угла |
| Width | Float | Ширина |
| Height | Float | Высота |
| Rotate | Float | Угол поворота кадра обнаружения |

# Уведомление о событии контроля качества

Если вы настроили адрес обратного вызова для контроля качества, во время процесса контроля качества backend отправит результаты в формате JSON на ваш URL обратного вызова. Вы можете использовать содержимое обратного вызова для выполнения последующих операций обработки.

Этот документ в основном объясняет поля уведомления обратного вызова, передаваемые Tencent Cloud Streaming Services (CSS) пользователям при запуске события контроля качества.

## Примечание

Данное руководство предполагает, что вы знаете, как настроить обратные вызовы и получать уведомления обратного вызова от CSS. Для получения подробной информации см. [Как получать уведомления о событиях](https://intl.cloud.tencent.com/document/product/267/38080).

## Параметры события контроля качества

### Тип события

| Тип события | Значение |
| --- | --- |
| Обратный вызов контроля качества | event_type = 343 |

### Общие параметры обратного вызова

| Параметр | Тип | Описание |
| --- | --- | --- |
| t | int64 | Время истечения, это временная метка Unix, когда истекает подпись уведомления о событии. Период действительности уведомления обратного вызова от Tencent Cloud по умолчанию составляет 10 минут. Если время, указанное значением `t` в уведомлении, истекло, такое уведомление считается недействительным. Это предотвращает атаки повторного воспроизведения сети. Значение `t` — это десятичная временная метка Unix, то есть количество секунд, прошедших с 00:00:00 (время UTC/GMT), 1 января 1970 года. |
| sign | string | Подпись безопасности. sign = MD5(key + t). Примечание: Tencent Cloud объединяет ключ шифрования **key** и `t`, генерирует хеш MD5 объединенной строки и встраивает его в сообщения обратного вызова. Ваш backend-сервер может выполнить те же вычисления при получении сообщения обратного вызова. Если подпись совпадает, это означает, что сообщение поступает от Tencent Cloud. |

> **Примечание:** Вы можете установить ключ обратного вызова **key** в разделе **Конфигурация функций** > [Обратный вызов прямого эфира](https://console.tencentcloud.com/live/config/callback), который используется для аутентификации. Рекомендуется установить это поле для обеспечения безопасности данных. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/65519e84cb6711f0a93d52540044a08e.png)

### Параметры обратного вызова

| Параметр | Типы данных | Описание |
| --- | --- | --- |
| appid | int | ID приложения. |
| appname | string | Путь потока трансляции. |
| stream_id | string | ID потока. |
| domain | string | Доменное имя трансляции. |
| event_time | int64 | Временная метка запроса, время эпохи UNIX (в секундах). |
| transcode_template_id | int | ID шаблона трансляции. |
| watermark_template_id | int | ID шаблона водяного знака. |
| score | float | Фактическое значение оценки. |
| function | string | Категория функции: QualityControl. |
| class | string | Тип исключения контроля качества. |

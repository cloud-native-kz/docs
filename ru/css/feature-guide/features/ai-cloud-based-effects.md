# AI облачные эффекты

Функция AI облачных эффектов Cloud Streaming Services (CSS) интегрируется с технологией AI преобразования текста в видео, позволяя пользователям генерировать персонализированные видео со спецэффектами в реальном времени путем ввода текстовых описаний (Prompts). Пользователи также могут управлять списком созданных спецэффектов через консоль или API, а также отправлять видео со спецэффектами на указанный прямой эфир. Эта функция улучшает интерактивный опыт пользователей, создавая уникальные методы эмоционального выражения для стримеров и аудитории. По сравнению с традиционными фиксированными спецэффектами подарков, это повысило интерактивность.

## **Обязательно знать**

- Использование функции AI облачных эффектов CSS будет включать две платежа: один за генерирование видео со спецэффектами и еще один за его отправку. Конкретные сборы взимаются в зависимости от фактического использования. Для получения соответствующей информации о биллинге см. [Документацию](https://www.tencentcloud.com/document/product/267/70336).
- Видео со спецэффектами генерируются большими моделями и имеют определенную степень случайности. Качество видеоэффектов будет постепенно улучшаться с эволюцией технологии и итерациями версий.
- Строго запрещено использование Prompts, содержащих запрещенные чувствительные слова (такие как насилие и незаконные чувствительные слова).

## **Руководство пользователя**

#### Консоль - AI облачные эффекты

a. Вы можете просмотреть список видео со спецэффектами, который отображает Prompt, использованный для генерирования каждого видео, а также информацию, такую как тип видео.

b. В списке операторы могут нажать **Отправить спецэффект**, ввести имя потока для соответствия онлайн потоку и запустить отправку видео со спецэффектами.

#### TencentCloud API - API, связанный с видео со спецэффектами

a. Полный список видео со спецэффектами может быть получен через API запроса DescribeLiveCloudEffectList .

**Отправить тело запроса:**

> **Примечание:** Когда входной параметр GiftType пуст, все видео со спецэффектами будут возвращены. Вы можете указать PGC (значение по умолчанию системы), UCG (загруженное пользователями) или AIGC (сгенерированное пользователями с помощью AI).

```
{	"Action": "DescribeLiveCloudEffectList",	"GiftType": "PGC",	"Region": "ap-guangzhou",	"RequestId": "1047d0dc-6dc8-4898-a7f3-03726a822b0e",	"RequestSource": "API",	"Operator": "admin"}
```

**Ответ API:**

```
{	"Response": {		"GiftInfoList": [{			"CreateTime": "2025-05-19T00:00:00Z",			"GiftId": "gift_1",			"GiftTag": "animals",			"GiftType": "PGC",			"PreviewImageUrl": "https://vinson-1258442684.cos.ap-nanjing.myqcloud.com/texiao_image/parrot_v3.png",			"Prompt": "parrot",			"UpdateTime": "2025-05-19T00:00:00Z"		},		{			"CreateTime": "2025-05-19T00:00:00Z",			"GiftId": "gift_2",			"GiftTag": "animals",			"GiftType": "PGC",			"PreviewImageUrl": "https://vinson-1258442684.cos.ap-nanjing.myqcloud.com/texiao_image/dog_v7.png",			"Prompt": "puppy",			"UpdateTime": "2025-05-19T00:00:00Z"		}],		"RequestId": "1047d0dc-6dc8-4898-a7f3-03726a822b0e"	}}
```

b. Отправить указанное видео со спецэффектами на прямой эфир через API отправки SendLiveCloudEffect, поддерживает указание коэффициента масштабирования спецэффектов при передаче, а также их относительное положение на экране трансляции. Кроме того, можно настроить количество циклов воспроизведения спецэффектов.

**Отправить тело запроса:**

> **Примечание:** API будет выполнять проверку активности входящего прямого эфира и проверку существования указанного GiftId. Вы должны предоставить правильные параметры.

```
{	"Action": " SendLiveCloudEffect",	"GiftId": "gift_1",	"PushDomainName": "5000.livepush.com",	"AppName": "live",	"StreamName": "mystream",	"Region": "ap-guangzhou",	"RequestId": "1047d0dc-6dc8-4898-a7f3-03726a822b0e",	"RequestSource": "API",    "LoopTimes": 3,    "ZoomFactor": 2,    "XPosition": 50,    "YPosition": 50,	"Operator": "admin"}
```

**Ответ API:**

```
{	"Response": {		"RequestId": "1047d0dc-6dc8-4898-a7f3-03726a822b0e"	}}
```

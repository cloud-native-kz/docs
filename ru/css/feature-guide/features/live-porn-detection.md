# Обнаружение порнографии в прямом эфире

Для использования функции обнаружения порнографии CSS необходимо включить захват экрана. Вы можете использовать функцию обнаружения порнографии либо путём настройки в [консоли CSS](https://intl.cloud.tencent.com/document/product/267/31072), либо с помощью API. В этом документе показано, как использовать API обнаружения порнографии.

## Важная информация

Если ваш bucket COS разрешает общедоступное чтение и содержит политически чувствительный, порнографический или другой неприемлемый контент, во избежание блокировки bucket удалите этот контент в первую очередь.

## Включение обнаружения порнографии

Поскольку функция обнаружения порнографии основана на захвате экрана, сначала необходимо включить захват экрана.

### 1. Создание шаблона захвата экрана с включённым обнаружением порнографии

Вызовите [CreateLiveSnapshotTemplate](https://intl.cloud.tencent.com/document/product/267/30834), установив `PornFlag` на `1` для создания шаблона захвата экрана с включённым обнаружением порнографии.

### 2. Создание правила захвата экрана

Вызовите [CreateLiveSnapshotRule](https://intl.cloud.tencent.com/document/product/267/30835), установив `TemplateId` на ID шаблона захвата экрана, созданного на шаге 1, для связи шаблона с целевыми `AppId`, `DomainName`, `AppName` и `StreamName`.

### 3. Запуск прямого эфира

После создания правила захвата экрана с включённым обнаружением порнографии функция обнаружения порнографии будет автоматически включена для новых потоков. Если вы хотите включить обнаружение порнографии для текущего потока, необходимо остановить и перезапустить поток.

## Получение результата обнаружения порнографии

После включения обнаружения порнографии вы можете настроить зарегистрированное имя домена в шаблоне обратного вызова обнаружения порнографии для получения обратных вызовов с результатами обнаружения.

> **Примечание:** По умолчанию обратные вызовы отправляются только при обнаружении подозрительного контента.

### 1. Создание шаблона обратного вызова обнаружения порнографии

Вызовите [CreateLiveCallbackTemplate](https://intl.cloud.tencent.com/document/product/267/30815), установив `PornCensorshipNotifyUrl` на ваше имя домена для создания шаблона обратного вызова обнаружения порнографии.

### 2. Создание правила обратного вызова обнаружения порнографии

Вызовите [CreateLiveCallbackRule](https://intl.cloud.tencent.com/document/product/267/30816), установив `TemplateId` на ID шаблона обратного вызова, созданного на шаге 1, для связи шаблона с целевыми `AppId`, `DomainName` и `AppName`.

### 3. Получение результата обнаружения порнографии

Backend CSS отправит результаты обнаружения порнографии на ваш домен в виде HTTP POST запросов. Результаты можно найти в формате JSON в теле запроса. Поле `type` указывает, содержит ли прямой эфир порнографический контент.

> **Примечание:** Система не может достичь 100% точности. Могут быть ложные срабатывания или ложные отрицания. Рекомендуется проверить изображения, подозреваемые в порнографичности, используя поле `type`.

#### LabelResult

Результат попадания метки, сгенерированный моделью категоризации.

| Параметр | Тип | Описание |
| --- | --- | --- |
| Scene | String | Сценарий, определённый моделью, такой как объявление, порнография и вредоносный контент. |
| Suggestion | String | Операция, предлагаемая системой для текущей метки. Пожалуйста, обрабатывайте различные типы нарушений на основе ваших деловых потребностей. Допустимые значения: BlockReviewPass |
| label | String | Обнаруженная метка. |
| SubLabel | String | Подметка. |
| Score | Integer | Оценка достоверности метки. |
| Details | Array of [LabelDetailItem](#labeldetailitem) | Детали попадания подметки. |

Детали обнаружения объекта. Если идентифицированный сценарий — объект, логотип объявления или QR-код, этот параметр возвращает имя метки, значение метки, оценку метки и информацию о местоположении.

| Параметр | Тип | Описание |
| --- | --- | --- |
| Id | Integer | ID идентифицированного объекта. |

| Score | Integer | Оценка обнаруженной метки. Диапазон значений: 0-100. Например, `QrCode 99` указывает на высокую вероятность того, что контент является QR-кодом. |

## Отключение обнаружения порнографии

Вы можете отключить обнаружение порнографии, удалив правило захвата экрана или изменив шаблон захвата экрана. Изменение применяется только к новым потокам. Если вы хотите отключить обнаружение порнографии для текущего потока, необходимо остановить и перезапустить поток.

### 1. Удаление правила захвата экрана

Вызовите [DeleteLiveSnapshotRule](https://intl.cloud.tencent.com/document/product/267/30833), передав `DomainName`, `AppName` и `StreamName`, связанные с ID шаблона захвата экрана, чтобы удалить правило захвата экрана.

### 2. Изменение шаблона захвата экрана

Вызовите [ModifyLiveSnapshotTemplate](https://intl.cloud.tencent.com/document/product/267/30828), установив `PornFlag` на `0`.

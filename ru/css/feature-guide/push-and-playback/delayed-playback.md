# Отложенное воспроизведение

Отложенное воспроизведение — это функция, которая позволяет отложить воспроизведение потоков. Она в основном используется на важных трансляциях для предоставления организаторам времени на обработку непредвиденных ситуаций. Вы можете включить эту функцию через настройку параметров.

## Примечания

- Вы можете включить отложенное воспроизведение тремя способами:
  - Непосредственно через консоль CSS. Подробнее см. в разделе [Конфигурация отложенного воспроизведения](https://www.tencentcloud.com/document/product/267/55264).
  - Вызовите [API отложения воспроизведения](https://intl.cloud.tencent.com/document/product/267/30850).
  - Добавьте параметр `txDelayTime` в конец **URL-адреса передачи**. Подробнее см. в разделе [Конфигурация передачи](https://www.tencentcloud.com/document/product/267/41660#push-configuration).
- Отложенное воспроизведение является платной дополнительной услугой. Чтобы активировать функцию отложенного воспроизведения, используйте консоль, вызовите интерфейс отложенной прямой трансляции или добавьте параметр конфигурации отложенного воспроизведения к домену передачи. После успешной передачи будет создана [Биллинг дополнительных функций](https://www.tencentcloud.com/document/product/267/54973).

> **Примечание:** метод API не рекомендуется, так как вызов API предполагает кэширование конфигурации, что затрудняет оценку времени вступления функции в силу. Рекомендуется включать функцию вторым способом.

## Подготовка

1. Активируйте [CSS](https://console.tencentcloud.com/live).
2. Войдите в консоль CSS, выберите [Управление доменами](https://console.tencentcloud.com/live/domainmanage) и нажмите **Добавить доменное имя** для добавления домена передачи. Подробнее см. в разделе [Добавление доменного имени](https://intl.cloud.tencent.com/document/product/267/35970).

## Конфигурация передачи

1. Войдите в консоль CSS, перейдите в **Инструменты** > [Генератор адресов](https://console.tencentcloud.com/live/addrgenerator/addrgenerator), выберите **Домен передачи** для **Типа домена** и нажмите **Создать адрес**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d5e4db2485f311ef8829525400fdb830.png)

2. Добавьте `txDelayTime` в конец адреса передачи и передавайте потоки через OBS. Подробные инструкции см. в разделе [Передача через OBS](https://intl.cloud.tencent.com/document/product/267/31569).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/60f7384685f211efaee3525400bdab9d.png)

> **Примечание:** установите `txDelayTime` равным количеству секунд, на которое вы хотите отложить воспроизведение. Значение должно быть целым числом и не может превышать 600.

## Отложенное воспроизведение

1. Войдите в консоль CSS, перейдите в **Инструменты** > [Генератор адресов](https://console.tencentcloud.com/live/addrgenerator/addrgenerator), выберите **Домен воспроизведения** для **Типа домена** и нажмите **Создать адрес**.
2. Используйте [VLC](https://intl.cloud.tencent.com/document/product/267/32483), FFmpeg или другие инструменты для воспроизведения. Подробнее см. в разделе [Воспроизведение CSS](https://intl.cloud.tencent.com/document/product/267/31559).

На приведенной выше схеме время задержки, установленное для воспроизведения с помощью параметра `txDelayTime` в адресе передачи, составляет 30 с, а фактическая задержка воспроизведения составляет 34 с, что указывает на то, что функция отложенного воспроизведения вступила в силу.

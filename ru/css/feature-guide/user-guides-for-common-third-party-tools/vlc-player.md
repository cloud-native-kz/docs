# VLC Player

VLC media player — это кроссплатформенный мультимедийный плеер с открытым исходным кодом и фреймворк, который может воспроизводить большинство мультимедийных файлов, а также DVD, аудио CD, VCD и различные потоковые протоколы бесплатно. Он поддерживает операционные системы OS X, Windows, Linux, iOS, Android и Chrome OS и все распространённые форматы файлов для потокового вещания, такие как RTMP, FLV и M3U8. Вы можете перейти на [официальный веб-сайт VLC](http://www.videolan.org/vlc/?spm=a2c4g.11186623.2.13.72e92ad705krbE) для загрузки последней версии.

## Воспроизведение видео в VLC

1. Чтобы получить [адрес потока для воспроизведения видео](https://intl.cloud.tencent.com/document/product/267/31559), выберите требуемый домен воспроизведения в [Domain Management](https://console.tencentcloud.com/live/domainmanage) и сформируйте адрес для воспроизведения в формате, указанном в конфигурации воспроизведения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0358763048c211ef92aa52540075b605.png)

2. Запустите VLC и выберите **File** > **Open Network** (Файл > Открыть сетевой поток).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7a448eb3e10c11eeb419525400ea3514.jpeg)

3. Введите адрес потока прямого эфира в появившееся диалоговое окно.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7a2dcfb4e10c11eeb1eb525400b5f95f.jpeg)

4. Затем нажмите **Open** (Открыть) для подтверждения воспроизведения. Если загрузка потока с адреса воспроизведения прошла успешно, откроется окно воспроизведения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7a32a980e10c11ee9ca3525400bb593a.jpeg)

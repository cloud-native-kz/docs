# Глоссарий

### Service Level Agreement

Service Level Agreement (SLA) — это официальное обязательство, действующее между поставщиком услуг и клиентом.

### Authentication

Authentication в CSS — это процесс, при котором Tencent Cloud расшифровывает ваше подписанное содержимое, проверяет его легитимность и отклоняет запросы, которые не могут быть расшифрованы или содержат недействительное содержимое, чтобы обеспечить получение соответствующих услуг только законными клиентами.

### Stream

Stream в CSS — это группа аудиовизуальных данных, которые непрерывно передаются, например аудиовизуальные данные, собираемые камерой и непрерывно отправляемые другой стороне при видеозвонке WeChat. Push — это отправка таких данных на указанный адрес, который обычно указывает на группу экземпляров CVM.

### Signature

Signature в CSS используется для шифрования содержимого, отправляемого с использованием вашего собственного закрытого ключа, чтобы убедиться, что другие не смогут подделать содержимое.

### SLA

См. [Service Level Agreement](https://intl.cloud.tencent.com/document/product/267/17023).

### StreamName

StreamName — это идентификатор потока, который обычно используется для уникальной идентификации потока вместе с именем домена.

### ISO date format

ISO date format — это формат времени, как указано в ISO 8601. В CSS, если не указано иное, все параметры, связанные со временем, используют время UTC в стандарте ISO 8601 (в формате YYYY-MM-DDThh:mm:ssZ). Например, 2022-10-01T10:00:00Z представляет 18:00:00 1 октября 2022 г. по пекинскому времени (UTC+8).

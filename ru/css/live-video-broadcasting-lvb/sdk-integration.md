# Интеграция SDK

CSS предоставляет LVB как профессиональный, стабильный и быстрый сервис облачной трансляции для типичных случаев использования. Вы можете интегрировать MLVB SDK в приложения, веб-сайты и другие платформы для быстрого включения функции прямой трансляции.

## Приложения

Вы можете интегрировать MLVB SDK с приложениями на клиентах iOS или Android для прямой трансляции и воспроизведения.

- **Прямая трансляция в приложениях**: поддерживает захват камеры или экрана мобильного телефона и последующую отправку контента на платформу CSS с использованием протокола RTMP. Дополнительную информацию см. в разделах [Camera Push](https://www.tencentcloud.com/document/product/1071/38157) и [Screen Recording Push](https://www.tencentcloud.com/document/product/1071/41878).
- **Воспроизведение прямой трансляции в приложениях**: поддерживает протоколы воспроизведения, включая RTMP, HTTP-FLV и HLS. Вы можете интегрировать MLVB SDK с LVB для быстрого создания приложений прямой трансляции. Дополнительную информацию см. в разделе [LVB Pull](https://www.tencentcloud.com/document/product/1071/38159).

> **Примечание:** MLVB SDK использует CSS, IM, TRTC и другие сервисы для маломощной аудиовизуальной коммуникации между несколькими сторонами. Он обеспечивает совместное вещание для взаимодействия между зрителями, а другие зрители, которые не участвуют в совместном вещании, также могут смотреть прямую трансляцию.

## Веб

- **Прямая трансляция в веб**: вы можете использовать стандартный протокол WebRTC для проектирования и мультиплексирования потоков, а также вставлять фрагменты кода на веб-сайты для включения функции прямой трансляции. Подробнее см. в разделе [WebRTC Push](https://intl.cloud.tencent.com/document/product/267/41620).

> **Примечание:** WebRTC push использует аудиокодек Opus. Если вы используете стандартный протокол прямой трансляции (RTMP, HTTP-FLV или HLS) для воспроизведения, серверная часть CSS автоматически преобразует аудиопотоки в формат AAC для обеспечения нормального воспроизведения, что будет сопровождаться взиманием платы за перекодирование аудио. Подробнее см. в разделе [Live Transcoding > Audio Transcoding](https://intl.cloud.tencent.com/document/product/267/39604).

# Настройка списков программ и автоматического вещания

Live Video Caster (LVC) позволяет создавать расписания для настройки планового вещания. Это позволяет публиковать источники входных сигналов или макеты в запланированном порядке.

## Предварительные требования

- Вы выполнили шаги из раздела [Добавление источников входных сигналов](https://www.tencentcloud.com/document/product/267/58532).
- Вы выполнили шаги из раздела [Режиссура и редактирование](https://www.tencentcloud.com/document/product/267/58531).

## Настройка расписания

### Создание расписания

1. В списке [Live Video Caster](https://console.tencentcloud.com/live/caster?rid=5) найдите целевой кастер и нажмите его **ID** или нажмите **Open** справа, чтобы перейти на страницу редактирования кастера.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/92dad179dc8611f08d525254001c06ec.png)

2. Выберите **Schedule**, нажмите **Create schedule** и выполните следующие настройки:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3903f5cf6d2111f087e15254005ef0f7.png)

| Элемент конфигурации | Описание |
| --- | --- |
| Schedule name | Введите имя расписания, которое может состоять из максимум 10 символов. |
| Start time | Выберите время начала расписания, которое должно быть позже времени сохранения расписания. Если плановое вещание включено, в момент начала расписания LVC автоматически включает основной монитор (PGM) для начала трансляции и начисление начинается. |
| End time | Выберите время окончания расписания, которое должно быть позже времени начала и не превышать время истечения кастера. Если плановое вещание включено, в момент окончания расписания LVC автоматически останавливает PGM для остановки трансляции, и начисление заканчивается. |
| Switch audio and video together | Функция переключения аудио и видео вместе включена по умолчанию. Вы можете выбрать отключение этой функции вручную в зависимости от ваших бизнес-требований. Если она включена, аудио/видео переключаются соответственно при переключении программ в расписании. Если она отключена, необходимо выбрать источник входного сигнала в качестве фонового звука расписания. Фоновый звук будет всегда воспроизводиться независимо от переключения программ. Вы можете установить громкость звука в микшере. Изменения фонового звука для текущего расписания вступят в силу в начале следующей программы. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b8dc2dc4201e11f08ca05254005ef0f7.png)

3. Нажмите **Next** и настройте следующие элементы в области **Add program**:

| Элемент конфигурации | Описание |
| --- | --- |
| Program name | Введите имя программы, которое может состоять из максимум 10 символов. |
| Stream time | Время трансляции первой программы должно совпадать с временем начала расписания. Для остальных время трансляции должно быть между временем начала и окончания расписания. |
| Program type | Вы можете выбрать предварительно настроенные источники или макеты. |
| Watermark | Вы можете выбрать максимум пять водяных знаков. |
| Text | Вы можете выбрать максимум пять текстов. |

![](https://staticintl.cloudcachetci.com/cms/backend-cms/daf429586d0e11f084fc525400bf7822.png)

4. Нажмите **Add to schedule** для добавления программы. Вы можете добавить несколько программ в одно расписание.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f76b4f736d0e11f0b9a25254007c27c5.png)

5. После добавления программ нажмите **Save** для завершения создания расписания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/38ed8d176d0f11f081ce52540044a08e.png)

> **Примечание:** Время трансляции, установленное для вставляемой программы, должно быть позже текущего времени.

### Редактирование расписания

1. В списке [Live Video Caster](https://console.tencentcloud.com/live/caster?rid=5) найдите целевой кастер и нажмите его **ID** или нажмите **Open** справа, чтобы перейти на страницу редактирования кастера.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/91c35e61dc8211f0a4f35254007c27c5.png)

2. Выберите успешно созданное расписание и нажмите **Edit** справа, чтобы перейти на страницу редактирования расписания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cbdeb7656d0f11f09f3d52540099c741.png)

> **Примечание:** Истекшие расписания не могут быть отредактированы. Если плановое вещание включено, время начала текущего расписания не может быть изменено, но время окончания может быть изменено. Нажмите **Schedule** для перехода на текущую программу.

3. Нажмите **Edit** справа от Schedule.
  3.1. Время начала трансляции расписания и время окончания трансляции расписания могут быть изменены.
  3.2. Функция переключения аудио и видео вместе включена по умолчанию. Вы можете выбрать отключение этой функции вручную в зависимости от ваших бизнес-требований.
  3.3. Нажмите **Confirm** для сохранения изменений.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1395d34c6d1011f096685254001c06ec.png)

4. Выберите программу, которую вы хотите отредактировать, и нажмите **Edit** справа для редактирования программы.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4c53fc7a6d1011f087e15254005ef0f7.png)

5. Выберите программу, которую вы хотите удалить, и нажмите **Delete** справа для удаления программы.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6df4dd7f6d1011f096685254001c06ec.png)

6. Нажмите **Change** для изменения содержания программы.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/899908d66d1011f09e39525400454e06.png)

7. После редактирования нажмите **Save** для сохранения изменений.

### Удаление расписания

> **Примечание:** Расписания не могут быть восстановлены после удаления. **Соблюдайте предельную осторожность при удалении расписания**. Система автоматически удаляет каждое расписание через семь дней после его времени окончания.

1. На вкладке Schedule выберите целевое расписание и нажмите **Delete** справа.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a8d4b1aa6d1011f0bcac525400e889b2.png)

2. Нажмите **Confirm** для удаления расписания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b9854a4a6d1011f09f3d52540099c741.png)

### Вставка расписания

1. Нажмите ![](https://staticintl.cloudcachetci.com/cms/backend-cms/d74d2a19a46a11eebd03525400c26da5.png) для вставки вновь созданного расписания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d7727df16d1011f084fc525400bf7822.png)

> **Примечание:** Период времени, установленный для вставляемого расписания, не должен быть раньше текущего времени.

2. Повторите предыдущий шаг для вставки дополнительных расписаний или редактирования и удаления существующих расписаний.

#### Закрытие расписания

1. В зависимости от ваших бизнес-требований вы можете нажать ![](https://staticintl.cloudcachetci.com/cms/backend-cms/eacadfa86d1011f087e15254005ef0f7.png) для закрытия расписания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0132ca5a6d1111f087e15254005ef0f7.png)

2. После закрытия вы можете нажать **Schedule** в верхнем правом углу для повторного открытия.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5a74b8846d2111f087e15254005ef0f7.png)

## Настройка планового вещания

### Включение планового вещания

1. После настройки расписаний вы можете включить плановое вещание.
2. Нажмите ![](https://staticintl.cloudcachetci.com/cms/backend-cms/d7506fcfa46a11eebd03525400c26da5.png) для включения планового вещания. Если эта функция включена, LVC автоматически запустит и остановит выход PGM в соответствии с временем начала и окончания расписаний. **Кастер будет работать и начисляться во время включения выхода PGM**. Чтобы остановить текущее и будущее начисление, убедитесь, что плановое вещание и PGM отключены.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/708d1f386d1111f0b9a25254007c27c5.png)

3. Нажмите **Enable** для включения планового вещания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3f43e9702d4011ef97da5254007d9c55.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6b443e256d2111f096685254001c06ec.png)

  - **Schedule start time** (когда начинается трансляция): Если плановое вещание включено, в момент начала расписания LVC автоматически включает PGM для начала трансляции первой программы и начисление начинается.
  - **Program stream time** (исключая первую программу): Если плановое вещание и PGM оба включены, в момент времени трансляции программы LVC автоматически переключается на программу и передает её на PGM.

> **Примечание:** Если PGM или плановое вещание отключены во время запланированной трансляции, программы больше не будут переключаться автоматически.

  - **Schedule end time** (когда заканчивается трансляция): Если плановое вещание включено, в момент окончания расписания LVC автоматически останавливает PGM и начисление заканчивается.

### Отключение планового вещания

> **Примечание:** После отключения планового вещания LVC больше не будет запускать и заканчивать трансляции в соответствии с временем начала и окончания расписаний. Чтобы избежать ненужных начислений, убедитесь, что основной монитор (PGM) также отключен.

1. Для завершения планового вещания нажмите ![](https://staticintl.cloudcachetci.com/cms/backend-cms/7e4ad3e9dcd811f0a6e3525400e889b2.png).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f38231086d1111f084fc525400bf7822.png)

2. Рассмотрите ваши фактические бизнес-требования перед принятием решения об остановке планового вещания; действуйте с осторожностью. Нажмите **Disable** для остановки планового вещания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/112284a1202211f0b21052540099c741.png)

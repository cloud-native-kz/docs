# Введение в область функций

Этот документ представляет основные компоненты консоли Live Video Caster (LVC).

- **Input Source**: позволяет настраивать источники ввода, включая по требованию, прямую трансляцию и источники изображений.
- **PVW**: позволяет просматривать вашу трансляцию в режиме предпросмотра.
- **PGM**: позволяет отслеживать то, что видит аудитория в вашей трансляции.
- **Layout**: позволяет настраивать макет для использования в смешанном потоке из нескольких источников ввода.
- **Soundboard**: позволяет регулировать громкость каждого входного потока в выходном потоке и отделять аудио от видео.
- **Feature**: позволяет устанавливать водяные знаки, текст, переходы и мониторинг потока.
- **Configure Schedule and Enable schedules**: позволяет создавать расписания для настройки запланированного вещания. Это позволяет публиковать источники ввода или макеты запланированным образом.

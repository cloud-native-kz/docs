# Live Video Caster

Live Video Caster (LVC) использует мощные возможности прямой трансляции видео Tencent Cloud для реализации облачного переключения потоков, смешивания потоков и т. д. LVC поддерживает пользовательские макеты экрана, механизмы резервирования, синхронизированное переключение аудио и видео и многое другое. Он позволяет легко составлять контент прямой трансляции без громоздкого оборудования традиционных студийных режиссерских пультов, помогая расширить вашу онлайн-бизнес-присутствие.

## Функции продукта

| Функция | Описание |
| --- | --- |
| Надежные облачные возможности | На основе мощных возможностей прямой трансляции Tencent Cloud LVC предлагает удобные и простые в использовании услуги трансляции, помогая расширить ваши сценарии онлайн-бизнеса. |
| Низкие затраты | Решение LVC не требует оборудования. Использование по требованию и модель оплаты по мере использования значительно снижают инвестиции в оборудование режиссера. |
| Механизм резервирования | Видео и изображения резервирования и другие механизмы экстренной помощи позволяют автоматически переключать поток в случае прерывания потока, обеспечивая надежную прямую трансляцию. |
| Смешивание потоков с пользовательским макетом видео | LVC поддерживает добавление более 20 входов видео и изображений в прямом эфире/по требованию, которые можно смешивать в 4 потока с пользовательским макетом. |
| Многоканальная прямая трансляция | В сценариях электронной коммерции, онлайн-образования, потоковой передачи игр, крупных конференций, представлений и шоу LVC может переключаться между потоками для достижения многоканальной прямой трансляции. |
| Отображение оценок в реальном времени для спортивных событий | С помощью LVC вы можете редактировать текст и изображения онлайн и отображать их в указанное время, как видно на телевидении. |
| Совместная трансляция в нескольких регионах | В отличие от традиционных студийных режиссерских пультов, LVC не использует кабели для передачи сигналов и поэтому может объединять сигналы из нескольких регионов и доставлять смешанный сигнал с низкими затратами. |
| Беспрепятственное переключение между промо-видео, рекламой и демонстрациями | Благодаря возможности LVC беспрепятственно переключаться между потоками, вы можете воспроизводить вводные видео перед началом события и переходить на рекламу или демонстрации в любой момент во время прямой трансляции. |

## Цены

Платежные элементы для LVC включают в себя плату за время выходного потока и плату за передачу потоков третьей стороне. Плата взимается по модели оплаты по мере использования. Связанные документы по счетам см. в разделе [Обзор цен Live Video Caster](https://www.tencentcloud.com/document/product/267/58538#).

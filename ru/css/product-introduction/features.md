# Функции

В этом документе представлен список возможностей CSS. Более подробную информацию о каждой функции можно найти в соответствующих документах.

### Трансляция

| Функция | Описание |
| --- | --- |
| [Поддержка протоколов трансляции](https://intl.cloud.tencent.com/document/product/267/7968) | Поддерживает протоколы RTMP, SRT, WebRTC и RTMP over SRT для трансляции. |
| [Поддержка инструментов трансляции](https://intl.cloud.tencent.com/document/product/267/31558) | Поддерживает MLVB SDK от Tencent Cloud (доступен для iOS, Android и web) или стороннее ПО для потоковой передачи, такое как OBS, XSplit и FMLE. |
| Поддержка устройств трансляции | Поддерживает распространённые сторонние устройства потоковой передачи RTMP, кодеры и цифровые приставки. |

### Воспроизведение

| Функция | Описание |
| --- | --- |
| [Поддержка протоколов воспроизведения](https://intl.cloud.tencent.com/document/product/267/7968) | Поддерживает протоколы воспроизведения RTMP, FLV, HLS и WebRTC. |
| [Поддержка инструментов воспроизведения](https://intl.cloud.tencent.com/document/product/267/31559) | Поддерживает MLVB SDK от Tencent Cloud (доступен для iOS, Android и web) или сторонние проигрыватели FLV, RTMP или HLS. |
| Управление воспроизведением | Воспроизведение исходного потока или потока, перекодированного в реальном времени. |

### Консоль CSS

| Раздел | Описание |
| --- | --- |
| [Обзор](https://intl.cloud.tencent.com/document/product/267/31054) | Отображает данные, такие как использование пакета трафика или данные о полосе пропускания и трафике в реальном времени. |
| [Управление доменами](https://intl.cloud.tencent.com/document/product/267/35970) | Поддерживает добавление, изменение, отключение и удаление доменов трансляции и воспроизведения, а также настройку CNAME, HTTPS-сертификатов и аутентификации трансляции и воспроизведения. |
| [Управление потоками](https://intl.cloud.tencent.com/document/product/267/31068) | Запрос информации об активных потоках, основных и резервных потоках, исторических потоках и заблокированных потоках. |
| [Специализированные услуги](https://intl.cloud.tencent.com/document/product/267/34223) | Запрос или изменение информации конфигурации [Водяного знака трансляции](https://www.tencentcloud.com/document/product/267/31073), [Перекодирования трансляции](https://www.tencentcloud.com/document/product/267/31071), [Записи трансляции](https://www.tencentcloud.com/document/product/267/34223), [Временного сдвига трансляции](https://www.tencentcloud.com/document/product/267/53312), [Снимков экрана трансляции](https://www.tencentcloud.com/document/product/267/31072), [Проверки контента трансляции](https://www.tencentcloud.com/document/product/267/58265), [Адаптивной скорости передачи трансляции](https://www.tencentcloud.com/document/product/267/50271), [Заполнения трансляции](https://www.tencentcloud.com/document/product/267/53400), [Субтитров трансляции](https://www.tencentcloud.com/document/product/267/58144), [Обратного вызова трансляции](https://www.tencentcloud.com/document/product/267/31074) и [Конфигурации шифрования DRM](https://www.tencentcloud.com/document/product/267/48068). |
| [Центр данных](https://intl.cloud.tencent.com/document/product/267/31076) | Поддерживает запрос статистики использования, включая полосу пропускания, трафик, запросы, одновременные подключения, снимки экрана, каналы трансляции и каналы записи. |

### Безопасность

| Функция | Описание |
| --- | --- |
| [Аутентификация трансляции](https://intl.cloud.tencent.com/document/product/267/31059) | Генерирует URL-адреса трансляции с защитой от hotlink-атак (можно использовать свой ключ и указать время истечения). |
| [Аутентификация воспроизведения](https://intl.cloud.tencent.com/document/product/267/31060) | Генерирует URL-адреса воспроизведения с защитой от hotlink-атак с использованием списков разрешений/блокировок IP или поля referer; поддерживает удалённую аутентификацию воспроизведения. |
| [Шифрование DRM](https://www.tencentcloud.com/document/product/267/48068) | Шифрует видео на основе схем DRM, включая Widevine, FairPlay или NormalAES, для предотвращения несанкционированной записи и hotlink-атак. |

### API

| Категория API | Описание |
| --- | --- |
| [API управления доменами](https://intl.cloud.tencent.com/document/product/267/30760) | Добавление, удаление, запрос, включение и отключение доменов трансляции и воспроизведения, а также изменение доменов воспроизведения. |
| [API управления отложенным воспроизведением](https://intl.cloud.tencent.com/document/product/267/30760) | Отложенное воспроизведение, запрос отложенных воспроизведений и возобновление воспроизведения в реальном времени. |
| [API управления записью](https://intl.cloud.tencent.com/document/product/267/30760) | Создание, удаление и завершение задач записи, создание, удаление, запрос и изменение шаблонов записи, а также создание, удаление и запрос правил записи. |
| [API захвата изображений и обнаружения порнографии](https://intl.cloud.tencent.com/document/product/267/30760) | Создание, удаление и запрос правил записи, а также создание, удаление, запрос и изменение шаблонов записи. |
| [API управления водяными знаками](https://intl.cloud.tencent.com/document/product/267/30760) | Добавление, удаление, запрос и изменение водяных знаков, а также создание, удаление и запрос правил водяных знаков. |
| [API обратного вызова трансляции](https://intl.cloud.tencent.com/document/product/267/30760) | Создание, удаление и запрос правил обратного вызова, а также создание, удаление, запрос и изменение шаблонов обратного вызова. |
| [API перетягивания потока](https://intl.cloud.tencent.com/document/product/267/30760) | Добавление, удаление, запрос и изменение конфигураций получения потока. |
| [API управления трансляцией](https://intl.cloud.tencent.com/document/product/267/30760) | Запрос отключённых, активных и исторических потоков, запрос прерываний трансляции и статуса потока, остановка потоков, блокирование потоков и возобновление потоков. |
| [API перекодирования трансляции](https://intl.cloud.tencent.com/document/product/267/30760) | Создание, удаление и запрос правил перекодирования, а также создание, удаление, запрос и изменение шаблонов перекодирования. |
| [API запроса данных выставления счётов](https://intl.cloud.tencent.com/document/product/267/30760) | Запрос полосы пропускания и использования трафика, подлежащих выставлению счётов, запрос данных воспроизведения по региону и ISP, запрос кодов статуса HTTP для запросов воспроизведения, запрос данных воспроизведения на уровне домена в реальном времени и запрос пакетов. |
| [API управления сертификатами](https://intl.cloud.tencent.com/document/product/267/30760) | Добавление, удаление, привязка, отвязка и запрос сертификатов, а также запрос и изменение доменов, привязанных к сертификату. |
| [API управления аутентификацией](https://intl.cloud.tencent.com/document/product/267/30760) | Запрос и изменение ключей аутентификации для воспроизведения и трансляции. |
| [API временного сдвига](https://intl.cloud.tencent.com/document/product/267/30760) | Создание, удаление, запрос и изменение шаблонов временного сдвига, создание, удаление и запрос правил временного сдвига, а также запрос потоков с временным сдвигом. |
| [API смешивания потоков трансляции](https://intl.cloud.tencent.com/document/product/267/30760) | Смешивание потоков и отмена смешивания потоков. |

### SDK

| SDK | Описание |
| --- | --- |
| [MLVB SDK](https://intl.cloud.tencent.com/document/product/1071) | Предоставляет возможности, включая трансляцию потока, базовое улучшение, фильтры, воспроизведение и временный сдвиг. |
| [Tencent Effect SDK](https://www.tencentcloud.com/document/product/1143) | Предоставляет возможности, включая фильтры, улучшение, стикеры и распознавание жестов для различных сценариев съёмки (SDK является решением для обработки видео, разработанным совместно Tencent Cloud, Pitu и YouTu). |

# Обзор

Cloud Streaming Services (CSS) предоставляет пользователям быстрые, стабильные и профессиональные облачные услуги прямой трансляции. Она предлагает Live Video Broadcasting (LVB) для крупномасштабного воспроизведения в режиме реального времени и Live Event Broadcasting (LEB) для прямой трансляции с минимальной задержкой, что делает её высокоподходящей для многих сценариев потокового вещания. CSS также может использоваться вместе с Mobile Live Video Broadcasting SDK (MLVB SDK) для создания комплексного решения для потокового видео- и аудиовещания.

| Продукт | Описание | Варианты использования |

| [LVC](https://www.tencentcloud.com/document/product/267/58539) | Используя мощные возможности прямой трансляции Tencent Cloud, LVC обеспечивает такие функции, как переключение потоков прямой трансляции и многоэкранное смешанное потоковое вещание в облаке. LVC поддерживает пользовательские макеты экрана, резервные потоки и синхронное переключение аудио-видео среди других функций, что исключает необходимость в тяжелом оборудовании традиционной консоли директора. Это делает услуги режиссуры удобными и расширяет сценарии онлайн-бизнеса. | Киберспортивные игры, прямые трансляции электронной коммерции, онлайн-образование, прямые трансляции событий |

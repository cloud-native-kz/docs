# Варианты использования

## Прямая продажа товаров

CSS помогает платформам электронной коммерции расти, позволяя продавцам отображать товары с большей информативностью и помогая потребителям принимать обоснованные решения, что помогает снизить затраты на маркетинг и увеличить продажи.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ca13916edc8d11f0a38652540044a08e.png)

## Потоковая трансляция живых шоу

Объединяя SDK MLVB и возможности управления комнатами IM, CSS предоставляет функции, включая мгновенную трансляцию, цветовые фильтры и подключение микрофона, которые идеально подходят для живых шоу. Простота использования out-of-the-box помогает вам легко повысить популярность живых шоу.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/417b82d8dc8d11f0a38652540044a08e.png)

## Спортивные события

Он обеспечивает возможности прямой трансляции с ультранизкой задержкой для спортивных событий, позволяя результатам матчей быстро достигать пользователей через потоковую трансляцию и позволяя зрителям наслаждаться удовольствием быть в курсе последних событий в режиме реального времени.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/78ed70f1dc8d11f09143525400bf7822.png)

## Дистанционное обучение

Вы можете использовать консоль Tencent Cloud или API для загрузки или скачивания видео для удаленного обучения, которые защищены паролями плеера и динамической защитой от горячих ссылок. Кроме того, ультранизкая задержка LEB предлагает возможность подключения микрофона в режиме реального времени.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7aa786cddc8e11f09143525400bf7822.png)

# Ограничения на использование

Перед использованием CSS необходимо ознакомиться со следующими ограничениями на использование:

| Элемент | Описание |
| --- | --- |
| Доменное имя CSS | Под одной учетной записью можно создать несколько доменных имен для воспроизведения и трансляции. Если регион ускорения доменного имени — это материковая часть Китая или глобальный регион, доменное имя должно иметь регистрацию ICP от MIIT, и информация о регистрации должна быть в настоящее время действительной и доступной. По умолчанию вы можете управлять до 100 доменными именами под одной учетной записью. Если масштаб вашего бизнеса велик, вы можете [отправить заявку](https://console.tencentcloud.com/workorder/category) для увеличения максимального количества доменных имен. Мы рекомендуем вам сохранять длину доменного имени не более 45 символов. Если вам требуется более длинное доменное имя, вам необходимо [отправить заявку](https://console.tencentcloud.com/workorder/category) для получения помощи. Доменное имя, управляемое CSS, может использоваться для трансляции или воспроизведения живых потоков нормально только после разрешения доменного имени. Дополнительную информацию см. в разделе [Настройка CNAME для доменного имени](https://intl.cloud.tencent.com/document/product/267/31057). |
| Прямая трансляция | Сервис CSS не ограничивает битрейт трансляции и поддерживает распространенные разрешения и соответствующие битрейты. Чтобы избежать задержек при трансляции, мы рекомендуем вам сохранять битрейт ниже 4 Мбит/с. Ограничение длины StreamName составляет 255 символов. |
| Воспроизведение в прямом эфире | Поток можно воспроизводить только тогда, когда `StreamName` адреса воспроизведения совпадает с адресом трансляции. Ограничение пропускной способности по умолчанию составляет 20 Гбит/с. Если вам нужно снять это ограничение, вы можете [отправить заявку](https://console.tencentcloud.com/workorder/category) для подачи заявки. Если вам требуется дополнительная поддержка при резком увеличении трафика прямой трансляции, вам необходимо [отправить заявку](https://console.tencentcloud.com/workorder/category) или связаться с вашим представителем Tencent Cloud за три рабочих дня до события для получения помощи. Такие скачки в основном включают следующие два типа: внезапный ежедневный пиковый трафик превышает 200 Гбит/с с скачком в 200% от ежедневного пика; внезапный ежедневный пиковый трафик превышает 500 Гбит/с. |
| Конфигурация шаблона | После успешного привязки шаблона его применение занимает 5–10 минут. |
| Переключение ежедневного режима выставления счетов | Вы можете выбрать выставление счетов по трафику или по пропускной способности для ежедневного выставления счетов. Вы можете переключать режим выставления счетов только один раз в день, и переключение вступит в силу на следующий день. |
| Поддерживаемые протоколы трансляции | Поддерживаются протоколы RTMP, SRT и WebRTC. Подробную информацию см. в разделе [Основы прямой трансляции](https://www.tencentcloud.com/document/product/267/7968#). При использовании протокола WebRTC для трансляции потока каждый домен трансляции имеет ограничение по умолчанию в **1 000** одновременных потоков. |
| Поддерживаемые протоколы воспроизведения | Стандартное видеовещание (LVB) поддерживает протоколы воспроизведения RTMP, FLV и HLS. Дополнительную информацию см. в разделе [Основы прямой трансляции](https://intl.cloud.tencent.com/document/product/267/7968). Трансляция живого события (LEB) поддерживает протокол воспроизведения WebRTC. |
| LVC | Для каждой учетной записи можно создать 5 экземпляров LVC. Вы можете добавить новые экземпляры LVC после удаления старых. Если вам требуется больше LVC, пожалуйста [отправьте заявку](https://console.tencentcloud.com/workorder/category) для подачи заявки. Плейлист входных данных VOD поддерживает до 5 файлов VOD. Трансляция на третьих лиц поддерживает до трех потоков. По умолчанию один поток передается на текущую учетную запись Tencent Cloud CSS, а два других могут передаваться на третьих лиц. |
| Ретрансляция | Можно создать максимум 200 задач ретрансляции. Если ваш бизнес ретрансляции требует большего количества задач, пожалуйста свяжитесь с нами путем [отправки заявки](https://console.tencentcloud.com/workorder/category) или обратитесь к своему менеджеру учетной записи для получения помощи. Функция ретрансляции предоставляет только услуги извлечения и отправки контента. Пожалуйста убедитесь, что контент авторизован и соответствует соответствующим законам и постановлениям относительно распространения контента. Если возникнут вопросы авторского права или нарушения контента, CSS остановит соответствующие услуги и сохранит право на преследование по закону. Для использования функции записи ретрансляции вы можете привязать ее к шаблону записи CSS, что потребует оплату [сборов за запись](https://www.tencentcloud.com/document/product/267/39605). Шаблон записи, привязанный к задаче ретрансляции, поддерживает только шаблоны, которые записывают исходный поток. Чтобы изменить шаблон записи во время выполняемой задачи извлечения, необходимо остановить текущую задачу извлечения на 30 секунд, а затем перезапустить задачу извлечения, чтобы новый шаблон записи вступил в силу. |

# Обзор выставления счетов

> **Примечание:** Вы можете использовать [калькулятор цен CSS](https://buy.tencentcloud.com/price/css/calculator) для оценки ваших сборов за LVB и LEB.

Платежные позиции CSS включают базовые услуги и дополнительные услуги. Вы также можете нести расширенные сборы за услуги, если используете функции CSS, которые зависят от возможностей других продуктов Tencent Cloud.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7fdf82d5df5211eea462525400bb593a.png)

- [Базовые сборы за услуги](#base) — это сборы за использование LVB и LEB, которые взимаются на основе трафика или пиковой пропускной способности.
- [Сборы за дополнительные услуги](#appreciation) — это сборы за использование дополнительных функций, таких как транскодирование, запись, сдвиг времени, снимок экрана, проверка, воспроизведение с задержкой, LVC и трансляция. Эти функции отключены по умолчанию и выставляются в счет только при включении.
- [Расширенные сборы за услуги](#extensions) — это сборы за использование возможностей других продуктов Tencent Cloud и взимаются в соответствии с правилами выставления счетов соответствующих продуктов.

## Базовые сборы за услуги

Базовые сборы за услуги включают сборы за LVB и LEB.

| Платежная позиция | Описание | Режим выставления счетов |
| --- | --- | --- |

Цены различаются для регионов внутри и вне материковой части Китая. | [Ежедневная оплата по мере использования](https://www.tencentcloud.com/document/product/267/39969#bandwidth)[Ежемесячная оплата по мере использования](https://www.tencentcloud.com/document/product/267/39969#bandwidth) |

> **Примечание:** Поскольку LEB использует каналы с ультранизкой задержкой, его сборы за трафик/пропускную способность немного выше, чем у LVB. CSS предоставляет ежемесячный режим выставления счетов для клиентов с высоким спросом. Чтобы перейти на ежемесячное выставление счетов, обратитесь к вашему торговому представителю. Дополнительную информацию о режимах выставления счетов см. в разделе [Изменение режимов выставления счетов](https://intl.cloud.tencent.com/document/product/267/30411).

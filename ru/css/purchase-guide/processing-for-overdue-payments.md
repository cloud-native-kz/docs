# Обработка просроченных платежей

> **Примечание:** Если вы являетесь клиентом партнера Tencent Cloud, правила обращения с ресурсами при просроченных платежах регулируются соглашением между вами и партнером.

## Просроченные платежи

- Счета за текущий день по услугам CSS с оплатой по факту будут созданы утром следующего дня. Счета за услуги CSS ежемесячной выставляемой планом будут созданы в первый день следующего месяца. Подробнее о выставлении счетов см. в разделе [Просмотр счетов](https://intl.cloud.tencent.com/document/product/267/36278).
- Когда обнаруживается, что баланс вашего аккаунта Tencent Cloud недостаточен, что приводит к просроченным платежам, вы получите уведомление о просрочке. У вас есть 24 часа на пополнение счета. Если задолженность погашена в течение 24 часов, аккаунт будет восстановлен (баланс больше или равен 0), и CSS не будет приостановлена. Если задолженность не погашена в течение 24 часов, CSS будет приостановлена, даже если предоплаченный план все еще доступен.
- Когда ваш аккаунт становится просроченным, CSS и LVC не будут немедленно прерывать текущие трансляции или вывод LVC PGM. Они продолжат работать и начисления будут продолжаться обычным образом. Чтобы прекратить использование услуги LVC, необходимо вручную отключить трансляцию или PGM (основной монитор).
- Если услуга VOD приостановлена из-за просроченного платежа, функция записи CSS будет затронута, однако связанные сборы за услуги будут продолжать начисляться обычным образом.

## Приостановка услуги

- Если платеж просрочен и баланс аккаунта не пополнен в требуемый период (баланс аккаунта отрицательный) или содержит контент, нарушающий правила или Условия обслуживания Tencent Cloud, CSS имеет право автоматически конфисковать ресурсы CSS под аккаунтом в течение **60 дней** после даты приостановки услуги.
- После приостановки вашего аккаунта файлы VOD, созданные при записи трансляции, станут недоступными, а скриншоты (включая скриншоты для определения порнографии), хранящиеся в COS, будут обработаны в соответствии с [политикой просроченных аккаунтов](https://intl.cloud.tencent.com/document/product/436/10044) COS.
- Если аккаунт Tencent Cloud приостановлен за содержание контента, нарушающего правила или Условия обслуживания Tencent Cloud, аккаунт не может быть восстановлен.

## Конфискация

- Если какой-либо аккаунт просрочен более чем на **60 дней**, Tencent Cloud будет автоматически конфисковывать ресурсы CSS и регулярно удалять данные и записи CSS (конфигурации, логи услуг, авторизации клиентов и т. д.) под этим аккаунтом.
- Если вы хотите деактивировать услугу CSS и удалить данные CSS под вашим аккаунтом, свяжитесь со своим представителем Tencent Cloud или отправьте [заявку](https://console.tencentcloud.com/workorder/category).

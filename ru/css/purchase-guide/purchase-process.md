# Процесс покупки

## LVB

LVB предлагает [предоплаченные пакеты](https://www.tencentcloud.com/document/product/267/52220) и режим [оплаты по мере использования](https://intl.cloud.tencent.com/document/product/267/2819). Это означает, что вы можете начать использовать LVB, если у вас есть учетная запись Tencent Cloud и активирована CSS.

### Регистрация учетной записи Tencent Cloud

Следуйте инструкциям в разделе [Регистрация](https://intl.cloud.tencent.com/document/product/378/17985), чтобы создать учетную запись Tencent Cloud (вы можете зарегистрироваться используя адрес электронной почты).

### Активация CSS

Перейдите в

консоль CSS

. Прочитайте

условия использования

и установите флажок для согласия с ними. Затем нажмите

**Подать заявку на активацию**

, чтобы начать использовать LVB немедленно. Режим выставления счетов по умолчанию для LVB — ежедневный расчет по трафику. Для получения дополнительной информации о выставлении счетов см.

Выставление счетов LVB

.

![](https://main.qcloudimg.com/raw/dec97eeb50fc64dbcbd2b443fa7b96d4.png)

### Покупка пакетов

Перейдите на [страницу покупки](https://buy.tencentcloud.com/live), чтобы приобрести пакеты трафика и транскодирования.

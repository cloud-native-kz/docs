# Отложенное воспроизведение

Отложенное воспроизведение — это функция, которая задерживает доставку прямых трансляций зрителям. Это часто используется на важных событиях, чтобы дать организаторам время на решение чрезвычайных ситуаций. Отложенное воспроизведение является расширенной функцией CSS. В этом документе описывается, как происходит расчет этой функции.

## Важная информация

В дополнение к комиссиям за расширенные функции использование функции отложенного воспроизведения также будет связано с [расходами на трафик/полосу пропускания](https://www.tencentcloud.com/document/product/267/2819).

## Тарификация

| Тип | Стоимость за единицу |
| --- | --- |
| Цена (USD/единица тарификации) | 0.01515 |

**Соотношение длительности тарификации к фактической длительности:**

| Расширенная функция | Соотношение длительности тарификации (единица тарификации) : фактическая длительность (минуты) |
| --- | --- |
| Отложенное воспроизведение | 0.05 : 1 |

## Сведения о тарификации

- Позиция: расширенная функция
- Режим тарификации: оплата по мере использования.
- Цикл тарификации: ежедневно. Плата, начисленная в каждый день, будет списана с вашего счета на следующий день. Точное время списания комиссии и создания счета см. в вашем платежном выписке.

## Формула расчета

- Длительность тарификации = Фактическая длительность × Соотношение
- Плата = Цена за единицу × Длительность тарификации

## Пример расчета тарификации

- Предположим, вы использовали функцию отложенного воспроизведения в течение 60 минут 15 июня 2023 г. 16 июня будет начислена следующая комиссия за расширенную функцию:
- 0.01515 (USD) × 0.05 (единица тарификации/минута) × 60 (минут) = 0.04545 (USD)

# Интеллектуальное определение порнографического контента

CSS предоставляет функцию определения порнографического контента. Поскольку CSS должен делать снимки экрана в прямых трансляциях для обнаружения порнографического контента, использование функции определения порнографического контента влечет [сборы за определение порнографического контента](https://www.tencentcloud.com/document/product/267/39607#8146365a-6b3b-466f-936b-3d8a468c3b8e) и [сборы за захват экрана](https://intl.cloud.tencent.com/document/product/267/39606). Определение порнографического контента **оплачивается по общему количеству изображений, проверенных на порнографический контент в месяц.**

## Примечания

- Определение порнографического контента отключено по умолчанию и может быть включено в консоли.
- Созданные снимки экрана хранятся в COS и будут влечь сборы за хранение в COS. Дополнительные сведения см. в разделе [Ценообразование COS](https://intl.cloud.tencent.com/pricing/cos).
- Определение порнографического контента — платная функция. **Первые 1000 проверленных снимков экрана в месяц бесплатны. За дополнительные проверленные снимки экрана взимается плата.**
- Списание за аудит экрана происходит во время использования. Убедитесь, что ваш счет не просрочен. Если ваш счет просрочен (отрицательный баланс и просроченный кредитный счет), списание не будет выполнено.

## Ценообразование

| Проверленные снимки экрана | Цена (USD/Тысяча снимков экрана) | Примечания |
| --- | --- | --- |
| ≤ 1000 | 0 | Первые 1000 проверленных снимков экрана в месяц бесплатны |
| ＞ 1000 | 0.2294 | Общее количество снимков экрана будет округлено вверх до ближайшей тысячи |

> **Примечание:** Для снимков экрана, хранящихся в [COS](https://intl.cloud.tencent.com/document/product/436/6222) вне материковой части Китая, будут взиматься дополнительные сборы за исходящий трафик общественной сети. Подробные цены на конкретные платные элементы COS см. в разделе [Ценообразование COS](https://buy.tencentcloud.com/price/cos/overview).

## Обзор выставления счетов

- Платный элемент: Количество снимков экрана, проверленных на порнографический контент.
- Режим выставления счетов: Оплата по мере использования.
- Цикл выставления счетов: Ежемесячный цикл выставления счетов. Счет за текущий месяц будет выставлен с 1 по 5 день следующего месяца. Подробные сведения см. в вашей фактической выписке по счету.
- Правила выставления счетов: Сборы рассчитываются путем умножения общего количества проверленных снимков экрана (с вычитанием первых 1000 бесплатных снимков экрана) в календарном месяце на цену за единицу.

## Пример расчета счета

Предположим, что вы использовали услугу определения порнографического контента с 1 января по 1 февраля 2021 г., а в январе было проверлено в общей сложности 168 000 снимков экрана на порнографический контент. Сборы за определение порнографического контента, которые вам потребуется оплатить 2 февраля 2021 г., будут следующими: Сборы за определение порнографического контента в январе = 0.2294 (USD/Тысяча снимков экрана) × (168 - 1) Тысяча снимков экрана = 38.3098 USD.

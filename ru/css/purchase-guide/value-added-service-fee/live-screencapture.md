# Живая скриншотография

CSS может делать скриншоты прямых трансляций и сохранять их в COS. Использование скриншотографии будет влечь за собой расходы, которые будут выставлены **на основе общего количества сделанных за месяц скриншотов**.

### Примечания

- Скриншотография отключена по умолчанию и может быть включена в консоли или через облачные API.
- Сгенерированные скриншоты хранятся в COS и будут влечь за собой плату за хранение в COS. Подробные сведения см. на странице [Цены COS](https://intl.cloud.tencent.com/pricing/cos).
- Скриншотография — платная функция. **Первые 1000 скриншотов каждого месяца бесплатны. За любые дополнительные скриншоты будет взиматься плата**.
- Списание расходов происходит во время использования. Вы должны убедиться, что ваш счет не просрочен. Если ваш счет просрочен (отрицательный баланс и просроченный кредитный счет), списание расходов не будет выполнено.

### Цены

| Скриншоты | Цена (USD/тысяча скриншотов) | Примечания |
| --- | --- | --- |
| ≤ 1000 | 0 | Первые 1000 скриншотов каждого месяца бесплатны |
| ＞ 1000 | 0.0176 | Общее количество скриншотов будет округлено до ближайшей 1000 |

### Обзор выставления счетов

- Элемент счета: количество скриншотов.
- Способ выставления счетов: почасовая оплата.
- Цикл выставления счетов: ежемесячный цикл выставления счетов. Счет за текущий месяц будет сгенерирован с 1-го по 5-й день следующего месяца. Подробные сведения см. в вашей фактической выписке по счету.
- Правила выставления счетов: плата рассчитывается путем умножения общего количества скриншотов, сделанных в месяц, на единичную цену.

### Пример выставления счетов

Предположим, что вы использовали услугу скриншотографии прямых трансляций с 1 января по 1 февраля 2021 года, и в течение месяца было сгенерировано 168000 скриншотов. В таком случае плата за услугу скриншотографии прямых трансляций, которую вы должны были бы оплатить 2 февраля 2021 года, была бы следующей:
Плата за скриншотографию прямых трансляций за январь = 0.0176 (USD/тысяча скриншотов) × (168 - 1) тысяч скриншотов = 2.9392 USD.

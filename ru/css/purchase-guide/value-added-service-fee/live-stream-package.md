# Пакет Live Stream

CSS предлагает возможности пакета Live Stream, поддерживающие различные протоколы воспроизведения, такие как LL-HLS и DASH. Инкапсулируя конфигурации, система обеспечивает разделение живых потоков на более мелкие фрагменты, обеспечивая значительно более низкую задержку при потоковой передаче.

Правила выставления счетов за использование функции Live Stream Package описаны следующим образом:

## Важно знать

- Для использования функции упаковки необходимо заранее завершить конфигурацию упаковки. Плата за упаковку взимается при извлечении потока. Если извлекаемый поток — это транскодированный поток, будут начислены только [сборы за транскодирование](https://www.tencentcloud.com/document/product/267/39604), плата за упаковку не взимается. Однако если извлекаемый поток — это исходный поток, **плата за упаковку** будет применена.
- Протоколы инкапсуляции потоков включают HLS - CMAF, DASH - ISO, DASH - CMAF, LL-HLS - TS, LL-HLS - CMAF, а также инкапсуляцию DRM при использовании функциональности DRM. Каждый инкапсулированный поток выставляется отдельно.
- Когда длительность задачи Live Stream Package составляет менее одной минуты, выставление счетов будет рассчитано на основе минимальной длительности в одну минуту для Live Stream Package.

## Цены

Выставление счетов основано на длительности извлечения исходного инкапсулированного потока.

| Позиция выставления счета | Цена (USD/мин) |
| --- | --- |
| Live Stream Package | 0.005 |

### Детали выставления счета

- Позиция выставления счета: длительность задач Live Stream Package.
- Режим выставления счета: Pay-as-you-go (оплата по мере использования)
- Цикл выставления счета: ежедневное выставление счета. Сборы, начисленные каждый день, вычитаются в 10:00 утра или при создании ежедневного счета на следующий день.
- Правила выставления счета: плата за задачу Live Stream Package рассчитывается в зависимости от ее длительности, при этом любое время менее одной минуты округляется до целой минуты.

### Примеры выставления счета

Предположим, что вы использовали функцию Live Stream Package 1 июня 2025 г. и извлекли исходные инкапсулированные потоки, включая извлечение исходного потока HLS-CMAF на протяжении 50 минут и исходного потока LL-HLS-TS на протяжении 50 минут. Счет, созданный 2 июня 2025 г., будет выглядеть следующим образом:

0.005（USD/мин）×（ 50+ 50 ）（минут）= 0.5 (USD)

# Ретрансляция

Сервис ретрансляции позволяет быстро получать контент из существующих видео или потоков в прямом эфире и отправлять его на адрес назначения. Этот сервис включает три типа расходов — [стоимость длительности задачи ретрансляции](https://www.tencentcloud.com/document/product/267/41059#relay-task-duration), [стоимость ретрансляции третьим сторонам](https://www.tencentcloud.com/document/product/267/41059#third-party-relaying-bandwidth) и [стоимость расширенных функций](https://www.tencentcloud.com/document/product/267/41059#extended-feature-cost).

## Важная информация

- Сервис ретрансляции **стал платным с 00:00 (UTC+8) 1 июля 2021 года**. Задачи ретрансляции, выполняемые после 1 июля 2021 года, будут облагаться сборами за ретрансляцию независимо от даты создания задач.
- Получение данных из существующего источника будет облагаться сборами за воспроизведение/загрузку. Если вы получаете данные из Tencent Cloud CSS, VOD или COS, применяются правила выставления счетов соответствующего продукта.

## Длительность задачи ретрансляции

### Тарифы

| Статья выставления счета | Цена (USD/мин) |
| --- | --- |
| Длительность задачи ретрансляции | 0.00032 |

### Детали выставления счета

- Статья выставления счета: Длительность задачи ретрансляции
- Способ выставления счета: Оплата по использованию
- Цикл выставления счета: Ежедневное выставление счета. Сборы, начисленные в каждый день, списываются в 10:00 утра или при создании ежедневного счета на следующий день.
- Правила выставления счета: Сборы рассчитываются на основе длительности выполнения задач ретрансляции. Приостановленные или истекшие задачи не влекут за собой сборы. Если вы возобновите приостановленную задачу, будут начислены сборы.

### Примеры выставления счета

Предположим, вы использовали функцию ретрансляции в течение 100 минут 1 октября 2022 года. 2 октября 2022 года вам потребуется оплатить следующий сбор за длительность задачи ретрансляции:
`0.00032 (USD/мин) x 100 (минут) = 0.032 (USD)`

> **Примечание:** Даже если получение данных не удается из-за ошибки источника, задача ретрансляции не будет остановлена до указанного времени окончания, и **сборы будут начисляться в течение этого периода**.

## Пропускная способность третьей стороны при ретрансляции

> **Примечание:** Когда адресом назначения для ретрансляции является адрес отправки CSS, по умолчанию используется передача по внутренней сети Tencent Cloud. В этом сценарии в настоящее время нет дополнительных расходов на ретрансляцию третьей стороне. Когда адресом назначения для ретрансляции является адрес третьей стороны, не связанный с CSS, будет использоваться передача по внешней сети и будут начислены сборы за ретрансляцию третьей стороне.

### Тарифы

Сборы за ретрансляцию третьим сторонам рассчитываются на основе максимальной пропускной способности (Мбит/с), используемой для ретрансляции в каждом периоде выставления счета. Цена зависит от региона, в который ретранслируются ваши потоки. Если вы ретранслируете в несколько регионов в течение периода выставления счета, сборы рассчитываются отдельно на основе максимального использования пропускной способности в каждом регионе.

| --- | --- |

| Гонконг (Китай) | 12.67 |
| Сингапур | 8.04 |
| Франкфурт | 7.1 |
| Сеул | 16.56 |
| Индия | 23.66 |
| Таиланд | 13.01 |

| Вирджиния | 7.1 |
| Джакарта | 17.4 |
| Япония | 13.01 |
| Сан-Паулу | 23.66 |
| Другие регионы | 12.67 |

### Детали выставления счета

- Способ выставления счета: Ежемесячная оплата по использованию
- Цикл выставления счета: Ежемесячное выставление счета. Счет за каждый месяц создается с 1 по 3 день следующего месяца.
- Правила выставления счета: По умолчанию сборы за ретрансляцию третьим сторонам начисляются в режиме оплаты по использованию на основе среднего ежедневного пикового использования пропускной способности (для всех задач ретрансляции третьим сторонам) в каждом месяце.

### Примеры выставления счета

Предположим, вы использовали функцию ретрансляции для отправки потоков третьим сторонам в течение пяти дней в октябре 2022 года. Регион, в который вы ретранслировали, был материковым Китаем, а максимальная пропускная способность, используемая для ретрансляции в эти пять дней, составляла 10 Мбит/с, 80 Мбит/с, 70 Мбит/с, 75 Мбит/с и 60 Мбит/с соответственно.
Ваш режим выставления счета LVB для материкового Китая — это ежедневное выставление счета по пропускной способности. В октябре 31 день.
Ваш сбор за ретрансляцию третьей стороне за октябрь будет следующим:
`(10 Мбит/с + 80 Мбит/с + 70 Мбит/с + 75 Мбит/с + 60 Мбит/с) / 31 день x 12.67 (USD/Мбит/с/месяц) = 120.569 USD`

## Стоимость расширенных функций

### Примечания

В дополнение к стоимости расширенных функций при использовании локального режима, будут также применяться [сборы за сервис ретрансляции](https://intl.cloud.tencent.com/document/product/267/41059).

### Тарифы

| Тип | Цена (USD/единица) |
| --- | --- |
| Цена за единицу | 0.01515 |

**Соотношение длительности выставления счета к фактической длительности:**

| Расширенная функция | Соотношение длительности выставления счета (единица выставления счета) : фактическая длительность (минуты) |
| --- | --- |
| Локальный режим ретрансляции | 0.02 : 1 |

### Детали выставления счета

- Статья: Расширенная функция
- Способ выставления счета: Оплата по использованию
- Цикл выставления счета: Ежедневно. Сбор, начисленный в каждый день, будет списан со вашего счета на следующий день. Для получения фактического времени списания сбора и создания счета см. ваше выписку по счету.

### Формулы расчета

- Длительность выставления счета = Фактическая длительность x Соотношение
- Сбор = Цена за единицу x Длительность выставления счета

### Примеры выставления счета

Предположим, вы включили локальный режим для задачи ретрансляции 23 ноября 2022 года. Длительность задачи составляет 100 минут, а локальный режим также включен на 100 минут. 24 ноября вам потребуется оплатить следующие сборы:

- Сбор за ретрансляцию: `0.00032 (USD/мин) x 100 (минут) = 0.032 (USD)`
- Сбор за расширенную функцию: `0.01515 (USD) x 0.02 (единица выставления счета/мин) × 100 (минут) = 0.0303 (USD)`

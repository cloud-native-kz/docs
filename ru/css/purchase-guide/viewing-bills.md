# Просмотр счетов

Вы можете просмотреть счета CSS и детали платежей в **Billing Center** > **Bills** > [Bill Details](https://console.tencentcloud.com/expense/bill/summary).

Страница Bill Details включает вкладки [Bill by Instance](#bill-by-instance) и [Bill Details](https://www.tencentcloud.com/document/product/267/36278#bill-details):

- Bill by Instance: отображает счета, агрегированные по экземплярам.
- Bill Details: отображает каждое списание без агрегирования.

## Bill by Instance

2. Нажмите **All products** и выберите **Cloud Streaming Services**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/059d139f3e4a11eeacb1525400cea498.png)

#### Поля счета

| Поле | Описание |
| --- | --- |
| Transaction type | Ежедневное урегулирование: сборы списываются ежедневно. Ежемесячное урегулирование: сборы списываются ежемесячно. Новая ежемесячная подписка: пакеты CSS |
| Configuration description | Функции CSS, используемые в текущем месяце, и их использование. Прямая трансляция с транскодированием. Прямая запись. Захват экрана прямой трансляции. Интеллектуальное определение контента для взрослых. Сдвиг времени прямой трансляции. Вытягивание и ретрансляция. Режим прямой трансляции. Модерация прямого видео. Запоздалое воспроизведение |
| Original cost | Общая стоимость использования функции CSS в текущем месяце. |
| Discount multiplier | Множитель скидки за месяц (1 указывает на отсутствие скидок). |
| Total amount after discount | Общая сумма после скидки = Первоначальная стоимость × Множитель скидки |

> **Примечание:** Другие поля присваиваются Tencent Cloud. Для получения дополнительной информации см. [Bills](https://www.tencentcloud.com/document/product/555/57403).

## Bill Details

2. Нажмите **All products** и выберите **Cloud Streaming Services**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/20e01ace3e4a11eeacb1525400cea498.png)

#### Поля счета

| Поле | Описание |
| --- | --- |
| Component type | Функция CSS, используемая в текущем месяце. |
| Component name | Подпункт используемой функции. |
| Component price | Цена компонента в каталоге без скидок. |
| Component usage | Использование компонента. |
| Discount multiplier | Множитель скидки за месяц (1 указывает на отсутствие скидок). Пользователям с ежемесячным выставлением счетов предоставляются скидки. Для получения дополнительной информации обратитесь в отдел продаж. |
| Usage duration | Общее использование компонента в текущем месяце. |
| Total amount after discount | Общая сумма после скидки = Первоначальная стоимость × Множитель скидки. Первоначальная стоимость = Цена компонента × Длительность использования. |

> **Примечание:** Другие поля присваиваются Tencent Cloud. Для получения дополнительной информации см. [Bills](https://www.tencentcloud.com/document/product/555/57403).

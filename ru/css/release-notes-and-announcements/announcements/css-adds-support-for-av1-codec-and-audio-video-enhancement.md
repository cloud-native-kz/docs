# CSS добавляет поддержку кодека AV1 и улучшения аудио/видео

CSS начал предоставлять функции кодирования AV1 и улучшения аудио/видео за плату с **00:00 1 июня 2022 года**.

Использование этих функций может привести к взиманию стандартных сборов за транскодирование, сборов за транскодирование кодека с максимальной скоростью и сборов за улучшение аудио/видео.

## Новые функции

### Кодек AV1

AV1 — это открытый, свободный от авторских отчислений видеокодек, разработанный для передачи видео в интернет Альянсом за открытые медиа (AOMedia).

- Он может передавать видео 1080p (1920 x 1080) со скоростью 1–2 Мбит/с.
- Он имеет 40% улучшенную эффективность сжатия по сравнению с H.264, что означает, что вы можете сэкономить 40% затрат на CDN без ущерба качеству воспроизведения.
- Он свободен от авторских отчислений и поддерживается большим количеством браузеров, чем H.265 (AV1 поддерживается 75% браузеров на рынке, включая Chrome).
- Поскольку технологии AV1 и WebRTC берут своё начало в Google, эти две технологии хорошо работают вместе. Вы можете использовать продукт [LEB](https://intl.cloud.tencent.com/document/product/267/42141) Tencent Cloud вместе с функцией кодирования AV1 для обеспечения низкозадержного и высокачественного потокового вещания при более низких затратах на CDN.

### Улучшение аудио/видео

На основе наших модулей восстановления и улучшения изображения и алгоритмов искусственного интеллекта мы предлагаем возможности улучшения качества видео, включая восстановление сверхвысокого разрешения, интеллектуальную интерполяцию кадров, улучшение HDR, удаление шума изображения, восстановление контуров и другие функции улучшения аудио/видео.

## Как использовать функции

### Кодек AV1

Чтобы использовать кодек AV1, просто установите кодек для вашего шаблона транскодирования на AV1. Подробные сведения см. в разделе [Live Transcoding](https://intl.cloud.tencent.com/document/product/267/31071).

### Улучшение аудио/видео

Пожалуйста, свяжитесь с вашим менеджером по продажам или обратитесь в службу поддержки клиентов, чтобы включить функцию улучшения аудио/видео.

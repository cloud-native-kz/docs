# CSS для корректировки цен на трансляцию для третьих сторон

CSS скорректирует цены на трансляцию для третьих сторон начиная с **00:00 (UTC+8) 1 августа 2022 г.** Цены на трансляцию для третьих сторон за пределами материкового Китая и Гонконга будут варьироваться в зависимости от региона. Для материкового Китая и Гонконга цена останется неизменной.

Это изменение повлияет на выставление счетов CSS. Статья платежей sp_live_dcrelay изменится с dcrelay_bandwidth_monthly на dcrelay_bandwidth_monthly_regional.

## Примечания

- Новые цены вступят в силу начиная с 00:00 (UTC+8) 1 августа 2022 г. Они будут применены к вашему счету за август, который будет выставлен с 1 по 3 сентября.
- Если вы подписали контракт с Tencent Cloud, пожалуйста, свяжитесь с вашим менеджером по продажам или [отправьте заявку](https://console.tencentcloud.com/workorder/category), чтобы узнать о ценах, которые будут применены к вашему аккаунту в будущем.

## Выставление счетов за трансляцию для третьих сторон

Вы можете использовать услугу трансляции CSS для трансляции потоков видео на адреса третьих сторон. Сборы за трансляцию для третьих сторон взимаются в соответствии с максимальной пропускной способностью (Мбит/с), используемой для трансляции в каждом периоде выставления счетов. Цена варьируется в зависимости от региона, в который транслируются потоки.

- Область применения: сборы взимаются только в том случае, если вы транслируете на адрес, не являющийся URL-адресом push CSS текущего аккаунта (аккаунта, который создал задачу трансляции).
- Режим выставления счетов: ежемесячная оплата по факту использования
- Цикл выставления счетов: ежемесячное выставление счетов. Ваш счет за каждый месяц выставляется со 1 по 3 число следующего месяца.
- Правила выставления счетов: по умолчанию сборы за трансляцию для третьих сторон взимаются в режиме оплаты по факту использования в соответствии с вашим средним суточным пиковым использованием пропускной способности (для всех задач трансляции для третьих сторон) в каждом месяце. Если для услуги LVB на вашем аккаунте используется другой режим выставления счетов, он будет применен к трансляции для третьих сторон.

## Цены после изменения

Если вы используете услугу трансляции в нескольких регионах в течение одного периода выставления счетов, она будет оплачиваться отдельно в соответствии с использованием пиковой пропускной способности в каждом регионе.

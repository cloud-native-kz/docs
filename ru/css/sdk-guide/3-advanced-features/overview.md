## Воспроизведение видео AV1

AV1 — это открытый формат видеосжатия, свободный от лицензионных платежей. По сравнению с предшественником H.265 (HEVC), он может дополнительно снизить битрейт более чем на 30% при сохранении того же уровня качества изображения. Это означает, что он может обеспечить более высокое качество изображения при одной и той же пропускной способности. В настоящее время CSS поддерживает кодирование AV1. Однако для воспроизведения видео AV1 ваш проигрыватель должен поддерживать декодирование формата AV1.

## TMIO SDK

TMIO SDK настраивает, инкапсулирует и оптимизирует протоколы потоковой передачи мультимедиа, такие как SRT и QUIC. Он обеспечивает безопасность при передаче вверх и реализует передачу с низкой задержкой с высокой степенью предотвращения потерь пакетов, оптимизацией многоканальной передачи и механизмом таймаута повторной передачи (RTO). Он обещает широкое применение в сценариях, требующих высокой стабильности источника данных и дальней передачи.

### Описание функций

- TMIO SDK подходит для дальней передачи и индустрии радио и телевидения.
- TMIO SDK поддерживает основные платформы, включая Android, iOS, Windows, macOS и Linux.

### Способ интеграции

Вы можете интегрировать SDK в соответствии с [Шагами интеграции](https://www.tencentcloud.com/document/product/267/51158).

## libLebConnection SDK

libLebConnection SDK предоставляет расширенные возможности передачи на основе собственного WebRTC. Он позволяет подключить ваш существующий проигрыватель к LEB всего с несколькими простыми изменениями. На основе потока, совместимого с LVB, и облачных возможностей обработки медиа, он может реализовать трансляцию в реальном времени с низкой задержкой даже при высокой одновременности, помогая вам плавно перейти со стандартной потоковой трансляции LVB на потоковую трансляцию LEB с низкой задержкой. Для больших комнат в современных сценариях реального взаимодействия (RTC) это также может помочь вам быстро реализовать ретрансляцию в реальном времени с низкими затратами и низкой задержкой.

### Описание функций

- libLebConnection SDK может воспроизводить потоки аудио/видео с низкой задержкой даже при плохих сетевых условиях.
- Он может воспроизводить видео H.264, H.265 и AV1 с B-кадрами и выводить их как необработанные данные видеокадра, такие как файлы Annex B и OBU для видеовходов H.264/H.265 и AV1 соответственно.
- Он может воспроизводить аудио AAC и Opus и выводить их как необработанные данные аудиокадра.
- Поддерживает Android, iOS, Windows, Linux и macOS.

### Способ интеграции

Вы можете интегрировать libLebConnection SDK в соответствии с [Интеграцией libLebConnection SDK в проигрыватель](https://www.tencentcloud.com/document/product/267/51159).

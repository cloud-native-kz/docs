# Объявление об оптимизации метода выставления счетов CSS согласно политике

Для предоставления более гибких и подходящих режимов выставления счетов мы планируем оптимизировать политику выставления счетов для [upstream push](https://www.tencentcloud.com/document/product/267/2818) и [relay to third parties](https://www.tencentcloud.com/document/product/267/41059) начиная с 00:00 23 декабря 2024 года, что не повлияет на ваш текущий метод выставления счетов и размер платежей. Подробные корректировки приведены ниже:

### **Корректировка политики выставления счетов для Upstream Push**

| До корректировки | Метод выставления счетов для upstream push следует методу выставления счетов для Live Video Broadcasting (LVB) downstream playback и не поддерживает отдельную настройку метода выставления счетов для upstream push. Для получения дополнительной информации. |
| --- | --- |
| После корректировки | Ваш текущий метод выставления счетов для upstream push останется без изменений, поддерживая конфигурацию по требованию, и больше не будет следовать методу выставления счетов для LVB downstream playback. |

### Корректировка политики выставления счетов для Relay to Third Parties

| До корректировки | Relay to third parties использует режим оплаты по мере использования на основе средней дневной пиковой ширины полосы пропускания по умолчанию. Если LVB под вашей учетной записью использует другие методы ежемесячного выставления счетов по ширине полосы пропускания, метод выставления счетов по ширине полосы пропускания для relay to third parties будет следовать методу выставления счетов LVB. Для получения дополнительной информации. |
| --- | --- |
| После корректировки | Ваш текущий метод выставления счетов для relay to third parties останется без изменений, поддерживая конфигурацию по требованию для других методов выставления счетов по ширине полосы пропускания, и больше не будет следовать методу выставления счетов LVB downstream playback bandwidth. |

Если у вас есть какие-либо вопросы, пожалуйста, [свяжитесь с нами](https://console.tencentcloud.com/workorder/category). Мы искренне благодарны вам за доверие и поддержку Tencent Cloud CSS.

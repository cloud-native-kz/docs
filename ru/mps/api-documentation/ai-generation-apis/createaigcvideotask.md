# CreateAigcVideoTask

## 1. Описание API

Доменное имя для запроса API: mps.intl.tencentcloudapi.com.

Этот API используется для создания задач генерации видео на основе AI.

Максимум 10 запросов могут быть инициированы в секунду для этого API.

Рекомендуем использовать API Explorer

Попробовать

API Explorer предоставляет диапазон возможностей, включая онлайн-вызов, аутентификацию подписи, генерацию кода SDK и быстрый поиск API. Это позволяет вам просматривать запрос, ответ и автоматически генерируемые примеры.

## 2. Входные параметры

Следующий список параметров запроса содержит только параметры запроса API и некоторые общие параметры. Полный список общих параметров см. в разделе [Общие параметры запроса](https://www.tencentcloud.com/document/api/1041/33628).

| Имя параметра | Обязательно | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/1041/33628). Значение, используемое для этого API: CreateAigcVideoTask. |
| Version | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/1041/33628). Значение, используемое для этого API: 2019-06-12. |
| Region | Нет | String | [Общие параметры](https://www.tencentcloud.com/document/api/1041/33628). Этот параметр не требуется для этого API. |
| ModelName | Нет | String | Имя модели. Поддерживаемые модели: Hunyuan, Hailuo, Kling, Vidu, OS, GV. |
| ModelVersion | Нет | String | Номер конкретной версии модели. По умолчанию система использует поддерживаемую стабильную версию модели. 1. Hailuo: [02 и 2.3]. 2. Kling: [2.0, 2.1, 2.5, O1 и 2.6]. 3. Vidu: [q2, q2-pro и q2-turbo]. 4. GV: [3.1]. 5. OS: [2.0]. |
| SceneType | Нет | String | Сценарий для генерируемого видео. Примечание: не все модели поддерживают сценарии. 1. Kling поддерживает управление движением (motion_control). 2. Mingmou поддерживает преобразование пейзажа в портрет (land2port). 3. Vidu поддерживает шаблоны специальных эффектов (template_effect). |
| Prompt | Нет | String | Описание генерируемого видео. (Примечание: поддерживается максимум 2000 символов.) Этот параметр требуется, когда не указано эталонное изображение. |
| NegativePrompt | Нет | String | Содержимое, которое вы хотите предотвратить генерацию моделью. Примечание: не все модели это поддерживают. Например: освещение сверху и яркие цвета. Люди и животные. Несколько транспортных средств и ветер. |
| EnhancePrompt | Нет | Boolean | Значение по умолчанию — False, что означает, что модель строго следует инструкциям. Для лучших результатов с более тонкими подсказками установите этот параметр в True, чтобы автоматически оптимизировать входную подсказку и улучшить качество генерации. |
| ImageUrl | Нет | String | URL изображения для генерации видео. URL должен быть доступен из открытой сети. Примечание: 1. Рекомендуемый размер изображения не более 10 МБ. Разные модели имеют разные ограничения по размеру. 2. Поддерживаемые форматы изображений: JPEG и PNG. 3. При использовании модели OS размерность входного изображения должна быть 1280x720 или 720x1280. |
| LastImageUrl | Нет | String | Модель будет генерировать видео, используя изображение этого параметра в качестве завершающего кадра. Модели, поддерживающие этот параметр: 1. GV. Если указано изображение завершающего кадра, ImageUrl требуется в качестве начального кадра. 2. Kling. Версия 2.1 поддерживает начальные и завершающие кадры с разрешением 1080P. 3. Vidu. q2-pro и q2-turbo поддерживают начальные и завершающие кадры. Примечание: 1. Рекомендуемый размер изображения не более 10 МБ. Разные модели имеют разные ограничения. 2. Поддерживаемые форматы изображений: JPEG и PNG. |
| ImageInfos.N | Нет | Array of [AigcVideoReferenceImageInfo](https://www.tencentcloud.com/document/api/1041/33690#AigcVideoReferenceImageInfo) | Список из максимум 3 ресурсных изображений, используемых для описания изображений, которые модель должна использовать для генерации видео. Модель, поддерживающая несколько изображений: 1. GV. Если указано несколько изображений, ImageUrl и LastImageUrl недоступны. 2. Vidu поддерживает генерацию видео с несколькими эталонными изображениями. Модель q2 принимает 1-7 изображений. ReferenceType в ImageInfos может использоваться для указания ID субъекта для входных данных. Примечание: 1. Размер изображения не может превышать 10 МБ. 2. Поддерживаемые форматы изображений: JPEG и PNG. |
| VideoInfos.N | Нет | Array of [AigcVideoReferenceVideoInfo](https://www.tencentcloud.com/document/api/1041/33690#AigcVideoReferenceVideoInfo) |  |
| Duration | Нет | Integer | Длительность генерируемого видео. Примечание: 1. Kling поддерживает 5 и 10 секунд. Значение по умолчанию: 5 секунд. 2. Режим std Hailuo поддерживает 6 и 10 секунд, другие режимы поддерживают 6 секунд. Значение по умолчанию: 6 секунд. 3. Vidu поддерживает 1-10 секунд. 4. GV поддерживает 8 секунд. Значение по умолчанию: 8 секунд. 5. OS поддерживает 4, 8 и 12 секунд. Значение по умолчанию: 8 секунд. |
| ExtraParameters | Нет | [AigcVideoExtraParam](https://www.tencentcloud.com/document/api/1041/33690#AigcVideoExtraParam) | Требуемые дополнительные параметры. |
| StoreCosParam | Нет | [AigcStoreCosParam](https://www.tencentcloud.com/document/api/1041/33690#AigcStoreCosParam) | Информация бакета COS для результата файла. Примечание: COS требуется, и роль MPS_QcsRole должна быть создана и авторизована. |
| AdditionalParameters | Нет | String | Специальные параметры сценария, требуемые моделью, отформатированные как сериализованная строка JSON. Пример: {"camera_control":{"type":"simple"}} |
| Operator | Нет | String | Имя оператора API. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| TaskId | String | ID успешно созданной задачи. Ход выполнения задачи и результаты генерации можно получить, вызвав API запроса. |
| RequestId | String | Уникальный ID запроса, генерируемый сервером, будет возвращен для каждого запроса (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId требуется для локализации проблемы. |

### Пример 1 Пример запроса

#### Пример входных данных

```
POST / HTTP/1.1
Host: mps.intl.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: CreateAigcVideoTask

{
    "ModelName": "GV",
    "Prompt": "move the picture",
    "NegativePrompt": "Top lighting,bright colors",
    "EnhancePrompt": false,
    "ImageUrl": "https://1500039689.vod-**.com/6cd2c44bvodcq1500039689/179cafb25145403699605999621/**oxE6jcA.png",

    "Duration": 8,
    "ExtraParameters": {
        "Resolution": "720P",
        "AspectRatio": "16:9"
    },
    "StoreCosParam": {
        "CosBucketName": "aigc-***-video-1**58",
```

# DescribeTranscodeTemplates

## 1. Описание API

Доменное имя для запроса API: mps.intl.tencentcloudapi.com.

Этот API используется для получения списка шаблонов транскодирования на основе уникального ID шаблона. Результат возвращает все подходящие пользовательские и [предустановленные шаблоны транскодирования](https://intl.cloud.tencent.com/document/product/266/33476?from_cn_redirect=1#.E9.A2.84.E7.BD.AE.E8.BD.AC.E7.A0.81.E6.A8.A1.E6.9D.BF).

Максимум 100 запросов можно инициировать в секунду для этого API.

Рекомендуем использовать API Explorer

Поддерживаются форматы для сценариев трансляции.LQ_material: восстановление видео низкого разрешения/старого видео. Улучшает общее разрешение и решает проблемы старых видео, такие как низкое разрешение, размытие, искажение, царапины и цветовая температура из-за их возраста.lecture: живые шоу, электронная коммерция, конференции и лекции. Улучшает эффект отображения лица и выполняет специфическую оптимизацию, включая улучшение области лица, шумоподавление и удаление артефактов, для сценариев с участием человека, таких как живые шоу, электронная коммерция, конференции и лекции. |

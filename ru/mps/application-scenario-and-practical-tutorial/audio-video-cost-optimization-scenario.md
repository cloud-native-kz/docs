# Сценарий оптимизации затрат на аудио/видео

## Обзор

Развитие живой торговли, онлайн-образования, удаленных конференций, коротких видео, коротких драм и тенденций искусственного интеллекта приводит к увеличению объема видеоконтента. Поставщики услуг должны балансировать между эффективностью и стоимостью. Оптимизация затрат является ключевой для устойчивости и конкурентоспособности. Тencent Cloud Media Processing помогает снизить затраты на обработку и хранение.

## Требования

- Резкий рост видеоконтента привел к увеличению затрат на хранение и передачу данных, что требует технических решений для снижения расходов.
- Сбалансированный выбор количества активированных функций имеет решающее значение при обработке аудио и видео, чтобы избежать минимальной оптимизации или высоких затрат.
- Дифференцированная обработка видео на основе конкретных проблем может максимизировать экономическую выгоду.

## Наши решения

### Ведущие возможности кодирования и сжатия

Поддерживает стандарты кодирования, такие как VP8, VP9, H.264, H.265, AV1, AVS3, H.266 и другие, а также поддерживает кодирование в реальном времени для 4K и 8K, обеспечивая сверхвысокое качество изображения и плавное воспроизведение видео. При сохранении субъективного качества видео экономит 50% затрат на полосу пропускания. В течение последних трех лет последовательно занимает первое место в конкурсе MSU Video Coding Competition; также достигла лучших общих показателей в оценке кодирования Streaming Learning Center. Вы можете оценить эффекты трансляции на нашей [странице онлайн-демонстрации](https://mps.live/demo/encoding/video).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ce9d88d3046c11f0aa8e525400bf7822.png)

В [консоли Media Processing](https://console.tencentcloud.com/mps/templates/avs) доступно множество шаблонов трансляции аудио и видео для вашего удобства. Каждый шаблон предварительно настроен с набором параметров трансляции, что позволяет быстро выбрать соответствующий шаблон трансляции в соответствии с вашим конкретным деловым сценарием.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5624a608046d11f08bc5525400454e06.png)

Вы также можете создавать новые шаблоны для настройки различных параметров. Подробную информацию о создании шаблонов и настройке параметров см. в документации [Шаблоны трансляции аудио и видео](https://www.tencentcloud.com/document/product/1041/48784). Информацию о стоимости трансляции аудио и видео см. в разделе [Инструкции по выставлению счетов](https://www.tencentcloud.com/document/product/1041/49204).

### Решение обработки по требованию

Услуги аудио и видео Tencent Cloud предоставляют высококачественную трансляцию и решение по требованию "Проверка качества + Трансляция и улучшение". Для оптимизации затрат видео можно проверить перед трансляцией, что позволит выбрать подходящий шаблон на основе результатов проверки.

| Функция | Описание | Как использовать |
| --- | --- | --- |
| Проверка качества | Проверка качества медиа поддерживает:Проверка качества формата: выявляет проблемы с форматом, такие как проблемы DTS, PTS, изменения разрешения, изменения частоты дискретизации, потеря кадров и дублирование кадров.Проверка качества видео и аудиоконтента: JitterResults, BlurResults, AbnormalLightingResults, CrashScreenResults, BlackWhiteEdgeResults, NoiseResults, MosaicResults и другие.Оценка качества без справочной информации | См. документацию [Интеграция проверки качества медиа](https://www.tencentcloud.com/document/product/1041/67727). |
| Обычная трансляция/TSC трансляция/Адаптивная потоковая передача | В сценариях обработки по требованию медиа-активы сначала проверяются на качество. На основе этой проверки видео с проблемами формата или высокими битрейтами транслируются, а те, у которых нет проблем, пропускают этот шаг, снижая затраты на трансляцию. | См. документацию [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33627). |

### Оценка после трансляции

После завершения процесса трансляции пользователи также могут количественно оценить результаты трансляции с помощью функции оценки трансляции. Tencent Cloud Media Processing предлагает комплексные возможности оценки трансляции, поддерживая оценку VMAF, PSNR, SSIM и VMAF-NEG для видео из различных источников и в различных форматах.

|  | Оценка сравнения BD-Rate | Выбирает шаблон Media Processing Service и оценивает различия в качестве видеотрансляции различных шаблонов при различных битрейтах.Поддерживает методы оценки, включая VMAF, PSNR, SSIM и VMAF-NEG.Поддерживает настраиваемый выбор периода времени или диапазона кадров для оценки.Поддерживает сравнение оценок оценивания при указанных битрейтах или сравнение битрейтов при указанном CRF (оценка качества видео). | См. документацию [Руководство пользователя](https://www.tencentcloud.com/document/product/1041/59862). |

# Сценарий улучшения качества изображения

## Обзор

Обработка медиа Tencent Cloud повышает качество видео при потоковой передаче по требованию и в режиме реального времени, увеличивая удержание пользователей и коммерческое преобразование. Она предлагает технические решения для игр, шоу талантов, образования и кинематографии, адаптируясь к различным потребностям контента благодаря своему технологическому опыту и опыту в отрасли.

## Ключевые требования

- Повышайте качество видео путем исправления визуальных дефектов, предоставляя превосходный просмотр.
- Способность к интеллектуальной обработке по требованию, снижение затрат и повышение эффективности.

## Наши решения

### Богатые возможности улучшения аудио и видео

Обработка медиа Tencent Cloud предоставляет различные варианты улучшения аудио и видео. Пользователи могут использовать одну или несколько функций для лучших результатов.

| Тип улучшения | Функции | Описание |
| --- | --- | --- |
| Улучшение видео | Супер разрешение | Супер разрешение может идентифицировать содержимое и контуры видео, реконструировать детали и локальные особенности видео в высокой четкости, преобразуя видео низкого разрешения в видео высокого разрешения, подходит для сценариев, таких как восстановление старых фильмов. |
|  | Улучшение при низком освещении | Из-за условий окружающей среды и ограничений аппаратного обеспечения камеры некоторые сцены могут страдать от недостатка яркости и контрастности, что приводит к темным изображениям или отсутствию деталей. Активация улучшения при низком освещении значительно улучшает детали и контрастность в темных областях, повышая субъективное качество для человеческого глаза. |
|  | HDR | Поддерживает HDR10 и HLG, предлагая более широкую цветовую гамму и больше деталей цвета, обеспечивая видеоконтент более высокого качества. |
|  | Комплексное улучшение | Благодаря комплексным аналитическим возможностям ИИ, он автоматически балансирует содержимое текстуры на изображении, улучшая ключевые детали при удалении артефактов сжатия и зубчатых краев, тем самым улучшая общее субъективное восприятие изображения. |
|  | Улучшение цвета | Улучшение цвета делает изображение более близким к реальным цветам и улучшает их в некоторой степени, чтобы соответствовать предпочтениям человеческого глаза. |
|  | Улучшение деталей | Улучшение деталей сосредоточено на деталях видео, которым требуется внимание (например, трава на спортивном поле), делая содержимое изображения более ясным и богатым. |
|  | Улучшение лица | Улучшайте области видео, на которые человеческая зрительная система особенно обращает внимание, такие как лица, делая детали в этих областях более ясными и улучшая субъективное восприятие. |
|  | Удаление царапин | Удаление царапин может восстанавливать царапины и пятна снега на видео, восстанавливая поврежденное содержимое. |
|  | Удаление артефактов | Из-за множественного сжатия видео при трансcodировании или нескольких процессов трансcodирования вводятся блочные эффекты, эффекты звенения, утечка цветности и москитный шум, вызывающие искажения, влияющие на визуальный эффект. Эффективное удаление артефактов сжатия восстанавливает искажения, введенные кодированием. |
|  | Снижение шума видео | Случайный шум может быть введен во время съемки фильма из-за камеры и окружающей среды. Эта служба предлагает подавление шума, устраняя случайный шум в изображении без потери деталей. |
| Улучшение аудио | Снижение шума аудио | Удаляет шум устройства, шум окружающей среды и т. д., подходит для сценариев, таких как запись классов и постпроизводство наружной съемки. |
|  | Разделение аудио | Разделяет человеческие голоса от фоновых звуков или голоса певца от аккомпанемента в аудиовизуальных файлах, создавая независимые аудиоматериалы для постпроизводственной художественной обработки. |
|  | Выравнивание громкости | Нормализация громкости: сохраняет последовательный общий уровень громкости, делая звук воспроизведения похожим по громкости, избегая проблем быть слишком громким или слишком тихим, и обеспечивает лучший слуховой опыт. Выравнивание уровня: сглаживает слишком громкие звуковые сегменты, избегая внезапных изменений громкости, и обеспечивает более стабильный слуховой опыт. |
|  | Улучшение аудио | Удаление шума: снижает нежелательный шум или помехи в аудио, улучшая качество и четкость аудио. Подавление свистящих звуков (подавление сибилянтов): Сибилянты — это резкие, пронзительные звуки в аудио, часто производимые, когда источник звука находится близко к микрофону. Подавление сибилянтов направлено на уменьшение или устранение этого неестественного звука, тем самым улучшая качество аудио. |

### Решения для обработки по требованию

Аудио- и видеослужбы Tencent Cloud предлагают улучшения и по требованию "Проверка качества + Трансcodирование и улучшение". Это позволяет оптимизировать затраты путем проверки качества видео перед улучшением. На основе результатов проверки выбирается подходящий шаблон улучшения. Улучшение измеряется путем сравнения оценок видео до и после обработки.

| Функция | Описание | Как использовать |
| --- | --- | --- |
| Проверка качества | Проверка качества включает: Проверка качества формата: обнаруживает проблемы с форматом, такие как проблемы DTS, PTS, изменения разрешения, изменения частоты дискретизации, потеря кадров и дублирование кадров. Поддерживает обнаружение качества изображения видео, с конкретными элементами проверки следующим образом: JitterResults, BlurResults, AbnormalLightingResults, CrashScreenResults, BlackWhiteEdgeResults, NoiseResults, MosaicResults и т. д. Оценка качества без эталона | Обратитесь к документации [Media Quality Inspection Integration](https://www.tencentcloud.com/document/product/1041/67727). |
| Улучшение аудио и видео | В сценариях обработки по требованию внимание сосредоточено на устранении проблем с аудио и видео, выявленных при проверке качества, путем целевых вмешательств. Улучшение при низком освещении Комплексное улучшение Улучшение лица Снижение шума аудио Видео без проблем не требуют улучшения, что может помочь снизить затраты. | Обратитесь к документации [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33627). |

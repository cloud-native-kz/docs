# Сценарии онлайн-образования

## Обзор

Индустрия онлайн-образования трансформируется благодаря технологиям. С растущей базой пользователей и новыми технологиями основное внимание уделяется поддержанию качества обучения при ограниченной полосе пропускания, снижению затрат и интеграции новых технологий. Этот документ описывает обработку мультимедиа Tencent Cloud для образования, помогая пользователям применять технологические решения для экономии затрат и повышения эффективности.

## Сценарии

### Четкая и стабильная трансляция в прямом эфире

- Описание требования: четкое визуальное представление и стабильное качество звука имеют решающее значение для того, чтобы студенты могли видеть содержимое доски и понимать информацию в учебных материалах без помех.
- Решение: используя возможности видеокодирования на терминале и шумоподавления аудио, мы улучшаем качество аудио и видео для учителей при проведении онлайн-уроков, обеспечивая четкое изображение и стабильный звук. Благодаря возможностям сверхбыстрого перекодирования и быстрой трансляции в прямом эфире мы предоставляем опыт прямой трансляции с низкой задержкой и высоким разрешением.

### ИИ повышает качество образования

- Описание требования: ИИ должен использоваться во время занятий и после них для улучшения качества образования.
- Решение: используйте функцию интеллектуальных субтитров для отображения содержания лекции инструктора в виде субтитров в реальном времени. С помощью функции резюмирования большой модели генерируйте краткое резюме курса и обзор ключевых моментов в конце сеанса, тем самым повышая результативность обучения.

### Улучшение опыта трансляции для инструкторов

Благодаря улучшению производительности кодирования на стороне учителя и повышению возможностей шумоподавления аудио можно оптимизировать качество потока аудио и видео с интерфейса трансляции учителя.

| Функция | Описание | Как использовать |
| --- | --- | --- |
| Terminal Video Codec SDK | Кодировщик видео терминала Tencent Top Speed Codec (TSC) разработан для сценариев, требующих низких вычислительных мощностей, низкой задержки и высокого качества изображения на стороне терминала. По сравнению с аппаратным кодированием его преимущества включают: стабильность, надежность и быстрый запуск. При одинаковом качестве он сохраняет битрейт, повышает стабильность передачи, снижает пропускную способность нисходящего распределения и снижает затраты на хранение. При одинаковом битрейте он улучшает качество изображения и повышает удобство пользователя. Богатый набор функций удовлетворяет разнообразные бизнес-требования, такие как использование кодирования с учетом интересующих регионов (ROI) для улучшения качества изображения в области лица и динамическая регулировка конфигурации кодирования для адаптации к колебаниям сети. | Шаг первый: требуется активировать пробную лицензию через консоль [Активировать тестовую лицензию](https://www.tencentcloud.com/document/product/1041/65954#c75f17d7-69ad-43e2-9fcd-16a2eace0d8c) или [Получить официальную лицензию SDK](https://www.tencentcloud.com/document/product/1041/65954#4abdea9b-657c-49f0-91df-21991d1c95c6); Шаг второй: для технической интеграции обратитесь к процессу [Интеграция SDK ультра-HD кодирования терминала](https://www.tencentcloud.com/document/product/1041/60831#294b81fb-0ba2-420c-99fc-40805708d0e2). |
| Terminal Audio SDK | Terminal Audio SDK, встроенный на стороне клиента, значительно повышает качество звука, устраняя эхо и шум. Используемые технологии включают: адаптивное подавление шума, устранение компонентов фонового шума в аудиосигнале и снижение помех для улучшения качества восприятия голоса. Акустическая отмена эха в основном решает проблему эха в аудиокоммуникации. Автоматическая регулировка усиления отвечает за регулировку громкости при передаче аудиосигналов. Возможность проверки качества аудио терминала позволяет оценить качество входного аудиосигнала, своевременно выявляя проблемы с аудио и предоставляя оповещения. | Шаг первый: требуется активировать пробную лицензию через консоль [Активировать тестовую лицензию](https://www.tencentcloud.com/document/product/1041/65954#c75f17d7-69ad-43e2-9fcd-16a2eace0d8c) или [Получить официальную лицензию SDK](https://www.tencentcloud.com/document/product/1041/65954#4abdea9b-657c-49f0-91df-21991d1c95c6); Шаг второй: для технической интеграции обратитесь к процессу [Интеграция SDK ультра-HD кодирования терминала](https://www.tencentcloud.com/document/product/1041/60831#47de165a-7753-4985-9415-539b9f772a74). |

### Улучшение опыта просмотра для студентов

| Функция | Описание | Как использовать |
| --- | --- | --- |
| Top Speed Codec (TSC) Перекодирование | Перекодирование TSC является "обновленной" версией стандартного перекодирования видео, способной адаптивно оптимизировать различные видео для предоставления более четкого опыта просмотра с более низкой полосой пропускания. При сохранении или улучшении качества изображения это сохраняет более 50% затрат на полосу пропускания. | Обратитесь к документации [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33627). |
| Проверка качества | Проверка качества мультимедиа поддерживает: проверку качества формата, проверку качества содержимого видео и аудио, оценку качества без эталона. Эта функция обнаруживает проблемы в прямой трансляции, оповещая операторов для быстрого выявления аномалий. Для сценариев видео по требованию она обнаруживает и оценивает содержимое, интегрируя функции перекодирования и улучшения для решения проблем. | Обратитесь к документации [Интеграция проверки качества мультимедиа](https://www.tencentcloud.com/document/product/1041/67727). |
| Улучшение видео | Использование передовых моделей обработки ИИ в MPS, наряду с множеством приложений в различных бизнес-сценариях, функция улучшения значительно повышает качество аудио и видео. Поддерживает распределенное улучшение качества изображения в реальном времени, удаление артефактов, шумоподавление, улучшение цвета, улучшение деталей и улучшение лица. В образовательной среде возможно улучшить видеоизображение путем оптимизации цветового представления, уточнения деталей лица и повышения четкости письменного содержимого на доске. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/3ad29c6413b311f09b3252540044a08e.png) | Обратитесь к документации [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33627). |
| Облачная запись потока | Традиционная запись только захватывает визуальные эффекты, а не содержимое аудио и видео на стороне клиента. Чтобы записать ровно то, что видно, необходимо использовать облачную Windows или Android для потоковой передачи через прямую трансляцию Tencent Cloud, интегрируя модуль записи для создания файлов. Эта функция может использоваться для создания повторов курса и презентаций учителя. | Свяжитесь с бизнес-командой для участия в бета-тестировании. |
| Интеллектуальные субтитры | Функция интеллектуальных субтитров распознает голоса из видео или прямых трансляций в реальном времени, преобразует речь в субтитры и переводит их на различные языки, такие как китайский, английский и японско-корейские. Это полезно для прямых субтитров, транскрипции видео для глобального распределения и извлечения субтитров из старых фильмов для восстановления. Функция поддерживает настройку горячих слов для более точного перевода. В образовательной среде функция субтитров может использоваться для создания субтитров в реальном времени во время прямых занятий в классе, проводимых учителями; она также может использоваться для добавления субтитров к видео по требованию, тем самым повышая опыт просмотра для пользователей. | Обратитесь к документации [Генерация и перевод субтитров](https://www.tencentcloud.com/document/product/1041/54517). |
| Руководство по резюмированию LLM | Опираясь на основу интеллектуального распознавания, резюмирование большой модели способно создавать краткие резюме для видеоконтента. Это автоматизирует создание ключевых моментов и комплексных резюме содержимого курса. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/8f1d061403a011f09c4a5254001c06ec.png) | Обратитесь к [mps.live](https://mps.live/demo/ai/llm) для бесплатной пробной версии. Обратитесь к документации [Руководство по резюмированию LLM](https://www.tencentcloud.com/document/product/1041/66042). |

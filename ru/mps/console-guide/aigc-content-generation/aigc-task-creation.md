# Создание задач AIGC

## **Сценарии**

Консоль Media Processing Service (MPS) предлагает различные типы возможностей создания задач генерации AIGC. Пользователи могут быстро оценить эффекты генерации через консоль.

## Предварительные условия

1. Вы [зарегистрировали аккаунт Tencent Cloud](https://www.tencentcloud.com/document/product/378/17985) и прошли проверку личности.
2. Вы активировали [Tencent Cloud MPS](https://www.tencentcloud.com/document/product/1041/44041).

## Этапы использования

1. Войдите в **консоль MPS**, перейдите на страницу [**Создание задач AIGC**](https://console.tencentcloud.com/mps/aigc/create) и выберите **Генерация видео** или **Генерация изображений**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2919adaee16511f0ab0f5254005ef0f7.png)

> **Примечание:** Диапазоны значений параметров, поддерживаемые различными моделями, различаются.

4. Нажмите **Генерация**, и затем вы можете остаться на текущей странице для создания другого содержания или перейти на страницу управления задачами AIGC для просмотра задачи.

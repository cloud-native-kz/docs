# Управление задачами AIGC

## Сценарии

После инициирования задачи AIGC через консоль или вызова API, информация об инициированной задаче может быть просмотрена и управляема в [Управлении задачами AIGC](https://console.tencentcloud.com/mps/aigc/tasks).

## Просмотр задачи генерации

1. [Список задач AIGC](https://console.tencentcloud.com/mps/aigc/tasks) включает два отдельных списка: "**Генерация изображений**" и "**Генерация видео**".

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f08e004fe16511f0948a52540099c741.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/fc7d6636e16511f095ab5254001c06ec.png)

Описание полей списка см. в следующей таблице:

| Название столбца | Описание |
| --- | --- |
| ID задачи | ID инициированной задачи. |
| Тип задачи | Тип задачи, соответствующий ID задачи. В частности, генерация изображений: включает генерацию текста в изображение и генерацию изображения в изображение. Генерация видео: включает генерацию текста в видео и генерацию изображения в видео. |
| Статус задачи | Включает: Ожидание: задача находится в очереди ожидания обработки. Выполнение: задача вошла в фазу выполнения. Успешно: выполнение задачи завершено. Вы можете просмотреть результат задачи. Ошибка: выполнение задачи не удалось, возможная причина — превышение времени ожидания. |
| Подсказка | Информация подсказки, переданная при отправке запроса. |
| Модель | Запрошенные параметры модели. |
| Разрешение | Фактическое разрешение созданного видео. |
| Длительность видео | Продолжительность созданного видео (это поле недоступно в списке задач изображений). |
| Время создания | Момент времени, когда инициирована задача. |
| Время окончания | Момент времени, когда завершена задача. |
| Операция | Результаты успешно созданных задач могут быть просмотрены или загружены на странице просмотра результатов. |

2. Нажмите **Просмотр результата**, чтобы просмотреть дополнительную информацию о задаче.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/36233657e16611f095b0525400e889b2.png)

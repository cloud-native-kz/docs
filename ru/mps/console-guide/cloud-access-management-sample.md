# Пример управления доступом в облаке

## Введение

Вы можете авторизовать пользователей для просмотра и использования определенных ресурсов в консоли Media Processing Service (MPS) через политики управления доступом облака (CAM). В этом документе приводятся примеры авторизации для просмотра и использования определенных ресурсов, которые помогут пользователям понять, как использовать определенные политики CAM в консоли MPS.

## Примеры

#### Полный доступ на чтение и запись (MPS)

Вы можете использовать политику с названием QcloudMPSFullAccess для пользователей. Эта политика предназначена для предоставления пользователям прав доступа ко всем ресурсам в MPS.
Подробные шаги выглядят следующим образом:
Обратитесь к разделу [Управление авторизацией](https://www.tencentcloud.com/document/product/598/10602) для получения инструкций по предоставлению пользователям предустановленной политики QcloudMPSFullAccess.

#### Доступ для роли MPS

Вы можете использовать политику с названием QcloudAccessForMPSRole для пользователей. Эта политика предназначена для предоставления пользователям прав на чтение списка бакетов Object Storage (COS), чтение и запись конфигураций бакетов, а также чтение или загрузку объектов. Она также включает возможность отправки сообщений через Message Queue (CMQ).

Подробные шаги выглядят следующим образом:
Обратитесь к разделу [Управление авторизацией](https://www.tencentcloud.com/document/product/598/10602) для получения инструкций по предоставлению пользователям предустановленной политики QcloudAccessForMPSRole.

#### Доступ для роли MPS при доставке в SCF

Вы можете использовать политику с названием QcloudAccessForMPSRoleInDeliverToSCF для пользователей. Эта политика предназначена для связи с QCSRole MPS и используется MPS для временного доступа к ресурсам других облачных сервисов.

Подробные шаги выглядят следующим образом:
Обратитесь к разделу [Управление авторизацией](https://www.tencentcloud.com/document/product/598/10602) для получения инструкций по предоставлению пользователям предустановленной политики QcloudAccessForMPSRoleInDeliverToSCF.

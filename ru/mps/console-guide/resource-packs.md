# Пакеты ресурсов

## Обзор

На этой странице показаны приобретённые вами пакеты ресурсов для общего транскодирования и транскодирования Top Speed Codec (TSC), а также их использование.

## Просмотр пакетов ресурсов

Войдите в

консоль MPS

и выберите

Пакеты ресурсов

на левой боковой панели. На вкладке

**Пакеты ресурсов**

вы можете просмотреть информацию о приобретённых пакетах ресурсов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/aee64e4cd6dc11eea2b0525400bb593a.png)

| Заголовок | Описание |
| --- | --- |
| ID пакета ресурсов | Уникальный идентификатор пакета ресурсов. Вы можете ввести ID пакета ресурсов в поле поиска для поиска пакета ресурсов. |
| Статус | Текущий статус пакета ресурсов, который может быть действительным, истекшим, заблокированным или исчерпанным. Действительный: пакет ресурсов действителен и может использоваться. Заблокированный: пакет ресурсов был заблокирован по какой-либо причине и больше не может использоваться. Истекший: пакет ресурсов истёк и больше не может использоваться. Период действия пакета ресурсов составляет один год. Исчерпанный: пакет ресурсов полностью использован. |
| Тип | Тип пакета ресурсов, который может быть общим транскодированием или транскодированием TSC. Вы можете просмотреть только пакеты общего транскодирования или пакеты транскодирования TSC, выбрав соответствующий тип. |
| Всего минут | Общее количество минут, предоставляемое пакетом ресурсов, которое не изменяется после начала использования пакета. |
| Оставшиеся минуты | Количество оставшихся минут в пакете ресурсов, которое уменьшается после начала использования пакета. Когда количество оставшихся минут становится нулевым, статус пакета изменится на истекший. |
| Время начала | Время, когда пакет ресурсов стал действительным. |
| Время окончания | Время истечения пакета ресурсов, которое наступает через один год после времени начала. |
| Операция | Вы можете нажать кнопку «Возврат» для возврата пакета ресурсов. Примечание: возврат пакетов ресурсов возможен только в течение пяти дней с момента покупки. |

## Просмотр использования

Войдите в

консоль MPS

и выберите

Пакеты ресурсов

на левой боковой панели. На вкладке

**Детали использования**

вы можете просмотреть детали использования ваших пакетов ресурсов.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/aeedb6f8d6dc11ee9409525400c26da5.png)

Вы можете просмотреть списания определённого пакета ресурсов (

**ID пакета ресурсов**

) за определённый период времени (

**время списания**

).

| Заголовок | Описание |
| --- | --- |
| Время списания | Время, когда произошло списание. |
| ID пакета ресурсов | ID пакета ресурсов, с которого были списаны минуты. |
| Тип | Тип пакета ресурсов. Вы можете выбрать тип для просмотра списаний этого типа. |
| Тип транскодирования | Тип транскодирования списания. Использование одного и того же типа транскодирования суммируется каждый день. |
| Минуты до списания | Оставшиеся минуты пакета ресурсов до того, как произошло списание. |
| Минуты после списания | Оставшиеся минуты пакета ресурсов после списания. |

# Инструмент редактирования субтитров

## Применимые сценарии

Этот инструмент можно использовать для калибровки файлов субтитров. Он поддерживает загрузку файлов субтитров, редактирование текстового содержимого и корректировку временной шкалы. Вы также можете добавлять видео для предварительного просмотра, чтобы обеспечить синхронизацию субтитров с видеоконтентом.

**Вы можете использовать** [**Инструмент редактирования субтитров**](https://console.tencentcloud.com/mps/subtitle_tool) **бесплатно.**

![](https://staticintl.cloudcachetci.com/cms/backend-cms/c56ae25c693211f08eae52540044a08e.png)

## Инструкции по использованию

### 1. Добавление файлов субтитров и видео

- Добавьте локальный файл субтитров или файл субтитров из облачного хранилища объектов Tencent Cloud (COS) для последующего редактирования. Поддерживаются форматы WebVTT и SRT.
- Добавьте локальный видеофайл или видеофайл из COS для предварительного просмотра. Поддерживаются форматы MP4, MOV, AVI, M4V и MPG.
- Нажмите **File Information** в верхнем правом углу, чтобы просмотреть пути добавленных файлов. Для локальных файлов отображается только имя файла. Вы можете нажать **Re-select File** в верхнем правом углу, чтобы выбрать и добавить другие файлы субтитров или видео.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2d146577693311f088f3525400e889b2.png)

![](https://staticintl.cloudcachetci.com/cms/backend-cms/49e5952a6ba611f088f3525400e889b2.png)

### 2. Редактирование и сохранение субтитров

#### **(1) Редактирование субтитров**

Нажмите **Edit**, чтобы перейти в режим редактирования после добавления файла субтитров. Вы можете редактировать временную шкалу и содержимое субтитров файла.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7a04718d6ba611f09cbf525400454e06.png)

- Корректировка временной шкалы

Выберите строку субтитров, которую необходимо отрегулировать. Затем нажмите кнопку регулировки или вручную введите точные временные точки для корректировки времени начала и окончания этой строки субтитров. Это обеспечит точное совпадение субтитров с видеоконтентом.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/86e4ed816ba611f09d77525400bf7822.png)

- Изменение содержимого субтитров

Выберите строку субтитров, которую необходимо отрегулировать, и вручную отредактируйте содержимое субтитров. Вы можете нажать Enter, чтобы добавить пустую строку субтитров для текущей временной точки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/9b14defc6ba611f0bcc652540099c741.png)

- Добавление/удаление строк субтитров

Наведите курсор на строку субтитров, чтобы вызвать кнопки **Insert to Below** и **Delete**. Нажмите их, чтобы добавить строку субтитров или удалить выбранную строку субтитров.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a91106f06ba611f0a1c55254005ef0f7.png)

- Поиск и замена

Нажмите **Find and Replace** в верхнем правом углу и введите содержимое для поиска. Система отобразит количество совпадающих записей (X entries) и автоматически выделит все результаты. Введите содержимое для замены и нажмите **Replace**, чтобы завершить замену в один клик.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bc4192166ba611f0bcc652540099c741.png)

- Отмена редактирования

Нажмите **Cancel Editing**, чтобы отменить все текущие изменения. Субтитры будут восстановлены в исходное состояние до редактирования.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ca891b246ba611f09cbf525400454e06.png)

#### **(2) Сохранение после редактирования**

- Для локального файла субтитров:

Нажмите **Confirm** после редактирования, чтобы сохранить изменения. Вы можете снова перейти в режим редактирования для внесения изменений или непосредственно нажать **Download**, чтобы сохранить отредактированный файл субтитров в локальный путь.

> **Примечание:** Отредактированные данные субтитров будут временно сохранены в браузере после успешного сохранения. Временно сохраненные данные будут потеряны при выходе со страницы (закрытии или обновлении). Не покидайте страницу. Рекомендуется нажать **Download**, чтобы оперативно сохранить файл в локальный путь.![](https://staticintl.cloudcachetci.com/cms/backend-cms/d81da7c96ba611f09dc95254007c27c5.png)

- Для файла субтитров в COS:
  - Нажмите **Save**, чтобы напрямую перезаписать исходный файл субтитров в COS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f81fdcf46ba611f08eae52540044a08e.png)

  - Нажмите **Save As**. Вы можете выбрать сохранение отредактированного файла субтитров в локальный путь или сохранить его как новый файл в COS (без перезаписи исходного файла).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/08221ef06ba711f08eae52540044a08e.png)

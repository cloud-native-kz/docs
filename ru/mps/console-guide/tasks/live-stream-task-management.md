# Управление задачами трансляции в реальном времени

## Описания операций

### Создание задачи трансляции

Нажмите [Создать задачу](https://console.tencentcloud.com/mps/tasks/create/live), следуйте инструкциям на странице для настройки адреса трансляции, схемы и пути сохранения выходных данных. В настоящее время консоль поддерживает запись трансляций в реальном времени. Для подробной информации о конфигурации шаблонов см. [Шаблон записи трансляции](https://www.tencentcloud.com/document/product/1041/56732).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/850cee86232c11efa62352540003d080.png)

> **Примечание:** При создании задачи записи убедитесь, что введенный адрес трансляции правильный. Если трансляция не может быть получена, операция получения будет повторена три раза. Если операция все равно не удастся, статус задачи записи изменится на Ошибка.

### **Управление задачами трансляции**

После успешного создания задачи трансляции она будет выполняться автоматически. Вы можете выполнять такие операции, как просмотр деталей или завершение задачи, в [списке задач](https://console.tencentcloud.com/mps/tasks/live).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/78f6d5d6232d11efa62352540003d080.png)

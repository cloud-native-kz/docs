# Управление задачами VOD

## Сценарии операций

После инициирования задачи обработки мультимедиа путем загрузки файла в корзину COS, запуска скриптов ручного перекодирования или вызовов API вы можете просматривать и управлять задачей в модуле [VOD](https://console.intl.cloud.tencent.com/mps/tasks/vod-list).

## Описания операций

### Список задач

![](https://staticintl.cloudcachetci.com/cms/backend-cms/86da7cfd846211ee8fb75254000524f8.png)

| Название столбца | Описание |
| --- | --- |
| ID задачи | Идентификатор инициированной основной задачи. |
| Статус | Текущий статус основной задачи, который может быть: Ожидание, В процессе, Завершено. **Ожидание**: задача находится в очереди в ожидании обработки. **В процессе**: статус основной задачи В процессе, пока выполняется любая из её подзадач. **Завершено**: статус задачи Завершено, когда ни одна подзадача не выполняется. |
| Время создания | Момент времени, когда была инициирована задача. |
| Время завершения | Момент времени, когда задача была завершена. |
| Операция | Дополнительные сведения см. в разделе [Операции с задачами](#.E4.BB.BB.E5.8A.A1.E6.93.8D.E4.BD.9C.5B.5D(id.3Aoperate)) ниже. |

### Запрос задач

1. Нажмите [VOD](https://console.tencentcloud.com/mps/tasks/vod), чтобы перейти на страницу управления задачами. Список на этой странице отображает записи об основных задачах, инициированных этой учётной записью.
2. Вы можете отфильтровать требуемую основную задачу, введя **ID задачи** в поле поиска в верхнем правом углу списка или выбрав **статус задачи** в списке.

### Создание задачи

1. Перейдите на страницу [VOD](https://console.tencentcloud.com/mps/tasks/vod) и нажмите [Создать задачу](https://console.tencentcloud.com/mps/tasks/vod/create), чтобы перейти на страницу создания задачи.
2. Выберите видеофайл для обработки, укажите путь вывода, шаблон перекодирования и другую необходимую информацию, затем инициируйте задачу.

### Операции с задачами

Поддерживаемые операции управления задачами включают: Подробности, Перезагрузить, Завершить и Воспроизвести исходное видео.

- Подробности: вы можете нажать **Подробности**, чтобы просмотреть информацию о всех подзадачах основной задачи.
- Перезагрузить: задачи со статусом "Завершено" можно перезагрузить. Вы можете нажать **Перезагрузить**, чтобы повторно выполнить все подзадачи основной задачи.
- Завершить: задачи со статусом "Ожидание" можно завершить. Вы можете нажать **Завершить**, чтобы отменить ожидающие задачи и дальнейшее выполнение.
- Воспроизвести исходное видео: вы можете нажать **Воспроизвести исходное видео**, чтобы получить и воспроизвести входной видеофайл задачи.

### Список подзадач

В списке задач вы можете нажать **Подробности**, чтобы отобразить информацию о всех подзадачах основной задачи. Подробная информация выглядит следующим образом:

| Название столбца | Описание |
| --- | --- |
| Номер подзадачи | Уникальные порядковые номера, различающие подзадачи основной задачи. |
| Статус подзадачи | Статус подзадач, включая: Ожидание, В процессе, Успешно, Ошибка. |
| Тип подзадачи | Тип подзадач, который может быть: Перекодирование аудио/видео, Улучшение аудио/видео, Снимок экрана и т. д. |
| Время начала | Время инициирования подзадачи. |
| Время завершения | Время завершения подзадачи. |
| Выход | Место выхода подзадачи. (Местоположение вывода не требуется, когда тип подзадачи — Интеллектуальный анализ, Идентификация или Проверка; вы можете использовать [API DescribeTaskDetail](https://www.tencentcloud.com/document/product/1041/33644) для запроса сведений о статусе выполнения и результатах подзадач) |
| Операция | Дополнительные сведения см. в разделе [Операции с подзадачами](#squery) ниже. |

### Операции с подзадачами

Поддерживаемые операции с подзадачами включают **Подробности**, **Воспроизведение/Просмотр** и **Загрузка**.

| Название операции | Описание |
| --- | --- |
| Подробности | Вы можете проверить подробности подзадачи. |
| Воспроизведение/Просмотр | Вы можете проверить результат обработки подзадачи и воспроизвести видео после обработки подзадачи. |
| Загрузка | Вы можете нажать **Загрузка**, чтобы загрузить выходной файл этой подзадачи. Эта операция поддерживается только для подзадач перекодирования аудио/видео, снимка экрана и улучшения аудио/видео. **Примечание: если создано несколько снимков экрана, в настоящее время можно загрузить только первый снимок экрана. В будущих версиях вы сможете упаковать несколько снимков экрана для загрузки.** |

# Шаблон улучшения аудио/видео

Используя передовые модели обработки искусственного интеллекта в рамках MPS, наряду с широким спектром приложений в различных сценариях бизнеса, функция улучшения значительно повышает качество аудио и видео. Она находит широкое применение в OTT, электронной коммерции и спортивных событиях, обеспечивая значительные бизнес-выгоды благодаря улучшению Quality of Experience (QoE) и Quality of Service (QoS).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/bf303d85da0711efa1f2525400bf7822.png)

## Обзор

Система предоставляет набор предустановленных шаблонов улучшения для вашего удобства. Кроме того, вы можете создавать пользовательские шаблоны, адаптированные к вашим конкретным потребностям бизнеса. Установив различные параметры обработки для различных сценариев приложений, эти шаблоны облегчают повторное использование в будущих проектах. Ниже приводится подробное объяснение того, как создавать пользовательские шаблоны улучшения аудио/видео с помощью консоли и API.

## Создание шаблона улучшения через консоль

1. Перейдите на страницу **Управление шаблонами** > [Шаблон улучшения аудио/видео](https://console.tencentcloud.com/mps/templates/enhs) в консоли Media Processing и нажмите **Создать шаблон улучшения аудио/видео**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7777dc95f82211f093ec525400074c32.png)

2. При входе на страницу создания шаблона улучшения аудио/видео можно настроить следующие параметры. Для получения подробных инструкций по параметрам см. раздел [Описание конфигурации](#f7916e0f-8d11-4205-8304-55bc43c10fc6) ниже.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2f285237f74311f0831c52540073fd3b.png)

### Конфигурация параметров, связанных с транскодированием

Улучшенная функция MPS основана на транскодировании, что означает, что улучшенная обработка накладывает параметры улучшения на основе процесса транскодирования. Поэтому при использовании улучшенной функции вы можете одновременно настраивать параметры транскодирования и параметры улучшения.

Общие параметры транскодирования включают тип транскодирования, выходной формат мультиплексирования, стандарт кодирования, разрешение, частоту кадров и битрейт.

Нажмите **Дополнительные параметры транскодирования**, чтобы настроить полные параметры транскодирования в появившемся интерфейсе справа. Для получения подробных объяснений параметров транскодирования см. [Шаблон транскодирования аудио/видео](https://www.tencentcloud.com/document/product/1041/48784).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/41396af2f74311f08586525400380f7d.png)

> **Примечание:** При выборе типа данных "транскодирование аудио и видео" можно включить функции улучшения видео и улучшения аудио. Если вы отмените "аудиопараметры" (только выходное видео) в конфигурации транскодирования или отключите "транскодирование аудио" (использовать исходное аудио) в аудиопараметрах, вы сможете включить только функцию улучшения видео и не сможете включить улучшение аудио. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/760f228cf74311f0a74f5254001d6acc.png) При выборе типа данных "только транскодирование аудио" (выходной аудиофайл) можно включить только функцию улучшения аудио.

## Создание шаблона улучшения через API

Улучшенная функция MPS основана на транскодировании, что означает, что улучшенная обработка накладывает параметры улучшения на основе процесса транскодирования. Вы можете использовать API MPS [создать шаблон транскодирования](https://www.tencentcloud.com/document/product/1041/33671) для создания пользовательских шаблонов транскодирования и улучшения, например:

```
{  "Container": "mp4",  "Name": "test",  "VideoTemplate": { //Video transcoding parameter configuration.    "Codec": "h264",    "Fps": 50,    "Bitrate": 5000,    "Width": 0,    "Height": 0,    "Gop": 0  },  "AudioTemplate": { //Audio transcoding parameter configuration.    "Codec": "aac",    "Bitrate": 60  },  "EnhanceConfig": {  //Enhancement parameter configuration.    "VideoEnhance": { //Video enhancement configuration.      "FrameRate": {  //Frame interpolation.        "Switch": "ON",        "Fps": 50      },      "SuperResolution": {  //Super resolution.        "Switch": "ON",        "Type": "lq"      }    },    "AudioEnhance": { //Audio enhancement configuration.      "Denoise": {  //Audio denoising.        "Switch": "ON"      }    }  }}
```

> **Примечание:** [API Explorer](https://console.tencentcloud.com/api/explorer?Product=mps&Version=2019-06-12&Action=CreateTranscodeTemplate) поддерживает быструю проверку. Введите страницу и заполните соответствующие параметры, и тогда вы сможете инициировать вызов онлайн-API. ![](https://staticintl.cloudcachetci.com/cms/backend-cms/9af361fd4ff011f09d0a525400454e06.png)

# Шаблон интеллектуального анализа

Система предоставляет предустановленные шаблоны интеллектуального анализа, которые вы можете напрямую использовать в управлении рабочими процессами. Вы также можете создать собственные шаблоны интеллектуального анализа в соответствии с потребностями бизнеса. Нажмите  **Create Intelligent Analysis Template**  для входа в параметры настройки шаблона.

| Элемент конфигурации | Описание |
| --- | --- |
| Template Name | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры, пробелы, подчеркивания (_), дефисы (-) и точки (.) |
| Content analysis configuration | Интеллектуальное маркирование, интеллектуальная категоризация, интеллектуальное создание эскизов и маркирование отдельных кадров |

> **Примечание:** Вы можете просмотреть  **system preset**  шаблоны интеллектуального анализа в [MPS console > Intelligent Analysis Templates](https://console.intl.cloud.tencent.com/mps/templates/analysises).

Созданные шаблоны отображаются в списке шаблонов анализа содержимого, где вы можете просмотреть детали, отредактировать или удалить шаблон.

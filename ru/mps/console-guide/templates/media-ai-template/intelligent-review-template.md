# Шаблон интеллектуальной проверки

1. MPS предоставляет предустановленные шаблоны модерации, которые можно добавить непосредственно в схему. Вы также можете нажать **Create template** для создания собственных настраиваемых шаблонов модерации.
  - Имя шаблона: До 64 символов; поддерживает китайские иероглифы, буквы, цифры, пробелы, подчеркивания (_), дефисы (-) и точки (.).
  - Элементы модерации: Распознавание изображений, распознавание речи и распознавание текста. Подэлементы выбранных элементов модерации будут отображаться в столбце справа.

| Элемент модерации | Подэлемент | Описание |
| --- | --- | --- |
| Распознавание изображений | Порнографический контент | Порнография, вульгарность, интимность и сексуальность |
|  | Терористический контент | Кровавые сцены, взрывы и пожары |
|  | Политически чувствительный контент | Запрещенные значки и знаменитости из спорта и индустрии развлечений |
| Распознавание речи | Порнографический контент | Порнография, вульгарность, интимность и сексуальность |
|  | Политически чувствительный контент | Запрещенные значки и знаменитости из спорта и индустрии развлечений |
| Распознавание текста | Порнографический контент | Порнография, вульгарность, интимность и сексуальность |
|  | Политически чувствительный контент | Запрещенные значки и знаменитости из спорта и индустрии развлечений |

2. Для каждого подэлемента можно установить **Confirm Threshold** и **Suspicion Threshold**, которые определяют строгость модерации. Если они остаются пустыми, будут использоваться значения по умолчанию.
  - **Confirm threshold**: MPS анализирует загруженные видео и присваивает им оценки подтверждения. Если оценка видео превышает порог подтверждения, видео будет отмечено как подтвержденное. Диапазон значений порога — 0–100. Рекомендуется использовать значение по умолчанию.
  - **Suspicion threshold**: MPS анализирует загруженные видео и присваивает им оценки подозрения. Если оценка видео превышает порог подозрения, видео будет отмечено как подозрительное. Вы можете инициировать ручную модерацию подозрительных видео на странице модерации видео. Диапазон значений порога — 0–100. Рекомендуется использовать значение по умолчанию.

> **Примечание:** Вы можете просмотреть **системные предустановленные** шаблоны аудита в [MPS console > Intelligent Auditing Templates](https://console.intl.cloud.tencent.com/mps/templates/audits).

3. Созданные шаблоны отображаются в списке шаблонов модерации, где можно просмотреть детали шаблона, редактировать его или удалить.

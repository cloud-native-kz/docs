# Шаблон интеллектуальных субтитров

## Обзор

Функция интеллектуальных субтитров может выполнять автоматическое распознавание речи (ASR) на основе информации о голосе в видеофайлах по требованию или прямых трансляциях, преобразовывать её в субтитры и переводить их на несколько языков. Она подходит для сценариев, таких как прямые субтитры при трансляции и перевод видео для зарубежного распространения. Вы можете создавать пользовательские шаблоны интеллектуальных субтитров и предустанавливать различные параметры обработки для разных сценариев использования, чтобы облегчить последующее повторное использование.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1fa643d0ef6e11efa8355254001c06ec.png)

## Предварительные требования

1. Вы [зарегистрировались в Tencent Cloud](https://www.tencentcloud.com/document/product/378/17985).
2. Вы активировали [Media Processing Service (MPS)](https://www.tencentcloud.com/products/mps?from_qcintl=422130401) и вошли в [консоль MPS](https://console.tencentcloud.com/mps/index).

## Руководство по конфигурации шаблона

1. Перейдите в **Templates >** [**Media AI Template**](https://console.tencentcloud.com/mps/templates/intel) **> Smart Subtitle**. Система предоставляет несколько предустановленных шаблонов, которые вы можете использовать напрямую, или вы можете нажать **Create Smart Subtitle Template**, чтобы создать пользовательский шаблон.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/00f3ff94a58b11f0930a5254007c27c5.png)

2. Откройте страницу создания шаблона интеллектуальных субтитров. Поддерживаются следующие параметры конфигурации.

### Тип обработки "Создание субтитров на основе ASR"

![](https://staticintl.cloudcachetci.com/cms/backend-cms/510c2825c51311f0a1dc52540044a08e.png)

Поддерживаются следующие элементы конфигурации:

| Элемент конфигурации |  | Описание |
| --- | --- | --- |
| Имя шаблона |  | Поддерживаются только китайские иероглифы, английские буквы, цифры, подчёркивания (_), дефисы (-) и точки (.), длина не должна превышать 64 символов. |
| Тип обработки |  | Создание субтитров на основе автоматического распознавания речи (ASR): введите аудиофайл или видеофайл, создайте файл субтитров на основе ASR и переведите субтитры. Перевод файлов субтитров: введите файл субтитров, переведите исходные субтитры на субтитры на разных языках через LLM и создайте новый файл субтитров. |
| Тип обработки "Создание субтитров на основе ASR" | Исходный язык аудиоконтента видео | Выберите исходный язык аудиоконтента видео. Ниже приведен [список поддерживаемых исходных языков](#ASRlanguages). |
|  | Связать пользовательский лексикон ключевых слов ASR | Часто используемые слова в речи обычно распознаются точно. Однако для специальных названий (таких как личные имена, названия продуктов и названия компаний) и терминов, специфичных для отрасли (таких как название бренда Zhiling, название здания Binhai Dasha, Hebao в страховой отрасли или Cunchutong в облачном хранилище), точность распознавания может снизиться. Чтобы решить эту проблему, мы предоставляем функцию пользовательского лексикона ключевых слов. Вы можете добавлять специализированные термины путём ручного ввода или импорта файлов, значительно повышая точность ASR. Для подробного руководства по конфигурации см. [Пользовательские лексиконы ключевых слов](https://www.tencentcloud.com/document/product/1041/68176). **Примечание:** В настоящее время лексикон ключевых слов поддерживает только мандаринский и английский языки. Таким образом, лексикон ключевых слов можно связать только когда исходный язык аудиоконтента видео — упрощённый китайский или английский. |
|  | Включить перевод | После включения субтитры на исходном языке можно перевести на указанный язык, что подходит для сценариев, таких как перевод видео для глобального распространения. Ниже приведен [список поддерживаемых целевых языков для перевода.](#ASRlanguages) **Примечание:** Некоторые языки в настоящее время недоступны для выбора. Если вам нужно их использовать, [свяжитесь с нами](https://console.tencentcloud.com/workorder/category). |
|  | Целевой язык для перевода |  |
|  | Тип субтитров | Если функция перевода не включена, выходные субтитры будут содержать только контент на исходном языке. Если функция перевода включена, поддерживаются следующие типы выходных субтитров: Одноязычный (целевой язык) Двуязычный (исходный язык + целевой язык) |
|  | Формат файла субтитров | В настоящее время поддерживаются форматы WebVTT и SRT. Если вам нужна только обратная связь о содержимом субтитров и вам не нужен выходной файл субтитров, вы можете выбрать Do Not Generate Subtitle Files. |

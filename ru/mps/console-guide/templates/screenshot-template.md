# Шаблон скриншота

MPS предоставляет предустановленные шаблоны скриншотов, которые можно добавить прямо в схему. Поддерживаются три типа скриншотов: скриншот в точке времени, выборочный скриншот и скриншот спрайта изображения. Вы также можете нажать **Создать шаблон** для создания собственных пользовательских шаблонов скриншотов.

## Скриншот в точке времени

| Элемент | Описание |
| --- | --- |
| Имя шаблона | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы и точки (.) |
| Формат изображения | JPG |
| Размеры изображения | Ширина и высота изображения должны быть в диапазоне 128–4096 пх. |

Предустановленные шаблоны:

| ID шаблона | Формат | Ширина | Высота | Режим заполнения |
| --- | --- | --- | --- | --- |
| 10 | JPG | Как источник | Как источник | Масштабировать по размеру |

## Выборочный скриншот

| Элемент | Описание |
| --- | --- |
| Имя шаблона | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы (-) и точки |
| Формат изображения | JPG |
| Размеры изображения | Ширина и высота изображения должны быть в диапазоне 128–4096 пх. |
| Интервал выборки | Интервал может быть значением в процентах (%) или значением времени (s). Если выбран `%`, введенное значение не может превышать 100. |

Предустановленные шаблоны:

| ID шаблона | Формат | Ширина | Высота | Единица измерения интервала | Интервал | Режим заполнения |
| --- | --- | --- | --- | --- | --- | --- |
| 10 | JPG | Как источник | Как источник | По проценту | 10% | Масштабировать по размеру |

## Скриншот спрайта изображения

| Элемент | Описание |
| --- | --- |
| Имя шаблона | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы и точки (.) |
| Формат изображения | JPG |
| Размеры изображения | Ширина и высота изображения должны быть в диапазоне 128–4096 пх. |
| Интервал выборки | Интервал может быть значением в процентах (%) или значением времени (s). Если выбран `%`, введенное значение не может превышать 100. |
| Строки | Положительное целое число. Произведение количества строк подизображений на количество столбцов не должно превышать 100. |
| Столбцы | Положительное целое число. Произведение количества строк подизображений на количество столбцов не должно превышать 100. |

Созданные шаблоны можно найти в списке шаблонов скриншотов, который отображает информацию, включая имя шаблона, тип скриншота, размеры изображения и тип шаблона. Вы также можете просмотреть сведения о шаблоне, отредактировать или удалить его на этой странице.

## Анимированный скриншот

| Элемент | Описание |
| --- | --- |
| Имя шаблона | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры и подчеркивания (_) |
| Формат изображения | WEBP или GIF |
| Частота кадров (fps) | 1–30 |
| Качество изображения | 0–100. Чем больше значение, тем выше качество и больше размер изображения. |
| Размеры изображения (пх) | 0 или 128–4096 для любого измерения |

Созданные шаблоны можно найти в списке шаблонов, который отображает информацию, включая имя шаблона, тип изображения, частоту кадров, качество изображения, размеры изображения и тип шаблона. Вы можете просматривать, редактировать или удалять пользовательский шаблон, но предустановленные шаблоны можно только просматривать, а не редактировать или удалять.

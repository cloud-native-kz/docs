# Шаблон записи потока

1. MPS предоставляет предустановленные шаблоны записи потока. Вы можете нажать **Создать шаблон** для настройки шаблонов записи потока. Заполните соответствующую информацию на странице создания шаблона.

| Элемент | Описание |
| --- | --- |
| Имя шаблона | Максимум 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы (-) и точки |
| Описание шаблона | Максимум 256 символов; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы (-) и точки |
| Формат записи | Поддерживает HLS и MP4. Для формата HLS можно настроить: длительность каждого TS-сегмента, максимальную длительность записи и время ожидания возобновления. Для формата MP4 можно настроить: максимальную длительность записи. |
| Длительность TS-сегмента | Должна быть в диапазоне 5～30 секунд. |
| Максимальная длительность записи | Должна быть в диапазоне 10～720 минут. При превышении установленной максимальной длительности записи будет создан новый файл. |
| Время ожидания возобновления | Период времени ожидания возобновления напрямую влияет на время создания файла записи. Если интервал прерывания потока не превышает установленный период ожидания возобновления, один прямой поток создаст только один файл. Установите разумный период времени ожидания возобновления, который должен быть в диапазоне 60～1800 секунд. Если поле не заполнено, это означает, что функция возобновления отключена. |

2. Созданные шаблоны находятся в [списке шаблонов записи потока](https://console.tencentcloud.com/mps/templates/lives), который отображает информацию, включая имя шаблона, описание шаблона, формат записи, время каждого TS, цикл записи, время ожидания возобновления. На этой странице вы также можете просмотреть детали, отредактировать или удалить шаблон.

# Обзор шаблонов

## Общие сведения

Служба обработки медиа (MPS) поддерживает трансходирование, улучшение и интеллектуальный анализ, а также другие богатые функции. Каждая функция имеет соответствующие детальные параметры настройки. Вы можете предварительно установить различные параметры обработки путем настройки шаблонов, что облегчает их последующее повторное использование.

## Указания

Войдите в [консоль MPS](https://console.tencentcloud.com/mps) и нажмите **Templates** на левой боковой панели. Выберите тип шаблона, который вы хотите настроить, и нажмите **Create template**.

## Типы шаблонов

В таблице ниже приведены типы шаблонов обработки медиа, которые вы можете создать и добавить в оркестрировку.

> **Примечание:** После [включения оркестрировки](https://www.tencentcloud.com/document/product/1041/58242#e881ece6-8b7e-4b13-9f14-04858f1024dc), если шаблон, используемый в оркестрировке, отредактирован, оркестрировка будет работать с отредактированными параметрами шаблона. После [включения оркестрировки](https://www.tencentcloud.com/document/product/1041/58242#e881ece6-8b7e-4b13-9f14-04858f1024dc), если шаблон, используемый в оркестрировке, удален, подзадачи, связанные с этим шаблоном, завершатся ошибкой.

| Тип шаблона | Подтип | Описание |
| --- | --- | --- |
| Трансходирование аудио/видео | TSC шаблон трансходирования | По сравнению с обычным трансходированием, TSC шаблон трансходирования значительно снижает размер файла видео, сохраняя его четкость. |
|  | Шаблон обычного трансходирования | Базовые возможности трансходирования видео. |
|  | Шаблон адаптивного потокового воспроизведения видео | Файлы исходного видео можно трансходировать в несколько потоков, подходящих для воспроизведения в различных сценариях, что позволяет пользователям выбирать подходящий битрейт для воспроизведения видео на основе условий сети, тем самым улучшая качество просмотра. |
|  | Шаблон обычного трансходирования аудио | Поддерживает базовую обработку трансходирования для чистых аудиофайлов. |
|  | TSC шаблон трансходирования аудио | По сравнению с обычным трансходированием аудио, достигает результатов трансходирования с более низкими битрейтами и превосходным качеством звука. |
|  | Шаблон адаптивного потокового воспроизведения аудио | Преобразование аудио в форматы адаптивного потокового воспроизведения, позволяя пользователям выбирать подходящий битрейт для воспроизведения аудио на основе условий сети, тем самым улучшая качество прослушивания. |
|  | Шаблон переупаковки | Трансформация формата упаковки исходного видео без его перекодирования. В настоящее время поддерживается преобразование исходных видео в форматы MP4 и HLS. |
| Улучшение аудио/видео | Улучшение аудио/видео | Наша служба поддерживает улучшение и восстановление различных проблем качества в аудио и видео, комплексно улучшая визуальный опыт. Она широко используется на международных крупных событиях, таких как Олимпийские игры и чемпионат мира FIFA, а также в различных сценариях, таких как восстановление классических фильмов и трансляция киберспорта. |
| Интеллектуальная проверка | Интеллектуальная проверка | Проверка изображений, голоса и текста в видео на предмет порнографии, противоправности и нарушений нормативных требований. |
| Интеллектуальное распознавание | Интеллектуальное распознавание | Распознавание лиц, объектов, текста и голоса. Автоматическое распознавание речи (ASR) также поддерживает интеллектуальный перевод и преобразование в субтитры. |
| Интеллектуальный анализ | Интеллектуальный анализ | Поддержка маркирования, классификации, генерации обложек, маркирования с привязкой к кадрам, обрезки, монтажа, вступления и заключения, маркирования игр и других возможностей интеллектуального анализа. |
| Снимок экрана | Снимок экрана в определенный момент времени | Создание снимков экрана в определенные моменты времени. |
|  | Выборочный снимок экрана | Создание снимков экрана с указанным интервалом (в секундах или процентах). |
|  | Снимок экрана спрайта изображения | Создание снимков экрана в определенные моменты времени и их объединение в спрайт. |
|  | Анимированный снимок экрана | Вырезание видеоклипа и его преобразование в анимированный снимок экрана. |
| Водяной знак | Водяной знак | Добавление текстового или графического водяного знака к видео. |
| Запись прямой трансляции | Запись прямой трансляции | Запись содержимого прямой трансляции. |

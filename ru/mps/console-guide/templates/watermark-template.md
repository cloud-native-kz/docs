# Шаблон водяного знака

## Обзор

![](https://staticintl.cloudcachetci.com/cms/backend-cms/4c190dbec9b011f087ad52540099c741.png)

Служба обработки медиа (MPS) поддерживает следующие типы водяных знаков:

| Тип | Описание |
| --- | --- |
| Цифровой водяной знак | Позволяет встраивать пользовательскую текстовую информацию, невидимую невооруженному глазу, в видео без влияния на визуальное качество и целостность. Поддерживаются как базовый цифровой водяной знак, так и цифровой водяной знак NAGRA NexGuard. |
| Видимый водяной знак | Поддерживает добавление изображений в указанных форматах, таких как PNG и JPG, или пользовательского текста в видео в видимой форме. Это решение подходит для сценариев, таких как защита авторских прав и продвижение бренда. |

В данном документе в основном рассматриваются параметры конфигурации для шаблонов водяных знаков. Для получения полного руководства по инициированию задач водяных знаков и подробной информации о функциональной интеграции обратитесь к [Интеграция цифровых водяных знаков и видимых водяных знаков](https://www.tencentcloud.com/document/product/1041/74443).

## Шаблон цифрового водяного знака

Цифровой водяной знак MPS поддерживает две версии: базовый водяной знак авторских прав и судебно-экспертный водяной знак NAGRA NexGuard.

| Версия | Описание |
| --- | --- |
| Базовый водяной знак авторских прав | Базовая версия: встраивает невидимую текстовую информацию в видео без влияния на визуальное качество и целостность. |
| Судебно-экспертный водяной знак NAGRA NexGuard | NAGRA NexGuard — это ведущее мировое решение в области цифровых водяных знаков, отличающееся сильной скрытностью и обладающее возможностями защиты от удаления уровня вещания. Разработано специально для отслеживания пиратства и защиты авторских прав на высокостоимостный контент, такой как фильмы и драматические сериалы. |

1. Выполните вход в консоль MPS и перейдите на страницу [шаблонов водяных знаков](https://console.tencentcloud.com/mps/templates/watermarks?tab=implicitWatermark).
2. Нажмите "Создать шаблон водяного знака", выберите тип водяного знака и введите имя шаблона и содержимое водяного знака для создания шаблона цифрового водяного знака.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a0646183c9b011f087ad52540099c741.png)

> **Примечание:** Каждый пользователь может создать до 50 шаблонов судебно-экспертных водяных знаков NexGuard. После создания пользователи не могут изменять или удалять их. Чтобы изменить или удалить шаблоны, свяжитесь с нами для обработки в фоновом режиме.

## Шаблон видимого водяного знака

1. Перейдите в консоль MPS > [Шаблоны водяных знаков](https://console.tencentcloud.com/mps/templates/watermarks).
2. Нажмите **Создать шаблон видимого водяного знака**, который поддерживает следующие конфигурации:

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d8bc21e5c9b011f09e745254007c27c5.png)

| Элемент конфигурации |  | Описание |
| --- | --- | --- |
| Имя шаблона | Макс. 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы (-) и точки | Макс. 64 символа; поддерживает китайские иероглифы, буквы, цифры, подчеркивания (_), дефисы (-) и точки |
| Тип водяного знака |  | Поддерживает как водяной знак в виде изображения, так и текстовый водяной знак. |
| Для "водяного знака в виде изображения" | Изображение | Изображения в формате PNG или JPG. Для лучшего визуального восприятия рекомендуются прозрачные изображения в формате PNG. Размер изображения не должен превышать 200 КБ или 200 × 200 пиксел. |
| Для "текстового водяного знака" | Содержимое водяного знака | Допускаются только буквенно-цифровые символы и стандартные символы английского языка; знак процента (%) не поддерживается, максимальная длина не должна превышать 100 символов. |
|  | Шрифт | Поддерживает `simkai.ttf` и `arial.ttf`. Для дополнительных требований к шрифтам обратитесь в нашу группу поддержки. |
|  | Размер шрифта | Диапазон размера шрифта составляет [0,1] или [8,4096]. |
|  | Цвет | Цвет содержимого водяного знака. |
|  | Прозрачность | Уровень прозрачности содержимого водяного знака, где 0% обозначает полную прозрачность, а 100% представляет полную непрозрачность. |
| Опорное положение | Верхний левый (по умолчанию), верхний правый, нижний левый или нижний правый, на основе которого вы можете изменять положение изображения водяного знака, регулируя вертикальное и горизонтальное смещение | Верхний левый (по умолчанию), верхний правый, нижний левый или нижний правый, на основе которого вы можете изменять положение изображения водяного знака, регулируя вертикальное и горизонтальное смещение |
| Вертикальное смещение | Процент представляет отношение вертикального расстояния между водяным знаком и опорным положением к высоте видео, которое используется для указания вертикального положения водяного знака. | Процент представляет отношение вертикального расстояния между водяным знаком и опорным положением к высоте видео, которое используется для указания вертикального положения водяного знака. |
| Горизонтальное смещение | Процент представляет отношение горизонтального расстояния между водяным знаком и опорным положением к ширине видео, которое используется для указания горизонтального положения водяного знака. | Процент представляет отношение горизонтального расстояния между водяным знаком и опорным положением к ширине видео, которое используется для указания горизонтального положения водяного знака. |
| Размеры изображения | Вы можете выбрать изменение размера водяного знака в процентах (%) или пикселях (px). | Вы можете выбрать изменение размера водяного знака в процентах (%) или пикселях (px). |

3. Созданные шаблоны можно найти в списке шаблонов водяных знаков, где отображаются предпросмотры водяных знаков и информация, включая имя шаблона, формат, тип, опорное положение, размеры и т. д. Вы также можете просматривать подробную информацию о шаблоне, редактировать или удалять его на этой странице.

# Terminal SDK

## Overview

Media Processing Service (MPS) Top Speed Codec (TSC) terminal SDK может предоставлять функции, такие как сжатие кодирования, улучшение звука и увеличение разрешения видео на клиентских устройствах пользователя (таких как мобильные телефоны, компьютеры и планшеты), улучшая общее качество и опыт использования медиа.

## Prerequisites

Предварительные требования для покупки или пробного использования MPS terminal SDK следующие:

1. Вы должны [зарегистрировать учетную запись Tencent Cloud](https://www.tencentcloud.com/zh/account/register?s_url=https%3A%2F%2Fconsole.tencentcloud.com%2F) и пройти проверку личности.
2. Учетная запись должна иметь разрешения на совершение покупок на странице продукта MPS.

## Directions

Войдите в [консоль Tencent Cloud MPS](https://console.tencentcloud.com/mps/client_sdk/license) и перейдите на страницу MPS **Terminal SDK License**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/89fb3f8ea00e11efa04c52540055f650.png)

### Подача заявки на пробную версию **Terminal SDK License**

#### Step 1: Самостоятельная подача заявки на пробную версию SDK License

На странице нажмите **Create Trial License**, выберите нужный тип и версию SDK для пробного использования, затем нажмите **Create License** для завершения подачи заявки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d66d2b63a00e11efaaca525400fdb830.png)

> **Note:**Каждая пробная License действительна в течение 60 дней. В течение периода действия вы не можете повторно подавать заявки на несколько SDK License одного типа и версии.

#### Step 2: Привязка информации приложения

Пробная SDK License, на которую вы подали заявку, неактивна и станет активной только после завершения привязки информации приложения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/fda1220fa00e11efaaca525400fdb830.png)

Нажмите **Bind** и заполните соответствующую информацию приложения для завершения процесса привязки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0be67ebaa00f11efa646525400329841.png)

> **Note:**В течение периода действия пробной версии вы можете неограниченно изменять информацию привязки. После обновления лицензии до официальной платной License информацию привязки невозможно будет менять по своему усмотрению. Пожалуйста, подтвердите информацию привязки перед обновлением до официальной License.

### Покупка Terminal License

#### Step 1: Выбор продукта для покупки

На странице нажмите **Create Terminal SDK License**, и вы будете перенаправлены на страницу покупки, где можно выбрать необходимую Terminal SDK License. Выберите по очереди:

1. License Type: TSC - Terminal Audio SDK, TSC - Terminal Enhancement SDK.
2. Billing Mode: годовая или ежемесячная покупка.
3. SDK Version.
4. Duration.
5. Number of Licenses (обычно 1).

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8b45de44a00f11efa0b3525400bdab9d.png)

#### Step 2: Подтверждение оплаты заказа

После выбора необходимой конфигурации SDK нажмите **Purchase Now** для просмотра информации о продукте по этому платежу. После подачи заказа выберите подходящий способ оплаты для завершения платежа.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/b9ae2c78a00f11efaaca525400fdb830.png)

#### Step 3: Привязка информации приложения

На купленной License заполните информацию привязки приложения. После подтверждения услуга вступит в силу.

> **Note:**Информация привязки может быть заполнена только один раз. Если допущена ошибка, пожалуйста, свяжитесь с Tencent Cloud для [отправки тикета](https://console.intl.cloud.tencent.com/workorder/category) для изменения.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cb4366eca70c11ef804852540055f650.png)

### Обновление с пробной версии на официальную версию

#### Step 1: Обновление до официальной License

На странице нажмите **Upgrade to Official License** для целевой пробной License для обновления и покупки License.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/246801dba01011efa0b3525400bdab9d.png)

> **Note:**После обновления до формальной License информацию привязки невозможно будет изменить. Пожалуйста, проверьте и подтвердите информацию привязки перед покупкой.После покупки официальной License пробный период автоматически завершится.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/2ee7b59ba01011ef992f52540075b605.png)

#### Step 2: Выбор типа выставления счетов и срока

Нажмите **Purchase Prepaid SDK**, затем вы можете выбрать покупку на ежемесячной или годовой основе. Нажмите **Next** для входа на страницу подтверждения заказа. После оплаты License будет обновлена до официального издания.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/52f28e72a01011ef834b525400f69702.png)

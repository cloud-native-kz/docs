# Оценка BD-Rate

Функция видеооценки в настоящее время поддерживает оценку качества видео и оценку BD-rate. Вы можете создавать задачи оценки на странице видеооценки консоли, сравнивать качество транскодирования видео и эффекты шаблонов транскодирования, а также поддерживает представление результатов визуализированных данных. Сведения о стандартах ценообразования соответствующих функций см. в разделе [Обзор выставления счетов](https://www.tencentcloud.com/document/product/1041/49204).

## Сценарии использования

### Оценка эффективности шаблона транскодирования

Шаблоны транскодирования Media Processing Service состоят из различных параметров, которые позволяют выбрать оптимальные параметры транскодирования посредством оценки BD-rate. Выберите исходное видео и шаблоны транскодирования с различными параметрами для оценки BD-rate шаблонов транскодирования. На основе как минимум четырех сеансов транскодирования сравните данные оценки, чтобы выбрать шаблон транскодирования, соответствующий вашим требованиям. Результаты оценки могут создавать таблицу оценок BD-rate и диаграмму кривой оценки между различными шаблонами транскодирования.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8e3658d2014011efbf03525400ae4d13.png)

## Инструкции

1. Создание задачи оценки

Перейдите на страницу [видеооценки](https://console.intl.cloud.tencent.com/mps/evaluation?tab=bd) консоли Media Processing Service и нажмите на кнопку **Создать задачу сравнительной оценки BD-Rate**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/3fc5ddf0950011efa50f52540075b605.png)

  1.1. Выберите исходное видео, которое будет использоваться как исходное видео при оценке. Поддерживается выбор видеофайлов из хранилища COS и доступ к видео через URL. Если вы используете локальный видеофайл, загрузите его в COS, а затем выберите через метод COS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8f210fca950011ef9dcd525400f69702.png)

> **Примечание:** Загрузка локальных файлов в хранилище COS Storage повлечет за собой соответствующие расходы на хранение.

  1.2. Выберите шаблоны транскодирования. Каждый выбранный шаблон будет выполнять одно транскодирование исходного видео, и результаты транскодирования будут оценены. Каждая оценка BD-rate позволяет выбрать от 2 до 4 шаблонов транскодирования.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ca559ff7950011ef810152540055f650.png)

  1.3. Выберите критерии оценки. Вы можете выбрать один из методов оценки: PSNR, SSIM, VMAF или VMAF-NEG.
  1.4. Выберите параметры оценки. Параметры оценки по умолчанию используют параметры качества видео (CRF = 18, 24, 29, 34), но вы также можете выбрать пользовательские параметры битрейта или параметры качества видео.

В сценарии пользовательских параметров определения необходимо выбрать от 4 до 8 параметров. При расчете счета результат транскодирования каждого параметра оценки будет рассчитан отдельно для количества кадров оценки.

Пример: При исходном видео из 1000 кадров, выборе 2 шаблонов транскодирования, использовании PSNR в качестве стандарта оценки и одновременном выборе 5 пользовательских параметров определения битрейта, стоимость, создаваемая задачей оценки, составляет:

(1000 кадров x 2 шаблона) * 0,105 USD / 1000 кадров * 1 тип метода оценки * 5 параметров оценки = 1,05 USD

![](https://staticintl.cloudcachetci.com/cms/backend-cms/45ab3eab950111efafd5525400d5f8ef.png)

  1.5. Во время процесса оценки также поддерживается добавление файлов для оценки. Вы можете создать контрольную группу и поместить от 4 до 8 видео, созданных из одного шаблона транскодирования, в контрольную группу. Каждая задача оценки поддерживает не более 4 контрольных групп.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cafdf584950111ef8bbd525400fdb830.png)

  1.6. Выберите место хранения результатов. Каждое видео результата транскодирования будет сохранено в этом пути, и эти видео не будут удаляться при удалении задачи оценки. Пожалуйста, помните об очистке вовремя.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/db6b0e1d950111efb488525400a9236a.png)

2. Просмотр результатов оценки

После создания задачи оценки запись о задаче будет создана на странице оценки качества видео. После успешного выполнения задачи оценки вы можете щелкнуть, чтобы просмотреть отчет о результатах оценки в деталях задачи.

> **Примечание:** Период сохранения задач оценки составляет 7 дней. Записи оценки, превышающие 7 дней, будут удалены. Пожалуйста, загрузите результаты отчета оценки вовремя. Если ни одно из сравниваемых видео не может создать результаты оценки, отчет оценки не будет доступен для просмотра.

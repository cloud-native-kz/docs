# Оценка качества видео

Функция оценки видео в настоящее время поддерживает оценку качества видео и оценку BD-rate. Вы можете создавать задачи оценки на странице оценки видео в консоли, сравнивать качество транскодирования видео и эффекты шаблонов транскодирования, а также поддерживает представление визуализированных результатов данных. Информацию о стандартах ценообразования соответствующих функций см. в [Обзоре биллинга](https://www.tencentcloud.com/document/product/1041/49204).

## Варианты использования

### Оценка качества транскодированного видео

Выберите исходное видео и видео для сравнения, затем оцените оценку качества видео для видео для сравнения относительно исходного видео с помощью оценки PSNR, SSIM и VMAF. Источники видео можно выбрать из хранилища COS, получить доступ по URL или загрузить локально. Результаты оценки могут генерировать общую оценку, диаграмму линий оценок по кадрам и снимок кадра с наихудшим качеством.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/489b2cc9f1b911eebd84525400c50087.png)

## Инструкции

1. Создание задачи оценки

Перейдите на страницу [Оценка видео](https://console.intl.cloud.tencent.com/mps/evaluation?tab=video) консоли Сервиса обработки медиа и нажмите на кнопку **Создать задачу оценки качества видео**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/26db8a4c94fd11efafd5525400d5f8ef.png)

  1.1. Выберите исходное видео; это видео будет использоваться в качестве эталонного видео при оценке. Вы можете выбрать видеофайлы из хранилища COS, получить доступ к видео по URL или, если используется локальный видеофайл, загрузите файл в COS и затем выберите его через COS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/616dac1df1b911eeae1c525400b554aa.png)

> **Примечание:** Частота кадров исходного видео должна совпадать с частотой кадров видео для сравнения, поскольку оценка проводится в поэлементном режиме по кадрам. Если частоты кадров различаются, содержимое оцениваемых кадров не будет совпадать, и результаты не будут релевантными; для пользователей, выбирающих доступ к видео по URL, убедитесь, что URL действительный и позволяет загружать содержимое через Интернет; в противном случае доступ к видео не удастся; загрузка локальных файлов в хранилище COS будет влечь соответствующие затраты на хранение.

  1.2. При выборе видео для сравнения каждое из них будет сравниваться с исходным видео во время оценки, максимум 10 видео для сравнения разрешено для каждой оценки. Видео для сравнения можно выбрать из хранилища COS или получить доступ по URL. Если используется локальный видеофайл, загрузите файл в COS и затем выберите его через COS.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/ab1a3e7694fd11ef8a1b525400bdab9d.png)

Видео для сравнения также можно создать, инициировав задачу транскодирования. Пользователи могут выбрать шаблон транскодирования для создания видео путем инициирования задач транскодирования.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6cef4e3294fe11ef810152540055f650.png)

> **Примечание:** Выбор создания через шаблон транскодирования будет влечь затраты на транскодирование в соответствии с правилами биллинга для транскодирования; в настоящее время невозможно смешивать и комбинировать несколько способов создания видео для сравнения, таких как хранилище COS, URL, локальная загрузка и шаблоны транскодирования. Видео для сравнения для каждой оценки могут поступать только из одного из этих способов.

  1.3. При выборе методов оценки можно выбрать по крайней мере один из PSNR, SSIM или VMAF. При биллинге каждый метод оценки будет рассчитываться отдельно на основе количества кадров оценки.

Например: если исходное видео содержит 1000 кадров, 2 видео для сравнения выбраны из COS, а методы оценки PSNR, SSIM и VMAF отмечены как методы оценки, стоимость задачи оценки составит:

(1000 кадров x 2) * 0,105 USD / 1000 кадров * 3 метода оценки = 0,63 USD

![](https://staticintl.cloudcachetci.com/cms/backend-cms/08c71ea994ff11efa3675254002693fd.png)

  1.4. При выборе места хранения для результатов этот путь будет хранить снимки экрана 10 кадров с наибольшей разницей в качестве от исходного видео для каждого метода оценки для каждого видео для сравнения. Изображения снимков экрана не будут удалены при удалении задачи оценки, пожалуйста, своевременно очистите.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/21144adf94ff11ef810152540055f650.png)

2. Просмотр результатов оценки

После создания задачи оценки на странице Оценка качества видео будет создана запись задачи. После успешного выполнения задачи оценки вы можете нажать для просмотра Отчета о результатах оценки.

> **Примечание:** Период хранения задач оценки составляет 7 дней. Записи оценки, превышающие 7 дней, будут удалены, пожалуйста, своевременно загрузите результаты отчета об оценке; если ни одно из видео для сравнения не может сгенерировать результаты оценки, отчет об оценке не будет доступен для просмотра.

Отчет об оценке содержит следующее содержимое.

  2.1. Просмотр общих оценок видео для сравнения, поддержка просмотра снимков экрана и параллельного сравнения видео.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6cb6827ff1ba11eeb6df5254005c8f7c.png)

  - Нажмите для просмотра снимка экрана, чтобы отобразить изображение кадра с наибольшей разницей в качестве картинки при транскодировании каждого видео для сравнения, отображающее время кадра и соответствующую оценку.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/e2c42c6bf1b911ee941952540085f884.png)

  - Нажмите на одновременное сравнение видео для одновременного воспроизведения нескольких видео, поддерживая синхронное изменение размера нескольких видео.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0c97283cf1ba11eeae1c525400b554aa.png)

  2.2. Просмотр графиков оценки видео, поддержка сравнения видео с помощью фильтрации и сортировки.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/23248032f1ba11eeae1c525400b554aa.png)

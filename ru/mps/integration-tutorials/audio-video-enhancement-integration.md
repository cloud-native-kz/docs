# Интеграция улучшения аудио/видео

## I. Обзор функции улучшения аудио/видео

### Обзор

Функция улучшения аудио/видео опирается на ведущие модели обработки аудио/видео AI от MPS и его богатый опыт накопления данных для обеспечения профессионального уровня решений по улучшению аудио/видео. Эта функция поддерживает распределённое улучшение качества изображения в реальном времени, такое как удаление артефактов видео, подавление шума, улучшение цвета, улучшение лица, SDR2HDR и улучшение с использованием больших моделей, что может значительно улучшить качество аудио и видео. Она широко используется в сценариях OTT, электронной коммерции, спортивных событий и других, эффективно достигая двумерного улучшения QoE и QoS, создавая значительную деловую ценность.

#### **Технические преимущества**

- Алгоритм AI улучшения для полного спектра сценариев: это ведущий в отрасли алгоритм AI улучшения, специально разработанный для различных сценариев, таких как игры, контент UGC, высокодефиниционные фильмы и телепередачи PGC, онлайн-образование, прямые трансляции, электронная коммерция и старые кинематографические источники, комплексно улучшающий качество аудио и видео.
- Комплексное улучшение аудио: поддерживает подавление шума голоса, разделение аудио, улучшение качества аудио и выравнивание громкости, значительно улучшая четкость и качество звука, удовлетворяя спросу на высокачественное аудио в различных сценариях.

### Как использовать функцию улучшения аудио/видео

#### (1) Примечание перед использованием доступ к интерфейсу

Перед использованием этой функции необходимо выполнить следующие предварительные операции:

- Зарегистрируйтесь/войдите в учётную запись Tencent Cloud, активируйте MPS и завершите авторизацию роли сервиса.
- Если вы используете дополнительную учётную запись Tencent Cloud, убедитесь, что учётная запись имеет достаточные разрешения для использования MPS.

Для подробного руководства см. [Быстрый старт](https://www.tencentcloud.com/document/product/1041/33482). Для вопросов авторизации учётной записи см. [Авторизация учётной записи](https://www.tencentcloud.com/document/product/1041/69220).

#### (2) Создание задач улучшения аудио/видео

Сервис обработки медиа Tencent Cloud (MPS) предоставляет три метода инициирования задач: быстрое создание задач через консоль, инициирование задач через API и автоматическое срабатывание задач. Следующая блок-схема показывает общий процесс операции для каждого метода инициирования задач. Для примера того, как инициировать конкретную задачу, см. подробные инструкции в разделе [Создание задач улучшения аудио/видео](https://www.tencentcloud.com/document/product/1041/70463#2d1448ea-975d-44be-857b-ed53ac9cec03) ниже.

## II. Создание задач улучшения аудио/видео

### Метод 1. Быстрое создание задач через консоль

1. Введите веб-страницу консоли MPS [Создать задачу](https://console.tencentcloud.com/mps/tasks/create/vod), выберите узел "улучшение аудио и видео" и выберите шаблон улучшения из всплывающего интерфейса справа. Вы можете выбрать предустановленный шаблон или пользовательский шаблон. Для подробного руководства по конфигурации см. [шаблон улучшения аудио/видео](https://www.tencentcloud.com/document/product/1041/48789).
2. Укажите путь входного файла и путь вывода, затем нажмите "Создать" для инициирования задачи.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a524bffcef8811f09d46525400a31896.png)

### Метод 2. Инициирование задачи через API

Вызовите API [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640), настройте пути входа и выхода и введите ID шаблона в `MediaProcessTask -> TranscodeTaskSet -> Definition` для инициирования задачи. Пример:

```
{	"InputInfo": {//Путь входного файла, поддерживает источники такие как COS, URL		"Type": "URL",		"UrlInputInfo": {			"Url": "xxxxx" 		}	},	"OutputStorage": { //Путь вывода, поддерживает COS, VODPro		"Type": "COS",		"CosOutputStorage": {			"Bucket": "xxx",			// Пример: media-1300111			"Region": "xxx" // например ap-beijing		}	},	"OutputDir": "/output/",	"MediaProcessTask": {		"TranscodeTaskSet": [{			"Definition": 100910,             // Обязательно. 100910 - это ID предустановленного шаблона. Замените на ID вашего пользовательского шаблона улучшения аудио/видео.			"OverrideParameter": { // Параметр переопределения, переопределяет соответствующие значения параметров в шаблоне, может быть удалён если не требуется				"Container": "m4a",				// Например, укажите вывод как m4a				"AudioTemplate": {					"Codec": "aac",					"Bitrate": 64,					"SampleRate": 44100,					"AudioChannel": 2				}			},			"OutputObjectPath": "{inputName}_transcode_{definition}.{format}" // Дополнительно. Укажите имя файла вывода		}]	},	"TaskNotifyConfig": {		"NotifyType": "URL",		"NotifyUrl": "xxx" // Дополнительно. URL обратного вызова	}}
```

> **Примечание:**Рекомендуется использовать[API Explorer](https://console.tencentcloud.com/api/explorer?Product=mps&Version=2019-06-12&Action=ProcessMedia)для быстрой отладки и проверки.**![](https://staticintl.cloudcachetci.com/cms/backend-cms/3d164534f73f11f0b34b525400ecee81.png)

### Метод 3. Автоматическое срабатывание задачи MPS после загрузки файла в COS

Если вы хотите загрузить видеофайл в бакет COS и достичь автоматического улучшения аудио/видео в соответствии с предустановленными параметрами, вы можете:

1. Щелкните **Сохранить оркестрацию** при создании задачи и настройте параметры, такие как срабатывающий бакет и срабатывающая директория во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7b4636364aa511f085275254001c06ec.png)

2. Затем перейдите в список **Автономная оркестрация**, найдите новую оркестрацию и включите переключатель на **Запуск**. После этого любые видеофайлы, добавленные в срабатывающую директорию, будут автоматически инициировать задачи в соответствии с предустановленным процессом и параметрами оркестрации, а обработанные видеофайлы будут сохранены в путь вывода, настроенный в оркестрации.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/6bfc3571f73f11f0831c52540073fd3b.png)

> **Примечание:**После включения оркестрации требуется 3-5 минут для её вступления в силу.

## III. Запрос результатов задачи

### Запрос результата задачи в консоли

Перейдите в консоль [Управление автономными задачами](https://console.tencentcloud.com/mps/tasks/vod-list), и только что инициированная задача будет отображена в списке задач.

Когда статус подзадачи «успешна», вы можете просмотреть, загрузить или перейти в соответствующую директорию бакета COS для выходного файла.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/d190a4d9f74211f09fbf525400370dda.png)

### Обратный вызов задачи

При инициировании задачи MPS с использованием [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640), вы можете установить информацию обратного вызова через параметр `TaskNotifyConfig`. После завершения обработки задачи результат задачи будет передан обратно через настроенную информацию обратного вызова. Вы можете анализировать результат уведомления о событии через [ParseNotification](https://www.tencentcloud.com/document/product/1041/33679).

### Запрос результата задачи путём вызова API

Вызовите API [DescribeTaskDetail](https://www.tencentcloud.com/document/product/1041/33644) и введите ID задачи (например, 24000022-ScheduleTask-774f101xxxxxxx1tt110253) для запроса результата задачи.

## IV. Часто задаваемые вопросы

### Как создать шаблон улучшения аудио/видео

Система предоставляет несколько предустановленных шаблонов улучшения для вашего выбора. Вы также можете создать пользовательские шаблоны улучшения аудио/видео в соответствии с бизнес-требованиями, предустанавливая различные параметры обработки для различных сценариев приложений для облегчения последующего повторного использования. Вы можете создать шаблон улучшения аудио/видео через консоль и API. Для подробного руководства по конфигурации см. [Шаблон улучшения аудио/видео](https://www.tencentcloud.com/document/product/1041/48789#32342eb8-8d70-405b-9315-f01302960a21).

### Поддерживает ли функция улучшения настройку параметров, связанных с кодированием, такой как битрейт и GOP?

Шаблон улучшения поддерживает выбор стандартного транскодирования или TSC транскодирования и позволяет настраивать параметры, связанные с кодированием.

### **Как достичь лучшего эффекта улучшения?**

Улучшение аудио/видео включает такие функции, как удаление артефактов видео, подавление шума, улучшение цвета, улучшение деталей, улучшение лица и SDR2HDR. Если вы всё ещё не удовлетворены результатами ваших собственных комбинированных тестов, свяжитесь с нами напрямую, чтобы получить подробные рекомендации по конфигурации и выполнить расширенную настройку параметров.

### Обзор выставления счётов

Улучшение аудио/видео реализуется на основе транскодирования. Поэтому инициирование задачи улучшения аудио/видео будет влекомых две платы: улучшение аудио/видео и транскодирование аудио/видео (стандартное транскодирование или TSC транскодирование). Подробные сведения см. в разделе [Выставление счётов за улучшение аудио/видео](https://www.tencentcloud.com/document/product/1041/49204#audio.2Fvideo-enhancement).

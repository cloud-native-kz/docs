# Интеграция цифровых и видимых водяных знаков

## Введение в функцию водяных знаков

### Обзор

![](https://staticintl.cloudcachetci.com/cms/backend-cms/1e2f676cc45c11f0a0935254007c27c5.png)

Сервис обработки медиа (MPS) поддерживает следующие типы водяных знаков:

| Тип | Описание |
| --- | --- |
| Цифровой водяной знак | Позволяет встраивать пользовательскую текстовую информацию, невидимую невооруженному глазу, в видео без влияния на визуальное качество и целостность. Поддерживаются как базовый цифровой водяной знак, так и цифровой водяной знак NAGRA NexGuard. |
| Видимый водяной знак | Поддерживает добавление изображений в указанных форматах, таких как PNG и JPG, или пользовательского текста к видео в видимой форме. Это решение подходит для сценариев, таких как защита авторских прав и продвижение бренда. |

### Предварительные условия интеграции

Перед использованием этой функции необходимо выполнить следующие подготовительные операции:

- Зарегистрируйтесь/войдите в аккаунт Tencent Cloud, активируйте MPS и завершите авторизацию роли сервиса**.**
- Если вы используете дополнительный аккаунт Tencent Cloud, убедитесь, что аккаунт имеет достаточные разрешения для использования MPS.

Для получения подробного руководства см. [Быстрый старт](https://www.tencentcloud.com/document/product/1041/33482). Для проблем с авторизацией аккаунта см. [Авторизация аккаунта](https://www.tencentcloud.com/document/product/1041/69220).

## Цифровой водяной знак

Цифровой водяной знак MPS поддерживает две версии: **базовый водяной знак авторских прав** и **водяной знак судебной экспертизы NAGRA NexGuard**.

| Версия | Описание |
| --- | --- |
| Базовый водяной знак авторских прав | Базовая версия: встраивает невидимую текстовую информацию в видео без влияния на визуальное качество и целостность. |
| Водяной знак судебной экспертизы NAGRA NexGuard | NAGRA NexGuard - это ведущее решение для цифровых водяных знаков в мире, отличающееся высокой скрытностью и обладающее возможностями защиты от удаления трансляционного уровня. Оно адаптировано для отслеживания пиратства и защиты авторских прав на высокоценный контент, такой как фильмы и сериалы. |

### Инструкции по выставлению счетов

#### Добавление цифрового водяного знака

"Добавление цифрового водяного знака" взимается на основе продолжительности выходного видео. Подробные сведения о ценах см. в [Инструкциях по выставлению счетов](https://www.tencentcloud.com/document/product/1041/49204#b68ac422-cfed-4dbb-852c-95aa418d61cc). Поскольку функция "добавление цифрового водяного знака" зависит от функций "кодирование аудио/видео" или "улучшение аудио/видео", вам необходимо связать шаблон цифрового водяного знака при инициировании задачи кодирования или улучшения для реализации добавления цифрового водяного знака. Поэтому будут начисляться комиссии за обе функции "добавление цифрового водяного знака" и "кодирование аудио/видео"/"улучшение аудио/видео".

#### Извлечение цифрового водяного знака

Извлечение водяного знака взимается на основе количества вызовов, при этом каждая задача извлечения поддерживает только одно видео. Подробные сведения о ценах см. в [Инструкциях по выставлению счетов](https://www.tencentcloud.com/document/product/1041/49204#b68ac422-cfed-4dbb-852c-95aa418d61cc).

### Создание шаблона цифрового водяного знака

1. Войдите в **консоль MPS** и перейдите на страницу [**шаблона водяного знака**](https://console.tencentcloud.com/mps/templates/watermarks?tab=implicitWatermark).
2. Нажмите **Создать шаблон водяного знака**, выберите тип водяного знака и введите имя шаблона и содержимое водяного знака для создания шаблона цифрового водяного знака.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/20a61d23c45b11f0aa02525400e889b2.png)

> **Примечание:**Каждый пользователь может создать до 50 шаблонов водяных знаков судебной экспертизы NexGuard. После создания пользователи не могут изменять или удалять их. Для изменения или удаления шаблонов свяжитесь с нами для обработки в фоновом режиме.

### Инициирование задач добавления цифрового водяного знака

Функция добавления цифрового водяного знака должна зависеть от функции кодирования аудио/видео. Вы можете добавить цифровой водяной знак, связав шаблон цифрового водяного знака при инициировании задачи кодирования.

Войдите в **консоль MPS**, перейдите на страницу [**Создать задачу**](https://console.tencentcloud.com/mps/tasks/create), создайте задачу автономной обработки файлов, выберите узел кодирования аудио/видео, включите функцию цифрового водяного знака на всплывающей панели параметров и свяжите шаблон цифрового водяного знака.

> **Примечание:**При использовании функции [улучшения аудио/видео](https://www.tencentcloud.com/document/product/1041/70463) вы также можете включить функцию добавления цифрового водяного знака.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/37ddfcd1c45b11f0b35752540099c741.png)

Если вызывается API [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640), пример параметра выглядит следующим образом:

```
{  "InputInfo": {    "Type": "URL",    "UrlInputInfo": {      "Url": "https://test-1234567.cos.ap-guangzhou.myqcloud.com/video/test.mp4"// Replace it with the video URL to be processed.    }  },  "OutputStorage": {    "Type": "cos",    "CosOutputStorage": {      "Bucket": "test-1234567",      "Region": "ap-guangzhou"    }  },  "MediaProcessTask": {    "TranscodeTaskSet": [      {        "Definition": 101005, //101005 is the preset template ID for video transcoding. It can be replaced with the custom audio/video transcoding template ID you create in the console.        "BlindWatermark": {          "Definition": "1234" //Enter the custom digital watermark template ID you create in the console. 1234 is an example and has no practical significance.        }      }    ]  }}
```

### Автоматическое запуск задач

Если вы хотите загрузить видеофайл в корзину облачного хранилища объектов (COS) и добиться автоматического кодирования аудио/видео MPS и добавления цифрового водяного знака в соответствии с предустановленными параметрами, вы можете:

1. Нажмите "Сохранить эту оркестровку" при создании задачи и настройте параметры, такие как корзина и каталог запуска, во всплывающем окне.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/7123de37c45b11f0a0935254007c27c5.png)

2. Затем введите [список автономных оркестровок](https://console.tencentcloud.com/mps/workflows/vod), найдите вновь созданную оркестровку и включите переключатель в столбце.
3. Для видеофайлов, добавленных впоследствии в каталог запуска, задачи будут инициированы автоматически в соответствии с предустановленным процессом и параметрами оркестровки, а обработанные видеофайлы сохраняются в путь вывода, настроенный оркестровкой.

> **Примечание:**После включения оркестровка вступает в силу в течение 3-5 минут.

### Извлечение цифрового водяного знака

Благодаря обнаружению и извлечению цифровых водяных знаков в передаваемом видео можно идентифицировать личность плагиатора, что обеспечивает ключевую основу для защиты авторских прав.

> **Примечание:**В консоли в настоящее время поддерживается только извлечение "базовых водяных знаков авторских прав". Для извлечения "водяных знаков судебной экспертизы NexGuard" [отправьте заявку](https://console.tencentcloud.com/workorder/category) для связи с нами и приложите видеофайл, связанный с подозрением на несанкционированное воспроизведение.

Войдите в **консоль MPS**, перейдите на страницу [**Извлечение цифрового водяного знака**](https://console.tencentcloud.com/mps/extract_watermark), выберите видео для извлечения водяного знака и нажмите **Извлечь водяной знак**.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/868d9720c45b11f0aa02525400e889b2.png)

При вызове API [ExtractBlindWatermark](https://www.tencentcloud.com/document/product/1041/74705) для инициирования задачи, ниже приведен пример параметра:

```
{    "Type": "blind-basic",    "InputInfo": {        "Type": "COS",        "CosInputInfo": {            "Bucket": "mycos-123456789",            "Region": "ap-nanjing",            "Object": "/mps_output/blind_watermark/test.mp4"        }    }}
```

## Видимый водяной знак

### Инструкции по выставлению счетов

За добавление видимого водяного знака не взимаются дополнительные комиссии.

### Создание шаблона видимого водяного знака

Войдите в **консоль MPS** и перейдите на страницу [**Шаблон водяного знака**](https://console.tencentcloud.com/mps/templates/watermarks) для создания шаблона видимого водяного знака. Поддерживаются водяные знаки изображения и текста.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/cbff067cc45b11f0aa02525400e889b2.png)

### Инициирование задач добавления видимого водяного знака

Функция добавления видимого водяного знака должна зависеть от функции кодирования аудио/видео. Вы можете добавить видимый водяной знак, связав шаблон видимого водяного знака при инициировании задачи кодирования.

Войдите в **консоль MPS**, перейдите на страницу [**Создать задачу**](https://console.tencentcloud.com/mps/tasks/create), создайте задачу автономной обработки файлов, выберите узел кодирования аудио/видео, включите функцию видимого водяного знака на всплывающей панели параметров и свяжите шаблон видимого водяного знака.

Можно добавить до 10 видимых изображений или текстовых водяных знаков.

> **Примечание:**При использовании функции [улучшения аудио/видео](https://www.tencentcloud.com/document/product/1041/70463) или функции снятия скриншотов вы также можете включить функцию добавления видимого водяного знака.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/f301bd68c45b11f09c26525400bf7822.png)

Если вызывается API [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640), пример параметра выглядит следующим образом:

```
{  "InputInfo": {    "Type": "URL",    "UrlInputInfo": {      "Url": "https://test-1234567.cos.ap-guangzhou.myqcloud.com/video/test.mp4"// Replace it with the video URL to be processed.    }  },  "OutputStorage": {    "Type": "cos",    "CosOutputStorage": {      "Bucket": "test-1234567",      "Region": "ap-guangzhou"    }  },  "MediaProcessTask": {    "TranscodeTaskSet": [      {        "Definition": 101005, //101005 is the preset template ID for video transcoding. It can be replaced with the custom audio/video transcoding template ID you create in the console.        "WatermarkSet": [          {            "Definition": 1234 //Enter the custom watermark template ID you create in the console. 1234 is an example and has no practical significance.          }        ]      }    ]  }}
```

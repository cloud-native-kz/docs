# Универсальный доступ к переводу видео

## Введение в функцию перевода видео с уровнем дубляжа

Функция перевода видео интегрирует несколько мощных атомарных возможностей MPS, **включая удаление субтитров, извлечение, перевод, внедрение и ИИ-дубляж, обеспечивая комплексное решение для всего рабочего процесса перевода**. Дополнительные сведения о тарификации см. в разделе [Инструкции по выставлению счетов](https://www.tencentcloud.com/en/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf). Сервис поддерживает следующие два решения:

- **Перевод видео на уровне субтитров**: удаление + извлечение субтитров + перевод субтитров. Автоматически определяет исходные субтитры в видео, выполняет точное удаление и перевод на основе больших моделей, затем без проблем встраивает переведенные субтитры в видео.
- **Перевод видео с уровнем дубляжа**: в дополнение к переводу на уровне субтитров это решение обеспечивает естественный и плавный ИИ-дубляж, создавая новое видео с ИИ-переводом субтитров и закадровым голосом на целевом языке.

## Интеграция перевода видео с уровнем дубляжа

### Предварительные требования

Перед доступом к Smart Erase необходимо выполнить следующие предварительные требования для использования MPS: зарегистрировать и войти в учетную запись Tencent Cloud, активировать MPS и предоставить разрешения ролям сервиса.

Подробное руководство см. в разделе [Быстрый старт](https://www.tencentcloud.com/document/product/1041/33482). По вопросам авторизации учетной записи см. раздел [Авторизация учетной записи](https://www.tencentcloud.com/document/product/1041/69220).

### Инициирование задачи перевода видео с уровнем дубляжа

#### Способ 1: инициирование задачи в консоли

1. Перейдите на страницу [Создать задачу](https://console.tencentcloud.com/mps/tasks/create/vod) в консоли и последовательно укажите путь входного файла, рабочий процесс оркестровки и путь вывода.
2. В конфигурации оркестровки выберите узел Media AI - Intelligent Analysis.
3. На появившейся странице справа выберите предустановленный шаблон 25, включите **Расширенные параметры** в разделе Дополнительные параметры, а затем заполните необходимые параметры в ExtendedParameter на основе [Описания ExtendedParameter](#extendedpara) в следующем разделе.

> **Примечание:** функция базовой редакции интеллектуального стирания - удаление водяного знака используется по умолчанию при выборе предустановленного шаблона 25. Если вам нужно инициировать задачу перевода видео, вы должны передать параметры в ExtendedParameter. В противном случае MPS выполняет только удаление водяного знака на видео. Консоль MPS автоматически экранирует данные JSON. Не заполняйте экранированные строки. В противном случае задача не выполнится.

## Описание ExtendedParameter

Установите ExtendedParameter соответствующим образом в зависимости от вариантов использования перевода видео. В этом разделе приведены распространенные сценарии.

### Сценарий 1: извлечение и перевод субтитров OCR + удаление исходных субтитров + внедрение новых субтитров + ИИ-дубляж с клонированным тембром (наиболее распространенный сценарий)

Субтитры и дубляж видео переводятся на указанный язык для создания нового видео с субтитрами и дубляжом на целевом языке. Содержание субтитров и дубляжа получено из OCR-извлечения текста (извлечение исходного языкового текста из видео). Для видео с исходными языковыми субтитрами на экране рекомендуется выполнить настройку в соответствии с этим сценарием.

#### Инструкции по выставлению счетов

Для использования возможностей в этом сценарии вам будут выставлены счета за три пункта выставления счетов: удаление субтитров, извлечение и перевод субтитров на основе OCR + внедрение субтитров и замену текста в речь (TTS) (клонированный тембр).

Если внедрение субтитров отключено, вам будут выставлены счета за три пункта: удаление субтитров, извлечение и перевод субтитров на основе OCR и замену TTS (клонированный тембр).

Дополнительные сведения о тарификации см. в разделе [Инструкции по выставлению счетов](https://www.tencentcloud.com/en/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

#### Передача параметров в **ExtendedParameter**

```
{    "delogo": {        "cluster_id": "gpu_pre",        "CustomerAppId": "audio_clone_ocr"    }}//Если вам нужно напрямую вызвать API или использовать режим JSON API Explorer, используйте экранированную версию: {\\"delogo\\":{\\"cluster_id\\":\\"gpu_pre\\",\\"CustomerAppId\\":\\"audio_clone_ocr\\"}}.
```

#### Указание языка

По умолчанию язык переводится с упрощенного китайского на английский. Если вам нужно использовать другие языки, передайте параметры `translate_src_language` и `translate_dst_language` в ExtendedParameter. Поддерживаемые языки и соответствующие коды см. в разделе [Поддерживаемые языки для перевода видео](#language). Пример:

```
{    "delogo": {        "cluster_id": "gpu_pre",        "CustomerAppId": "audio_clone_ocr",        "subtitle_param": {            "translate_src_language": "de", //Немецкий            "translate_dst_language": "id" //Индонезийский        }    }}/*Если вам нужно напрямую вызвать API или использовать режим JSON API Explorer, используйте экранированную версию:{\\"delogo\\":{\\"cluster_id\\":\\"gpu_pre\\",\\"CustomerAppId\\":\\"audio_clone_ocr\\",\\"subtitle_param\\":{\\"translate_src_language\\":\\"de\\",\\"translate_dst_language\\":\\"id\\"}}}translate_src_language: указывает исходный язык видео. Если этот параметр не передан, исходный язык по умолчанию — китайский.translate_dst_language: указывает целевой язык для перевода видео. Если этот параметр не передан, целевой язык по умолчанию — английский.*/
```

#### Отключение внедрения субтитров

Если вам не нужно встраивать субтитры целевого языка в видео, вы можете добавить параметр `"use_draw": false`.

```
{    "delogo": {        "cluster_id": "gpu_pre",        "CustomerAppId": "audio_clone_ocr",         "subtitle_param": {            "use_draw": false        }    }}/*use_draw: если этот параметр не передан или имеет значение true, внедрение субтитров включено. Если этот параметр имеет значение false, внедрение субтитров отключено.Если вам нужно напрямую вызвать API или использовать режим JSON API Explorer, используйте экранированную версию:{\\"delogo\\":{\\"cluster_id\\":\\"gpu_pre\\",\\"CustomerAppId\\":\\"audio_clone_ocr\\",\\"subtitle_param\\":{\\"use_draw\\":false}}}*/
```

### Сценарий 2: извлечение и перевод субтитров ASR + внедрение новых субтитров + ИИ-дубляж с клонированным тембром

Обработка в сценарии 2 аналогична обработке в сценарии 1. То есть субтитры и дубляж видео переводятся на указанный язык для создания нового видео с субтитрами и дубляжом на целевом языке. Однако в сценарии 2 содержание субтитров и дубляжа получено из автоматического распознавания речи (ASR) (распознавание содержания исходного голоса в видео и его транскрипция в субтитры). В сценарии 2 удаление субтитров не выполняется автоматически.

Дополнительные сведения о тарификации см. в разделе [Инструкции по выставлению счетов](https://www.tencentcloud.com/en/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

Если внедрение субтитров отключено, вам будут выставлены счета за два пункта: преобразование речи на основе ASR и замену TTS (клонированный тембр). Дополнительные сведения о тарификации см. в разделе [Инструкции по выставлению счетов](https://www.tencentcloud.com/en/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

### Сценарий 3: ИИ-дубляж со стандартным тембром

Функция ИИ-дубляжа для перевода видео поддерживает два типа тембра:

- Стандартный тембр: поддерживает китайский, английский и японский языки. Предоставляет различные варианты мужских, женских и детских голосов. Будет взиматься плата за замену TTS (стандартный тембр).
- Клонированный тембр: восстанавливает характеристики голоса на основе передовой технологии клонирования ИИ-голоса. Будет взиматься плата за замену TTS (клонированный тембр).

По умолчанию в сценариях 1 и 2 используется клонированный тембр. Возможность стандартного тембра находится в стадии обновления и доступна только для бета-тестирования. Если вам нужно использовать эту функцию, обратитесь к менеджеру по продажам или [отправьте заявку](https://console.tencentcloud.com/workorder/category) для получения поддержки.

### Обратный вызов уведомления о событии

Когда вы используете [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640) для инициирования задачи MPS, вы можете указать параметр TaskNotifyConfig для настройки обратного вызова события. После завершения обработки задачи результат задачи будет вызван через настроенную информацию обратного вызова. Вы можете проанализировать результат уведомления о событии с помощью [ParseNotification](https://www.tencentcloud.com/document/product/1041/33679).

### Запрос результата задачи путем вызова API

Когда вы используете [ProcessMedia](https://www.tencentcloud.com/document/product/1041/33640) для инициирования задачи обработки мультимедиа, возвращается идентификатор задачи (TaskId), например 24000022-WorkflowTask-b20a8exxxxxxx1tt110253 и 24000022-ScheduleTask-774f101xxxxxxx1tt110253. Вызовите API [DescribeTaskDetail](https://www.tencentcloud.com/document/product/1041/33644) и введите идентификатор задачи для получения результата задачи. Вам нужно проанализировать поле WorkflowTask ->AiAnalysisResultSet, чтобы получить результат задачи.

## Интеграция перевода видео на уровне субтитров

### Сценарий 1: без удаления, создание субтитров с помощью распознавания и перевода речи ASR

Подробные сведения см. в разделе [Учебник по доступу к интеллектуальным субтитрам](https://www.tencentcloud.com/document/product/1041/54517).

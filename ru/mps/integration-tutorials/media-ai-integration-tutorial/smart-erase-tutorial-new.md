# Учебное пособие Smart Erase (новое)

> **Примечания:** Начиная с сентября 2025 г. модуль Smart Erase добавлен в консоль и API. Это учебное пособие описывает методы доступа к каждой функции на основе шаблона Smart Erase. Для клиентов, которые используют шаблон Intelligent Analysis для доступа к функции стирания, см. документ [Учебное пособие Smart Erase (старое)](https://www.tencentcloud.com/document/product/1041/58269). Модуль Smart Erase поддерживает больше пользовательских конфигураций и более гибкий, чем шаблон Intelligent Analysis. Два модуля отличаются только использованием функций. Они используют одинаковые цены и не влекут за собой корректировок выставления счетов.

## Введение в функцию Smart Erase

Smart Erase позволяет размывать, применять мозаику или бесшовную обработку элементов, таких как логотипы, субтитры, человеческие лица и номерные знаки в видеоизображениях, что облегчает распространение и совместное использование контента. Функция широко используется в нескольких областях, таких как платформы коротких драм, платформы коротких видео, трансграничная электронная коммерция и независимые медиастудии. Информацию о ценах см. в [Инструкции по выставлению счетов Smart Erase](https://www.tencentcloud.com/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

В следующем примере, кроме области автоматического стирания (позиция субтитра), добавлены две дополнительные области стирания (область текста предупреждения в верхнем левом углу и область названия шоу внизу) для обеспечения стирания текста в этих фиксированных областях.

### Удаление субтитров и извлечение субтитров на основе OCR

В дополнение к удалению субтитров поддерживается распознавание текста OCR для извлечения исходных субтитров из видео и их перевода.

> **Примечание:** В сценарии стирания исходное видео обычно поставляется с исходными субтитрами. Использование OCR для извлечения субтитров дает лучшие и более точные результаты. Поэтому функция умного стирания интегрирует только возможности OCR. Если вы хотите создать файлы субтитров с помощью автоматического распознавания речи (ASR), вы можете использовать функцию умного субтитра. Подробности см. в [Учебное пособие по доступу к умному субтитру](https://www.tencentcloud.com/document/product/1041/54517).

#### Обзор выставления счетов

Будут взиматься сборы за удаление субтитров и извлечение субтитров на основе OCR (перевод субтитров не включен) или сборы за удаление субтитров и извлечение и перевод субтитров на основе OCR (перевод субтитров включен). Информацию о ценах см. в [Инструкции по выставлению счетов Smart Erase](https://www.tencentcloud.com/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

#### Пример конфигурации шаблона

Укажите позиции субтитров для извлечения и стирания в области автоматического стирания, включите извлечение субтитров на основе OCR и включите перевод по мере необходимости.

> **Примечания:** Если субтитры на китайском, английском языках или на обоих языках, использование OCR для извлечения может дать наилучший эффект. Распознаются только текст в области автоматического стирания. Текст в других областях не извлекается и не стирается. Если существует несколько текстовых областей в области автоматического стирания, система выбирает только **одну** текстовую область для извлечения и стирания, обычно самую стабильную, которая существует дольше всего.

### Извлечение субтитров на основе OCR

Стирание не выполняется. Субтитры извлекаются и переводятся исключительно на основе распознавания текста OCR.

#### Обзор выставления счетов

Если извлекаются только исходные субтитры, взимается плата за извлечение субтитров на основе OCR. Если перевод субтитров включен, взимаются сборы за извлечение и перевод субтитров на основе OCR. Информацию о ценах см. в [Инструкции по выставлению счетов Smart Erase](https://www.tencentcloud.com/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

#### Пример конфигурации шаблона

Вы можете использовать шаблон 25 функции Intelligent Analysis MPS и инициировать его через расширенные параметры. Подробные инструкции см. в [Учебное пособие по локализации видео на уровне озвучки](https://www.tencentcloud.com/document/product/1041/74090).

### Удаление логотипа

Удаление логотипа означает стирание графического содержимого, такого как логотипы, значки и фотографии профилей пользователей в видеоизображениях. Поддерживает два режима: базовое издание и продвинутое издание.

- Базовое издание: обеспечивает базовый эффект размытия. Это экономичный вариант, подходящий для анимации или видео с чистым фоном. Будет взиматься плата за "базовый сбор smart erase (удаление логотипа - базовое издание)".
- Продвинутое издание: обеспечивает лучший эффект стирания, подходит для видео в стиле реальности, таких как короткие драмы. Будет взиматься плата за "удаление логотипа - продвинутое издание".

![Слева: исходное видео. Красное поле показывает позицию логотипа. В центре: эффект обработки базового издания. Справа: эффект обработки продвинутого издания.](https://staticintl.cloudcachetci.com/cms/backend-cms/a77764af887911f0b321525400e889b2.png)

#### Обзор выставления счетов

Взимаются сборы за "удаление логотипа - продвинутое издание" и "удаление логотипа - базовое издание". Информацию о ценах см. в [Инструкции по выставлению счетов Smart Erase](https://www.tencentcloud.com/document/product/1041/49204#b71b5f78-d20e-4560-8b9e-a4f02f0d75cf).

**Автоматическое стирание:** модели AI используются для автоматического распознавания и стирания логотипов в полнометражных видеоизображениях, поддерживая как статические, так и динамические логотипы.

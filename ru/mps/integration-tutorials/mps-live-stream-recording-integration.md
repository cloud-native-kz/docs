# Интеграция записи прямых трансляций MPS

С помощью Media Processing Service (MPS) вы можете записывать контент прямых трансляций по URL.

## Инструкции

1. На странице [Шаблоны записи прямых трансляций](https://console.tencentcloud.com/mps/templates/lives) создайте новый шаблон записи прямой трансляции. Консоль предоставляет шаблон записи по умолчанию.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/8728c75b65a511ee94c3525400d793d0.png)

2. На странице [Схемы прямых трансляций](https://console.tencentcloud.com/mps/workflows/liverecord) создайте новую схему. На этой странице необходимо выбрать бакет COS и указать каталог для вывода. В диаграмме действий ниже нажмите и добавьте шаг "Запись прямой трансляции".

![](https://staticintl.cloudcachetci.com/cms/backend-cms/0eec822365a611eeabd75254005810a4.png)

3. Нажмите кнопку редактирования на правой стороне шага "Запись прямой трансляции" для выполнения детальной конфигурации.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/5ee9eea265a611eeabd75254005810a4.png)

4. В детальной конфигурации:

(1) Выберите шаблон записи;

(2) Выберите выходной бакет;

(3) Отредактируйте выходной путь. Путь выходного файла может быть относительным или абсолютным.

![](https://staticintl.cloudcachetci.com/cms/backend-cms/a02238a665a611eeabd75254005810a4.png)

> **Примечание:** Обратите внимание, что удаление случайного поля {rand} из пути может привести к перезаписи файла другим файлом с идентичными значениями полей.

5. После заполнения приведённой выше информации нажмите "Создать" для установления схемы прямой трансляции.
6. Перейдите на страницу управления [Задачи прямых трансляций](https://console.tencentcloud.com/mps/tasks/live) и создайте новую задачу. Введите адрес прямой трансляции, который необходимо записать, и выберите схему. Нажмите "Создать задачу".

> **Примечание:** Убедитесь, что адрес прямой трансляции введён правильно. Если не удастся получить прямую трансляцию, будут выполнены 3 повторные попытки. Если прямую трансляцию всё ещё не удастся получить, задача записи не будет выполнена.

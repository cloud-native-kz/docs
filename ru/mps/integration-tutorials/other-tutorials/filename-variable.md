# Переменная имени файла

MPS поддерживает рендеринг целевых путей выходных файлов со следующими переменными:

| Имя переменной | Описание | Инструкция по использованию |
| --- | --- | --- |
| inputName | Имя входного файла. | Применимо для задачи VOD. |
| inputFormat | Формат входного файла. | Применимо для задачи VOD. |
| number | Номер выходного файла. Примечание: эта переменная действует только на выходные файлы .ts, созданные в формате HLS. | Применимо для задачи VOD. |
| taskId | ID задачи. | Применимо для задачи Live. |
| rand | Переменная случайного числа. В задачах записи живого потока она может использоваться для настройки выходного пути. Таким образом, когда другие параметры переменных совпадают, эта переменная случайного числа может использоваться для предотвращения перекрытия нескольких файлов результатов записи. | Применимо для задачи Live. |
| streamId | ID потока в задаче живого потока. | Применимо для задачи Live. |
| format | Формат выходного файла. | Применимо для задачи VOD и Live. |
| definition | ID шаблона параметров. | Применимо для задачи VOD и Live. |

### Пример 1

Если ваши требования к трансодированию выглядят следующим образом:

- Имя входного файла — `AnimalWorldE01.mp4`.
- Используются шаблоны трансодирования 100010, 100020 и 100030.
- Имена выходных файлов — `AnimalWorldE01_100010.mp4`, `AnimalWorldE01_100020.mp4` и `AnimalWorldE01_100030.mp4` соответственно.

Затем при использовании API [ProcessMedia](https://intl.cloud.tencent.com/document/product/1041/33640) для запуска трансодирования:
Вы должны установить параметр `InputInfo.CosInputInfo.OutputObjectPath` как `{inputName}_{definition}.{format}`.

#### Пример 2

Если ваши требования к трансодированию выглядят следующим образом:

- Имя входного файла — `AnimalWorldE01.mp4`.
- Используется шаблон трансодирования 100210.
- Имя выходного файла .m3u8 — `AnimalWorldE01_from_mp4.m3u8`.
- Имена выходных файлов .ts — `AnimalWorldE01_from_mp4_0.ts`, `AnimalWorldE01_from_mp4_1.ts`, `AnimalWorldE01_from_mp4_2.ts` и так далее.

Затем при использовании API [ProcessMedia](https://intl.cloud.tencent.com/document/product/1041/33640) для запуска трансодирования:

- Вы должны установить параметр `InputInfo.CosInputInfo.OutputObjectPath` как `{inputName}_from_{inputFormat}.{format}`.
- Вы должны установить параметр `InputInfo.CosInputInfo.SegmentObjectName` как `{inputName}_from_{inputFormat}_{number}.{format}`.

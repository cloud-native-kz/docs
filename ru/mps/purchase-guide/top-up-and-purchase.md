# Пополнение счета и покупка

## Ежедневная модель оплаты по фактическому использованию

Если вы используете [ежедневный режим оплаты по фактическому использованию](https://www.tencentcloud.com/document/product/1041/49204#daily-billing), плата рассчитывается на основе фактического использования функций MPS. Обращайте внимание на баланс вашего счета и пополняйте его своевременно.

1. Войдите в консоль Tencent Cloud и нажмите [Billing Center](https://console.tencentcloud.com/account/recharge) в правом верхнем углу. В левой боковой панели выберите **Payment Management > Payment**.
![](https://staticintl.cloudcachetci.com/cms/backend-cms/5398c261d6d811ee9409525400c26da5.png)
2. Введите сумму пополнения и нажмите **Pay Now**.

## Ежемесячная модель оплаты по фактическому использованию

Для [ежемесячной оплаты по фактическому использованию](https://www.tencentcloud.com/document/product/1041/49204#monthly-billing) применяются единые цены, указанные в вашем контракте с Tencent Cloud. Для получения дополнительной информации обратитесь к вашему менеджеру по продажам.

## Пакеты предоплачиваемых ресурсов

1. Войдите в [консоль MPS](https://console.tencentcloud.com/mps). На странице обзора нажмите **Buy resource pack**, чтобы перейти на [страницу покупки](https://buy.tencentcloud.com/mps).
2. Выберите нужные пакеты ресурсов.

> **Примечание:** Вы можете приобрести несколько пакетов ресурсов. Сначала будет использован пакет с наиболее ранней датой истечения. Каждый пакет ресурсов имеет собственный период действия. Если вы приобретаете несколько пакетов, их периоды действия не суммируются. Каждый пакет ресурсов MPS может использоваться только для определенного типа ресурсов MPS. После исчерпания всех ваших пакетов ресурсов вы будете автоматически переведены в режим [оплаты по фактическому использованию](https://www.tencentcloud.com/document/product/1041/49204).

# Объявление об официальном внедрении платежей за функции захвата экрана и создания анимированных скриншотов MPS

Функции захвата видео и создания анимированных скриншотов видео MPS официально начнут взиматься с 1 октября 2023 года. Конкретные тарифы и правила выставления счетов следующие:

## Захват экрана

- MPS может выполнять обработку захвата экрана для видео. Использование функции захвата экрана будет взиматься, выставление счетов производится на основе накопленного количества захватов экрана видео в день.
- Файлы скриншотов, создаваемые при захвате экрана видео, хранятся в облачном хранилище объектов (COS). Для получения информации о соответствующих расходах см. [Тарификацию COS](https://www.tencentcloud.com/pricing/cos).

Использование функции захвата экрана будет взиматься, единица выставления счетов: **USD/тысяча**.

**Ниже приведены детали тарификации:**

| Наименование | Единица выставления счетов | Цена (USD за 1000 скриншотов) |
| --- | --- | --- |
| Захват экрана | Количество скриншотов | 0.0176 |

## Детали выставления счетов

- Формула выставления счетов: плата за захват экрана = Количество (тысячи) полученных скриншотов × Единица цены (USD за 1000 скриншотов)
- Количество скриншотов рассчитывается ежедневно и округляется до ближайшей тысячи в большую сторону.
- Способ оплаты: постоплата.
- Цикл выставления счетов: ежедневно. Расходы, понесенные в каждый день, выставляются счетом и вычитаются из баланса вашего счета между 12:00 и 18:00 следующего дня.

## Создание анимированного скриншота

MPS может создавать анимированные скриншоты из видео. Использование этой функции будет взиматься плата за создание анимированного скриншота.

Ниже приведены детали тарификации:

| Выходное разрешение | Цена (USD/мин) |
| --- | --- |
| 4K (короткая сторона ≤ 2160 px) | 0.0421 |
| 2K (короткая сторона ≤ 1440 px) | 0.0206 |
| FHD (короткая сторона ≤ 1080 px) | 0.0095 |
| HD (короткая сторона ≤ 720 px) | 0.0049 |
| SD (короткая сторона ≤ 480 px) | 0.0024 |

Детали выставления счетов

- Формула выставления счетов: плата за создание анимированного скриншота = Длительность скриншота (минуты) × Единица цены (USD/минуту)
- Каждая задача создания скриншота выставляется счетом только один раз. Длительность скриншота округляется до ближайшей минуты в большую сторону.
- Способ оплаты: постоплата.
- Цикл выставления счетов: ежедневно. Расходы, понесенные в каждый день, выставляются счетом и вычитаются из баланса вашего счета между 12:00 и 18:00 следующего дня.

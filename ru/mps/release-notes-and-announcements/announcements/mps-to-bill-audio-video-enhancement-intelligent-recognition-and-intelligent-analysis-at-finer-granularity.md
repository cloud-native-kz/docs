**Объяснение счета**

- Правило выставления счета: платежи основаны на разрешении и продолжительности видео после улучшения с ежедневным циклом выставления счета.
- Формула расчета: плата за обработку видеоулучшения = продолжительность выходного файла (минуты) × цена за единицу выходного файла разного разрешения (USD/минута).
- Для каждой выполненной задачи улучшения видео взимается только один платеж. В пределах цикла выставления счета, если продолжительность менее одной минуты, она будет рассчитана как одна минута.

## Интеллектуальное распознавание контента

Интеллектуальное распознавание контента использует технологию ИИ для распознавания видеоконтента, включая распознавание лиц, распознавание текста, распознавание речи, перевод речи и распознавание объектов. Единица измерения счета — USD/минута.

Подробная информация о счете выглядит следующим образом:

| Статья счета | Цена (USD/минута) |
| --- | --- |
| Распознавание лиц | 0.0088 |
| Распознавание текста | 0.0176 |
| Распознавание речи | 0.0053 |
| Перевод речи | 0.0529 |
| Распознавание объектов | 0.0176 |

**Объяснение счета**

- Формула расчета: плата за интеллектуальное распознавание контента = продолжительность исходного видео (минуты) × цена за единицу интеллектуального распознавания контента (USD/минута).
- Для каждой выполненной задачи интеллектуального распознавания контента взимается только один платеж. В пределах цикла выставления счета, если продолжительность менее одной минуты, она будет рассчитана как одна минута.

## Интеллектуальный анализ контента

Интеллектуальный анализ контента использует технологию видеоискусственного интеллекта и огромную базу данных образцов для анализа и обработки видеоконтента, включая видеотегирование, классификацию видео, умную обложку, тегирование кадр за кадром, сегментацию видео, выделение ярких моментов и анализ вводных и заключительных титров. Единица измерения счета — USD/минута.

| Статья счета | Цена (USD/минута) |
| --- | --- |
| Видеотегирование | 0.0071 |
| Классификация видео | 0.0071 |
| Умная обложка | 0.0071 |
| Тегирование кадр за кадром | 0.0265 |
| Сегментация видео | 0.0071 |
| Выделение ярких моментов | 0.0176 |
| Анализ вводных и заключительных титров | 0.0026 |

**Объяснение счета**

- Формула расчета: плата за интеллектуальный анализ контента = продолжительность исходного видео (минуты) × цена за единицу интеллектуального анализа контента (USD/минута).
- Для каждой выполненной задачи интеллектуального анализа контента взимается только один платеж. В пределах цикла выставления счета, если продолжительность менее одной минуты, она будет рассчитана как одна минута.

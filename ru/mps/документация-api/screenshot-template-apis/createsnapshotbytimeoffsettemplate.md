# CreateSnapshotByTimeOffsetTemplate

## 1. Описание API

Доменное имя для запроса API: mps.intl.tencentcloudapi.com.

Этот API используется для создания пользовательского шаблона снимков экрана в определённые моменты времени. Можно создать до 16 шаблонов.

Для этого API можно инициировать максимум 100 запросов в секунду.

Рекомендуем использовать API Explorer

## 2. Входные параметры

Следующий список параметров запроса содержит только параметры API запроса и некоторые распространённые параметры. Полный список распространённых параметров см. в разделе [Распространённые параметры запроса](https://www.tencentcloud.com/document/api/1041/33628).

| Название параметра | Требуется | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Распространённые параметры](https://www.tencentcloud.com/document/api/1041/33628). Значение для этого API: CreateSnapshotByTimeOffsetTemplate. |
| Version | Да | String | [Распространённые параметры](https://www.tencentcloud.com/document/api/1041/33628). Значение для этого API: 2019-06-12. |
| Region | Нет | String | [Распространённые параметры](https://www.tencentcloud.com/document/api/1041/33628). Этот параметр не требуется для этого API. |
| Name | Нет | String | Название шаблона снимков экрана в определённые моменты времени. Ограничение по длине: 64 символа. |
| Width | Нет | Integer | Ширина изображения в пикселях. Диапазон значений: [128, 4,096]. |
| Height | Нет | Integer | Высота изображения в пикселях. Диапазон значений: [128, 4,096]. |
| ResolutionAdaptive | Нет | String | Адаптация разрешения. Допустимые значения: open: включено. В этом случае `Width` представляет длинную сторону видео, а `Height` — короткую сторону; close: отключено. В этом случае `Width` представляет ширину видео, а `Height` — высоту. Значение по умолчанию: open. |
| Format | Нет | String | Формат изображения. Допустимые значения: jpg (по умолчанию), png, webp. |
| Comment | Нет | String | Описание шаблона. Ограничение по длине: 256 символов. |
| FillType | Нет | String | Тип заполнения. "Fill" — это способ обработки снимка экрана, когда его соотношение сторон отличается от соотношения сторон исходного видео. Поддерживаются следующие типы заполнения: stretch: растяжение. Снимок экрана будет растянут кадр за кадром, чтобы соответствовать соотношению сторон исходного видео, что может сделать снимок экрана «короче» или «длиннее»; black: заполнение чёрным цветом. Этот параметр сохраняет соотношение сторон исходного видео для снимка экрана и заполняет несовпадающую область чёрными блоками. white: заполнение белым цветом. Этот параметр сохраняет соотношение сторон исходного видео для снимка экрана и заполняет несовпадающую область белыми блоками. gauss: заполнение размытием Гаусса. Этот параметр сохраняет соотношение сторон исходного видео для снимка экрана и заполняет несовпадающую область размытием Гаусса. Значение по умолчанию: black. |

## 3. Выходные параметры

| Название параметра | Тип | Описание |
| --- | --- | --- |
| Definition | Integer | Уникальный ID шаблона снимков экрана в определённые моменты времени. |
| RequestId | String | Уникальный идентификатор запроса, генерируемый сервером, будет возвращён при каждом запросе (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId необходим для определения проблемы. |

## 4. Примеры

### Пример 1. Создание шаблона снимков экрана в определённые моменты времени

#### Пример входных данных

```
POST / HTTP/1.1
Host: mps.intl.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: CreateSnapshotByTimeOffsetTemplate
<Common request parameters>

{
    "Width": "540",
    "Format": "jpg",
    "Name": "Time point screenshot template 1",
    "Height": "960"
}
```

#### Пример выходных данных

```json
{
    "Response": {
        "Definition": 1008,
        "RequestId": "12ae8d8e-dce3-4151-9d4b-5594145287e1"
    }
}
```

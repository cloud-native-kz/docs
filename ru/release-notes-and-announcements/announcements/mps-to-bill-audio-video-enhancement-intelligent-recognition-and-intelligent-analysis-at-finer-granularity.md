**Объяснение выставления счетов**

- Правило выставления счетов: Плата рассчитывается на основе разрешения и длительности видео после улучшения с ежедневным циклом выставления счетов.
- Формула выставления счетов: Плата за обработку улучшения видео = Длительность выходного файла (минуты) × Цена единицы для выходного файла разного разрешения (USD/минута).
- Для каждого выполненного задания по улучшению видео взимается только одна плата. В течение цикла выставления счетов, если длительность составляет менее одной минуты, она будет рассчитана как одна минута.

## Интеллектуальное распознавание содержимого

Интеллектуальное распознавание содержимого использует технологию искусственного интеллекта для распознавания видеоконтента, включая распознавание лиц, распознавание текста, распознавание речи, перевод речи и распознавание объектов. Единица выставления счетов — USD/минута.

Конкретные детали выставления счетов выглядят следующим образом:

| Элемент выставления счетов | Цена (USD/минута) |

| Перевод речи | 0.0529 |
| Распознавание объектов | 0.0176 |

**Объяснение выставления счетов**

- Формула выставления счетов: Плата за интеллектуальное распознавание содержимого = Длительность видеоисточника (минуты) × Цена единицы интеллектуального распознавания содержимого (USD/минута).
- Для каждого выполненного задания по интеллектуальному распознаванию содержимого взимается только одна плата. В течение цикла выставления счетов, если длительность составляет менее одной минуты, она будет рассчитана как одна минута.

## Интеллектуальный анализ содержимого

Интеллектуальный анализ содержимого использует технологию видео ИИ и массивную базу данных образцов для анализа и обработки видеоконтента, включая видеотеги, классификацию видео, интеллектуальное изображение обложки, теговку кадр за кадром, сегментацию видео, выделение ключевых моментов и анализ вступления и завершения. Единица выставления счетов — USD/минута.

- Для каждого выполненного задания по интеллектуальному анализу содержимого взимается только одна плата. В течение цикла выставления счетов, если длительность составляет менее одной минуты, она будет рассчитана как одна минута.

# Объявление о начале бета-тестирования конференции с многоабонентским аудио и видео

TRTC Conference（TUIRoomKit）— это продукт, разработанный для сценариев многоабонентского аудио и видео общения, таких как деловые встречи, онлайн-классы и вебинары. Интегрировав этот продукт, вам нужно написать всего несколько строк кода, чтобы добавить функцию многоабонентского аудио и видео общения в ваше приложение, сэкономив 90% времени разработки и быстро запустив свой бизнес. Дисплей базовых функций показан на следующем изображении:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9d93360651fc11ee974d5254005f490f.png)

## Инструкции по внутреннему тестированию

Продукт в настоящее время находится на стадии бета-тестирования, **интерфейс открыт с исходным кодом и бесплатен в использовании, управление комнатой, управление микрофоном и другие возможности предоставляются бесплатно в течение ограниченного времени**. Однако использование Chat IM и услуги Tencent Real-Time Communication TRTC во время использования продукта будет продолжать взиматься по обычным тарифам. Подробности см. в разделе [Обзор выставления счетов Chat IM](https://www.tencentcloud.com/document/product/1047/34349) и [Обзор выставления счетов TRTC](https://www.tencentcloud.com/document/product/647/34610). Если произойдут какие-либо изменения в методе выставления счетов продукта, функциях и периоде бета-тестирования, мы заранее объявим об этом на официальном веб-сайте и уведомим вас через сообщения на сайте, текстовые сообщения и электронные письма. Оставайтесь на связи.

## Введение в функции

TUIRoomKit предоставляет интерфейс для стандартных встреч, поддерживает общий доступ к экрану, управление участниками, управление микрофоном и рисованием, текстовый чат, и другие функции управления конференцией. Совместим с платформами Windows, Mac, Android, iOS, Web и Electron для бесшовного взаимодействия. Подробные функции можно найти в таблице ниже:

| **Тип функции** |  | **Описание функции** |
| --- | --- | --- |
| Представление интерфейса | Плавающее окно встречи | Плавающее окно интерфейса встречи |
|  | Пользовательский интерфейс | Поддержка пользовательской настройки макета интерфейса встречи |
| Возможность управления комнатой | Создание/уничтожение/присоединение/выход из комнаты | Создание, уничтожение, присоединение и выход из комнаты встречи |
|  | Отключение видео/микрофона/чата для одного пользователя или всех | Отключение видео, микрофона и чата для одного или всех пользователей |
|  | Исключение из комнаты (встречи) | Поддержка исключения указанных пользователей из комнаты встречи |
|  | Текстовые сообщения в комнате | Участники отправляют текстовые сообщения |
|  | Пользовательская информация комнаты | Пользовательское описание комнаты, имя и т. д. |
|  | Общий доступ к экрану | Участники предоставляют общий доступ к экрану компьютера на встрече |
| Возможность аудио и видеозвонков | Многоабонентский аудио/видеозвонок | **Поддержка до 300 участников, 50 человек с включенным микрофоном, если вам нужно увеличить лимит, пожалуйста,**[отправьте заявку](https://console.tencentcloud.com/workorder/category)**и свяжитесь с нами** |
|  | Включение/отключение камеры | Участники включают/отключают камеру |
|  | Включение/отключение микрофона | Участники включают/отключают микрофон |
|  | Регулировка громкости микрофона | Участники регулируют громкость микрофона |
|  | Шумоподавление на основе ИИ | Эффективная фильтрация фонового шума с помощью возможностей ИИ, восстановление качества вызова |
|  | Качество изображения | Опция SD, HD, FHD, 2K |
| Вход с нескольких устройств | Вход на встречу с нескольких устройств | Подключение на любой платформе, автоматическое прекращение запросов доступа с других устройств |
|  | Вход на конференцию с нескольких устройств на одной платформе | Поддержка входа с одной платформы (например, нескольких устройств iOS) |

# Требуется лицензия для воспроизведения видео в новой версии LiteAV SDK

Мы выпустили версию 10.1 LiteAV SDK для мобильных устройств (Android, iOS и Flutter). Новая версия использует тот же механизм воспроизведения, который работает в Tencent Video, и поставляется с оптимизированными функциями воспроизведения. Подробнее см. [Что нового](#up).

В этой новой версии вам потребуется лицензия для использования модуля **воспроизведения видео**.

Вы можете **бесплатно подать заявку на получение лицензии для использования возможностей трансляции и воспроизведения видео по требованию в новом SDK**. Подробные инструкции см. в разделе [Бесплатная лицензия](#warrant). Вам не нужна лицензия, если вы не используете функции воспроизведения видео или не обновились до v10.1 или более поздних версий.

> **Примечание:** **Если ваше приложение уже имеет лицензию на потоковую трансляцию или лицензию UGSV, вы сможете продолжать использовать функции воспроизведения видео после обновления до v10.1.** Вы можете просмотреть информацию о своей лицензии в [консоли CSS](https://console.intl.cloud.tencent.com/live) или [консоли VOD](https://console.intl.cloud.tencent.com/vod).

## Бесплатная лицензия

1. Войдите в [консоль TRTC](https://console.tencentcloud.com/trtc). Если у вас еще нет учетной записи, сначала зарегистрируйтесь.
2. Выберите **Связанные облачные сервисы** на левой боковой панели. В разделе **Лицензии** нажмите **Создать официальную лицензию**, введите требуемую информацию и нажмите **Подтвердить**.

## Что нового

| Функция | Описание |
| --- | --- |
| Механизм воспроизведения Tencent Video | Впервые Tencent предлагает общественности возможности воспроизведения своего продукта Tencent Video, которые предоставляют LiteAV SDK новые преимущества, такие как превосходное качество аудио/видео, точный поиск, автономное воспроизведение, а также более плавное, стабильное воспроизведение и лучшую совместимость. |
| Новые форматы | Новый SDK добавляет поддержку форматов, включая QUIC, AV1 и H.266, удовлетворяя потребности в большем количестве сценариев и помогая снизить затраты, сокращая использование полосы пропускания на 20-55% по сравнению с существующими форматами H.264 и H.265. |
| Новое решение для защиты контента | Помимо собственного шифрования протокола, локального шифрования и защиты от горячих ссылок, новый SDK представляет схему DRM-шифрования на основе хорошо зарекомендовавших себя технологий DRM, обеспечивающую полную защиту вашего контента. |
| Улучшение качества видео | Новый SDK добавляет поддержку воспроизведения видео HDR10. Он также предлагает решения для транскодирования Top Speed Codec, которые помогут вам снизить использование полосы пропускания без ущерба для качества воспроизведения. |
| Специфичная для сценариев интеграция с низким кодом | Мы предоставляем код для сценариев приложений, таких как короткие видео и видеоленты, а также возможности, включая создание эскизов, динамические водяные знаки и воспроизведение плейлистов, чтобы помочь вам быстро создавать приложения с низкими затратами. |

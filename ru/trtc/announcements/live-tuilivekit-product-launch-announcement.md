# Объявление о запуске Live (TUILiveKit)

Уважаемые пользователи,

**TRTC Live (TUILiveKit) официально запускается 22 апреля 2024 года по пекинскому времени (UTC+8)**. Этот продукт разработан специально для интерактивного потокового вещания в сценариях социальных развлечений, онлайн-шопинга, фитнес-консультаций и онлайн-обучения, целью которого является обеспечить вашему приложению новый интерактивный опыт.

## ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/4111208ffdf811ee80d9525400ae4d13.png)

## Основные особенности продукта

- **Готов к использованию**: Live предоставляет UI-наборы как для трансляторов, так и для зрителей с полным набором компонентов и пользовательскими интерфейсами, тесно адаптированными к API бизнеса потокового вещания, что значительно снижает сложность разработки. С помощью всего нескольких строк изменения кода возможно быстро запустить возможности интерактивного потокового вещания, сэкономив 90% времени разработки.
- **Богатые функции интерактивного потокового вещания**: поддерживает множество интерактивных функций, таких как чат и эмодзи, отправка подарков, отметки "нравится" и поддержка нескольких гостей, а также поддерживает функции конфигурации, такие как управление аудиторией, теги аудитории, улучшение внешности, фоновая музыка и плавающее окно, подходит для различных сценариев бизнеса потокового вещания.
- **Сверхнизкая задержка и высокое качество**: обеспечивает задержку ниже 300 миллисекунд, поддерживает 2K и 4K, использует оптимизированные сетевые технологии и эффективное кодирование для обеспечения более плавного и немедленного опыта взаимодействия между зрителями и трансляторами.
- **Гибкая настройка UI**: предоставляет компоненты с открытым исходным кодом для стандартного интерфейса потокового вещания, которые можно использовать напрямую или гибко модифицировать в соответствии с конкретными бизнес-сценариями.

## Начните бесплатно

- TRTC Live предоставит бесплатную пробную версию (0 USD/месяц) при запуске с периодом тестирования 14 дней, который может быть продлен один раз, всего 28 дней. Дополнительно, ежемесячно будет предоставляться 10 000 минут аудио- и видеоконтента. Для подробных инструкций по активации см. [Активировать сервис (TUILiveKit)](https://www.tencentcloud.com/document/product/647/60033). Для получения подробной информации о тарификации и функциях Live см. [Пакеты TRTC Live на месяц](https://www.tencentcloud.com/document/product/647/59407). Для получения дополнительной информации см. [Введение в компоненты (TUILiveKit)](https://www.tencentcloud.com/document/product/647/60034).

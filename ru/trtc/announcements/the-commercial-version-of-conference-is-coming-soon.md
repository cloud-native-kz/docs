# Коммерческая версия Conference выходит в ближайшее время

Коммерческая версия TRTC Conference (TUIRoomKit) запустится 12 марта 2024 года. Уже сейчас вы можете связаться с нами, чтобы получить бесплатный доступ к мощным функциям коммерческого SDK (версия 2.0).

### Основные возможности новой версии

В новой версии SDK функциональность продукта значительно расширена:

- Новые функции, такие как бронирование комнаты.
- Комплексная оптимизация основных функций, включая управление комнатой, управление участниками, управление сигналами управления конференцией и управление микрофоном.
- Абсолютно новый REST API бэкенда для вашего использования.

### Способ подачи заявки

Вы можете применить для получения доступа к испытанию и узнать больше подробностей о продукте двумя следующими способами:

- **Отправьте письмо на адрес: info_rtc@tencent.com. Обязательно укажите SDKAPPID в письме, чтобы мы могли быстро ответить и активировать его для вас**. (Рекомендуется для быстрого ответа).
- [Присоединитесь к группе Telegram](https://t.me/+EPk6TMZEZMM5OGY1?s_url=https%3A%2F%2Ftrtc.io).

> **Примечание:** загрузка по умолчанию на GitHub — это SDK версии 2.0. Вам необходимо связаться с нами одним из указанных выше способов, чтобы активировать его для использования.

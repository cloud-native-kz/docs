# Лицензия на бесплатный пробный период

Для улучшения работы с функцией Beauty AR SDK мы предлагаем бесплатный пробный период для каждого SDKAppID, обеспечивающий бесплатный доступ к 4 редакциям:**Все функции красоты, Виртуальный фон, Распознавание лиц и Распознавание жестов,** а также возможность **Gift AR**. На мобильной платформе каждый SDKAppID может использовать каждую функцию **бесплатно один раз (каждый пробный период длится 14 дней, может быть продлен один раз на общий срок 28 дней).**

Этот документ содержит инструкции и рекомендации по [подаче заявки на пробную лицензию](#mobileapplytrial) и [продлению пробной лицензии](#mobilerenewtrial) для Beauty AR SDK Mobile и Gift AR.

## Функции бесплатного пробного периода

Функции, которые пользователи могут протестировать бесплатно:

| **Редакция** | **Функции** | **Описание** |
| --- | --- | --- |

|  | Регулировка качества изображения | Контрастность, насыщенность, резкость, усиление резкости, яркость, подавление шума, теплота и оттенок |
|  | Коррекция лица | Большие глаза, ширина лица, худое лицо (естественное, женское и мужское), втягивание челюсти, V-образная форма, подбородок, длина лица, форма лица, линия волос, морщины, линии улыбки, мешки под глазами, яркие глаза, расстояние между глазами, внешние уголки глаз, уголки глаз, ширина глаз, высота глаз, положение глаз, угол бровей, расстояние между бровями, высота бровей, длина бровей, толщина бровей, гребень брови, тонкий нос, крылья носа, скулы, положение носа, спинка носа, назион, белые зубы, форма губ, высота губ, ширина губ, положение губ, улыбающееся лицо |
|  | Точечный макияж | Помада, румяна, трёхмерность, тени для век, подводка для глаз, тушь для ресниц, брови |
|  | Наклейки | 2D наклейки, 3D наклейки и наклейки с жестами |
|  | Фильтры | Раскрашивание изображения в реальном времени |
|  | Наборы макияжа | Модные образы макияжа, создание естественного красивого образа одним нажатием. |
|  | Виртуальный фон | Размытый, прозрачный и пользовательские фоны |
|  | Коррекция фигуры | Одноклик для похудения, удлинение ног, стройные ноги, стройные плечи, маленькая голова, увеличение груди |

## Подайте заявку на пробную лицензию

Вы можете активировать бесплатный пробный период Beauty AR SDK в консоли TRTC. Конкретные шаги следующие:

1. Войдите в [консоль TRTC > Соответствующие услуги](https://console.trtc.io/aboutcloud) и нажмите **Начать работу.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/845a3da805f911efa2e5525400488742.png)

2. Выберите **Mobile**, и в зависимости от ваших потребностей заполните `Имя приложения`, `Имя пакета` и `Bundle ID`. Отметьте функции, которые хотите протестировать: **Все функции красоты, Виртуальный фон, Распознавание лиц, Распознавание жестов, Gift AR**, затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1fef7185516f11f0adb15254005ef0f7.png)

3. После активации вы можете просмотреть вашу информацию на текущей странице и обратиться к **Руководству интеграции** выше для интеграции. `License Url` и `License Key` будут использованы в [**руководстве интеграции**](https://trtc.io/document/60195).

> **Примечание:** Для интеграции специальных эффектов анимации подарков обратитесь к: [Gift AR (Android)](https://www.tencentcloud.com/document/product/1143/70531) и [Gift AR (iOS)](https://www.tencentcloud.com/document/product/1143/70537).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7a5c4d522faa11f08caa5254005ef0f7.png)

## Продление пробной лицензии

Первоначальное применение пробной лицензии поставляется со сроком действия по умолчанию 14 дней, после чего вы можете продлить **один раз**. Нажмите кнопку **Продлить** в нижней части блока информации пакета, а затем нажмите **Подтвердить**, чтобы продлить функцию еще на 14 дней.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a5ac67822fcc11f09e67525400bf7822.png)

## Обновление пробной лицензии

Если вам нужно обновить пробную лицензию Beauty AR SDK Web&H5 на официальную лицензию, чтобы продлить ее срок действия, сначала [выберите и приобретите пакет Beauty AR SDK для Mobile](https://console.trtc.io/subscription/buy/beauty-mobile), затем выполните следующие действия:

1. Нажмите **Обновить** в нижней части блока информации пакета пробной лицензии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/8e3683e92fcc11f0b44b5254007c27c5.png)

2. Перейдите на **интерфейс обновления**, нажмите **Привязать**, выберите непривязанный пакет Beauty AR SDK Mobile и нажмите **Подтвердить**, чтобы обновить и создать официальное приложение с тем же именем пакета, при этом разблокировав официальную лицензию Beauty AR SDK без необходимости прохождения проверки выпуска. Если нет привязываемых пакетов Beauty AR SDK, вы можете перейти на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty-mobile) для приобретения. **Обратите внимание, что ресурсы вступят в силу немедленно после покупки, а период действия начнется.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b7e41f162fcc11f091625254001c06ec.png)

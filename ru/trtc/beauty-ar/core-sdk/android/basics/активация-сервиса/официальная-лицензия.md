# Официальная лицензия

Beauty AR SDK поддерживает интеграцию как с **Web & H5**, так и с **мобильными** платформами. Эта статья проведет вас через процесс **мобильной платформы** [Покупка официальной лицензии](https://www.tencentcloud.com/document/product/1143/69831#12a202ad-aa1c-4e97-b644-269058424e5f), [Обновление периода действия официальной лицензии](https://www.tencentcloud.com/document/product/1143/69831#0d255ca9-ee85-4ce2-ac14-3a7e2a35e82f) и [Автоматическое продление](https://www.tencentcloud.com/document/product/1143/69831#c402189a-da8e-4930-8300-982455de59d3).

## Мобильная платформа

Перед покупкой официальной версии мы в настоящее время предлагаем 28-дневный бесплатный пробный период для веб-версии. Вы можете [начать бесплатный пробный период](https://trtc.io/document/60219).

### Покупка официальной лицензии

Вы можете выбрать и купить пакет SDK, который соответствует вашим потребностям, на [странице покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty-mobile) и получить официальную лицензию на использование. **Период действия начинается с момента покупки. Пожалуйста, используйте его как можно скорее**. Различия в поддерживаемых функциях между различными версиями SDK см. в [Обзоре счетов](https://trtc.io/zh/document/60214#).

1. Конкретные шаги следующие: перейдите на [TRTC Console > Связанные услуги](https://console.trtc.io/aboutcloud), нажмите **Начать**, войдите в управление лицензиями и нажмите для создания официальной лицензии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/57e05211309d11f08caa5254005ef0f7.png)

2. Следуя графическому руководству, перейдите на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty-mobile) для покупки нужной версии пакета, затем нажмите **Покупка завершена**. После этого вы можете редактировать `Имя приложения`, `Имя пакета` и `Bundle ID` официального приложения и нажать для привязки. Выберите мобильный пакет Beauty AR SDK и нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/800f314030ad11f0b0b1525400e889b2.png)

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/481bc8e9309e11f0b0b1525400e889b2.png)

3. После успешного создания официальной лицензии на странице будет отображена информация созданной официальной лицензии. Для интеграции вы можете обратиться к приведенному выше **Руководству по интеграции**. `License Key` и `License Token` будут использоваться в руководстве по интеграции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9a0556a8309e11f0b0b1525400e889b2.png)

### Обновление периода действия официальной лицензии

Вы можете войти в [TRTC Console > Связанные услуги](https://console.trtc.io/aboutcloud), нажать **Начать**, войти на страницу **Управление лицензиями** и просмотреть период действия официальной лицензии мобильного Beauty AR SDK. Официальная лицензия Beauty AR будет отправлять вам напоминания об истечении за 32 дня, 7 дней, 3 дня и 1 день до истечения срока, напоминая вам о своевременном продлении во избежание влияния на обычные бизнес-операции. Если ваша официальная лицензия мобильного Beauty AR SDK скоро истечет, вы можете выполнить следующие шаги для продления:

1. Выберите лицензию, которую вы хотите продлить, затем нажмите **Обновить действительность** внизу поля ресурса.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/65a74f29309f11f0b0b1525400e889b2.png)

2. Beauty AR License поддерживает два способа продления периода действия: **Продлить текущую лицензию** и **Выбрать другой ресурс лицензии для замены**, как показано ниже.

> **Примечание:** **Ресурсы с включенным автоматическим продлением не поддерживают метод замены ресурса лицензии для продления.** Если вы хотите изменить период действия на период действия другой лицензии, отключите функцию автоматического продления.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9814fd6930ae11f0b0b1525400e889b2.png)

Продлить текущую лицензию

Выбрать другой ресурс лицензии для замены

1. Выберите **Продлить текущую лицензию**, затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/22c54de630af11f091625254001c06ec.png)

> **Примечание:** Если выбрать **Продлить текущую лицензию**: когда ресурс лицензии **находится в периоде действия или истек менее семи дней назад**, в соответствии с правилами выставления счетов Tencent Cloud поддерживается продление текущей лицензии. После продления период действия будет обновлен на: **текущий период действия + длительность продления.** Когда ресурс лицензии **истек более семи дней назад**, исходный ресурс был уничтожен, и мы купим для вас ресурс того же типа и автоматически завершим привязку замены. После продления период действия будет обновлен на: **дата платежа + длительность продления**.

2. На странице **Продление** выберите **Период продления**. Мобильный endpoint Beauty AR Web продлевается периодически **По месяцам/годам**. Нажмите **Подтвердить** для расширения периода действия лицензии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3249635c348011f0bd655254007c27c5.png)

1. Выберите **Выбрать другой ресурс лицензии для замены**, затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a0e0005630af11f0b0b1525400e889b2.png)

2. На странице **Обновить действительность** нажмите **Привязать**, выберите **Непривязанный** мобильный пакет Beauty AR SDK (если нет доступных пакетов для привязки, перейдите на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty-mobile) для покупки), затем нажмите **Подтвердить**.

> **Примечание:** В настоящее время расширение действительности пакета поддерживает только пакеты одного типа. То есть, если тип привязанного пакета лицензии — Standard, то продление может выбрать только пакет Standard для продления. Если вы хотите изменить тип привязанного пакета, вам необходимо [Связаться с нами](https://trtc.io/contact) или обратиться в команду продаж для обработки.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cabaf4c930af11f09e67525400bf7822.png)

3. Проверьте период действия после обновления.

> **Примечание:** **Официальная лицензия Beauty AR не поддерживает изменение информации**. Если вам нужно изменить информацию о лицензии, не используйте купленный пакет ресурсов для обновления периода действия лицензии. Вместо этого нажмите **Создать официальную лицензию** для повторного создания приложения, добавления новой лицензии и привязки информации о новом имени пакета.

### Автоматическое продление

Рекомендуется использовать опцию **Включить автоматическое продление** для продления. После включения система будет автоматически продлевать ежемесячно после истечения срока, при условии, что на вашем счете **достаточно средств**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/402d54d730b011f0b47352540044a08e.png)

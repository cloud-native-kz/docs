# Руководство по расширенному режиму

## Что такое расширенный режим

В SDK рекомендуется устанавливать параметры красоты в диапазон от 0 до 100 или от -100 до 100 (см. [Параметры эффектов](https://www.tencentcloud.com/document/product/1143/60207)). Регулировка параметров в этом диапазоне обычно дает удовлетворительные результаты красоты. Если установка параметра на максимальное или минимальное значение по-прежнему не удовлетворяет вашим требованиям, вы можете рассмотреть возможность использования расширенного режима. Расширенный режим может сделать эффекты красоты более выраженными, например более заметное разглаживание кожи и дальнейшее сужение лица.

## Как использовать расширенный режим

В версиях после SDK 3.5.0 мы оптимизировали метод использования расширенного режима. **Вам просто нужно установить большие значения в SDK**, например, если рекомендуемый диапазон значений составляет -100 до 100, то вы можете установить -120 до 120 в SDK.

1. **Если вы используете наш компонент пользовательского интерфейса TEBeautyKit:**

Вызовите метод `enableEnhancedMode` из `TEBeautyKit`. После вызова TEBeautyKit умножит значение, отображаемое на панели, на подходящий множитель перед установкой его в SDK. ## Рекомендуемый множитель повышения для расширенного режима

Мы предоставляем справочное значение для множителя повышения. Не рекомендуется превышать рекомендуемое нами значение. В противном случае эффект красоты может ухудшиться. Справочное значение см. ниже:

| Название элемента красоты | Рекомендуемый максимальный множитель повышения |
| --- | --- |
| Отбеливание, сокращение лица, V-лицо, расстояние между глазами, положение носа, удаление морщин смеха, помада, трёхмерность | 1,3x |
| Осветление глаз | 1,5x |
| Румяна | 1,8x |
| Прочие | 1,2x |

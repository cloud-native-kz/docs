# Использование материалов

## Фильтры

Каждый фильтр — это изображение в формате png. При использовании необходимо передать путь к изображению в SDK. Этапы работы см. ниже.

### Сценарий 1: Использование TEBeautyKit

TEBeautyKit — это библиотека панели пользовательского интерфейса для BeautyAR. Она разработана для удобного и быстрого использования и управления функциями красоты.

Этапы работы см. ниже.

1. **Ознакомьтесь с**[Интеграцией UIKit (Android)](https://www.tencentcloud.com/document/product/1143/60196)**.**
2. **Добавление ресурсов фильтра**

Поместите добавленное изображение фильтра в директорию `assets/lut` вашего проекта. Затем отредактируйте файл конфигурации панели `assets/beauty_panel/lut.json`, добавив новый элемент на основе существующего содержимого json. При запуске приложения вызов метода `TEBeautyKit` `copyRes` скопирует изображение фильтра из директории assets в директорию downloadPath, указанную в `lut.json`.

3. **Конфигурирование иконок фильтров**

Поле icon в `lut.json` представляет иконку фильтра. Поместите иконку в директорию `assets/beauty_panel/panel_icon/lut_icon`. Значение поля icon также может быть URL-адресом иконки, начинающимся с `http` или `https`, и TEBeautyKit загрузит иконку из интернета.

4. **Конфигурирование ресурсов фильтра**

Поле resourceUri в `lut.json` — это путь, где сохраняется изображение фильтра в приватной директории приложения. Конфигурируйте ресурсы на основе существующих элементов и измените расширение resourceUri с "xxx.png" на имя файла добавленного фильтра, избегая конфликтов с существующими фильтрами в `lut.json`. Поле resourceUri также может быть URL-адресом изображения фильтра, начинающимся с `http` или `https`. При клике на URL-адрес изображение фильтра будет загружено из интернета и сохранено в директории downloadPath, указанной в `lut.json`.

### Сценарий 2: Прямая интеграция SDK BeautyAR

1. Поместите добавленное изображение фильтра в любую директорию папки assets вашего проекта. Затем при инициализации приложения скопируйте изображение в приватную директорию приложения или на карту SD, чтобы получить путь к изображению, обозначаемый как `/path/to/your/lut_xxx.png`. Для простоты рекомендуется поместить изображение в директорию assets/lut, а затем скопировать код copyRes из TEBeautyKit из демо-проекта для использования.
2. При использовании фильтра вызовите метод [setEffect](https://www.tencentcloud.com/document/product/1143/60201#32a345ed-73c4-4c60-bc7e-3f91b9a4755c) SDK и передайте путь к изображению фильтра в SDK.

## Анимированные наклейки

Каждая анимация — это папка. При использовании необходимо передать путь папки в SDK. Этапы работы см. ниже.

### Сценарий 1: Использование TEBeautyKit

TEBeautyKit — это библиотека панели пользовательского интерфейса для BeautyAR. Она разработана для удобного и быстрого использования и управления функциями красоты.

Этапы работы см. ниже.

1. **Ознакомьтесь с**[Интеграцией UIKit (Android)](https://www.tencentcloud.com/document/product/1143/60196)**.**
2. **Добавление анимационных материалов**

Поместите добавленную папку анимации в директорию `assets/MotionRes` вашего проекта. Затем отредактируйте файл конфигурации панели `assets/beauty_panel/motions.json`, добавив новый элемент на основе существующего содержимого. При запуске приложения вызов метода `copyRes` у `TEBeautyKit` скопирует папку анимации из директории assets в директорию downloadPath, указанную в `motions.json`.

3. **Конфигурирование иконок анимации**

Поле icon в `motions.json` представляет иконку анимации. Поместите иконку в директорию `assets/beauty_panel/panel_icon/motions_icon`. Значение поля icon также может быть URL-адресом иконки, начинающимся с `http` или `https`, и TEBeautyKit загрузит иконку из интернета.

4. **Конфигурирование анимационных материалов**

Поле resourceUri в `motions.json` — это путь, где сохраняется анимация в приватной директории приложения. Конфигурируйте материалы на основе существующих элементов, избегая конфликтов с существующими анимациями в `motions.json`. Поле resourceUri также может быть URL-адресом zip-файла анимации, начинающимся с `http` или `https`. При клике на URL-адрес zip-файл будет загружен из интернета и сохранен в директории downloadPath, указанной в `motions.json`.

### Сценарий 2: Прямая интеграция SDK BeautyAR

Поместите добавленную папку анимации в любую директорию папки assets вашего проекта. Затем при инициализации приложения скопируйте папку в приватную директорию приложения или на карту SD, чтобы получить папку анимации, обозначаемую как `/path/to/your/motion`. При использовании анимации вызовите метод [setEffect](https://www.tencentcloud.com/document/product/1143/60201#32a345ed-73c4-4c60-bc7e-3f91b9a4755c) SDK и передайте путь в SDK.

## Макияж красоты и сегментация фона

Их использование аналогично использованию анимированных наклеек, описанному выше, с соответствующими файлами json `makeup.json` и `segmentation.json`.

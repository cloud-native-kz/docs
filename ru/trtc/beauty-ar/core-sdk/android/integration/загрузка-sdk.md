# Загрузка SDK

## Мобильные версии

SDK для мобильных приложений доступен в 19 версиях, включая семь базовых версий, семь расширенных версий и пять возможностей. Подробные сведения о функциях различных версий см. в разделе [Обзор цен](https://www.tencentcloud.com/document/product/1143/69949). Текущая доступная для загрузки версия — 4.0.0. Информацию об истории выпусков см. в разделе [История версий](https://www.tencentcloud.com/document/product/1143/60203).

> **Примечание:** Поскольку существует множество категорий пакетов, если вы еще не выбрали пакет, вы можете загрузить SDK пакета **S1-07** (который включает все функции) для использования. После выбора пакета позже вы сможете заменить его. Выбор пакетов SDK можно просмотреть в [описании функций](https://www.tencentcloud.com/document/product/1143/45376), в котором подробно описаны функции, включенные в пакет красоты.

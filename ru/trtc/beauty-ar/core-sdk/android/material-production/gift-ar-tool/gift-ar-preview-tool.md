# Инструмент предпросмотра Gift AR

Этот документ в основном описывает способ использования инструмента предпросмотра Gift AR от Tencent в Gift AR. После прочтения этого документа вы узнаете, как использовать инструмент предпросмотра Gift AR для быстрого предпросмотра материалов анимации.

## Загрузка

Вы можете загрузить инструмент предпросмотра Gift AR на локальный компьютер с помощью [TEP Preview](https://mediacloud-76607.gzc.vod.tencent-cloud.com/MediaX/TEP_Preview/TEP_Preview_Latest.zip). После завершения загрузки распакуйте архив TEP_Preview_Latest.zip, чтобы получить инструмент предпросмотра Gift AR. Затем дважды щелкните его, чтобы начать работу.

> **Примечание:** Инструмент предпросмотра Gift AR в настоящее время поддерживает только систему MacOS. Пожалуйста, используйте соответствующую систему для последующей установки и использования.

## Использование

Указатели управления внутренней панели инструмента предпросмотра Gift AR отмечены следующим образом на рисунке:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/94cafbe44b2611f092f25254007c27c5.png)

### Предпросмотр анимации

1. Заполните информацию License Key анимации, которую требуется предпросмотреть, в области License Key в верхней части панели инструмента предпросмотра Gift AR, чтобы обеспечить правильный предпросмотр.
2. С помощью мыши перетащите файл анимации, который требуется предпросмотреть, в область предпросмотра анимации на панели инструмента предпросмотра Gift AR, а затем отпустите кнопку мыши для начала предпросмотра.
3. Во время предпросмотра вы можете нажать кнопку функции в области нижнего левого угла, чтобы выполнить паузу и начало воспроизведения анимации.

### Просмотр детальной информации об анимации

Во время предпросмотра вы можете нажать область в нижнем правом углу

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/fe163ec5095811f09d28525400bf7822.png)

для просмотра дополнительной информации о предпросматриваемой анимации. Схематическая диаграмма после нажатия выглядит следующим образом:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9c11cf354b2611f085275254001c06ec.png)

На приведенном выше рисунке в правом верхнем углу отображается информация об объединенной анимации предпросматриваемой анимации. Информация по этому вопросу приведена в таблице ниже:

| Значение тега | Тип объединенной анимации | Описание |
| --- | --- | --- |
| receiver_avatar | Тип изображения | Впоследствии вы можете заменить этот регион на **Image**. |
| sender_avatar | Тип изображения | Впоследствии эту область можно заменить на **Image**. |
| img_405 | Тип текста | Впоследствии эту область можно заменить на **text**. |

Одновременно в правом нижнем углу отображается детальная информация о предпросматриваемой анимации. Информация по этому вопросу приведена в таблице ниже:

| Тип | Значение | Описание |
| --- | --- | --- |
| Duration | 14 | Общая длительность анимации в текущем предпросмотре составляет 14 с. |
| FrameRate | 20 fps | Частота кадров анимации в текущем предпросмотре составляет 20 fps. |
| Width | 712 | Ширина анимации в текущем предпросмотре составляет 712 px. |
| Height | 1267 | Высота анимации в текущем предпросмотре составляет 1267 px. |
| FrameCount | 299 | Общее количество кадров анимации в текущем предпросмотре составляет 299. |

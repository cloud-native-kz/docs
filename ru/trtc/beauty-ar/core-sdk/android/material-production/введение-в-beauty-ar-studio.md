# Введение в Beauty AR Studio

## Введение в продукт

Beauty AR Studio — это инструмент создания материалов, поддерживающий клиентов в настройке 2D и 3D стикеров для персонализированного производства материалов. После завершения производства материалы можно импортировать в SDK для использования.

## Возможности инструмента

- AI Configuration: модульное редактирование возможностей искусственного интеллекта
- 3D Editing: проектирование материалов 3D-моделей
- Process Control: богатые опции управления процессом для создания различных сложных динамических эффектов
- Dynamic Effect Import: поддержка импорта локальных динамических материалов для предварительного просмотра эффектов

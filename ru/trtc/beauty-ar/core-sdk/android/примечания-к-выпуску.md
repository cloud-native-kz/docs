# Примечания к выпуску

### Версия 4.1.0.13 @2025.12.22

- Добавлена функция сегментации Chroma Key.
- Улучшена функция Beautiful Body: повышена точность и стабильность распознавания человеческого тела.
- Обновлена библиотека libpag: с версии 4.4.24-noffavc на 4.4.35-noffavc.
- Исправлены известные ошибки.

### Версия 4.0.0.13 @2025.10.22

- Добавлено свойство "skin highlight" (блеск кожи).
- Добавлена функция определения улыбки.
- Оптимизация эффекта красоты.
- Исправлены известные ошибки.
- Оптимизирован размер файла ресурсов пакета.

### Версия 3.9.3.12 @2025.07.16

- Добавлены несколько эффектов сглаживания кожи в режиме EffectMode_Pro.
- Интегрирована функция GAN косметического улучшения кожи в режиме EffectMode_Pro.
- Добавлены новые ресурсы материалов.
- Оптимизированы эффекты красоты.
- Исправлены известные ошибки.

### Версия 3.9.2.7 @2025.04.24

- Добавлена функция автоматического подавления шума в условиях контрового света.
- Обновлена и улучшена библиотека libpag с поддержкой совместимости Android 15 16k pagesize.
- Оптимизация производительности.
- Исправлены известные ошибки.

### Версия 3.9.1.6 @2025.04.02

- Введена функция "Forehead 2" (лоб 2).
- Исправлены известные ошибки.

### Версия 3.9.0.8 @2024.11.19

- Добавлена функция "Light Makeup" (легкий макияж). Подробности см. в руководстве пользователя Light Makeup.
- Добавлена функция регулировки уголка глаза; в Whitening добавлен эффект "Brightening" (осветление); Whitening поддерживает Custom Filter Path (пользовательский путь фильтра). Подробности см. в разделе [Effect Parameters](https://www.tencentcloud.com/document/product/1143/60207).
- High-Performance Mode (режим высокой производительности) изменен на EffectMode. Подробности см. в руководстве пользователя EffectMode (High-Performance Mode).
- Известные ошибки исправлены.

### Версия 3.8.0.15 @2024.10.15

Исправлены известные ошибки.

### Версия 3.8.0.9 @2024.08.29

- Добавлены атрибуты красоты/макияжа красоты/тела: регулировка яркости, положение глаз, двойное веко, мешки под глазами, стройные руки.
- Оптимизирована проблема отображения макияжа на закрытых объектах.
- Оптимизирована проблема размазывания макияжа на устройствах низкого уровня.
- Добавлены новые материалы макияжа и стикеров.
- Исправлены известные ошибки.
- Оптимизация производительности.

### Версия 3.7.0.5 @2024.05.29

- Добавлены функции загара кожи.
- Добавлена функция окрашивания волос для макияжа в одной точке.
- Добавлена функция естественного сужения лица в режиме высокой производительности.
- Добавлены новые материалы 2D-анимации.
- Добавлен API `getDeviceLevel` для получения уровня устройства.
- Оптимизирован эффект лба (линии роста волос).
- Оптимизирована панель эффектов красоты.

### Версия 3.6.0.4 @2024.03.21

- Добавлена функция "Denoise" (подавление шума).
- Добавлена функция "camera move" (движение камеры) и улучшена пользовательская опыт.
- Исправлены известные ошибки.

### Версия 3.5.0.5 @2024.01.29

- Добавлены функции регулировки цветовой температуры и тона.
- Добавлены функции интеллектуальной красоты и определения пола, обеспечивающие смягченные эффекты косметического макияжа для мужчин и младенцев.
- SDK поддерживает параметры pag-файла при использовании пользовательской сегментации и зеленого экрана.
- Оптимизирована функция сегментации фона.
- Оптимизированы эффекты сглаживания и осветления кожи, минимизировано влияние на не-кожные области.
- Оптимизирована чувствительность обнаружения лица.
- Оптимизирован интерфейс установки эффектов красоты и добавлен новый интерфейс setEffect.
- Исправление ошибки: диапазон резкости, который ранее был от -100 до 100, изменен на 0 до 100. Увеличение груди оптимизировано как регулировка груди с изменением диапазона с 0 до 100 на -100 до 100.
- Добавлены материалы 2D/3D-стикеров и материалы стилей макияжа на все лицо.
- Обновлена библиотека libpag до версии 4.3.33. Клиенты могут при необходимости заменить ее своей версией pag.
- Переработан стиль демонстрации и добавлена быстродоступная библиотека TEBeautyKit.

### Версия 3.3.0.2 @2023.10.30

- Добавлены новые функции, такие как цветные контактные линзы, ширина переносицы и корень носа.
- В демонстрацию добавлена новая функция обрезки текстуры.
- В демонстрации точки лицевых ориентиров используют разные цвета для обозначения точек в зависимости от их видимости.

### Версия 3.2.0.5 @2023.09.20

- Добавлена функция эффекта резкости.
- Добавлен новый уровень отбеливания кожи - whitening 3.
- Добавлена эксклюзивная функция для осветления кожи.
- Оптимизирован эффект размытия фона.
- Оптимизирована логика макияжа в отдельной области.
- Решена проблема опускания головы при управлении аватаром.
- Оптимизирована логика управления голосом.
- Обновлена версия libpag.
- Оптимизированы правила сглаживания, чтобы избежать появления класса `a.a.a.a.a`.
- Устранены проблемы зубчатых краев и нечеткости периферии при сглаживании кожи.

### Версия 3.1.0.2 @2023.08.02

- Вновь добавлен макияж в одной точке: тени для век, подводка, ресницы и брови, доступные в пакете серии S.
- Добавлена функция распознавания жестов, которая может распознавать 14 жестов и положения рук. Подробности см. в [документе по жестам](https://www.tencentcloud.com/document/product/1143/56174).
- В функцию красоты добавлен режим высокой производительности, подходящий для длительного использования на устройствах низкого уровня. Подробности см. в документации Android API.
- Аватар получил новые материалы анимации.
- Аватар поддерживает пользовательские фоны.
- В функциях пользовательской сегментации фона и сегментации зеленого экрана красоты добавлено определение ориентации изображения.
- Исправлены известные ошибки.

### Версия 3.0.1.2 @2023.07.18

- Добавлена функция наложения материалов, подробности см. в документации Android API.
- Исправлены некоторые известные ошибки.

### Версия 3.0.0.13 @2023.05.12

- Добавлены новые функции красоты, включая ширину глаз, высоту глаз, угол бровей, расстояние между бровями, высоту бровей, длину бровей, толщину бровей, пик бровей, ширину губ, положение губ и улыбающиеся губы.
- Добавлена новая функция увеличения груди в инструмент коррекции фигуры.
- Изменения библиотеки so включают переименование libteffmpeg.so на libtecodec.so.
- Исправлены проблемы пользовательского опыта и ошибки, связанные с сегментацией фона.
- Обновлен 3D-движок в SDK. Пожалуйста, проверьте, что все существующие анимированные стикеры работают правильно после обновления. Если у вас возникнут какие-либо проблемы, пожалуйста, свяжитесь с нами.
- Изменения API:

Удален интерфейс setYTDataListener, и данные, которые ранее вызывались в onYTDataUpdate, перемещены в AIDataListener как onAIDataUpdated. Содержание обратного вызова остается без изменений.

- Изменения материалов:

(1) Удалены следующие материалы: утиная шапка, фея цветов, очки 2, волшебный бант. Эти материалы несовместимы с текущей версией SDK.

(2) Обновлены следующие материалы: шлем, лапы, ягненок, очки. Если вы ранее использовали какой-либо из этих материалов, пожалуйста, свяжитесь с нами, чтобы получить обновленные файлы.

### Версия 2.4.2 @ 2022.07.07

- Оптимизирован алгоритм определения лица для устранения нестабильного распознавания и отслеживания в некоторых сценариях.
- Добавлена поддержка проверки параметров API `updateProperty`. Подробности о требованиях к параметрам см. в updateProperty.
- Исправлена проблема, при которой значение, возвращаемое `isDeviceSupport`, иногда бывает неточным.
- Добавлен `beacon.aar` в папку `libs` SDK. Необходимо скопировать его в проект. Если в проекте уже есть beacon, это может привести к конфликту. Чтобы избежать этого, не импортируйте файл AAR.
- Добавлена поддержка Unity. Подробности см. в папке `Unity` в пакете SDK.

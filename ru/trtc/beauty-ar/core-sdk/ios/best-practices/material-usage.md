# Использование материалов

## Фильтры

Каждый фильтр — это изображение в формате png. При использовании необходимо передать путь к изображению в SDK.

### Сценарий 1: использование TEBeautyKit

`TEBeautyKit` — это библиотека панели интерфейса для BeautyAR. Она предназначена для быстрого и удобного использования и управления функциями красоты пользователями. Операции описаны ниже.

1. **Ознакомьтесь с**[Интеграция UIKit (iOS)](https://www.tencentcloud.com/document/product/1143/60194)**.**
2. **Добавление материалов фильтра**

Поместите изображение нового фильтра в директорию `lut.bundle` вашего проекта. Затем измените файл конфигурации панели `TEBeautyKit/Assets/json/lut.json`, добавив новый элемент на основе существующего содержимого в json.

3. **Конфигурирование значков фильтра**

Поле `icon` в `lut.json` представляет значок фильтра. Поместите значок в директорию `TEBeautyKit/Assets/BeautyRes`. Значение поля `icon` также может быть URL-адресом значка, начинающимся с http или https, и `TEBeautyKit` будет загружать значок из интернета.

4. **Конфигурирование ресурсов фильтра**

Поле `resourceUri` в `lut.json` — это путь, где сохраняется изображение фильтра в приватной директории приложения. Конфигурируйте ресурсы на основе существующих элементов в json и измените суффикс `resourceUri` с "xxx.png" на имя файла нового фильтра, избегая конфликтов с существующими фильтрами в lut.json. Поле `resourceUri` также может быть URL-адресом изображения фильтра, начинающимся с http или https. При клике на URL-адрес изображение фильтра будет загружено из интернета и сохранено в директорию `downloadPath`, указанную в `lut.json`.

### Сценарий 2: прямая интеграция BeautyAR SDK

1. Поместите изображение нового фильтра в директорию `lut.bundle` вашего проекта. При использовании динамического подхода загрузки скачайте изображение фильтра в песочницу и запишите путь к нему.
2. При использовании фильтра вызовите метод SDK `setEffect` и передайте путь к изображению фильтра в SDK. Для инструкций по операции см. [Effect Parameters](https://www.tencentcloud.com/document/product/1143/60207).

## Анимированные стикеры

Каждая анимация — это папка. При использовании необходимо передать путь к этой папке в SDK. Операции описаны ниже.

### Сценарий 1: использование TEBeautyKit

TEBeautyKit — это библиотека панели интерфейса для BeautyAR. Она предназначена для быстрого и удобного использования и управления функциями красоты пользователями.

1. **Ознакомьтесь с**[Интеграция UIKit (iOS)](https://www.tencentcloud.com/document/product/1143/60194)**.**
2. **Добавление ресурсов анимации**

Поместите новую папку анимации в соответствующую директорию `resource bundle` вашего проекта: в `2dMotionRes.bundle` для 2D-анимаций, в `3dMotionRes.bundle` для 3D-анимаций, в `ganMotionRes.bundle` для забавных анимаций или в `handMotionRes.bundle` для анимаций жестов. Затем измените файл конфигурации панели `TEBeautyKit/Assets/json/motions.json`, добавив новый элемент на основе существующего содержимого.

3. **Конфигурирование значков анимации**

Поле `icon` в `motions.json` представляет значок анимации. Поместите значок в директорию `TEBeautyKit/Assets/BeautyRes`. Значение поля `icon` также может быть URL-адресом значка, начинающимся с `http или https`, и `TEBeautyKit` будет загружать значок из интернета.

4. **Конфигурирование ресурсов анимации**

Поле `resourceUri` в `motions.json` — это путь, где сохраняется анимация в приватной директории приложения. Конфигурируйте ресурсы на основе существующих элементов, избегая конфликтов с существующими анимациями в `motions.json`. Поле `resourceUri` также может быть URL-адресом zip-файла анимации, начинающимся с `http или https`. При клике на URL-адрес zip-файл будет загружен из интернета и сохранен в директорию `downloadPath`, указанную в `motions.json`. Zip-файл анимации необходимо распаковать перед использованием.

### Сценарий 2: прямая интеграция BeautyAR SDK

Поместите новую папку анимации в соответствующую директорию `resource bundle` вашего проекта, в `2dMotionRes.bundle` для 2D-анимаций, в `3dMotionRes.bundle` для 3D-анимаций, в `ganMotionRes.bundle` для забавных анимаций или в `handMotionRes.bundle` для анимаций жестов. В BeautyAR SDK версии 3.6.0 или более ранней, если файл анимации зашифрован, необходимо скопировать его в песочницу и записать его путь. При использовании динамического подхода загрузки скачайте файл анимации в песочницу, распакуйте его и запишите путь к распакованной папке анимации. При использовании анимации вызовите метод SDK setEffect и передайте путь в SDK. См. [Effect Parameters](https://www.tencentcloud.com/document/product/1143/60207).

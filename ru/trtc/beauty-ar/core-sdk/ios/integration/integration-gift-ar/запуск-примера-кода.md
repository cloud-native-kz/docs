# Запуск примера кода

Tencent Gift AR SDK предоставляет демонстрационные проекты для различных платформ разработки. В этой статье основное внимание уделяется тому, как быстро запустить iOS Demo Gift AR SDK.

## Загрузка

[Нажмите для загрузки](https://mediacloud-76607.gzc.vod.tencent-cloud.com/MediaX/download/demo/TCEffectPlayerDemo_iOS.zip) проекта iOS Gift AR плеера Demo.

## Требования к окружению

- Xcode 9.0+.
- iPhone или iPad под управлением iOS 9.0 или более поздней версии.
- Проект имеет настроенную действительную подпись разработчика.

## Импорт проекта

Распакуйте загруженный zip-файл, перейдите в каталог TCEffectPlayerDemo_iOS, выполните `pod install` в терминале. После успешного выполнения нажмите TCEffectPlayerDemo.xcworkspace, чтобы открыть проект demo.

> **Примечание:** Библиотеки TCMediaX, TCEffectPlayer и YTCommonXMagic в Podfile могут быть не последней версии. Вы можете получить последнюю версию через официальный сайт.

## Изменение конфигурации

Замените информацию о лицензии в проекте. Откройте файл AppDelegate.m, найдите поля LICENCE_URL и LICENCE_KEY и замените соответствующие значения на информацию о лицензии Tencent Gift AR SDK, на которую вы подали заявку. Tencent Gift AR SDK поддерживает подачу заявки на пробную лицензию для тестирования. Дополнительную информацию о подаче заявки можно найти [здесь](https://trtc.io/document/60219?platform=android&product=beautyar).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/224e32b24b7511f0b25352540099c741.png)

## Конфигурация ресурса анимации

> **Примечание:** Если вам не нужно воспроизводить собственные файлы анимации, вы можете пропустить этот шаг.

Найдите Directory Resource. В настоящее время добавьте ресурс анимации в этот каталог. Одновременно настройте имя ресурса анимации для воспроизведения в файле ViewController.m.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/426672dc4b7511f08548525400454e06.png)

> **Примечание:** Встроенные файлы анимации в Demo не привязаны к лицензии. Поэтому вы можете использовать собственную лицензию для тестирования воспроизведения. Если вы хотите заменить собственную анимацию, учтите, что информация о лицензии анимации должна совпадать с информацией о лицензии, установленной в проекте. В противном случае воспроизведение не будет выполнено из-за несовпадения информации о лицензии.

## Компиляция и запуск

1. Убедитесь, что тестовое устройство iPhone или iPad уже подключено к Xcode.
2. Измените Bundle Id, соответствующий целевому объекту TCEffectPlayerDemo, на Bundle Id, который вы заполнили при подаче заявки на лицензию Gift AR плеера, импортируйте сертификат и запустите.

В этот момент консоль сначала выведет:

```
TCMediaXBase setLicense authresult: 0 ,errorMsg: { }
```

Это указывает на то, что проверка подлинности лицензии SDK уже прошла успешно.

> **Примечание:** Tencent Gift AR SDK требует сетевого подключения для первоначальной проверки подлинности лицензии. Поэтому при запуске Demo при запуске необходимо установить сетевое подключение.

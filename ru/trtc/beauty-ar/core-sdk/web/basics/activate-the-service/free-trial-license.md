# Бесплатная пробная лицензия

Чтобы улучшить ваш опыт работы с функцией Beauty AR SDK, мы предлагаем бесплатный пробный период для каждого SDKAppID. На веб-платформе Web&H5 каждый AppID может использовать бесплатный пробный период в течение 14 дней с возможностью однократного продления, в общей сложности 28 дней.

В этой статье содержатся рекомендации по [запросу пробной лицензии](#webapplytrial), [продлению пробной лицензии](#webrenewtrial) и [обновлению пробной лицензии](#webupgradetrial) для Beauty AR SDK Web&H5.

## Функции бесплатного пробного периода

Функции, которые пользователи могут тестировать бесплатно, включают:

| **Функции** | **Описание** |
| --- | --- |
| Beauty | Отбеливание, сглаживание, складки носогубные, резкость, скулы, тонкое лицо, V-образная форма, лоб, маленькая голова, губы, нос, большие глаза, яркость глаз, темные круги, подбородок |
| Filters | Раскраска изображений в реальном времени |
| Makeup | Полный набор модных эффектов макияжа, которые можно быстро применить для естественного улучшения внешности. |
| 2D stickers | Забавные виджеты для лица |
| 3D stickers | Реалистичные 3D анимированные стикеры |
| Virtual background | Фон может быть размыт, настроен или сделан прозрачным для удаления людей из кадра. |
| Animojis | Animojis поддерживают зеркальное отражение мимики лица в реальном времени. |
| Virtual avatars | Технология распознавания ключевых точек лица используется вместе с ИИ и компьютерной графикой для точного моделирования выражений лица. Поддерживается настройка модели. |
| Domain | При запросе бесплатного пробного периода поддерживаются только абсолютные имена доменов |

## Запрос пробной лицензии

Вы можете активировать бесплатный пробный период Beauty AR SDK Web&H5 в консоли TRTC. Конкретные действия выглядят следующим образом:

1. Войдите в [TRTC Console > Relevant Services](https://console.trtc.io/aboutcloud) и нажмите **Get Started.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9c49b2605fb11efa2e5525400488742.png)

2. Выберите **Web&H5** и заполните `Project Name` и `Domain` в соответствии с вашей ситуацией. После завершения нажмите **Confirm**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/898d79ea0c3011ef9ef35254002977b6.png)

3. После активации вы сможете просмотреть информацию на текущей странице и обратиться к приведенному ниже **Integration Guide** для интеграции. `License Key` и `License Token` будут использованы в [руководстве по интеграции](https://trtc.io/document/50099).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9d0f9c505fb11efa2e5525400488742.png)

## Продление пробной лицензии

Пробная лицензия изначально действительна 14 дней по умолчанию. После истечения срока вы можете продлить ее **один раз**, нажав **Renew** в нижней части поля информации о пакете, а затем нажав **Confirm** для продления функции еще на 14 дней.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9b5dbb705fb11ef8a9c5254003e1eb1.png)

## Обновление пробной лицензии

Если вам требуется обновить пробную лицензию Beauty AR SDK Web&H5 на официальную лицензию, чтобы продлить период ее действия, сначала [выберите и приобретите пакет Beauty AR SDK для Web&H5](https://console.trtc.io/subscription/buy/beauty), а затем выполните следующие действия:

1. Нажмите **Upgrade** в нижней части поля информации о пакете пробной лицензии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9c32d8a05fb11efa2e5525400488742.png)

2. Перейдите на **интерфейс обновления**, нажмите **Bind**, выберите неподключенный пакет Beauty AR SDK Web&H5 и нажмите **Confirm** для обновления и создания официального приложения с тем же именем пакета, при этом разблокировав официальную лицензию Beauty AR SDK без необходимости проверки издания. Если нет связываемых пакетов Beauty AR SDK, вы можете перейти на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty) для совершения покупки. **Обратите внимание, что ресурсы вступят в силу сразу же после покупки, и период действия начнется.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9c60faf05fb11ef9aef5254009d370c.png)

# Официальная лицензия

Beauty AR SDK поддерживает интеграцию с платформами **Web & H5** и **Mobile**. В этой статье описывается процесс **платформы Web&H5**: [Покупка официальной лицензии](#webpurchaseofficial), [Обновление периода действия официальной лицензии](#webupgradeofficial) и [Автоматическое продление](#webrenewal).

## Web&H5

Перед покупкой официальной версии мы предоставляем 28 дней бесплатного пробного периода на веб-сайте. Вы можете [Начать бесплатный пробный период](https://www.tencentcloud.com/document/product/1143/60219#).

### Покупка официальной лицензии

Вы можете выбрать и приобрести нужный вам пакет SDK на [странице покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty) и получить официальную лицензию для использования. **Период действия начинается с момента покупки, поэтому используйте его как можно скорее**. Описание различий между функциями разных версий SDK см. в разделе [Обзор биллинга](https://www.tencentcloud.com/document/product/1143/60214#).

Подробные инструкции приведены ниже:

1. Перейдите на [консоль TRTC > Соответствующие услуги](https://console.trtc.io/aboutcloud), нажмите **Начать** для входа в управление лицензией, затем нажмите **Создать официальную лицензию**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cef683f4053311efa7e752540052a3fc.png)

2. Выберите версию пакета, введите `Project Name` и `Domain` официального приложения, нажмите **Привязать**, чтобы выбрать **непривязанный** пакет Beauty AR SDK Web&H5 (если нет доступных пакетов для привязки, вы можете перейти на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty) и приобрести один), затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/820292db0c4c11ef9ef35254002977b6.png)

3. После успешного создания официальной лицензии страница отобразит информацию о созданной лицензии. Вы можете ознакомиться с **руководством интеграции** ниже для выполнения интеграции. `License Key` и `License Token` будут использованы в руководстве интеграции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a3f2b87e054411ef8a9c5254003e1eb1.png)

### Обновление периода действия официальной лицензии

Вы можете войти в [консоль TRTC > Соответствующие услуги](https://console.trtc.io/aboutcloud), нажать **Начать** для входа на страницу **Управление лицензией** и проверить период действия официальной лицензии Beauty AR SDK Web&H5. Официальные лицензии Beauty AR будут отправлять вам напоминания об истечении за 32 дня, 7 дней, 3 дня и 1 день до истечения срока действия, напоминая вам вовремя продлить лицензию, чтобы избежать нарушения нормальной работы бизнеса. Если срок действия вашей официальной лицензии Beauty AR SDK Web&H5 подходит к концу, выполните следующие шаги для продления:

1. Выберите лицензию, которую хотите продлить, и нажмите **Обновить период действия** в нижней части информационного блока пакета.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5a4132f2054511ef99495254007c64fd.png)

2. Beauty AR License поддерживает два способа продления периода действия: **Продлить текущую лицензию** и **Выбрать другой ресурс лицензии для замены**, как показано ниже.

> **Примечание:** **Ресурсы с включенным автоматическим продлением не поддерживают способ замены ресурса лицензии при продлении.** Если вы хотите изменить период действия на период другой лицензии, отключите функцию автоматического продления.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9635cea9060711ef9aef5254009d370c.png)

Продлить текущую лицензию

Выбрать другой ресурс лицензии для замены

1. Выберите **Продлить текущую лицензию**, затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6e9896ef053b11ef9f9c525400d4e181.png)

> **Примечание:** При выборе **Продлить текущую лицензию**: когда ресурс лицензии находится в действительном периоде или истек менее семи дней назад, согласно правилам биллинга Tencent Cloud поддерживается продление текущей лицензии. После продления период действия будет обновлен на: текущий период действия + период продления. Когда ресурс лицензии истек более семи дней назад, исходный ресурс был удален, и мы приобретем для вас ресурс того же типа и автоматически завершим замену привязки. После продления период действия будет обновлен на: дата платежа + период продления.

2. На странице **Продление** выберите **Период продления**. Beauty AR Web конечная точка продлевается периодически **Ежемесячно**. Нажмите **Подтвердить**, чтобы продлить срок действия лицензии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dae0a007054411efa7e752540052a3fc.png)

1. Выберите **Выбрать другой ресурс лицензии для замены**, затем нажмите **Подтвердить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e163b94e053b11ef81955254003a4131.png)

2. На странице **Обновить период действия** нажмите **Привязать**, выберите **Непривязанный** пакет Beauty AR SDK Web&H5 (если нет доступных пакетов для привязки, перейдите на [страницу покупки Beauty AR SDK](https://console.trtc.io/subscription/buy/beauty) и приобретите один), затем нажмите **Подтвердить**.

> **Примечание:** В настоящее время продление периода действия пакета поддерживает только пакеты одного типа. То есть, если тип привязанного пакета лицензии — Standard, то при продлении можно выбрать только пакет Standard. Если вы хотите изменить тип привязанного пакета, необходимо [Связаться с нами](https://trtc.io/contact) или обратиться к команде бизнеса для обработки.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/31b0f95b05fa11ef8b3f5254007c64fd.png)

3. Проверьте период действия после обновления.

> **Примечание:** **Beauty AR Official License не поддерживает изменение информации**. Если вам нужно изменить информацию лицензии, не используйте приобретенный пакет ресурсов для обновления периода действия лицензии. Вместо этого нажмите **Создать официальную лицензию**, чтобы пересоздать приложение, добавить новую лицензию и привязать информацию нового имени пакета.

### Автоматическое продление

Рекомендуется использовать опцию **Включить автоматическое продление** для продления. После включения лицензия будет автоматически продлеваться ежемесячно после истечения срока действия, при условии что на вашем аккаунте **достаточно средств**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6b59fb03054411efa7e752540052a3fc.png)

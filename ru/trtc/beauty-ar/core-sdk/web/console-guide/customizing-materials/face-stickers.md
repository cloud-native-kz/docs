# Наклейки на лице

Вы можете загрузить файл PNG или последовательность кадров PNG в консоль для создания собственных наклеек с отслеживанием лица.

## Ограничения

| Требование | Описание |
| --- | --- |
| Формат | Для статических наклеек загрузите одно изображение PNG. Для анимированных наклеек загрузите последовательность **пронумерованных** изображений PNG, например `001.PNG`, `002.PNG`. |
| Размеры | Рекомендуется 1000 x 1000 пиксель или меньше. |
| Количество кадров | 100 кадров или менее. |
| Размер файла | Один файл изображения не должен превышать 1 МБ. |

## Инструкции

### Шаг 1. Создайте ваш материал

### Шаг 2. Импортируйте материал в консоль

2. В области **Choose material** (Выбор материала) панели **Parameter configuration** (Конфигурация параметров) справа загрузите ваш материал, используя перетаскивание или обзор.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/37488b04e01611ed95af525400c56988.png)

### Шаг 3. Измените параметры

- На основной панели перетащите изображение наклейки, чтобы позиционировать её относительно лица.
- На панели конфигурации параметров справа измените параметры, включая **режим наложения**, **интенсивность**, **скорость воспроизведения** (для последовательности кадров) и **точку привязки**.

> **Примечание** Мы не рекомендуем устанавливать скорость воспроизведения выше 30 кадров в секунду. **Точка привязки** — это опорная точка наклейки, которая по умолчанию является центром изображения наклейки. X и Y обозначают горизонтальное и вертикальное расстояния (в процентах) точки привязки от верхнего левого угла изображения наклейки. Например, точка привязки по умолчанию — (0.5, 0.5). Если вы хотите использовать верхний левый угол наклейки в качестве точки привязки, установите параметр (0, 0). Правильно выбранная точка привязки обеспечивает лучшие эффекты отслеживания. Например, для наклеек-украшений на голову рекомендуется точка привязки на лбу, а для наклеек, таких как очки, рекомендуется точка привязки вдоль переносицы. Вы можете продолжить экспериментировать, чтобы найти лучшую точку привязки для вашей наклейки.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/49347317e01611edbd56525400088f3a.gif)

### Шаг 4. Просмотрите материал

В окне предпросмотра справа вы можете просмотреть свою наклейку на встроенном фото или с помощью камеры вашего устройства.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/643d6585e01611edbd56525400088f3a.png)

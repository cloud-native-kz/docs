# Запуск примера кода

**Beauty AR Web** позволяет реализовать на веб-сайтах функции AR-красоты, фильтры, макияж, стикеры и другие возможности. В этом документе описано, как быстро запустить веб-приложение с поддержкой красоты в реальном времени локально. На основе этого приложения вы можете реализовать другие функции согласно инструкциям в соответствующих документах.

> **Примечание:** Этот проект — тестовый проект, предназначенный только для локального тестирования. Для официального запуска проекта с возможностями Beauty AR Web необходимо приобрести официальную лицензию и настроить домен веб-сайта в консоли продукта. Дополнительные сведения см. в разделе [Официальная лицензия](https://trtc.io/document/60213?platform=web&product=beautyar).

## Шаг 1. Создание лицензии

### Подача заявки на пробную лицензию

Убедитесь, что вы создали веб-лицензию согласно инструкциям в разделе [Подача заявки на пробную лицензию](https://www.tencentcloud.com/document/product/1143/60218#).

Заполните название проекта и введите локальный адрес разработки для домена, например `test.webar.com`. Нажмите кнопку подтверждения для завершения создания.

- Веб-лицензия авторизована для доменного имени, поэтому заполните фактический домен вашего проекта (не вводите localhost:xxxx).
- При использовании веб-лицензии информация о домене будет проверена, поэтому очень важно убедиться, что домен, используемый при фактической работе, совпадает с тем, который был введен при подаче заявки на лицензию, **исключая адреса localhost**. Поскольку адрес localhost автоматически подтверждается как успешный, даже если выше был заполнен `test.webar.com`, вы все равно можете использовать `localhost:xxxx` как адрес разработки для локального запуска демонстрации без проблем.

> **Примечание:** Пробная лицензия действительна только 14 дней (может быть продлена один раз на дополнительные 14 дней, всего 28 дней). Каждый аккаунт может создать максимум одну тестовую лицензию. Для официальных проектов посетите страницу [Официальная лицензия](https://trtc.io/document/60213?platform=web&product=beautyar) для покупки по мере необходимости.

### Получение App ID, ключа лицензии и токена

После создания вы можете увидеть информацию созданного тестового проекта в списке проектов и получить `License Token` для Beauty AR Web и `License key` для тестового проекта. Вы также можете получить `App ID`.

> **Примечание:** **Токен лицензии** используется для вычисления подписи аутентификации, поэтому убедитесь, что он хранится в безопасности и конфиденциальности. Здесь токен используется для вычисления подписи на фронтенде просто для помощи в запуске демонстрации локально. В реальной производственной среде необходимо реализовать алгоритм подписи на сервере.

## Шаг 2. Локальный запуск

1. Загрузите код [примера проекта](https://github.com/tencentcloud-webar/web-demo-en/tree/master/quick-start).

```
 git clone https://github.com/tencentcloud-webar/web-demo-en.git cd quick-start
```

2. Замените указанные элементы конфигурации в `index.js` на ключ лицензии, токен и `APPID`, полученные на [шаге 1](https://www.tencentcloud.com/document/product/1143/68963#.E6.AD.A5.E9.AA.A41.EF.BC.9A.E5.88.9B.E5.BB.BA-license.5B.5D(id.3Astep1)), как показано ниже:

```
/** ----- Authentication configuration ----- *//** * Tencent Cloud account's APPID *  * You can also view your APPID in the [Account Center](https://console.tencentcloud.com/developer). */const APPID = ''; // Set it to your Tencent Cloud account APPID./** * Web LicenseKey *  * obtained from Before You Start */const LICENSE_KEY = ''; // Set it to your license key./** * The token used to calculate the signature. *  * Note: This method is only suitable for debugging. In a production environment, you should store the token and calculate the signature on your server. The front end can get the signature by calling an API. For details, see * https://trtc.io/document/50099?platform=web&product=beautyar#e4ce3483-41f7-4391-83ae-f4b61e221ea0 */const token = ''; // Set it to your token./** ----------------------- */
```

3. Запустите проект в локальной среде разработки.

> **Примечание:** Перед локальным запуском проекта убедитесь, что на устройстве установлена среда `nodejs`.

Последовательно выполните следующие команды в каталоге проекта и откройте `localhost:8090` в браузере для проверки возможностей Beauty AR Web.

```
# Install dependenciesnpm i # Compile and run the codenpm run dev
```

После выполнения приведенных выше шагов вы можете проверить фильтры и эффекты SDK для настольного браузера. Вы можете использовать встроенные материалы для проверки различных фильтров и эффектов макияжа согласно инструкциям в разделе [Начало интеграции](https://www.tencentcloud.com/document/product/1143/68777), или использовать дополнительные возможности Beauty AR Web, такие как пользовательские стикеры, макияж и фильтры. Подробные инструкции по настройке материалов эффектов см. в [Руководство по настройке материалов](https://trtc.io/document/53887?platform=web&product=beautyar).

# Примечания к выпуску

> **Примечание:** Версия Web SDK V2 официально выпущена. По сравнению с V1, V2 предлагает значительные улучшения производительности, а интерфейсы API обратно совместимы. Мы настоятельно рекомендуем обновиться до V2, чтобы享受 улучшенный пользовательский опыт. Обратите внимание, что в будущем версия V1 будет получать только исправления ошибок и не будет предоставлять обновления новых функций. Поэтому, чтобы обеспечить доступ к новейшим функциям и оптимальную производительность, пожалуйста, завершите обновление как можно скорее.

### SDK Версия 2.0.2 @ 2026-01-07

- Исправлена проблема, при которой инициализация SDK не устанавливала параметр **module**, что приводило к ошибке.
- Отключены все обнаружения, включая распознавание жестов, при вызове интерфейса **disable**.
- Исправлена проблема, при которой интерфейс **disable** не отключал эффекты отбеливания и сглаживания кожи.
- Исправлена проблема, при которой повторное включение и выключение камеры в Windows вызывало неожиданное зависание.
- Исправлены другие проблемы.

# Управление пакетами

TRTC позволяет просматривать использование пакетов в вашей учетной записи через консоль. Вы можете нажать [Управление пакетами > Мои пакеты](https://console.trtc.io/package) на левой боковой панели, чтобы просмотреть информацию о пакетах в списке пакетов.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/526ba4b6c06211f0b4c35254001c06ec.png)

## Список пакетов

> **Примечание:** Типы пакетов разделяются на Пробный период и ежемесячный пакет. Первый позволяет вычитать время аудио и видео, записи, микстранскодирования, последний позволяет вычитать время аудио и видео. Конкретные детали коэффициента вычитания можно найти в [описании свободного времени](https://trtc.io/document/42735).

| Функции | Описание |
| --- | --- |
| Название пакета | Название приобретенного пакета поддерживает следующие варианты: RTC Engine - StandardRTC Engine - ProTRTC Free Trial1-to-1 CallGroup Call |
| ID пакета | Система автоматически генерирует уникальный ID пакета после успешной покупки |
| Тип пакета | Существует два типа пакетов: пакет свободного времени и ежемесячный пакет, из которых: Пакеты Standard/Professional — это приобретенные клиентом продукты, включающие Call/RTC Engine. Пробный период — это подарок платформы, распределяемый на 12 месяцев за раз, 10 000 минут в месяц |
| Длительность пакета (мин) | Количество минут, включенное в пакет |
| Оставшееся время этого цикла (минуты)/（%） | По количеству оставшихся минут и доле использования вы можете более наглядно увидеть использование пакета |
| Время начала этого цикла | Время успешной покупки пакета |
| Время окончания этого цикла | Время истечения срока действия пакета |
| Статус этого цикла | Возвращено, Истекло, Исчерпано, Не используется, Используется |

## Покупка пакета

Если вашим реальным потребностям бизнеса требуется продление пакета, перейдите на [Управление пакетами > Подписка на пакеты](https://console.trtc.io/subscription), выберите **продукт, для которого вам нужно приобрести пакет**, выберите **нужную вам версию** и нажмите **Подписаться сейчас**, чтобы завершить покупку.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7474f464cb6611f0b011525400bf7822.png)

> **Примечание:** Перед покупкой пакета убедитесь, что вы понимаете информацию о выставлении счетов. Для получения дополнительных сведений см. [Обзор выставления счетов](https://trtc.io/document/34610).

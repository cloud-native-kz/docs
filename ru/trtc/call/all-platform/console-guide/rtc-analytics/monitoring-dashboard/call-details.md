# Подробная информация о вызове

В этом документе описана информация, которую можно просмотреть на странице подробной информации конкретной комнаты вызовов в информационной панели мониторинга.

## Инструкции

1. Перейдите в [консоль Tencent RTC > Room Call Survey](https://console.trtc.io/monitor) (информационная панель мониторинга) и выберите комнату, которую вы хотите просмотреть.
2. Нажмите **Room ID** или **View Call Details** справа, чтобы перейти на страницу подробной информации.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ed8f28aea61d11eeae9a525400c26da5.png)

## Отображаемая информация

### Основная информация

- **Информация о комнате вызовов:** основная информация о комнате вызовов, включая **SDKAppID**, **Application Name**, **Room ID**, **User Count**, **Start Time - End Time** и **Room Duration**.
- **Текущий диапазон времени:** диапазон времени запроса. За один запрос можно отобразить вызовы продолжительностью до 5 часов.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/00b941c7a61e11eeae9a525400c26da5.png)

### Список пользователей

В этом разделе показана информация о пользователях вызова, включая идентификатор пользователя, роль, тип, регион, время пребывания в выбранном диапазоне времени, время входа и выхода из комнаты, продолжительность пребывания, версию SDK, тип SDK, а также информацию об устройстве и сети.

| Элемент | Описание |
| --- | --- |
| User ID | По умолчанию отображаются 6 последних присоединившихся пользователей. Чтобы отобразить более 6 пользователей, нажмите **Add user** или **View All Users**. Можно отобразить максимум 20 пользователей. |
| User Role | Якорь или аудитория. Якори могут отправлять и получать данные, а аудитория может только получать данные. |
| User Type | Входящий или исходящий. Зеленая стрелка вверх указывает на то, что пользователь отправил данные, а желтая стрелка вниз указывает на то, что пользователь получил данные. |
| Region | Географическое местоположение пользователя |
| Userâs Stay in This Period | Синий сегмент представляет период нахождения пользователя в комнате, а серый сегмент представляет период отсутствия пользователя в комнате. |
| Room In/Out Time | Время первого входа и последнего выхода пользователя. |
| Duration | Продолжительность пребывания пользователя в комнате. |
| SDK Version | Номер версии SDK пользователя. |
| SDK Type | Тип SDK, используемый пользователем. Описание см. в документации [SDK Download](https://www.tencentcloud.com/document/product/647/34615). |
| Device | Модель или номер устройства, используемого пользователем |
| Network | Тип сети, используемый пользователем |

### Полученные и отправленные данные

Пользователь в вызове получает данные других пользователей и отправляет локальные данные. Поэтому необходимо отобразить информацию о данных в обе стороны. По умолчанию страница подробной информации о вызове в информационной панели мониторинга отображает статистику с точки зрения [получателя](https://www.tencentcloud.com/document/product/647/39070#data-received-and-sent). Вы также можете переключиться на точку зрения [отправителя](https://www.tencentcloud.com/document/product/647/39070#data-received-and-sent).

#### Получатель

- Вы можете переключаться между четырьмя вкладками: **All**, **Video**, **Audio** и **Screen Sharing**. Вкладка отображается только если соответствующие данные получены.
- По умолчанию график показывает данные, полученные от всех удаленных пользователей, обозначенные разными цветами. Вы можете просмотреть данные одного пользователя, нажав **Select sender to view details** и выбрав указанного отправителя.
- Красные линии на графике указывают на дрожание сети. Вы можете нажать **Select sender to view details** в верхнем правом углу и выбрать соответствующий идентификатор пользователя, чтобы проверить подробную информацию в разделе [End-to-End Details](https://www.tencentcloud.com/document/product/647/39071).
- Если отображается большой диапазон времени, вы можете использовать колесико мыши для увеличения части графика и просмотра подробной информации о конкретной точке.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0ec8b5cc9a4811eea869525400c26cb9.png)

#### Отправитель

- Вы можете переключаться между четырьмя вкладками: **All**, **Video**, **Audio** и **Screen Sharing**. Вкладка отображается только если соответствующие данные отправлены.
- Красные линии на графике указывают на дрожание сети. Вы можете нажать **View details** в верхнем правом углу, чтобы проверить подробную информацию в разделе [End-to-End Details](https://www.tencentcloud.com/document/product/647/39071).
- Если отображается большой диапазон времени, вы можете использовать колесико мыши для увеличения части графика и просмотра подробной информации о конкретной точке.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/23c1e38a9a4811ee8964525400321be2.png)

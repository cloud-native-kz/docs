# Список вызовов

Tencent RTC предоставляет панель мониторинга для разработчиков, позволяющую отслеживать качество вызовов. В панели мониторинга можно просмотреть сведения о вызовах и узнать статус вызовов пользователей.

## Просмотр списка вызовов

Войдите в [консоль Tencent RTC > Панель мониторинга](https://console.trtc.io/monitor), чтобы просмотреть сведения о вызовах во всех комнатах учетной записи.

В списке вызовов на панели мониторинга по умолчанию отображается 10 записей вызовов на одной странице, отсортированных в обратном порядке по времени начала вызова, то есть первая запись — это наиболее свежая запись о вызове. Каждая запись содержит 6 полей: ID комнаты, Время начала — Время окончания, Статус вызова, Длительность сеанса, Присоединившиеся пользователи, Операция.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7ada9daba61e11eeae9a525400c26da5.png)

Информация, отображаемая в списке вызовов, включает:

| Элемент | Описание |
| --- | --- |
| ID комнаты | ID комнаты вызова |
| Время начала — Время окончания | Время начала и окончания вызова |
| Статус вызова | Включает два состояния: не завершен и завершен. |
| Длительность сеанса | Время между входом первого пользователя и выходом последнего пользователя. Если вызов идет, отображается время от входа первого пользователя до момента запроса. |
| Присоединившиеся пользователи | Общее количество пользователей, вошедших в комнату |
| Операция | Нажмите [Сведения о вызове](https://www.tencentcloud.com/document/product/647/39070), чтобы перейти на страницу сведений и просмотреть подробную информацию о вызове. |

## Поиск вызовов

На панели мониторинга можно отфильтровать список вызовов несколькими способами поиска.

- Чтобы просмотреть информацию о вызовах в комнатах определенного приложения, нажмите **Выбрать приложение**, выберите приложение из раскрывающегося списка и нажмите **Поиск**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/53d1378c9a3c11ee8964525400321be2.png)

- Чтобы просмотреть информацию о вызовах в определенный период времени, выберите **Сегодня**, **Вчера**, **Последние 6 часов** или введите диапазон времени и нажмите **Поиск**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5d07ad069a3c11ee8964525400321be2.png)

- Чтобы просмотреть информацию о конкретном вызове, выберите диапазон времени, введите ID комнаты (`roomid`) или ID пользователя (`userid`) и нажмите **Поиск**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/63dc11d39a3c11ee8964525400321be2.png)

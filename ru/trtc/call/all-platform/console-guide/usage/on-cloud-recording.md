# Облачная запись

Статистика использования облачной записи в консоли TRTC предназначена для отображения сведений об использовании облачной записи в TRTC по [длительности записи](https://trtc.io/document/52817#.E4.BA.91.E6.9C.8D.E5.8A.A1.E6.B6.88.E8.80.97). Если вы используете тарифицирование CSS по [пиковому маршруту одновременных подключений](https://www.tencentcloud.com/document/product/267/39605), вам необходимо перейти в [консоль CSS](https://console.tencentcloud.com/live/livestat) для проверки.

## Примечание

Данные обновляются каждые 5 минут, а не в реальном времени, поэтому отображение статистики может быть отложено на 5–180 минут.

### Инструкции

1. Перейдите в [консоль Tencent RTC > Использование](https://console.trtc.io/statistics), выберите **Облачная запись**.
2. Выберите приложение, которое вы хотите просмотреть, а затем выберите период времени.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2d200b95cb6811f0a4a55254001c06ec.png)

## Сведения об использовании

Длительность рассчитывается в секундах и округляется до минут. Поэтому совокупная используемая длительность для выставления счетов может немного отличаться от вашего фактического использования. Для получения фактической суммы к счету см. счета, созданные в [Центре выставления счетов](https://console.intl.cloud.tencent.com/expense/bill).

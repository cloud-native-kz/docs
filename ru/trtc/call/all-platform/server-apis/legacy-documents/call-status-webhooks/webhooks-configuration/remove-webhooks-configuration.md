# Удаление конфигурации Webhooks

## Обзор функции

Администратор может удалить callback через этот интерфейс.

## Описание API

### Пример URL запроса

```
https://xxxxxx/v1/callback/delete?sdkappid=88888888&identifier=administrator&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, изменяемые при вызове этого API, и их описание. Для получения дополнительной информации см. [Обзор REST API](https://www.tencentcloud.com/document/product/647/60138#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: `callkit-intl.trtc.tencent-cloud.com` |
| v1/callback/delete | API запроса |
| sdkappid | Sdkappid, назначенный консолью при создании приложения |
| identifier | Должна быть учетная запись администратора приложения Chat |
| usersig | Подпись, сгенерированная учетной записью администратора приложения, подробные инструкции см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/39074#) |
| random | Введите случайное 32-битное целое число без знака, диапазон от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

10 раз в секунду.

### Примеры пакетов запроса

Достаточно передать пустое значение.

```
{}
```

### Примеры пакетов ответа

```
{    "errorCode": 0,    "errorMessage": "Success",    "requestId": "5b0fa500064397cad3554506e27e18e1",    "data": ""}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| errorCode | Integer | Код ошибки, 0 указывает на успех |
| errorMessage | String | Сообщение об ошибке |
| requestId | String | Уникальный ID запроса |
| data | String | Нет данных |

## Коды ошибок

| Код ошибки | Описание |
| --- | --- |
| 0 | Запрос выполнен успешно |
| 50001 | Текущее приложение должно приобрести пакет версии группового вызова TUICallKit для использования |

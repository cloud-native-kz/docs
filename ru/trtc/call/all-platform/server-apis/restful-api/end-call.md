# Завершение звонка

## Описание функции

Администратор приложения может использовать этот API для завершения звонка.

## Описание вызова API

### Пример URL запроса

```
https://console.tim.qq.com/v4/call_engine_http_srv/end_call?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, используемые при вызове этого API, и их описание. Дополнительную информацию о параметрах см. в разделе [REST API Introduction](https://www.tencentcloud.com/document/product/647/68926).

| Параметры | Описание |
| --- | --- |
| xxxxxx | SDKAppID расположен в стране/регионе выделенного доменного имени: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/call_engine_http_srv/end_call | Запрашиваемый API. |
| sdkappid | SDKAppID, назначенный консолью IM при создании приложения. |
| identifier | Должно быть учетной записью администратора приложения. Дополнительные сведения см. в разделе App admin. |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробности см. в разделе [generate UserSig](https://www.tencentcloud.com/document/product/647/35166). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

200 запросов в секунду.

### Пример пакета запроса

**Базовая форма**

```
{  "CallId": "055662e1-bc8a-469c-a334-1126c8c17d58"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CallId | String | Обязательно | ID звонка |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-431454f25a44462d8155bdff4fed38cc-O-Seq-19029769"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на отказ. |
| ErrorInfo | String | Сообщение об ошибке. |
| ActionStatus | String | Результат обработки запроса. OK: обработка выполнена успешно; FAIL: обработка не удалась. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для локализации проблемы. |

## Описание кодов ошибок

Если не происходит ошибка сети (например, ошибка 502), код возврата HTTP для этого API равен 200. Фактический код ошибки и информация об ошибке представлены ErrorCode и ErrorInfo в полезной нагрузке ответа.

Распространенные коды ошибок (60000–79999) см. в документе [Error Code](https://www.tencentcloud.com/document/product/647/54901).

Собственные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 101001 | Внутренняя ошибка сервера, повторите попытку. |
| 101002 | Недопустимый параметр. Проверьте, правильно ли сформирован запрос в соответствии с описанием ошибки. |
| 101004 | Звонок не существует, или он мог существовать, но уже завершился. |

# Соглашение об обработке и защите данных

## 1. ПРЕДПОСЫЛКИ

Этот модуль применяется, если вы используете функцию Tencent Real-Time Communication (далее **«Функция»**). Этот модуль включен в Соглашение об обработке и защите данных, расположенное по адресу ([DPSA](https://intl.cloud.tencent.com/document/product/301/17347)). Термины, используемые, но не определенные в этом модуле, имеют значение, которое им присваивается в DPSA. В случае любого конфликта между DPSA и этим модулем этот модуль применяется в той мере, в которой существует несоответствие.

## 2. ОБРАБОТКА

Мы будем обрабатывать следующие данные в связи с Функцией:

| **Персональная информация** | **Использование** |
| --- | --- |
| **Аудио- и видеоконтент конечных пользователей** | Мы обрабатываем эти данные только для целей предоставления Функции вам, то есть для передачи аудиовизуального контента (мы не храним такие данные для этой цели). |
| **Информация о временном ключе клиента:** SDK APPID, имя пользователя, закрытый ключ | Мы используем эту информацию для идентификации вашего использования Функции. |
| **Данные журналов клиента, относящиеся к приложениям:** SDK APPID, имя приложения, тег, статус службы, время создания, операция | Мы используем эту информацию для ведения записей, связанных с приложениями, созданными вами с использованием Функции. |

## 3. РЕГИОН ОБСЛУЖИВАНИЯ

Как указано в DPSA.

## 4. СУБПРОЦЕССОРЫ

Как указано в DPSA.

## 5. СОХРАНЕНИЕ ДАННЫХ

Мы будем хранить персональные данные, обрабатываемые в связи с Функцией, следующим образом:

| **Персональная информация** | **Политика сохранения** |
| --- | --- |
| **Информация о временном ключе клиента:** SDK APPID, имя пользователя, закрытый ключ | Мы сохраняем такие данные в течение всего времени использования вами Функции. После прекращения использования вами Функции или удаления вашей учетной записи мы удалим эти данные в течение 7 дней. |
| **Данные журналов клиента, относящиеся к приложениям:** SDK APPID, имя приложения, тег, статус службы, время создания, операция | Мы сохраняем такие данные в течение всего времени использования вами Функции. После прекращения использования вами Функции или удаления вашей учетной записи мы удалим эти данные в течение 7 дней. |

Вы можете запросить удаление такой персональной информации в соответствии с DPSA.

## 6. СПЕЦИАЛЬНЫЕ УСЛОВИЯ

Вы должны гарантировать, что эта Функция используется только конечными пользователями, которые достигли минимального возраста, требуемого для согласия на обработку их персональных данных, или что получено согласие родителя для лиц младше минимального возраста. Это может различаться в зависимости от юрисдикции, в которой находится конечный пользователь.

Вы гарантируете, подтверждаете и обещаете, что получите и будете поддерживать все необходимые согласия от конечных пользователей в отношении обработки их персональных данных в отношении Функции в соответствии с применимым законодательством и таким образом, чтобы позволить нам соответствовать применимому законодательству.

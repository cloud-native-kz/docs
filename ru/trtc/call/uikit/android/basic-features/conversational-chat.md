# Conversational Chat

Chat предоставляет многоплатформенные API чата, компоненты пользовательского интерфейса и серверные API, позволяя вам создать полнофункциональное приложение для чата за десять минут. В приложении для чата вы можете легко добавить функции аудио и видеовызовов всего с помощью трёх строк кода.

## Демонстрация интеграции

После интеграции Chat и TUICallKit вы можете легко инициировать аудио и видеовызовы на странице сообщений чата и странице информации о пользователе.

## Быстрая интеграция

### Подготовка

- Убедитесь, что ваше приложение [интегрировало Chat](https://trtc.io/document/60520?platform=android&product=chat&menulabel=uikit).

### Активация сервиса

1. Войдите в [консоль чата](https://console.trtc.io/) для активации аудио и видеосервисов. Подробные шаги см. в документации [Activate Service](https://trtc.io/document/60220).
2. В диалоговом окне активации сервиса TRTC нажмите "Confirm". Система создаст приложение TRTC с тем же SDKAppID, что и текущее приложение для чата. [Консоль TRTC](https://trtc.io/login?s_url=https://console.trtc.io/) позволяет повторно использовать учётную запись и аутентификацию.

### Интеграция TUICallKit

Добавьте зависимость TUICallKit в файл build.gradle приложения:

```
api project(':tuicallkit-kt')
```

После интеграции компонента TUICallKit интерфейс чата и интерфейс информации контактов по умолчанию будут отображать кнопки "видеовызов" и "голосовой вызов". Когда пользователь нажимает кнопку, TUICallKit автоматически отобразит UI приглашения на вызов и отправит запрос на вызов другой стороне.

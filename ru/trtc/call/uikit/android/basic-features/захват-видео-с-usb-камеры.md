# Захват видео с USB-камеры

В этой статье рассказывается о том, как интегрировать USB-камеру с компонентом `TUICallKit` на устройствах Android для включения функции видеозвонков. В настоящее время эта функция поддерживается компонентами Android TUICallkit и Flutter (Android) TUICallKit.

## Подготовка окружения

- Android 5.0 (SDK API Level 21) или выше.
- Gradle 4.2.1 или выше.
- Устройства на базе Android: мобильный телефон, планшет или другие пользовательские устройства.

## Шаг 1: Подготовка условий

1. Перед использованием функции USB Camera, предоставляемой Tencent Cloud, необходимо перейти в консоль, активировать аудио- и видеослужбы для вашего приложения и приобрести план группового звонка. Подробные шаги см. в разделе [Активация сервиса](https://www.tencentcloud.com/document/product/647/59832#).
2. Этот плагин должен использоваться вместе с компонентом TUICallKit. Сначала интегрируйте компонент TUICallKit:
  - [Android](https://www.tencentcloud.com/document/product/647/50991#)
  - [Flutter](https://www.tencentcloud.com/document/product/647/54896#)

## Шаг 2: Интеграция компонента

В файле `build.gradle` в директории приложения вашего проекта добавьте следующий код зависимости:

```
implementation "io.trtc.uikit:usb-camera:latest.release"
```

После выполнения вышеописанных шагов вы сможете использовать внешнюю камеру для видеозвонков в компоненте `TUICallKit`.

## Часто задаваемые вопросы

### Невозможно открыть камеру на устройствах Android 9 и Android 10?

**Причина**: На Android 9 и Android 10 при запросе USB-разрешений также проверяются разрешения камеры. Однако из-за аномалии в фреймворке Android даже если разрешения камеры предоставлены, проверка все равно не пройдет, что предотвращает предоставление USB-разрешений.

**Решение**: Измените targetSdkVersion приложения на 27 или ниже.

```
defaultConfig {
    targetSdkVersion 27
}
```

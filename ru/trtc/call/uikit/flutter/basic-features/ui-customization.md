# Настройка пользовательского интерфейса

В этой статье рассказывается о том, как настроить пользовательский интерфейс TUICallKit. Мы предоставляем два решения на выбор: **решение точной настройки интерфейса** и **решение самостоятельной реализации пользовательского интерфейса**.

Примечание: Решение по настройке страницы требует использования плагина [tencent_calls_uikit](https://pub.dev/packages/tencent_calls_uikit) версии 1.8.0 или выше.

## Решение 1. Незначительная настройка пользовательского интерфейса

Вы можете загрузить последнюю версию плагина [tencent_calls_uikit](https://pub.dev/packages/tencent_calls_uikit/versions) локально, а затем использовать метод локальной зависимости для подключения плагина к вашему проекту. Метод локальной зависимости выглядит следующим образом:

В узле **dependencies** файла проекта **pubspec.yaml** добавьте зависимость плагина **tencent_calls_uikit**, как показано ниже:

```
dependencies:    tencent_calls_uikit:        path: your file path
```

### Замена иконок

Вы можете напрямую заменить иконки в папке **assets\\images**, чтобы обеспечить согласованность цветовой гаммы иконок во всем приложении. При замене сохраняйте имя файла иконки без изменений.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/95eccefd372811eeb231525400c56988.png)

### Замена рингтона

Вы можете заменить три аудиофайла в папке **assets\\audios** для достижения цели замены рингтона:

| Название | Назначение |
| --- | --- |
| phone_dialing.mp3 | Звук набора номера |
| phone_hangup.mp3 | Звук разъединения |
| phone_ringing.mp3 | Рингтон входящего вызова |

### Замена текста

Вы можете изменить содержание строк на интерфейсе видеовызова, отредактировав строки в файле **strings.g.dart** в директории **lib\\src\\i18n**.

## Решение 2. Реализация пользовательского интерфейса

Вся функциональность вызовов TUICallKit реализована на основе компонента без пользовательского интерфейса TUICallEngine. Вы можете напрямую использовать API TUICallEngine, предоставляемый [tencent_calls_uikit](https://pub.dev/packages/tencent_calls_uikit/versions), для полной реализации собственного интерфейса на основе TUICallEngine.

### TUICallEngine

[TUICallEngine](https://www.tencentcloud.com/document/product/647/54907) — это базовый API всего компонента вызовов, предоставляющий основные API для одноранговых и групповых аудио- и видеовызовов, включая инициирование, ответ, отклонение, завершение, а также управление устройствами.

### TUICallObserver

[TUICallObserver](https://www.tencentcloud.com/document/product/647/54908) — это класс событий обратного вызова для TUICallEngine. Вы можете получать интересующие вас события обратного вызова через этот наблюдатель.

# Примечания к выпуску

Подробный журнал обновлений Flutter можно найти на [странице изменений](https://pub.dev/packages/tencent_calls_uikit/changelog) на сайте pub.dev.

> **Примечания для обновления версии 2.9.0:** Для пользователей, которым необходимо обновиться до API calls, обратите внимание на постепенное развертывание версии. Если вы выполняете вызов с использованием API calls, а версия другой стороны не была обновлена, это приведет к сбою вызова.

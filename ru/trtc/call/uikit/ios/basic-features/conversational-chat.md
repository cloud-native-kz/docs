# Conversational Chat

Chat предоставляет многоплатформенные API чата, компоненты пользовательского интерфейса и серверные API, позволяя вам создать полнофункциональное приложение чата за десять минут. В приложении чата вы можете легко добавить функции аудио и видеозвонков всего тремя строками кода.

## Демонстрация интеграции

После интеграции Chat и TUICallKit вы можете легко инициировать аудио и видеозвонки на странице сообщений чата и странице информации о пользователе.

## Быстрая интеграция

### Подготовка

- Убедитесь, что ваше приложение [интегрировало Chat](https://trtc.io/document/60521?platform=ios%20and%20macos&product=chat&menulabel=uikit).

### Активация сервиса

1. Войдите в [консоль чата](https://console.trtc.io/) и активируйте сервисы аудио и видео. Подробные инструкции см. в документации [Активация сервиса](https://trtc.io/document/60220).
2. В диалоговом окне активации сервиса TRTC нажмите "Подтвердить". Система создаст приложение TRTC с тем же SDKAppID, что и текущее приложение чата. [Консоль TRTC](https://trtc.io/login?s_url=https://console.trtc.io/) позволяет повторно использовать учетную запись и аутентификацию.

### Интеграция TUICallKit

Добавьте следующее содержимое в ваш Podfile:

```
// Integrate the TUICallKit componentpod 'TUICallKit'
```

После интеграции компонента TUICallKit интерфейс чата и интерфейс информации о контактах по умолчанию будут отображать кнопки "видеозвонок" и "голосовой вызов". Когда пользователь нажимает кнопку, TUICallKit автоматически отображает пользовательский интерфейс приглашения на звонок и отправляет запрос на приглашение на звонок другой стороне.

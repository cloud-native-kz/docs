# Языковые настройки

## Поддерживаемые языки

В настоящее время поддерживаются **упрощённый китайский, английский, японский и арабский языки.**

## Переключение языка

TUICallKit по умолчанию использует тот же язык, что и операционная система телефона. Если вам нужно переключить язык, вы можете использовать `TUIGlobalization.setPreferredLanguage` для переключения языков. В качестве примера приведено переключение на английский язык:

```
…import TUICorefunc steLanguage() {    TUIGlobalization.setPreferredLanguage("en")}
```

## Добавление нового языка

### Шаг 1: Интеграция исходного кода

1. Клонируйте/загрузите код из [GitHub](https://github.com/Tencent-RTC/TUICallKit).
2. Загрузите зависимость исходного кода TUICallKit локально в файл Podfile проекта приложения.

```
  target 'TUICallKitApp' do  use_frameworks!  …  pod 'TUICallKit-Swift/Professional', :path => "Your Download Path/TUICallKit/iOS"  end
```

3. Запустите команду `pod update` для обновления зависимостей.

### Шаг 2: Добавление нового языкового пакета

#### На примере **испанского языка:**

1. Добавьте новый файл испанского языка.

Перейдите в каталог исходных файлов TUICallKit, в папку iOS/TUICallKit-Swift/Resources, и создайте новый файл `es.lproj/Localized.strings`.

2. Скопируйте содержимое файла `iOS/TUICallKit-Swift/Resources/en.lproj/Localized.strings` в новый файл `iOS/TUICallKit-Swift/Resources/es.lproj/Localized.strings`.
3. Переведите английское содержимое в файле `iOS/TUICallKit-Swift/Resources/es.lproj/Localized.strings` на испанский язык.
4. Перейдите в каталог, содержащий Podfile проекта приложения, и выполните команду `pod install` для обновления зависимостей.

```
  …import com.tencent.qcloud.tuicore.TUIThemeManager;public class MainActivity extends BaseActivity {    @Override  public void onCreate(Bundle savedInstanceState) {      super.onCreate(savedInstanceState);      Locale locale = new Locale("es");
      TUIThemeManager.addLanguage("es", locale);      …    }    …}
```

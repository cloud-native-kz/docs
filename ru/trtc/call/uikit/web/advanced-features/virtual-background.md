# Виртуальный фон

TUICallKit запустил новую функцию виртуального фона, позволяющую пользователям устанавливать размытый или графический фон во время видеозвонков. Это скрывает реальную среду звонка, защищает конфиденциальность и делает звонок более интересным. Далее в этой статье описывается, как использовать эту функцию в компоненте TUICallKit.

## Требования подготовки

Перед использованием функции виртуального фона от Tencent Cloud вам необходимо посетить консоль, чтобы активировать услуги аудио и видео для вашего приложения и приобрести пакет `Group Call`. Подробные шаги см. в разделе [Активация услуги](https://www.tencentcloud.com/document/product/647/59832).

## Включение размытого фона

Дизайн пользовательского интерфейса TUICallKit поддерживает установку размытого фона. Вызвав следующий интерфейс, вы можете отобразить кнопку функции размытого фона в пользовательском интерфейсе. Нажатие кнопки напрямую включит функцию размытого фона.

> **Примечание:** H5 пока не поддерживается, поддерживается только ПК.

```
import { TUICallKitAPI } from "@trtc/calls-uikit-vue"; TUICallKitAPI.enableVirtualBackground(true);
```

## Установка графического фона (опционально)

Реализация графического фона требует, чтобы пользователи сохранили изображение локально. После сохранения вызовите следующий интерфейс для установки (в настоящее время поддерживаются только изображения с локальным путём, изображения URI пока не поддерживаются).

```
import { TUICallKitAPI } from "@trtc/calls-uikit-vue"; TUICallKitAPI.getTUICallEngineInstance().setVirtualBackground(imagePath: string)
```

## Часто задаваемые вопросы

### Включение размытого фона не срабатывает или задерживается

- Убедитесь, что вы приобрели пакет аудио и видеозвонков `Group Call`, см. [Активация услуги](https://www.tencentcloud.com/document/product/647/59832).
- При плохом качестве сети файл модели виртуального фона может быть загружен не полностью, что приведёт к невозможности включить виртуальный фон.

### Можно ли включить виртуальный фон, если камера выключена?

Недоступно.

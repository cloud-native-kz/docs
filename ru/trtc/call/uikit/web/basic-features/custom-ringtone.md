# Пользовательский рингтон

В данной статье рассказывается, как использовать функцию пользовательского рингтона и беззвучного входящего вызова на основе определения.

## Установка рингтона входящего вызова

- Можно использовать только адреса локальных файлов в формате MP3, убедившись, что файл доступен.
- Для сброса рингтона передайте пустую строку для filePath.
- Используйте метод импорта ES6 для импорта файла рингтона.

```
import
```

## Беззвучный рингтон входящего вызова

- Включение/отключение рингтона входящего вызова.
- После включения рингтон входящего вызова не будет проигрываться при получении запроса на вызов.

```
try {  await TUICallKitAPI.enableMuteMode(enable: boolean);} catch (error: any) {  alert(`[TUICallKit] enableMuteMode API failed. Reason: ${error}`);}
```

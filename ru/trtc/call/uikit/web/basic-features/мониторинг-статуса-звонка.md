# Мониторинг статуса звонка

В этой статье описывается, как использовать обратные вызовы статуса звонка с компонентом TUICallKit.

## Мониторинг состояния звонка

Если ваш бизнес требует **мониторинга статуса звонка**, например событий во время процесса звонка ( [TUICallEvent](https://www.tencentcloud.com/document/product/647/51017) для получения подробной информации), вы можете обратиться к следующему коду:

```
import { TUICallEvent } from '@trtc/call-engine-lite-js';let handleUserEnter = function(event) {    console.log('TUICallEvent.USER_ENTER: ', event);};TUICallKitAPI.getTUICallEngineInstance().on(TUICallEvent.USER_ENTER, handleUserEnter);TUICallKitAPI.getTUICallEngineInstance().off(TUICallEvent.USER_ENTER, handleUserEnter);
```

## Событие обратного вызова компонента

Компонент TUICallKit предоставляет обратные вызовы статуса звонка, которые можно использовать для реализации дополнительной логики взаимодействия на стороне бизнеса. Дополнительные сведения см. в разделе [Обзор атрибутов TUICallKit](https://www.tencentcloud.com/document/product/647/51015#e57a5c34-b984-4372-af01-abaa5cd44ffe).

- `beforeCalling`: Выполняется перед началом звонка.
- `afterCalling`: Выполняется после завершения звонка.

React

Vue

```
function App() {  const handleBeforeCalling = () => {    console.log("[TUICallKit Demo] beforeCalling");  };  const handleAfterCalling = () => {    console.log("[TUICallKit Demo] afterCalling");  };  return (    <TUICallKit       beforeCalling={handleBeforeCalling}       afterCalling={handleAfterCalling} />  )}
```

```
<template>  <TUICallKit     :beforeCalling="handleBeforeCalling"     :afterCalling="handleAfterCalling" /></template><script setup>function handleBeforeCalling() {  console.log("[TUICallKit Demo] beforeCalling");}function handleAfterCalling() {  console.log("[TUICallKit Demo] afterCalling");}</script>
```

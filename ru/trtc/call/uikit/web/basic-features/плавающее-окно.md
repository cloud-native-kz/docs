## Функция плавающего окна

Метод 1: используйте API `enableFloatWindow(enable: boolean)` для включения/отключения плавающего окна.

```
import { TUICallKitAPI } from "@trtc/calls-uikit-react";try {  await TUICallKitAPI.enableFloatWindow(enable: Boolean)} catch (error: any) {  alert(`[TUICallKit] enableFloatWindow failed. Reason: ${error}`);}
```

Метод 2: управляйте включением/отключением плавающего окна и полноэкранного режима через контроль атрибутов.

- Атрибут `allowedMinimized` управляет включением/отключением плавающего окна.
- Атрибут `allowedMinimized` управляет включением/отключением полноэкранного режима.

React

Vue

```
<TUICallKit     allowedMinimized={true}     allowedFullScree={true}    />
```

```
<TUICallKit    :allowedMinimized="true"    :allowedFullScreen="true"   />
```

# Активация сервиса

## Активация пробного сервиса

Чтобы лучше ознакомиться с возможностями TRTC Call, мы предоставляем 7-дневный бесплатный пробный период. Каждый SDKAppID может дважды бесплатно попробовать TRTC Call, каждый раз на 7 дней. Каждая учетная запись может попробовать TRTC Call 10 раз в общей сложности.

Вы можете следовать приведенным ниже рекомендациям для активации пробной версии Call.

1. Войдите в [консоль Tencent RTC](https://console.trtc.io) и нажмите на **Создать приложение**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0519000cc5e711f0a2c65254005ef0f7.png)

2. Во всплывающем окне введите **имя приложения**, выберите **Call** и выберите подходящий **регион развертывания**. Затем нажмите **Создать**. Это создаст приложение TRTC, привязанное к пробной версии TRTC Call.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1384ad61c5e711f0a7ca5254001c06ec.png)

3. После создания приложения вы автоматически перейдете на страницу сведений о продукте Call. На этом этапе вы быстро создали приложение и успешно получили пробную версию TRTC Call (TUICallKit). Вы можете просмотреть информацию на текущей странице [**Сведения о продукте Call**](https://console.trtc.io/call) или [**Мои приложения**](https://console.trtc.io/app) и обратиться к [Руководству по интеграции](https://trtc.tencentcloud.com/document/46660/50990/50991) для интеграции. `SDKAppID` и `SDKSecretKey` будут использоваться в Руководстве по интеграции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2e0d3856c5e711f086d9525400e889b2.png)

## Покупка официальной версии

**Для использования TUICallKit необходимо приобрести ежемесячную подписку на Call**. Для получения подробной информации о цене и сравнении функций пакетов ежемесячной подписки на Call см. [Инструкции по выставлению счетов за ежемесячные пакеты Call](https://trtc.io/document/54632?platform=android&product=call). **Для покупки выполните следующие шаги:**

1. Перейдите на [страницу покупки Call](https://console.trtc.io/subscription/buy/call), выберите **приложение (SDKAppID) и тип пакета для покупки**, **рекомендуется включить автоматическое продление, чтобы избежать влияния на использование бизнеса**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Подписаться сейчас**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/78bf4d21c5e711f0a2c65254005ef0f7.png)

2. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**Страницу сведений о продукте Call**](https://console.trtc.io/call) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [Руководство по интеграции](https://trtc.tencentcloud.com/document/46660/50990/50991) для интеграции.

## Продление официальной версии

1. Перейдите на [**Страницу сведений о продукте Call**](https://console.trtc.io/call), нажмите **Продление / Обновление**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f024633cc6a511f09e745254007c27c5.png)

2. Перейдите на [страницу покупки Call](https://console.trtc.io/subscription/buy/call), подтвердите приложение (SDKAppID) для покупки, выберите **версию того же пакета приложения**, **рекомендуется включить автоматическое продление, чтобы избежать влияния на использование бизнеса**. После включения система будет автоматически продлевать подписку ежемесячно по истечении срока действия **при наличии достаточного баланса на счете**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Подписаться сейчас**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f01afd2bc6a511f0a7775254005ef0f7.png)

3. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**Страницу сведений о продукте Call**](https://console.trtc.io/call) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [Руководство по интеграции](https://trtc.tencentcloud.com/document/46660/50990/50991) для интеграции.

## Обновление официальной версии

1. Перейдите на [**Страницу сведений о продукте Call**](https://console.trtc.io/call), нажмите **Продление / Обновление**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/fe465870c6a511f0b011525400bf7822.png)

2. Перейдите на [страницу покупки Call](https://console.trtc.io/subscription/buy/call), подтвердите приложение (SDKAppID), требующее обновления, выберите **более продвинутую версию, чем текущий пакет приложения**, затем нажмите **Подписаться сейчас** после подтверждения информации о покупке и согласия с соответствующим соглашением.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/07775784c6a611f0a7775254005ef0f7.png)

3. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**Страницу сведений о продукте Call**](https://console.trtc.io/call) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения.

## Автоматическое продление

Конкретные шаги для включения автоматического продления в консоли следующие:

1. Перейдите на [**Страницу сведений о продукте Call**](https://console.trtc.io/call).
2. Выберите **приложение, для которого вы хотите включить автоматическое продление**.
3. В информации о продукте Call нажмите кнопку **Автоматическое продление** **Включить**. Появится всплывающее окно подтверждения. Нажмите **Включить**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/21f9be5ec6a611f0aedb525400454e06.png)

# Введение в продукт

## Обзор компонента

TUICallKit — это компонент пользовательского интерфейса для аудио и видеозвонков, разработанный компанией Tencent RTC. Он поддерживает в реальном времени общение до девяти участников одновременно. Интегрировав этот компонент, вам нужно написать всего несколько строк кода, чтобы добавить функции аудио и видеозвонков в ваше приложение.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/79e7b727f94a11ed80405254002af9e0.png)

# Коды ошибок

Уведомление пользователей об ошибках и предупреждениях, возникающих во время аудио и видеовызовов.

## Коды ошибок Chat

Видео и аудиовызовы используют Chat SDK Tencent Cloud в качестве базового сервиса для связи, такого как основная логика сигнализации вызова и сигнализации занятости. Ниже представлены распространенные коды ошибок:

| Код ошибки | Описание |
| --- | --- |
| 6014 | Вы не вошли в Chat SDK или были принудительно выведены из системы. Сначала войдите в Chat SDK и повторите попытку после успешного обратного вызова. Чтобы проверить, находитесь ли вы в сети, используйте TIMManager getLoginUser. |
| 6017 | Невалидный параметр. Проверьте сообщение об ошибке, чтобы определить невалидный параметр. |
| 6206 | UserSig истек. Получите новый действительный UserSig и войдите снова. Для получения дополнительной информации о том, как получить UserSig, см. [Генерирование UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| 7013 | Текущий пакет не поддерживает это API. Пожалуйста, обновитесь до пакета Flagship Edition. |
| 8010 | ID запроса сигнализации невалиден или уже обработан. |

> **Примечание:** Дополнительные коды ошибок Chat SDK доступны по адресу: [Коды ошибок Chat](https://www.tencentcloud.com/document/product/1047/34348)

## Коды ошибок TRTC

Видео и аудиовызовы используют Chat SDK Tencent Cloud в качестве базового сервиса для вызова, такого как основная логика переключения камеры и микрофона. Ниже представлены распространенные коды ошибок:

| Enum | Значение | Описание |
| --- | --- | --- |
| ERR_CAMERA_START_FAIL | -1301 | Не удалось включить камеру. Это может произойти при наличии проблемы с программой конфигурации камеры (драйвером) на Windows или macOS. Отключите и повторно включите камеру, перезагрузите камеру или обновите программу конфигурации. |
| ERR_CAMERA_NOT_AUTHORIZED | -1314 | Нет разрешения на доступ к камере. Это обычно происходит на мобильных устройствах и может быть вызвано тем, что пользователь отклонил доступ. |
| ERR_CAMERA_SET_PARAM_FAIL | -1315 | Неправильные параметры камеры (неподдерживаемые значения или другие). |
| ERR_CAMERA_OCCUPY | -1316 | Камера используется. Попробуйте другую камеру. |
| ERR_MIC_START_FAIL | -1302 | Не удалось включить микрофон. Это может произойти при наличии проблемы с программой конфигурации микрофона (драйвером) на Windows или macOS. Отключите и повторно включите микрофон, перезагрузите микрофон или обновите программу конфигурации. |
| ERR_MIC_NOT_AUTHORIZED | -1317 | Нет разрешения на доступ к микрофону. Это обычно происходит на мобильных устройствах и может быть вызвано тем, что пользователь отклонил доступ. |
| ERR_MIC_SET_PARAM_FAIL | -1318 | Не удалось установить параметры микрофона. |
| ERR_MIC_OCCUPY | -1319 | Микрофон используется. Микрофон не может быть включен, когда, например, пользователь проводит вызов на мобильном устройстве. |
| ERR_TRTC_ENTER_ROOM_FAILED | -3301 | Не удалось войти в комнату. Причину см. в сообщении об ошибке -3301. |

> **Примечание:** Дополнительные коды ошибок TRTC SDK доступны по адресу: [Коды ошибок TRTC](https://www.tencentcloud.com/document/product/647/35130)

## Коды ошибок сервера CallKit

Коды ошибок новой версии REST API CallKit, например при получении статуса вызова и записей вызовов. Ниже представлены распространенные коды ошибок:

| Код ошибки | Описание |
| --- | --- |
| 101001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 101002 | Невалидный параметр, проверьте, соответствуют ли параметры требованиям. Обратитесь к сообщению об ошибке для получения информации о конкретном невалидном поле. |
| 101004 | Вызов не существует или уже завершен. |
| 101050 | Запись вызова не существует. |

# Обзор функции

Администраторы могут получать доступ к записям звонков по callId через этот интерфейс.

> **Примечание:** REST API в настоящее время находится в бета-версии. Вы можете запрашивать данные звонков, созданные в течение последних 7 дней.

# Описание вызова API

## Пример URL запроса

```
https://xxxxxx/v1/records/get_record_by_callId?sdkappid=88888888&identifier=administrator&usersig=xxx&random=99999999&contenttype=json
```

## Параметры запроса

В таблице ниже указаны только параметры, изменяемые при вызове этого API, и их описание. Для получения дополнительной информации обратитесь к [Обзор REST API](https://www.tencentcloud.com/document/product/647/60138#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, в котором находится SDKAppID: `callkit-intl.trtc.tencent-cloud.com` |
| v1/records/get_record_by_callId | API запроса |
| sdkappid | SDKAppID, назначенный консолью при создании приложения |
| identifier | Использование учётной записи администратора чата |
| usersig | Подпись, созданная учётной записью администратора приложения. Для получения подробных инструкций см. [Генерация UserSig](https://www.tencentcloud.com/document/product/647/39074#) |
| random | Введите случайное 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

## Максимальная частота вызовов

20 раз в секунду.

## Пример пакета запроса

```
{    "callId": "2ae7d549-c441-4a9b-87c0-61810fe19582"}
```

## Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| callId | String | Обязательное | Уникальный ID звонка |

## Пример пакета ответа

```
{    "errorCode": 0,     "errorMessage": "Success",      "requestId": "3fdc344c7a67461c6af3cfc3e77744b5",     "data": {        "callId": "2ae7d549-c441-4a9b-87c0-61810fe19582",        "sdkAppId": 88888888,        "mediaType": "video",        "roomId": "123456",        "startCallTs": 1688705638,        "acceptTs": 1688705641,        "endTs": 1688705668,        "caller": "alice",        "totalUserNumber": 2,        "callType": "single",         "callResult": "normal_end",         "callees": [            "bob1",            "bob2"        ]     } }
```

## Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| errorCode | Integer | Код ошибки, 0 означает успех |
| errorMessage | String | Сообщение об ошибке |
| requestId | String | Уникальный ID запроса |
| callId | String | Уникальный ID звонка |
| sdkAppId | Integer | Ваш sdkAppId |
| mediaType | String | Тип медиа: `video` видеозвонок, `audio` аудиозвонок |
| roomId | String | ID комнаты звонка |
| startCallTs | Integer | Временная метка начала звонка (в секундах) |
| acceptTs | Integer | Временная метка подключения звонка (в секундах) |
| endTs | Integer | Временная метка окончания звонка (в секундах) |
| caller | String | userId звонящего |
| totalUserNumber | Integer | Общее количество участников звонка |
| callType | String | Тип звонка: `single` один на один, `group` групповой звонок |
| callResult | String | Результат звонка: `cancel` Отмена: звонящий отменил звонок перед подключением, `reject` Отклонено: вызываемый отклонил звонок, `not_answer` Не ответили: вызываемый не ответил вовремя, `normal_end` Завершено: звонок был подключен и завершен нормально, `call_busy` Занято: линия была занята во время звонка, `interrupt` Прерывание: звонок был прерван из-за сетевых или других причин. |
| callees | Array | Список ID пользователей вызываемых абонентов, участвовавших в звонке |

> **Примечание:** Поле callResult может отображать все типы только для одноранговых звонков, в то время как групповые чаты содержат только `normal_end`.

# Коды ошибок

За исключением сетевых ошибок (например, ошибка 502), HTTP код возврата для этого интерфейса всегда равен 200. Фактический код ошибки и сообщение об ошибке передаются через errorCode и errorMessage в теле ответа.
Для общих кодов ошибок (70000-79999) см. [Код ошибки](https://www.tencentcloud.com/document/product/647/60138#rest-api-.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81).

| Код ошибки | Описание |
| --- | --- |
| 0 | Запрос выполнен успешно |
| 50001 | Текущему приложению необходимо приобрести пакет версии TUICallKit Group Call для использования |
| 70011 | Записи не найдены |
| Unknown error code | Неизвестная ошибка, пожалуйста [отправьте заявку](https://console.intl.cloud.tencent.com/workorder/category) для связи с технической поддержкой |

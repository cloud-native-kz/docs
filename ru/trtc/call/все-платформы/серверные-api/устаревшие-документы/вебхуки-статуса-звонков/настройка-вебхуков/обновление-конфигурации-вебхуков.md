# Обновление конфигурации вебхуков

## Обзор функции

Администраторы могут обновлять обратные вызовы через этот интерфейс.

## Описание API

### Пример URL запроса

```
https://xxxxxx/v1/callback/update?sdkappid=88888888&identifier=administrator&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого API, и их описание. Для получения дополнительной информации см. [Обзор REST API](https://www.tencentcloud.com/document/product/647/60138#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: `callkit-intl.trtc.tencent-cloud.com` |
| v1/callback/update | API запроса |
| sdkappid | sdkappid, назначенный консолью при создании приложения |
| identifier | Должна быть учетная запись администратора Chat App |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробные инструкции см. в [Генерация UserSig](https://www.tencentcloud.com/document/product/647/39074#) |
| random | Случайное 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

10 раз в секунду.

### Пример пакета запроса

```
{    "address": "https://www.example2.com/callback",    "actions": [        "call_busy",        "cancel",        "normal_end"    ]}
```

### Описание полей запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| address | String | Обязательное | Адрес обратного вызова, должен начинаться с http/https, рекомендуется использовать более безопасный https |
| actions | Array | Обязательное | Сценарии, требующие запуска обратного вызова. См. [Список слов команд обратного вызова](https://www.tencentcloud.com/document/product/647/60135#2147ef13-cf14-4134-9b18-5e3f5d4a026c) |

### Пример пакета ответа

```
{    "errorCode": 0,    "errorMessage": "Success",    "requestId": "5b0fa500064397cad3554506e27e18e1",    "data": "https://www.example2.com/callback"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| errorCode | Integer | Код ошибки, 0 означает успех |
| errorMessage | String | Сообщение об ошибке |
| requestId | String | Уникальный ID запроса |
| data | String | Адрес обратного вызова успешно обновлен |

## Коды ошибок

| Код ошибки | Описание |
| --- | --- |
| 0 | Запрос выполнен успешно |
| 50001 | Текущему приложению необходимо приобрести пакет TUICallKit Group Call Version для использования |
| 999 | Информация конфигурации отсутствует |

# Управление группами

Войдите в [консоль Chat](https://console.trtc.io), выберите **Chat >** [**Groups**](https://console.trtc.io/chat/qun-manage) на левой панели и **выберите целевое приложение** в верхней части. Вы можете управлять группами в соответствии с фактическими потребностями бизнеса.

Кроме того, вы можете вызвать соответствующие RESTful API для управления группами. Для получения дополнительной информации см. [документацию по API управления группами](https://intl.cloud.tencent.com/document/product/1047/34960).

## Добавление групп

1. На странице **Group Management** нажмите **Add Group**.
2. Настройте следующие параметры в появившемся диалоговом окне добавления группы:
  - Group Name: введите имя группы. Это обязательный параметр, длина не должна превышать 30 байт.
  - Group Owner ID: введите ID владельца группы. Это необязательный параметр. Необходимо ввести имя зарегистрированного пользователя.
  - Group Type: установите тип группы, который может быть Work, Public, Meeting или Audio-video Group. Для получения дополнительной информации о типах групп см. [Типы групп](https://intl.cloud.tencent.com/document/product/1047/33529).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/66132af7cb3711f087ad52540099c741.png)

3. Нажмите **OK** для сохранения конфигурации.
 После создания группы вы сможете просмотреть ID группы, имя группы, владельца группы, тип группы и время создания группы в списке групп.

## Просмотр деталей группы

На странице **Group Management** вы можете нажать **Learn more** в строке целевой группы, чтобы перейти на страницу **Group Details**, где вы сможете просмотреть и изменить основную информацию о группе и управлять членами группы.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cc97f5747b1311efb9d8525400f69702.png)

### Изменение основной информации

1. На странице **Group Details** нажмите **Edit** в области основной информации.
2. В появившемся диалоговом окне изменения информации о группе вы можете изменить имя группы и описание группы.
 ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dfc49f3b7b1311ef82535254002693fd.png)
3. Нажмите **Confirm** для сохранения конфигурации.

### Управление членами группы

**Добавление члена группы**

1. На странице **Group Details** нажмите **Add Member** в области управления членами группы.
2. В появившемся диалоговом окне добавления членов введите имя пользователя.

> **Примечание:** необходимо ввести имя зарегистрированного пользователя.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e71effdd7b1311efa87e52540055f650.png)

3. Нажмите **Confirm** для сохранения конфигурации. После добавления члена группы вы сможете просмотреть имя пользователя, ник, время присоединения, время последнего сообщения и роль члена в списке членов группы.

**Удаление члена группы**

1. На странице **Group Details** вы можете удалить членов группы следующими способами:
  - Удалить одного члена: нажмите **Delete** в строке целевого члена.
  - Удалить несколько членов: выберите членов для удаления и нажмите **Delete Members** над списком членов группы.
2. В появившемся диалоговом окне нажмите **Confirm**.
 После удаления выбранные члены больше не будут принадлежать группе.

## Отправка сообщений

1. На странице **Group Management** вы можете отправлять сообщения следующими способами:
  - Отправить сообщение в одну группу: нажмите **Send Messages** в строке целевой группы.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/fe002e7a7b1311ef8631525400a9236a.png)

  - Отправить сообщение в несколько групп: выберите целевые группы, в которые вы хотите отправить сообщение, и нажмите **Send Messages** над списком групп.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/06a073607b1411ef852f52540075b605.png)

2. В появившемся диалоговом окне отправки группового сообщения введите содержание сообщения.

> **Примечание:** длина сообщения не может превышать 300 символов.

3. Нажмите **OK** для отправки сообщения.

## Удаление групп

**После удаления группы вся её информация будет удалена и не может быть восстановлена. Будьте осторожны при удалении групп.**

1. На странице **Group Management** вы можете удалять группы следующими способами:
  - Удалить одну группу: нажмите **Delete** в строке целевой группы.
  - Удалить несколько групп: выберите целевые группы для удаления и нажмите **Delete Groups** над списком групп.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/19b0229d7b1411ef852f52540075b605.png)

2. В появившемся диалоговом окне нажмите **Confirm**. **После удаления группы вся её информация будет удалена и не может быть восстановлена.**

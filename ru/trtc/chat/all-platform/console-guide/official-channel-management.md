# Управление официальными каналами

Войдите в [Chat Console](https://console.trtc.io/), затем в левой панели навигации выберите [Chat > Configuration > Official Channel Management](https://console.trtc.io/chat/public-account-management). Вы можете управлять официальными каналами в соответствии с потребностями вашего бизнеса. Вы также можете управлять официальными каналами, вызывая соответствующие REST API. Подробные инструкции см. в [документации API управления официальными каналами](https://www.tencentcloud.com/document/product/1047/60755).

## Создание официального канала

1. На странице **Official Channel Management** нажмите **Create Official Channel**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/4f2d7cf8fc2111f0837f525400ecee81.png)

2. В окне **Create Official Channel** настройте следующие параметры:
  - Official Channel Name: введите имя официального канала. Этот параметр обязателен и не должен превышать 150 байт.
  - Profile Photo: загрузите изображение для использования в качестве аватара официального канала. Этот параметр обязателен. Ограничений на размер или формат изображения нет.
  - Official Channel Profile: по желанию введите краткое описание для официального канала. Этот параметр не должен превышать 400 байт.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2625119efc2311f0a98b52540073fd3b.png)

3. Нажмите **Create** для сохранения официального канала.

## Редактирование официального канала

1. На странице **Official Channel Management** нажмите кнопку **Edit**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f38ddc5ffc2411f0afd0525400370dda.png)

2. В окне **Edit Official Channel** при необходимости обновите имя официального канала, описание или фото.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f204bbf9fc2511f0a98b52540073fd3b.png)

## Удаление официального канала

1. На странице **Official Channel Management** выберите официальный канал, который вы хотите удалить, и нажмите кнопку **Delete** справа.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5bf3b086fc2711f0840952540097cba1.png)

2. После нажатия кнопки **Delete** появится окно подтверждения. Удалённые официальные каналы не могут быть восстановлены.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/da67a21cfc2711f084a0525400380f7d.png)

## Создание публикации

Страница управления публикациями позволяет создавать, просматривать и искать все публикации, отправленные с официальных каналов

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/354ac46afcb911f084a0525400380f7d.png)

> **Примечание:** На странице управления публикациями отображаются только опубликованные публикации и их детали. Редактирование публикаций не поддерживается.

Шаги по созданию публикаций:

1. **Select Post Type**. Выберите тип создаваемой публикации. Система поддерживает следующие типы:
  - Image and Text: публикация с изображениями и текстом.
  - Image Only: публикация только с изображениями.
  - Text Only: публикация только с текстом.
2. **Edit Post Content**. Введите содержимое публикации в поле ввода и загрузите изображения в область загрузки изображений.
3. **Add Links**. Нажмите **Add Link** для вставки гиперссылок для вторичной навигации.
4. **Preview Content**. Вы можете просмотреть публикацию в реальном времени в области предпросмотра справа.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c33ce6f4fc2b11f0afd0525400370dda.png)

5. **Publish Post.** После заполнения всех обязательных полей нажмите Publish для отправки публикации. После публикации публикация появится в списке публикаций.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/696cc16efcb511f0837f525400ecee81.png)

6. Если публикация не удастся отправить, всплывающее окно в верхнем правом углу покажет сообщение об ошибке. При необходимости обновите публикацию в соответствии с подсказкой.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/d7a3a351fcb611f084a0525400380f7d.png)

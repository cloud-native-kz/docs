# Монитор в реальном времени

## Мониторинг в реальном времени

> **Примечание:** функция мониторинга в реальном времени находится в бета-тестировании и постоянно обновляется. Если у вас есть обратная связь или предложения, [отправьте заявку](https://console.tencentcloud.com/workorder/category?level1_id=29&level2_id=40&source=0&data_title=%E5%8D%B3%E6%97%B6%E9%80%9A%E4%BF%A1%20IM&level3_id=237&radio_title=%E7%99%BB%E5%BD%95%E5%8F%8A%E5%A4%9A%E7%AB%AF%E5%9C%A8%E7%BA%BF%E9%97%AE%E9%A2%98&queue=3235&scene_code=27293&step=2).

1. На левой боковой панели выберите **Chat > >** [**Monitor**](https://console.trtc.io/chat/monitor) и **выберите целевое приложение вверху**.
2. В области обзора вы можете просмотреть **Current Online Users**, **One-to-One Messages Today**, **Ordinary Group Messages Today** и **Audio-Video Group Messages Today**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/fe2d311fcb5711f09e745254007c27c5.png)

3. В области детальных данных мониторинга данные за 24 часа календарного дня по умолчанию отображаются по метке времени. Когда курсор мыши указывает на область графика данных, вы можете использовать колесико прокрутки для увеличения временной метки для просмотра деталей, перетаскивать временную метку влево и вправо для просмотра данных до и после времени, а также нажимать на легенду под временной меткой, чтобы скрыть или показать соответствующее значение в графике.
  - В области мониторинга входов вы можете просмотреть количество входов и процент успеха входа для каждого клиента.

> **Примечание:** в настоящее время отображаются только данные входа, сообщаемые Chat SDK версии 4.8.10 или позже для iOS, Android, Windows и macOS. Рекомендуется обновиться до [последней версии SDK](https://intl.cloud.tencent.com/document/product/1047/33996).

  - В области мониторинга сообщений вы можете просмотреть количество сообщений один-на-один или групповых сообщений, отправленных каждым клиентом, и процент успешной доставки сообщений.

> **Примечание:** в настоящее время отображаются только данные входа, сообщаемые Chat SDK версии 4.8.10 или позже для iOS, Android, Windows и macOS. Рекомендуется обновиться до [последней версии SDK](https://intl.cloud.tencent.com/document/product/1047/33996). В настоящее время веб-SDK не поддерживает сбор статистики сообщений по типу чата.

  - В области мониторинга обратных вызовов вы можете просмотреть количество обратных вызовов и процент успеха обратного вызова.
  - В области мониторинга RESTful API вы можете просмотреть количество запросов RESTful API и процент успеха запроса.
  - В области мониторинга оффлайн-доставки вы можете просмотреть количество попыток оффлайн-доставки и процент успеха доставки.

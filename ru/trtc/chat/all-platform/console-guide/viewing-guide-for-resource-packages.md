# Руководство по просмотру пакетов ресурсов

## Просмотр пакетов ресурсов объема сообщений сообщества

Для просмотра объема сообщений нижестоящего сообщества, охватываемого текущим [Chat Monthly Package](https://www.tencentcloud.com/document/product/1047/67650), перейдите в [Chat Console](https://console.trtc.io/chat), выберите **Chat > Configuration >** [**Group Configuration**](https://console.trtc.io/chat/qun-setting) в левой боковой панели, **выберите целевое приложение в верхней части** и **просмотрите его в разделе Group feature configuration > Community**.**.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/bf69872bcb5711f0a4a55254001c06ec.png)

## Просмотр пакетов ресурсов плагинов

Для просмотра использования плагинов, таких как облачный поиск и преобразование речи в текст после активации, перейдите в [Chat Console](https://console.trtc.io/chat/plugin), выберите **Chat >**[**Advanced Features**](https://console.trtc.io/chat/plugin) в левой боковой панели, **выберите целевое приложение в верхней части**, нажмите на карточку любого плагина и откройте сведения плагина для просмотра **.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c748e997cb5711f0a93d52540044a08e.png)

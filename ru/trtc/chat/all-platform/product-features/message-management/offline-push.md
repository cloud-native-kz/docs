# Автономная отправка уведомлений

## Обзор

В случаях, когда приложение отправляется в фоновый режим или процесс завершается, функция автономной отправки уведомлений можно использовать при наличии новых сообщений, которые требуют уведомления пользователя. Instant Messaging Chat предоставляет вам TIMPush для реализации функции автономной отправки уведомлений. Для получения информации о поддерживаемых функциях обратитесь к таблице ниже и нажмите на [Service Overview](https://trtc.io/document/60537) для получения дополнительной информации о продукте.

# React Native

## 1.0.0 (2024-12-4)

- Добавлен API `disablePostNotificationInForeground`, переключение уведомлений панели уведомлений когда приложение находится в foreground (по умолчанию включено).
- Добавлен API `createNotificationChannel`, поддержка для OPPO/FCM установки пользовательских каналов уведомлений.
- Добавлено событие клика по панели уведомлений `Push.EVENT.NOTIFICATION_CLICKED`.
- Оптимизирован опыт доступа.
- Поддержка функции пользовательских звуков уведомлений.

## 0.3.1 (2024-10-15)

- обновление процесса доступа Android в readme

## 0.2.0 (2024-10-11)

- Добавлен API `addPushListener/removePushListener`, поддержка получения online push-сообщений и уведомлений об отзыве push-сообщений.

## 0.1.0 (2024-09-10)

- Для React Native, основанный на Tencent Cloud Push Service, поддерживает отправку push-уведомлений на iOS и Android, и адаптирован к push-сервисам крупных производителей

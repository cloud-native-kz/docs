# Супер крупное развлекательное и совместное сообщество

## Обзор

Сообщество — это мощный инструмент для совместной работы в области развлечений. Оно поддерживает

**сообщество**

-

**группа**

-

**тема**

иерархию, в которой сообщения изолированы, а большое количество участников имеют общий набор отношений дружбы. Кроме того, это позволяет вам группировать участников и устанавливать разрешения группы для просмотра, общения и управления.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/393cd4ab049d11f08c4452540044a08e.jpeg)

## Варианты использования

### Поиск единомышленников: новый способ расширения пользовательской базы

- Сообщество поддерживает размещение большого количества энтузиастов в иерархии **сообщество**-**группа**-**тема**.
- Большое открытое сообщество предоставляет более детализированные темы, позволяя пользователям общаться и взаимодействовать более свободно и проактивно.
![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/398ae26d049d11f0aa8e525400bf7822.jpeg)

### Социальная сеть на основе игр: повышенная лояльность и активность пользователей

Различные темы сообщества позволяют игрокам получать доступ к новостям, искать товарищей по команде, обсуждать сюжеты и делиться советами. Перед игрой пользователи могут искать рекомендации; во время игры они всегда могут общаться с другими (через постоянно активный чат); после игры они могут углубиться в изучение тем.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/39d289d0049d11f0aa645254007c27c5.jpeg)

### Маркетинг для фанатов: эффективный инструмент управления

Разнообразные темы в сообществе могут идеально заменить бесконечные группы (такие как группа 1 пользователей из Шэньчжэня, группа 2 пользователей из Шэньчжэня, группа 1 пользователей из Гуанчжоу и группа 1 пользователей из Шанхая), избавляя вас от необходимости управлять множеством утомительных групп и делая маркетинг для фанатов более точным.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3967557f049d11f08bc5525400454e06.jpeg)

### Управление организацией: четкий и многоуровневый способ общения

Все члены организации могут присоединиться к одному сообществу, которое поддерживает иерархическое общение через иерархию сообщества и настройки разрешений.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/394f0c32049d11f08c4452540044a08e.jpeg)

## Технические преимущества

### Супер крупное количество участников сообщества

Сообщество Tencent Chat почти в десять тысяч раз больше, идеально отвечая вашим потребностям в размещении массового количества участников в таких вариантах использования, как поиск единомышленников, маркетинг для фанатов, социальная сеть на основе игр и управление организацией.

### Надежность сообщений

Сообщество Tencent Chat значительно увеличивает количество участников, включив мощные возможности обмена сообщениями Tencent. С полной и надежной системой обмена сообщениями, основанной на более чем 20 годах накопления технологий, она обеспечивает успешность отправки/получения сообщений свыше 99,99% и доступность сервиса, помогая вам легко обрабатывать сотни миллионов одновременных запросов.

### Производительность отправки сообщений

Сообщество Tencent Chat использует новую архитектуру отправки сообщений с «быстрым и медленным каналами» плюс «двухуровневая отправка». Этот специальный тип архитектуры хорошо балансирует время и пространство для повышения производительности системной отправки и снижения потребления производительности терминала. Пользователи могут наслаждаться одинаковым опытом взаимодействия с сообщениями в супер-крупных группах, как и в обычных.

### Управление статусом сообщений и разрешениями пользователей

Сообщество Tencent Chat предоставляет расширенные возможности, такие как редактирование, отзыв и пересылка сообщений. Это позволяет устанавливать отключение звука, включать режим «не беспокоить», подсчитывать непрочитанные сообщения и редактировать профили глобально или по сообществу или теме.

## Руководство по интеграции Native SDK

> **Внимание:** Функция темы сообщества поддерживается только Chat Native SDK Enhanced edition версии 6.2.2363 или более поздней. Чтобы использовать её, [приобретите Pro edition, Pro Plus edition или Enterprise edition](https://buy.tencentcloud.com/avc?from=17182).

Ниже описаны API тем сообщества для Android.

### Вызов API темы сообщества

1. Вызовите API `createGroup` для создания сообщества с поддержкой тем, как описано в разделе [Управление сообществом](https://www.tencentcloud.com/document/product/1047/48172#.E5.88.9B.E5.BB.BA.E7.A4.BE.E7.BE.A4).
2. Вызовите API `getJoinedCommunityList` для получения списка созданных и присоединённых сообществ.

> **Внимание:** Сообщество используется для управления участниками группы, но не поддерживает отправку или получение сообщений. Дополнительную информацию о других функциях сообщества см. в списке «других API управления».

3. После успешного создания сообщества вызовите API `createTopicInfoCommunity` для создания тем, как описано в разделе [Управление темами](https://www.tencentcloud.com/document/product/1047/36271?lang=zh).
4. Управление темами также включает удаление темы, изменение информации о теме, получение списка тем и прослушивание обратных вызовов темы. Кроме того, темы поддерживают отправку и получение сообщений для общения пользователей. Дополнительную информацию об API см. в разделе [Сообщения темы](https://www.tencentcloud.com/document/product/1047/36271?lang=zh).
5. Группировка участников сообщества: Вы можете указать информацию о группе в пользовательских полях участника группы для отображения участников сообщества в группах. Для достижения этого эффекта все участники сообщества должны быть загружены локально для сортировки по группам. Если количество участников группы велико, рекомендуется реализовать группировку участников на стороне сервера.

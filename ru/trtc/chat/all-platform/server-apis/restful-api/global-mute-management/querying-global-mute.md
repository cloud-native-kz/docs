# Запрос глобального отключения звука

## Справочная информация

- Глобальное отключение звука включает глобальное отключение звука для личных сообщений и для групповых сообщений.
- По умолчанию глобальное отключение звука личных сообщений отключено для учетных записей. Если для учетной записи включено глобальное отключение звука личных сообщений, все личные сообщения не могут быть отправлены в течение периода отключения звука. После истечения периода отключения звука система Chat backend автоматически отключает глобальное отключение звука личных сообщений, и после этого все личные сообщения могут быть отправлены в обычном режиме. Для постоянного глобального отключения звука личных сообщений период отключения звука никогда не истекает.
- По умолчанию глобальное отключение звука групповых сообщений отключено для учетных записей. Если для учетной записи включено глобальное отключение звука групповых сообщений, все групповые сообщения не могут быть отправлены в течение периода отключения звука. После истечения периода отключения звука система Chat backend автоматически отключает глобальное отключение звука групповых сообщений, и после этого все групповые сообщения могут быть отправлены в обычном режиме. Для постоянного глобального отключения звука групповых сообщений период отключения звука никогда не истекает.

## Описание функции

- Этот API используется для запроса глобального отключения звука личных сообщений для учетной записи.
- Этот API используется для запроса глобального отключения звука групповых сообщений для учетной записи.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/openconfigsvr/getnospeaking?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором расположен ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/openconfigsvr/getnospeaking | API запроса |
| sdkappid | SDKAppID, присвоенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Дополнительные сведения см. в разделе [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение зафиксировано как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

```
{    "Get_Account": "lumotuwe"}
```

### Поля запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| Get_Account | String | Обязательное | Аккаунт, для которого запрашивается информация об отключении звука |

### Пример ответа

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "C2CmsgNospeakingTime": 4294967295,    "GroupmsgNospeakingTime": 7196}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ErrorCode | Integer | Код ошибки запроса. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |
| C2CmsgNospeakingTime | Number | Период отключения звука для личных сообщений в секундах. Значение является неотрицательным целым числом. Значение `0` указывает, что отключение звука сообщений отключено. Максимальное значение `4294967295` (или 0xFFFFFFFF в шестнадцатеричном формате) указывает, что для учетной записи включено постоянное отключение звука. Другие значения указывают конкретный период отключения звука для учетной записи. Например, значение `3600` указывает, что период отключения звука для учетной записи составляет один час. |
| GroupmsgNospeakingTime | Number | Период отключения звука для групповых сообщений в секундах. Значение является неотрицательным целым числом. Значение `0` указывает, что отключение звука сообщений отключено. Максимальное значение `4294967295` (или 0xFFFFFFFF в шестнадцатеричном формате) указывает, что для учетной записи включено постоянное отключение звука. Другие значения указывают конкретный период отключения звука для учетной записи. Например, значение `3600` указывает, что период отключения звука для учетной записи составляет один час. |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и подробности можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 130001 | Ошибка при разборе формата JSON. Проверьте, соответствует ли JSON запрос спецификациям JSON. |
| 130002 | JSON запрос не содержит поле `Get_Account`. |
| 130003 | Поле `Get_Account` в JSON запросе недействительно. |
| 130014 | Произошла системная ошибка JSON. Повторите попытку или обратитесь в техническую поддержку. |

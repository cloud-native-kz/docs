# Установка глобального отключения звука

## Справочная информация

- Глобальное отключение звука включает отключение звука для личных сообщений и групповых сообщений.
- Если глобальное отключение звука для личных сообщений включено для аккаунта, в течение периода отключения звука не могут быть отправлены никакие личные сообщения, отправитель получает код ошибки 20012 (подробнее см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348)), и webhook [Перед отправкой личного сообщения](https://intl.cloud.tencent.com/document/product/1047/34364) не запускается. После истечения периода отключения звука система бэкенда Chat автоматически отключает глобальное отключение звука для личных сообщений, после чего все личные сообщения могут быть отправлены нормально. Для постоянного глобального отключения звука для личных сообщений период отключения звука никогда не истекает. По умолчанию глобальное отключение звука для личных сообщений отключено для аккаунтов.
- Если глобальное отключение звука для групповых сообщений включено для аккаунта, в течение периода отключения звука не могут быть отправлены никакие групповые сообщения, отправитель получает код ошибки 10017 (подробнее см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348)), и webhook [Перед отправкой группового сообщения](https://intl.cloud.tencent.com/document/product/1047/34374) не запускается. После истечения периода отключения звука система бэкенда Chat автоматически отключает глобальное отключение звука для групповых сообщений, после чего все групповые сообщения могут быть отправлены нормально. Для постоянного глобального отключения звука для групповых сообщений период отключения звука никогда не истекает. По умолчанию глобальное отключение звука для групповых сообщений отключено для аккаунтов.

## Описание функции

- Этот API используется для включения глобального отключения звука для личных сообщений аккаунта.
- Этот API используется для включения глобального отключения звука для групповых сообщений аккаунта.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/openconfigsvr/setnospeaking?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором расположен ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/openconfigsvr/setnospeaking | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Для получения дополнительной информации см. раздел **App Admin** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробнее см. [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | 32-битное случайное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение зафиксировано как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

```
{    "Set_Account": "lumotuwe",    "C2CmsgNospeakingTime": 4294967295, // `C2CmsgNospeakingTime` и `GroupmsgNospeakingTime` являются необязательными, но один из них должен быть указан.    "GroupmsgNospeakingTime": 7200}
```

### Поля запроса

| Поле | Тип | Требуется | Описание |
| --- | --- | --- | --- |
| Set_Account | String | Требуется | Аккаунт, для которого будет включено отключение звука |
| C2CmsgNospeakingTime | Integer | Необязательно | Период отключения звука (в секундах) для личных сообщений. Значение является неотрицательным целым числом, максимальное значение составляет 4294967295 (0xFFFFFFFF в шестнадцатеричном формате).0: аккаунт включен4294967295: аккаунт постоянно отключенДругие значения: определенный период отключения звука аккаунта |
| GroupmsgNospeakingTime | Integer | Необязательно | Период отключения звука (в секундах) для групповых сообщений. Значение является неотрицательным целым числом, максимальное значение составляет 4294967295 (0xFFFFFFFF в шестнадцатеричном формате).0: аккаунт включен4294967295: аккаунт постоянно отключенДругие значения: определенный период отключения звука аккаунта |

### Пример ответа

```
{    "ErrorCode": 0,    "ErrorInfo": "",}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ErrorCode | Integer | Код ошибки запроса. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

HTTP-статус, возвращаемый этим API, всегда составляет 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 130001 | Ошибка парсинга формата JSON. Проверьте, соответствует ли JSON-запрос спецификациям JSON. |
| 130004 | JSON-запрос не содержит поле `Set_Account`. |
| 130005 | Поле `Set_Account` в JSON-запросе недействительно. |
| 130008 | В JSON-запросе не указано ни `GroupmsgNospeakingTime`, ни `C2CmsgNospeakingTime`. |
| 130014 | Произошла системная ошибка. Повторите попытку или обратитесь в техническую поддержку. |

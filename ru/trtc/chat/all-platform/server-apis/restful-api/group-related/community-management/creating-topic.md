# Создание раздела

## Обзор функции

Этот API используется администратором приложения для создания раздела.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддерживается ли этот REST API |
| --- | --- |
| Private | Нет |
| Public | Нет |
| ChatRoom | Нет |
| AVChatRoom | Нет |
| Community | Этот API применяется только к сообществам с включенными разделами. |

Это предустановленные типы групп в Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание:** Чтобы использовать функцию разделов, перейдите в [консоль](https://console.trtc.io/chat/qun-setting), выберите **Конфигурация функций** > **Конфигурация групп** > **Конфигурация функций групп** > **Сообщество**, включите функцию сообщества, а затем включите функцию разделов.

### Пример URL запроса

```
https://xxxxxx/v4/million_group_open_http_svc/create_topic?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для получения информации о других параметрах см. раздел [Обзор REST API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/million_group_open_http_svc/create_topic | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Дополнительную информацию см. в разделе **Администратор приложения** в [Проверка подлинности входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Дополнительную информацию см. в разделе [Генерация UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение зафиксировано как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Базовый формат**
Создание раздела. Здесь `GroupId` является обязательным, что указывает на группу раздела. Перед созданием раздела должна существовать группа с включенными разделами.

```
{  "GroupId": "@TGS#_@TGS#cQVLVHIM62CJ", // ID группы раздела, обязательно  "TopicName": "TestTopic" 		// Название раздела, обязательно}
```

```
{  "GroupId": "@TGS#_@TGS#cQVLVHIM62CJ", // ID группы раздела, обязательно  "TopicId": "@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_TestTopic", 	// Пользовательский ID раздела, опционально  "TopicName": "TestTopic" 		// Название раздела, обязательно}
```

- **ВСЕ В ОДНОМ**

```
{  "GroupId": "@TGS#_@TGS#cQVLVHIM62CJ", // ID группы раздела, обязательно  "TopicId": "@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_TestTopic", 	// Пользовательский ID раздела, опционально  "TopicName": "TestTopic", 		// Название раздела, обязательно  "From_Account": "1400187352", // Член, создающий раздел  "CustomString": "This is a custom string",	// Пользовательская строка  "FaceUrl": "http://this.is.face.url", // (Опционально) URL фото профиля раздела  "Notification": "This is topic Notification", // (Опционально) Уведомление раздела  "Introduction": "This is topic Introduction" // (Опционально) Описание раздела}
```

### Поля запроса

### Пример ответа

- **Базовая форма, содержащая только базовую информацию раздела**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "TopicId": "@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_@TOPIC#cETE3HIM62CQ"}
```

- **Пользовательский ID и** `ВСЕ В ОДНОМ` **раздела**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "TopicId": "@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_TestTopic"}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Ошибка |
| ErrorInfo | String | Информация об ошибке |
| TopicId | String | ID успешно созданного раздела, который может быть кастомизирован или назначен бэкэндом Chat. |

## Коды ошибок

HTTP код статуса, возвращаемый этим API, всегда равен 200, если не происходит ошибка сети (например, ошибка 502). Конкретный код ошибки и подробности можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для публичных кодов ошибок (60000–79999) см. раздел [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Недопустимое слово команды. |
| 10004 | Неправильный параметр. Проверьте описание ошибки и устраните проблему. |
| 10006 | Операция превышает лимит частоты. Пожалуйста, снизьте частоту вызовов. Эта ошибка обычно возникает из-за слишком большого увеличения чистой группы в один день или слишком частых вызовов для получения всех групп в приложении. |
| 10007 | Недостаточные разрешения на операцию. Проверьте параметры запроса на основе сообщения об ошибке. |
| 10008 | Запрос недействителен, вероятно, потому что проверка информации подписи в запросе не прошла. Повторите попытку или [отправьте запрос в службу поддержки](https://console.tencentcloud.com/workorder/category?level1_id=29&level2_id=40&source=0&data_title=%E4%BA%91%E9%80%9A%E4%BF%A1%20%20IM&step=1). |
| 10010 | Текущая группа не существует или была удалена. |
| 10015 | Запрошенный ID группы недействителен. Проверьте параметр запроса на основе сообщения об ошибке. |
| 10016 | Бэкэнд приложения отклонил эту операцию через вебхук. Проверьте возвращаемое значение вебхука перед созданием раздела. |
| 10025 | Вы уже использовали этот ID группы. Сначала распустите существующую группу или выберите другой ID группы. |
| 10036 | Превышен лимит созданных групп видео (AVChatRoom). Удалите некоторые группы или приобретите обновление согласно [Ценообразованию](https://intl.cloud.tencent.com/document/product/1047/34350). |
| 10037 | Превышен лимит созданных предоплаченных разделов. Удалите некоторые разделы или приобретите обновление согласно [Ценообразованию](https://intl.cloud.tencent.com/document/product/1047/34350). |
| 11000 | Текущая группа не поддерживает функцию разделов сообщества. Для использования этой функции необходимо приобрести [Pro, Pro Plus или Enterprise издание](https://www.tencentcloud.com/document/product/1047/34577) и [включить его в консоли](https://intl.cloud.tencent.com/document/product/1047/34419). |
| 80001 | Не прошла проверку безопасности. Проверьте параметры запроса на основе сообщения об ошибке. |

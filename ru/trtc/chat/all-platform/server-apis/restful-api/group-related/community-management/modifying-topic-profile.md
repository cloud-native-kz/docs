# Изменение профиля темы

## Обзор функции

Этот API используется для изменения профиля темы.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддерживается ли этот REST API |
| --- | --- |
| Private | Нет |
| Public | Нет |
| ChatRoom | Нет |
| AVChatRoom | Нет |
| Community | Этот API применяется только к сообществам с включенной функцией тем. |

Это встроенные типы групп в Chat. Дополнительная информация доступна в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание:** Чтобы использовать функцию тем, необходимо перейти в [консоль](https://console.trtc.io/chat/qun-setting), выбрать **Feature Configuration** > **Group configuration** > **Group feature configuration** > **Community**, включить функцию сообщества, а затем включить функцию тем.

### Пример URL запроса

```
https://xxxxxx/v4/million_group_open_http_svc/modify_topic?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор REST API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Силиконовая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/million_group_open_http_svc/modify_topic | API запроса |
| sdkappid | `SDKAppID`, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительная информация доступна в разделе **App Admin** в статье [Аутентификация при входе](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробнее см. в статье [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Фиксированное значение: `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Изменение основной информации темы**
Изменение основной информации темы, такой как название темы и уведомление темы.

```
{  "GroupId":"@TGS#_@TGS#cQVLVHIM62CJ",// Группа темы, которую нужно изменить, требуется  "TopicId":"@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_@TOPIC#cRTE3HIM62C5",// ID темы, которую нужно изменить, требуется  "TopicName":"TestTopicName",// Название темы, опционально  "Notification":"Notification"    // Уведомление темы, опционально}
```

- **ВСЕ В ОДНОМ**

```
{  "GroupId":"@TGS#_@TGS#cQVLVHIM62CJ",	// Группа темы, которую нужно изменить, требуется  "TopicId":"@TGS#_@TGS#cQVLVHIM62CJ@TOPIC#_@TOPIC#cRTE3HIM62C5",	// ID темы, которую нужно изменить, требуется  "From_Account":"1400187352",	// Член, изменяющий тему  "TopicName":"TestTopicName",// Название темы, опционально  "FaceUrl": "http://this.is.new.face.url", // Фотография профиля темы, опционально  "Notification":"Notification",    // Уведомление темы, опционально  "Introduction": "Introduction",// Описание темы, опционально  "MuteAllMember": "On", // Отключение звука для всех членов, опционально. `On`: Включить. `Off`: Отключить   "CustomString":"This is a customs string." // Пользовательская строка, опционально}
```

### Поля запроса

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "ok",    "ErrorCode":0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

HTTP-код состояния, возвращаемый этим API, всегда составляет 200, если не происходит сетевой ошибки (например, ошибка 502). Конкретный код ошибки и сведения можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (от 60000 до 79999) см. раздел [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, характерные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверное слово команды. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10006 | Операция превышает лимит частоты. Сократите частоту вызовов. Эта ошибка обычно вызвана слишком большим увеличением количества групп в сети за один день или слишком частыми вызовами для получения всех групп в приложении. |
| 10007 | Недостаточно прав для операции. Проверьте параметры запроса на основе сообщения об ошибке. |
| 10008 | Неверный запрос, вероятно, потому что проверка информации подписи в запросе не прошла. Повторите попытку или [отправьте запрос в поддержку](https://console.tencentcloud.com/workorder/category?level1_id=29&level2_id=40&source=0&data_title=%E4%BA%91%E9%80%9A%E4%BF%A1%20%20IM&step=1). |
| 10010 | Неверный запрос. Тема была удалена. |
| 10015 | Неверный ID запрашиваемой группы. Проверьте параметр запроса на основе сообщения об ошибке. |
| 10016 | Серверная часть приложения отклонила эту операцию через обратный вызов третьей стороны. Проверьте возвращаемое значение вашего вебхука "Topic Profile Change Webhook". |
| 11000 | Текущая группа не поддерживает функцию темы сообщества. Чтобы использовать эту функцию, необходимо приобрести [Pro edition, Pro Plus edition или Enterprise edition](https://buy.intl.cloud.tencent.com/avc) и [включить его в консоли](https://intl.cloud.tencent.com/document/product/1047/34419). |
| 80001 | Не удалось пройти проверку безопасности. Проверьте параметры запроса на основе сообщения об ошибке. |

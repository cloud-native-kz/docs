# Обновление счётчиков группы

## Обзор функции

Администратор приложения может использовать этот API для обновления (установки/увеличения/уменьшения) счётчиков группы.

> **Примечание:** Для использования этой функции необходимо приобрести [Pro edition, Pro Plus edition или Enterprise edition](https://www.tencentcloud.com/document/product/1047/34577).

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. То же, что и рабочие группы (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. То же, что и группа встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Нет |

Это встроенные типы групп в Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/update_group_counter?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны только изменённые параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/modify_group_attr | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учётная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учётной записью администратора приложения. Подробнее см. [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | 32-битное неподписанное целое число, варьирующееся от 0 до 4294967295. |
| contenttype | Формат запроса. Значение фиксировано как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

**Базовый формат**
Обновление счётчиков группы:

```
{    "GroupId": "@TGS#aC5SZEAEF",    "GroupCounter":[        {            "Key":"like", // Ключ счётчика группы            "Value":5 // Значение счётчика группы        },        {            "Key":"unlike",            "Value":1        }    ],    "Mode": "Set"  // `Set`, `Increase`, `Decrease`}
```

### Поля запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы |
| GroupCounter | Array | Да | Список счётчиков группы. `Key` указывает ключ счётчика группы. Список может содержать до 20 ключей, каждый ключ содержит не более 128 байт. `Value` указывает значение счётчика группы, которое является 64-битным знаковым целым числом. |
| Mode | String | Нет | `Increase` и `Decrease` указывают на увеличение или уменьшение исходного значения соответственно. Если `Key` не существует, создайте новый со `Value`, равным 0, а затем увеличивайте или уменьшайте значение. `Set` указывает на установку `Value` для соответствующего `Key`. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "GroupCounter":[  // Список последних обновленных пар ключ-значение счётчика группы        {            "Key":"like", // Ключ счётчика группы            "Value":5 // Последнее значение счётчика группы        },        {            "Key":"unlike",            "Value":1        }    ]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: Успешно; `FAIL`: Ошибка |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Ошибка |
| ErrorInfo | String | Информация об ошибке |
| GroupCounter | Array | Список последних обновленных пар ключ-значение счётчика группы |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не происходит ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Недостаточно прав на операцию. Например, текущий пользователь не является членом группы. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Неверный ID группы. Проверьте, правильно ли введен ID группы и существует ли группа. |
| 10059 | Для использования этой функции необходимо приобрести Pro edition, Pro Plus edition или Enterprise edition. |
| 10063 | Количество ключей в счётчике группы превышает лимит в 20. |

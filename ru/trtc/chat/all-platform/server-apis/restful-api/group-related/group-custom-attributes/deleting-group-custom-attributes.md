# Удаление пользовательских атрибутов группы

## Обзор функции

Этот API позволяет [администраторам приложения](https://www.tencentcloud.com/document/product/1047/33517#app-admin) удалять пользовательские атрибуты групп.

## Описание вызова API

### Поддерживаемые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Аналогично группе Work в новой версии. |
| Public | Да. |
| ChatRoom | Да. Аналогично группе Meeting в новой версии. |
| AVChatRoom | Да. |
| Community | Да. |

Это встроенные типы групп системы обмена мгновенными сообщениями (IM). Дополнительную информацию см. в разделе [Система групп](https://www.tencentcloud.com/document/product/1047/33529).

### Пример URL запроса

```
https://console.tim.qq.com/v4/group_open_http_svc/delete_group_attr?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые можно изменять при вызове этого API. Дополнительную информацию о параметрах см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| v4/group_open_http_svc/delete_group_attr | API запроса. |
| sdkappid | SDKAppID, назначенный консолью IM при создании приложения. |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительную информацию см. в разделе [Генерация UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, зафиксированный как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Базовый запрос**
Этот пример показывает, как удалить пользовательские атрибуты группы.

```
{  "GroupId": "@TGS#aC5SZEAEF",  "GroupAttr":[      {          "key":"attr_key" // Ключ атрибута      },      {          "key":"attr_key1"      }  ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, пользовательские атрибуты которой необходимо удалить. |
| GroupAttr | Array | Да | Список пользовательских атрибутов, подлежащих удалению, с ключами пользовательских атрибутов. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса.- OK: Успешно- FAIL: Ошибка |
| ErrorCode | Integer | Код ошибки.- 0: Успешно- Другие значения: Ошибка |
| ErrorInfo | String | Информация об ошибке. |

## Коды ошибок

HTTP код состояния, возвращаемый этим API, всегда равен 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и информация об ошибке указываются в ErrorCode и ErrorInfo соответственно в ответе.

Для общих кодов ошибок (60000–79999) см. раздел [Коды ошибок](https://www.tencentcloud.com/document/product/1047/34348).

В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10004 | Недопустимый параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Отсутствует разрешение на операцию. Например, группа этого типа не имеет разрешения на операцию с пользовательскими атрибутами. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Недопустимый ID группы. Используйте правильный ID группы. |
| 10045 | Размер ключа пользовательского атрибута превышает лимит в 32 байта. |
| 10049 | Частота записи пользовательских атрибутов в группу превышает лимит. Вы можете добавлять, удалять или изменять атрибуты не более 5 раз в секунду. |
| 10050 | Ошибка удаления несуществующего пользовательского атрибута. |

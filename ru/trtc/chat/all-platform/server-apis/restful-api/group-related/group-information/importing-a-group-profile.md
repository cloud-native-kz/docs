# Импорт профиля группы

## Обзор функции

Этот API позволяет администратору приложения импортировать данные группы без запуска обратных вызовов или доставки уведомлений. Когда ваше приложение необходимо перенести в Chat из другой системы мгновенного обмена сообщениями, вы можете использовать этот API для импорта существующих данных группы.

## Описание вызова API

### Поддерживаемые типы групп

| Тип группы | Поддержка этого RESTful API |
| --- | --- |
| Private | Да. В новой версии аналогично рабочим группам (Work). |
| Public | Да |
| ChatRoom | Да. В новой версии аналогично группам встреч (Meeting). |
| AVChatRoom | Нет |
| Community | Да |

Выше приведены встроенные типы групп сервиса Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание:** Группы AVChatRoom не поддерживают импорт базовых данных группы. Если вы попытаетесь импортировать базовые данные группы для групп AVChatRoom, будет возвращена ошибка 10007. Чтобы достичь эффекта импорта базовых данных группы, вы можете [создать группу](https://intl.cloud.tencent.com/document/product/1047/34895) и [изменить базовые данные группы](https://intl.cloud.tencent.com/document/product/1047/34962).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/import_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где расположен ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/import_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в документе [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробнее см. в документе [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение фиксировано на `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Базовый формат**
Импорт группы. Вы можете использовать `CreateTime` для указания времени создания группы.

```
{  "Owner_Account": "leckie", // ID пользователя владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom или Community (обязательно)  "Name": "TestGroup", // Имя группы (обязательно)  "CreateTime": 1448357837 // Время создания группы (необязательно). Если это поле не указано, время создания по умолчанию — это время запроса.}
```

- **Формат содержащий только параметр включения тем**
Импорт группы и указание ее типа как группы сообщества с поддержкой тем.

```
{  "Type": "Community", // Тип группы (обязательно)  "Name": "test_import_group",  // Имя группы (обязательно)  "SupportTopic": 1// Поддерживается ли опция темы. Допустимые значения: `1`: да; `0`: нет.}
```

- **Указание других дополнительных полей**
Вы можете указать дополнительные поля, такие как `Introduction` и `Notice`. Формат запроса совпадает с форматом запроса создания группы.

```
{  "Owner_Account": "leckie", // ID пользователя владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom или Community (обязательно)  "GroupId":"MyFirstGroup", // Определяемый пользователем ID группы для внешнего отображения (необязательно)  "Name": "TestGroup", // Имя группы (обязательно)  "Introduction": "This is group Introduction", // Описание группы (необязательно)  "Notification": "This is group Notification", // Уведомление группы (необязательно)  "FaceUrl": "http://this.is.face.url",  "MaxMemberNum": 500, // Максимальное количество членов группы (необязательно)  "ApplyJoinOption": "FreeAccess", // Способ обработки запросов на присоединение к группе (необязательно)  "CreateTime": 1448357837, // Время создания группы (необязательно). Если это поле не указано, время создания по умолчанию — это время запроса.  "AppDefinedData": [ // Пользовательское поле группы (необязательно)      {          "Key": "GroupTestData1", // Ключ пользовательского поля приложения          "Value": "xxxxx" // Значение пользовательского поля      },      {          "Key": " GroupTestData2",          "Value": "abc\\u0000\\u0001" // Пользовательское поле поддерживает двоичные данные.      }  ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| Owner_Account | String | Необязательно | ID владельца группы, который будет автоматически добавлен в членов группы. Если это поле не указано, группа не будет иметь владельца. |
| Type | String | Обязательно | Тип группы, который может быть Public, Private (Work), ChatRoom или Community. |
| GroupId | String | Необязательно | Чтобы упростить ID групп и сделать их легко запоминающимися, Tencent Cloud позволяет приложениям настраивать ID групп во время создания группы через RESTful API. Подробнее см. в документе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529). |
| Name | String | Обязательно | Имя группы, максимальная длина которого составляет 30 байт. |
| Introduction | String | Необязательно | Описание группы, максимальная длина которого составляет 240 байт. |
| Notification | String | Необязательно | Уведомление группы, максимальная длина которого составляет 300 байт. |
| FaceUrl | String | Необязательно | URL фото профиля группы, максимальная длина которого составляет 100 байт. |
| MaxMemberNum | Integer | Необязательно | Максимальное количество членов группы, которое составляет максимум 6000. Значение по умолчанию — 2000. В настоящее время группа сообщества поддерживает до 100000 членов группы. |
| ApplyJoinOption | String | Необязательно | Способ обработки запросов на присоединение к группе. Допустимые значения: `FreeAccess`, `NeedPermission` (по умолчанию) и `DisableApply`. Это поле не поддерживается группами сообщества. |
| AppDefinedData | Array | Необязательно | Пользовательское поле группы. По умолчанию это поле недоступно и должно быть включено перед использованием. Подробнее см. в документе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529). |
| CreateTime | Integer | Необязательно | Время создания группы |
| SupportTopic | Integer | Нет | Поддерживается ли опция темы. Это поле действительно только для сообществ. |

### Пример ответа

- **Базовый формат**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "@TGS#2J4SZEAEL"}
```

- **Формат содержащий только параметр включения тем**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "@TGS#_@TGS#c4YHCIIM62CX"}
```

- **Указание других дополнительных полей**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "MyFirstGroup"}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |
| GroupId | String | ID группы после успешного создания, который назначается бэкэндом Chat или указывается пользователями. |

## Коды ошибок

HTTP статус-код для этого API всегда равен 200, если не возникает сетевая ошибка (такая как ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверное командное слово. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Нет прав операции. Эта ошибка возникает, например, когда член публичной группы пытается удалить других пользователей из группы (только администратор приложения может выполнить эту операцию). |
| 10021 | ID группы уже используется. Укажите другой ID группы. |

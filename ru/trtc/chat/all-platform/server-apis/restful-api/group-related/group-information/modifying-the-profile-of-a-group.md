# Изменение профиля группы

## Обзор функции

Этот API используется администратором приложения для изменения профиля указанной группы.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Аналогично рабочим группам (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Аналогично группам встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Да |

Это предустановленные типы групп в Chat. Дополнительная информация в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Образец URL запроса

```
https://xxxxxx/v4/group_open_http_svc/modify_group_base_info?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/modify_group_base_info | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительная информация в разделе **App Admin** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробнее см. [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который должен быть всегда `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Образец запроса

- **Изменение основной информации о группе**
Изменение основной информации о группе, такой как имя группы и объявление группы.

```
{  "GroupId": "@TGS#1NVTZEAE4", // Группа, чей основной профиль вы хотите изменить (обязательно)  "Name": "NewName", // Имя группы (опционально)  "Introduction": "NewIntroduction", // Введение группы (опционально)  "Notification": "NewNotification", // Уведомление группы (опционально)  "FaceUrl": "http://this.is.new.face.url", // Фото профиля группы (опционально)  "MaxMemberNum": 500, // Максимальное количество членов группы (опционально)  "ApplyJoinOption": "NeedPermission", // Способ подачи заявления на присоединение к группе (опционально)  "MuteAllMember": "On" // Отключает звук для всех членов (опционально). "On": Включить, "Off": Отключить}
```

- **Установка пользовательской информации группы**
Установка информации пользовательского поля группы. По умолчанию пользовательская информация недоступна и должна быть включена в [консоли Chat](https://console.tencentcloud.com/im). Подробнее см. в таблице описания полей пакета запроса.

```
{  "GroupId": "@TGS#1NVTZEAE4", // Группа, чей основной профиль вы хотите изменить (обязательно)  "AppDefinedData": [ // Пользовательское поле (опционально)      {          "Key": "GroupTestData1", // Ключ пользовательского поля для изменения          "Value": "NewData"  // Новое значение пользовательского поля      }  ]}
```

- **Удаление пользовательской информации группы**
Удаление информации пользовательского поля группы, которую вы установили.

```
{  "GroupId": "@TGS#1NVTZEAE4", // Группа, чей основной профиль вы хотите изменить (обязательно)  "AppDefinedData": [  // Пользовательское поле (опционально)      {          "Key": "GroupTestData2",          "Value": "" // Если этот параметр пуст, пользовательское поле будет удалено.      }  ]}
```

- **ВСЕ В ОДНОМ**

```
{  "GroupId": "@TGS#2J4SZEAEL", // Группа, чей основной профиль вы хотите изменить (обязательно)  "Name": "NewName", // Имя группы (опционально)  "Introduction": "NewIntroduction", // Введение группы (опционально)  "Notification": "NewNotification", // Уведомление группы (опционально)  "FaceUrl": "http://this.is.new.face.url", // Фото профиля группы (опционально)  "MaxMemberNum": 500, // Максимальное количество членов группы (опционально)  "ApplyJoinOption": "NeedPermission", // Способ подачи заявления на присоединение к группе (опционально)  "MuteAllMember": "On", // Отключает звук для всех членов, что опционально. `On`: Включить. `Off`: Отключить  "AppDefinedData": [ // Пользовательское поле (опционально)      {          "Key": "GroupTestData1", // Ключ пользовательского поля для изменения          "Value": "NewData" // Новое значение пользовательского поля      },      {          "Key": "GroupTestData2",          "Value": "" // Если этот параметр пуст, пользовательское поле будет удалено.      }  ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Обязательно | ID группы, основную информацию которой вы хотите изменить. |
| Name | String | Опционально | Максимальная длина составляет 30 байт. Это поле кодируется в UTF-8, и один китайский символ занимает три байта. |
| Introduction | String | Опционально | Максимальная длина составляет 240 байт. Это поле кодируется в UTF-8, и один китайский символ занимает три байта. |
| Notification | String | Опционально | Максимальная длина составляет 300 байт. Это поле кодируется в UTF-8, и один китайский символ занимает три байта. |
| FaceUrl | String | Опционально | URL фото профиля группы, максимальная длина составляет 100 байт. |
| MaxMemberNum | Integer | Нет | Максимальное количество членов группы. Для рабочих групп (Work), публичных групп (Public), групп встреч (Meeting) и сообществ см. верхний лимит количества членов группы в [Системе групп](https://intl.cloud.tencent.com/document/product/1047/33529#.E7.BE.A4.E7.BB.84.E9.99.90.E5.88.B6.E5.B7.AE.E5.BC.82). Для аудиовизуальных групп (AVChatRoom) это поле невалидно и не требует указания. |
| ApplyJoinOption | String | Опционально | Способ обработки заявлений на присоединение к группе. Допустимые значения: `FreeAccess`, `NeedPermission` и `DisableApply`. Группы сообщества в настоящее время не поддерживают это поле. |
| AppDefinedData | Array | Опционально | По умолчанию это поле недоступно. Чтобы узнать, как включить пользовательские поля на уровне группы, см. [Пользовательские поля](https://intl.cloud.tencent.com/document/product/1047/33529#.E8.87.AA.E5.AE.9A.E4.B9.89.E5.AD.97.E6.AE.B5). |
| MuteAllMember | String | Опционально | Отключает звук для всех членов группы и позволяет говорить только администратору группы, владельцу группы и системному администратору. |
| InviteJoinOption | String | Опционально | Способ обработки приглашений присоединиться к группе. Допустимые значения: FreeAccess (прямое приглашение пользователей в группу без одобрения и других операций), NeedPermission (требование одобрения администратора группы или владельца) и DisableInvite (не поддержка SDK приглашений в группу). Этот параметр не поддерживается типом группы AVChatRoom. |
| From_Account | String | Опционально | Учетная запись источника оператора, опционально. Если это поле не заполнено, оператор, изменяющий информацию о группе, является учетной записью администратора приложения, используемой при вызове этого интерфейса. Кроме того, приложение также может "подделать" оператора, изменяющего информацию о группе, через это поле для реализации некоторых специальных функциональных требований. Если вы указываете это поле, вы должны убедиться, что учетная запись в поле существует. |
| EnablePermission | Integer | Опционально | Только сообщества, поддерживающие темы, могут установить это поле. Включена ли группа разрешений в качестве основы для определения разрешений: 0: Отключить 1: Включить |
| DefaultPermission | Integer | Опционально | Только группы, включившие группу разрешений в качестве основы для определения разрешений, могут установить это поле. Разрешения по умолчанию для члена группы в группе. |

### Образец ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode":0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: не удалось |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Не удалось |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

Возвращаемый HTTP код состояния для этого API всегда 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, характерные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверное слово команды. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10026 | Команда SDKAppID запроса отключена. Пожалуйста, [отправьте билет](https://console.tencentcloud.com/workorder/category?level1_id=29&level2_id=40&source=0&data_title=%E4%BA%91%E9%80%9A%E4%BF%A1%20%20IM&step=1). |
| 10007 | Недостаточные разрешения на операцию. Проверьте, является ли оператор администратором приложения или имеет ли оператор разрешение на изменение полей в запросе. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Неверный ID группы. Используйте правильный ID группы. |

# Импорт участников группы

## Обзор функции

- Этот API используется для импорта участников группы без срабатывания обратных вызовов и отправки уведомлений.
- Когда вашему приложению требуется миграция в Chat из другой системы мгновенных сообщений, вы можете использовать этот API для импорта существующих данных участников группы.

## Описание вызова API

### Применимые типы групп

| Тип группы | Применимо |
| --- | --- |
| Private | Да. Аналогично рабочим группам (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Аналогично группам встреч (Meeting) в новой версии. |
| AVChatRoom | Нет |
| Community | Да |

Выше приведены встроенные типы групп сервиса Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание** Обычно импорт участников группы не требуется для аудиовидео-групп (AVChatRoom), поэтому аудиовидео-группы не поддерживают этот API. Если вы попытаетесь использовать этот API для аудиовидео-группы, будет возвращена ошибка 10007.

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/import_group_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. раздел [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где расположен ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/import_group_member | Запрос API |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **Администратор приложения** в документе [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Подробные сведения см. в разделе [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака, значение которого варьируется от 0 до 4294967295. |
| contenttype | Формат запроса. Значение зафиксировано как `json`. |

### Максимальная частота вызовов

200 вызовов в секунду.

### Пример запроса

Вы можете вызвать этот API для отправки запроса на импорт участников в группу. Один запрос может импортировать до 20 участников. Если количество непрочитанных сообщений, указанное этим API, больше, чем текущее количество сообщений группы, количество непрочитанных сообщений устанавливается на текущее количество сообщений группы.

> **Внимание** Перед импортом участника убедитесь, что время присоединения участника к группе позже времени создания группы и раньше текущего момента времени. В противном случае участник не может быть импортирован.

```
{    "GroupId": "@TGS#2J4SZEAEL", // (Обязательно) Целевая группа    "MemberList": [ // За один раз можно добавить до 300 участников.    {        "Member_Account": "tommy", // (Обязательно) Идентификатор добавляемого участника        "Role":"Admin", // (Необязательно) Роль участника группы, который будет импортирован. В настоящее время поддерживается только роль Admin.        "JoinTime":1448357837, // (Необязательно) Время присоединения импортируемого участника группы к группе        "UnreadMsgNum":5          // (Необязательно) Количество непрочитанных сообщений участника    },    {        "Member_Account": "jared",        "JoinTime":1448357857,        "UnreadMsgNum":2    }]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | Идентификатор целевой группы |
| MemberList | Array | Да | Список добавляемых участников |
| Member_Account | String | Да | Учетные записи участников группы, которые будут импортированы |
| Role | String | Нет | Роль участника группы, который будет импортирован. В настоящее время поддерживается только `Admin`. Если это поле не указано, используется значение `Member` (обычный пользователь). |
| JoinTime | Integer | Нет | Время присоединения импортируемого участника группы к группе |
| UnreadMsgNum | Integer | Нет | Количество непрочитанных сообщений участника группы, который будет импортирован |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "MemberList": [    {         "Member_Account": "tommy",         "Result": 1 // Результат импорта. 0: Ошибка; 1: Успех; 2: Уже является участником группы    },    {         "Member_Account": "jared",         "Result": 1    }]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Ошибка |
| ErrorInfo | String | Информация об ошибке |
| MemberList | Array | Результат добавления участников |
| Member_Account | String | Учетная запись участника группы |
| Result | Integer | Результат импорта. 0: Ошибка; 1: Успех; 2: Уже является участником группы |

## Коды ошибок

Возвращаемый этим API код состояния HTTP всегда равен 200, если только не возникает сетевая ошибка (например, ошибка 502). Конкретный код ошибки и подробные сведения можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. раздел [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, относящиеся к этому API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверное слово команды. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните неполадки. |
| 10005 | Пакет запроса содержит слишком много учетных записей. |
| 10006 | Достигнут лимит частоты операций. Пожалуйста, сократите частоту вызовов. |
| 10007 | Нет прав на операцию. Например, обычный участник в открытой группе пытается удалить других пользователей из группы, но это может сделать только администратор приложения. |
| 10010 | Группа не существует или была распущена. |
| 10015 | Неверный идентификатор группы. Используйте правильный идентификатор группы. |

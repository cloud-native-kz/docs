# Получение количества активных пользователей в аудиовизуальной группе

## Описание функции

Администратор приложения может использовать идентификатор группы для получения количества активных пользователей в группе прямой трансляции.

## Описание API

### Применимые типы групп

| ID типа группы | Поддержка REST API |
| --- | --- |
| Private | Да. Аналогично Work (рабочая группа) в новой версии. |
| Public | Да. |
| ChatRoom | Да. Аналогично Meeting (временная группа встреч) в новой версии. |
| AVChatRoom | Да. |
| Community | Да. |

Это встроенные в IM группы. Дополнительную информацию см. в [Системе групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание**: Количество активных пользователей обновляется каждые 10 секунд. Если в группе есть веб-пользователи, количество активных пользователей обновляется через 10–20 секунд после присоединения или выхода пользователя из группы. Для групп Private, Public, ChatRoom и Community требуется [пакет Pro, Pro Plus или Enterprise](https://buy.tencentcloud.com/avc?from=17473), а функция "Статус активности участников группы" должна быть активирована (через "Конфигурация функций группы" в консоли).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/get_online_member_num?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже описаны только измененные параметры при вызове этого API. Дополнительную информацию о других параметрах см. в [Обзоре RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| https | Протокол запроса — HTTPS, метод запроса — POST. |
| xxxxxx | Страна/регион, в котором находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/get_online_member_num | API, на который отправляется запрос. |
| sdkappid | SDKAppID, назначенный консолью IM при создании приложения. |
| identifier | Должна быть учетная запись администратора приложения. Дополнительную информацию см. в разделе **Администратор приложения** в [Аутентификации входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительную информацию об операции см. в [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение всегда `json`. |

### Максимальная частота вызовов

200 раз/сек

### Пример пакета запроса

Этот API используется для получения количества активных пользователей в группе прямой трансляции (AVChatRoom). Количество активных пользователей обновляется каждые 10 секунд. Рекомендуется, чтобы backend запрашивал количество активных пользователей каждые 10 секунд и синхронизировал его со всеми пользователями в группе.

```
{    "GroupId":"@TGS#a6I4ZUUGO"}
```

### Поля пакета запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | Идентификатор группы. |

### Пример пакета ответа

```
{    "ActionStatus":"OK",    "ErrorInfo":"",     "ErrorCode": 0,     "OnlineMemberNum":1000 // Количество активных пользователей}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно. `FAIL`: ошибка. |
| ErrorCode | Integer | Код ошибки. `0`: успешно. Другие значения: ошибка. |
| ErrorInfo | String | Информация об ошибке. |
| OnlineMemberNum | Integer | Количество активных пользователей в группе. |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не произойдет ошибка сети (например, ошибка 502). Конкретный код ошибки и подробную информацию можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок 60000–79999 см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В таблице ниже описаны коды ошибок, относящиеся к этому API.

| Код ошибки | Описание |
| --- | --- |
| 10002 | Произошла ошибка системы. Повторите попытку или обратитесь в техническую поддержку. |
| 10003 | Команда запроса недействительна. Повторите попытку или обратитесь в техническую поддержку. |
| 10004 | Параметр недействителен. Проверьте поле `ErrorInfo` в пакете ответа, чтобы убедиться, что обязательные поля указаны или что поля установлены согласно требованиям протокола. |
| 10007 | У вас нет необходимых разрешений. Вы можете получить количество активных пользователей только для групп прямой трансляции. |
| 10010 | Группа не существует или была распущена. |
| 10015 | Идентификатор группы недействителен. Проверьте идентификатор группы. |

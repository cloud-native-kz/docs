# Добавление подписчиков

## Обзор функции

API используется администратором приложения для добавления пользователя в качестве подписчика на официальный канал.

> **Примечание:** Функция доступна только для Enterprise. Чтобы её включить, [свяжитесь с нами](https://trtc.io/zh/contact). Вы можете [испробовать](https://trtc.io/document/34577?product=chat#.E5.89.8D.E6.8F.90.E6.9D.A1.E4.BB.B6) эту функцию в консоли. Пробная версия позволяет создавать до 5 открытых аккаунтов сроком на один месяц.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/official_account_open_http_svc/add_subscriber?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже представлены только параметры, которые изменяются при вызове этого API, и их описания. Подробнее о параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя, соответствующее стране/региону SDKAppID: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/official_account_open_http_svc/add_subscriber | Запрос API. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Аккаунт администратора приложения. Подробнее см. [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробнее см. [Генерирование UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Значение всегда `json`. |

### Максимальная частота вызовов

200 раз/секунду

### Пример пакета запроса

```
{    "Official_Account": "@TOA#_@TOA#dOEF",		// Требуемый официальный канал (обязательно)    "SubscriberList": [		// За один раз можно добавить до 100 подписчиков.        {            "Subscriber_Account": "107867"	// UserID подписчика, который нужно добавить        },      	{            "Subscriber_Account": "966582476"        }    ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| Official_Account | String | Да | ID официального канала для подписки. Используйте поле OfficialAccountUserID, возвращаемое API при создании. |
| SubscriberList | Array | Да | Массив добавляемых подписчиков. |
| Subscriber_Account | String | Да | UserID подписчика, который нужно добавить. |

### Пример пакета ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "SubscriberList": [        {            "Subscriber_Account": "107867",            "Result": 2        },        {            "Subscriber_Account": "966582476",            "Result": 1		// Результат добавления. 0: ошибка; 1: успех; 2: уже подписан.        }    ]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки: OK: успех. FAIL: ошибка. |
| ErrorCode | Integer | Код ошибки: 0: успех. Другие значения: ошибка. |
| ErrorInfo | String | Информация об ошибке. |
| SubscriberList | Array | Возвращённые результаты добавленных подписчиков. |
| Subscriber_Account | String | Возвращённый UserID подписчика. |
| Result | Integer | Результат добавления: 0: ошибка. 1: успех. 2: уже подписан на этот официальный канал. |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), HTTP код возврата для этого API всегда равен 200. ErrorCode и ErrorInfo в пакетах ответа представляют фактический код и информацию об ошибке.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://www.tencentcloud.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 6017 | Недопустимый параметр запроса. Измените и повторите попытку. |
| 10004 | Внутренняя ошибка сервера. Попробуйте снова. |
| 20001 | Недопустимый пакет запроса. Измените и повторите попытку. |
| 20005 | Внутренняя ошибка сервера. Попробуйте снова. |
| 130002 | Недостаточно прав доступа. Проверьте параметры запроса в соответствии с сообщением об ошибке. |
| 130004 | Пользователь официального канала не существует, или существовал, но был удалён. |
| 130006 | Количество подписчиков для этого официального канала достигло лимита. Обратитесь к разделу [Тарификация](https://www.tencentcloud.com/document/product/1047/34349), чтобы приобрести или обновить соответствующие пакеты и расширить максимальное количество подписчиков для официального канала. (После обновления необходимо вызвать API [Изменение информации об официальном аккаунте](https://www.tencentcloud.com/document/product/1047/60757), чтобы обновить максимальное количество поддерживаемых подписчиков.) |
| 130007 | Неправильный ID пользователя официального канала. Измените и повторите попытку. |
| 130008 | ID пользователя официального канала уже используется. |
| 131000 | Для этого AppId не включена служба официальных каналов. |
| 130010 | Количество официальных каналов, на которые может подписаться подписчик, превысило лимит. Проверьте и отпишитесь от некоторых официальных каналов. |

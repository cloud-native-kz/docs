# Обновление данных группы разговора

## Обзор функции

Данные метки группы разговора независимы от последних контактов. Вы можете использовать RESTful API для добавления, удаления, изменения и запроса группы разговора, стандартной метки разговора, пользовательской метки разговора и других полей. Метки группы разговора поддерживают до 1000 разговоров. Этот API можно использовать для обновления данных группы разговора. Операции с данными группы разговора поддерживаются только в редакциях Pro, Pro Plus и Enterprise.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/recentcontact/update_contact_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны изменённые параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://trtc.io/document/34620?product=chat&menulabel=serverapi).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com`. Сингапур: `adminapisgp.im.qcloud.com`. Сеул: `adminapikr.im.qcloud.com`. Франкфурт: `adminapiger.im.qcloud.com`. Кремниевая долина: `adminapiusa.im.qcloud.com`. Джакарта: `adminapiidn.im.qcloud.com` |
| v4/recentcontact/update_contact_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учётная запись администратора приложения. Дополнительные сведения см. в разделе **Администратор приложения** в [Аутентификация входа](https://trtc.io/document/33517?platform=javascript&product=chat&menulabel=coresdk). |
| usersig | Подпись, созданная учётной записью администратора приложения. Подробнее см. [Создание UserSig](https://trtc.io/document/34385?product=chat&menulabel=serverapi). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

```
{    "From_Account":"user20",    "UpdateType":1,    "UpdateGroup":{        "UpdateGroupType":2,        "OldGroupName":"test1",        "ContactUpdateItem":[            {                "ContactOptType":1,                "ContactItem":{                    "Type": 1,                    "To_Account": "user1"                }            }        ]    }}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| From_Account | String | Да | Заполните UserID. Запрос на обновление группы разговора для этого пользователя. |
| UpdateType | Integer | Да | `1`: Добавление разговора в группу или удаление разговора из группы |
| UpdateGroup | Object | Да | Добавление разговора в группу или удаление разговора из группы |
| UpdateGroupType | Integer | Да | Тип обновления. `1`: Обновление имени группы; `2`: Обновление группы разговора |
| OldGroupName | String | Да (если `UpdateGroupType` равен 1) | Имя группы для обновления |
| NewGroupName | String | Да (если `UpdateGroupType` равен 1) | Обновленное имя группы, которое может быть до 32 байт |
| ContactUpdateItem | Array | Нет | Разговор для обновления |
| ContactOptType | Integer | Обязательно | Тип обновления. `1`: Добавление разговора в группу; `2`: Удаление разговора из группы. |
| ContactItem | Object | Да | Объект разговора |
| Type | Integer | Да | Тип разговора: 1 — личный разговор; 2 — групповой разговор |
| ToGroupId | String | Да | Присваивается только для группового разговора, заполните ID группы стороны разговора. |
| To_Account | String | Да | Присваивается только для личного разговора, заполните UserID стороны разговора. |

### Пример ответа

```
{    "UpdateType": 1,    "UpdateGroupResult": {        "UpdateGroupType": 2,        "GroupName": "test1",        "GroupId": 1,        "OldGroupName": "test1",        "ContactResultItem": [            {                "ContactOptType": 1,                "ContactItem": {                    "Type": 1,                    "To_Account": "user1"                },                "ResultCode": 0,                "ResultInfo": ""            }        ]    },    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": ""}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: Успешно; `FAIL`: Неудачно |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Неудачно |
| ErrorInfo | String | Информация об ошибке |
| UpdateType | Integer | Обязательно |
| UpdateGroupResult | Object | Результат обновления группы |
| UpdateGroupType | Integer | Да |
| GroupName | String | Текущее имя группы |
| OldGroupName | String | Исходное имя группы |
| GroupId | Integer | Текущий ID группы |
| ContactResultItem | Array | Результат операции разговора |
| ContactOptType | Integer | Обязательно |
| ContactItem | Object | Объект разговора |
| Type | Integer | Тип разговора: 1 — личный разговор; 2 — групповой разговор |
| ToGroupId | String | Присваивается только для группового разговора, заполните ID группы стороны разговора |
| To_Account | String | Присваивается только для личного разговора, заполните UserID стороны разговора |
| ResultCode | Integer | Код ошибки в результате операции разговора. `0`: Успешно |
| ResultInfo | String | Описание ошибки в результате операции разговора |

## Коды ошибок

Возвращаемый HTTP код состояния для этого API всегда равен 200, если не возникает сетевая ошибка (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ResultCode`, `ResultInfo`, `ErrorCode` и `ErrorInfo`.

Для кодов ошибок общего пользования (60000–79999) см. [Коды ошибок](https://trtc.io/document/34348?platform=javascript&product=chat&menulabel=coresdk).

В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 50001 | Запрошенный UserID не был импортирован в бэкенд Tencent Cloud Chat. Пожалуйста, выполните импорт. |
| 50002 | Неправильный параметр запроса. Проверьте ваш запрос в соответствии с описанием ошибки. |
| 50003 | Запрос требует разрешений администратора приложения. |
| 50004 | Внутренняя ошибка сервера. Пожалуйста, попробуйте снова. |
| 50005 | Timeout сети. Попробуйте позже. |
| 51006 | При изменении метки разговора количество разговоров пусто или превышает верхний лимит 100. |
| 51007 | Не удалось заменить GroupID на GroupCode из-за внутренней ошибки или распада группы. |
| 51008 | Общее количество разговоров в группе разговора превышает 1000. |
| 51009 | Группа разговора не существует при попытке удаления. |
| 51010 | Количество групп разговора превышает верхний лимит 20. |
| 51011 | Имя группы разговора содержит более 32 байт. |
| 51012 | Превышено максимальное количество разговоров, закреплённых вверху. |
| 51013 | Если запрошенное приложение не относится к редакциям Pro, Pro Plus или Enterprise, при изменении стандартной метки разговора или группы разговора будет выведена ошибка. |

# Вебхук после отзыва сообщений официального канала

## Обзор функции

Бэкенд приложения может просматривать отзыв сообщений официального канала в реальном времени через этот вебхук.

## Примечания

- Для включения вебхука необходимо настроить URL вебхука и включить соответствующий протокол. Дополнительные сведения о методе настройки см. в документе [Конфигурация вебхука](https://www.tencentcloud.com/document/product/1047/34520).
- Во время работы этого вебхука бэкенд Chat инициирует HTTP POST запрос к бэкенду приложения.
- После получения запроса вебхука бэкенд приложения должен проверить, что SDKAppID, содержащийся в URL запроса, является `SDKAppID` приложения.
- Для дополнительных соображений безопасности см. документ [Обзор вебхука: Соображения безопасности](https://www.tencentcloud.com/document/product/1047/34354).

## Сценарии срабатывания вебхука

- Пользователи приложения отзывают сообщения официального канала через клиент.
- Администратор приложения отзывает сообщение официального канала через REST API.

## Время срабатывания вебхука

После успешного отзыва сообщения официального канала.

## Описание вызова API

### Пример URL запроса

В последующем примере URL вебхука, настроенный в приложении, — `https://www.example.com`.
**Пример:**

```
https://www.example.com?SdkAppid=$SDKAppID&CallbackCommand=$CallbackCommand&contenttype=json&ClientIP=$ClientIP&OptPlatform=$OptPlatform
```

### Параметры запроса

| Параметр | Описание |
| --- | --- |
| https | Протокол запроса — HTTPS, метод запроса — POST. |
| www.example.com | URL вебхука. |
| SdkAppid | SDKAppID, назначенный консолью Chat при создании приложения. |
| CallbackCommand | Фиксированное значение: OfficialAccount.CallbackAfterMsgWithDraw. |
| contenttype | Фиксированное значение: JSON. |
| ClientIP | IP-адрес клиента, например: 127.0.0.1. |
| OptPlatform | Платформа клиента. Описание значений см. в документе [Обзор вебхука: Протокол вебхука](https://www.tencentcloud.com/document/product/1047/34354) для значения параметра OptPlatform. |

### Пример запроса

```
{    "CallbackCommand":"OfficialAccount.CallbackAfterMsgWithDraw", // Команда вебхука    "Official_Account":"#TOA#_123", // ID пользователя официального канала    "WithdrawMsgList":[ // Список MsgKey для отзыва сообщений                   {          "MsgKey":"71_1_1698741698"        }    ],    "EventTime": 1670574414123// Временная метка срабатывания события в миллисекундах		}
```

### Поля запроса

| Объект | Характеристики | Функция |
| --- | --- | --- |
| CallbackCommand | String | Команда вебхука. |
| Official_Account | String | ID пользователя официального канала. |
| WithdrawMsgList | Array | Список MsgKey для отзыва сообщений. |
| EventTime | Integer | Временная метка срабатывания события в миллисекундах. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0 // Игнорировать результат вебхука}
```

### Поля ответа

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| ActionStatus | String | Обязательно | Результат обработки запроса: OK: указывает на успешную обработку; FAIL: указывает на ошибку |
| ErrorCode | Integer | Обязательно | Код ошибки. Значение 0 здесь означает игнорировать ответ. |
| ErrorInfo | String | Обязательно | Сообщение об ошибке. |

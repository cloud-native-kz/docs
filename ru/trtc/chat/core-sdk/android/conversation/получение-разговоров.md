# Получение разговоров

## Обзор

SDK предоставляет API для получения разговоров, который можно использовать для получения информации объекта `V2TIMConversation` одного или нескольких указанных разговоров.

### Получение указанного разговора

Вызовите `getConversation` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMConversationManager.html#a619aaff2bb5664e094d2341819b95096) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Conversation.html#v2timmanager.getconversation(conversationid:succ:fail:)) /  [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Conversation_08.html#ad4b7b80fbe0cff25027371b416ede9f9) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMConversationManager.html#a9891f4b029e7a1fd3d17398cbe1b367c)) для получения информации о разговоре, который представляет собой объект `V2TIMConversation`. Состав `conversationID` можно найти в [Обзоре](https://www.tencentcloud.com/document/product/1047/48329).

Пример кода:

Java

Swift

Objective-C

C++

```

```

```
template <class T>class ValueCallback final : public V2TIMValueCallback<T> {public:    using SuccessCallback = std::function<void(const T&)>;    using ErrorCallback = std::function<void(int, const V2TIMString&)>;    ValueCallback() = default;    ~ValueCallback() override = default;    void SetCallback(SuccessCallback success_callback, ErrorCallback error_callback) {        success_callback_ = std::move(success_callback);        error_callback_ = std::move(error_callback);    }    void OnSuccess(const T& value) override {        if (success_callback_) {            success_callback_(value);        }    }    void OnError(int error_code, const V2TIMString& error_message) override {        if (error_callback_) {            error_callback_(error_code, error_message);        }    }private:    SuccessCallback success_callback_;    ErrorCallback error_callback_;};V2TIMString conversationID = u8"conversationID";auto callback = new ValueCallback<V2TIMConversation>{};callback->SetCallback(    [=](const V2TIMConversation& conversation) {        // Obtained the conversation successfully        delete callback;    },    [=](int error_code, const V2TIMString& error_message) {        // Failed to obtain the conversation        delete callback;    });V2TIMManager::GetInstance()->GetConversationManager()->GetConversation(conversationID, callback);
```

### Получение указанных разговоров

Вызовите `getConversationList` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMConversationManager.html#a41ebb09032a6bbda0a78e8734c61fb93) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Conversation.html#v2timmanager.getconversationlist(conversationidlist:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Conversation_08.html#ab1f5e86e270b122cb725266d234d9dd5) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMConversationManager.html#aa1aec337ed398f603f15f14ee9023d62)) для получения списка указанных разговоров, содержащего объекты `V2TIMConversation`.

Java

Swift

Objective-C


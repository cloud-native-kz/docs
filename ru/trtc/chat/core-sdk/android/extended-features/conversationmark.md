# ConversationMark

### Функция плагина

Начиная с версии 7.3, можно интегрировать плагин отметки разговора `TUIConversationMarkPlugin`. Плагин отметки разговора — это закрытая библиотека пользовательского интерфейса, которая зависит от `TUIConversation`. После интеграции плагина вы сможете отмечать и снимать отметки с разговоров в `TUIConversation`.

### Интеграция Android

`TUIConversationMarkPlugin` — это закрытый плагин, который необходимо интегрировать через gradle. Найдите build.gradle приложения и добавьте зависимость плагина отметки в раздел dependencies.

Классическая версия

```
  dependencies {    ...    # Интегрировать функцию разговора.    api project(':tuiconversation')    ...    # Интегрировать плагин отметки разговора, поддерживается начиная с версии 7.3.    api "com.tencent.imsdk:tuiconversationmark-plugin:7.3.4358"    ...}
```

### Интеграция iOS

`TUIConversationMarkPlugin`, как обычные компоненты TUIKit, можно быстро интегрировать через CocoaPods. Подробные шаги интеграции можно найти в [Интеграция базовых функций](https://trtc.io/zh/document/60521).

Для плагина отметки разговора вам нужно добавить только одну строку в Podfile для завершения интеграции:

Классическая версия

```
# Раскомментируйте следующую строку для определения глобальной платформы вашего проекта# ...  # Интегрировать функцию разговора.  pod 'TUIConversation/UI_Classic'   ...  # Интегрировать плагин отметки разговора, поддерживается начиная с версии 7.3.  pod 'TUIConversationMarkPlugin'  ...end
```

> **Примечание**: `TUIConversationMarkPlugin` зависит от `TUIConversation`. Интеграция только `TUIConversationMarkPlugin` приведет к аномалиям функций, и интерфейс отметки разговора не будет отображаться нормально. `TUIConversationMarkPlugin` поддерживается начиная с версии 7.3. Для использования плагина отметки разговора обновите `TUIConversation` до версии 7.3 или выше.

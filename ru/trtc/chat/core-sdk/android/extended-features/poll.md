# Опрос

### Функция плагина

Начиная с версии 7.1, можно интегрировать плагин голосования `TUIPollPlugin`. Плагин голосования — это библиотека пользовательского интерфейса с закрытым исходным кодом, которая зависит от `TUIChat`.

После интеграции плагина вы сможете инициировать групповое голосование, просматривать результаты статистики голосования и участвовать в голосованиях других пользователей непосредственно в `TUIChat`.

### Интеграция на Android

`TUIPollPlugin` — это закрытый плагин, который нужно интегрировать через gradle. Найдите build.gradle приложения и добавьте зависимость плагина голосования в раздел dependencies.

Классическая версия

```
  dependencies {    ...  # Интегрировать функцию чата.  api project(':tuichat')  ...  # Интегрировать плагин голосования, поддерживаемый начиная с версии 7.1.  api "com.tencent.imsdk:tuipoll-plugin:7.3.4358"  ...}
```

### Интеграция на iOS

`TUIPollPlugin`, как и обычные компоненты TUIKit, можно быстро интегрировать через CocoaPods. Для плагина голосования вам нужно просто добавить одну строку в Podfile для завершения интеграции:

Классическая версия

```
# Раскомментируйте следующую строку для определения глобальной платформы вашего проекта# ...  # Интегрировать функцию чата.  pod 'TUIChat/UI_Classic'   ...  # Интегрировать плагин голосования, поддерживаемый начиная с версии 7.1.  pod 'TUIPollPlugin'  ...end
```

> **Примечание:** `TUIPollPlugin` зависит от `TUIChat`. Интеграция только `TUIPollPlugin` приведет к аномалиям функций и интерфейс голосования не будет отображаться нормально. `TUIPollPlugin` поддерживается начиная с версии 7.1. Для использования плагина голосования обновите `TUIChat` до версии 7.1 или более поздней версии.

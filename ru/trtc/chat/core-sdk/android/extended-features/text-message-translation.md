# Перевод текстовых сообщений

### Функция плагина

После входа в интерфейс чата вы можете вручную долгим нажатием на элемент текстового сообщения в списке сообщений. В появившемся меню нажмите **кнопку перевода** для перевода текста.

> **Примечание:** Функция перевода сообщений предоставляется плагином. Требуется интегрировать `TUITranslationPlugin`, который поддерживается версией 7.2 и выше. Функция перевода сообщений доступна только для клиентов Pro Plus и Enterprise. Она может быть использована после [приобретения Pro Plus и Enterprise](https://console.trtc.io/subscription/buy/chat?packType=pro&language=en); версия Free Trial поддерживает [определённый лимит бесплатного пробного использования](https://trtc.io/document/67651?product=chat&menulabel=core%20sdk&platform=javascript#0182a443-4747-42cb-b5de-ad165eb5a2ff), действительный в течение одного месяца.

- Если вам не требуется функция перевода, не интегрируйте этот плагин. **Кнопка перевода** не будет отображаться при долгом нажатии на текстовое сообщение.
- Если вам требуется функция перевода, интегрируйте `TUIChat` и `TUITranslationPlugin`. Методы интеграции см. в разделе Основы интеграции ([Android](https://www.tencentcloud.com/document/product/1047/50057#)/[iOS](https://www.tencentcloud.com/document/product/1047/50056#)). После интеграции дополнительные настройки не требуются, и **кнопка перевода** будет автоматически отображаться при долгом нажатии на текстовое сообщение.

После интеграции `TUITranslationPlugin` вы также можете установить целевой язык для перевода в интерфейсе **Me**. Целевой язык по умолчанию — это текущий язык, используемый `TUIChat`.

> **Примечание:** `TUITranslationPlugin` зависит от `TUIChat` и не может быть интегрирован отдельно. Перевод поддерживается только для текстовых сообщений, текстовых цитат и текстовых ответных сообщений. Изображения, аудио, видео, файлы, эмодзи и пользовательские сообщения не поддерживают перевод. Не все исходные языки можно переводить на целевые языки. Например, английский можно переводить на хинди, но китайский — нет. См. раздел [Поддерживаемые языки для перевода](https://trtc.io/zh/document/53434) для получения информации о поддерживаемых в настоящее время языках. Если перевод не удаётся, обратитесь к этому документу для переключения исходного или целевого языка.

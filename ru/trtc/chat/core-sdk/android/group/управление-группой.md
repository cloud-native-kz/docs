# Управление группой

## Обзор

Функция управления группой позволяет создавать группу, присоединяться к группе, получать список присоединённых групп, покидать группу или распускать группу.

## Слушатель событий группы

В функции управления группой, описанной ниже, SDK будет автоматически запускать обратный вызов уведомления о событии группы, например, когда кто-то присоединяется или покидает группу.

- Вызовите `addGroupListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#a8dde0362f00a454945e7811c8a726a38) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.addgrouplistener(listener:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#af583b113cdec570b08ae80d682fba52c) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#a05af3d083cef4d667cf972e0cf340289)) для добавления слушателя событий группы.
- События группы определены в `V2TIMGroupListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html)), включая уведомления о членах группы, уведомления о жизненном цикле группы, уведомления о запросах присоединения к группе и обратный вызов слушателя событий тем.
- Чтобы перестать получать события группы, вызовите `removeGroupListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#ab4d57bd5fd4b2d71c29daaeee8b9b760) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.removegrouplistener(listener:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#a2b2093489bf869f70c03be39f4ed08a1) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#a60fe2aac014661aeaf3cbbafaea830e3)) для удаления слушателя событий группы.
- Если вы настроили слушатель событий группы, но не получаете события, как ожидалось, вы можете войти в [Консоль](https://console.trtc.io/) и проверить `конфигурацию системного уведомления группы`. Страница конфигурации выглядит следующим образом:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/8f367433217711efa62352540003d080.png)

## Поиск групп

Вы можете искать только группы, которые были сохранены локально, такие как список присоединённых групп и профили групп, которые были загружены.

Вы можете вызвать API `searchGroups` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#a94a72082b7e2682942f35196a7e28023) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.searchgroups(searchparam:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#ac9a960921e512621340159d82a4b5259) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#a63b463ce1a5952adf8d88bc794b32f22)) для поиска локальных групп.
Вы можете установить ключевое слово поиска `keywordList` и указать область поиска, то есть поиск по полям `userID`, `groupName`.

> **Примечание:** Функция локального поиска групп поддерживается только версией v5.4.666 и более поздними. Локальный поиск групп — это функция повышенного уровня, [приобретите издание Pro, Pro Plus или Enterprise](https://trtc.io/buy/chat) для использования. Список ключевых слов для поиска, поддерживается до 5.

## Создание группы

### Базовый API

Вызовите обычный API `createGroup` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#af836e4912f668dddf6cc679233cfb0bb) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.creategroup(info:memberlist:succ:fail:)) /  [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#a3bbcf819c1ec70e520b7f9a42cfbb989) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#a0325514b94a734186be684eb9bb5cc80)) и укажите параметры для создания группы.

Параметры `createGroup` описаны ниже:

| Параметр | Определение | Обязательный | Описание |
| --- | --- | --- | --- |
| groupType | Тип группы | Да | Дополнительную информацию см. в разделе [Типы групп](https://www.tencentcloud.com/document/product/1047/33529#group-types). |
| groupID | ID группы | Нет | Если оставить пустым, ID будет назначен автоматически после успешного создания группы. Может быть настроен согласно инструкциям в разделе [Пользовательские ID групп](https://www.tencentcloud.com/document/product/1047/33529?lang=en&pg=#custom-group-ids). |
| groupName | Имя группы | Да | Может содержать до 30 байт. |

Если вы вызвали `addGroupListener` для добавления слушателя событий группы, как описано в разделе [Слушатель событий группы](https://www.tencentcloud.com/document/product/1047/48466#group-event-listener), `onGroupCreated` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a93504c985480a2bb87ca716809e3dd00) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.ongroupcreated(groupid:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#aa5c08185c8c9ddaace6421e1e153711d) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#af7cf426687a86f033051ad6bed8fc087)) будет вызван после успешного создания группы.

## Присоединение к группе

| Тип | Способ присоединения к группе |
| --- | --- |
| Рабочая группа (Work) | По приглашению |
| Открытая группа (Public) | По запросу пользователя и с одобрения владельца группы или администратора |
| Группа встреч (Meeting) | Свободное присоединение |
| Сообщество (Community) | Свободное присоединение |
| Аудио-видео группа (AVChatRoom) | Свободное присоединение |

Далее описано, как присоединиться к группам в последовательности от простого к сложному.

> **Примечание:** Вам нужно вызвать `addGroupListener` для добавления слушателя событий группы заранее, как описано в разделе [Слушатель событий группы](https://www.tencentcloud.com/document/product/1047/48466#group-event-listener) для получения следующих событий группы.

#### Свободное присоединение

Группы встреч (Meeting), аудио-видео группы (AVChatRoom) и сообщества в основном используются в сценариях свободного взаимодействия, таких как онлайн-встреча и прямая трансляция. Процесс присоединения к таким группам является самым простым:

1. Пользователь вызывает `joinGroup` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#ad64a09bea508672d6d5a402b3455b564) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.joingroup(groupid:msg:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#a4762156b7a98489eb4715de53028e12a) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#adf3dc4604f30fde1d34dceb1990b38fe)) для присоединения к группе.
2. После успешного присоединения пользователя к группе все члены группы (включая пользователя) получат обратный вызов `onMemberEnter` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a85cbb33a40aaa41781e4835bf802db6d) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.onmemberenter(groupid:memberlist:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#aed6a96e6b304863586a107c45ba33f11) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#a420505f6b58f887dfc95aaa8526913d4)).

> **Примечание:** Рабочая группа по умолчанию не поддерживает свободное присоединение к группе, вы можете изменить поле `groupAddOpt` переменной `V2TIMGroupInfo` путём вызова `setGroupInfo` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1ims

## Выход из группы

Вызовите `quitGroup` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#a6d140dbeb44906de9cb69f69c2ce5919) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.quitgroup(groupid:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#ac2a43b3ada447131df0c5f19e8079be5) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#a43ef277f0eb49d6087d140a09152eced)) для выхода из группы.

Остальные участники группы получат обратный вызов `onMemberLeave` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a2169676423875e4c9c376796245ca8d5) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.onmemberleave(groupid:member:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#a27ca737d915d21810b7f54e8677922e0) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#ace37ae003f37fa8ae1aa3035b6c668c6)).

Вызовите `dismissGroup` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#afd0221c0c842a6dcfa0acc657e50caeb)  / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.dismissgroup(groupid:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#a5bd55cb04867985253949d8cc78f860e) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#abfa30c09968c3b6d07c31d8d5a741502)) для распуска группы, и все участники группы получат обратный вызов `onGroupDismissed` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a6e89728e160e126460a6b8eeddf00ad5) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.ongroupdismissed(groupid:opuser:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#a9063ce1847731024581f2b1ad6b9ed0a) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#a5f27455e6b0a90e9d34f716cd0fffaeb)).

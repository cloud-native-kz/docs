# Управление участниками

## Обзор

Управление участниками группы включает получение списка участников, отключение микрофона участников группы, удаление участников группы, предоставление прав доступа и передачу прав владельца группы.

## Получение списка участников группы

Вызовите API `getGroupMemberList` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#a69fc0831aacaa0585c1855f4c91320be)/ [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.getgroupmemberlist(groupid:filter:nextseq:succ:fail:)) /[Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#a98681b9036e73acbe8f84737b5291326)/[C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#ade696bf03f06de9cdfb534570de35254)) для получения списка участников указанной группы.
Этот список содержит профиль каждого участника группы, такой как ID пользователя (`userID`), карточка имени группы (`nameCard`), аватар (`faceUrl`), имя пользователя (`nickName`), время присоединения к группе (`joinTime`), статус онлайн (`isOnline`) и т. д.

Поскольку аудиовидео группа (AVChatRoom) имеет большое количество участников, функция получения списка всех участников недоступна. Пользователь Edition Pro, Edition Pro Plus или Enterprise edition может вызвать `getGroupMemberList` для получения последних 1000 участников группы. Интервал обновления данных списка участников группы на сервере составляет 40 сек.

Эта функция должна быть включена в [Консоли](https://console.trtc.io/), путь конфигурации: Applications > Your App > Chat > Configuration > Group Configuration > Group feature configuration > List of online audio-video group members.

Детальное описание параметров API `getGroupMemberList` следует далее.

### Параметр filter

Параметр `filter` имеет тип `V2TIMGroupMemberFilter` и означает получение списка определённой роли:

| Filter | Type |
| --- | --- |
| V2TIM_GROUP_MEMBER_FILTER_ALL | Получить список информации всех участников группы |
| V2TIM_GROUP_MEMBER_FILTER_OWNER | Получить список информации владельца группы |
| V2TIM_GROUP_MEMBER_FILTER_ADMIN | Получить список информации администраторов группы |
| V2TIM_GROUP_MEMBER_FILTER_COMMON | Получить список информации обычных участников группы |

Для аудиовидео групп вы можете установить `filter` на пользовательскую метку участника группы, отличную от `V2TIMGroupMemberFilter`, для фильтрации участников группы по необходимости. Дополнительная информация см. в разделе [Маркировка участников группы](https://www.tencentcloud.com/document/product/1047/48181#marking-group-members).

### Параметр nextSeq

Во многих случаях необходимо отображать только первую страницу списка участников группы, а не информацию всех участников группы. Больше участников нужно получить только при нажатии пользователем **Следующей страницы** или при прокрутке страницы списка. В этом случае вы можете использовать получение по страницам. Процедура получения по страницам:

1. При первом вызове `getGroupMemberList` установите `nextSeq` на `0` (это означает получение списка участников группы с начала). За раз можно получить до 500 объектов участников группы.

> **Примечание:** Для версии 7.2 и ниже, обычная группа по умолчанию получает 50 участников группы за раз. Для версии 7.3 и выше, обычная группа по умолчанию получает 500 участников группы за раз.

2. После первого успешного получения списка участников группы, callback `V2TIMGroupMemberInfoResult` функции `getGroupMemberList` будет содержать `nextSeq` (поле для следующего получения):
  - Если `nextSeq` равно `0`, все участники группы были получены.
  - Если `nextSeq` больше `0`, есть ещё участники группы, которых можно получить.
3. Когда пользователь продолжит прокручивать список участников группы, если есть ещё участники для получения, вы можете продолжить вызов API `getGroupMemberList` и передать параметр `nextSeq` (значение из объекта `V2TIMGroupMemberInfoResult`, возвращённого последним получением) для следующего получения.
4. Повторите [шаг 3] до тех пор, пока `nextSeq` не станет равно `0`.

## Маркировка участников группы

Владелец группы может вызвать API `markGroupMemberList` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#abdd5f157904dd50012c7a93f66f85dba) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.markgroupmemberlist(groupid:memberlist:marktype:enablemark:succ:fail:)) /[Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#a21238536e7cb2c3fd086797e7dc1b970)/[C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#abda7d60a02581930b2071cac01d41cfd)) для маркировки или отмены маркировки участников аудиовидео группы. Правила следующие:

1. Эта функция поддерживается только для аудиовидео групп и Community групп, и только владельцы групп имеют право маркировать или отмечать участников группы.
2. На одну аудиовидео группу можно добавить до десяти пользовательских меток.
3. Пользовательские метки, настроенные через Chat SDK, должны быть типа uint32_t и больше или равны 1000.
4. За раз можно маркировать до 100 участников группы.
5. Можно маркировать только онлайн участников. Когда участник покидает группу или переходит в офлайн, его информация о метке будет очищена. Когда участник присоединяется к группе или снова переходит в онлайн, вы можете установить метку для участника через третий вызов обратного вызова.

После маркировки участников группы, владелец группы может вызвать [getGroupMemberList](https://www.tencentcloud.com/document/product/1047/48181#getting-the-group-member-list) чтобы установить `filter` на настроенную пользовательскую метку участника группы для фильтрации участников группы по необходимости.

> **Примечание:** Эта функция доступна только в Enhanced edition v6.6 или позже. Для использования этой функции необходимо [приобрести Pro edition, Pro Plus edition или Enterprise edition](https://trtc.io/buy/chat).

## Отключение микрофона участников группы

### Отключение микрофона определённого участника группы

Владелец группы или администратор могут вызвать `muteGroupMember` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#a450230c4d129611e1b0519827ec0f8b5) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.mutegroupmember(groupid:memberuserid:mutetimeseconds:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#aa8a0206f75d75400b517f7e0d80fe9ee) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#ab19d433e5552205fcba61627e54f7569)) и установить `muteTime` (в секундах) для отключения микрофона участника группы. Если `muteTime` установлено на `120`, участник будет отключен на 120 секунд.
Если участнику группы отключен микрофон, отправка сообщений будет неудачной и будет сообщен код ошибки. Чтобы включить микрофон участника, установите `muteTime` на `0`.

Временная метка отключения микрофона хранится в поле `muteUntil` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupMemberFullInfo.html#a2caecbec07bdd4fa8e6b8072bc39be58) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupMemberFullInfo.html#v2timgroupmemberfullinfo.muteuntil) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMGroupMemberFullInfo.html#a953345fe02297a7192b727abe4b772c6) / [C++](https://im.sdk.qcloud.com/doc/en/structV2TIMGroupMemberFullInfo.html#a953345fe02297a7192b727abe4b772c6)) информации участника группы. Обратите внимание, что `muteUntil` указывает время окончания отключения микрофона участника группы.

После отключения микрофона участника группы, все участники группы (включая участника с отключённым микрофоном) получат callback `onMemberInfoChanged` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a4ac777faad07e32408ae7ef5e2e3fc86) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.onmemberinfochanged(groupid:changeinfolist:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#a18b147301296642e7193e4321e33fd24) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#af87ae2fa909e0e10a521074371359125)).

> **Примечание:** Владелец группы может отключать/включать микрофон администраторам и обычным участникам группы. Администратор может отключать/включать микрофон обычным участникам группы.

### Отключение микрофона всей группе

Владелец группы или администратор также могут вызвать API `muteAllGroupMembers` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#ab3f057b439e976b17dcc0e2e9be5e7d1) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.muteallgroupmembers(groupid:ismute:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#a7749c65c3adca4a48611ab4265407cf6) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#a746446022f8869942903c5e19d1a4c80)) для отключения микрофона всей группе путём установки поля `isMute` на `true`/`YES`. Микрофон всей группе может быть отключен на неограниченный период времени.

Отключение микрофона всем участникам группы вызовет callback `onAllGroupMembersMuted` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#ad982cf2f3538d940f7e9d7b6dc9eb5af)/ [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.onallgroupmembersmuted(groupid:ismute:)) /  [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#a7bdd2e8fb2025ba34f3595ca29137202)/[C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#a2ecb07b1904d6ac4d0e65809367fe492)).

Это уведомление отключено по умолчанию. Чтобы включить его, вы можете войти в [Консоль](https://console.trtc.io/) и изменить соответствующие настройки. Путь конфигурации: Applications > Your App > Chat > Configuration > Group Configuration > Group system notification configuration > Notification of group profile change > Notification of muting all change.

> **Примечание:** Только владелец группы может отключать микрофон администратору.

## Удаление участников группы

SDK предоставляет базовый API удаления и продвинутый API удаления для всех типов групп. Владелец группы или администратор могут вызвать API удаления для удаления определённого участника группы из группы.

### Базовый API

Владелец группы или администратор могут вызвать `kickGroupMember` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupManager.html#a6da6755c6e0c46e96cb02575074a5333)/ [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Group.html#v2timmanager.kickgroupmember(groupid:memberlist:reason:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Group_08.html#a0581f28fddf2ade890aa62e4318d7a97) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupManager.html#ad2e4f74f4e26fb0db455d8e92f774032)) для удаления определённого участника группы из группы. После удаления участника группы, все члены (включая удаленного пользователя) получат callback `onMemberKicked` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMGroupListener.html#a2874b768866c2d255144c128a766c7fe) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMGroupListener.html#v2timgrouplistener.onmemberkicked(groupid:opuser:memberlist:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMGroupListener-p.html#a85e7f32f403d876018a26a1b37607edc) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMGroupListener.html#a08eea992617b046a983ca8757414d8d5)).

За исключением AVChatRoom, удалённый пользователь может повторно подать заявку на присоединение к группе или быть приглашённым присоединиться к группе после использования этого API. См. детали в разделе [Удаление участников группы из AVChatRoom](https://www.tencentcloud.com/document/product/1047/48181#ce057de3-3267-462a-8720-9c7892074335).

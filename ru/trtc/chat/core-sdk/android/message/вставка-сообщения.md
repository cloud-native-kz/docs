# Вставка сообщения

## Описание функции

При вставке сообщения оно будет добавлено только в локальную базу данных, но не будет отправлено на сервер.
Этот API используется для вставки советов в беседу, таких как "Вы покинули группу" и "Защищайте свою информацию. Не отправляйте приватную информацию, такую как учетная запись, пароль и код подтверждения в групповой чат". Такие сообщения должны отображаться в области чата, но не должны отправляться другим пользователям.

> **Внимание** Вставленные сообщения будут потеряны, если учетная запись войдет на другое мобильное устройство или приложение будет удалено и переустановлено.

### Вставка локального сообщения в личные сообщения

Вызовите `insertC2CMessageToLocalStorage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#a5afe4461b4a47205d2865ea94317d4aa) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.insertc2cmessagetolocalstorage(msg:to:sender:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#acc1dccd310d1965248cff0d4fd5ca45f) /  [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#abba2adf81fa2bb457c14fffb9ae0eda4)) для вставки локального сообщения в личные сообщения.

# Отозвать сообщение

## Обзор

Отправитель может отозвать успешно отправленное сообщение.

По умолчанию отправитель может отозвать сообщение, отправленное в течение 2 минут. Вы можете изменить временное ограничение для отзыва сообщения. Подробные инструкции см. в разделе [Параметры отзыва сообщений](https://www.tencentcloud.com/document/product/1047/34419?lang=en&pg=#message-recall-settings).

Отзыв сообщения можно реализовать через код пользовательского интерфейса получателя: когда сообщение отозвано, получатель получит уведомление `onRecvMessageRevoked`, которое содержит `msgID` отозванного сообщения. Вы можете идентифицировать отозванное сообщение на уровне пользовательского интерфейса на основе `msgID` и изменить пузырь сообщения на статус "Сообщение отозвано".

В личных чатах только отправитель может отозвать свои собственные сообщения через этот API.

В групповых чатах не только отправитель может отозвать своё сообщение, но администраторы или владельцы группы также могут отзывать сообщения, отправленные другими участниками группы, через этот API.

Этот API доступен для всех типов групп, включая группы аудио-видео (AVChatRoom) и группы сообщества, в версии 7.4 и позже.

## Отзыв сообщения (отправителем)

Отправитель может вызвать `revokeMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#ad0dfce6be749165cd90a9ff67a1308b1) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.revokemessage(msg:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#a972ac3fb7744458eb0d6abd96ce35126) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#a3f271fcb935ada0ef05709367638a1a6)) для отзыва сообщения.

Пример кода:

class Callback final : public V2TIMCallback {public:    using SuccessCallback = std::function<void()>;    using ErrorCallback = std::function<void(int, const V2TIMString&)>;    Callback() = default;    ~Callback() override = default;    void SetCallback(SuccessCallback success_callback, ErrorCallback error_callback) {        success_callback_ = std::move(success_callback);        error_callback_ = std::move(error_callback);    }    void OnSuccess() override {        if (success_callback_) {            success_callback_();        }    }    void OnError(int error_code, const V2TIMString& error_message) override {        if (error_callback_) {            error_callback_(error_code, error_message);        }    }private:    SuccessCallback success_callback_;    ErrorCallback error_callback_;};auto callback = new Callback;callback->SetCallback(    [=]() {        // Сообщение успешно отозвано        delete callback;    },    [=](int error_code, const V2TIMString& error_message) {        // Не удалось отозвать сообщение        delete callback;    });V2TIMManager::GetInstance()->GetMessageManager()->RevokeMessage(message, callback);

## Получение уведомления об отзыве сообщения (получателем)

1. Получатель вызывает `addAdvancedMsgListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#aaccdec10b9fbee5e43eaf908e359c823) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.addadvancedmsglistener(listener:)) /[Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#acf794752cc6bfa786aea5cd7fabadfab) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#a498688ee0f672f114e28d830761dfbf8)) для установки расширенного прослушивателя сообщений.
2. Получатель получает уведомление об отзыве сообщения в `onRecvMessageRevoked` ([Java](https://im.sdk.qcloud.com/doc/zh-cn/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMAdvancedMsgListener.html#ac7ca5a853276542a27b0a57783478ef6) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMAdvancedMsgListener.html#v2timadvancedmsglistener.onrecvmessagerevoked(msgid:operateuser:reason:)) / [Objective-C](https://im.sdk.qcloud.com/doc/zh-cn/protocolV2TIMAdvancedMsgListener-p.html#afea79b252918b586d0157fba8a8ba00e) / [C++](https://im.sdk.qcloud.com/doc/zh-cn/classV2TIMAdvancedMsgListener.html#a026f0c46fc41b8f6294f7f423d2ce771)).

> **Примечание:** Начиная с версии 7.4, когда сообщение отозвано, получатель может получить причину отзыва и автора отзыва сообщения через API onRecvMessageRevoked.

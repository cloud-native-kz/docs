# Очистка истории сообщений

## Обзор

Можно очищать сообщения в одиночных чатах и групповых чатах.
При очистке сообщений в беседе все сообщения в беседе будут удалены **локально и из облака**, но сама беседа не будет удалена.

> **Примечание**: **Не** используйте этот API, если вы не хотите удалять сообщения из облака. Этот API поддерживается SDK версии v5.4.666 и выше.

После очистки сообщений `lastMessage` в беседе станет пустым, и:

- Если версия вашего SDK старше v5.5.892 и `lastMessage` используется для сортировки, это может повлиять на последовательность в списке бесед.
- Если ваш SDK версии v5.5.892 или выше и `orderKey` используется для сортировки, последовательность в списке бесед не будет затронута.
Дополнительную информацию см. в разделе [Список бесед](https://intl.cloud.tencent.com/document/product/1047/48326).

### Очистка сообщений в одиночных чатах

Вызовите `clearC2CHistoryMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#a29aa6e75c2238c35cc609bef0e5a46ce) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.clearc2chistorymessage(userid:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#a005c7767172d9a3980974b68c780c33b) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#aade0f8d9a53a87473990714f17a297bc)) для очистки сообщений в одиночных чатах.

Пример кода:

Java

Swift

Objective-C

C++

```
V2TIMManager.getMessageManager().clearC2CHistoryMessage(userID, new V2TIMCallback() {    @Override    public void onSuccess() {        // One-to-one messages cleared successfully    }    @Override    public void onError(int code, String desc) {        // Failed to clear one-to-one messages    }});
```

```
V2TIMManager.shared.clearC2CHistoryMessage(userID: "userID") {        print("clearC2CHistoryMessage succ")    } fail: { code, desc in        print("clearC2CHistoryMessage fail, \\(code), \\(desc)")    }
```

```
[V2TIMManager.sharedInstance clearC2CHistoryMessage:userID                                               succ:^{    NSLog(@"One-to-one messages cleared successfully");} fail:^(int code, NSString *desc) {    NSLog(@"Failed to clear one-to-one messages, code: %d, desc: %@", code, desc);}];
```

```
class Callback final : public V2TIMCallback {public:    using SuccessCallback = std::function<void()>;    using ErrorCallback = std::function<void(int, const V2TIMString&)>;    Callback() = default;    ~Callback() override = default;    void SetCallback(SuccessCallback success_callback, ErrorCallback error_callback) {        success_callback_ = std::move(success_callback);        error_callback_ = std::move(error_callback);    }    void OnSuccess() override {        if (success_callback_) {            success_callback_();        }    }    void OnError(int error_code, const V2TIMString& error_message) override {        if (error_callback_) {            error_callback_(error_code, error_message);        }    }private:    SuccessCallback success_callback_;    ErrorCallback error_callback_;};auto callback = new Callback;callback->SetCallback(    [=]() {        // One-to-one messages cleared successfully        delete callback;    },    [=](int error_code, const V2TIMString& error_message) {        // Failed to clear one-to-one messages        delete callback;    });
```

### Очистка групповых сообщений

Вызовите `clearGroupHistoryMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#a6e1a1dce441243d0bd5ac2f8bcecb3d9) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.cleargrouphistorymessage(groupid:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#a16c01c19a285e2bd11443c868c8256e6) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#a1207155633e3deb59616d4deb779d1eb)) для очистки групповых сообщений.

Пример кода:

Java

Swift

Objective-C

C++

```
V2TIMManager.getMessageManager().clearGroupHistoryMessage(GroupID, new V2TIMCallback() {    @Override    public void onSuccess() {        // Group messages cleared successfully    }    @Override    public void onError(int code, String desc) {        // Failed to clear group messages    }});
```

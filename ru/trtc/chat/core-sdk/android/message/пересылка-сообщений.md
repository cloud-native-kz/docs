# Пересылка сообщений

## Описание функции

Вы можете объединять и пересылать сообщения следующим образом:

1. Создайте объединенное сообщение на основе списка исходных сообщений.
2. Отправьте объединенное сообщение получателю.
3. Получатель получает объединенное сообщение и анализирует список исходных сообщений.

Для отображения объединенного сообщения необходимы название и краткое содержание, как показано ниже:

## Объединение и пересылка сообщений

### Создание и отправка объединенного сообщения

Объединенное сообщение можно создать, установив список сообщений вместе с названием и кратким содержанием объединенного сообщения. Процесс выглядит следующим образом:

1. Вызовите API `createMergerMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#acebe275789ab49cc8abe6af5e07aa3b0) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.createmergermessage(messagelist:title:abstractlist:compatibletext:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#a2943bb31403aeb22f8582cd9966cf13e) /Â [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#afc00a2a85b3d29ccfc472ea6544eccf3))Â для создания объединенного сообщения. Необходимо также установить список исходных сообщений, а также название и краткое содержание объединенного сообщения.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dc8c4e2f1f9a11eea27e525400c56988.png)

| Атрибут | Описание | Примечания |
| --- | --- | --- |
| messageList | Список исходных сообщений | Список исходных сообщений, которые должны быть объединены и переданы |
| title | Название | Название объединенного сообщения, например "История чата между xixiyah и Hello", как показано выше |
| abstractList | Список кратких содержаний | Список кратких содержаний объединенного сообщения, как показано выше. Краткие содержания исходных сообщений необходимо отобразить для объединенного сообщения, которое будет развернуто после нажатия пользователем на ячейку. |
| compatibleText | Текст совместимости | Если ранняя версия SDK не поддерживает объединенное сообщение, пользователь по умолчанию получит текстовое сообщение с содержимым `compatibleText`. |

### Получение объединенного сообщения

#### Добавление слушателя

Получатель может вызвать `addAdvancedMsgListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#aaccdec10b9fbee5e43eaf908e359c823) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.addadvancedmsglistener(listener:))  / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#acf794752cc6bfa786aea5cd7fabadfab) /Â [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#a498688ee0f672f114e28d830761dfbf8)) для добавления продвинутого слушателя сообщений.

#### Анализ сообщения

После добавления слушателя получатель получит объединенное сообщение `V2TIMMessage` в `onRecvNewMessage`.
Вы можете использовать элемент объединенного сообщения `V2TIMMergerElem` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMergerElem.html) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMMergerElem.html) /  [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMMergerElem.html) /Â [C++](https://im.sdk.qcloud.com/doc/en/structV2TIMMergerElem.html)) для получения `title` и `abstractList` для отображения в пользовательском интерфейсе.
Затем, когда пользователь нажимает на объединенное сообщение, вы можете вызвать API `downloadMergerMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMergerElem.html#af34d8228a9842875652a726f24ac3d30) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMMergerElem.html#v2timmergerelem.downloadmergermessage(succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMMergerElem.html#ad77abfe27eabf237aee7c951100e6755) /Â [C++](https://im.sdk.qcloud.com/doc/en/structV2TIMMergerElem.html#ae980573283ec250c82c26b8e25a3dd59)) для загрузки списка объединенных сообщений для отображения в пользовательском интерфейсе.

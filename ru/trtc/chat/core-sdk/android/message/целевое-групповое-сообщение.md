# Целевое групповое сообщение

## Описание функции

Целевое групповое сообщение — это сообщение, отправляемое указанным участникам группы, которое не могут получить другие участники группы.

> **Примечание** Целевые групповые сообщения поддерживаются только SDK Enhanced edition версии 6.0.1975 и выше. Для использования этой функции необходимо [приобрести Pro edition, Pro Plus edition или Enterprise edition](https://trtc.io/buy/chat). Исходный объект сообщения для создания целевого группового сообщения не может быть групповым сообщением @. Функция целевых групповых сообщений недоступна для групп сообществ и аудиовидео (AVChatRoom). По умолчанию целевые групповые сообщения исключаются из числа непрочитанных сообщений групповой беседы.

## Отправка целевого группового сообщения

Целевое групповое сообщение — это сообщение, отправляемое указанным участникам группы, которое не могут получить другие участники группы.

- Для отправки целевого группового сообщения одному участнику группы передайте как `groupID`, так и `receiver`.
- Для отправки целевого группового сообщения нескольким участникам группы выполните следующие действия:
  1.1. Вызовите API `createXxxMessage` (здесь `Xxx` указывает на тип сообщения) для создания исходного объекта сообщения `V2TIMMessage`.
  1.2. Вызовите API `createTargetedGroupMessage` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMMessageManager.html#a4def1515746b2840e4b82047a53b91a2) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Message.html#v2timmanager.createtargetedgroupmessage(message:receiverlist:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Message_08.html#a8bddd2f566a53362b4da5448fdd18fbc) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMMessageManager.html#aceeef38fd6308e91154cdd8310c6012f)) для создания объекта целевого сообщения `V2TIMMessage` на основе исходного объекта сообщения и укажите список участников группы, которые будут получать сообщение.
  1.3. Вызовите API `sendMessage` для отправки целевого сообщения.

## Получение целевого группового сообщения

По умолчанию целевые групповые сообщения исключаются из числа непрочитанных сообщений групповой беседы.
Целевое групповое сообщение можно получить так же, как обычное сообщение. Подробные инструкции см. в разделе [Получение сообщения](https://intl.cloud.tencent.com/document/product/1047/47995).

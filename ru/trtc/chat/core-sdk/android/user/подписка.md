# Подписка

## Описание

- **Подписка** — функция, которая позволяет пользователям подписываться на других интересующих их пользователей, чтобы своевременно получать последние новости, ленту или информацию о событиях этих пользователей. Система может предоставлять персонализированные рекомендации контента на основе списка подписок пользователя.
- **Подписчики** — состояние пользователя, который за ним следят другие пользователи. Когда пользователь A подписывается на пользователя B, A становится подписчиком B. Пользователи могут просматривать количество подписчиков, список подписчиков или профиль подписчиков на своей странице профиля.

С помощью этих функций вы можете создать активную, взаимосвязанную сеть пользователей, способствующую распространению информации и формированию сообщества.

> **Примечание:** Эта функция доступна только в улучшенной версии SDK v7.8 и выше. Эта функция поддерживается только в редакциях Pro, Pro-plus и Enterprise. Пожалуйста, [приобретите редакцию Pro, Pro-plus или Enterprise](https://console.trtc.io/subscription/buy/chat?language=en) для использования этой функции.

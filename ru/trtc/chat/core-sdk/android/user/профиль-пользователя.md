# Профиль пользователя

## Описание функции

Пользователи могут запрашивать информацию о себе, друзьях и пользователях, не являющихся друзьями. Они также могут изменять свои никнеймы, фотографии профиля, статусы, а также замечания о друзьях и списки.

## Прослушиватель событий отношений

Вызовите `addFriendListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMFriendshipManager.html#af09d0d2297fe73cc81b8e8941bcd35b2) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Friendship.html#v2timmanager.addfriendlistener(listener:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Friendship_08.html#a1de011b63b3c20b1be519dc7ba124704) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMFriendshipManager.html#ac4c542617008471fa1fe7a64ba963fbb)) для добавления прослушивателя событий отношений.

Чтобы прекратить получение событий отношений, вызовите `removeFriendListener` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMFriendshipManager.html#a6a823459f109bcd8744806f8f1b8bfea) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager+Friendship.html#v2timmanager.removefriendlistener(listener:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/categoryV2TIMManager_07Friendship_08.html#aed40ccbbc4e15be79154077a6d3ec085) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMFriendshipManager.html#ae21ca2737c35305ecc1f25e054265ed8)) для удаления прослушивателя событий отношений.

### Профиль пользователя, не являющегося другом

Вызовите API `getUsersInfo` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#a7ca8c0f71a9875021fc35dfcaff68d1e) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.getusersinfo(useridlist:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#ac638c1dd6ec1e5204637e4b87129cd38) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#ac7aa404aec07fc0d9823d9da5fd4e443)) и введите `UserID` пользователя, не являющегося другом, для параметра `userIDList`, чтобы запросить профиль пользователя, не являющегося другом.

> **Примечание:** Профиль пользователя, не являющегося другом, не может быть изменен. Когда профиль пользователя, не являющегося другом, обновляется, серверная часть не может отправить какое-либо системное уведомление в SDK, потому что отношение дружбы отсутствует, поэтому профиль пользователя, не являющегося другом, **не будет обновляться в реальном времени** в SDK. Чтобы избежать отправки сетевого запроса на сервер каждый раз при получении профиля пользователя и сэкономить сетевые ресурсы, SDK добавляет логику кэширования, устанавливая интервал в 10 минут между активными извлечениями профиля одного и того же пользователя с сервера.

## Подписка на профиль пользователя, не являющегося другом

Вы можете использовать API `subscribeUserInfo` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#a4d14dda5e99d821c84dea4ae25e403af) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.subscribeuserinfo(useridlist:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#ac23fa2c088905352957d21789fc2b9d3) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#a211e335c9ea2188db1596347abec825e)) для подписки на профили пользователей, не являющихся друзьями. После успешной подписки SDK уведомит вас об изменениях в профиле подписанного пользователя через обратный вызов `onUserInfoChanged` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMSDKListener.html#a9d68adb99ba287374cf63e4260357b69) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMSDKListener.html#v2timsdklistener.onuserinfochanged(userinfolist:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/protocolV2TIMSDKListener-p.html#a18279639e8c805cd6b7676fadbd76465) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMSDKListener.html#aa0a34bb257c759b0f9909fd46863f3a6)).

Подписываясь на профили пользователей, не являющихся друзьями, вы можете **оставаться в курсе изменений в профилях конкретных пользователей, не являющихся друзьями, в реальном времени**. **Вам не нужно подписываться** на профили друзей или свой профиль. Изменения этих профилей будут автоматически уведомлены вам через обратный вызов `onFriendInfoChanged` или `onSelfInfoUpdated`.

> **Примечание:** Эта функция доступна только для клиентов Pro edition, Pro Plus edition, Enterprise edition. После [приобретения Pro edition, Pro Plus edition, Enterprise edition](https://trtc.io/buy/chat) и включения функции "User Profile Change Subscription" в [Консоли](https://console.trtc.io/) вы можете использовать эту функцию. Путь переключения: Applications > Your App > Chat > Configuration > Login and Message > User Profile Change Subscription. Эта функция доступна только в SDK enhanced edition v7.4 или позже. Список подписок позволяет максимум 200 подписок. После превышения лимита самый ранний подписанный пользователь будет автоматически исключен (подписка на друзей также займет эти 200 мест). Лимит частоты по умолчанию составляет максимум 20 запросов в течение 5 секунд, а максимальное количество пользователей для одного запроса не должно превышать 100.

## Отписка от профиля пользователя, не являющегося другом

Если вы не хотите получать уведомления об изменениях в профилях подписанных пользователей, вы можете вызвать API `unsubscribeUserInfo` ([Java](https://im.sdk.qcloud.com/doc/en/classcom_1_1tencent_1_1imsdk_1_1v2_1_1V2TIMManager.html#af42e383f564f7d5bb9b4d39a3e00d4d9) / [Swift](https://im.sdk.qcloud.com/doc/en/swift_V2TIMManager.html#v2timmanager.unsubscribeuserinfo(useridlist:succ:fail:)) / [Objective-C](https://im.sdk.qcloud.com/doc/en/interfaceV2TIMManager.html#a4875d417b0ae85ac41ac63b5ac5681a3) / [C++](https://im.sdk.qcloud.com/doc/en/classV2TIMManager.html#aa9888ebe50dc33f007da9bd40ad6c3df)) для отмены подписки для определенных пользователей.

Когда список пользователей в параметрах API пуст, все текущие подписки будут отменены.

> **Примечание:** Эта функция доступна только для клиентов Advanced edition. После [приобретения Pro edition, Pro Plus edition, Enterprise edition](https://trtc.io/buy/chat) и включения функции "User Profile Change Subscription" в [Консоли](https://console.trtc.io/) вы можете использовать эту функцию. Путь переключения: Applications > Your App > Chat > Configuration > Login and Message > User Profile Change Subscription. Эта функция доступна только в SDK enhanced edition v7.4 или позже. Лимит частоты API по умолчанию составляет максимум 20 запросов в течение 5 секунд, а максимальное количество пользователей для одного запроса не должно превышать 100.

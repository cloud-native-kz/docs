# Получение беседы

## Описание функции

Chat SDK предоставляет API для получения бесед, который можно использовать для получения информации объекта `V2TimConversation` одной или нескольких указанных бесед.

### Получение указанной беседы

Вызовите `getConversation` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_conversation_manager/V2TIMConversationManager/getConversation.html)) для получения информации о беседе, которая является объектом `V2TimConversation`.

Пример кода:

```
V2TimValueCallback<V2TimConversation> conv = await conversationManager.getConversation(conversationID: "conversationID");
```

### Получение указанных бесед

Вызовите `getConversationList` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_conversation_manager/V2TIMConversationManager/getConversationList.html)) для получения списка указанных бесед, который содержит объекты `V2TimConversation`.

Пример кода:

```
V2TimValueCallback<V2TimConversationResult> convList = await conversationManager.getConversationList(nextSeq: '', count: 10);
```

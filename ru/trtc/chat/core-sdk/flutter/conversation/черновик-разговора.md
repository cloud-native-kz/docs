# Черновик беседы

## Описание функции

При отправке сообщения пользователь может захотеть переключиться на другое окно чата, прежде чем закончить написание сообщения. Незавершённое сообщение можно сохранить через API `setConversationDraft`, чтобы пользователь мог вернуться к нему позже через поле `draftText` объекта `V2TIMConversation` и завершить его.

> **Внимание:** Черновик может содержать только текст. Черновик сохраняется только в локальной базе данных и не передаётся на сервер. Поэтому он не синхронизируется между устройствами и будет недоступен после удаления и переустановки приложения.

## Установка черновика беседы

Вызовите API `setConversationDraft` ([Подробнее](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_conversation_manager/V2TIMConversationManager/setConversationDraft.html)), чтобы установить черновик беседы.

Если параметр `draftText` этого API пуст, черновик отменяется.

Пример кода:

```
conversationManager.setConversationDraft(conversationID: "conversationID",draftText: "Draft");
```

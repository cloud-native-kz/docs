# Отзыв сообщения

## Описание функции

- Метод отзыва сообщения находится в базовом классе `TencentImSDKPlugin.v2TIMManager.getMessageManager()`.
- Получатель слушает уведомление об отзыве сообщения через `addAdvancedMsgListener`.

## Отзыв сообщения

Отправитель может отозвать успешно отправленное сообщение.

По умолчанию отправитель может отозвать сообщение, отправленное в течение двух минут. Вы можете изменить временное ограничение для отзыва сообщения согласно инструкциям в [Feature Configuration](https://intl.cloud.tencent.com/document/product/1047/34419).

Отзыв сообщения можно реализовать через код пользовательского интерфейса получателя: когда сообщение отозвано, получатель получит уведомление `onRecvMessageRevoked`, содержащее `msgID` отозванного сообщения. Вы можете идентифицировать отозванное сообщение на уровне пользовательского интерфейса на основе `msgID` и изменить всплывающее окно для сообщения на статус "Сообщение отозвано".

### Отзыв сообщения (отправителем)

Отправитель вызывает `revokeMessage` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/revokeMessage.html)) для отзыва сообщения.

Пример кода:

```
 V2TimCallback revokeMessage = await  TencentImSDKPlugin.v2TIMManager.getMessageManager().revokeMessage(msgID: "");
```

### Получение уведомления об отзыве сообщения (получателем)

1. Вызовите `addAdvancedMsgListener` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/addAdvancedMsgListener.html)) для установки продвинутого слушателя сообщений.
2. Получайте уведомление об отзыве сообщения через `onRecvMessageRevoked` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/removeAdvancedMsgListener.html)).

Пример кода:

```
onRecvMessageRevoked: (String messageid) {      // Process the recalled message among locally maintained messages},
```

# Отключение уведомлений

## Описание функции

Вы можете установить параметр получения сообщений для **личного чата или группового чата**, чтобы реализовать функцию отключения уведомлений.

Chat SDK поддерживает следующие три параметра получения сообщений, определённые в `V2TIMReceiveMessageOpt`:

| Параметр получения сообщений | Описание функции |
| --- | --- |
| ReceiveMsgOptEnum.V2TIM_RECEIVE_MESSAGE | Сообщения будут получены, когда пользователь онлайн, и будут получены уведомления офлайн-доставки, когда пользователь офлайн. |
| ReceiveMsgOptEnum.V2TIM_NOT_RECEIVE_MESSAGE | Сообщения не будут получены независимо от того, находится ли пользователь онлайн или офлайн. |
| ReceiveMsgOptEnum.V2TIM_RECEIVE_NOT_NOTIFY_MESSAGE | Сообщения будут получены, когда пользователь онлайн, и уведомления офлайн-доставки не будут получены, когда пользователь офлайн. |

Различные параметры `V2TIMReceiveMessageOpt` можно использовать для реализации отключения уведомлений о групповых сообщениях:
**Сообщения не будут получены.**
После установки параметра получения сообщений на `V2TIM_NOT_RECEIVE_MESSAGE` личные сообщения и групповые сообщения не будут получены, и список разговоров не будет обновляться.

**Сообщения будут получены, но не будут уведомлены пользователю, и на интерфейсе списка разговоров будет отображена иконка без количества непрочитанных сообщений.**

1. Параметр получения сообщений установлен на `V2TIM_RECEIVE_NOT_NOTIFY_MESSAGE`.
2. Когда получатель получает личное сообщение или групповое сообщение и требуется обновить список разговоров, он может получить количество непрочитанных сообщений через `unreadCount` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/models_v2_tim_conversation/V2TimConversation/unreadCount.html)) в `V2TIMConversation` разговора.
3. Получатель отображает иконку вместо количества непрочитанных сообщений, когда параметр получения сообщений определён как `V2TIM_RECEIVE_NOT_NOTIFY_MESSAGE` на основе `recvOpt` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/models_v2_tim_conversation/V2TimConversation/recvOpt.html)) в `V2TIMConversation`.

> **Примечание:** Поскольку этот метод требует функции подсчёта непрочитанных сообщений, он применяется только к рабочим группам (Work) и открытым группам (Public). Дополнительные сведения о типах групп см. в разделе Обзор групп.

## Установка параметра получения сообщений для личного чата

Вызовите API `setC2CReceiveMessageOpt` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/setC2CReceiveMessageOpt.html)) для установки параметра получения сообщений для личного чата.

Вы можете использовать параметр `userIDList` для указания до 30 пользователей одновременно.

> **Внимание:** Этот API можно вызывать максимум 5 раз в секунду.

Пример кода:

```
// Set not to receive messages no matter whether the user is online or offlineTencentImSDKPlugin.v2TIMManager.getMessageManager().setC2CReceiveMessageOpt(userIDList: ['user1','user2'], opt: ReceiveMsgOptEnum.V2TIM_NOT_RECEIVE_MESSAGE);
```

## Получение параметра получения сообщений для личного чата

Вызовите API `getC2CReceiveMessageOpt` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/getC2CReceiveMessageOpt.html)) для получения параметра получения сообщений для личного чата.

Пример кода:

```
V2TimValueCallback<List<V2TimReceiveMessageOptInfo>> messageOpt = await TencentImSDKPlugin.v2TIMManager.getMessageManager().getC2CReceiveMessageOpt(userIDList: ['user1','user2']);  messageOpt.data.forEach((element) {    // Message receiving option    element.c2CReceiveMessageOpt;    element.userID;  });
```

## Установка параметра получения сообщений для группового чата

Вызовите API `setGroupReceiveMessageOpt` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/setGroupReceiveMessageOpt.html)) для установки параметра получения сообщений для группового чата.

Пример кода:

```
TencentImSDKPlugin.v2TIMManager.getMessageManager().setGroupReceiveMessageOpt(groupID: "groupID", opt: ReceiveMsgOptEnum.V2TIM_NOT_RECEIVE_MESSAGE);
```

## Получение параметра получения сообщений для группового чата

Вызовите API `getGroupsInfo` ([Подробности](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_group_manager/V2TIMGroupManager/getGroupsInfo.html)) для получения объекта списка профилей группы `V2TIMGroupInfo`. Здесь поле `recvOpt` указывает параметр получения сообщений для группового чата.

Пример кода:

```
V2TimValueCallback<List<V2TimGroupInfoResult>> groups = await TencentImSDKPlugin.v2TIMManager.getGroupManager().getGroupsInfo(groupIDList: ['groupID']);  groups.data.forEach((element) {     // Get the message receiving option of the group    element.groupInfo.recvOpt;  });
```

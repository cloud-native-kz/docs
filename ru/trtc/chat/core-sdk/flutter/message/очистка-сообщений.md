# Очистка сообщений

## Описание функции

Можно очищать сообщения в личных переписках и групповых чатах.
При очистке сообщений в переписке все сообщения в этой переписке будут удалены **как локально, так и из облака**, однако сама переписка не будет удалена.

> **Примечание:** **Не** используйте этот API, если вы не хотите удалять сообщения из облака.

Если последнее сообщение удалено, `lastMessage` в переписке станет предпоследним сообщением.

### Очистка личных сообщений

Вызовите `clearC2CHistoryMessage` ([Подробнее](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/clearC2CHistoryMessage.html)) для очистки личных сообщений.

Пример кода:

```
TencentImSDKPlugin.v2TIMManager.getMessageManager().clearC2CHistoryMessage(userID: "userid");
```

### Очистка групповых сообщений

Вызовите `clearGroupHistoryMessage` ([Подробнее](https://pub.dev/documentation/tencent_cloud_chat_sdk/latest/manager_v2_tim_message_manager/V2TIMMessageManager/clearGroupHistoryMessage.html)) для очистки групповых сообщений.

Пример кода:

```
 TencentImSDKPlugin.v2TIMManager.getMessageManager().clearGroupHistoryMessage(groupID: "");
```

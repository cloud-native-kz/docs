# Поиск членов облачной группы

## Описание функции

Поиск членов облачной группы с поддержкой нечеткого поиска по ключевым словам. Пользователи могут быстро найти членов группы, соответствующих ключевым словам в группах, к которым они присоединились, что облегчает поиск конкретных членов или управление группой.

> **Примечания:** Функция поиска облачных сообщений доступна только для клиентов **Pro Plus и Enterprise** и может использоваться после [приобретения Pro Plus и Enterprise](https://console.trtc.io/subscription/buy/chat?packType=pro&language=en); версия Free Trial поддерживает [определенный лимит бесплатного использования](https://www.tencentcloud.com/document/product/1047/67651#d1113f0d-47e8-4211-82c0-00d2efb72586), действительный в течение одного месяца. Функция поиска членов облачной группы поддерживается начиная с версии **v3.5.0**. Область поиска: выполняется нечеткий поиск по ключевым словам в никнеймах членов группы nick и названиях карточек группы nameCard во всех группах, в которых находится текущий авторизованный пользователь; точный поиск по userID всех членов группы. Результаты поиска включают только частичную информацию о группе (идентификатор группы groupID, имя группы name, тип группы type, аватар группы avatar) и частичную информацию о членах группы (userID члена группы, никнейм члена группы nick, карточка группы nameCard), соответствующие требованиям отрисовки списка результатов поиска членов группы. Для получения подробной информации о членах группы обратитесь к: [getGroupMemberProfile](https://www.tencentcloud.com/document/product/1047/48177#getting-the-profile-of-group-members). Максимальное количество результатов поиска за один раз — 100. Прямые трансляции (TencentCloudChat.TYPES.GRP_AVCHATROOM) не поддерживаются для поиска. Этот API имеет ограничение облачной частоты 2 раза/сек и ограничение локального вызова API 20 раз/5 сек.

## Поиск членов облачной группы

##### API

```
chat.searchCloudGroupMembers(options);
```

##### Параметры

Параметр options имеет тип Object и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| keywordList | Array | `обязательный` Список ключевых слов, поддерживает до 5 ключевых слов. |
| keywordListMatchType | String \| undefined | Тип сопоставления списка ключевых слов: or: поиск с отношением "или" (по умолчанию) and: поиск с отношением "и" |
| groupTypeList | Array \| undefined | Если не указано, по умолчанию выполняется поиск во всех типах групп (прямые трансляции `TencentCloudChat.TYPES.GRP_AVCHATROOM` не поддерживаются для поиска). Указание типов групп: `TencentCloudChat.TYPES.GRP_WORK` рабочая группа друзей `TencentCloudChat.TYPES.GRP_PUBLIC` социальная группа незнакомцев `TencentCloudChat.TYPES.GRP_MEETING` временная группа встреч `TencentCloudChat.TYPES.GRP_COMMUNITY` сообщество |
| groupIDList | Array \| undefined | Поиск в указанном списке идентификаторов групп, если не указано, по умолчанию выполняется поиск во всех группах. |
| cursor | String \| undefined | Начальная позиция для каждого поиска членов облачной группы. Не указывайте `cursor` для первого поиска, используйте значение `cursor`, возвращенное предыдущим вызовом API `searchCloudGroupMembers`, для продолжения поиска. |
| count | Number \| undefined | Количество результатов для каждого поиска членов облачной группы, по умолчанию 20, максимум 100. |

##### **Возвращаемое значение**

`Promise`

| Имя | Тип | Описание |
| --- | --- | --- |
| totalCount | Number | Общее количество членов группы, соответствующих критериям поиска. |
| searchResultList | Array | Список членов группы, соответствующих критериям поиска, возвращенный сгруппированным по `groupID`. |
| cursor | String | Курсор, необходимый для вызова API поиска для продолжения выборки. |

Здесь `searchResultList` — это список, содержащий объекты результатов поиска. Параметры описаны ниже:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupInfo | Object | Информация о группе, включая следующие детали группы: `groupID` идентификатор группы. `name` имя группы. `type` тип группы. `avatar` аватар группы. |
| memberList | Array | Список членов группы, соответствующих критериям поиска в текущей группе. Каждый член группы в списке включает следующую информацию: `userID` идентификатор пользователя члена группы. `nick` никнейм члена группы. `nameCard` карточка группы. |

##### **Пример**

```
// Поиск указанного списка ключевых слов и списка идентификаторов групп
// - Поиск членов группы в текущих присоединенных группах, содержащих 'test' или 'user', и идентификатор группы — 'group1' или 'group2'
let promise = chat.searchCloudGroupMembers({   keywordList: ['test', 'user'],   groupIDList: ['group1', 'group2'],}); promise.then(function(imResponse) { // Поиск членов группы успешно выполнен   const { totalCount, cursor, searchResultList } = imResponse.data;   console.log(totalCount); // Общее количество членов группы, соответствующих критериям поиска   console.log(cursor); // Начальная позиция для следующего облачного поиска, если отсутствует, указывает, что результаты поиска полностью получены   console.log(searchResultList); // Список членов группы, соответствующих критериям поиска, возвращенные с разбиением на страницы сгруппированные результаты, максимум возврата на странице 100, на основе входного count   for (let i = 0; i < searchResultList.length; i++) {      const searchResultItem = searchResultList[i];      const { groupInfo, memberList } = searchResultItem;      const { groupID, name, type, avatar } = groupInfo;      // Идентификатор группы / Имя группы / Тип группы / Аватар группы      console.log(groupID, name, type, avatar);      console.log(memberList); // Список членов, соответствующих критериям в группе      for (let i = 0; i < memberList.length; i++) {        const memberItem = memberList[i];        const { userID, nick, nameCard } = memberItem;        console.log(userID); // Идентификатор пользователя члена группы        console.log(nick); // Никнейм члена группы        console.log(nameCard); // Карточка группы      }    }}).catch(function(imError) {   console.error(imError); // Поиск членов группы не выполнен});
```

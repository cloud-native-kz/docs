# Список разговоров

## Обзор функции

- С помощью списков разговоров пользователи могут легко найти нужные разговоры после входа в приложение.
- Функция списка разговоров включает получение списка разговоров и прослушивание событий обновления списка разговоров.
- Основная структура данных — это `Conversation`.

## Получение списка разговоров

Вызовите API `getConversationList` на стороне доступа, чтобы получить список разговоров.

##### **API**

```
chat.getConversationList(options);
```

##### Параметры

| Название | Тип | Описание |
| --- | --- | --- |
| options | undefined \| Array \| Object | Если `options` имеет значение `undefined`, SDK вернёт все разговоры. Если `options` имеет тип `Array`, он не должен быть пустым, SDK вернёт указанные разговоры. Если `options` имеет тип `Object`, например `{type, markType, groupName}`, SDK вернёт отфильтрованные разговоры. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```

```

Прослушивайте события `TencentCloudChat.EVENT.CONVERSATION_LIST_UPDATED` на стороне доступа для получения уведомлений об обновлении списка разговоров.

# Счетчик группы

## Обзор функции

Chat SDK предоставляет функцию счетчика группы, позволяющую установить определенное количество счетчиков для каждой группы.

В отличие от пользовательских атрибутов группы, счетчики группы в основном используются для хранения данных целочисленного типа. Вы можете использовать счетчики группы для хранения дополнительной информации, связанной с группой, такой как совокупное количество просмотров, количество просмотров, количество лайков, полученных хостом, и общее количество подарков, отправленных аудиторией хосту в прямой трансляции группы.

> **Примечание:** Счетчики группы поддерживаются всеми типами групп, за исключением групп сообществ с включенными темами. Эта функция поддерживается только в [издании Premium](https://trtc.io/buy/chat).

Учитывайте следующее для счетчиков группы:

1. Одна группа может поддерживать до 20 счетчиков группы, что означает, что количество ключей в одной группе не должно превышать 20.
2. Ключ для одного счетчика группы не может превышать 128 символов, а значение должно быть целым числом (до 64-битных целых чисел со знаком).
3. API `setGroupCounters`, `increaseGroupCounter` и `decreaseGroupCounter` объединяют вычисления с ограничением SDK в 20 вызовов за 5 секунд на вход в систему пользователя. При превышении интерфейс возвращает код ошибки 2996.
4. Интерфейс `getGroupCounters` вычисляется независимо. SDK ограничивает максимум 20 вызовов отдельного пользователя за 5 секунд при входе в систему. При превышении лимита интерфейс возвращает код ошибки 2996.

## Установка счетчиков группы

После успешной установки счетчика срабатывает событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Если ключ счетчика, который вы собираетесь установить, существует, то значение счетчика будет обновлено напрямую; если нет, то будет добавлена новая пара ключ-значение. Если несколько пользователей устанавливают один и тот же счетчик одновременно, будет использовано окончательное значение счетчика. Рекомендуется, чтобы операции установки счетчика выполнял владелец группы.

##### **API**

```
chat.setGroupCounters(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| counters | Object | Пара ключ-значение счетчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Установить значения счетчиков key1 и key2 в 0let promise = chat.setGroupCounters({  groupID: 'group1',  counters: { key1: 0, key2: 0 }});promise.then(function(imResponse) { // Успешно установлено  console.log(imResponse.data.counters); // Счетчики группы успешно установлены}).catch(function(imError) { // Ошибка при установке  console.warn('setGroupCounters error:', imError); // Информация об ошибке});
```

## Увеличение значения счетчика группы

После успешной установки счетчика срабатывает событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Значение в параметрах интерфейса — это размер изменения; после вызова интерфейса переданный размер изменения будет добавлен к текущему значению. Если ключ счетчика, который вы собираетесь установить, уже существует, система напрямую увеличит текущее значение на основе переданного значения; в противном случае она добавит ключ и увеличит на основе переданного значения, начиная со значения по умолчанию 0.

##### **API**

```
chat.increaseGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| key | String | Ключ счетчика группы |
| value | Number | Размер изменения счетчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Предположим, текущее значение счетчика key1 равно 8. После вызова интерфейса increaseGroupCounter и передачи // размера изменения 2 окончательное значение key1 становится 10.let promise = chat.increaseGroupCounter({  groupID: 'group1',  key: 'key1',  value: 2,});promise.then(function(imResponse) { // Успешное увеличение  console.log(imResponse.data);  const { groupID, key, value } = imResponse.data;}).catch(function(imError) { // Ошибка при увеличении  console.warn('increaseGroupCounter error:', imError);});
```

## Уменьшение значения счетчика группы

После успешной установки счетчика срабатывает событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Значение в параметрах интерфейса — это размер изменения; после вызова интерфейса переданный размер изменения будет вычтен из текущего значения. Если ключ счетчика, который вы собираетесь установить, уже существует, система напрямую уменьшит текущее значение на основе переданного значения; в противном случае она добавит ключ и уменьшит на основе переданного значения, начиная со значения по умолчанию 0.

##### **API**

```
chat.decreaseGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| key | String | Ключ счетчика группы |
| value | Number | Размер изменения счетчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Предположим, текущее значение счетчика key1 равно 8.// После вызова API decreaseGroupCounter с размером уменьшения 2// Окончательное значение key1 становится 6.let promise = chat.decreaseGroupCounter({  groupID: 'group1',  key: 'key1',  value: 2});promise.then(function(imResponse) { // Успешное уменьшение  console.log(imResponse.data);  const { groupID, key, value } = imResponse.data;}).catch(function(imError) { // Ошибка при установке  console.warn('decreaseGroupCounter error:', imError);});
```

## Получение счетчиков группы

> **Примечание:** Если keyList не передан, возвращаются все счетчики группы.

##### **API**

```
chat.getGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| keyList | Array \| undefined | Список ключей счетчиков группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Получить значения счетчиков группы key1 и key2let promise = chat.getGroupCounters({  groupID: 'group1',  keyList: ['key1', 'key2']});promise.then(function(imResponse) { // Успешно получено  console.log(imResponse.data.counters);}).catch(function(imError) {  console.warn('getGroupCounters error:', imError); // Информация об ошибке});
```

```
// Получить все счетчики группыlet promise = chat.getGroupCounters({  groupID: 'group1'});promise.then(function(imResponse) { // Успешно получено  console.log(imResponse.data.counters);}).catch(function(imError) {  console.warn('getGroupCounters error:', imError); // Информация об ошибке});
```

## Уведомление об изменении счетчика группы

Когда вы вызываете интерфейсы `setGroupCounters`, `increaseGroupCounter` или `decreaseGroupCounter` для изменения счетчиков группы, срабатывает событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED` и возвращается обновленное значение.

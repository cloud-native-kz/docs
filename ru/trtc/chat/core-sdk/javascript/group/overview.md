# Обзор

## Обзор групп

Chat SDK новой версии поставляется с обновленными типами групп, включая рабочие группы (Work), открытые группы (Public), группы встреч (Meeting), сообщества и аудиовидео-группы (AVChatRoom).

Группы в более ранней версии SDK и новой версии SDK сравниваются ниже:

| Предыдущие типы групп | Новые типы групп | Функции группы |
| --- | --- | --- |
| Public | Public (Открытая группа социального общения незнакомцев) | Позволяет владельцу группы назначать администраторов группы. Для присоединения к группе пользователи должны найти ID группы и запросить одобрение у владельца группы или администратора. |
| Private | Work (Рабочая группа друзей) | Рабочая группа позволяет пользователям присоединиться после приглашения друга, который является членом группы, без необходимости принятия пользователем или одобрения владельцем группы. |
| ChatRoom | Meeting (Временная группа встреч) | Группа встреч позволяет пользователям свободно присоединяться и выходить, а также просматривать историю сообщений до того момента, когда они присоединились к группе. Этот тип группы подходит для сценариев использования Tencent Real-Time Communication (TRTC), например аудио и видеоконференции и онлайн-образование. |
| AVChatRoom | AVChatRoom (Аудиовидео-группа) | Аудиовидео-группа позволяет пользователям свободно присоединяться и выходить. Поддерживает неограниченное количество членов группы и не сохраняет историю сообщений. Этот тип группы можно использовать с продуктами прямого вещания для поддержки сценариев чата с комментариями на экране.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/03a0a259cf0411ef8c8a525400454e06.png) |
| - | Community (Группа сообщества) | Сообщество позволяет пользователям свободно присоединяться и выходить. Это мощный новый инструмент для развлекательного сотрудничества и подходит для сценариев чата с очень большим количеством членов сообщества, таких как поиск единомышленников, социальные сети в играх, маркетинг фанатов и управление организацией. В рамках одного сообщества большое количество членов можно разделить на разные группы и темы для разделения сообщений в целях иерархического общения, но они также могут использовать один и тот же набор отношений дружбы. ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/330a1461cf0411efa4a3525400bf7822.png) |

> **Примечание:** Функция Community поддерживается только Enhanced версией SDK 5.8.1668 и выше, а также Web SDK 2.17.0 и выше. Функция Community требует [приобретения Premium Edition](https://trtc.io/buy/chat), и для использования необходимо активировать переключатель в [Console](https://console.trtc.io/). Путь переключателя: Applications > Your App > Chat > Configuration > Group Configuration > Group feature configuration > Community. Для Friends Work Group (Work) и Stranger Social Group (Public) история сообщений до присоединения к группе по умолчанию не загружается. Для включения этой функции необходимо активировать переключатель в [Console](https://console.trtc.io/). Путь переключателя: Applications > Your App > Chat > Configuration > Group Configuration > Group message configuration > Pull message history before group join.

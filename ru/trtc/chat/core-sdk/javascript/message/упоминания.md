# Упоминания

## Обзор функции

Отправитель прослушивает символы в поле ввода. Когда пользователь вводит @, появляется UI выбора участников группы. После выбора целевых участников группы сообщение будет отображаться в поле ввода в формате `"@A @B @C......"`, что можно дополнительно отредактировать перед отправкой.
В списке групповых чатов UI разговора получателя будет отображаться идентификатор `"someone@me"` или `"@all"`, чтобы напомнить пользователю, что его упомянул кто-то в групповом чате.

> **Примечание** В настоящее время поддерживаются только текстовые @ сообщения.

## Отправка группового сообщения с упоминанием @

1. Отправитель прослушивает поле ввода текста в UI чата и запускает UI выбора участников группы. После выбора участников группы вызывается информация об ID и никнеймах участников.
2. Отправитель вызывает API `createTextAtMessage` для создания группового сообщения @.
3. Отправитель вызывает `sendMessage` для отправки созданного объекта сообщения @.

##### API

```
chat.createTextAtMessage(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object`. Он содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| to | String | `groupID` получателя сообщения |
| conversationType | String | Тип разговора, который должен быть `TencentCloudChat.TYPES.CONV_GROUP` |
| payload | Object | Контейнер содержимого сообщения |
| cloudCustomData | String | Пользовательские данные сообщения, которые сохраняются в облаке, отправляются получателю и могут по-прежнему быть извлечены после удаления и переустановки приложения. |
| priority | String | Приоритет сообщения. Если сообщения в группе превышают ограничение частоты, сервер будет доставлять сообщения с высоким приоритетом в первую очередь. Поддерживаемые значения перечисления: `TencentCloudChat.TYPES.MSG_PRIORITY_HIGH` `TencentCloudChat.TYPES.MSG_PRIORITY_NORMAL (по умолчанию)` `TencentCloudChat.TYPES.MSG_PRIORITY_LOW` `TencentCloudChat.TYPES.MSG_PRIORITY_LOWEST` |

Параметр `payload` описан ниже:

| Имя | Тип | Описание |
| --- | --- | --- |
| text | String | Текстовое содержимое сообщения |
| atUserList | Array.<String> | Список пользователей, которых нужно упомянуть @. Если вам нужно упомянуть @ALL, передайте `TencentCloudChat.TYPES.MSG_AT_ALL`. Например, если это текстовое сообщение хочет упомянуть denny и lucy, а также всех, `atUserList` будет передан как `['denny', 'lucy', TencentCloudChat.TYPES.MSG_AT_ALL]`. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let message = chat.createTextAtMessage({  to: 'group1',  conversationType: TencentCloudChat.TYPES.CONV_GROUP,  payload: {    text: '@denny @lucy Dinner gathering tonight, reply with 1 if you are coming.',    atUserList: ['denny', 'lucy', TencentCloudChat.TYPES.MSG_AT_ALL] // 'denny' 'lucy' are both userID  },  // cloudCustomData: 'your cloud custom data'});let promise = chat.sendMessage(message);promise.then(function(imResponse) {  console.log(imResponse);}).catch(function(imError) {  console.warn('sendMessage error:', imError);});
```

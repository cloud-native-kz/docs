# Целевое сообщение группы

## Обзор функции

Целевое сообщение группы — это сообщение, отправленное указанным членам группы, которое не могут получить другие члены группы.

> **Примечание:** Для использования этой функции необходимо приобрести выпуск Pro, выпуск Pro Plus или Enterprise согласно инструкциям в [Pricing](https://trtc.io/pricing/chat). При создании группового @ сообщения нельзя указать список членов группы для получения сообщения (`receiverList`). Функция целевого сообщения группы недоступна для групп сообщества и групп аудио-видео (AVChatRoom). По умолчанию целевые сообщения группы исключаются из счетчика непрочитанных сообщений группового разговора.

## Отправка целевого сообщения группы

Для отправки целевого сообщения группы указанным членам группы выполните следующие действия:

- Вызовите API `createXxxMessage` (здесь `Xxx` обозначает тип сообщения) для создания сообщения (укажите получателей сообщения через `receiverList`).
- Вызовите API `sendMessage` для отправки сообщения.

##### Примеры

```
// Create a targeted group messagelet message = chat.createTextMessage({  to: 'test',  conversationType: TencentCloudChat.TYPES.CONV_GROUP,  payload: {    text: 'Hello world!'  },  receiverList: ['user0', 'user1']});// Send the messagelet promise = chat.sendMessage(message);promise.then(function(imResponse) {  // Message sent successfully  console.log(imResponse);}).catch(function(imError){  // The message failed to be sent  console.warn('sendMessage error:', imError);});
```

## Получение целевого сообщения группы

По умолчанию целевые сообщения группы исключаются из счетчика непрочитанных сообщений группового разговора.
Целевое сообщение группы можно получить так же, как обычное сообщение.

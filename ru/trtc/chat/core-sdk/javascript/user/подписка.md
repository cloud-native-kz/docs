# Подписки

## Обзор функции

- **Подписка** — функция, которая позволяет пользователям подписываться на других пользователей, чтобы своевременно получать последние новости, ленту или информацию о событиях этих пользователей. Система может предоставлять персонализированные рекомендации контента на основе списка подписок пользователя.
- **Подписчики** — состояние, при котором на пользователя подписаны другие пользователи. Когда пользователь A подписывается на пользователя B, A становится подписчиком B. Пользователи могут просматривать количество подписчиков, список подписчиков или профиль подписчиков на своей странице профиля.

Благодаря этим функциям можно создать активную, взаимосвязанную сеть пользователей, способствующую распространению информации и развитию сообщества.

> **Примечание:** Эта функция поддерживается только в выпусках Pro, Pro-plus и Enterprise. Пожалуйста, [приобретите Pro, Pro-plus или Enterprise](https://console.trtc.io/subscription/buy/chat?language=en) для использования.

## Подписка на пользователей

> **Примечание:** Этот интерфейс поддерживает подписку на до 20 пользователей одновременно (кроме себя, можно подписаться на любого другого пользователя). Каждый пользователь может иметь до 5000 подписчиков, но количество подписок не ограничено.

**API**

```
chat.followUser(options);
```

##### **Параметры**

| Название | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список ID пользователей, поддерживающий подписку на до 20 пользователей одновременно. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.followUser(['user1', 'user2']);
promise.then(function(imResponse) {
  console.log(imResponse.data);
}).catch(function(imError) {
  console.warn('followUser error:', imError);
});
```

## Получение списка моих подписчиков

> **Примечание:** Этот API возвращает до 500 пользователей за раз.

**API**

```
chat.getMyFollowersList(nextCursor);
```

##### **Параметры**

| Название | Тип | Описание |
| --- | --- | --- |
| nextCursor | String \| undefined | Начальная позиция для постраничной выборки. По умолчанию пусто для первой выборки и может быть опущено. При успешной выборке nextCursor не равен `''`. Если требуется постраничная выборка, вы можете передать это значение снова, пока nextCursor не вернёт `''`. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.getMyFollowersList(nextCursor);
promise.then(function(imResponse) {
  const { resultList, nextCursor = '' } = imResponse.data;
  // ressultList - список моих подписчиков
  // nextCursor - идентификатор для продолжения постраничной выборки
  if (nextCursor != '') {
    // выполнить постраничную выборку
  }
}).catch(function(imError) {
  console.warn('getMyFollowersList error:', imError);
});
```

## Получение количества подписок/подписчиков/взаимных подписчиков

**API**

```
chat.getUserFollowInfo(userIDList);
```

##### **Параметры**

| Название | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список ID пользователей; если userIDList не предоставлен, это означает получение информации о подписках для себя. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// получить количество моих подписок/подписчиков/взаимных подписчиков
let promise = chat.getUserFollowInfo();
promise.then(function(imResponse) {
  console.log(imResponse.data);
}).catch(function(imError) {
  console.warn('getUserFollowInfo error:', imError);
});
```

```
// получить количество подписок/подписчиков/взаимных подписчиков для указанных пользователей
let promise = chat.getUserFollowInfo(['user1', 'user2']);
promise.then(function(imResponse) {
  console.log(imResponse.data);
}).catch(function(imError) {
  console.warn('getUserFollowInfo error:', imError);
});
```

## Проверка типа подписки

**API**

```
chat.checkFollowType(userIDList);
```

##### **Параметры**

| Название | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список ID пользователей для проверки; один запрос поддерживает до 100 ID пользователей. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.checkFollowType(['user1', 'user2']);promise.then(function(imResponse) {  console.log(imResponse.data);  imResponse.data.forEach((item) => {    // item.userID - userID    // item.followType - 0 - нет отношений, 1 - мой подписчик, 2 - я подписан, 3 - взаимные подписчики  });}).catch(function(imError) {  console.warn('checkFollowType error:', imError);});
```

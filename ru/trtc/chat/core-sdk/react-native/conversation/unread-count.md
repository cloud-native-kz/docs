# Количество непрочитанных сообщений

## Обзор функции

Список разговоров пользователя обычно содержит несколько разговоров. Если в одном из разговоров появляется новое сообщение, в ячейке списка необходимо отобразить значок с количеством непрочитанных сообщений.

После того как пользователь входит в разговор и возвращается в список разговоров, количество непрочитанных сообщений очищается и значок исчезает. В некоторых приложениях рассчитывается и отображается общее количество непрочитанных сообщений всех разговоров в нижней вкладке списка разговоров.

## Получение общего количества непрочитанных сообщений во всех разговорах

> **Примечание:** Применимо только к рабочим группам, публичным группам, сообществам и конференциям. AVChatRoom в настоящее время не поддерживаются. Общее количество непрочитанных сообщений будет вычитать количество непрочитанных сообщений из разговоров, установленных в режим "Не беспокоить". Общее количество непрочитанных сообщений не включает количество непрочитанных системных разговоров. Поскольку этот интерфейс зависит от синхронизации списка разговоров из облака SDK, общее количество непрочитанных сообщений, полученное сразу после успешного входа, может быть 0. Сторона доступа может обновить локальное общее количество непрочитанных сообщений в сочетании с событием `TencentCloud.EVENT.TOTAL_UNREAD_MESSAGE_COUNT_UPDATED`.

##### **API**

```
chat.getTotalUnreadMessageCount();
```

##### **Возвращаемое значение**

`Number`

##### **Примеры**

```
// Общее количество непрочитанных сообщений в диалогах один-на-один и групповых разговорахlet totalUnreadCount = chat.getTotalUnreadMessageCount();
```

## Уведомление об изменении общего количества непрочитанных сообщений во всех разговорах

##### **Примеры**

```
let onTotalUnreadMessageCountUpdated = function(event) {  console.log(event.data);};chat.on(TencentCloudChat.EVENT.TOTAL_UNREAD_MESSAGE_COUNT_UPDATED, onTotalUnreadMessageCountUpdated);
```

## Очистка количества непрочитанных сообщений в разговоре

##### **API**

```
chat.setMessageRead(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object`. Он содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| conversationID | String | ID разговора. Допустимые значения: `C2C${userID}` (для диалога один-на-один) `GROUP{groupID}` (для группового чата) `@TIM#SYSTEM` (для разговора системных уведомлений) `GROUP${topicID}` (для темы) |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Отметить все непрочитанные сообщения в разговоре как прочитанныеlet promise = chat.setMessageRead({conversationID: 'C2Cexample'});promise.then(function(imResponse) {  // Успешно отмечены непрочитанные сообщения как прочитанные.  // Значение атрибута `unreadCount` разговора с указанным ID установлено на `0`.}).catch(function(imError) {  // Ошибка при отметке непрочитанных сообщений как прочитанные  console.warn('setMessageRead error:', imError);});
```

## Очистка количества непрочитанных сообщений всех разговоров

##### API

```
chat.setAllMessageRead(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object`. Он содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| scope | String \| undefined | Установить область обработки сообщений. Допустимые значения: `TencentCloudChat.TYPES.READ_ALL_C2C_MSG`: отметить непрочитанные сообщения всех диалогов один-на-один как прочитанные `TencentCloudChat.TYPES.READ_ALL_GROUP_MSG`: отметить непрочитанные сообщения всех групповых разговоров как прочитанные `TencentCloudChat.TYPES.READ_ALL_MSG ` (значение по умолчанию): отметить непрочитанные сообщения всех диалогов один-на-один и групповых разговоров как прочитанные |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Отметить все непрочитанные сообщения во всех разговорах как прочитанные// Аналогично `chat.setAllMessageRead({scope: TencentCloudChat.TYPES.READ_ALL_MSG})`let promise = chat.setAllMessageRead();promise.then(function(imResponse) {  // Успешно отмечены непрочитанные сообщения как прочитанные.  // Значения атрибута `unreadCount` всех разговоров установлены на `0`.}).catch(function(imError) {  // Ошибка при отметке непрочитанных сообщений как прочитанные  console.warn('setAllMessageRead error:', imError);});
```

```
// Отметить все непрочитанные сообщения всех диалогов один-на-один как прочитанныеlet promise = chat.setAllMessageRead({scope: TencentCloudChat.TYPES.READ_ALL_C2C_MSG});promise.then(function(imResponse) {  // Успешно отмечены непрочитанные сообщения как прочитанные.  // Значения атрибута `unreadCount` всех диалогов один-на-один установлены на `0`.}).catch(function(imError) {  // Ошибка при отметке непрочитанных сообщений как прочитанные  console.warn('setAllMessageRead error:', imError);});
```

```
// Отметить все непрочитанные сообщения всех групповых разговоров как прочитанныеlet promise = chat.setAllMessageRead({scope: TencentCloudChat.TYPES.READ_ALL_GROUP_MSG});promise.then(function(imResponse) {  // Успешно отмечены непрочитанные сообщения как прочитанные.  // Значения атрибута `unreadCount` всех групповых разговоров установлены на `0`.}).catch(function(imError) {  // Ошибка при отметке непрочитанных сообщений как прочитанные  console.warn('setAllMessageRead error:', imError);});
```

## Отправка сообщений, исключённых из количества непрочитанных сообщений разговора

В обычных случаях отправленные сообщения один-на-один и групповые сообщения включаются в количество непрочитанных сообщений. Атрибут `unreadCount` объекта `Conversation` указывает количество непрочитанных сообщений в разговоре.

Если вы хотите отправлять сообщения, которые не будут включены в количество непрочитанных сообщений, такие как подсказки или управляющие сообщения, вы можете обратиться к следующему примеру кода.

##### **Примеры**

```
chat.sendMessage(message, {  messageControlInfo: {    // `unreadCount` разговора не обновляется (сообщение сохраняется на сервере роумингов).    excludedFromUnreadCount: true,    // `lastMessage` разговора не обновляется (сообщение сохраняется на сервере роумингов).     excludedFromLastMessage: true  }});
```

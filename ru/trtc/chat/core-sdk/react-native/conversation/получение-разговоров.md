# Получение разговоров

## Обзор функции

Chat SDK предоставляет API для получения разговоров, который можно использовать для получения информации объекта `Conversation` одного или нескольких указанных разговоров.

## Получение списка разговоров

> **Примечание:** Профиль в списке разговоров, полученный через этот API, неполный (содержит только фотографию профиля, псевдоним и т.д., что достаточно для отображения списка разговоров). Для запроса подробных профилей разговоров см. `getConversationProfile`. По умолчанию клиент может получить 100 последних разговоров из облака. После обновления до Pro edition, Pro Plus edition или Enterprise edition можно получить до 500 последних разговоров из облака. Период хранения разговора совпадает с периодом хранения последнего сообщения, по умолчанию составляет 7 дней. См. [Pricing](https://trtc.io/pricing/chat). Период хранения разговора совпадает с периодом хранения последнего сообщения, по умолчанию составляет 7 дней. Возвращаемое поле данных `isSyncCompleted` этого API используется для указания того, завершена ли синхронизация списка разговоров из облака. После успешного входа в систему SDK активно получает список разговоров способом постраничного запроса. Результат возврата этого API — это список разговоров, полученный SDK. Если SDK еще не получил данные из облака, вызов этого API вернет пустой массив.

##### **API**

```
chat.getConversationList(options);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| options | undefined \| Array \| Object | Параметры опций. Если `options` не переданы, это означает получение всех разговоров. Если передан непустой параметр массива, это означает получение нескольких указанных разговоров. Передача `{ type, markType, groupName, hasUnreadCount, hasGroupAtInfo }` в `options` означает фильтрацию списка разговоров согласно этим условиям. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Получить полный список разговоровlet promise = chat.getConversationList();promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList; }).catch(function(imError) {  console.warn('getConversationList error:', imError); // Не удалось получить список разговоров});
```

```
// Получить список указанных разговоровlet promise = chat.getConversationList([conversationID1, conversationID2]);promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;}).catch(function(imError) {  console.warn('getConversationList error:', imError); // Не удалось получить список разговоров});
```

```
// Получить все групповые разговорыlet promise = chat.getConversationList({ type: TencentCloudChat.TYPES.CONV_GROUP });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить все "избранные" разговорыlet promise = chat.getConversationList({ markType: TencentCloudChat.TYPES.CONV_MARK_TYPE_STAR });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить все немечены разговорыlet promise = chat.getConversationList({ markType: 0 });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить все разговоры в указанной группе разговоровlet promise = chat.getConversationList({ groupName: 'Suppliers' });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить все разговоры, которые не принадлежат ни одной группеlet promise = chat.getConversationList({ groupName: '' });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить все разговоры со счетчиком непрочитанныхlet promise = chat.getConversationList({ hasUnreadCount: true });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

```
// Получить разговоры, включающие групповые сообщения @let promise = chat.getConversationList({ hasGroupAtInfo: true });promise.then(function(imResponse) {  const conversationList = imResponse.data.conversationList;});
```

## Получение деталей разговора

Этот API можно использовать для получения подробной информации о разговоре.

##### **API**

```
chat.getConversationProfile(conversationID);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| conversationID | String | ID разговора. Метод формирования ID разговора: `C2C${userID}` (Приватный чат) `GROUP{groupID}` (Групповой чат) `@TIM#SYSTEM` (Сеанс системного уведомления) |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.getConversationProfile(conversationID);promise.then(function(imResponse) {  // Успешно получено  console.log(imResponse.data.conversation); // Профиль разговора}).catch(function(imError) {  console.warn('getConversationProfile error:', imError); // Не удалось получить профиль разговора});
```

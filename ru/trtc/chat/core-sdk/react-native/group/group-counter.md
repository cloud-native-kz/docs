# Счётчик группы

## Обзор функции

Chat SDK предоставляет функцию счётчика группы, позволяя установить определённое количество счётчиков для каждой группы.

В отличие от пользовательских атрибутов группы, счётчики группы в основном используются для хранения данных целочисленного типа. Вы можете использовать счётчики группы для хранения дополнительной информации, связанной с группой, такой как совокупное количество зрителей, количество просмотров, количество "лайков", полученных ведущим, и общее количество подарков, переданных аудиторией ведущему в прямую группу.

> **Примечание:** Счётчики группы поддерживают все типы групп, за исключением групп сообществ с включённой функцией тем. Эта функция поддерживается только в [Premium edition](https://trtc.io/buy/chat).

Учитывайте следующее для счётчиков группы:

1. Одна группа может поддерживать до 20 счётчиков группы, то есть количество ключей в одной группе не должно превышать 20.
2. Ключ одного счётчика группы не может превышать 128 символов, а значение должно быть целым числом (до 64-битных знаковых целых чисел).
3. API `setGroupCounters`, `increaseGroupCounter` и `decreaseGroupCounter` объединяют вычисления с ограничением SDK в 20 вызовов за 5 секунд на вход в систему пользователя. При превышении интерфейс возвращает код ошибки 2996.
4. Интерфейс `getGroupCounters` рассчитывается независимо. SDK ограничивает одного пользователя максимум 20 вызовами в течение 5 секунд на вход в систему. При превышении лимита интерфейс возвращает код ошибки 2996.

## Установка счётчиков группы

После успешной установки счётчика инициируется событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Если ключ устанавливаемого вами счётчика существует, значение счётчика будет обновлено напрямую; если нет, будет добавлена новая пара ключ-значение. Если несколько пользователей устанавливают один и тот же счётчик одновременно, будет использовано окончательное значение счётчика. Рекомендуется, чтобы владелец группы выполнял операции установки счётчика.

##### **API**

```
chat.setGroupCounters(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| counters | Object | Пара ключ-значение счётчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Установите значения счётчиков key1 и key2 в 0let promise = chat.setGroupCounters({  groupID: 'group1',  counters: { key1: 0, key2: 0 }});promise.then(function(imResponse) { // Установка выполнена успешно  console.log(imResponse.data.counters); // Счётчики группы успешно установлены}).catch(function(imError) { // Установка не удалась  console.warn('setGroupCounters error:', imError); // Информация об ошибке});
```

## Увеличение значения счётчика группы

После успешного увеличения счётчика инициируется событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Значение в параметрах интерфейса — это размер изменения; после вызова интерфейса входящий размер изменения будет добавлен к текущему значению. Если ключ счётчика, который вы собираетесь установить, уже существует, система будет напрямую увеличивать текущее значение на основе входящего значения; в противном случае будет добавлен ключ и увеличение произойдёт на основе входящего значения, начиная с значения по умолчанию 0.

##### **API**

```
chat.increaseGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| key | String | Ключ счётчика группы |
| value | Number | Размер прироста счётчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Предположим, что текущее значение счётчика key1 равно 8. После вызова интерфейса increaseGroupCounter и передачи // размера прироста value = 2, окончательное значение key1 становится 10.let promise = chat.increaseGroupCounter({  groupID: 'group1',  key: 'key1',  value: 2,});promise.then(function(imResponse) { // Прирост выполнен успешно  console.log(imResponse.data);  const { groupID, key, value } = imResponse.data;}).catch(function(imError) { // Прирост не удался  console.warn('increaseGroupCounter error:', imError);});
```

## Уменьшение значения счётчика группы

После успешного уменьшения счётчика инициируется событие `TencentCloudChat.EVENT.GROUP_COUNTER_UPDATED`.

> **Примечание:** Значение в параметрах интерфейса — это размер изменения; после вызова интерфейса входящий размер изменения будет вычтен из текущего значения. Если ключ счётчика, который вы собираетесь установить, уже существует, система будет напрямую уменьшать текущее значение на основе входящего значения; в противном случае будет добавлен ключ и уменьшение произойдёт на основе входящего значения, начиная с значения по умолчанию 0.

##### **API**

```
chat.decreaseGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| key | String | Ключ счётчика группы |
| value | Number | Размер уменьшения счётчика группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Предположим, что текущее значение счётчика key1 равно 8.// После вызова API decreaseGroupCounter с размером уменьшения 2// Окончательное значение key1 становится 6.let promise = chat.decreaseGroupCounter({  groupID: 'group1',  key: 'key1',  value: 2});promise.then(function(imResponse) { // Уменьшение выполнено успешно  console.log(imResponse.data);  const { groupID, key, value } = imResponse.data;}).catch(function(imError) { // Установка не удалась  console.warn('decreaseGroupCounter error:', imError);});
```

## Получение счётчиков группы

> **Примечание:** Если keyList не передан, возвращаются все счётчики группы.

##### **API**

```
chat.getGroupCounter(options);
```

##### **Параметры**

Параметр `options` имеет тип `Object` и содержит следующие значения атрибутов:

| Имя | Тип | Описание |
| --- | --- | --- |
| groupID | String | ID группы |
| keyList | Array \| undefined | Список ключей счётчиков группы |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// Получите значения счётчиков группы key1 и key2let promise = chat.getGroupCounters({  groupID: 'group1',  keyList: ['key1', 'key2']});promise.then(function(imResponse) { // Получено успешно  console.log(imResponse.data.counters);}).catch(function(imError) {  console.warn('getGroupCounters error:', imError); // Информация об ошибке});
```

```
// Получите все счётчики группыlet promise = chat.getGroupCounters({  groupID: 'group1'});promise.then(function(imResponse) { // Получено успешно  console.log(imResponse.data.counters);}).catch(function(imError) {  console.warn('getGroupCounters error:', imError); // Информация об ошибке});
```

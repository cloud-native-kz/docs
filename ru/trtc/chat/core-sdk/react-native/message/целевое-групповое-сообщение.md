# Целевое групповое сообщение

## Обзор функции

Целевое групповое сообщение — это сообщение, отправленное указанным участникам группы, которое не могут получить другие члены группы.

> **Примечание:** Для использования этой функции необходимо приобрести издание Pro, Pro Plus или Enterprise согласно инструкциям в [Pricing](https://trtc.io/pricing/chat). При создании группового сообщения @ вы не можете указать список членов группы для получения сообщения (`receiverList`). Функция целевого группового сообщения недоступна для сообществ и аудиовидео групп (AVChatRoom). По умолчанию целевые групповые сообщения исключены из количества непрочитанных сообщений групповой беседы.

## Отправка целевого группового сообщения

Чтобы отправить целевое групповое сообщение указанным участникам группы, следуйте приведённым ниже инструкциям:

- Вызовите API `createXxxMessage` (здесь `Xxx` обозначает тип сообщения) для создания сообщения (укажите получателей сообщения через `receiverList`).
- Вызовите API `sendMessage` для отправки сообщения.

##### Пример

```
// Create a targeted group messagelet message = chat.createTextMessage({  to: 'test',  conversationType: TencentCloudChat.TYPES.CONV_GROUP,  payload: {    text: 'Hello world!'  },  receiverList: ['user0', 'user1']});// Send the messagelet promise = chat.sendMessage(message);promise.then(function(imResponse) {  // Message sent successfully  console.log(imResponse);}).catch(function(imError){  // The message failed to be sent  console.warn('sendMessage error:', imError);});
```

## Получение целевого группового сообщения

По умолчанию целевые групповые сообщения исключены из количества непрочитанных сообщений групповой беседы.

Целевое групповое сообщение можно получить так же, как обычное сообщение.

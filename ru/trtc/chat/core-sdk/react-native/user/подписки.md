# Подписки

## Обзор функции

- **Подписка** позволяет пользователям подписываться на интересующих их пользователей и своевременно получать последние новости, ленты или информацию о событиях этих пользователей. Система может предоставлять персонализированные рекомендации контента на основе списка подписок пользователя.
- **Подписчики** — это состояние, когда пользователь подписан другими пользователями. Когда пользователь A подписывается на пользователя B, A становится подписчиком B. Пользователи могут просматривать количество подписчиков, список подписчиков или профиль подписчиков на своей странице профиля.

С помощью этих функций вы можете создать активную, взаимосвязанную сеть пользователей, способствуя распространению информации и развитию сообщества.

> **Примечание:** Эта функция поддерживается только в выпусках Pro, Pro-plus и Enterprise. Пожалуйста, [приобретите Pro, Pro-plus или Enterprise](https://console.trtc.io/subscription/buy/chat?language=en) для использования этой функции.

## Подписка на пользователей

> **Примечание:** Этот интерфейс поддерживает подписку на до 20 пользователей одновременно (кроме самого себя, на любого другого пользователя можно подписаться). Каждый пользователь может иметь максимум 5000 подписчиков, но на количество подписок ограничений нет.

**API**

```
chat.followUser(options);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список идентификаторов пользователей. Поддерживает подписку на до 20 пользователей одновременно. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.followUser(['user1', 'user2']);promise.then(function(imResponse) {  console.log(imResponse.data);}).catch(function(imError) {  console.warn('followUser error:', imError);});
```

## Получение списка моих подписчиков

> **Примечание:** Этот API возвращает до 500 пользователей за один раз.

**API**

```
chat.getMyFollowersList(nextCursor);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| nextCursor | String \| undefined | Начальная позиция для получения с разбиением по страницам. По умолчанию пусто для первой страницы и может быть опущено. После успешного получения nextCursor не равен `''`. Если требуется разбиение по страницам, вы можете передать это значение снова для получения до тех пор, пока nextCursor не вернет `''`. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.getMyFollowersList(nextCursor);promise.then(function(imResponse) {  const { resultList, nextCursor = '' } = imResponse.data;  // ressultList - список моих подписчиков  // nextCursor - идентификатор для продолжения получения по страницам  if (nextCursor != '') {    // выполнить получение по страницам  }}).catch(function(imError) {  console.warn('getMyFollowersList error:', imError);});
```

## Получение количества подписок/подписчиков/взаимных подписок

**API**

```
chat.getUserFollowInfo(userIDList);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список идентификаторов пользователей; если userIDList не предоставлен, это означает получение информации о подписках для себя. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
// получить количество моих подписок/подписчиков/взаимных подписокlet promise = chat.getUserFollowInfo();promise.then(function(imResponse) {  console.log(imResponse.data);}).catch(function(imError) {  console.warn('getUserFollowInfo error:', imError);});
```

```
// получить количество подписок/подписчиков/взаимных подписок для указанных пользователейlet promise = chat.getUserFollowInfo(['user1', 'user2']);promise.then(function(imResponse) {  console.log(imResponse.data);}).catch(function(imError) {  console.warn('getUserFollowInfo error:', imError);});
```

## Проверка типа подписки

**API**

```
chat.checkFollowType(userIDList);
```

##### **Параметры**

| Имя | Тип | Описание |
| --- | --- | --- |
| userIDList | Array.<String> | Список идентификаторов пользователей для проверки; один запрос поддерживает до 100 идентификаторов пользователей. |

##### **Возвращаемое значение**

`Promise`

##### **Примеры**

```
let promise = chat.checkFollowType(['user1', 'user2']);promise.then(function(imResponse) {  console.log(imResponse.data);  imResponse.data.forEach((item) => {    // item.userID - userID    // item.followType - 0 - нет связи, 1 - мой подписчик, 2 - моя подписка, 3 - взаимная подписка  });}).catch(function(imError) {  console.warn('checkFollowType error:', imError);});
```

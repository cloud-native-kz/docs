# Журнал изменений

## Chat React Native SDK 0.1.9 @2022.12.05

- Новое: URL-адреса мультимедийных сообщений в интернете (`url`) больше не будут возвращаться по умолчанию. Их необходимо получить, вызвав API `getMessageOnlineUrl`.
- Новое: Локальные URL-адреса мультимедийных сообщений (`localurl`) больше не будут возвращаться по умолчанию. Они будут возвращены только после успешной загрузки сообщений путем вызова API `downloadMessage`.
- Новое: Для `advanceMessageListener` добавлен обратный вызов `onMessageDownloadProgressCallback`, который будет срабатывать при обновлении прогресса загрузки мультимедийного сообщения.
- Новое: Добавлена функция расширения сообщений. Подробности см. в разделе Message > Message Extension.
- Улучшение: Базовый нативный SDK был обновлен до версии 6.9.3557.

> **Примечание** В мультимедийные сообщения и файловые сообщения были внесены существенные изменения. Пожалуйста, измените вашу существующую логику получения и отрисовки таких сообщений в соответствии с первыми тремя обновлениями. В противном случае сообщения не будут отображаться. Если у вас возникнут вопросы в процессе изменения, пожалуйста, [свяжитесь с нами](https://www.tencentcloud.com/contact-us).

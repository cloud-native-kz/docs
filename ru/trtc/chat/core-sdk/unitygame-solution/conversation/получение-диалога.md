#  Получение диалога

## Описание функции

Chat SDK предоставляет API для получения диалогов, который можно использовать для получения информации объекта `ConvInfo` одного или нескольких указанных диалогов.
Вызовите `ConvGetConvInfo` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/ConvApi/ConvGetConvInfo.html)) для получения информации об одном или нескольких диалогах.

Пример кода:

```
List<ConvParam> list = new List<ConvParam>();ConvParam conv1 = new ConvParam();conv1.get_conversation_list_param_conv_id = "group_conv_id";conv1.get_conversation_list_param_conv_type = TIMConvType.kTIMConv_Group;list.Add(conv1);ConvParam conv2 = new ConvParam();conv2.get_conversation_list_param_conv_id = "c2c_conv_id";conv2.get_conversation_list_param_conv_type = TIMConvType.kTIMConv_C2C;list.Add(conv2);TIMResult res = TencentIMSDK.ConvGetConvInfo(list, (int code, string desc, List<ConvInfo> info_list, string user_data)=>{ // Process the async logic});
```

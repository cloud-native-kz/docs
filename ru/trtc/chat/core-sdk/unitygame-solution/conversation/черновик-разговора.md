# Черновик Разговора

## Описание Функции

При отправке сообщения пользователь может захотеть переключиться на другое окно чата перед завершением сообщения. Незаконченное сообщение можно сохранить через API `ConvSetDraft`, чтобы пользователь мог получить доступ к нему через поле `conv_draft` объекта `ConvInfo` и завершить его.

> **Внимание:** Черновик может содержать только текст. Черновик будет сохранён только в локальной базе данных и не на сервере. Поэтому он не может синхронизироваться между устройствами и не будет доступен после удаления и переустановки приложения.

## Установка Черновика Разговора

Вызовите API `ConvSetDraft` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/ConvApi/ConvSetDraft.html)) для установки черновика разговора.

Пример кода:

```
DraftParam draft_param = new DraftParam{  draft_msg = new Message  {    message_conv_id = "1234",    message_conv_type = TIMConvType.kTIMConv_Group,    message_elem_array = new List<Elem>{new Elem    {      elem_type = TIMElemType.kTIMElem_Text,      text_elem_content = Input.text    }},  }}TIMResult res = TencentIMSDK.ConvSetDraft(conv_id, conv_type, draft_param);
```

# Счетчик группы

## Обзор

Начиная с нативного SDK v7.0, Tencent Cloud Chat SDK предоставляет функцию счетчика группы. Вы можете настроить определенное количество счетчиков для каждой группы.

В отличие от [пользовательских атрибутов группы](https://intl.cloud.tencent.com/document/product/1047/48175), счетчики группы используются для хранения данных целого числа. Счетчики группы можно использовать для хранения дополнительной информации, связанной с группой, например кумулятивное количество зрителей, количество просмотров, количество лайков и количество подарков, которые аудитория подарила хосту аудиовидео группы.

> **Примечание:** Счетчики группы поддерживают все типы групп, кроме групп сообществ с включенными темами. Функция счетчика группы поддерживается только версией Pro, Pro Plus или Enterprise.

Имейте в виду следующее для счетчиков группы:

1. Одна группа поддерживает до 20 счетчиков группы, то есть до 20 ключей на группу.
2. Для одного счетчика группы ключ может содержать до 128 байт, а значение должно быть целым числом (целое число со знаком до 64 бит).
3. API `GroupSetGroupCounters`, `GroupIncreaseGroupCounter` и `GroupDecreaseGroupCounter` в совокупности могут быть вызваны авторизованным пользователем до 20 раз в течение пяти секунд в SDK, и код ошибки 8516 будет возвращен, если лимит превышен.
4. API `GroupGetGroupCounters` может быть вызван авторизованным пользователем до 20 раз в течение пяти секунд в SDK, и код ошибки 8516 будет возвращен, если лимит превышен.

## Установка счетчиков группы

> **Примечание:** Если `key` счетчика группы, который необходимо установить, уже существует, значение `key` обновляется напрямую. В противном случае будет добавлена пара ключ-значение. Если несколько пользователей устанавливают один и тот же счетчик одновременно, будет использоваться окончательное значение счетчика. Рекомендуется, чтобы владелец группы выполнял операции установки счетчика.

Пример: вызовите API `GroupSetGroupCounters` для установки значений счетчиков `key1` и `key2` в 0

```
    // Set a group counter    List<GroupCounter> list = new List<GroupCounter>    {      new GroupCounter      {        group_counter_key = "key",        group_counter_value = 1      }    };    TIMResult res = TencentIMSDK.GroupSetGroupCounters("groupID", list, (int code, string desc, List<GroupCounter> results, string user_data)=>{      // Async result of the group counter setting    });
```

## Увеличение значения счетчика группы

> **Примечание:** Параметр API `value` является значением изменения. Каждый раз при вызове API текущее значение увеличивается на переданное значение. Если `key` счетчика группы, который необходимо установить, уже существует, текущее значение напрямую увеличивается на переданное значение. В противном случае будет добавлен `key`, и значение по умолчанию (0) будет увеличено на переданное значение.

Пример: предположим, что текущее значение счетчика `key1` равно 8. После вызова API `GroupIncreaseGroupCounter` с передачей значения увеличения 2, окончательное значение `key1` становится 10.

```
    // Increment the value of a group counter    TIMResult res = TencentIMSDK.GroupIncreaseGroupCounter("groupID", "key1", 2, (int code, string desc, List<GroupCounter> results, string user_data)=>{      // Async result of the group counter increment    });
```

## Уменьшение значения счетчика группы

> **Примечание:** Параметр API `value` является значением изменения. Каждый раз при вызове API текущее значение уменьшается на переданное значение. Если `key` счетчика группы, который необходимо установить, уже существует, текущее значение напрямую уменьшается на переданное значение. В противном случае будет добавлен `key`, и значение по умолчанию (0) будет уменьшено на переданное значение.

Пример: предположим, что текущее значение счетчика `key1` равно 8. После вызова API `GroupDecreaseGroupCounter` с передачей значения уменьшения 2, окончательное значение `key1` становится 6.

```
    // Decrement the value of a group counter    TIMResult res = TencentIMSDK.GroupDecreaseGroupCounter("groupID", "key1", 2, (int code, string desc, List<GroupCounter> results, string user_data)=>{      // Async result of the group counter decrement    });
```

## Получение счетчиков группы

> **Примечание:** Если переданный список ключей пуст, возвращаются все счетчики группы.

Пример: вызовите API `GroupGetGroupCounters` для получения значений счетчиков `key1` и `key2`

```
    // Get group counters    TIMResult res = TencentIMSDK.GroupGetGroupCounters("groupID", new List<string> {"key1", "key2"}, (int code, string desc, List<GroupCounter> results, string user_data)=>{      // Async result of the group counter getting    });
```

## Уведомление об изменении счетчика группы

> **Примечание:** Прежде чем вы сможете вызвать указанный выше обратный вызов, вам необходимо вызвать API `SetGroupCounterChangedCallback` ([подробнее](https://comm.qq.com/im/doc/unity/zh/api/SDKRegisteringCallback/SetGroupCounterChangedCallback.html)) для добавления прослушивателя группы.

Пример кода:

```
    // Set the group counter change callback    TencentIMSDK.SetGroupCounterChangedCallback((string group_id, string group_counter_key, ulong group_counter_new_value, string user_data) => {    });
```

# Управление группами

## Описание функции

Функция управления группами позволяет создавать группу, присоединяться к группе, получать список присоединённых групп, выходить из группы или распускать группу.

## Обработчик событий группы

Вызовите `SetGroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/SDKRegisteringCallback/SetGroupTipsEventCallback.html)), чтобы добавить обработчик событий группы.

Чтобы прекратить получение событий группы, вызовите `SetGroupTipsEventCallback` ещё раз ([Подробности](https://comm.qq.com/im/doc/unity/en/api/SDKRegisteringCallback/SetGroupTipsEventCallback.html)) и передайте `null`, чтобы удалить обработчик событий группы.

> **Внимание:** Необходимо заранее установить обработчик событий группы, чтобы получать уведомления о событиях группы.

Пример кода:

```
TencentIMSDK.SetGroupTipsEventCallback((GroupTipsElem message, string user_data)=>{ // Process the callback logic});
```

## Создание группы

Для инициализации информации группы, такой как описание группы, фото профиля группы и существующих членов группы при создании группы, вызовите расширенный API `GroupCreate` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupCreate.html)), и `create_group_result_groupid` будет возвращён в обратном вызове при успешном создании.

Пример кода:

```
// Create a public group and specify group attributesCreateGroupParam param = new CreateGroupParam{  create_group_param_group_id = "group_id",  create_group_param_group_name = "group_name",  create_group_param_group_type = TIMGroupType.kTIMGroup_Public,  create_group_param_add_option = TIMGroupAddOption.kTIMGroupAddOpt_Any,};TIMResult res = TencentIMSDK.GroupCreate(param, (int code, string desc, CreateGroupResult result, string user_data)=>{ // Process the async logic});
```

## Присоединение к группе

| Тип | Способ присоединения к группе |
| --- | --- |
| Рабочая группа (Work) | По приглашению |
| Открытая группа (Public) | По запросу пользователя и одобрению владельца группы или администратора |
| Группа встреч (Meeting) | Свободное присоединение |
| Сообщество (Community) | Свободное присоединение |
| Аудиовидео группа (AVChatRoom) | Свободное присоединение |

Ниже описано, как присоединиться к группам в порядке от простого к сложному.

> **Внимание:** Необходимо заранее вызвать `SetGroupTipsEventCallback`, чтобы добавить обработчик событий группы и получать следующие события группы.

#### Свободное присоединение к группе

Группы встреч (Meeting), аудиовидео группы (AVChatRoom) и сообщества предназначены в основном для сценариев свободного взаимодействия, таких как онлайн-конференции и трансляции. Процесс присоединения к таким группам наиболее простой:

1. Пользователь вызывает `GroupJoin` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupJoin.html)), чтобы присоединиться к группе.
2. После успешного присоединения пользователя к группе все члены группы (включая пользователя) получат обратный вызов `GroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/callback/GroupTipsEventCallback.html)).

Пример кода:

```
// Listen for the group join eventTencentIMSDK.SetGroupTipsEventCallback((GroupTipsElem message, string user_data)=>{ // Process the callback logic});// Join a groupTIMResult res = TencentIMSDK.GroupJoin(group_id, "greeting message", (int code, string desc, string user_data)=>{ // Process the async logic});
```

#### Присоединение к группе по приглашению

Рабочие группы (Work), подобно группам WeChat и WeCom, подходят для общения в рабочих условиях. Модель взаимодействия разработана таким образом, чтобы отключить активное присоединение к группе и разрешить пользователям присоединяться к группе только по приглашению от членов группы.
Этапы присоединения к группе следующие:

1. Член группы вызывает `GroupInviteMember` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupInviteMember.html)), чтобы пригласить пользователя в группу.
2. Все члены группы (включая пользователя) получат обратный вызов `GroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/callback/GroupTipsEventCallback.html)).

Пример кода:

```
// Listen for the group invitation eventTencentIMSDK.SetGroupTipsEventCallback((GroupTipsElem message, string user_data)=>{ // Process the callback logic});// Invite the `userA` user to join the `groupA` groupGroupInviteMemberParam param = new GroupInviteMemberParam{  group_invite_member_param_group_id = "group_id",  group_invite_member_param_identifier_array = new List<string> {    "1234"  } // Array of IDs of the users invited to the group};TIMResult res = TencentIMSDK.GroupInviteMember(param, (int code, string desc, List<GroupInviteMemberResult> result, string user_data)=>{ // Process the async logic});
```

#### Присоединение к группе по запросу и одобрению

Открытая группа (Public) аналогична группе интересов и групповому чату QQ. Любой может присоединиться к ней по запросу и одобрению владельца группы или администратора.

Описание процесса:

1. Установите обратный вызов для системного сообщения группы `SetGroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/SDKRegisteringCallback/SetGroupTipsEventCallback.html)).
2. Пользователь вызывает `GroupJoin` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupJoin.html)), чтобы запросить присоединение к группе.
3. Владелец группы или администратор просматривает список запросов на присоединение к группе и вызывает `GroupHandlePendency` ([Подробности](https://comm.qq.com/im/sdk/unity_plus/_site_en/api/com.tencent.imsdk.unity.TencentIMSDK.html#com_tencent_imsdk_unity_TencentIMSDK_GroupHandlePendency_com_tencent_imsdk_unity_types_GroupHandlePendencyParam_com_tencent_imsdk_unity_callback_NullValueCallback_)), чтобы одобрить/отклонить запрос.
4. При одобрении все члены группы (включая пользователя) получат обратный вызов `SetGroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/SDKRegisteringCallback/SetGroupTipsEventCallback.html)), уведомляющий членов группы о том, что кто-то присоединился к группе.

Владелец группы или администратор также может вызвать API `GroupModifyGroupInfo` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupModifyGroupInfo.html)), чтобы изменить опцию присоединения к группе (`group_modify_info_param_add_option`) на "присоединение к группе запрещено" или "одобрение не требуется".

`group_modify_info_param_add_option` имеет следующие варианты:

| Опция присоединения к группе | Описание |
| --- | --- |
| TIMGroupAddOption.kTIMGroupAddOpt_Forbid | Никакие пользователи не могут присоединиться к группе. |
| TIMGroupAddOption.kTIMGroupAddOpt_Auth | Требуется одобрение от владельца группы или администратора, чтобы присоединиться к группе (значение по умолчанию). |
| TIMGroupAddOption.kTIMGroupAddOpt_Any | Любой пользователь может присоединиться к группе без одобрения. |

## Получение списка присоединённых групп

Вызовите `GroupGetJoinedGroupList` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupGetJoinedGroupList.html)), чтобы получить список присоединённых рабочих групп (Work), открытых групп (Public), групп встреч (Meeting) и сообществ (Community, которые **не поддерживают** функцию тем). Аудиовидео группы (AVChatRoom) и сообщества (Community, которые **поддерживают** функцию тем) не включаются в этот список.

Пример кода:

```
// Get the joined groupsTIMResult res = TencentIMSDK.GroupGetJoinedGroupList((int code, string desc, List<GroupBaseInfo> info_list, string user_data)=>{ // Process the async logic});
```

## Выход из группы

> **Внимание:** Владелец группы **не может** выйти из открытой группы (Public), группы встреч (Meeting), сообщества или аудиовидео группы (AVChatRoom) и может только распустить её.

Пример кода:

```
// Leave a groupTIMResult res = TencentIMSDK.GroupQuit(group_id, (int code, string desc, string user_data)=>{ // Process the async logic});
```

## Распуск группы

Пример кода:

```
// Disband a groupTIMResult res = TencentIMSDK.GroupDelete(group_id, (int code, string desc, string user_data)=>{ // Process the async logic});
```

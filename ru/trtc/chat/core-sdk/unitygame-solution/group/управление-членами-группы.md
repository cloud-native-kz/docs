# Управление членами группы

## Описание функции

Управление членами группы включает получение списка членов, отключение членов группы, удаление членов группы, предоставление разрешений и передачу владения группой.

## Получение списка членов группы

Группа может иметь большое количество членов (например, более 5000). В этом случае вы можете использовать фильтр (`group_get_members_info_list_param_option`) и постраничное получение данных (`group_get_members_info_list_param_next_seq`) — продвинутые функции API для получения списка членов группы.

### Фильтр (`filter`)

При вызове API `GroupGetMemberInfoList` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupGetMemberInfoList.html)) вы можете указать получение списка информации о членах с указанными ролями.

| Фильтр | Тип |
| --- | --- |
| TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_All | Получить список информации всех членов группы |
| TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_Owner | Получить список информации владельца группы |
| TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_Admin | Получить список информации администраторов группы |
| TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_Member | Получить список информации обычных членов группы |

Пример кода:

```
// Specify to pull the profile of the group owner through the `filter` parameterGroupGetMemberInfoListParam param = new GroupGetMemberInfoListParam{  group_get_members_info_list_param_group_id = "group_id",  group_get_members_info_list_param_option = new GroupMemberGetInfoOption  {    group_member_get_info_option_role_flag = TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_Owner  }};TIMResult res = TencentIMSDK.GroupGetMemberInfoList(param, (int code, string desc, GroupGetMemberInfoListResult result, string user_data)=>{ // Process the async logic});
```

### Получение по страницам (`nextSeq`)

Во многих случаях на пользовательском интерфейсе требуется отображать только первую страницу списка членов группы, а не информацию о всех членах группы. Дополнительные члены группы необходимо получить только тогда, когда пользователь нажимает **Следующая страница** или прокручивает страницу списка. В этом случае можно использовать постраничное получение.

Направления постраничного получения:

1. При первом вызове `GroupGetMemberInfoList` установите `nextSeq` на `0` (это означает получение списка членов группы с начала). За один раз можно получить до 50 объектов членов группы.
2. После первого успешного получения списка членов группы обратный вызов `GroupGetMemberInfoListResult` из `GroupGetMemberInfoList` будет содержать `group_get_memeber_info_list_result_next_seq` (это поле для следующего получения):
  - Если `group_get_memeber_info_list_result_next_seq` равно `0`, все члены группы были получены.
  - Если `group_get_memeber_info_list_result_next_seq` больше `0`, есть дополнительные члены группы, которые можно получить. Это не означает, что следующая страница списка членов будет получена немедленно. В обычном программном обеспечении для общения постраничное получение часто инициируется операцией прокрутки.
3. Когда пользователь продолжает прокручивать список членов группы, если есть дополнительные члены для получения, вы можете продолжить вызов API `GroupGetMemberInfoList` и передать параметр `nextSeq` (значение из объекта `GroupGetMemberInfoListResult`, возвращённого последним получением) для следующего получения.
4. Повторяйте [шаг 3] до тех пор, пока `nextSeq` не будет равно `0`.

Пример кода:

```
// Specify to pull the profile of the group owner through the `filter` parameterGroupGetMemberInfoListParam param = new GroupGetMemberInfoListParam{  group_get_members_info_list_param_group_id = "group_id",  group_get_members_info_list_param_option = new GroupMemberGetInfoOption  {    group_member_get_info_option_role_flag = TIMGroupMemberRoleFlag.kTIMGroupMemberRoleFlag_Owner  },  group_get_members_info_list_param_next_seq = 0};TIMResult res = TencentIMSDK.GroupGetMemberInfoList(param, (int code, string desc, GroupGetMemberInfoListResult result, string user_data)=>{ // Process the async logic});
```

## Отключение звука

### Отключение звука у указанного члена группы

После отключения звука члену группы все члены группы (включая члена с отключённым звуком) получат обратный вызов `GroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/callback/GroupTipsEventCallback.html)).

### Отключение звука для всей группы

Владелец группы или администратор также может вызвать API `GroupModifyGroupInfo` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupModifyGroupInfo.html)) для отключения звука во всей группе, установив атрибут `group_modify_info_param_is_shutup_all` на `true`. Вся группа может быть отключена на неограниченный период времени и требует включения звука путём изменения `group_modify_info_param_is_shutup_all` на `false`.

> **Примечание:** Только владелец группы может отключить звук администратору.

Пример кода:

```
// Mute the group member `userB` for ten minutesGroupModifyMemberInfoParam param = new GroupModifyMemberInfoParam{  group_modify_member_info_group_id = "group_id",  group_modify_member_info_identifier = "userB",  group_modify_member_info_modify_flag = TIMGroupMemberModifyInfoFlag.kTIMGroupMemberModifyFlag_ShutupTime,  group_modify_member_info_shutup_time = 600};TIMResult res = TencentIMSDK.GroupModifyMemberInfo(param, (int code, string desc, string user_data)=>{ // Process the async logic});// Mute allGroupModifyInfoParam param = new GroupModifyInfoParam{  group_modify_info_param_group_id = "group_id",  group_modify_info_param_modify_flag = TIMGroupModifyInfoFlag.kTIMGroupModifyInfoFlag_ShutupAll,  group_modify_info_param_is_shutup_all = true};TIMResult res = TencentIMSDK.GroupModifyGroupInfo(param, (int code, string desc, string user_data)=>{ // Process the async logic});
```

## Удаление члена группы

После удаления обычного члена группы все члены (включая удалённого члена) получат обратный вызов `GroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/callback/GroupTipsEventCallback.html)).

Поскольку видеогруппа (AVChatRoom) может быть присоединена свободно, нет API для удаления члена группы из видеогруппы (AVChatRoom). Вы можете использовать `GroupModifyMemberInfo` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupModifyMemberInfo.html)) для отключения звука указанному члену, чтобы реализовать аналогичные элементы управления.

> **Примечание:** Только владелец группы может удалить администратора из группы.

Пример кода:

```
GroupDeleteMemberParam param = new GroupDeleteMemberParam{  group_delete_member_param_group_id = "group_id",  group_delete_member_param_identifier_array = new List<string>  {    "user_id"  }};TIMResult res = TencentIMSDK.GroupDeleteMember(param, (int code, string desc, List<GroupDeleteMemberResult> result, string user_data)=>{ // Process the async logic});
```

## Установка администратора

Владелец группы может вызвать `GroupModifyMemberInfo` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupModifyMemberInfo.html)), чтобы установить члена группы в открытой группе (Public) или группе встреч (Meeting) в качестве администратора.

Обычный член, назначенный администратором, имеет разрешения администратора для выполнения следующих операций:

- Изменение базового профиля группы
- Удаление обычного члена из группы
- Отключение звука обычному члену (предотвращение участия члена в разговоре в течение указанного периода времени)
- Одобрение запроса на присоединение к группе

Для получения дополнительной информации см. [Групповая система](https://intl.cloud.tencent.com/document/product/1047/33529).

После установки/отмены установки обычного члена в качестве администратора все члены (включая обычного члена) получат обратный вызов `GroupTipsEventCallback` ([Подробности](https://comm.qq.com/im/doc/unity/en/callback/GroupTipsEventCallback.html)).

Пример кода:

```
GroupModifyMemberInfoParam param = new GroupModifyMemberInfoParam{  group_modify_member_info_group_id = "group_id",  group_modify_member_info_identifier = "user_id",  group_modify_member_info_modify_flag = TIMGroupMemberModifyInfoFlag.kTIMGroupMemberModifyFlag_MemberRole,  group_modify_member_info_member_role = TIMGroupMemberRole.kTIMMemberRole_Admin};TIMResult res = TencentIMSDK.GroupModifyMemberInfo(param, (int code, string desc, string user_data)=>{ // Process the async logic});
```

## Получение количества членов группы, находящихся в сети

Вызовите `GroupGetOnlineMemberCount` ([Подробности](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupGetOnlineMemberCount.html)) для получения количества членов группы, находящихся в сети.

> **Примечание:** В настоящее время можно получить только количество членов, находящихся в сети, в видеогруппе (AVChatRoom). Этот API можно вызывать один раз в 60 секунд в SDK.

Пример кода:

```
TIMResult res = TencentIMSDK.GroupGetOnlineMemberCount("group_id", (int code, string desc, GroupGetOnlineMemberCountResult result, string user_data)=>{ // Process the async logic});
```

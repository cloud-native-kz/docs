# Поиск группы

## Описание функции

Можно искать только локально сохраненные группы, такие как список присоединенных групп и профили групп, которые были загружены.

## Поиск локальной группы

Вызовите API `GroupSearchGroups` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupSearchGroups.html)) для поиска локальной группы.
Вы можете установить ключевое слово поиска `group_search_params_keyword_list` и указать область поиска, чтобы установить, нужно ли выполнять поиск по полям `groupID` и `groupName` группы.

Пример кода:

```
// Search for a group by the keywordGroupSearchParam param = new GroupSearchParam{  group_search_params_keyword_list = new List<string>  {    "Keyword 1"  },  group_search_params_field_list = new List<TIMGroupSearchFieldKey>  {    TIMGroupSearchFieldKey.kTIMGroupSearchFieldKey_GroupId,    TIMGroupSearchFieldKey.kTIMGroupSearchFieldKey_GroupName,  }};TIMResult res = TencentIMSDK.GroupSearchGroups(param, (int code, string desc, List<GroupDetailInfo> result, string user_data)=>{ // Process the async logic});
```

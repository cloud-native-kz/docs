# Поиск участников группы

## Описание функции

Можно выполнять поиск только по локально сохраненным участникам группы, например по списку участников группы или загруженным профилям участников группы.

> **Примечание:** Эта функция недоступна для аудио-видео групп (AVChatRoom), так как участники группы не сохраняются локально.

## Поиск в локальной группе

Вызовите API `GroupSearchGroupMembers` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/GroupApi/GroupSearchGroupMembers.html)) для поиска локального участника группы.
Вы можете установить ключевое слово поиска `group_search_member_params_keyword_list` и указать область поиска, чтобы определить, будут ли выполняться поиски по полям `memberUserID`, `memberNickName`, `memberRemark` и `memberNameCard` участника группы.

В зависимости от того, пуста ли (`null`) переменная `group_search_member_params_groupid_list` параметра `GroupMemberSearchParam` ([Подробнее](https://comm.qq.com/im/doc/unity/en/types/GroupsAttributes/GroupMemberSearchParam.html)) в `GroupSearchGroupMembers`, возможны два случая:

- Если `group_search_member_params_groupid_list` оставлена пустой, выполняется поиск участников во всех группах и результаты возвращаются по `groupID`.
- Если `group_search_member_params_groupid_list` заполнена, выполняется поиск участников в указанной группе.

Пример кода:

```
// Search for group members by the keyword and group IDGroupMemberSearchParam param = new GroupMemberSearchParam{  group_search_member_params_groupid_list = new List<string>  {    "group_id"  },  group_search_member_params_keyword_list = new List<string>  {    "Keyword 1"  },  group_search_member_params_field_list = new List<TIMGroupMemberSearchFieldKey>  {    TIMGroupMemberSearchFieldKey.kTIMGroupMemberSearchFieldKey_Identifier,    TIMGroupMemberSearchFieldKey.kTIMGroupMemberSearchFieldKey_NikeName,    TIMGroupMemberSearchFieldKey.kTIMGroupMemberSearchFieldKey_Remark,    TIMGroupMemberSearchFieldKey.kTIMGroupMemberSearchFieldKey_NameCard,  }};TIMResult res = TencentIMSDK.GroupSearchGroupMembers(param, (int code, string desc, List<GroupGetOnlineMemberCountResult> result, string user_data)=>{ // Process the async logic});
```

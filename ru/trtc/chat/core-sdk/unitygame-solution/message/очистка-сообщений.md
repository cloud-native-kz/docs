# Очистка сообщений

## Описание функции

Можно очищать сообщения в личных и групповых чатах.
При очистке сообщений в беседе все сообщения в этой беседе будут удалены **как локально, так и из облака**, однако сама беседа не будет удалена.

> **Примечание:** **Не** используйте этот API, если вы не хотите удалять сообщения из облака.

Если последнее сообщение удалено, значение `lastMessage` в беседе станет предпоследним сообщением.

### Очистка личных сообщений

Вызовите `MsgClearHistoryMessage` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/MessageApi/MsgClearHistoryMessage.html)) для очистки личных сообщений.

Пример кода:

```
TIMResult res = TencentIMSDK.MsgClearHistoryMessage(conv_id, TIMConvType.kTIMConv_C2C, (int code, string desc, string user_data) => {  // Process the callback logic });
```

### Очистка групповых сообщений

Вызовите `MsgClearHistoryMessage` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/MessageApi/MsgClearHistoryMessage.html)) для очистки групповых сообщений.

Пример кода:

```
 TIMResult res = TencentIMSDK.MsgClearHistoryMessage(conv_id, TIMConvType.kTIMConv_Group, (int code, string desc, string user_data) => {  // Process the callback logic });
```

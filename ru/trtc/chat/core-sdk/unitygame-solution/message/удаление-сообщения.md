#  Удаление сообщения 

## Описание функции

Можно удалять как локальные сообщения, так и облачные сообщения.
При удалении облачных сообщений такие сообщения будут удалены как локально, так и из облака и **не могут быть восстановлены**.

Если удалено последнее сообщение, `lastMessage` в диалоге станет предпоследним сообщением.

### Удаление локального сообщения

Вызовите `MsgDelete` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/MessageApi/MsgDelete.html)) для удаления локального сообщения.

> **Примечание:** После удаления исторического локального сообщения SDK пометит сообщение как удаленное локально, и его больше нельзя будет извлечь с помощью `MsgGetMsgList`. Если приложение удалено и переустановлено, маркер удаления будет потерян локально, и сообщение все еще можно будет извлечь с помощью `MsgGetMsgList`.

Пример кода:

```
public static void MsgDelete()    {      string conv_id = "287646";      MsgDeleteParam message_delete_param = new MsgDeleteParam();      message_delete_param.msg_delete_param_msg = new Message(); // The message to be deleted      message_delete_param.msg_delete_param_is_remble = false; // Delete the local message      TIMResult res = TencentIMSDK.MsgDelete(conv_id, TIMConvType.kTIMConv_C2C, message_delete_param, (int code, string desc, string user_data) => {        // Process the callback logic      });  }
```

### Удаление сообщения из облака

Вызовите `MsgDelete` ([Подробнее](https://comm.qq.com/im/doc/unity/en/api/MessageApi/MsgDelete.html)) для удаления сообщений из облака.

Этот API удаляет сообщения как локально, так и из облака, которые не могут быть восстановлены.

> **Примечание:** За один вызов можно удалить максимум 30 сообщений. Сообщения для удаления за один вызов **должны** быть из одного диалога. Этот API можно вызывать не более одного раза в секунду. Если сообщения были загружены на устройство учетной записью, они останутся на устройстве после вызова API для их удаления из облака; другими словами, удаленные сообщения не синхронизируются.

Пример кода:

```
public static void MsgDelete()    {      string conv_id = "287646";      MsgDeleteParam message_delete_param = new MsgDeleteParam();      message_delete_param.msg_delete_param_msg = new Message(); // The message to be deleted      message_delete_param.msg_delete_param_is_remble = true; // Delete the roaming message      TIMResult res = TencentIMSDK.MsgDelete(conv_id, TIMConvType.kTIMConv_C2C, message_delete_param, (int code, string desc, string user_data) => {        // Process the callback logic      });  }
```

## 1.8.3 @2023.01.13

- Добавлена поддержка добавления нескольких функций callback к событию callback.
- Добавлена поддержка удаления указанных функций callback.
- Добавлены API и callback событий, связанные с группами диалогов.
- Добавлена поддержка установки callback событий для пользовательских данных диалога.
- Добавлены API маркировки диалогов.
- Добавлено расширенное API для получения списка диалогов.
- Добавлены API и callback события, связанные со счетчиками групп.
- Добавлено API для перевода текстовых сообщений.
- Добавлены API и callback события для получения/подписки на общее количество непрочитанных сообщений с фильтром.

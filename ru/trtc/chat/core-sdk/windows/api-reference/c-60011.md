## API групп

Tencent Cloud Chat SDK поддерживает пять предустановленных типов групп, каждый из которых применяется к различным сценариям.

- Рабочая группа (Work): пользователи могут присоединиться к группе только после приглашения членами группы. Этот тип группы аналогичен приватной группе (Private) в более ранних версиях.
- Открытая группа (Public): пользователи могут присоединиться к открытой группе путем запроса, который должен быть одобрен владельцем группы или администратором группы.
- Группа встреч (Meeting): используется совместно с [TRTC](https://trtc.io/document) для обеспечения сценариев видеоконференций и онлайн-обучения. Пользователи могут свободно присоединяться и покидать группу, а также просматривать историю сообщений перед присоединением. Аналогична группе чата (ChatRoom) в более ранних версиях.
- Сообщество: пользователь может свободно присоединиться и покинуть сообщество. Подходит для сценариев чата с очень большим количеством членов сообщества, таких как обмен знаниями и обсуждение игр. Эта функция поддерживается клиентом с SDK улучшенным изданием v5.8.1668 или позже и веб-SDK v2.17.0 или позже. Для использования необходимо приобрести [Pro edition, Pro Plus edition или Enterprise edition](https://trtc.io/buy/chat), а затем включить его в [Console](https://console.trtc.io/), путь: Applications > Your App > Chat > Configuration > Group Configuration > Community.
- Аудио-видео группа (AVChatRoom): аудио-видео группа позволяет пользователям свободно присоединяться и покидать ее, подходит для сценариев прямого эфира и комнат чата с комментариями на экране. На количество членов группы нет ограничений.

# Добавление контакта

В этой статье вы найдете руководство по добавлению контактов в TUIKit.

## Демонстрация

Если вы раньше не добавляли контакты, интерфейс контактов будет пустым. После добавления контактов они будут отображены в списке контактов, как показано ниже:

| Пустой список контактов | Заполненный список контактов |
| --- | --- |
| ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a9bc7ac02d4c11efb8c45254005a8b94.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a997b08f2d4c11efa01d5254005235d8.png) |

## Требования к среде разработки

- Android Studio-Giraffe
- Gradle-7.2
- Android Gradle Plugin Version-7.0.0
- kotlin-gradle-plugin-1.5.31

## Предварительные условия

Перед построением интерфейса убедитесь, что вы выполнили следующие 4 пункта:

1. Создали приложение в консоли.
2. Создали несколько учетных записей пользователей в консоли.
3. Интегрировали `TUIKit` или `TUIContact`.
4. Вызвали API `login` в `TUILogin` для входа в компонент.

> **Примечание:** Все компоненты используют этот API входа. Вы можете входить один раз каждый раз при запуске приложения. Убедитесь, что вход выполнен успешно, и рекомендуется выполнить следующее в колбэке успешного входа.

Если вы не выполнили вышеуказанные 4 шага, сначала обратитесь к соответствующим шагам в разделе [Начало работы](https://www.tencentcloud.com/document/product/1047/60520), иначе вы можете столкнуться с проблемами при реализации следующих функций.

Если вы уже выполнили их, продолжайте читать ниже.

## Инструкции по шагам

Рекомендуем вам работать непосредственно в `TUIKit`, добавляя контакты вручную следующим образом:

1. Выполните шаги из раздела [Построение интерфейса контактов](https://www.tencentcloud.com/document/product/1047/61218), чтобы отобразить интерфейс контактов.
2. Нажмите кнопку `+` в верхнем правом углу интерфейса и в подменю выберите `Add to Contacts`.
3. Введите допустимый userID для поиска пользователя. Вы можете получить допустимый userID со страницы `Account Management` в консоли. Путь к странице: Applications > Your App > Chat > Users > Account Management.
4. Добавьте пользователя в контакты.

Результат показан ниже:

| Нажать [Add to Contacts] | Поиск пользователя | Отправка запроса на добавление в контакты |
| --- | --- | --- |
| ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a99a47ef2d4c11efa4f552540077de32.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a999bb1f2d4c11ef97da5254007d9c55.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a99afdbc2d4c11efa01d5254005235d8.png) |

Обычно после отправки запроса на добавление в друзья получатель принимает его автоматически. Однако если учетная запись получателя имеет установленную `Friend Request Verification`, поведение может отличаться. Путь к конфигурации:

Applications > `Your App` > Chat > Users > Account Management > `Choose an account` > Edit > Friend Request Verification.

Параметры `Friend Request Verification` следующие:

| Значения Friend Request Verification | Значение |
| --- | --- |
| Accept all friend requests | По умолчанию принимать все запросы на добавление в друзья |
| Manually accept or reject friend requests | Получатель должен вручную принять или отклонить запрос на добавление в друзья. |
| Reject all friend requests | Отклонить все запросы на добавление в друзья |

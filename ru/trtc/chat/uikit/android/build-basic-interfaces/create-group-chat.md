# Создание группового чата

Эта статья поможет вам создать группу в TUIKit.

## Требования к среде разработки

- Android Studio-Giraffe
- Gradle-7.2
- Android Gradle Plugin Version-7.0.0
- kotlin-gradle-plugin-1.5.31

## Предварительные условия

Перед созданием интерфейса убедитесь, что вы выполнили следующие 4 пункта:

1. Создали приложение в консоли.
2. Создали несколько учетных записей пользователей в консоли.
3. Интегрировали `TUIKit`.
4. Вызвали API `TUILogin` `login` для входа в компонент.

> **Примечание:** Все компоненты используют этот API для входа. Вы можете входить один раз каждый раз при запуске приложения. Убедитесь, что вход выполнен успешно, и мы рекомендуем выполнить следующие действия в успешном обратном вызове входа.

Если вы еще не выполнили вышеуказанные 4 шага, сначала обратитесь к соответствующим шагам в [Начало работы](https://www.tencentcloud.com/document/product/1047/60520), в противном случае при реализации следующих функций могут возникнуть проблемы.

Если вы их уже выполнили, продолжайте читать ниже.

## Создание группы

Для создания группы вручную в TUIKit необходимо выполнить два предварительных условия:

1. Загрузить список диалогов. Обратитесь к документу: [Построение списка диалогов](https://www.tencentcloud.com/document/product/1047/61216).
2. Добавить некоторые контакты. Обратитесь к документу: [Добавление контактов](https://www.tencentcloud.com/document/product/1047/61220).

Необходимо выполнить еще 3 шага:

1. На экране списка диалогов после загрузки нажмите на значок `+` в верхнем правом углу, чтобы открыть подменю, и выберите `Create Group Chat`.
2. Выберите нескольких участников группы. Эти участники — это контакты, которые вы добавили. Если вы ранее не добавляли никаких контактов, на этом экране никто не будет доступен для выбора.
3. Установите название группового чата, тип, аватар и т. д.

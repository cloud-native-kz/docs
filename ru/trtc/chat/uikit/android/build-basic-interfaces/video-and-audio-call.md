## Предварительные условия

Перед созданием интерфейса убедитесь, что вы выполнили следующие 4 действия:

1. Создали приложение в консоли.
2. Создали учетные записи пользователей в консоли.
3. Интегрировали `TUICallKit`.
4. Вызвали API `login` в `TUILogin` для входа в компонент.

> **Примечание:** Все компоненты используют этот API для входа. Вы можете входить один раз каждый раз при запуске приложения. Убедитесь, что вход выполнен успешно, и мы рекомендуем выполнить следующие действия в обратном вызове успешного входа.

Если вы еще не выполнили указанные 4 шага, сначала обратитесь к соответствующим шагам в [Начало работы](https://trtc.io/document/60520), в противном случае при реализации следующих функций могут возникнуть проблемы.

Если вы уже выполнили их, продолжайте читать ниже.

## Этапы интеграции

1. Войдите в [консоль Chat](https://console.trtc.io/) и активируйте сервис видео и аудио. Подробные инструкции см. в документе: [Активация сервиса](https://trtc.io/document/59832).
2. В диалоговом окне активации сервиса TRTC нажмите "Подтвердить". Система создаст приложение TRTC с тем же SDKAppID, что и текущее приложение Chat в [консоли TRTC](https://trtc.io/login?s_url=https://console.trtc.io/), что позволит повторно использовать учетные записи и аутентификацию.
3. Интегрируйте компонент `TUICallKit`. В файле build.gradle приложения добавьте зависимость для **TUICallKit**:

```
// Integrate the TUICallKit componentapi project(':tuicallkit-kt')
```

После интеграции компонента `TUICallKit` кнопки "Видеозвонок" и "Голосовой звонок" автоматически появятся в интерфейсе чата и на экране профиля контакта. Когда пользователи нажимают на эти кнопки, `TUICallKit` отображает интерфейс приглашения на звонок и отправляет запрос приглашения на звонок другой стороне.

Когда пользователь получает приглашение на звонок в сети, `TUICallKit` автоматически отображает интерфейс приема звонка, позволяя пользователю принять или отклонить звонок.

Когда пользователь получает приглашение на звонок в оффлайне, необходимо использовать возможности оффлайн-уведомлений для пробуждения приложения звонка.

## Добавление оффлайн-уведомлений для push-сообщений

Подробную информацию см. в документации: [Оффлайн push-уведомления для звонков](https://trtc.io/document/50999). После настройки при нажатии на полученное оффлайн push-уведомление для видео- и аудиозвонков `TUICallKit` автоматически откроет интерфейс приглашения на видео/аудиозвонок.

# Настройка смайликов и стикеров

TUIChat позволяет добавлять пользовательские смайлики.

> **Примечание:** TUIChat поддерживает добавление изображений в следующих форматах в качестве пользовательских эмодзи: JPEG, JPG, PNG, BMP. Также поддерживается формат GIF, начиная с версии 7.8.

## Добавление пользовательского смайлика

TUIChat позволяет добавлять пользовательские смайлики из песочницы, директории `assets` и сетевых путей.
В следующем примере показано добавление смайлика `programmer` из директории `assets`.

### Подготовка ресурсов стикеров

Создайте папку `assets` в папке `src/main` приложения и поместите папку смайликов в директорию `assets`:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/31920e415b6d11ee94c3525400d793d0.png)

### Добавление стикера

При запуске приложения вызовите API для добавления пользовательского смайлика в `FaceManager`:
Каждый стикер имеет уникальный `faceGroupID`, а каждый смайлик в стикере имеет `faceKey`. После добавления стикера в `FaceManager` стикеры будут отсортированы по `faceGroupID` на UI ввода **Другие смайлики**.

```
public class DemoApplication extends Application {    @Override    public void onCreate() {        FaceGroup programmerGroup = new FaceGroup();        // Количество смайликов, отображаемых на строку на UI ввода **Другие смайлики**        programmerGroup.setPageColumnCount(5);        // Миниатюра стикера        programmerGroup.setFaceGroupIconUrl("file:///android_asset/programmer/programmer00@2x.png");        // Название стикера        programmerGroup.setGroupName("programmer");        for (int i = 0; i < 16; i++) {            CustomFace customFace = new CustomFace();            String index = "" + i;            if (i < 10) {                index = "0" + i;            }            // Поместите смайлики в директорию `assets`. Если используется путь песочницы или сетевой путь, можно использовать метод `setFaceUrl`.            customFace.setAssetPath("programmer/programmer" + index + "@2x.png");            // `key` смайлика            String faceKey = "programmer" + index;            customFace.setFaceKey(faceKey);            // Ширина смайлика на UI ввода **Другие смайлики**            customFace.setWidth(170);            // Высота смайлика на UI ввода **Другие смайлики**            customFace.setHeight(170);            programmerGroup.addFace(faceKey, customFace);        }        // Зарегистрируйте стикер в `FaceManager`. `FaceGroupID` равен `1`.        FaceManager.addFaceGroup(1, programmerGroup);    }}
```

### Эффект после успешного добавления

Успешно добавленный стикер отображается на UI ввода **Другие смайлики** в интерфейсе чата.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/41e9afdee1af11ee9f745254008eb8a8.png)

> **Внимание:** `faceGroupID` — это целое число больше 0 и должно быть уникальным.

## Отправка пользовательского смайлика

После добавления пользовательского смайлика он будет отображаться на UI ввода **Другие смайлики** и может быть отправлен одним щелчком.

Сообщение со смайликом также можно создать с помощью кода и отправить, например:

```
V2TIMMessage v2TIMMessage = V2TIMManager.getMessageManager()        .createFaceMessage(faceGroupID, faceKey.getBytes());V2TIMManager.getMessageManager().sendMessage(v2TIMMessage,        userID,        null,        V2TIMMessage.V2TIM_PRIORITY_DEFAULT,        false,        null,        new V2TIMSendCallback<V2TIMMessage>() {...}
```

## Парсинг пользовательского смайлика

После получения сообщения с пользовательским смайликом TUIKit выполнит парсинг `V2TIMMessage` из IM SDK в тип `FaceMessageBean`. `FaceMessageBean` может быть использован для получения `faceGroupID` и `faceKey` пользовательского смайлика.

```
TUIMessageBean messageBean = ChatMessageParser.parseMessage(v2TIMMessage);FaceMessageBean faceMessageBean = null;if (messageBean instanceof FaceMessageBean) {    faceMessageBean = (FaceMessageBean) messageBean;}if (faceMessageBean != null) {    int faceGroupID = faceMessageBean.getIndex();    String faceKey = null;    if (faceMessageBean.getData() != null) {        faceKey = new String(faceMessageBean.getData());    }}
```

## Рендеринг пользовательского смайлика

### Вызов существующего API для рендеринга

После получения `faceGroupID` и `faceKey` пользовательского смайлика можно вызвать метод `loadFace` класса `FaceManager` для загрузки их в переданное значение `imageView`:

```
FaceManager.loadFace(faceGroupID, faceKey, imageView);
```

### Пользовательский рендеринг

Также можно использовать `faceGroupID` и `faceKey` смайлика для получения реального URL смайлика из `FaceManager` для пользовательского рендеринга, например:

```
String faceUrl = "";List<FaceGroup> faceGroupList = FaceManager.getFaceGroupList();for(FaceGroup faceGroup : faceGroupList) {    if (faceGroup.getGroupID() == faceGroupID) {        ChatFace face = faceGroup.getFace(faceKey);        if (face != null) {            faceUrl = face.getFaceUrl();        }    }}// load faceUrl into view
```

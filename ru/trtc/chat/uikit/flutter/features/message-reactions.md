# Реакции на сообщения

## Обзор функции

TUIKit поддерживает реакции эмодзи, позволяя пользователям отвечать на сообщения эмодзи. Эта функция построена на основе возможности редактирования сообщений.

> **Примечание:** Реакции эмодзи доступны только в пакетах Pro edition plus и Enterprise edition. Чтобы использовать эту функцию, [приобретите Pro-plus edition или Enterprise edition](https://console.trtc.io/subscription/buy/chat?packType=pro).

## Демонстрация функции

### Отправка реакций эмодзи

Длительно нажмите на сообщение, чтобы открыть меню сообщения, где можно выбрать эмодзи для реакции. Нажмите кнопку справа, чтобы развернуть и просмотреть больше вариантов эмодзи. Чтобы добавить реакцию, нажмите на эмодзи. Если вы уже отреагировали этим эмодзи, повторный клик удалит вашу реакцию.

### Отображение реакций эмодзи

Все реакции эмодзи на сообщение отображаются ниже этого сообщения, и все участники беседы могут их видеть.

Число рядом с каждым эмодзи указывает, сколько пользователей отреагировали этим эмодзи. Клик на эмодзи открывает диалоговое окно с подробностями.

В диалоговом окне подробностей можно просмотреть список пользователей, которые отреагировали каждым эмодзи. Если вы отреагировали, вы можете нажать, чтобы удалить свою реакцию.

## Включение или отключение реакций эмодзи

Параметр `config` в `MessageList` позволяет включать или отключать реакции эмодзи для сообщений. По умолчанию эта функция включена (`true`). Чтобы отключить её, установите значение `false`. Подробнее об использовании компонента `MessageList` см. в [демонстрации](https://github.com/Tencent-RTC/TUIKit_Flutter/blob/main/chat/uikit/lib/chat_page.dart).

```
MessageList(  config: ChatMessageListConfig(isSupportReaction: false),),
```

### Chat Flutter UIKit V2 - 1.3.0 @2024.04.17

#### Основное

- Улучшена функция `initUIKit` несколькими улучшениями, упрощающими процесс конфигурации и повышающими общую удобство использования. Ключевые обновления включают:
  - Новый параметр `components`, который консолидирует конфигурации, связанные с компонентами, включая требуемый `usedComponentsRegister` для ручного объявления используемых компонентов. Он также позволяет дополнительные глобальные конфигурации, построители, контроллеры и обработчики событий для каждого компонента, влияющие на все экземпляры каждого компонента.
  - Новый обратный вызов `onTencentCloudChatSDKEvent` в параметре `callbacks` обрабатывает события, связанные с SDK, заменяя предыдущий `sdkListener` из `options`.
  - Параметр `config` теперь сосредоточен на глобальных конфигурациях для UIKit, удаляя `usedComponentsRegister` и `preloadDataConfig`. `usedComponentsRegister` был перемещён в параметр `components`.
  - Удалено требование передачи `context`.
- Введён новый менеджер для каждого компонента, названный добавлением `Manager` к имени компонента (например, `TencentCloudChatMessageManager`), обеспечивающий следующие функции для лучшей и более лёгкой интеграции:
  - `register`: [Ручное объявление использования каждого компонента](https://pub.dev/packages/tencent_cloud_chat#basic-usage) при вызове `initUIKit`.
  - `controller`: [Управление каждым компонентом](https://pub.dev/packages/tencent_cloud_chat#global-taking-control-of-each-component) в глобальном масштабе.
  - `eventHandlers`: [Обработка событий уровня компонента](https://pub.dev/packages/tencent_cloud_chat#global-handling-component-level-events) в глобальном масштабе.
  - `builder`: [Динамическое обновление построителей пользовательского интерфейса](https://pub.dev/packages/tencent_cloud_chat#dynamically-updating-ui-builders) для всех экземпляров.
  - `config`: [Конфигурирование компонентов](https://pub.dev/packages/tencent_cloud_chat#global-configuring-components) для всех экземпляров динамически.
- Миграция `register` из `экземпляра` каждого компонента в `менеджер`, как описано в предыдущем пункте.
- Уточнена основная структура хранения данных и производительность, подготавливая почву для будущих улучшений функций.

#### Сообщение (TencentCloudChatMessage)

- Сокращено количество перестройки элементов списка сообщений для повышения производительности.

- В соответствии с этим наше примерное приложение в репозитории GitHub также было перенесено на этот новый подход. Если вы хотите перейти на этот новый подход, см. наше [репо примерного приложения](https://github.com/TencentCloud/chat-demo-flutter).

### Chat Flutter UIKit V2 - 1.1.0 @2024.1.23

Полностью переработанная и модернизированная версия UIKit официально выпущена. По сравнению с предыдущей версией UIKit, есть несколько значительных улучшений и оптимизаций, которые включают:

1. **Персонализация темы**: Переключайтесь между **светлым и тёмным режимом** на лету или персонализируйте собственную тему с нашим богатым набором опций.
2. **Интернационализация**: Мы добавили поддержку большего количества языков, включая арабский, и представили **пользовательский интерфейс Ближнего Востока**. Наши мощные и удобные [инструменты локализации](https://pub.dev/packages/tencent_cloud_chat_intl) делают [персонализацию конфигурации локализации и перевода](https://pub.dev/packages/tencent_cloud_chat_intl) проще, чем когда-либо, и обеспечивают локализированный пользовательский опыт.
3. **Улучшения производительности**: Мы внесли значительные улучшения в **производительность списка сообщений** и представили эффективные и точные возможности позиционирования сообщений.
4. **Поддержка мультимедиа**: Испытайте улучшенную обработку мультимедиа и сообщений с файлами с непрерывным воспроизведением голосовых сообщений и **проведением пальцем по предпросмотрам мультимедийных сообщений**.
5. **Детальные оптимизации**: Мы добавили многочисленные детальные оптимизации, включая богатую анимацию, хаптическую обратную связь и уточненные интерфейсы для повышения пользовательского опыта.
6. **Новые функции**: Наслаждайтесь новыми функциями, такими как аватар в стиле сетки, переработанная панель перенаправления, селектор участников группы и новое меню долгого нажатия сообщения.
7. **Модульные пакеты**: Компоненты разбиты на **модульные пакеты**, позволяя импорту по требованию и сокращая ненужное заполнение.
Каждый модульный пакет поддерживает встроенную навигацию. Например, вы можете автоматически перейти от разговора к сообщению, чтобы начать чат,
без необходимости вручную создавать экземпляры нескольких страниц и обрабатывать переходы самостоятельно. Это значительно упрощает сложность разработки и интеграции.
8. **Дизайн для разработчиков**: Мы представили более унифицированный, стандартизированный дизайн параметров компонентов, более чёткие имена кода и более подробные комментарии для облегчения разработки и повышения эффективности.

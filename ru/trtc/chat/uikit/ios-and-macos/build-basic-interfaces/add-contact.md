# Добавление контакта

Эта статья проведет вас через процесс добавления контактов в TUIKit.

## Эффект отображения

Если вы не добавляли контакты заранее, загруженный интерфейс контактов будет пустым. После добавления контактов они будут отображаться в списке интерфейса, как показано ниже:

| Список контактов пуст | Список контактов не пуст |
| --- | --- |

## Требования к среде

- Xcode 10 или более поздняя версия
- iOS 9.0 или более поздняя версия

## Предварительные условия

Перед построением интерфейса убедитесь, что вы выполнили следующие 4 шага:

1. Создали приложение в консоли.
2. Создали некоторые учетные записи пользователей в консоли.
3. Интегрировали `TUIKit` или `TUIContact`.
4. Вызвали API `login` в `TUILogin` для входа в компонент.

> **Примечание:** Все компоненты используют этот API для входа. Вы можете входить один раз каждый раз при запуске приложения. Убедитесь, что вход выполнен успешно, и мы рекомендуем выполнить следующие действия в обратном вызове успешного входа.

Если вы еще не выполнили указанные выше 4 шага, обратитесь сначала к соответствующим шагам в [Начало работы](https://www.tencentcloud.com/document/product/1047/60521), иначе вы можете столкнуться с препятствиями при реализации следующих функций.

Если вы уже их выполнили, продолжайте чтение ниже.

## Инструкции по шагам

Мы рекомендуем вам работать непосредственно в `TUIKit`, вручную добавляя контакты следующим образом:

1. Следуйте шагам в [Построение интерфейса контактов](https://www.tencentcloud.com/document/product/1047/61215#) для отображения интерфейса контактов.
2. Нажмите кнопку `+` в правом верхнем углу интерфейса и в подменю выберите `Add to Contacts`.
3. Введите действительный userID для поиска пользователя. Действительный userID можно получить на странице `Account Management` в консоли. Путь страницы: Applications > Your App > Chat > Users > Account Management.
4. Добавьте пользователя в контакты.

Эффект показан ниже:

| Нажмите `Add to Contacts` | Поиск пользователя | Отправка запроса контакта |
| --- | --- | --- |
| ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c125c69b2c7411efb0275254006c0558.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c12dfa972c7411efa4f552540077de32.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c1258cf92c7411efa01d5254005235d8.png) |

Обычно после отправки запроса на добавление в друзья получатель автоматически принимает запрос. Однако если учетная запись получателя настроила `Friend Request Verification`, поведение может отличаться. Путь конфигурации:

Applications > `Your App` > Chat > Users > Account Management > `Choose an account` > Edit > Friend Request Verification.

Значения `Friend Request Verification` следующие:

| Значения Friend Request Verification | Значение |
| --- | --- |
| Accept all friend requests | По умолчанию принимать все запросы на добавление в друзья |
| Manually accept or reject friend requests | Требуется от получателя запроса вручную согласиться или отклонить запрос на добавление в друзья. |
| Reject all friend requests | Отклонить все запросы на добавление в друзья |

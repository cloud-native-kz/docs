# Создание группового чата

Эта статья покажет вам, как создать группу в TUIKit.

## Эффект отображения

После создания группового чата вы можете начать отправлять сообщения и взаимодействовать в группе. Если в этот момент вернуться к списку диалогов, вы найдете только что созданный групповой чат:

## Требования к среде

- Xcode 10 или выше
- iOS 9.0 или выше

## Предварительные условия

Перед построением интерфейса убедитесь, что вы выполнили следующие 4 действия:

1. Создали приложение в консоли.
2. Создали несколько учетных записей пользователей в консоли.
3. Интегрировали `TUIKit`.
4. Вызвали API `TUILogin` `login` для входа в компонент.

> **Примечание:** Все компоненты используют этот API для входа. Вы можете выполнять вход один раз при каждом запуске приложения. Убедитесь, что вход выполнен успешно, и рекомендуется выполнить следующие действия в обратном вызове успешного входа.

Если вы не выполнили вышеуказанные 4 шага, сначала обратитесь к соответствующим шагам в [Начало работы](https://www.tencentcloud.com/document/product/1047/60521), иначе вы можете столкнуться с трудностями при реализации следующих функций.

Если вы уже выполнили их, продолжайте читать ниже.

## Создание группы

Существует два предварительных условия для ручного создания группы в TUIKit:

1. Загрузка списка диалогов. Обратитесь к документации: [Построение списка диалогов](https://www.tencentcloud.com/document/product/1047/61217#).
2. Добавление некоторых контактов. Обратитесь к документации: [Добавление контактов](https://www.tencentcloud.com/document/product/1047/61221#).

Нужно выполнить еще 3 шага:

1. На загруженном интерфейсе списка диалогов нажмите значок `+` в верхнем правом углу, чтобы открыть подменю, и выберите `Create Group Chat`.
2. Выберите несколько участников группы. Эти участники — это контакты, которые вы добавили. Если вы ранее не добавили никаких контактов, на этом экране никто не будет доступен для выбора.
3. Установите имя группового чата, тип, аватар и т. д.

# Список контактов

В этой статье вы узнаете, как создать интерфейс для работы с контактами.

## Эффект отображения

Если вы предварительно не добавили контакты, загруженный интерфейс контактов будет пустым. После добавления контактов они будут отображены в списке интерфейса, как показано ниже:

## Требования к окружению

- Xcode 10 или более поздняя версия
- iOS 9.0 или более поздняя версия

## Предварительные условия

Перед созданием интерфейса убедитесь, что вы выполнили следующие 4 шага:

1. Создали приложение в консоли.
2. Создали несколько учетных записей пользователей в консоли.
3. Интегрировали `TUIKit` или `TUIContact`.
4. Вызвали API `login` компонента `TUILogin` для входа в компонент.

> **Примечание:** Все компоненты используют этот API входа. Просто входите один раз при каждом запуске приложения. Убедитесь в успешном входе перед продолжением; мы рекомендуем выполнить следующие действия в обратном вызове при успешном входе.

Если вы не выполнили вышеупомянутые 4 шага, сначала обратитесь к соответствующим шагам в разделе [Начало работы](https://www.tencentcloud.com/document/product/1047/60521), в противном случае при реализации следующих функций могут возникнуть проблемы.

Если вы уже выполнили их, продолжайте чтение ниже.

## Пошаговые инструкции

Чтобы отобразить интерфейс контактов, просто создайте объект `TUIContactController` и отобразите его.

Функции интерфейса контактов распределены следующим образом:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ac9b165e2c7411ef9130525400bf8054.png)

`TUIContact` по умолчанию обрабатывает действия клика на этом интерфейсе следующим образом:

| Действие | Эффект |
| --- | --- |
| Нажмите на `New Contacts` | Показать ожидающие запросы на добавление в друзья |
| Нажмите на `Group Chats` | Показать все групповые чаты текущей вошедшей в систему учетной записи |
| Нажмите на `Blocked List` | Показать список заблокированных контактов текущей вошедшей в систему учетной записи |
| Нажмите на аватар контакта | Перейти в интерфейс управления контактами |
| Нажмите на значок `+` в верхней правой части интерфейса | Всплывающее меню с `Add to Contacts` и `Add Group` |

Вы можете настроить поведение при нажатии на аватар контакта, нажатии `Add to Contacts` и нажатии на `Group Chats`, реализовав методы в `TUIContactControllerListener`

Классическая версия

Objective-C

protocol TUIContactControllerListener_Minimalist: AnyObject {    func onSelectFriend(_ cell: TUICommonContactCell_Minimalist) -> Bool    func onAddNewFriend(_ cell: TUICommonTableViewCell) -> Bool    func onGroupConversation(_ cell: TUICommonTableViewCell) -> Bool}

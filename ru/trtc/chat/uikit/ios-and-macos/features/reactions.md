# Реакции

## Описание

Начиная с версии 7.8 функция ответа с помощью эмодзи предоставляется плагином TUIEmojiPlugin.

- Если вам не требуется эта функция, просто пропустите интеграцию плагина. В этом случае при длительном нажатии на текстовое сообщение модуль ответа с эмодзи не будет отображаться.
- Если вам нужна эта функция, необходимо интегрировать TUIChat и TUIEmojiPlugin. Методы интеграции см. в разделе ["Интеграция базовых функций"](https://www.tencentcloud.com/document/product/1047/50056?lang=en&pg=). После интеграции дополнительные настройки не требуются. При длительном нажатии на текстовое сообщение кнопка ответа с эмодзи будет отображаться автоматически.

> **Примечание:** Функция ответа с эмодзи поддерживается только версиями Pro Plus и Enterprise. Пожалуйста, [приобретите издание Pro Plus или Enterprise](https://console.trtc.io/subscription/buy/chat?packType=pro) для использования этой функции.

## Эффект

### Отправка ответа с эмодзи

После интеграции функции ответа с эмодзи над меню длительного нажатия для сообщений появится область выбора эмодзи. Вы можете нажать кнопку справа, чтобы развернуть область и просмотреть больше эмодзи. Нажатие на эмодзи отправит ответ на сообщение этим эмодзи. Если вы уже ответили на сообщение эмодзи, нажатие на эмодзи отменит ответ.

| Меню длительного нажатия для сообщений | Больше эмодзи |
| --- | --- |
| ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1a23fa48167c11efb8ef5254002fd0a8.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a0e5ff89dc7211ee8c73525400ea3514.jpeg) |

Если функция ответа с эмодзи не интегрирована, запись для ответа с эмодзи не будет отображаться над всплывающим меню длительного нажатия для сообщений, как показано на рисунке:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2852d5a7167c11efb6495254005ac0ca.png)

### Отображение ответа с эмодзи

Все полученные ответы на сообщение будут отображаться под сообщением, и их смогут видеть все участники чата. Под сообщением будут отображаться эмодзи ответа и ник участника чата, который ответил этим эмодзи. Нажмите на эмодзи или ник, и вы увидите детали ответа с эмодзи для этого сообщения. В интерфейсе детализации ответа с эмодзи нажатие на отправленный вами ответ с эмодзи позволит быстро отозвать ответ с эмодзи.

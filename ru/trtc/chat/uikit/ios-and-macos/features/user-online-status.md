# Статус пользователя онлайн

## Описание

TUIKit поддерживает отображение статуса пользователя онлайн, начиная с версии [6.5.2803](https://www.tencentcloud.com/document/product/1047/34282?lang=en&pg=#6.5.2803-.402022.07.15---enhanced-edition).

Если функция "Показывать статус пользователя онлайн" включена, статус пользователя онлайн будет отображаться на аватаре каждого пользователя в списке чатов и списке контактов. Зелёный кружок указывает на онлайн-статус; отсутствие зелёного кружка указывает на оффлайн-статус.

Если функция "Показывать статус пользователя онлайн" отключена, статус пользователя онлайн отображаться не будет.

> **Примечание:** Функция "Статус пользователя онлайн" поддерживается только [Pro-изданием, Pro Plus-изданием, Enterprise-изданием](https://www.tencentcloud.com/document/product/1047/34350?lang=en&pg=). Пожалуйста, убедитесь, что премиум-пакет активирован перед использованием этой функции. Функция "Статус пользователя онлайн" требует включения переключателя статуса пользователя в [консоли Chat](https://console.trtc.io/). Пожалуйста, убедитесь, что переключатель включен перед использованием этой функции. Только друзья в списке разговоров поддерживают отображение их статуса онлайн. Если вам нужна поддержка не-друзей, вы должны вызвать API для подписки на статус не-друзей и реализовать пользовательский интерфейс самостоятельно. Справка по API: [Статус пользователя](https://trtc.io/document/49562?platform=ios%20and%20macos&product=chat&menulabel=coresdk#change-in-a-non-friend-user&).

## Включение статуса пользователя онлайн

В компоненте `TUICore`, в файле `TUIConfig`, предоставляется переключатель для функции "Статус пользователя онлайн" с названием `displayOnlineStatusIcon`. Его тип — BOOL, значение по умолчанию — NO.

Чтобы включить отображение статуса пользователя онлайн в списке чатов, сначала подпишитесь на премиум-пакет, затем включите переключатель функции статуса пользователя в [консоли Chat](https://console.trtc.io/) и измените значение `displayOnlineStatusIcon` по умолчанию на `YES`, или вызовите следующий метод перед инициализацией страницы чата.

Swift

Objective-C

```
TUIConfig.default().displayOnlineStatusIcon = true
```

```
[TUIConfig defaultConfig].displayOnlineStatusIcon = YES;
```

## Часто задаваемые вопросы

- **При вызове API подписки/отписки API возвращает код ошибки "72001".**

Код ошибки 72001 указывает на то, что соответствующая функция не была активирована в консоли. Пожалуйста, войдите в [консоль Chat](https://console.trtc.io/) и включите соответствующий переключатель функции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0a9d068e135811efa2935254005ac0ca.png)

- **Ошибка: пакет не поддерживает использование этого API, пожалуйста, обновитесь до премиум-пакета.**

Функция "Статус пользователя онлайн" поддерживается только премиум-пакетом. Это сообщение об ошибке указывает на то, что ваш текущий пакет не поддерживает эту функцию. Пожалуйста, войдите на [страницу покупки Chat](https://console.trtc.io/buy/active), чтобы активировать премиум-пакет и испробовать его.

# Введение в TUIKit

## Обзор TUIKit

TUIKit — это библиотека компонентов пользовательского интерфейса, созданная на основе Tencent Cloud Chat SDK. Она предоставляет универсальные компоненты пользовательского интерфейса, включая функции для работы с беседами, чатом и группами. Благодаря тщательно продуманным компонентам пользовательского интерфейса вы можете быстро создавать элегантные, надежные и масштабируемые приложения для обмена сообщениями. UIKit, разработанный на основе React Native, предлагает стиль интерфейса, лучше соответствующий привычкам использования зарубежных клиентов, и поддерживает интернационализацию. Мы приглашаем вас к интеграции.

## Компоненты TUIKit

TUIKit предоставляет следующие компоненты пользовательского интерфейса: ConversationList, Chat, ChatSetting и Contact. Каждый из этих компонентов отвечает за отображение различного содержимого.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/314a8601a26911efa0b3525400bdab9d.png)

## Функции Chat

Chat главным образом отвечает за отображение интерфейса сообщений. Вы можете использовать его для прямой отправки различных типов сообщений, включая текст/эмодзи/изображения/видео/файлы/пользовательские сообщения. Вы также можете использовать его для пересылки/удаления/цитирования сообщений путем длительного нажатия на сообщение, запроса деталей квитанции о прочтении сообщения и многого другого.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/31479b27a26911efaaca525400fdb830.png)

> **Примечание:** В целях соблюдения авторских прав на дизайн эмодзи проект Chat Demo/TUIKit не включает вырезки больших элементов эмодзи. Пожалуйста, замените их на ваши собственные дизайны или другие наборы эмодзи, на которые у вас есть авторские права, перед официальным запуском в коммерческое использование. **Показанный ниже набор эмодзи со смайликом по умолчанию защищен авторским правом Tencent RTC**, вы можете перейти на [Chat Pro Plus Edition и Enterprise Edition](https://console.trtc.io/subscription/buy/chat?packType=pro), чтобы использовать его бесплатно.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3135e50aa26911ef992f52540075b605.png)

## Функции ConversationList

ConversationList отвечает за отображение и редактирование списка бесед, включая закрепление бесед, отключение уведомлений о беседах, удаление бесед и т. д.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3149c4fca26911ef834b525400f69702.png)

## Функции ChatSetting

ChatSetting отвечает за управление профилями групп, членами групп и разрешениями групп.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3143ca05a26911efa04c5254002693fd.png)

## Функции Contact

Contact отвечает за отображение контактов и настройки разрешений.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/314a88c4a26911efa0b3525400bdab9d.png)

## Функции Profile

Profile отвечает за управление профилем пользователя.

# Оплата использования Chat сверх лимита пакета

## Квота свободных ежемесячных активных пользователей (MAU)

- MAU: Свободные ежемесячные активные пользователи
- Способ расчета: После вызова операции входа в Chat SDK и установки долгосрочного соединения с бэкендом Chat значение MAU увеличивается на 1. То есть один пользователь, вошедший в Chat в течение месяца, считается 1 MAU. Повторные входы одного пользователя не накапливаются в MAU.
- Детали конкретной стоимости показаны в таблице ниже:

| Описание | Free Trial | Standard | Pro | Pro-plus | Enterprise |
| --- | --- | --- | --- | --- | --- |
| Квота свободных ежемесячных активных пользователей (MAU) | 100/месяц | 10 000/месяц | 10 000/месяц | 25 000/месяц | 50 000/месяц |
| Плата за превышение MAU сверх плана | Не поддерживается. | 0,015 USD/MAU/месяц |  |  |  |

> **Примечание:** Используйте операцию входа Chat SDK в соответствии с вашим бизнес-сценарием, чтобы избежать высокого MAU. С 1 апреля 2026 года **плата за превышение MAU сверх плана** будет скорректирована. Дополнительную информацию об эффективных правилах и ценах см. в [официальном объявлении](https://www.tencentcloud.com/en/announce/detail/100941).

## Облачная отправка

Обеспечивает стабильные, своевременные и разнообразные услуги отправки. Плагин отправки поддерживает обычную отправку сообщений и отправку всем членам/по тегам, предлагая полный запрос жизненного цикла отправки, статистику данных и услугу устранения неполадок.

| Описание | Free Trial | Платная версия облачной отправки |
| --- | --- | --- |
| Свободная квота в пределах пакета | Каждый SDKAppID имеет две бесплатные пробные возможности для облачной отправки. Каждая бесплатная пробная версия действительна в течение 7 дней. Во время бесплатного пробного периода нет ограничений на количество вызовов API отправки всем членам/по тегам и количество отправок всем членам/по тегам. | Количество вызовов API отправки всем членам/по тегам: 100 раз/деньКоличество отправок всем членам/по тегам: 30 млн записей/месяц |

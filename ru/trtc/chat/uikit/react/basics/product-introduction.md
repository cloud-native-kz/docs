# Введение в продукт

Chat предоставляет многоплатформенные API для чата, компоненты UI, серверные API и вебхуки, позволяя вам создать полнофункциональное приложение для чата всего за десять минут.

Вы можете напрямую испытать демонстрацию Chat UIKit ниже. Кроме того, вы можете быстро изучить реализацию кода в интернете, [**попробовав нашу песочницу для чата**](https://codesandbox.io/p/sandbox/trtc-chat-full-48sffc?from-embed=).

Chat Component отвечает за отображение пользовательского интерфейса сообщений. Вы можете использовать его для прямой отправки различных типов сообщений, включая текст, эмодзи, изображения, видео, файлы и пользовательские сообщения. Chat также поддерживает переадресацию сообщений/отзыв/цитирование/поиск/получение подтверждения о прочтении сообщений.

> **Примечание:** Чтобы уважать авторские права на дизайн эмодзи, проект Chat Demo/TUIKit не включает вырезки больших элементов эмодзи. Пожалуйста, замените их на свои собственные дизайны или другие пакеты эмодзи, на которые у вас есть авторские права, перед официальным запуском для коммерческого использования. **Пакет эмодзи со смайликом по умолчанию, показанный ниже, защищен авторским правом Tencent RTC**, вы можете перейти на [Chat Pro Plus Edition и Enterprise Edition](https://console.trtc.io/subscription/buy/chat?packType=pro), чтобы использовать его бесплатно.![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0a25b7e497f711efa04c5254002693fd.png)

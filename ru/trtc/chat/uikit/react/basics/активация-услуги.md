# Активация услуги

В этой статье описано, как [активировать Trial Edition бесплатно](#free), [приобрести официальную версию](#official) Chat, а также [обновить официальную версию](#renew), [перейти на официальную версию](#upgrade) и [включить автоматическое продление](#autorenew). На основе [описания цен и функций версий Chat](https://trtc.io/zh/document/67650?product=chat&menulabel=uikit&platform=react#function) вы можете выбрать необходимую версию и следовать приведённому ниже руководству для бесплатной активации, покупки, продления и обновления официальной версии Chat.

## Включение пробной услуги

С помощью услуги Chat вы можете быстро интегрировать мгновенные сообщения для реализации различных функций обмена сообщениями и управления группами.

Чтобы помочь вам лучше ознакомиться с функциями Chat, мы предоставляем Trial Edition на 1 месяц для каждого SDKAppID (Trial Edition не включает бесплатную длительность звонков). Каждый SDKAppID предлагает бесплатную квоту MAU в размере 100 и позволяет 2 бесплатных пробных периода, каждый с действительностью 1 месяц. При этом общее время пробных версий для всех SDKAppID под одной учётной записью ограничено 10.

1. Войдите в [консоль Tencent RTC](https://console.trtc.io) и нажмите **Create Application**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/8ac73a67c6ac11f08942525400e889b2.png)

2. В появившемся окне введите **название приложения**, выберите **Chat**, выберите подходящий **регион развёртывания** и нажмите **Create** для завершения настройки пробной версии.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/bb9befd4c6ac11f0a4a55254001c06ec.png)

3. После завершения создания приложения вы по умолчанию попадёте на страницу сведений о продукте Chat. На данный момент вы быстро создали приложение и успешно получили бесплатную Trial Edition Chat. Вы можете просмотреть информацию на текущей [**странице сведений о продукте Chat**](https://console.trtc.io/chat) или [**Мои приложения**](https://console.trtc.io/app) и обратитесь к [руководству интеграции](https://trtc.io/document/50056?product=chat&menulabel=uikit&platform=ios%20and%20macos) для интеграции. `SDKAppID` и `SDKSecretKey` здесь будут использоваться в руководстве интеграции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c89ebcf5c6ad11f0a7775254005ef0f7.png)

### Получение UserSig

UserSig — это подпись безопасности, создаваемая на основе SDKAppID, SDKSecretKey и ID пользователя для проверки личности пользователя, аналогично учётным данным входа.

При использовании базовых облачных услуг SDKAppID, UserID и UserSig требуются при инициализации SDK или входе.

В формальной рабочей среде для обеспечения безопасности бизнеса вам необходимо развернуть **генератор UserSig на сервере** для создания UserSig для аутентификации. Подробнее см. в разделе [Аутентификация UserSig](https://trtc.io/zh/document/34385?product=chat&menulabel=uikit&platform=react).

Однако на этом этапе в целях тестирования и ознакомления с продуктом см. следующую процедуру получения UserSig для тестирования из консоли.

> **Примечание:** UserSig, полученный из консоли, имеет период действия по умолчанию 180 дней.

Процесс получения UserSig из консоли выглядит следующим образом:

1. Перейдите в [консоль Tencent RTC](https://console.trtc.io), выберите **Development Tools >**[**UserSig Tools**](https://console.trtc.io/usersig) на левой боковой панели.
2. В модуле **UserSig generation** слева нажмите на выпадающий список и выберите **созданное приложение (SDKAppID)**. После завершения соответствующий **SecretKey (Key)** будет автоматически создан.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0cc273a0c6ae11f087ad52540099c741.png)

3. Введите имя пользователя (UserID).
4. Нажмите **Generate** для создания соответствующей подписи UserSig.

## Покупка официальных версий

**Вам необходимо приобрести Chat Monthly Package для использования Chat**. Подробнее о цене и сравнении функций Chat Monthly Package см. в разделе [Описание функций и выставления счетов версий Chat](https://trtc.io/zh/document/67650?product=pricing). **Чтобы приобрести, выполните следующие действия:**

1. Перейдите на [страницу покупки Chat](https://console.trtc.io/subscription/buy/chat), выберите подходящий **регион развёртывания**, выберите **приложение (SDKAppID) и версию для покупки**, **рекомендуется включить автоматическое продление, чтобы не повлиять на использование услуги**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Subscribe now**.

> **Примечание:** Для покупки версии Enterprise Edition [свяжитесь с нами](https://trtc.io/zh/contact).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e028ed8ac6ae11f0a7775254005ef0f7.png)

2. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [руководство интеграции](https://trtc.io/document/50056?product=chat&menulabel=uikit&platform=ios%20and%20macos) для интеграции.

## Продление официальной версии

1. Перейдите на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat), нажмите **Renewal / Upgrade**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/298890a6c6b011f0a93d52540044a08e.png)

2. Перейдите на [страницу покупки Chat](https://console.trtc.io/subscription/buy/chat), выберите подходящий **регион развёртывания**, подтвердите приложение (SDKAppID) для покупки, выберите версию того же пакета приложения, **рекомендуется включить автоматическое продление, чтобы не повлиять на использование бизнеса**. Когда эта функция включена, при **достаточном балансе счёта будет автоматически продлено ежемесячно по истечении срока**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Subscribe now**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/c022660fc6b011f0a4a55254001c06ec.png)

3. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [руководство интеграции](https://trtc.io/document/50056?product=chat&menulabel=uikit&platform=ios%20and%20macos) для интеграции.

## Обновление до официальной версии

1. Перейдите на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat), нажмите **Renewal / Upgrade**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dcf96991c6b011f0a7775254005ef0f7.png)

2. Перейдите на [страницу покупки Chat](https://console.trtc.io/subscription/buy/chat), выберите подходящий **регион развёртывания**, подтвердите приложение (SDKAppID) для обновления, выберите **версию выше, чем пакет приложения**, подтвердите информацию о покупке и согласитесь с соответствующим соглашением, затем нажмите **Subscribe now**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2da972e3c6b111f0a93d52540044a08e.png)

3. Перейдите на страницу оплаты для завершения платежа. После завершения покупки вы можете перейти на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat) или [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [руководство интеграции](https://trtc.io/document/50056?product=chat&menulabel=uikit&platform=ios%20and%20macos) для интеграции.

## Автоматическое продление

Конкретные шаги для включения автоматического продления в консоли выглядят следующим образом:

1. Перейдите на [**страницу сведений о продукте Chat**](https://console.trtc.io/chat).
2. Выберите **приложение, для которого требуется автоматическое продление**.
3. В информации о продукте Chat нажмите кнопку **Enable** рядом с **Auto-renewable**. Появится диалоговое окно подтверждения. Нажмите **Enable**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b272b7e5c6b111f087ad52540099c741.png)

### Функции группы

На основе распространенных вариантов использования Chat предоставляет следующие типы групп по умолчанию:

- Рабочая группа (Work) позволяет пользователям присоединиться к группе по приглашению друга, являющегося членом группы. Приглашение не требует принятия приглашенным или одобрения владельца группы.
- Открытая группа (Public) позволяет владельцу группы назначать администраторов группы. Чтобы присоединиться к группе, пользователь должен найти ID группы и отправить запрос, который должен быть одобрен владельцем группы или администратором перед тем, как пользователь сможет присоединиться к группе.
- Группа встреч (Meeting) позволяет пользователям свободно присоединяться и выходить, поддерживает просмотр истории сообщений до присоединения пользователя к группе. Группы встреч идеальны для сценариев, интегрирующих TRTC (Tencent Real-Time Communication), таких как аудио и видео конференции и онлайн-образование.
- Аудио-видео группа (AVChatRoom) позволяет пользователям свободно присоединяться и выходить, поддерживает неограниченное количество членов и не сохраняет историю сообщений. Группы трансляции можно использовать с LVB (Live Video Broadcasting) для поддержки сценариев чата на экране комментариев.
- Группа сообщества (Community) позволяет пользователям свободно присоединяться и выходить, поддерживает до 100 000 членов и сохраняет историю сообщений. Чтобы присоединиться к группе, пользователь должен найти ID группы и отправить заявку, которая не требует одобрения администратором перед тем, как пользователь сможет присоединиться к группе.

> **Примечание:** Community — это новый мощный инструмент для развлекательного сотрудничества. В рамках одного сообщества большое количество участников может быть разделено на разные группы и темы для разделения сообщений для иерархической коммуникации, но они также могут использовать одну и ту же систему отношений друзей. Это помогает вам найти уникальный путь социального расширения. Функция сообщества подходит для разнообразных вариантов использования, такие как поиск единомышленников, социальная сеть на основе игр, маркетинг поклонников и управление организацией. Community: Функция сообщества поддерживается клиентом с улучшенным выпуском SDK v5.8.1668 или позже и веб-SDK v2.17.0 или позже. Для использования вам необходимо приобрести [Pro、Pro Plus、Enterprise](https://console.trtc.io/subscription/buy/chat?packType=pro), а затем включить его через [консоль](https://console.trtc.io/chat) > **Feature Configuration** > **Group configuration** > **Group feature configuration** > **Community**.

# Обзор выставления счетов

## Компоненты выставления счетов

Chat предлагает два метода выставления счетов:

- Пакет предоплаченный: сначала покупается, затем списывается. Покупка по циклам, действительна с дня T по день T следующего месяца. Например, если покупка произведена 1 марта 2023 года, она действительна с 1 марта 2023 года по 1 апреля 2023 года.
- Оплата по мере использования: выставляется счет по календарному месяцу. Сборы, понесенные в календарном месяце, будут удержаны в течение первых 3 дней следующего месяца.

# Поиск сообщений

`Search` — это важная функция в сценариях обслуживания клиентов, социальных сетей, онлайн-образования, онлайн-консультаций и офисных приложений. Она помогает пользователям быстро искать группы, пользователей и сообщения, улучшая пользовательский опыт продукта и повышая вовлеченность пользователей.

Из-за особенностей локального хранилища веб-платформы и других причин, React **не может реализовать локальный поиск**. Для лучшего удовлетворения потребности в возможности поиска была введена **облачная функция поиска**.

**Облачный поиск поддерживает глобальный поиск и поиск в диалоговом окне, а также поиск по группам, пользователям и сообщениям.**

> **Примечание:** Эта функция относится к дополнительным услугам (VAS). Вам необходимо приобрести плагин облачного поиска. Пожалуйста, нажмите [купить](https://console.trtc.io/chat/plugin/TUICloudSearch).

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/83a137315e3711f0bac1525400454e06.png)

## Сценарии использования

### Сценарий 1: Глобальный поиск

Через поиск пользователи могут глобально искать ВСЕ группы, пользователей и сообщения, содержащие указанные ключевые слова. Пользователи могут щелкнуть на поиск результатов групп, пользователей и сообщений. Как показано ниже:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/d3eda50a622a11f0a64452540099c741.png)

### Сценарий 2: Поиск в указанной сессии

Через функцию поиска сообщений пользователи могут напрямую искать исторические сообщения в определенных диалогах. Пользователи могут щелкнуть на поиск результатов исторических сообщений для прямой навигации к соответствующему месту чата с выделенным сообщением. Как показано ниже:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e8fda1b7622a11f091585254001c06ec.png)

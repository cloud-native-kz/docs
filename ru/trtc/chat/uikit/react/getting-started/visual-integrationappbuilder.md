# Визуальная интеграция: App Builder

## Введение в App Builder

App Builder предоставляет готовую к использованию систему визуальных компонентов пользовательского интерфейса. Он поддерживает конфигурацию всех типов — от базовых элементов управления (таких как Color и т. д.) до основных компонентов (таких как messageList, Search и т. д.). Разработчики могут эффективно создавать и отлаживать приложения. Поддерживает односекундный экспорт конфигурации в файл конфигурации, который можно быстро интегрировать в существующие проекты.

## Как испытать пользовательский интерфейс через App Builder

Вы можете испытать пользовательскую конфигурацию ChatKit UI в App Builder по следующей ссылке: [Ссылка для опробования](https://trtc.io/demo/homepage/#/detail?scene=messenger). Как показано на рисунке ниже, вы можете управлять интерфейсом ChatKit через конфигурации App Builder и настраивать стиль пользовательского интерфейса в соответствии со своими потребностями.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/79e5581c72a011f0b9a25254007c27c5.png)

## Настройка пользовательского интерфейса в вашем проекте

> **Примечание:** Перед этим необходимо запустить демонстрацию ChatKit, обратившись к [документации среды выполнения ChatKit](https://trtc.io/document/45912?platform=web&product=chat&menulabel=uikit), а затем импортировать соответствующий файл appConfig.json, чтобы он работал.

Вы можете загрузить настроенное содержимое пользовательского интерфейса через App Builder с помощью кнопки Download Customised Demo. В этом случае вам необходимо импортировать загруженный файл appConfig.json в src/config вашего уже запущенного проекта Chatkit и заменить исходный файл appConfig.json. После замены исходного файла appConfig.json перезапустите проект и пользовательский интерфейс вступит в силу.

# Пользовательские эмодзи и наклейки

> **Примечание:** Использование возможностей пользовательских эмодзи, описанных в этой статье, применимо к **@tencentcloud/chat-uikit-vue ≥ 2.2.0** или **@tencentcloud/chat-uikit-uniapp ≥ 2.2.0**.

В нашем компоненте `TUIChat` поддерживается отправка и получение двух типов эмодзи:

| Типы эмодзи | Формат отправки | Смешанный текстовый макет | Содержимое отправки | Включено по умолчанию в TUIKit |
| --- | --- | --- | --- | --- |
| Маленькое эмодзи | Текстовое сообщение MSG_TEXT | Да | Ключ изображения эмодзи | Включено по умолчанию |
| Большая наклейка | Сообщение лица MSG_FACE | Нет | index: emojiGroupIDdata: Ключ изображения эмодзи | Включено по умолчанию |

Конкретные эффекты выглядят следующим образом:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/98051a8d2c7c11efa01d5254005235d8.png)

## Пользовательские большие наклейки

TUIChat поддерживает загрузку пользовательских больших эмодзи из **сетевого пути**. Ниже мы рассмотрим добавление набора эмодзи кошек в качестве примера того, как добавить набор пользовательских больших эмодзи.

### Шаг 1: Подготовка ресурсов большой наклейки

1. Назовите каждый файл ресурса пакета эмодзи с идентификатором формата `@custom`: TUIKit требует соответствия `@custom` для анализа пользовательских ресурсов эмодзи.
2. Поскольку метод анализа пакетов эмодзи — `URL + ключ изображения эмодзи`, убедитесь, что загружаете новые ресурсы пакета эмодзи по единому сетевому пути `URL`.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/d901bca32c7711ef918f52540005b090.png)

### Шаг 2: Добавление пакета большой наклейки

После загрузки всех ресурсов пакета эмодзи в сеть необходимо добавить объявления для новых ресурсов пакета эмодзи в **исходный код TUIKit**.

Файл `TUIKit/components/TUIChat/emoji-config/custom-emoji.ts` является файлом объявления для пользовательских эмодзи.

> **Примечание:** Если у вас есть потребность в многоплатформенном взаимодействии, убедитесь, что emojiGroupID каждой пользовательской группы эмодзи и ключ каждого эмодзи в группе согласованы на всех платформах.

```
import { EMOJI_TYPE } from '../../../constant';// Заполните путь вашей пользовательской сети больших эмодзи, пример ниже
export const CUSTOM_BIG_EMOJI_URL = 'https://web.sdk.qcloud.com/im/test/emoji-test/';// Объявите список пользовательской группы больших эмодзи, пример ниже
export const CUSTOM_BIG_EMOJI_GROUP_LIST: IEmojiGroupList = [  {    emojiGroupID: 4, // ID группы эмодзи    type: EMOJI_TYPE.CUSTOM, // Тип группы эмодзи: CUSTOM Пользовательская группа эмодзи    url: CUSTOM_BIG_EMOJI_URL, // Публичный префикс URL группы эмодзи    // Объявление списка эмодзи, отформатированное как ключ + суффикс типа файла эмодзи, каждый ключ в списке должен быть уникальным    list: ['@custom_cat32.png', '@custom_cat33.png', '@custom_cat34.png', '@custom_cat35.png', '@custom_cat36.png', '@custom_cat37.png', '@custom_cat38.png', '@custom_cat39.png',      '@custom_cat40.png', '@custom_cat45.png', '@custom_cat46.png', '@custom_cat47.png', '@custom_cat49.png', '@custom_cat50.png', '@custom_cat51.png'],  },];
```

### Шаг 3: Отображение пользовательской большой наклейки

TUIKit внутренне поддерживает функцию анализа пользовательской большой наклейки. Следуя приведённым выше шагам, вы добьётесь следующего эффекта:

## Пользовательское маленькое эмодзи

### Встроенные ресурсы маленького эмодзи

TUIKit по умолчанию включает набор ресурсов эмодзи жёлтого лица. Авторские права на эмодзи жёлтого лица принадлежат Tencent Cloud. Для лицензирования [свяжитесь с нами](https://www.tencentcloud.com/document/product/1047/41676). Сравнение изображений эмодзи жёлтого лица и их значений приведено ниже:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a1d7609a2c7811efb0275254006c0558.png)

### Шаг 1: Подготовка ресурсов маленького эмодзи

Поскольку метод анализа пакетов эмодзи — `baseURL + ключ изображения эмодзи`, убедитесь, что загружаете новые ресурсы пакета эмодзи по единому сетевому пути `baseURL`.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f0abe92d2c7711ef97da5254007d9c55.png)

### Шаг 2: Замена ресурсов маленького эмодзи

#### 2.1 Добавление объявления пользовательского маленького эмодзи

Поскольку TUIKit требует соответствия `@custom` для анализа ресурсов пользовательских эмодзи, **каждый ключ ресурса пользовательского маленького эмодзи должен включать идентификатор формата**`@custom`**.**

Целевой каталог файла: `TUIKit/components/TUIChat/emoji-config/custom-emoji.ts`

> **Примечание:** Если у вас есть потребность в многоплатформенном взаимодействии, убедитесь, что ключ каждого пользовательского маленького эмодзи определен последовательно на всех платформах.

```
@custom идентификатор для анализа, пример ниже
```

#### 2.2 Добавление интернационализированного текстового утверждения для пользовательских маленьких эмодзи

##### Добавление пар ключ-значение для объявления значений на английском языке

Целевой каталог файла: `TUIKit/components/TUIChat/emoji-config/locales/en.ts`

```
// Переписание утверждения пары ключ-значение на английском языке, пример ниже
const Emoji = {  '[@custom_Expect]': '[Expect]',  '[@custom_Blink]': '[Blink]',  '[@custom_Guffaw]': '[Guffaw]',  ... }
```

##### Добавление пар ключ-значение для объявления значений на китайском языке

Целевой каталог файла: `TUIKit/components/TUIChat/emoji-config/locales/zh_cn.ts`

```
// Переписание утверждения пары ключ-значение на китайском языке, пример ниже
const Emoji = {  '[@custom_Expect]': '[Expect]',  '[@custom_Blink]': '[Blink]',  '[@custom_Guffaw]': '[Guffaw]',  ... }
```

### Шаг 3: Отображение пользовательских маленьких эмодзи

TUIKit внутренне поддерживает функцию анализа пользовательских маленьких эмодзи. После завершения приведённых выше шагов замены эмодзи они будут автоматически анализироваться и отображаться в TUIChat. Эффект выглядит следующим образом:

## Удаление системных эмодзи

emojiGroupID системных эмодзи соответствует следующим пакетам мемов:

| emojiGroupID = 0 | emojiGroupID = 1 | emojiGroupID = 2 | emojiGroupID = 3 |
| --- | --- | --- | --- |
| ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f364f75122eb11ef860b52540049c929.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/060f7b0122ec11ef860b52540049c929.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1352e0e122ec11efb2cb5254006568c0.png) | ![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1ff22dec22ec11ef91395254000a29ac.png) |

Если вам не требуются некоторые из приведённых выше групп эмодзи, вы можете удалить определение этой группы в `TUIKit/components/TUIChat/emoji-config/default-emoji.ts`:

> **Примечание:** Если у вас есть потребность в многоплатформенном взаимодействии, убедитесь, что одни и те же группы эмодзи удалены на всех платформах и конфигурация групп эмодзи остаётся согласованной после удаления.

```
// Например, удаление группы больших эмодзи с emojiGroupID = 1
export const BIG_EMOJI_GROUP_LIST: IEmojiGroupList = [  // {  //   emojiGroupID: 1,  //   type: EMOJI_TYPE.BIG,  //   url: DEFAULT_BIG_EMOJI_URL,  //   list: ['yz00', 'yz01', 'yz02', 'yz03', 'yz04', 'yz05', 'yz06', 'yz07', 'yz08',  //     'yz09', 'yz10', 'yz11', 'yz12', 'yz13', 'yz14', 'yz15', 'yz16', 'yz17'],  // },  ...];
```

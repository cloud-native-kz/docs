# Перевод сообщений

## Описание функции

Начиная с версии v2.1.0, TUIKit предлагает функцию перевода текстовых сообщений. Инструмент обмена сообщениями в компоненте TUIChat поддерживает перевод текстовых сообщений. В настоящее время он может переводить с основных языков на языки, такие как английский, китайский, русский, японский, корейский и другие. Подробнее о поддерживаемых языках см. в разделе [Список поддерживаемых языков текстового перевода](https://www.tencentcloud.com/document/product/1047/60746#437f6f49-c77f-4b59-b138-7c219e8d21f8).

После перевода текста результаты сохраняются в течение текущего сеанса входа и остаются сохраненными даже после переключения беседы. Если вы переключитесь на другую беседу и измените целевой язык для перевода, предыдущие результаты перевода будут автоматически переведены на новый целевой язык.

> **Примечание:** Функция перевода текстовых сообщений в настоящее время предоставляется бесплатно. Вы можете связаться с нами через [группу технического обмена Telegram](https://t.me/+EPk6TMZEZMM5OGY1), чтобы включить и опробовать полный функционал.

## **Введение в функцию**

### Установка целевого языка для перевода

Целевой язык перевода по умолчанию — китайский, но вы также можете вручную переключить целевой язык. Импортируйте `TUIChatService` из `@tencentcloud/chat-uikit-engine` и вызовите метод `setTranslationLanguage`, как показано в следующем коде:

```
import { TUIChatService } from "@tencentcloud/chat-uikit-engine";// Переключение на английский.TUIChatService.setTranslationLanguage('en');// Переключение на китайский.TUIChatService.setTranslationLanguage('zh');// Переключение на японский и поддержка других языков.TUIChatService.setTranslationLanguage('jp');
```

> **Примечание:** При повторном входе язык перевода по умолчанию переустановится на китайский.

### Установка исходного языка для перевода

В настоящее время в TUIKit рекомендуется **по умолчанию использовать** автоматическое определение исходного языка, но вы также можете вручную установить исходный язык для обеспечения точности перевода.

В файле `TUIChat/utils/translation.ts` реализован класс утилит перевода `Translator`, где вы можете изменить атрибут `sourceLanguage` в соответствии со [списком поддерживаемых языков текстового перевода](https://www.tencentcloud.com/document/product/1047/60746#437f6f49-c77f-4b59-b138-7c219e8d21f8).

```
TUIChatService.translateText({    sourceTextList: [text],    sourceLanguage: 'auto', // zh/en/jp/kr}).then((response) => {    // response data});
```

### Дополнительное чтение

Следующее содержание служит только дополнительным материалом для чтения. Перевод текстовых сообщений и связанные функции уже реализованы в TUIKit версии 2.1.0 и выше и не требуют ручной реализации пользователями.

### Проверка поддержки функции перевода в текущем пакете

Путем запроса `StoreName.APP` в `TUIStore` по ключу `enabledTranslationPlugin` полученное логическое значение указывает, включена ли функция.

```
const enable = TUIStore.getData(StoreName.APP, 'enabledTranslationPlugin');
```

### Удаление функции перевода

В файле `TUIKit/components/TUIChat/message-list/message-tool/index.vue` просто удалите объект с ключом `translate` в `actionItems`.

```
{    key: 'translate',    text: TUITranslateService.t('TUIChat.Translation'),    visible: false,    iconUrl: translateIcon,    renderCondition() {},    clickEvent: translateMessage,}
```

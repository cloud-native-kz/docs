# Язык

## Описание

> **Примечание:** Расширенные возможности многоязычной поддержки претерпели значительные улучшения и изменения в версии @tencentcloud/chat-uikit-vue v2.0.0 и выше. Данный документ демонстрирует использование новой версии. Пожалуйста, **убедитесь, что версия зависимости @tencentcloud/chat-uikit-vue в вашем проекте составляет ≥ v2.0.0**.

`Vue TUIKit` на веб-платформах и H5 поставляется со стандартными языковыми пакетами **упрощённого китайского и английского языков**, которые служат языком отображения интерфейса.

В соответствии с руководством в этом документе вы можете либо использовать встроенный языковой пакет, либо расширенные возможности интернационализации, которые включают добавление новых языков, новых терминов или изменение существующих переводов.

## Использование встроенного языка и лексикона

Если ваше приложение требует только **английского/упрощённого китайского языков** и нет необходимости в добавлении новых записей или изменении существующих переводов, обратитесь к этому разделу.

> **Примечание:** В будущем будет доступна поддержка большего количества языков, следите за обновлениями!

### Указание языка отображения

Чтобы установить определённый язык для интерфейса TUIKit, необходимо вызвать следующий код при инициализации приложения.

```
import { TUITranslateService } from "@tencentcloud/chat-uikit-engine";// change language to chineseTUITranslateService.changeLanguage("zh");// change language to englishTUITranslateService.changeLanguage("en");
```

### Динамическое изменение в реальном времени

При использовании `TUITranslateService.changeLanguage` для переключения языков текущий язык не обновится мгновенно. Вам необходимо обновить страницу, чтобы изменения вступили в силу.

Вы можете добиться динамического изменения и отображения языка в реальном времени путём переключения `page/component key`.

Например, динамическое переключение языка `TUIConversation` в реальном времени:

```
<template>  <div class="home">    <div class="button-container">      <div class="button" @click="changeLanguage('en')">English</div>      <div class="button" @click="changeLanguage('zh')">Simplified Chinese</div>    </div>    <div class="conversation-container">      <TUIConversation :key="locale" />    </div>  </div></template><script setup lang="ts">import { ref } from "vue";import { TUITranslateService } from "@tencentcloud/chat-uikit-engine";import { TUIConversation } from "./TUIKit";const locale = ref("zh");const changeLanguage = (language: string) => {  TUITranslateService.changeLanguage(language).then(() => {    locale.value = language;  });};</script><style scoped lang="scss">.home {  width: 400px;  .button-container {    display: flex;    flex-direction: column;    .button {      margin: 10px;      background-color: #006eff;      color: #ffffff;      text-align: center;      padding: 5px;      border-radius: 30px;      cursor: pointer;    }  }}</style>
```

## Пользовательские записи языка

### Добавление записи языка

Если вам нужно расширить или изменить записи языковых пакетов **упрощённого китайского и английского языков**, вы можете добавить или изменить записи в директории `src/TUIKit/locales`.

Записи локализации разделены на две части. Директория 'en' содержит записи на английском языке, а директория 'zh_cn' содержит записи на упрощённом китайском языке.

> **Примечание:** Если вам нужно использовать упрощённый китайский и английский языки, при добавлении или изменении записей **убедитесь, что вы синхронизируете изменения в папках 'en' и 'zh_cn' в соответствующих подпапках**.

Структура директории для пакета терминов 'locales' представлена в следующей схеме:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/19044c5ec01e11eeb629525400170219.png)

### Использование языковых терминов

Здесь `TUITranslateService.t()` используется для перевода терминов. Дополнительную информацию об этом интерфейсе можно найти в [TUITranslateService](https://web.sdk.qcloud.com/im/doc/chat-engine/en/ITUITranslateService.html).

```
<template>    <div>{{TUITranslateService.t("TUIChat.${yourLocaleKey}")}}</div></template><script setup lang="ts">import { TUITranslateService } from "@tencentcloud/chat-uikit-engine";</script>
```

# Цитирование сообщений

## Описание функции

В сеансах TUIChat вы можете цитировать предыдущие сообщения, чтобы указать ответ на конкретное сообщение. После ответа вы также можете нажать на цитируемое сообщение, чтобы перейти к исходному сообщению, которое будет выделено и мигать.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9cee0f701cd311ef949d52540019e87e.png)

## Цитирование сообщений

### Цитирование сообщения

В веб-версии щелкните правой кнопкой мыши на сообщение; в мобильной версии удерживайте нажатие на сообщение. На сообщении появится панель инструментов. Нажмите кнопку "Цитирование" в панели инструментов, чтобы процитировать сообщение.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/aac70c6a1cd311ef949d52540019e87e.png)

### Отмена цитирования сообщения

Когда сообщение процитировано, но еще не отправлено, нажатие кнопки закрытия после цитаты позволяет отменить цитирование сообщения.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b192d9d81cd311ef9b0c525400762795.png)

### Просмотр цитируемого сообщения

Нажатие на содержимое цитаты в цитируемом сообщении найдет исходное сообщение, которое будет выделено и мигать:

- Когда цитируемое сообщение находится в видимой области экрана, нажатие на содержимое цитаты цитируемого сообщения только выделит его и вызовет мигание.
- Когда цитируемое сообщение находится вне видимой области, но есть в списке сообщений, нажатие на содержимое цитаты автоматически прокрутит список сообщений к исходному сообщению, и оно будет выделено и мигать.
- Когда цитируемое сообщение ни видимо на экране, ни находится в списке сообщений, нажатие на содержимое цитаты не приведет к переходу на исходное сообщение и не вызовет его выделение и мигание.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b864a7531cd311efabdf525400f65c2a.png)

## Дополнительное чтение

> **Примечание:** Следующее содержимое служит только дополнительным материалом для чтения. Функция цитирования сообщений уже включена в TUIKit по умолчанию и не требует ручной реализации пользователями.

### Как реализовать цитирование сообщений

#### Цитирование сообщения

При нажатии кнопки "Цитирование" вызывается соответствующий методу `quoteMessage()` сообщения, и запись о цитировании сообщения сохраняется в Store.

```
function quoteMessage(message) {  message?.quoteMessage();}
```

#### Отображение цитируемых сообщений и отмена цитирования

Отслеживайте изменения записи цитирования TUIStore в компоненте поля ввода, и цитируемое сообщение можно получить через функцию обратного вызова.

```
const quoteMessage = ref<IMessageModel>();onMounted(() => {  TUIStore.watch(StoreName.CHAT, {    quoteMessage: (options: { message: IMessageModel, type: string }) => {      if (options.message && options.type === "quote") {        // Обнаружено новое цитирование сообщения.        quoteMessage.value = options.message;      } else {        // Обнаружена отмена цитирования сообщения.        quoteMessage.value = undefined;      }    },  });});
```

При отмене цитирования просто удалите запись о цитировании из TUIStore.

```
function cancelQuote() {  TUIStore.update(StoreName.CHAT, "quoteMessage", { message: undefined, type: "quote" });}
```

# Активация услуги

## Активация пробной версии услуги

Чтобы вы смогли лучше ознакомиться с функциями услуги Push, мы предлагаем бесплатную 7-дневную пробную версию для каждого SDKAppID. Каждый SDKAppID может использовать пробную версию один раз, срок действия каждой пробной версии составляет 7 дней.

Вы можете активировать пробную версию услуги Push в Chat Console. Подробные шаги приведены ниже:

1. Войдите в [Chat Console](https://console.trtc.io/) и нажмите кнопку **Create Application**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/616616dd7a4211ef9e7a5254000e0cea.png)

2. Введите название приложения в диалоговом окне **Create Application** и нажмите кнопку **Create**. В появившемся окне создания выберите **Chat**, введите название приложения, выберите **Region** для хранения данных и нажмите кнопку **Create**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/417a6aee7a4211ef80ff525400d5f8ef.png)

3. После завершения создания Chat приложения перейдите на страницу [Chat > Push > Access Settings](https://console.trtc.io/chat/push-plugin-push-identifier), нажмите кнопку бесплатной пробной версии, чтобы получить пробную версию услуги Push.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/753a37d5c35f11efbcd1525400bf7822.png)

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b954ef0abc3f11efba8d525400f69702.png)

4. После успешного получения вы можете просмотреть оставшееся время действия пробной версии на странице.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7dcaf3ddc35f11efbaf8525400454e06.png)

## Приобретение официальной версии

Описание цен месячного пакета услуги Push выглядит следующим образом:

> **Примечание:** Услуга Push предоставляется как плагин к продукту Chat. При приобретении месячного пакета услуги Push необходимо также приобрести платную версию Chat для её использования.

| Элемент расчётов | Режим расчётов | Описание сборов |
| --- | --- | --- |
| Push Service (Push) | Плата за ресурсный план (Предоплата) | 299 USD/месяц (включает неограниченное количество обычных push-уведомлений, 100 бесплатных вызовов API универсального/тегового push в день, 30 млн бесплатных сообщений универсального/тегового push в месяц) |
|  | Плата за перерасход ресурсов (Постоплата) | Превышение вызовов интерфейса универсального/тегового push, каждый дополнительный вызов/день, постоплата после ежедневного расчёта: 10 USD/день. Сообщения универсального/тегового push, каждые дополнительные 10 000 сообщений, постоплата после ежемесячного расчёта: 0,1 USD/10 000 сообщений. |

**Чтобы приобрести месячный пакет услуги Push (Push), выполните следующие шаги:**

1. Перейдите на [страницу покупки Chat](https://buy.tencentcloud.com/avc), выберите правильный центр обработки данных и приложение (SDKAppID), для которого вы хотите осуществить покупку, выберите месячный пакет Chat (Standard или Pro или Pro-plus или Enterprise) и добавьте пакет Push Notifications, затем оформите заказ.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/47d9d814bc4f11ef96d352540075b605.png)

2. После завершения покупки вы можете перейти на страницу [Push Overview](https://console.trtc.io/chat/push-plugin-push-identifier), чтобы просмотреть информацию приложения.

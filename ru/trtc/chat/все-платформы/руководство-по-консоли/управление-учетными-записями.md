# Управление учётными записями

Войдите в [консоль Chat](https://console.trtc.io/), выберите **Chat >** [**Users**](https://console.trtc.io/chat/account-management) на левой боковой панели и **выберите целевое приложение в верхней части**. Вы можете управлять учётными записями в соответствии с реальными потребностями бизнеса.

## Создание учётной записи

1. На странице **Account Management** нажмите **Create account**.
2. Настройте следующие параметры во всплывающем диалоговом окне:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9e233cf3cb3311f0a93d52540044a08e.png)

  - Account Type: Выберите **General** или **Admin**. Роль администратора приложения имеет наивысший уровень разрешений управления. Она позволяет вызывать REST API для выполнения операций, таких как создание или распуск группы, отправка сообщений всем участникам. Каждое приложение поддерживает до 10 администраторов.
  - Username: Введите идентификатор пользователя. Это поле обязательно.
  - Nickname: Введите никнейм. Это поле необязательно.
  - Profile Photo: Введите URL фотографии профиля пользователя. Это поле необязательно.
3. Нажмите **Confirm**.
После создания учётной записи в списке учётных записей можно увидеть имя пользователя, никнейм, тип, фотографию профиля и время создания учётной записи.

## Удаление учётных записей

1. По умолчанию удаление учётных записей запрещено. Однако это ограничение можно снять, чтобы разрешить удаление учётных записей. Для этого нажмите **Click here to remove the restriction**. В появившемся диалоговом окне подтверждения, ознакомившись с соответствующими рисками, нажмите **Confirm**, чтобы снять ограничение.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a76450a5cb3311f08942525400e889b2.png)

2. На странице Account Management выберите из списка учётные записи, которые вы хотите удалить, и нажмите **Batch Delete**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/d5d46c37cb3a11f09e745254007c27c5.png)

3. В появившемся диалоговом окне подтверждения, разобравшись с соответствующими рисками, нажмите **Confirm** для удаления учётной записи. Учтите, что после удаления данные не могут быть восстановлены. Будьте осторожны при выполнении этой операции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dc6a41b2cb3a11f0a93d52540044a08e.png)

## Редактирование учётной записи

1. На странице **Account Management** найдите учётную запись для редактирования и нажмите **Edit** в столбце **Operation**.
2. Настройте следующие параметры во всплывающем диалоговом окне:
  - Username: Это поле нельзя редактировать.
  - Nickname: Вы можете отредактировать никнейм пользователя. Это поле необязательно.
  - Account Type: Это поле нельзя редактировать.
  - Gender: Вы можете отредактировать пол пользователя. Это поле необязательно.
  - Birth Date: Вы можете отредактировать дату рождения пользователя. Это поле необязательно.
  - Location: Вы можете отредактировать местоположение. Это поле необязательно.
  - What's Up: Вы можете отредактировать статус. Это поле необязательно.
  - Friend Request Verification: Вы можете отредактировать режим проверки запроса в друзья.
  - Language: Вы можете отредактировать язык. Это поле необязательно.
  - Profile Photo: Вы можете отредактировать фотографию профиля. Это поле необязательно.
  - Message settings: Вы можете выбрать параметры сообщений. Это поле необязательно.
  - Friending: Вы можете указать, запрещает ли администратор пользователю инициировать запрос в друзья. Это поле необязательно.
  - Level: Вы можете отредактировать уровень. Это поле необязательно.
  - Role: Вы можете отредактировать роль. Это поле необязательно.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/93c463c8cb3911f0a93d52540044a08e.png)

3. Нажмите **Confirm**.

## Экспорт учётных записей

1. Экспорт отдельной учётной записи: На странице **Account Management** найдите учётную запись для экспорта и нажмите **Export** в столбце **Operation**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/117d920ecb3a11f09e745254007c27c5.png)

2. Массовый экспорт: На странице **Account Management** выберите учётные записи для экспорта и нажмите **Batch Export** над списком учётных записей.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/16c2c31dcb3a11f09e745254007c27c5.png)

3. Во всплывающем диалоговом окне **Export successful** нажмите **Download**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1a877b93cb3a11f087ad52540099c741.png)

# Получение счетчиков группы

## Обзор функции

Администратор приложения может использовать этот API для получения счетчиков группы.

> **Примечание:** Для использования этой функции необходимо приобрести [выпуск Pro, выпуск Pro Plus или Enterprise](https://www.tencentcloud.com/document/product/1047/34577).

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Совпадает с рабочими группами (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Совпадает с группой встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Нет |

Это предустановленные типы групп в Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/get_group_counter?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны только измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/modify_group_attr | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Дополнительную информацию см. в разделе **App Admin** в статье [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробности см. в разделе [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

**Базовый формат**
Получение счетчиков группы:

```
{    "GroupId": "@TGS#aC5SZEAEF",    "GroupCounterKeys":[    // Список ключей счетчика группы для получения. Если значение не передано, будут возвращены все ключи счетчика группы.        "like",        "unlike"    ]}
```

### Поля запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы |
| GroupCounterKeys | Array | Нет | Список ключей счетчика группы для получения. Если значение не передано, будут возвращены все ключи счетчика группы. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "GroupCounter":[        {            "Key":"like", // Ключ счетчика группы            "Value":5 // Последнее значение счетчика группы        },        {            "Key":"unlike",            "Value":1        }    ]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |
| GroupCounter | Array | Список полученных пар ключ-значение счетчика группы |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не произойдет сетевая ошибка (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Попробуйте снова. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Недостаточно прав для выполнения операции. Например, текущий пользователь не является членом группы. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Неверный ID группы. Проверьте, правильно ли введен ID группы и существует ли группа. |
| 10059 | Для использования этой функции необходимо приобрести выпуск Pro, выпуск Pro Plus или Enterprise. |

# Получение списка заблокированных членов аудио-видео группы.

## Обзор функции

Вы можете использовать этот API для получения списка заблокированных членов аудио-видео группы.

> **Примечание** Блокировка членов аудио-видео группы поддерживается SDK Enhanced edition v6.6 или позже и web SDK v2.22.0 или позже. Чтобы использовать эту функцию, необходимо [приобрести Pro edition, Pro Plus edition или Enterprise edition](https://www.tencentcloud.com/document/product/1047/34577), перейти в [консоль](https://console.trtc.io/chat/qun-setting), выбрать **Feature configuration** > **Group configuration** > **Group feature configuration** и включить **Audio-video group member banning**.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. То же самое, что и рабочие группы (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. То же самое, что и группы встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Да |

Это предустановленные типы групп в Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/get_group_ban_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже описаны измененные параметры при вызове этого API. Для других параметров см. раздел [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где расположен ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/get_group_ban_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Login Authentication](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробно см. в разделе [Generating UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение фиксировано как `json`. |

### Максимальная частота вызова

200 вызовов в секунду

### Пример запроса

- **Базовый формат**

Используется для доставки широковещательных сообщений всем аудио-видео группам.

```
{    "GroupId": "@TGS#aJRGC4MH6",  // Group ID    "Limit":20, // Number of banned members to be pulled per page each time. The maximum value allowed is 100.    "Offset": 0 // Offset. For the first request, set `Offset` to 0. For subsequent requests, set `Offset` to the value of `NextOffset` in the response packet. If `NextOffset` is 0, the entire list of banned members of the audio-video group is obtained.}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, членов которой необходимо разблокировать |
| Limit | Integer | Нет | Количество разблокированных пользователей, получаемых каждый раз. Максимальное значение - 100. |
| Offset | Integer | Да | Смещение. Для первого запроса установите `Offset` на 0. Для последующих запросов установите `Offset` на значение `NextOffset` в пакете ответа. |

### Пример ответа

```
{    "ActionStatus": "OK",    "BannedAccountList": [        {            "BannedUntil": 1660875916,            "Member_Account": "brennanli1"        },        {            "BannedUntil": 1660875916,            "Member_Account": "brennanli12"        }    ],    "ErrorCode": 0,    "ErrorInfo": "",    "NextOffset": 2}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |
| BannedAccountList | Array | Информация о заблокированных членах. `BannedUntil` указывает время окончания блокировки заблокированных членов. `Member_Account` указывает ID учетных записей заблокированных членов. |
| NextOffset | Integer | Значение смещения следующего запроса. Если это поле равно `0`, список заблокированных членов группы AVChatRoom полностью получен. |

## Коды ошибок

Возвращаемый HTTP-статус для этого API всегда равен 200, за исключением ошибок сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (от 60000 до 79999) см. раздел [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В таблице ниже описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Попробуйте снова. |
| 10003 | Неверное слово команды. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10005 | Количество членов, которых необходимо разблокировать, превышает максимальное ограничение в 20 членов на запрос. |
| 10007 | Недостаточно прав операции. Например, переключатель не включен в консоли или операционная учетная запись не является корневой учетной записью. |

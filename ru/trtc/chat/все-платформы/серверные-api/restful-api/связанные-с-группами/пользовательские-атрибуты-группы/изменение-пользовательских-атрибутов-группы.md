# Изменение пользовательских атрибутов группы

## Обзор функции

Этот API используется администратором приложения для изменения пользовательских атрибутов группы.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Это то же самое, что рабочие группы (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Это то же самое, что группа встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Да |

Это предустановленные типы групп в Chat. Для получения дополнительной информации см. [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/modify_group_attr?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Имя домена, соответствующее стране/региону, в котором находится ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/modify_group_attr | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительные сведения см. в разделе **Администратор приложения** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака, варьирующееся от 0 до 4294967295. |
| contenttype | Формат запроса, который должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Базовый формат**
Изменить пользовательские атрибуты группы.

```
{  "GroupId": "@TGS#aC5SZEAEF",  "GroupAttr":[      {          "key":"attr_key", //Ключ атрибута          "value":"attr_val" //Значение атрибута      },{          "key":"attr_key1",          "value":"attr_val1"      }  ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, для которой требуется изменить пользовательские атрибуты |
| GroupAttr | Array | Да | Список пользовательских атрибутов. `key`: Ключ пользовательского атрибута, максимальная длина — 32 байта. Поддерживается до 16 ключей. `value`: Значение пользовательского атрибута, максимальная длина — 4000 байт. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode":0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: не удалось |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Не удалось |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (от 60000 до 79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Попробуйте еще раз. |
| 10004 | Недействительный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Нет прав доступа. Эта ошибка возникает, например, когда группа этого типа не поддерживает операции с пользовательскими атрибутами. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Недействительный ID группы. Проверьте, правильно ли введен ID группы и существует ли группа. |
| 10045 | Размер ключа пользовательского атрибута превышает предел 32 байта. |
| 10046 | Размер значения пользовательского атрибута превышает предел 4000 байт. |
| 10047 | Количество ключей пользовательского атрибута превышает предел 16. |
| 10048 | Общий размер значений пользовательских атрибутов превышает предел 16000 байт. |
| 10049 | Частота записи пользовательских атрибутов в группу превышает предел. Вы можете добавлять, удалять или изменять их не более 5 раз в секунду. |

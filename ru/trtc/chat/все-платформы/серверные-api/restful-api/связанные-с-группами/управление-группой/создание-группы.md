# Создание группы

## Обзор функции

Этот API используется администратором приложения для создания групп.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Совпадает с рабочими группами (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Совпадает с группами встреч (Meeting) в новой версии. |
| AVChatRoom | Да. Для использования аудио-видео группы (AVChatroom) пользователи (включая владельца группы) должны использовать SDK для запроса присоединения к группе. |
| Community | Да |

Это предустановленные типы групп в Chat. Для получения дополнительной информации см. [Group System](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание** Если тип группы указан как аудио-видео группа (AVChatRoom): При создании группы этого типа создатель группы не может добавлять пользователей в группу. Если при создании группы указан член группы, будет возвращена ошибка 10007. Пользователи могут только запросить присоединение к этому типу группы. Если указан владелец группы, владелец группы должен использовать SDK API для присоединения к группе каждый раз при использовании группы. Если тип группы указан как тип, отличный от аудио-видео группы (AVChatRoom): Если при отправке запроса не указаны ни владелец группы, ни список членов группы, количество создаваемых групп не ограничено. Если при отправке запроса указаны владелец группы или список членов группы, указанные владелец группы или члены группы автоматически добавляются в группу. Если общее количество групп в приложении превышает 100 000, будут взиматься определенные сборы. Для получения дополнительной информации см. [Pricing](https://intl.cloud.tencent.com/document/product/1047/34350). Количество групп, к которым пользователь может одновременно присоединиться, ограничено. При достижении этого числа присоединение или создание группы завершится неудачей. Для получения дополнительной информации см. [Pricing](https://intl.cloud.tencent.com/document/product/1047/34350).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/create_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [RESTful API Overview](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/create_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Login Authentication](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробную информацию см. в [Generating UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Базовый формат**
Создание группы. Поле `Owner_Account` необязательно. Если оно не указано, группа не будет иметь владельца. Если вы хотите указать владельца группы, убедитесь, что UserId владельца группы [был импортирован](https://intl.cloud.tencent.com/document/product/1047/34953).

```
{  "Owner_Account": "leckie", // UserId владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom, AVChatRoom или Community  "Name": "TestGroup" // Название группы (обязательно)}
```

- **Содержит только базовую информацию о группе**
Создание группы и указание базовой информации о группе, такой как описание группы и уведомление группы.

```
{  "Owner_Account": "leckie", // UserId владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom, AVChatRoom или Community  "Name": "TestGroup", // Название группы (обязательно)  "Introduction": "This is group Introduction", // Описание группы (необязательно)  "Notification": "This is group Notification", // Уведомление группы (необязательно)  "FaceUrl": "http://this.is.face.url", // URL фотографии профиля группы (необязательно)  "MaxMemberNum": 500, // Максимальное количество членов группы (необязательно)  "ApplyJoinOption": "FreeAccess"  // Метод обработки запросов на присоединение к группе (необязательно)}
```

- **Содержит только информацию о членах группы**
Создание группы и инициализация списка членов группы, который описан в таблице описания пакета запроса. При инициализации членов группы можно устанавливать только поля `Member_Account` и `Role`.

```
{  "Name": "TestGroup", // Название группы (обязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom или Community (AVChatRoom не поддерживается) (обязательно)  "MemberList": [ // Начальный список членов группы, который содержит максимум 20 членов (необязательно)       {          "Member_Account": "bob", // Член (обязательно)          "Role": "Admin" // Роль, присвоенная члену (необязательно). В настоящее время доступен только вариант Admin.       },       {          "Member_Account": "peter"       }   ]}
```

- **Пользовательский ID группы**
Чтобы упростить ID групп, Tencent Cloud позволяет приложениям настраивать ID групп при создании групп через RESTful API.

```
{  "Owner_Account": "leckie", // UserId владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom, AVChatRoom или Community (обязательно)  "GroupId": "MyFirstGroup", // Определяемый пользователем ID группы (необязательно)  "Name": "TestGroup"   // Название группы (обязательно)}
```

- **Содержит только пользовательскую информацию о группе**
Создание группы и указание пользовательских полей группы. Поле `AppDefineData` недоступно по умолчанию и требует включения в [консоли Chat](https://console.tencentcloud.com/im) перед использованием. Подробную информацию см. в таблице описания полей пакета запроса.

```
{  "Name": "TestGroup", // Название группы (обязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom, AVChatRoom или Community (обязательно)  "AppDefinedData": [ // Пользовательское поле группы (необязательно)      {          "Key": "GroupTestData1", // Ключ пользовательского поля приложения          "Value": "xxxxx" // Значение пользовательского поля      },      {          "Key": " GroupTestData2",          "Value": "abc\\u0000\\u0001" // Пользовательское поле поддерживает двоичные данные.      }  ]}
```

- **Содержит только пользовательскую информацию о членах группы**
Создание группы и указание пользовательских полей членов группы. По умолчанию AppMemberDefinedData недоступна и требует включения в [консоли Chat](https://console.tencentcloud.com/im) перед использованием. Подробную информацию см. в таблице полей пакета запроса.

```
{  "Owner_Account": "leckie", // UserId владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom или Community (AVChatRoom не поддерживается) (обязательно)  "Name": "TestGroup", // Название группы (обязательно)  "MemberList": [     {        "Member_Account":"bob",        "AppMemberDefinedData":[ // Пользовательское поле члена группы (необязательно)             {                 "Key": "MemberDefined1", // Пользовательский ключ члена группы                 "Value": "MemberData1" // Значение пользовательского поля члена группы             },             {                 "Key": "MemberDefined2",                 "Value": "MemberData2"             }         ]     },     {        "Member_Account":"peter",        "AppMemberDefinedData":[             {                 "Key": "MemberDefined1",                 "Value": "MemberData1"             },             {                 "Key": "MemberDefined2",                 "Value": "MemberData2"             }         ]     }  ]}
```

- **Содержит только формат параметров с поддержкой тематики**
Создание группы и указание типа как сообщества с поддержкой тематики.

```
{  "Type": "Community", 	// Тип группы (обязательно)  "Name": "TestCommunityGroup", // Название группы (обязательно)  "SupportTopic": 1			// Поддерживается ли опция темы. Допустимые значения: `1`: да; `0`: нет.}
```

- **ВСЕ В ОДНОМ**

```
{  "Owner_Account": "leckie", // UserId владельца группы (необязательно)  "Type": "Public", // Тип группы: Private, Public, ChatRoom или Community (AVChatRoom не поддерживается) (обязательно)  "GroupId":"MyFirstGroup", // Определяемый пользователем ID группы (необязательно)  "Name": "TestGroup", // Название группы (обязательно)  "Introduction": "This is group Introduction", // Описание группы (необязательно)  "Notification": "This is group Notification", // Уведомление группы (необязательно)  "FaceUrl": "http://this.is.face.url", // URL фотографии профиля группы (необязательно)  "MaxMemberNum": 500, // Максимальное количество членов группы (необязательно)  "ApplyJoinOption": "FreeAccess", // Метод обработки запросов на присоединение к группе (необязательно)  "AppDefinedData": [ // Пользовательское поле группы (необязательно)      {          "Key": "GroupTestData1", // Ключ пользовательского поля приложения          "Value": "xxxxx" // Значение пользовательского поля      },      {          "Key": "GroupTestData2",          "Value": "abc\\u0000\\u0001" // Пользовательское поле поддерживает двоичные данные.      }  ],  "MemberList": [ // Начальный список членов группы, который содержит максимум 20 членов (необязательно)      {          "Member_Account": "bob", // Член (обязательно)          "Role": "Admin", // Роль, присвоенная члену. В настоящее время доступен только вариант Admin. (необязательно)          "AppMemberDefinedData":[ // Пользовательское поле члена группы (необязательно)             {                 "Key":"MemberDefined1", // Пользовательский ключ члена группы                 "Value":"MemberData1" // Значение пользовательского поля члена группы             },             {                 "Key":"MemberDefined2",                 "Value":"MemberData2"             }         ]      },      {          "Member_Account": "peter",          "AppMemberDefinedData":[             {                 "Key":"MemberDefined1",                 "Value":"MemberData1"             },             {                 "Key":"MemberDefined2",                 "Value":"MemberData2"             }         ]      }  ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| Owner_Account | String | Необязательно | ID владельца группы, который должен быть [импортированной](https://intl.cloud.tencent.com/document/product/1047/34953) учетной записью. Значение будет автоматически добавлено в члены группы. Если это поле не указано, группа не будет иметь владельца, и членам группы нужно вызвать API присоединения к группе при использовании аудио-видео группы (AVChatroom). |
| Type | String | Обязательно | Тип группы, включая Public, Work (Private), Meeting, AVChatRoom и Community. |
| GroupId | String | Необязательно | Чтобы упростить ID групп и сделать их более запоминаемыми и общими, мы позволяем приложениям [настраивать ID групп](https://intl.cloud.tencent.com/document/product/1047/33529#.E8.87.AA.E5.AE.9A.E4.B9.89.E7.BE.A4.E7.BB.84-id) при создании групп через RESTful API. |
| Name | String | Обязательно | Название группы, максимальная длина которого составляет 30 байт. Это поле кодируется в UTF-8, и один китайский символ занимает 3 байта. |
| Introduction | String | Необязательно | Описание группы, максимальная длина которого составляет 240 байт. Это поле кодируется в UTF-8, и один китайский символ занимает 3 байта. |
| Notification | String | Необязательно | Уведомление группы, максимальная длина которого составляет 300 байт. Это поле кодируется в UTF-8, и один китайский символ занимает 3 байта. |
| FaceUrl | String | Необязательно | URL фотографии профиля группы, максимальная длина которого составляет 100 байт. |
| MaxMemberNum | Integer | Необязательно | Максимальное количество членов группы. Значение по умолчанию — верхний предел платного пакета. Например, для издания Developer это 20. При обновлении пакета вам нужно изменить это поле в соответствии с базовой информацией измененной группы. |
| ApplyJoinOption | String | Необязательно | Метод обработки запросов на присоединение к группе, который может быть `FreeAccess`, `NeedPermission` или `DisableApply`. Метод по умолчанию — `NeedPermission`. Это поле действительно только для [групп](https://intl.cloud.tencent.com/document/product/1047/33529#.E5.8A.A0.E7.BE.A4.E6.96.B9.E5.BC.8F.E5.B7.AE.E5.BC.82), к которым могут присоединиться пользователи. Группы Community в настоящее время не поддерживают это поле. |
| AppDefinedData | Array | Необязательно | Пользовательское поле уровня группы. Это поле недоступно по умолчанию и требует включения в [консоли Chat](https://console.tencentcloud.com/im). Подробную информацию см. в [Custom Fields](https://intl.cloud.tencent.com/document/product/1047/33529#.E8.87.AA.E5.AE.9A.E4.B9.89.E5.AD.97.E6.AE.B5). |
| MemberList | Array | Необязательно | Начальный список членов группы, который содержит максимум 20 членов. Подробную информацию о полях информации членов группы см. в [Group member profile](https://intl.cloud.tencent.com/document/product/1047/33529#SelfInfoFilter). |
| AppMemberDefinedData | Array | Необязательно | Пользовательские поля членов группы. Это поле недоступно по умолчанию и требует включения в [консоли Chat](https://console.tencentcloud.com/im). Подробную информацию см. в [Custom Fields](https://intl.cloud.tencent.com/document/product/1047/33529#.E8.87.AA.E5.AE.9A.E4.B9.89.E5.AD.97.E6.AE.B5). |
| SupportTopic | Integer | Нет | Поддерживается ли опция темы. Это поле действительно только для сообществ. |

### Пример ответа

- **Базовая форма, содержит только базовую информацию о группе, содержит только информацию о членах группы или содержит только пользовательскую информацию**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "@TGS#2J4SZEAEL"}
```

- **Содержит только формат параметров с поддержкой тематики**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "@TGS#_@TGS#cRDH3HIM62CP",  "HugeGroupFlag": 0,  "Type": "Community"}
```

- **Пользовательский ID группы и ВСЕ В ОДНОМ**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "GroupId": "MyFirstGroup"}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: неудачно |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: неудачно |
| ErrorInfo | String | Информация об ошибке |
| GroupId | String | ID группы после успешного создания, назначенный backend Chat. |

## Коды ошибок

Возвращаемый код статуса HTTP для этого API всегда 200, если не возникает сетевая ошибка (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для публичных кодов ошибок (60000–79999) см. [Error Codes](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, характерные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверная команда запроса. |
| 10004 | Неверный параметр. Проверьте описание ошибки и исправьте проблему. |
| 10005 | Количество импортированных в пакет запроса членов превышает 20. Уменьшите количество членов, импортированных в параметр `MemberList`. |
| 10006 | Операция превышает предел частоты. Пожалуйста, уменьшите частоту вызовов. Эта ошибка обычно вызывается слишком большим увеличением сетевой группы за один день или слишком частыми вызовами для получения всех групп в приложении. |
| 10007 | Недостаточно прав на выполнение операции. Проверьте параметры запроса на основе информации об ошибке. Например, указанный тип группы не позволяет добавлять членов, но в запросе был указан `MemberList`. |
| 10008 | Запрос недействителен, вероятно, потому что проверка информации подписи в запросе не удалась. Пожалуйста, повторите попытку или [отправьте заявку](https://console.tencentcloud.com/workorder/category?level1_id=29&level2_id=40&source=0&data_title=%E4%BA%91%E9%80%9A%E4%BF%A1%20%20IM&step=1). |
| 10016 | Backend приложения отклонил эту операцию через webhook. Проверьте возвращаемое значение вашего webhook "Before a group is created". |
| 10021 | ID группы уже используется другим пользователем. Выберите другой ID группы. |
| 10025 | Вы уже использовали этот ID группы. Сначала распустите существующую группу или выберите другой ID группы. |
| 10036 | Количество созданных аудио-видео групп (AVChatRoom) превышает предел. Удалите некоторые из групп или приобретите обновление, ссылаясь на [Pricing](https://intl.cloud.tencent.com/document/product/1047/34350). |
| 10037 | В запросе указан Owner_Account, но количество групп, созданных или присоединенных владельцем группы, превышает предел. Владельцу группы необходимо сначала покинуть некоторые группы или ссылаться на [Pricing](https://intl.cloud.tencent.com/document/product/1047/34350) для приобретения обновления. |
| 10038 | Количество импортированных в пакет запроса членов превышает предел. Уменьшите количество членов, импортированных в параметр `MemberList`, или приобретите обновление, ссылаясь на [Pricing](https://intl.cloud.tencent.com/document/product/1047/34350). |

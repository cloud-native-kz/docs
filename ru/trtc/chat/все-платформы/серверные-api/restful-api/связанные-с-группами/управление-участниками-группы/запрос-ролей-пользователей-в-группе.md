# Запрос ролей пользователей в группе

## Обзор функции

Этот API используется администратором приложения для массового получения ролей пользователей в группе.

## Описание вызова API

### Поддерживаемые типы групп

| ID типа группы | Поддержка этого RESTful API |
| --- | --- |
| Private | Да. То же самое, что рабочие группы (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. То же самое, что группы встреч (Meeting) в новой версии. |
| AVChatRoom | Нет |
| Community | Да |

Выше приведены предустановленные типы групп Chat. Для получения дополнительной информации см. [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание** Аудиовидео-группы (AVChatRoom) не поддерживают этот API. Если этот API будет вызван для аудиовидео-группы, будет возвращена ошибка 10007. Для получения текущего списка администраторов см. [Получение списка администраторов группы прямых трансляций](https://www.tencentcloud.com/document/product/1047/60524).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/get_role_in_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Силиконовая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/get_role_in_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Login Authentication](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробнее см. в разделе [Generating UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

Получить список групп, в которые вступил пользователь. Информация о группе включает ID группы и ID членов группы, чьи роли должны быть запрошены.

```
{    "GroupId": "@TGS#2C5SZEAEF",    "User_Account": [ // Поддерживается до 500 аккаунтов членов        "leckie",        "peter",        "wesley"    ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, которая должна быть запрошена |
| User_Account | Array | Да | Аккаунты пользователей для запроса. Поддерживается до 500 аккаунтов. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "UserIdList": [ // Результат        {            "Member_Account": "leckie",            "Role": "Owner" // Роль члена: Owner/Admin/Member/NotMember        },        {            "Member_Account": "peter",            "Role": "Member"        },        {            "Member_Account": "wesley",            "Role": "NotMember"        }    ]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: Успешно; `FAIL`: Ошибка |
| ErrorCode | Integer | Код ошибки. `0`: Успешно; другие значения: Ошибка |
| ErrorInfo | String | Информация об ошибке |
| UserIdList | Array | Извлеченные роли членов, включая Owner, Admin, Member и NotMember |

## Коды ошибок

HTTP код состояния, возвращаемый этим API, всегда равен 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000-79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Попробуйте снова. |
| 10003 | Неверное слово команды. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Нет прав доступа. Оператор должен иметь разрешения для выполнения соответствующих операций. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Ошибка при разборе пакета JSON. Проверьте, соответствует ли пакет спецификациям JSON. |

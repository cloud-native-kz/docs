# Массовое отключение и включение звука

## Обзор функции

- Администратор приложения может отключить звук определённых участников группы на определённый период времени.
- Администратор приложения может включить звук определённых участников группы.
- Когда пользователи с отключённым звуком покидают группу и повторно её присоединяются, они остаются с отключённым звуком.

## Описание вызова API

### Применимые типы групп

| Тип группы | Поддержка этого RESTful API |
| --- | --- |
| Private | Нет. То же самое, что рабочие группы (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. То же самое, что группы встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Да |

Это встроенные типы групп службы Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание:** приватные группы (т. е. рабочие группы в новой версии) не поддерживают отключение звука участников.

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/forbid_send_msg?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Отключение звука участников**
Вы можете установить определённый период времени в поле `MuteTime` для отключения звука указанных участников.

```
{  "GroupId": "@TGS#2C5SZEAEF",  "Members_Account": [ // До 500 учетных записей участников      "peter",      "leckie"  ],  "MuteTime": 60 // Период отключения звука, в секундах}
```

- **Включение звука участников**
Чтобы включить звук участников, установите `MuteTime` в значение `0`.

```
{  "GroupId": "@TGS#2C5SZEAEF",  "Members_Account": [ // До 500 учетных записей участников      "peter",      "leckie"  ],  "MuteTime": 0 // 0 указывает на включение звука участников.}
```

### Поля запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, в которой нужно выполнить запрос |
| Members_Account | Array | Да | Учетные записи участников для отключения звука. Поддерживается максимум 500 учетных записей. |
| MuteTime | Integer | Да | Период отключения звука целого числа без знака, в секундах. `0`: включить звук; `4294967295`: постоянное отключение звука |
| TopicId | String | Опционально | ID темы, статус отключения звука которой требуется установить. Это поле применяется только к группам сообществ с поддержкой тем. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode":0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если только не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и подробности можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, характерные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Ошибка внутреннего сервера. Повторите попытку. |
| 10003 | Неверная команда. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Нет прав доступа. Например, обычный участник в публичной группе пытается удалить других пользователей из группы, но это может делать только администратор приложения. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Неверный ID группы. Используйте правильный ID группы. |

# Разблокирование участников группы

## Обзор функции

Этот API используется для разблокирования участника группы. Разблокированный участник может повторно присоединиться к аудиовизуальной группе для получения сообщений.

> **Примечание:** разблокирование участников аудиовизуальной группы поддерживается SDK Enhanced Edition v6.6 и более поздних версий, а также веб-SDK v2.22.0 и более поздних версий. Для использования этой функции необходимо [приобрести редакцию Pro, Pro Plus или Enterprise](https://www.tencentcloud.com/document/product/1047/34577), перейти в [консоль](https://console.trtc.io/chat/qun-setting), выбрать **Конфигурация функций** > **Конфигурация группы** > **Конфигурация функций группы** и отключить **Блокирование участников аудиовизуальной группы**.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Аналогично рабочим группам (Work) в новой версии. |
| Public | Да |
| ChatRoom | Да. Аналогично группам встреч (Meeting) в новой версии. |
| AVChatRoom | Да |
| Community | Да |

Это предустановленные типы групп в Chat. Дополнительную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/unban_group_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. раздел [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/unban_group_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **Администратор приложения** в документе [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробную информацию см. в разделе [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |

| contenttype | Формат запроса. Значение фиксировано на `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

- **Основной формат**

Разблокирование участников аудиовизуальной группы

```
{    "GroupId": "@TGS#aJRGC4MH6",  // ID группы, участники которой будут разблокированы    "Members_Account":["brennanli3", "brennanli12"], // ID учетных записей участников, которые будут разблокированы. На один запрос поддерживается до 20 ID учетных записей.}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID группы, участники которой будут разблокированы |
| Members_Account | Array | Да | ID учетных записей участников, которые будут разблокированы. На один запрос поддерживается до 20 ID учетных записей. |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode":0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |

## Коды ошибок

Код состояния HTTP для этого API всегда 200, если не происходит ошибка сети (например, ошибка 502). Конкретный код ошибки и подробности можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. раздел [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неверная команда. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните неполадку. |
| 10005 | Количество участников для разблокирования превышает максимальный лимит в 20 участников на один запрос. |
| 10007 | Недостаточно прав доступа для выполнения операции. Например, переключатель не включен в консоли или операция выполняется не учетной записью администратора. |

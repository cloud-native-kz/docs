# Удаление беседы

## Обзор функции

Этот API используется для удаления беседы. Он также может очищать роуминговые сообщения.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/recentcontact/delete?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны изменённые параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Имя домена, соответствующее стране/региону, где расположен ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/recentcontact/delete | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учётная запись администратора приложения. Дополнительную информацию см. в разделе **Администратор приложения** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учётной записью администратора приложения. Подробнее см. [Генерация UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

#### Удаление беседы один-на-один

```
{    "From_Account":"id1",    "Type":1,    "To_Account":"id2",    "ClearRamble":1}
```

#### Удаление групповой беседы

```
{    "From_Account":"id1",    "Type":2,    "ToGroupid":"id2",    "ClearRamble":1}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| From_Account | String | Да | `UserID` учётной записи, для которой нужно удалить беседу |
| Type | Integer | Да | Тип беседы. `1`: беседа один-на-один; `2`: групповая беседа |
| To_Account | String | Нет | UserID другого участника беседы, значение присваивается только для беседы один-на-один |
| ToGroupid | string | Нет | ID группы, значение присваивается только для групповой беседы. |
| ClearRamble | Integer | Нет | Нужно ли очищать роуминговые сообщения. `1`: да; `0`: нет |

### Пример ответа

```
{    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": "",    "ErrorDisplay": ""}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно. `FAIL`: ошибка. |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка. Подробнее о ненулевых результатах см. [Коды ошибок](#ErrorCode). |
| ErrorInfo | String | Подробное сообщение об ошибке |
| ErrorDisplay | String | Подробная информация, отображаемая на клиенте |

## Коды ошибок

| Код ошибки | Описание |
| --- | --- |
| 50001 | Запрошенный UserID не был импортирован в бэкенд Tencent Cloud Chat. Пожалуйста, импортируйте его. |
| 50002 | Неправильный параметр запроса. Проверьте запрос в соответствии с описанием ошибки. |
| 50003 | Запрос требует разрешений администратора приложения. |
| 50004 | Внутренняя ошибка сервера. Пожалуйста, попробуйте ещё раз. |
| 50005 | Превышено время ожидания сети. Попробуйте позже. |

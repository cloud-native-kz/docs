# Установка количества непрочитанных сообщений участника

> **Примечание Chat**: Этот документ предназначен только для пользователей, переносящих свои приложения в Tencent Cloud Chat.

## Описание функции

- Этот API используется администраторами приложений для установки количества непрочитанных сообщений участника группы. При вызове этого API обратные вызовы не запускаются и уведомления не отправляются.
- При переносе приложения из другой системы Chat в Tencent Cloud Chat можно использовать этот API для установки количества непрочитанных сообщений участников группы.

## Описание вызова API

### Применимые типы групп

| ID типа группы | Поддержка RESTful API |
| --- | --- |
| Private | Да. Аналогично рабочей группе (Work) в новой версии. |
| Public | Да. |
| ChatRoom | Нет. Аналогично группе встреч (Meeting) в новой версии. |
| AVChatRoom | Нет. |
| Community | Да. |

Это встроенные типы групп в Chat. Подробную информацию см. в разделе [Система групп](https://intl.cloud.tencent.com/document/product/1047/33529).

> **Примечание**: Группы ChatRoom и AVChatroom (аудио-видео) не поддерживают подсчет непрочитанных сообщений. Поэтому вы не можете установить количество непрочитанных сообщений для участников этих двух типов групп. Если вы попытаетесь это сделать, ошибка не будет возвращена.

### Пример URL запроса

```
https://xxxxxx/v4/group_open_http_svc/set_unread_msg_num?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже описаны только измененные параметры при вызове этого API. Для получения информации о других параметрах см. раздел [Введение в RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| https | Протокол запроса — HTTPS, метод запроса — POST. |
| xxxxxx | Страна/регион, в котором находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/group_open_http_svc/set_unread_msg_num | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная в учетной записи администратора приложения. Подробную информацию о создании подписи см. в разделе [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса. Значение всегда `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример пакета запроса

Этот пример устанавливает количество непрочитанных сообщений указанного участника группы.
Если количество непрочитанных сообщений, указанное этим API, больше текущего количества сообщений в группе, количество непрочитанных сообщений будет установлено на текущее количество сообщений в группе.

```
{    "GroupId": "@TGS#2CLUZEAEJ",  // Целевая группа (обязательно)    "Member_Account": "bob",  // Целевой участник группы (обязательно)    "UnreadMsgNum":5          // Количество непрочитанных сообщений целевого участника}
```

### Поля пакета запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| GroupId | String | Да | ID целевой группы |
| Member_Account | String | Да | Целевой участник группы |
| UnreadMsgNum | Integer | Да | Количество непрочитанных сообщений целевого участника |

### Пример пакета ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка. |
| ErrorInfo | String | Информация об ошибке |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка. |

## Коды ошибок

Код состояния HTTP, возвращаемый этим API, всегда равен 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ErrorCode` и `ErrorInfo` соответственно.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В таблице ниже описаны коды ошибок, характерные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Повторите попытку. |
| 10003 | Неправильная команда. |
| 10004 | Неверный параметр. Проверьте описание ошибки и устраните проблему. |
| 10007 | Нет прав доступа. Например, обычный участник в открытой группе пытается удалить других пользователей из группы, но это может сделать только администратор приложения. |
| 10010 | Группа не существует или была удалена. |
| 10015 | Неверный ID группы. Убедитесь, что используется правильный ID группы. |

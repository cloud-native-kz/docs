# Получение данных группы разговоров пользователя

## Обзор функции

Этот API поддерживает пакетное получение данных групп разговоров пользователя.

> **Примечание:** Каждый пользователь может создать до 20 групп разговоров. Сеанс может присоединиться к нескольким группам. Лимит емкости сеанса на группу составляет 1000. Превышение этого лимита вернет код ошибки: [51008](https://www.tencentcloud.com/document/product/1047/34348). Верхний лимит для тегов сеанса и пользовательских полей составляет 1000 сеансов. Превышение этого лимита вернет код ошибки: [51008](https://www.tencentcloud.com/document/product/1047/34348). Функции группировки разговоров и маркировки требуют включения **Ultimate Edition или Enterprise Edition**. Количество облачных сеансов, которые можно получить, ограничено [Chat Basic Package](https://www.tencentcloud.com/document/product/1047/67650) "Number of Cloud-based Session List Pulls". Если количество сгруппированных или отмеченных тегами сеансов превышает соответствующий лимит, полное получение сеансов будет недоступно. Вы можете обновить версию пакета Chat, чтобы увеличить лимит облачных сеансов.

## Описание вызова API

### Примерный URL запроса

```
https://xxxxxx/v4/recentcontact/get_contact_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [обзор RESTful API](https://trtc.io/document/34620?product=chat&menulabel=serverapi).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Силиконовая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/recentcontact/get_contact_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Дополнительную информацию см. в разделе **App Admin** в [Login Authentication](https://trtc.io/document/33517?platform=javascript&product=chat&menulabel=coresdk). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробные сведения см. в [Generating UserSig](https://trtc.io/document/34385?product=chat&menulabel=serverapi). |
| random | Случайное 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Примерный запрос

```
{    "From_Account":"user0",    "StartIndex":0}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| From_Account | String | Да | Заполните UserID, запросите получение группы разговоров/метки для этого пользователя. |
| StartIndex | Integer | Да | Начальный индекс для получения текущей страницы. Введите `0` для первого получения и введите значение `NextStartIndex`, возвращенное в последний раз. |

### Примерный ответ

```
{    "ContactItem": [        {            "Type": 1,            "To_Account": "teacher4",            "StandardMark": "111001",            "ContactGroupId": [                1,                2            ],            "Timestamp": 1670843110        },        {            "Type": 2,            "ToGroupId": "@TGS#1N3RSUYG2",            "StandardMark": "111001",            "CustomMark": "abcd",            "Timestamp": 1672998266        }    ],    "GroupItem": [        {            "GroupName": "test1x",            "GroupId": 1        },        {            "GroupName": "test10",            "GroupId": 2        }    ],    "CompleteFlag": 1,    "NextStartIndex": 0,    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": ""}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: не удалось |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: не удалось |
| ErrorInfo | String | Информация об ошибке |
| ContactItem | Integer | Объект разговора |
| Type | String | Тип разговора: 1 — персональный разговор; 2 — групповой разговор |
| ToGroupId | String | Назначение только для группового разговора, заполните ID группы стороны разговора |
| To_Account | String | Назначение только для персонального разговора, заполните UserID стороны разговора |
| Timestamp | Integer | Временная метка последнего обновления группы разговоров |
| StandardMark | String | Возможность стандартной метки, например `011111110` |
| CustomMark | String | Пользовательская метка |
| ContactGroupId | Array | Группа, которой принадлежит разговор |
| CompleteFlag | Integer | Обязательно |
| NextStartIndex | Integer | Обязательно |

## Коды ошибок

HTTP код состояния, возвращаемый этим API, всегда 200, если не возникает ошибка сети (например, ошибка 502). Конкретный код ошибки и детали можно найти в полях ответа `ResultCode`, `ResultInfo`, `ErrorCode` и `ErrorInfo`.

Для общих кодов ошибок (60000-79999) см. [Коды ошибок](https://trtc.io/document/34348?platform=javascript&product=chat&menulabel=coresdk).

В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 50001 | Запрошенный UserID не был импортирован в бэкэнд Tencent Cloud Chat. Пожалуйста, импортируйте. |
| 50002 | Неправильный параметр запроса. Проверьте свой запрос в соответствии с описанием ошибки. |
| 50003 | Запрос требует разрешения администратора приложения. |
| 50004 | Ошибка внутреннего сервера. Пожалуйста, попробуйте еще раз. |
| 50005 | Истечение времени ожидания сети. Попробуйте позже. |
| 51006 | При изменении метки разговора количество разговоров пусто или превышает верхний лимит 100. |
| 51007 | Не удалось заменить GroupID на GroupCode из-за внутренней ошибки или распада группы. |
| 51008 | Общее количество разговоров в группе разговоров превышает 1000. |
| 51009 | Группа разговоров не существует при попытке удаления. |
| 51010 | Количество групп разговоров превышает верхний лимит 20. |
| 51011 | Имя группы разговоров содержит более 32 байтов. |
| 51012 | Превышено максимальное количество разговоров, закрепленных вверху. |
| 51013 | Если запрашиваемое приложение не входит в Pro Edition, Pro Plus Edition, Enterprise Edition, будет сообщена ошибка при изменении стандартной метки разговора или группы разговоров. |

# Создание данных группы разговоров

## Обзор функции

Данные группы разговоров независимы от недавних контактов. REST API поддерживает добавление, удаление, изменение и запрос полей для групп разговоров, стандартных меток разговоров и пользовательских тегов. Этот API поддерживает создание данных группы разговоров.

> **Примечание:** Каждый пользователь может создать до 20 групп разговоров. Сеанс может присоединиться к нескольким группам. Лимит емкости сеанса на группу составляет 1000. Превышение этого лимита вернет код ошибки: [51008](https://www.tencentcloud.com/document/product/1047/34348). Теги сеанса и пользовательские поля сеанса имеют объединенный верхний лимит в 1000 сеансов. Превышение этого лимита вернет код ошибки: [51008](https://www.tencentcloud.com/document/product/1047/34348). Примечание: функция группирования и маркировки разговоров требует включения **Ultimate Edition или Enterprise Edition**. Количество облачных сеансов, которые можно получить, ограничено параметром "количество получаемых разговоров" в [базовом пакете Chat](https://www.tencentcloud.com/document/product/1047/67650). Если количество сгруппированных или помеченных сеансов превышает соответствующий лимит, полное получение сеансов будет недоступно. Вы можете обновить версию пакета IM для увеличения лимита облачных сеансов.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/recentcontact/create_contact_group?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://trtc.io/document/34620?product=chat&menulabel=serverapi).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/recentcontact/create_contact_group | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Для получения дополнительной информации см. раздел **App Admin** в [Login Authentication](https://trtc.io/document/33517?platform=javascript&product=chat&menulabel=coresdk). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения подробной информации см. [Generating UserSig](https://trtc.io/document/34385?product=chat&menulabel=serverapi). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Пример запроса

```
{    "From_Account":"user15",    "GroupContactItem":[        {            "GroupName":"test0",            "ContactItem": [                {                    "Type": 1,                    "To_Account": "user0"                }            ]        }    ]}
```

### Поля запроса

| Поле | Тип | Обязательное | Описание |
| --- | --- | --- | --- |
| From_Account | String | Да | Заполните UserID, запрос на создание группы разговоров для этого пользователя. |
| GroupContactItem | Array | Да | Группа разговоров для добавления. За раз можно добавить только одну группу разговоров. |
| GroupName | String | Да | Имя группы разговоров для добавления, может содержать до 32 байт |
| ContactItem | Array | Да | Список разговоров для добавления |
| Type | Integer | Да | Тип разговора: 1: личное сообщение. 2: групповой разговор |
| ToGroupId | String | Нет | Назначьте значение только для группового разговора, заполните ID группы стороны разговора. |
| To_Account | String | Нет | Назначьте значение только для личного сообщения, заполните UserID стороны разговора. |

### Пример ответа

```
{    "GroupResultItem": [        {            "GroupItem": {                "GroupName": "test2",                "GroupId": 2            },            "ResultItem": [                {                    "ContactItem": {                        "Type": 1,                        "To_Account": "user1"                    },                    "ResultCode": 0,                    "ResultInfo": ""                }            ]        }    ],    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": "",    "ErrorDisplay": ""}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно; `FAIL`: ошибка |
| ErrorCode | Integer | Код ошибки. `0`: успешно; другие значения: ошибка |
| ErrorInfo | String | Информация об ошибке |
| GroupResultItem | Array | Результат добавления группы разговоров |
| GroupItem | Object | Объект группы разговоров |
| GroupName | String | Имя группы разговоров |
| GroupId | Integer | ID группы разговоров |
| ResultItem | Array | Результат операции |
| ContactItem | Integer | Объект разговора |
| Type | Integer | Тип разговора: 1: личное сообщение. 2: групповой разговор |
| ToGroupId | String | Назначьте значение только для группового разговора, заполните ID группы стороны разговора |
| To_Account | String | Назначьте значение только для личного сообщения, заполните UserID стороны разговора |
| ResultCode | Integer | Код ошибки в результате операции разговора. `0`: успешно |
| ResultInfo | String | Описание ошибки в результате операции разговора |

## Коды ошибок

Код HTTP статуса, возвращаемый этим API, всегда равен 200, если не происходит ошибка сети (например, ошибка 502). Конкретный код ошибки и подробности можно найти в полях ответа `ResultCode`, `ResultInfo`, `ErrorCode` и `ErrorInfo`.

Для общих кодов ошибок (60000–79999) см. [Error Codes](https://trtc.io/document/34348?platform=javascript&product=chat&menulabel=coresdk).

В следующей таблице описаны коды ошибок, относящиеся к этому API:

| Код ошибки | Описание |
| --- | --- |
| 50001 | Запрашиваемый UserID не был импортирован в серверную часть Tencent Cloud Chat. Пожалуйста, произведите импорт. |
| 50002 | Неправильный параметр запроса. Проверьте ваш запрос в соответствии с описанием ошибки. |
| 50003 | Запрос требует разрешения администратора приложения. |
| 50004 | Внутренняя ошибка сервера. Пожалуйста, попробуйте еще раз. |
| 50005 | Истечение времени ожидания сети. Попробуйте позже. |
| 51006 | При изменении метки разговора количество разговоров пусто или превышает верхний лимит в 100. |
| 51007 | Ошибка при замене GroupID на GroupCode из-за внутренней ошибки или распуска группы. |
| 51008 | Общее количество разговоров в группе разговоров превышает 1000. |
| 51009 | Группа разговоров не существует при попытке удаления. |
| 51010 | Количество групп разговоров превышает верхний лимит в 20. |
| 51011 | Имя группы разговоров содержит более 32 байт. |
| 51012 | Превышено максимальное количество разговоров, закрепленных наверху. |
| 51013 | Если запрашиваемое приложение не является Pro Edition, Pro Plus Edition или Enterprise Edition, при изменении стандартной метки разговора или группы разговоров будет выдана ошибка. |

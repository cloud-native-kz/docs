# Отправка сообщений в официальный аккаунт

## Обзор функции

- Администраторы приложения могут использовать этот API для отправки обычных сообщений всем подписчикам официального аккаунта.
- Максимальная частота отправки для одного официального аккаунта составляет 1 сообщение/сек, не более 2 массовых рассылок в час.
- Если содержание двух сообщений идентично в течение 5 минут, второе сообщение будет отклонено как дубликат.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/official_account_open_http_svc/send_official_account_msg?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, измененные при вызове этого API, и их описания. Дополнительную информацию о параметрах см. в [Обзоре RESTful API](https://www.tencentcloud.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенный домен для страны/региона, где находится ваш SDKAppID: Китай: `console.tim.qq.com`, Сингапур: `adminapisgp.im.qcloud.com`, Сеул: `adminapikr.im.qcloud.com`, Франкфурт: `adminapiger.im.qcloud.com`, Токио: `adminapijpn.im.qcloud.com`, Кремниевая долина: `adminapiusa.im.qcloud.com`, Джакарта: `adminapiidn.im.qcloud.com` |
| v4/official_account_open_http_svc/send_official_account_msg | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунт администратора приложения. Дополнительную информацию см. в [Администраторы приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98) |
| usersig | Подпись, созданная аккаунтом администратора приложения. Дополнительную информацию см. в [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Примеры пакетов запросов

- **Базовая форма**
Отправка обычных сообщений официального аккаунта.

```
{  "Official_Account": "@TOA#_2C5SZEAEF",  "Random": 9875,  "MsgBody": [ // Message body, consists of an element array, see field description for details, messages with the same content within five minutes are considered duplicates      {          "MsgType": "TIMTextElem", // Text          "MsgContent": {              "Text": "red packet"          }      },      {          "MsgType": "TIMFaceElem", // Expression          "MsgContent": {              "Index": 6,              "Data": "abc\\u0000\\u0001"          }      }  ],  "CloudCustomData": "your cloud custom data",  "OfflinePushInfo": {      "PushFlag": 0, // Normal push,      "Desc": "Offline Push Content",      "Ext": "This is the transmitted content",      "AndroidInfo": {          "Sound": "android.mp3"      },      "ApnsInfo": {          "Sound": "apns.mp3",          "BadgeMode": 1, // This field is omitted or 0 means counting is required, 1 means this message does not require counting, i.e., the number on the upper right corner icon does not increase          "Title":"apns title", // apns title          "SubTitle":"apns subtitle", // apns subtitle          "Image":"www.image.com" // image url      }  }}
```

- **Указание того, что сообщение не обновляет последний контакт**
Если в сообщении указан SendMsgControl, установка NoLastMsg означает, что последний контакт не обновляется.

```
{   "Official_Account": "@TOA#_2C5SZEAEF",   "Random": 9875,   "SendMsgControl":["NoLastMsg"],// Specifies that the message does not trigger an update to the recent contact session.   "MsgBody": [ // Message body, consists of an element array, see field description for details, messages with the same content within five minutes are considered duplicates       {           "MsgType": "TIMTextElem", // Text           "MsgContent": {               "Text": "red packet"           }       },       {           "MsgType": "TIMFaceElem", // Expression           "MsgContent": {               "Index": 6,               "Data": "abc\\u0000\\u0001"           }       }   ] }
```

- **Указание того, что сообщение не вызывает обратный вызов**
Если функция обратного вызова включена, пользователи могут контролировать, вызывает ли одно сообщение обратный вызов через ForbidCallbackControl, по умолчанию инициируется обратный вызов.

```
{  "Official_Account": "@TOA#_2C5SZEAEF",  "Random": 9875,  "ForbidCallbackControl":[          "ForbidBeforeSendMsgCallback",          "ForbidAfterSendMsgCallback"], // Prohibit callback control option  "MsgBody": [ // Message body, consists of an element array, see field description for details, messages with the same content within five minutes are considered duplicates      {          "MsgType": "TIMTextElem", // Text          "MsgContent": {              "Text": "red packet"          }      },      {          "MsgType": "TIMFaceElem", // Expression          "MsgContent": {              "Index": 6,              "Data": "abc\\u0000\\u0001"          }      }  ]}
```

- **Указание того, что сообщение не будет сохранено в режиме «оффлайн» или ротамер**
Если OnlineOnlyFlag указан в теле сообщения и его значение больше 0, это означает, что сообщение будет отправлено только в режиме онлайн и не будет сохранено для использования в режиме оффлайн или роуминга.

```
  {      "Official_Account": "@TOA#_2C5SZEAEF",      "Random": 9875,      "OnlineOnlyFlag": 1, // Indicates that the message is only sent online (only online subscribers can receive it), and is not stored for offline or roaming      "MsgBody": [ // Message body, consists of an element array, see field description for details, messages with the same content within five minutes are considered duplicates          {              "MsgType": "TIMTextElem", // Text              "MsgContent": {                  "Text": "red packet"              }          },          {              "MsgType": "TIMFaceElem", // Expression              "MsgContent": {                  "Index": 6,                  "Data": "abc\\u0000\\u0001"              }          }      ]  }
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| Official_Account | String | Обязательное | Пользователь официального аккаунта, отправляющий сообщения. |
| Random | Integer | Обязательное | Целое число без знака 32 бита (диапазон: от 0 до 4294967295). Если содержание и значение Random двух сообщений совпадают в течение 5 минут, второе сообщение будет считаться дубликатом и отклонено. |
| MsgBody | Array | Обязательное | Тело сообщения, дополнительную информацию см. в [Формат сообщения](https://www.tencentcloud.com/document/product/1047/33527). |
| OfflinePushInfo | Object | Дополнительное | Конфигурация уведомления об автономной отправке, дополнительную информацию см. в [Описание OfflinePushInfo](https://www.tencentcloud.com/document/product/1047/77715). |
| ForbidCallbackControl | Array | Дополнительное | Переключатель запрета обратного вызова сообщения, применяется только к отдельным сообщениям, ForbidBeforeSendMsgCallback указывает на запрет обратного вызова перед отправкой сообщения, ForbidAfterSendMsgCallback указывает на запрет обратного вызова после отправки сообщения |
| OnlineOnlyFlag | Integer | Дополнительное | 1 указывает, что сообщение отправляется только подписчикам в режиме онлайн, по умолчанию 0 указывает на отправку всем подписчикам |
| SendMsgControl | Array | Дополнительное | Разрешения на отправку сообщений, NoLastMsg применяется только к отдельным сообщениям, указывает на отсутствие обновления последнего разговора (Если OnlineOnlyFlag сообщения установлен на 1, то это поле не должно использоваться.) |
| CloudCustomData | String | Дополнительное | Пользовательские данные сообщения (хранятся в облаке, будут отправлены собеседнику и могут быть получены даже после удаления и переустановки приложения). |

### Примеры пакетов ответов

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "MsgTime": 1497249503,    "MsgKey": "89541_1_1572870301"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса: OK: успешная обработка, FAIL: ошибка |
| ErrorCode | Integer | Идентификатор ошибки: 0: успех, значение отличное от нуля: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| MsgTime | Integer | Временная метка отправки сообщения, соответствующая серверному времени |
| MsgKey | String | Уникальный идентификатор сообщения, используется для отзыва. Длина не должна превышать 50 символов |

## Коды ошибок

Если не происходит ошибка сети (например, ошибка 502), код возврата HTTP для этого интерфейса всегда составляет 200. Реальный код ошибки и сообщение об ошибке указаны в ErrorCode, ErrorInfo в теле ответа.
Распространенные коды ошибок (60000–79999) можно найти в документе [Коды ошибок](https://www.tencentcloud.com/document/product/1047/34348).
Приватные коды ошибок для этого API выглядят следующим образом:

| Код ошибки | Описание |
| --- | --- |
| 10002 | Внутренняя ошибка сервера. Пожалуйста, попробуйте еще раз. |
| 10004 | Неверные параметры. Проверьте запрос в соответствии с описанием ошибки. |
| 10007 | Недостаточно прав на выполнение операции. |
| 10010 | Пользователь официального аккаунта не существует или когда-то существовал, но был удален. |
| 10015 | ID пользователя официального аккаунта неверен, проверьте, правильно ли заполнен ID пользователя официального аккаунта. |
| 10016 | Бэкэнд приложения отказывает в этой операции через обратный вызов третьей стороны. |
| 10017 | Невозможно отправить сообщение из-за отключения звука, проверьте, был ли отправитель переведен в режим отключения звука. |
| 10023 | Частота отправки сообщений превышает лимит, пожалуйста, увеличьте интервал между отправками двух сообщений. |
| 80002 | Содержание сообщения слишком длинное, в настоящее время поддерживается до 12K байт сообщений, пожалуйста, отрегулируйте длину сообщения. |

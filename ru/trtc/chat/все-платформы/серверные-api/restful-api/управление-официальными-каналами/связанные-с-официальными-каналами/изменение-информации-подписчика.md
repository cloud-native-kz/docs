# Изменение информации подписчика

## Обзор функции

Этот API используется администратором приложения для изменения соответствующей информации подписчиков.

> **Примечание:** Функция доступна только для Enterprise. Для её активации [свяжитесь с нами](https://trtc.io/zh/contact). Вы можете [попробовать](https://trtc.io/document/34577?product=chat#.E5.89.8D.E6.8F.90.E6.9D.A1.E4.BB.B6) эту функцию в консоли. Пробная версия позволяет создавать до 5 публичных аккаунтов, действительных в течение одного месяца.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/official_account_open_http_svc/modify_subscriber_info?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, изменяемые при вызове этого API, и их описания. Подробное описание параметров см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя, соответствующее стране/региону SDKAppID: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/official_account_open_http_svc/modify_subscriber_info | API запроса. |
| sdkappid | SDKAppID, назначаемый консолью Chat при создании приложения. |
| identifier | Аккаунт администратора приложения. Подробнее см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробнее см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295. |
| contenttype | Значение всегда `json`. |

### Максимальная частота вызовов

200 раз/сек

### Пример пакета запроса

```
{    "Official_Account": "@TOA#_test_for_penn",    "Subscriber_Account": "107867",    "CustomString": "test_modify_subscriber_custom_string"}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| Official_Account | String | Да | ID официального канала, который необходимо изменить. Используйте поле OfficialAccountUserID, возвращаемое API при создании. |
| Subscriber_Account | String | Да | ID подписчика, информация которого должна быть изменена. |
| CustomString | String | Да | Пользовательское поле подписчика, которое может использоваться слоем приложения для реализации некоторых специальных требований функций. |

### Пример пакета ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки: OK: успешно. FAIL: ошибка. |
| ErrorCode | Integer | Код ошибки: 0: успешно. Другие значения: ошибка. |
| ErrorInfo | String | Информация об ошибке. |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), код возврата HTTP для этого API всегда равен 200. ErrorCode и ErrorInfo в пакетах ответа представляют фактический код ошибки и информацию об ошибке.
Для распространённых кодов ошибок (60000–79999) см. раздел [Коды ошибок](https://www.tencentcloud.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 6017 | Недействительный параметр запроса. Пожалуйста, измените и повторите попытку. |
| 10004 | Внутренняя ошибка сервера. Пожалуйста, попробуйте снова. |
| 20001 | Недействительный пакет запроса. Пожалуйста, измените и повторите попытку. |
| 20005 | Внутренняя ошибка сервера. Пожалуйста, попробуйте снова. |
| 130002 | Недостаточно прав для выполнения операции. Пожалуйста, проверьте параметры запроса на основе сообщения об ошибке. |
| 130004 | Пользователь официального канала не существует или когда-то существовал, но теперь был удалён. |
| 130007 | Неправильный ID пользователя официального канала. Пожалуйста, измените и повторите попытку. |
| 130008 | ID пользователя официального канала уже используется. |
| 131000 | Для этого AppId не активирована служба официального канала. |

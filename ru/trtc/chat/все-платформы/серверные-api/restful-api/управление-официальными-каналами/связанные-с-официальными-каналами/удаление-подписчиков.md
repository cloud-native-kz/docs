# Удаление подписчиков

## Обзор функции

Этот API используется администратором приложения для отписания пользователя от официального канала и удаления его из списка подписчиков.

> **Примечание:** Функция доступна только для Enterprises. Чтобы включить её, [свяжитесь с нами](https://trtc.io/zh/contact). Вы можете [опробовать](https://trtc.io/document/34577?product=chat#.E5.89.8D.E6.8F.90.E6.9D.A1.E4.BB.B6) эту функцию в консоли. Пробная версия позволяет создавать до 5 открытых аккаунтов с действительностью один месяц.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/official_account_open_http_svc/delete_subscriber?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, изменяемые при вызове этого API, и их описания. Подробную информацию о параметрах см. в [обзоре RESTful API](https://www.tencentcloud.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя, соответствующее стране/региону SDKAppID: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/official_account_open_http_svc/delete_subscriber | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Аккаунт администратора приложения. Подробную информацию см. в [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, сгенерированная аккаунтом администратора приложения. Подробную информацию см. в [Генерирование UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Значение всегда `json`. |

### Максимальная частота вызовов

200 раз/секунду

### Пример пакетов запроса

```
{    "Official_Account": "@TOA#_@TOA#dOEF",  // Требуемый официальный канал (обязательно)    "SubscriberToDel_Account": [        ////UserID подписчика, который будет удален        "107867",        "966582476"    ]}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| Official_Account | String | Да | ID официального канала для отписки. Используйте поле OfficialAccountUserID, возвращаемое API при создании. |
| SubscriberList | Array | Да | Массив добавляемых подписчиков. |
| Subscriber_Account | String | Да | UserID подписчика для добавления. |

### Пример пакетов ответа

```
{    "ActionStatus": "OK",    "ErrorInfo": "",    "ErrorCode": 0,    "SubscriberList": [        {            "Subscriber_Account": "107867",            "Result": 1        },        {            "Subscriber_Account": "966582476",            "Result": 1        }    ]}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки: OK: успешно. FAIL: ошибка. |
| ErrorCode | Integer | Код ошибки: 0: успешно. Другие значения: ошибка. |
| ErrorInfo | String | Информация об ошибке. |
| SubscriberList | Array | Возвращаемые результаты удалённых подписчиков. |
| Subscriber_Account | String | Возвращаемый UserID подписчика. |
| Result | Integer | Результат удаления: 0: ошибка. 1: успешно. 2: подписка на этот официальный канал отсутствует. |

## Коды ошибок

За исключением ошибок сети (таких как ошибка 502), код возврата HTTP для этого API всегда равен 200. ErrorCode и ErrorInfo в пакетах ответа представляют фактический код ошибки и информацию об ошибке.
Для общих кодов ошибок (60000–79999) см. [Коды ошибок](https://www.tencentcloud.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 6017 | Недопустимый параметр запроса. Измените и повторите попытку. |
| 10004 | Внутренняя ошибка сервера. Пожалуйста, попробуйте снова. |
| 20001 | Недопустимый пакет запроса. Измените и повторите попытку. |
| 20005 | Внутренняя ошибка сервера. Пожалуйста, попробуйте снова. |
| 130002 | Недостаточные права доступа. Проверьте параметры запроса согласно сообщению об ошибке. |
| 130004 | Пользователь официального канала не существует или когда-то существовал, но был прекращён. |
| 130007 | Неправильный ID пользователя официального канала. Измените и повторите попытку. |
| 130008 | ID пользователя официального канала уже используется. |
| 131000 | Для этого AppId не включена служба официального канала. |

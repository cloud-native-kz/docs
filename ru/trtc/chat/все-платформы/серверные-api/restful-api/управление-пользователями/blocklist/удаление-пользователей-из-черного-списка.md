# Удаление пользователей из черного списка

## Описание функции

Этот API используется для удаления одного или нескольких пользователей из черного списка.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/sns/black_list_delete?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены и описаны только параметры, которые необходимо изменить при вызове этого API. Подробную информацию о других параметрах см. в разделе [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| https | Протокол запроса — HTTPS, метод запроса — POST. |
| xxxxxx | Страна/регион, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Силиконовая долина: `adminapiusa.im.qcloud.com` Джакарта: `adminapiidn.im.qcloud.com` |
| v4/sns/black_list_delete | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Это должна быть учетная запись администратора приложения. Подробную информацию см. в разделе [Администраторы приложений](https://intl.cloud.tencent.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробную информацию см. в разделе [Генерирование UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Введите случайное 32-битное неподписанное целое число в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса. Значение всегда `json`. |

### Максимальная частота вызовов

Максимальная частота вызовов составляет 200 раз в секунду.

### Пример пакета запроса

```
{"From_Account":"id","To_Account":["id1","id2","id3"]}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| From_Account | String | Обязательное | Идентификатор, для которого удаляются конкретные пользователи из его черного списка. |
| To_Account | Array | Обязательное | Список идентификаторов для удаления из черного списка. Количество To_Account в одном запросе не может превышать 1000. |

### Пример пакета ответа

```
{"ResultItem":[{"To_Account":"id1","ResultCode":0,"ResultInfo":""},{"To_Account":"id2","ResultCode":0,"ResultInfo":""},{"To_Account":"id3","ResultCode":30006,"ResultInfo":"Err_SNS_BlackListCheck_Check_Reverse_BlackList_Fail"}],"Fail_Account":["id3"],"ActionStatus":"OK","ErrorCode":0,"ErrorInfo":"","ErrorDisplay":""}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ResultItem | Array | Массив объектов результатов пакетного удаления из черного списка. |
| To_Account | String | Идентификатор, который вы запросили для удаления из черного списка. |
| ResultCode | Integer | Результат обработки To_Account. 0: успешно. Остальные: ошибка. |
| ResultInfo | String | Описание ошибки для To_Account. Если обработка успешна, это поле будет пусто. |
| Fail_Account | Array | Список пользователей, для которых не удалось выполнить удаление. Это поле возвращается только при наличии ошибок удаления. |
| ActionStatus | String | Результат обработки пакета запроса. OK: успешно. FAIL: ошибка. |
| ErrorCode | Integer | Код ошибки. 0: успешно. Остальные: ошибка. |
| ErrorInfo | String | Подробная информация об ошибке. |
| ErrorDisplay | String | Подробная информация об ошибке для отображения на клиенте. |

## Коды ошибок

Код возврата HTTP для этого API всегда 200, если не произойдет ошибка сети (например, ошибка 502). ErrorCode и ErrorInfo в пакете ответа представляют фактический код ошибки и информацию об ошибке соответственно.
Общие коды ошибок (от 60000 до 79999) см. в разделе [Коды ошибок](https://intl.cloud.tencent.com/document/product/1047/34348).
В следующей таблице описаны коды ошибок, специфичные для этого API.

| Код ошибки | Описание |
| --- | --- |
| 30001 | Параметр запроса некорректен. В этом случае проверьте параметры запроса на основе описания ошибки. |
| 30003 | Запрошенная учетная запись пользователя не существует. |
| 30004 | Запрос требует разрешений администратора приложения. |
| 30006 | Произошла внутренняя ошибка сервера. В этом случае попробуйте еще раз. |
| 30007 | Истекло время ожидания сетевого соединения. В этом случае повторите попытку позже. |
| 30008 | Произошел конфликт записи из-за одновременных операций записи. В этом случае рекомендуется использовать пакетный режим. |

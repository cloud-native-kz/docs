# Получение друзей

## Обзор функции

- Этот API используется для получения данных всех друзей постранично.
- Он не может получать данные профиля.
- Вам не нужно указывать поля для получения. По умолчанию будут возвращены все стандартные и пользовательские данные друзей.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/sns/friend_get?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для остальных параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Имя домена, соответствующее стране/региону, где находится ваш SDKAppID.Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Сеул: `adminapikr.im.qcloud.com`Токио: `adminapijpn.im.qcloud.com`Франкфурт: `adminapiger.im.qcloud.com`Кремниевая долина: `adminapiusa.im.qcloud.com`Джакарта: `adminapiidn.im.qcloud.com` |
| v4/sns/friend_get | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Аккаунт администратора приложения. Дополнительные сведения см. в разделе **App Admin** в разделе [Аутентификация входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробные сведения см. в разделе [Создание UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который всегда должен быть `json`. |

### Максимальная частота вызовов

200 вызовов в секунду

### Примеры запросов

```
{    "From_Account": "id",    "StartIndex": 0,    "StandardSequence": 0,    "CustomSequence": 0}
```

### Поля запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| From_Account | String | Да | UserID аккаунта, который запрашивает получение данных друзей |
| StartIndex | Integer | Да | Начальная точка страницы для получения |
| StandardSequence | Integer | Нет | `StandardSequence`, возвращенный для предыдущего получения данных друзей. Если значение поля `StandardSequence` совпадает со значением в бэкенде, бэкенд не вернет стандартные данные друзей. |
| CustomSequence | Integer | Нет | `CustomSequence`, возвращенный для предыдущего получения данных друзей. Если значение поля `CustomSequence` совпадает со значением в бэкенде, бэкенд не вернет пользовательские данные друзей. |

### Примеры ответов

```
{    "UserDataItem": [        {            "To_Account": "id1",            "ValueItem": [                {                    "Tag": "Tag_SNS_IM_AddSource",                    "Value": "AddSource_Type_Android"                },                {                    "Tag": "Tag_SNS_IM_Remark",                    "Value": "Remark1"                },                {                    "Tag": "Tag_SNS_IM_Group",                    "Value":["Group1","Group2"]                },                {                    "Tag": "Tag_SNS_IM_AddTime",                    "Value": 1563867420                },                {                    "Tag": "Tag_SNS_Custom_Test",                    "Value": "CustomData1"                }            ]        },        {            "To_Account": "id2",            "ValueItem": [                {                    "Tag": "Tag_SNS_IM_AddSource",                    "Value": "AddSource_Type_IOS"                },                {                    "Tag": "Tag_SNS_IM_Group",                    "Value":["Group1"]                },                {                    "Tag": "Tag_SNS_IM_AddTime",                    "Value": 1563867425                }            ]        }    ],    "StandardSequence": 88,    "CustomSequence": 46,    "FriendNum": 20,    "CompleteFlag": 1,    "NextStartIndex": 0,    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": "",    "ErrorDisplay": ""}
```

### Поля ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| UserDataItem | Array | Массив объектов друзей. Каждый объект друга содержит поле `To_Account` и массив `ValueItem`. |
| To_Account | String | UserID друга |
| ValueItem | Array | Массив для хранения данных друзей. Каждый элемент массива содержит поле `Tag` и поле `Value`. |
| Tag | String | Название поля друга |
| Value | String/Integer/Array | Значение поля друга. Дополнительные сведения см. в разделе [Поля цепи отношений](https://intl.cloud.tencent.com/document/product/1047/33521). |
| StandardSequence | Integer | Последовательность для стандартных данных друзей. Клиент может сохранить эту последовательность и вернуть ее в бэкенд через поле `StandardSequence` в следующем запросе. |
| CustomSequence | Integer | Последовательность для пользовательских данных друзей. Клиент может сохранить эту последовательность и вернуть ее в бэкенд через поле `CustomSequence` в следующем запросе. |
| FriendNum | Integer | Общее количество друзей |
| CompleteFlag | Integer | Тег конца страницы. Ненулевое значение указывает на то, что все данные друзей получены. |
| NextStartIndex | Integer | Начальная точка следующей страницы |
| ActionStatus | String | Результат запроса. `OK`: успешно. `FAIL`: ошибка. |
| ErrorCode | Integer | Код ошибки. `0`: успешно. Другие значения: ошибка. Подробные сведения о ненулевых результатах см. в разделе [Коды ошибок](#ErrorCode). |
| ErrorInfo | String | Подробная информация об ошибке |
| ErrorDisplay | String | Подробная информация, отображаемая на клиенте |

## Коды ошибок

| Код ошибки | Описание |
| --- | --- |
| 30001 | Некорректный параметр запроса. Проверьте запрос в соответствии с описанием ошибки. |
| 30003 | Запрошенный аккаунт не существует. |
| 30004 | Запрос требует прав администратора приложения. |
| 30006 | Внутренняя ошибка сервера. Повторите попытку. |
| 30007 | Истекло время ожидания сети. Повторите попытку позже. |

# Коды ошибок

## Общие коды ошибок

| Код ошибки | Описание |
| --- | --- |
| 0 | Операция успешна |
| -1 | Временно неклассифицированная общая ошибка |
| -2 | Превышен лимит частоты запросов, пожалуйста, повторите попытку позже |
| -1000 | SDKAppID не найден, пожалуйста, проверьте информацию приложения в [консоли TRTC](https://console.tencentcloud.com/trtc/allapp) |
| -1001 | Передача недопустимых параметров при вызове API, проверьте корректность параметров |
| -1002 | Не выполнена аутентификация, пожалуйста, вызовите API Login |
| -1003 | Ошибка получения разрешения, отсутствует авторизация на доступ к аудио/видео, пожалуйста, проверьте, включены ли разрешения устройства |
| -1004 | Эта функция требует дополнительный пакет, пожалуйста, включите соответствующий пакет при необходимости |

### Определение ошибок обратного вызова API локального отображения пользователя, управления видео, управления аудио

| Код ошибки | Описание |
| --- | --- |
| -1100 | Системная ошибка, не удалось открыть камеру. Проверьте, работает ли устройство камеры нормально |
| -1101 | Камера не имеет системной авторизации, проверьте системную авторизацию |
| -1102 | Камера занята, проверьте, использует ли другой процесс камеру |
| -1103 | Устройство камеры отсутствует, пожалуйста, подключите устройство камеры |
| -1104 | Системная ошибка, не удалось открыть микрофон. Проверьте, работает ли устройство микрофона нормально |
| -1105 | Микрофон не имеет системной авторизации, проверьте системную авторизацию |
| -1106 | Микрофон занят |
| -1107 | Устройство микрофона отсутствует |
| -1108 | Не удалось получить объект экрана, проверьте разрешение на запись экрана |
| -1109 | Не удалось включить демонстрацию экрана, проверьте, не включена ли уже демонстрация экрана в комнате |

### Определение ошибок обратного вызова API управления комнатой

| Код ошибки | Описание |
| --- | --- |
| -2100 | Комната не существует при входе, может быть закрыта |
| -2101 | Эта функция может быть использована только после входа в комнату |
| -2102 | Владелец комнаты не может покидать комнату, тип конференц-комнаты: сначала передайте право собственности на комнату, затем покидайте комнату. Тип гостиной: владелец комнаты может только закрыть комнату |
| -2103 | Эта операция не поддерживается в текущем типе комнаты |
| -2104 | Эта операция не поддерживается в текущем режиме речи |
| -2105 | Недопустимый пользовательский ID комнаты, должны быть печатаемые символы ASCII (0x20-0x7e), максимум 48 байт |
| -2106 | ID комнаты уже используется, пожалуйста, выберите другой ID комнаты |
| -2107 | Недопустимое имя комнаты, максимум 30 байт, если содержит китайские символы, должно быть в кодировке UTF-8 |
| -2108 | Пользователь уже находится в другой комнате, один экземпляр RoomEngine поддерживает вход пользователя только в одну комнату, для входа в другую комнату, пожалуйста, покидайте комнату или используйте новый экземпляр RoomEngine |

### Определение ошибок обратного вызова API информации пользователя комнаты

| --- | --- |
| -2200 | Пользователь не найден |
| -2201 | Пользователь не найден в комнате |

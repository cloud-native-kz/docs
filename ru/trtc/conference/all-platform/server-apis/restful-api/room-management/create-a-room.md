# Создание комнаты

## Обзор функции

Администраторы приложения могут создавать комнаты и планировать встречи через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/create_room?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, которые изменяются при вызове этого API. Для получения дополнительной информации см. раздел [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/create_room | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Для получения дополнительной информации см. раздел [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная аккаунтом администратора приложения. Для получения дополнительной информации см. раздел [Генерирование UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Примеры пакетов запроса

**Основная форма**

```
{  "RoomInfo":{      "RoomId": "room-test",      "RoomName": "room-name-test",      "RoomType": "Conference",      "Owner_Account": "user2",      "MaxMemberCount": 300,      "ScheduleStartTime" : 1693271355,      "ScheduleEndTime" : 1693272355,      "IsVideoDisabled": true,      "IsAudioDisabled": true,      "IsMessageDisabled": true,      "IsScreenSharingDisabled": true,      "IsCloudRecordingDisabled": true,      "CustomInfo": "custom123",      "IsSeatEnabled": true,      "MaxSeatCount": 16,      "TakeSeatMode": "ApplyToTake"  },  "ScheduleInviteeList_Account":[      "user1", "user2", "user3"  ]}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательный | ID комнаты, до 48 байт |
| RoomName | String | Опциональный | Имя комнаты, по умолчанию используется ID комнаты, до 100 байт |
| RoomType | String | Обязательный | Тип комнаты: Conference (Переговорная) |
| Owner_Account | String | Опциональный | ID хозяина (должен быть [импортированным](https://www.tencentcloud.com/document/product/1047/34953) аккаунтом), по умолчанию использует ID пользователя того, кто вызывает API |
| MaxMemberCount | Integer | Опциональный | Максимальное количество членов комнаты, значение по умолчанию при отсутствии указания: верхний предел оплачиваемого пакета, например, пробная версия составляет 20, при обновлении пакета это поле должно быть обновлено в соответствии с изменением информации о комнате. |
| ScheduleStartTime | Integer | Опциональный | Плановое время начала встречи, по умолчанию текущее время. |
| ScheduleEndTime | Integer | Опциональный | Плановое время окончания встречи, по умолчанию на 1 час позже времени начала, минимальная длительность встречи не может быть менее 5 минут, максимальная не может превышать 24 ч. |
| IsVideoDisabled | Bool | Опциональный | Отключить всё видео, по умолчанию false |
| IsAudioDisabled | Bool | Опциональный | Отключить весь звук, по умолчанию false |
| IsMessageDisabled | Bool | Опциональный | Отключить отправку текстовых сообщений всеми участниками, по умолчанию false |
| IsScreenSharingDisabled | Bool | Опциональный | Отключить общий доступ к экрану, по умолчанию false |
| IsCloudRecordingDisabled | Bool | Опциональный | Отключить облачную запись, по умолчанию false |
| CustomInfo | String | Опциональный | Пользовательская информация, максимум 500 байт |
| IsSeatEnabled | Bool | Опциональный | Поддерживает ли позиции сидений. По умолчанию не поддерживается. |
| MaxSeatCount | Integer | Опциональный | Максимальное количество сидений, по умолчанию ограничение пакета (максимум 20 по умолчанию). |
| TakeSeatMode | String | Опциональный | Режим сидений: FreeToTake (открытый микрофон), ApplyToTake (микрофон по запросу) |
| ScheduleInviteeList_Account | Array | Опциональный | Список плановых участников, до 300 участников |

> **Примечание:** После окончания встречи пользователи должны вручную вызвать интерфейс [Уничтожение комнаты](https://www.tencentcloud.com/document/product/647/60706#), чтобы завершить встречу. Если интерфейс "Уничтожение комнаты" не вызывается вручную, серверная часть попытается очистить комнату через 6 часов после окончания встречи. Условием очистки является отсутствие оставшихся участников в комнате.

### Примеры пакетов ответа

**Основная форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха; FAIL для ошибки. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, другие значения для ошибок. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса, требуется предоставить этот RequestId при локализации проблем. |

## Коды ошибок

Если не происходит сетевая ошибка (например, ошибка 502), HTTP-код состояния для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для распространённых кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Частные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр, проверьте запрос на корректность согласно описанию ошибки. |
| 100003 | ID комнаты уже существует, выберите другой ID комнаты. |
| 100007 | Нет информации об оплате, необходимо приобрести пакет в консоли. |
| 100010 | ID комнаты был использован, оператор является владельцем, вы можете использовать его напрямую. |
| 100011 | ID комнаты был использован чатом, вы можете изменить ID комнаты или уничтожить группу через интерфейс чата. |
| 100012 | Превышена частота создания комнаты. Один и тот же ID комнаты можно создавать только один раз в секунду. |

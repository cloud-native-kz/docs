# Уничтожение комнаты

## Обзор функции

Администраторы приложения могут уничтожать комнаты через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/destroy_room?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, которые изменяются при вызове этого API. Для получения дополнительной информации обратитесь к [Обзору RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/destroy_room | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации обратитесь к [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Для получения дополнительной информации обратитесь к [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{  "RoomId": "room-test"}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |

### Пример пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса: OK для успеха, FAIL для неудачи. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, другие значения для неудачи. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый с каждым запросом. При обращении в поддержку необходимо предоставить этот RequestId. |

## Коды ошибок

Если не происходит ошибка сети (например, ошибка 502), HTTP код состояния для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для распространенных кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр, проверьте корректность запроса на основе описания ошибки. |
| 100004 | Комната не существует или была уничтожена. |
| 100006 | Недостаточно прав на выполнение операции. |

# Обновление информации о комнате

## Обзор функции

Администраторы приложения могут обновлять информацию о комнате через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/update_room_info?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого API. Дополнительные сведения см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, в котором находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/update_room_info | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакетов запроса

**Основная форма**

```
{  "RoomInfo":{      "RoomId": "room-test",      "RoomName": "room-name-test",      "MaxMemberCount": 300,      "IsVideoDisabled": true,      "IsAudioDisabled": true,      "IsMessageDisabled": true,      "IsScreenSharingDisabled": true,      "IsCloudRecordingDisabled": true,      "CustomInfo": "custom123",      "TakeSeatMode": "ApplyToTake"  }}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательный | ID комнаты |
| RoomName | String | Опциональный | Имя комнаты, по умолчанию равно ID комнаты, максимум 100 байт |
| MaxMemberCount | Integer | Опциональный | Максимальное количество участников комнаты |
| IsVideoDisabled | Bool | Опциональный | Отключить видео для всех |
| IsAudioDisabled | Bool | Опциональный | Отключить аудио для всех |
| IsMessageDisabled | Bool | Опциональный | Запретить всем участникам отправлять текстовые сообщения |
| IsScreenSharingDisabled | Bool | Опциональный | Отключить общий доступ к экрану |
| IsCloudRecordingDisabled | Bool | Опциональный | Отключить облачную запись |
| CustomInfo | String | Опциональный | Пользовательская информация, максимум 500 байт |
| TakeSeatMode | String | Опциональный | Режим сидения: FreeToTake (открытый микрофон), ApplyToTake (микрофон по запросу) |

### Пример пакетов ответа

**Основная форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха, FAIL для ошибки. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, иные значения для ошибки. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Возвращается уникальный ID запроса для каждого запроса; необходимо предоставить этот RequestId при локализации проблем. |

## Коды ошибок

Если не происходит сетевая ошибка (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр, пожалуйста, проверьте правильность запроса на основе описания ошибки. |
| 100004 | Комната не существует, или она когда-то существовала, но теперь была уничтожена. |
| 100005 | Не участник комнаты |
| 100006 | Недостаточно прав для операции |

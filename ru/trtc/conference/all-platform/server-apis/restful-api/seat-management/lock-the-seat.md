# Заблокировать место

## Обзор функции

Администраторы приложения могут блокировать позицию места через этот API. После блокировки пользователи не могут занять это место. При блокировании, если кто-то находится на месте, пользователь будет удален перед блокировкой.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_mic/lock_seat?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, измененные при вызове этого API. Дополнительные сведения см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_mic/lock_seat | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должно быть учетной записью администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса — фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{  "RoomId":"room-test",  "Index":2,  "ForbidTaken":true,  "ForbidVideo":true,  "ForbidAudio":true}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| Index | Integer | Обязательное | Номер позиции места |
| ForbidTaken | Bool | Опциональное | Заблокировать позицию места |
| ForbidVideo | Bool | Опциональное | Отключить видеопоток |
| ForbidAudio | Bool | Опциональное | Отключить аудиопоток |

### Пример пакета ответа

**Базовая форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-7c1680be52734bdc8d6de398ab9505e7-O-Seq-57717"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK для успеха, FAIL для сбоя. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, другие значения для сбоя. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется указать этот RequestId при поиске неполадок. |

## Коды ошибок

Если не возникает сетевая ошибка (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию по [кодам ошибок](https://www.tencentcloud.com/document/product/647/57276#).
Закрытые коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр, пожалуйста, проверьте правильность запроса на основе описания ошибки. |
| 100004 | Комната не существует или она существовала, но теперь была удалена. |
| 100211 | Эта комната не поддерживает подключение к месту. |

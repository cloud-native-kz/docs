# Блокировка членов комнаты

## Обзор функции

Администраторы приложения могут блокировать членов комнаты через этот API. После блокировки члены будут удалены из комнаты и не смогут повторно войти в течение периода блокировки.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/ban_room_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого API. Для получения дополнительных сведений обратитесь к [обзору RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/ban_room_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть учетной записью администратора приложения. Для получения дополнительных сведений обратитесь к [администратору приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения дополнительных сведений обратитесь к [созданию UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Основная форма**

```
{  "RoomId":"room-test",  "MemberList_Account":["user1"],  "Reason":"you are banned because of irregularities",  "Duration":3600}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| MemberList_Account | Array | Обязательное | ID блокируемых членов, до 20 в одном запросе |
| Reason | String | Обязательное | Информация о блокировке, до 1000 байт |
| Duration | Integer | Обязательное | Продолжительность блокировки в секундах |

### Пример пакета ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-d4e0a71869c84352ac3a730fda71cdb0-O-Seq-47158"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK для успеха, FAIL для ошибки. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, остальные для ошибки. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется предоставить этот RequestId при выявлении проблем. |

## Коды ошибок

Если не происходит сетевой ошибки (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000-79999) см. документацию [код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Недопустимый параметр, проверьте правильность запроса на основе описания ошибки. |
| 100004 | Комната не существует, или она существовала, но теперь была удалена. |
| 100005 | Не-члены комнаты |
| 100006 | Недостаточные права доступа |

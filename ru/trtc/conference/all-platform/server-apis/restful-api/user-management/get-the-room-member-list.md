# Получить список участников комнаты

## Обзор функции

Администраторы приложения могут получить список участников комнаты через этот API.

## Описание вызова API

### Образец URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/get_room_member_list?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого API. Подробную информацию см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/get_room_member_list | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Подробную информацию см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Подробную информацию см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное целое число без знака, состоящее из 32 бит, в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Образец пакетов запроса

**Основная форма**

```
{  "RoomId": "room-test",  "Count": 10,  "Next": ""}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| Count | Integer | Опциональное | Количество получаемых участников, по умолчанию 50 |
| Next | String | Опциональное | Позиция участников, полученных в последний раз, используйте `""` для первого вызова, продолжайте получение, используя значение Next, возвращаемое в ответе |

### Образец пакетов ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-52f049ba12d741b7a84959700acdb1d5-O-Seq-56287",  "Response": {      "Next": "",      "MemberList": [          {              "Member_Account": "user2",              "NameCard": "123",              "Role": "RoomOwner",              "Marks": [                  1000,                  1002,                  1003              ],              "CustomInfo": ""          }      ]  }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха, FAIL для ошибки. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, остальные для ошибки. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается для каждого запроса и требуется при предоставлении этого RequestId при локализации проблем. |
| MemberList | Array | Список участников комнаты |
| Member_Account | String | ID участника комнаты |
| NameCard | String | Имя участника комнаты |
| Role | String | Разрешения участника комнаты: Owner, Admin, Member |
| Marks | Array | Тег участника комнаты |
| CustomInfo | String | Пользовательские поля участника комнаты |

## Коды ошибок

Если не произойдет ошибка сети (например, ошибка 502), код статуса HTTP для этого интерфейса всегда будет 200. Фактические коды и сообщения об ошибках передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр, проверьте корректность запроса на основе описания ошибки. |
| 100004 | Комната не существует, или она существовала ранее, но теперь была удалена. |
| 100005 | Не участник комнаты |
| 100006 | Недостаточно прав для выполнения операции |

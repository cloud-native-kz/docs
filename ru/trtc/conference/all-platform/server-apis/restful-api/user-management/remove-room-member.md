# Удаление члена комнаты

## Обзор функции

Администраторы приложения могут удалять членов из комнаты через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/kick_user_out?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого API. Для получения дополнительной информации см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/kick_user_out | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения дополнительной информации см. [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

- **Основная форма**
Создание комнаты

```
{  "RoomId":"room-test",  "MemberList_Account":["user1"],  "Reason":"you are kicked because of irregularities"}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| MemberList_Account | Array | Обязательное | ID членов, которых нужно удалить, не более 20 за один запрос |
| Reason | String | Обязательное | Информация об удалении, не более 1000 байт |

### Пример пакета ответа

- **Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-d4e0a71869c84352ac3a730fda71cdb0-O-Seq-47158"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK для успешного выполнения, FAIL для сбоя. |
| ErrorCode | Integer | Код ошибки. 0 для успешного выполнения, другие значения для сбоя. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом, требуется указать этот RequestId при решении проблем. |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), HTTP код состояния этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Закрытые коды ошибок для этого API указаны ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр, проверьте корректность запроса на основе описания ошибки. |
| 100004 | Комната не существует или она существовала, но теперь была удалена. |
| 100005 | Не член комнаты |
| 100006 | Недостаточно прав доступа |

# Обновление информации участника комнаты

## Обзор функции

Администраторы приложения могут обновлять информацию участников комнаты через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/update_room_member_info?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, которые были изменены при вызове этого API. Для получения дополнительной информации см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/update_room_member_info | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения дополнительной информации см. [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакетов запроса

- **Базовая форма**

```
{  "RoomId":"room-test",  "MemberInfo":{      "Member_Account": "user4",      "NameCard": "jack",      "Role": "Admin",      "CustomInfo": "custom info"}}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |
| Member_Account | String | Обязательно | ID участника комнаты |
| NameCard | String | Опционально | Прозвище участника комнаты, максимум 32 байта |
| Role | String | Опционально | Права участника комнаты: Admin, Member |
| CustomInfo | String | Опционально | Пользовательские поля определения участника комнаты, максимум 60 байт |

### Пример пакетов ответа

- **Базовая форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-8d5f107ce00f420597a2db9fbe7f71ca-O-Seq-102831"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха, FAIL для отказа. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, другие значения для отказа. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется при предоставлении этого RequestId при локализации проблем. |

## Коды ошибок

За исключением ошибок сети (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды и сообщения об ошибках передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неправильный параметр, проверьте правильность запроса на основе описания ошибки. |
| 100004 | Комната не существует или она существовала, но теперь была удалена. |
| 100005 | Не участник комнаты |
| 100006 | Недостаточные права на выполнение операции |

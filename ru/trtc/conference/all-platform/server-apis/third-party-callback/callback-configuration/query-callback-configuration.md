# Конфигурация обратного вызова запроса

## Обзор функции

Администраторы приложения могут получить обратный вызов через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_config/get_callback?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, которые изменяются при вызове этого API. Дополнительные сведения см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60722#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_config/get_callback | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса — фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Примеры пакетов запроса

**Базовая форма**
Просто передайте пусто.

```
{}
```

### Описание полей пакета запроса

### Примеры пакетов ответа

**Базовая форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-1cc8828fd3d84795ac866ced43b15b5c-O-Seq-61309",  "Response": {      "Url": "http://www.example.com/callback",      "CallbackCommandList": [          "Room.CallbackAfterCreateRoom",          "Room.CallbackAfterDestroyRoom",          "Room.CallbackUpdateRoomInfo",          "Room.CallbackAfterMemberEnter",          "Room.CallbackAfterMemberLeave",          "Mic.CallbackAfterSeatInfoChanged"      ]  }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK — успех, FAIL — ошибка. |
| ErrorCode | Integer | Код ошибки. 0 — успех, другие значения — ошибка. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом и требуемый для предоставления при локализации проблем. |
| Url | String | Адрес обратного вызова, должен начинаться с http/https, рекомендуется использовать более безопасный https. |
| CallbackCommandList | Array | Для команд, которые запускают обратный вызов, см. [Список команд обратного вызова](https://www.tencentcloud.com/document/product/647/60720#). |

## Коды ошибок

Если не происходит ошибка сети (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Параметр недействителен, проверьте корректность запроса на основе описания ошибки. |
| 100301 | Конфигурация обратного вызова не существует, вы можете создать ее с помощью интерфейса создания конфигурации обратного вызова. |

# Обзор обратных вызовов

Для облегчения управления формой функций вашего приложения RoomKit предоставляет возможности обратных вызовов.

## Обзор функций

Пользователи могут настроить обратный вызов на указанный URL через RESTful API. Когда выполненный CallbackCommand находится в списке конфигурации, будет активирован обратный вызов.

## Примечания

- На каждый sdkAppId можно установить только один обратный вызов.
- Убедитесь, что URL обратного вызова доступен.

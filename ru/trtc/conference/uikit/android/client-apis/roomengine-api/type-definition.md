### RoomType

Тип комнаты

| Перечисление | Значение | Описание |
| --- | --- | --- |
| CONFERENCE | 1 | Комната типа конференции, подходит для сценариев конференций и образования, эта комната может включить режимы свободного выступления, подачи запроса на выступление, прямой трансляции и другие режимы. |
| LIVE_ROOM | 2 | Комната типа прямой трансляции, подходит для сценариев трансляции, эта комната может включить режимы свободного выступления и управления микрофоном, места в этой комнате пронумерованы. |

### SpeechMode

Режим управления микрофоном

| Перечисление | Значение | Описание |
| --- | --- | --- |
| FREE_TO_SPEAK | 1 | Режим свободного выступления. |
| APPLY_TO_SPEAK | 2 | Режим подачи запроса на выступление. (Работает только в комнатах типа конференции) |
| SPEAK_AFTER_TAKING_SEAT | 3 | Режим прямой трансляции. |

### MediaDevice

Тип медиа-устройства комнаты

| Перечисление | Значение | Описание |
| --- | --- | --- |
| MICROPHONE | 1 | Микрофон |
| CAMERA | 2 | Камера |
| SCREEN_SHARING | 3 | Общий доступ к экрану |

### Role

Типы ролей в комнате

| Перечисление | Значение | Описание |
| --- | --- | --- |
| ROOM_OWNER | 0 | Владелец комнаты, обычно относится к создателю комнаты, имеет высшую власть в комнате |
| MANAGER | 1 | Администратор комнаты |
| GENERAL_USER | 2 | Обычный участник в комнате |

### VideoQuality

Качество видео

| Перечисление | Значение | Описание |
| --- | --- | --- |
| Q_360P | 1 | Низкое качество 360P |
| Q_540P | 2 | Стандартное определение 540P |
| Q_720P | 3 | Высокое определение 720P |
| Q_1080P | 4 | Ультра-четкое 1080P |

### AudioQuality

Качество аудио

| Перечисление | Значение | Описание |
| --- | --- | --- |
| SPEECH | 0 | Режим речи |
| DEFAULT | 1 | Режим по умолчанию |
| MUSIC | 2 | Режим музыки |

### VideoStreamType

Тип видеопотока

| Перечисление | Значение | Описание |
| --- | --- | --- |
| CAMERA_STREAM | 0 | Видеопоток камеры высокого качества |
| SCREEN_STREAM | 1 | Видеопоток общего доступа к экрану |
| CAMERA_STREAM_LOW | 2 | Видеопоток камеры низкого качества |

### ChangeReason

Причина изменения (причина изменения статуса аудио и видео пользователя: самостоятельное изменение или изменение владельцем/администратором комнаты)

| Перечисление | Значение | Описание |
| --- | --- | --- |
| BY_SELF | 0 | Самостоятельная операция |
| BY_ADMIN | 1 | Операция владельца комнаты или администратора |

### RequestAction

Тип запроса

| Перечисление | Значение | Описание |
| --- | --- | --- |
| INVALID_ACTION | 0 | Неверный запрос |
| REQUEST_TO_OPEN_REMOTE_CAMERA | 1 | Запрос на открытие камеры удаленного пользователя |
| REQUEST_TO_OPEN_REMOTE_MICROPHONE | 2 | Запрос на открытие микрофона удаленного пользователя |
| REQUEST_TO_CONNECT_OTHER_ROOM | 3 | Запрос на подключение к другой комнате |
| REQUEST_TO_TAKE_SEAT | 4 | Запрос на прямую трансляцию |
| REQUEST_REMOTE_USER_ON_SEAT | 5 | Запрос удаленному пользователю на прямую трансляцию |
| REQUEST_APPLY_TO_ADMIN_TO_OPEN_LOCAL_CAMERA | 6 | Запрос администратору на открытие локальной камеры |
| REQUEST_APPLY_TO_ADMIN_TO_OPEN_LOCAL_MICROPHONE | 7 | Запрос администратору на открытие локального микрофона |

### NetworkQuality

Качество сети

| Перечисление | Значение | Описание |
| --- | --- | --- |
| UNKNOWN | 0 | Не определено |
| EXCELLENT | 1 | Текущая сеть отличная |
| GOOD | 2 | Текущая сеть хорошая |
| POOR | 3 | Текущая сеть средняя |
| BAD | 4 | Текущая сеть плохая |
| VERY_BAD | 5 | Текущая сеть очень плохая |
| DOWN | 6 | Текущая сеть не соответствует минимальным требованиям TRTC |

### RoomInfo

Информация о комнате

| Поле | Тип | Описание |
| --- | --- | --- |
| roomId | String | ID комнаты |
| ownerId | String | ID хозяина, по умолчанию является создателем комнаты (только для чтения) |
| roomType | [RoomType](/document/product/647/54866#RoomType) | Тип комнаты |
| name | String | Имя комнаты, по умолчанию является ID комнаты |
| speechMode | [SpeechMode](/document/product/647/54866#SpeechMode) | Режим управления микрофоном |
| isCameraDisableForAllUser | boolean | Отключить открытие камеры (опционально при создании комнаты), значение по умолчанию: false |
| isMicrophoneDisableForAllUser | boolean | Отключить открытие микрофона (опционально при создании комнаты), значение по умолчанию: false |
| isMessageDisableForAllUser | boolean | Отключить отправку сообщений (опционально при создании комнаты), значение по умолчанию: false |
| maxSeatCount | int | Максимальное количество мест для микрофона (поддерживается только при входе в комнату и создании комнаты) |
| enableCDNStreaming | boolean | Включить трансляцию CDN (опционально при создании комнаты, для комнат прямой трансляции), значение по умолчанию: false |
| cdnStreamDomain | String | Домен push для прямой трансляции (опционально при создании комнаты, для комнат прямой трансляции), значение по умолчанию: пусто |
| createTime | long | Время создания комнаты (только для чтения) |
| memberCount | int | Количество участников в комнате (только для чтения) |

### LoginUserInfo

Информация входа пользователя

| Поле | Тип | Описание |
| --- | --- | --- |
| userId | String | ID пользователя |
| userName | String | Имя пользователя |
| avatarUrl | String | URL аватара пользователя |

### UserInfo

Информация о пользователе в комнате

| Поле | Тип | Описание |
| --- | --- | --- |
| userId | String | ID пользователя |
| userName | String | Имя пользователя |
| avatarUrl | String | URL аватара пользователя |
| userRole | [Role](/document/product/647/54866#Role) | Тип роли пользователя |
| hasAudioStream | boolean | Наличие аудиопотока, значение по умолчанию: false |
| hasVideoStream | boolean | Наличие видеопотока, значение по умолчанию: false |
| hasScreenStream | boolean | Наличие потока общего доступа к экрану, значение по умолчанию: false |

### SeatInfo

Информация о месте в комнате

| Поле | Тип | Описание |
| --- | --- | --- |
| index | int | Номер места для микрофона |
| userId | String | ID пользователя |
| isLocked | boolean | Заблокировано ли место для микрофона, по умолчанию false |
| isVideoLocked | boolean | Запрещено ли открытие камеры на месте для микрофона, по умолчанию false |
| isAudioLocked | boolean | Запрещено ли открытие микрофона на месте для микрофона, по умолчанию false |

### SeatLockParams

Параметры операции блокировки места

| Поле | Тип | Описание |
| --- | --- | --- |
| lockSeat | boolean | Заблокировать место для микрофона, по умолчанию false |
| lockVideo | boolean | Заблокировать камеру места для микрофона, по умолчанию false |
| lockAudio | boolean | Заблокировать микрофон места для микрофона, по умолчанию false |

### UserVoiceVolume

Громкость голоса пользователя в комнате

| Поле | Тип | Описание |
| --- | --- | --- |
| userId | String | ID пользователя |
| volume | int | Размер громкости, диапазон значений 0 - 100 |

| upLoss | int | Коэффициент потерь пакетов восходящего канала |
| downLoss | int | Коэффициент потерь пакетов нисходящего канала |

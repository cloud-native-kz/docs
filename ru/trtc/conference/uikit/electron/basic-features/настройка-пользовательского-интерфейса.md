# Кастомизация пользовательского интерфейса

В этой статье подробно описано, как кастомизировать пользовательский интерфейс TUIRoomkit. TUIRoomkit предлагает два метода кастомизации: первый — это простое использование Custom UI API, второй — замена существующих компонентов UI. Подробное описание этих методов приведено ниже.

## Схема 1: Тонкая настройка интерфейса

TUIRoomkit предоставляет серию [API](https://www.tencentcloud.com/document/product/647/54885#), позволяющих легко кастомизировать UI. В таблице ниже перечислены некоторые основные API и их функции:

| API | Описание |
| --- | --- |
| [setLanguage](https://www.tencentcloud.com/document/product/647/54885#51991a76-0777-4773-a2dd-91ce51b92b18) | Установить язык интерфейса. |
| [setTheme](https://www.tencentcloud.com/document/product/647/54885#5bb291c2-edb5-48b0-968a-db52ed2252ae) | Установить тему интерфейса. |
| [enableWatermark](https://www.tencentcloud.com/document/product/647/54885#daceee1b-175d-41a1-b495-d158fe01d63c) | Включить функцию текстового водяного знака в приложении. Подробнее см.: [Text Watermark](https://www.tencentcloud.com/document/product/647/60531#). |
| [disableTextMessaging](https://www.tencentcloud.com/document/product/647/54885#9ed81bb6-73a0-4b75-878e-22cc641b43bc) | Отключить функцию текстовых сообщений в приложении. После вызова этой функции пользователи не смогут отправлять и получать текстовые сообщения. |
| [disableScreenSharing](https://www.tencentcloud.com/document/product/647/54885#833fce4f-6e77-45bf-bea0-33a618b540fb) | Отключить функцию общего доступа к экрану в приложении. После вызова этой функции пользователи не смогут делиться своим экраном с другими. |
| [hideFeatureButton](https://www.tencentcloud.com/document/product/647/54885#49c62018-9bc4-4117-bf68-aa985b868890) | Скрыть определённые кнопки функций в приложении. Вызывая эту функцию и передавая соответствующие значения перечисления [FeatureButton](https://www.tencentcloud.com/document/product/647/54885#6f28a0a9-c315-400e-a73f-b1fbd0b039eb), соответствующие кнопки будут скрыты из пользовательского интерфейса. |

## Схема 2: Экспорт и модификация исходного кода UIKit

Вы можете непосредственно изменить предоставленный исходный код UI для настройки пользовательского интерфейса TUIRoomKit в соответствии с вашими потребностями. Выполните следующий скрипт node для автоматического копирования исходного кода интерфейса TUIRoomKit в ваш проект (путь по умолчанию `./src/components/TUIRoom`).

> **Примечание:**
> После экспорта исходного кода необходимо вручную изменить ссылку компонента TUIRoom с адреса пакета npm на относительный адрес пути исходного кода TUIRoom.

Vue3

Vue2

```
node ./node_modules/@tencentcloud/roomkit-electron-vue3/scripts/eject.js
```

```
node ./node_modules/@tencentcloud/roomkit-electron-vue2.7/scripts/eject.js
```

### 1. Замена значков

Вы можете непосредственно изменить компоненты значков в папке `/TUIRoom/assets/icons/svg`, чтобы обеспечить согласованность цвета и стиля значков во всём приложении. При замене значков сохраняйте имена файлов неизменными.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/235499ef1bfc11efac7f5254007bbd8c.png)

### 2. Настройка макета UI

Вы можете настроить макет UI интерфейса видеоконференции с несколькими участниками, изменив различные компоненты в папке `/TUIRoom/components/`

```
- components/        - Chat                Чат    - common              Компоненты общих значков    - ManageMember        Управление участниками    - RoomContent         Видео комнаты    - RoomFooter          Нижний колонтитул страницы комнаты    - RoomHeader          Верхний колонтитул страницы комнаты    - RoomHome            Главная страница    - RoomInvite          Приглашение участников    - RoomLogin           Страница входа    - RoomMore            Ещё    - RoomSetting         Параметры    - RoomSidebar        Компоненты панели инструментов
```

## Решение 3: Реализуйте собственный UI

Общая функциональность TUIRoomKit основана на TUIRoomEngine, SDK без UI. Вы можете полностью реализовать свой собственный пользовательский интерфейс на основе TUIRoomEngine. Дополнительные сведения см. в:

- [Руководство по интеграции TUIRoomEngine](https://www.tencentcloud.com/document/product/647/54844#)
- [Адрес API TUIRoomEngine](https://www.tencentcloud.com/document/product/647/54882#)

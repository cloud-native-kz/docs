# Кастомизация UI

В этой статье рассказывается о том, как кастомизировать пользовательский интерфейс `TUIRoomKit`. Мы предоставляем две решение на выбор: **решение точной настройки** и **решение пользовательского UI**.

## Решение 1: Решение точной настройки

Путем прямого изменения исходного кода UI, который мы предоставляем, вы можете настроить пользовательский интерфейс `TUIRoomKit`.

### Замена иконок

Вы можете напрямую изменить компоненты иконок в папке

`rtc_conference_tui_kit/assets/images`

чтобы убедиться, что стиль цветовой гаммы иконок согласован во всем приложении. Пожалуйста, сохраняйте имя файла иконки без изменений при замене.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a7b05aaa84f211ee8fb75254000524f8.png)

### Замена текстовых строк

Вы можете изменить файлы `en_us.dart` и `zh_cn.dart` в папке `rtc_conference_tui_kit/lib/common/languages` для изменения содержимого строк интерфейса видеоконференции.

## Решение 3: Решение полной кастомизации UI

Общая функциональность `TUIRoomKit` основана на `TUIRoomEngine`, SDK без UI. Вы можете полностью реализовать свой собственный интерфейс пользователя на основе `TUIRoomEngine`. Для получения дополнительной информации см. [адрес интерфейса API TUIRoomEngine](https://www.tencentcloud.com/document/product/647/57512#).

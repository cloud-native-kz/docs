# Чат в конференции

На видеоконференциях участники могут отправлять сообщения в режиме реального времени в области чата, делиться мнениями и идеями, а также создавать расслабленную и приятную среду общения благодаря обмену эмодзи и анимациями. Для поддержания порядка на встречах хост или администратор может установить запрет на отправку сообщений участниками в чат, обеспечивая сосредоточенность и эффективность содержания конференции. Благодаря гибкому использованию этих функций видеоконференции могут обеспечить эффективное и удобное общение для различных сценариев.

## Описание функции

В нижней строке меню интерфейса конференции вы найдете опцию **Чат**. Нажмите на неё, и вы будете перенаправлены на страницу чата. На странице чата участники могут общаться с другими участниками, отправляя пользовательские текстовые сообщения. Такой дизайн облегчает связь между участниками в режиме реального времени без влияния на выступающего.

### Взаимодействие с эмодзи

Нажмите на **значок эмодзи** в строке редактирования сообщений интерфейса чата, чтобы отобразить список эмодзи. Нажмите на соответствующее эмодзи, чтобы отправить его и показать пальец вверх за блестящую речь выступающего.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/8cc34702fc9611eeac26525400ae4d13.png)

> **Примечание:** Чтобы соблюдать авторские права на дизайн эмодзи, проект Conference не включает большие срезы элементов эмодзи. Перед официальным коммерческим запуском замените их на разработанные вами эмодзи или другие пакеты эмодзи, права на которые вы принадлежат. По умолчанию **авторские права на набор эмодзи Little Yellow Face принадлежат Tencent Cloud** и могут быть лицензированы за плату. Если вы хотите получить лицензию, вы можете [Отправить обращение](https://console.tencentcloud.com/workorder/category), чтобы связаться с нами.

### Управление разрешениями чата в конференции

Хост/администратор может установить разрешения на чат для участника при управлении участниками. Обычные участники, которым запрещено говорить, не смогут редактировать или отправлять текстовые сообщения или эмодзи.

### Интеграция исходного кода модуля

1. Клонируйте/загрузите код из [Github](https://github.com/tencentyun/TUIRoomKit), а затем скопируйте подпапку `tuichat` из каталога Android на тот же уровень, что и каталог app вашего текущего проекта.
2. Найдите файл `setting.gradle` в корневом каталоге проекта и добавьте в него следующий код. Этот код импортирует `tuichat` как локальный модуль в ваш текущий проект.

```
  include ':tuichat'
```

3. Найдите файл `build.gradle` в каталоге app и добавьте в него следующий код. Этот код объявляет зависимость текущего приложения от вновь добавленного компонента `tuichat`.

```
  api project(':tuichat')
```

4. Добавьте хранилище maven и поддержку Kotlin в файл `build.gradle` корневого проекта (на том же уровне, что и `settings.gradle`):

```
  buildscript {    repositories {        mavenCentral()        maven { url "https://mirrors.tencent.com/nexus/repository/maven-public/" }    }    dependencies {        classpath 'com.android.tools.build:gradle:7.0.0'        classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:1.5.31"    }}
```

Если вы используете Gradle 8.x, вам необходимо добавить следующий код.

```
buildscript {    repositories {        mavenCentral()        maven { url "https://mirrors.tencent.com/nexus/repository/maven-public/" }    }    dependencies {        classpath 'com.android.tools.build:gradle:8.0.2'        classpath "org.jetbrains.kotlin:kotlin-gradle-plugin:1.9.0"    }}
```

> **Примечание:** Соответствие между версиями Kotlin, Gradle и AGP можно [посмотреть здесь](https://kotlinlang.org/docs/gradle-configure-project.html#apply-the-plugin).

### Доступ к виджету чата

Чтобы импортировать виджет чата с помощью CocoaPods, выполните следующие действия:

1. Добавьте следующие зависимости в свой `Podfile`.

```
pod 'TUIChat'          # [Optional] Chat widget
```

2. Выполните следующую команду для установки компонентов.

```
pod install
```

# Виртуальный фон

TUIRoomKit запустил функцию виртуального фона, позволяющую пользователям устанавливать размытый фон во время групповых встреч, скрывать реальную среду встречи, защищать конфиденциальность и добавлять веселья на встречу. В этой статье подробно описано, как использовать эту функцию в компоненте TUIRoomKit.

## Требования подготовки

Перед использованием функции виртуального фона Tencent Cloud необходимо перейти в консоль и активировать услугу групповых встреч для вашего приложения, а также приобрести пакет **Advanced Interactive Version/Large Room Interactive Version**. Подробные инструкции см. в разделе [Активировать услугу](https://www.tencentcloud.com/document/product/647/59973#).

## Включение размытого фона

> **Примечание:** **H5 не поддерживается, применимо только к Web PC.****Требуется TUIRoomKit v2.3.3 или более поздняя версия.**

Решение UI в TUIRoomKit поддерживает установку виртуального фона. Вы можете отобразить кнопку функции виртуального фона на интерфейсе, вызвав интерфейс [enableVirtualBackground](https://www.tencentcloud.com/document/product/647/54880#ac620717-1b58-42ed-b8a3-589bf0ad545e). Нажатие этой кнопки активирует функцию виртуального фона.

Vue3

Vue2

```
import { conference } from '@tencentcloud/roomkit-web-vue3';conference.enableVirtualBackground();
```

```
import { conference } from '@tencentcloud/roomkit-web-vue2.7';conference.enableVirtualBackground();
```

## Часто задаваемые вопросы

### Отсутствует ответ или задержка при активации виртуального фона?

- Убедитесь, что вы приобрели пакет групповой встречи **Advanced Interactive Version/Large Room Interactive Version**, подробности см. в разделе [Активировать услугу](https://www.tencentcloud.com/document/product/647/59973#).
- При плохом сетевом соединении файл модели виртуального фона может быть загружен не полностью, что приведет к сбою активации виртуального фона.

### Могу ли я активировать виртуальный фон с отключенной камерой?

Нет.

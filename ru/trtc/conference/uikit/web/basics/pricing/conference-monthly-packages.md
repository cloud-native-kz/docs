# Пакеты конференций по месячной подписке

TRTC Conference (TUIRoomKit) подходит для сценариев многопользовательского аудио-/видеочата, таких как **деловые встречи, вебинары и онлайн-образование**. Интегрировав этот компонент, вы сможете добавить управление комнатой, управление участниками, совместное использование экрана и другие функции в свое приложение всего за три шага в течение одного дня.

## Разбор стоимости

Плата за конференцию включает плату за пакет конференции по месячной подписке и плату по факту использования.

- Плата за пакет конференции по месячной подписке: **Компонент TRTC Conference можно использовать только после покупки пакета Conference по месячной подписке.** Сведения о ценах и функциях пакетов см. в документе [Пакеты конференций по месячной подписке](https://www.tencentcloud.com/document/product/647/59409#914711df-1caf-4b81-b9ab-ba37b02c08a0).
- Плата по факту использования: После исчерпания квоты вашего пакета Conference по месячной подписке использование будет оплачиваться в режиме оплаты по факту, **поэтому обратите внимание, что даже если вы подписались на пакет по месячной подписке, могут быть взяты дополнительные платежи.** Плата по факту использования включает [плату за MAU вне пакета](https://www.tencentcloud.com/document/product/1047/34350) и [Биллинг по длительности аудио и видео](https://www.tencentcloud.com/document/product/647/42734). Плата по факту использования включает [плату за MAU в Chat](https://trtc.io/document/67651?product=pricing#ef86ff3a-429e-4065-aa12-254a92e2b10d) и [плату за длительность аудио/видео](https://www.tencentcloud.com/document/product/647/42734). Кроме того, если вы используете базовые функции TRTC, могут взиматься также [плата за запись в облаке](https://www.tencentcloud.com/document/product/647/45176), [плата за доставку записи](https://www.tencentcloud.com/document/product/647/45176), [плата за MixTranscoding](https://www.tencentcloud.com/document/product/647/47631) и [плата за трансляцию](https://www.tencentcloud.com/document/product/647/47631).

> **Уведомление о правилах расчета MAU и использовании пакетов:** **Метод расчета MAU** – Метод расчета MAU следующий: после вызова операции TUILogin.login и установления длительного соединения с серверной частью Call значение MAU увеличивается на 1. То есть один пользователь, входящий в Call в течение месяца, учитывается как 1 MAU, а повторные входы одного и того же пользователя не накапливаются в MAU. Кроме того, если один и тот же пользователь использует Chat, Call, Live или Conference в рамках одного приложения (SDKAppID), он будет учтен только один раз как один MAU. Пожалуйста, разумно используйте операцию TUILogin.login в соответствии со своим сценарием бизнеса, чтобы избежать чрезмерно высокого значения MAU. **Ограниченная по времени акция на пакеты Conference:** На ограниченный период минуты аудио-/видеовызовов, включенные в платный пакет Conference по месячной подписке, теперь можно использовать для компенсации использования, создаваемого **любым приложением Real-Time Communication (RTC) (SDKAppID)** под вашей учетной записью Tencent Cloud, а не ограничиваться только конкретным приложением, связанным с покупкой. Дата окончания этого предложения будет объявлена официально на нашем веб-сайте.

## Детали биллинга

- Функции TRTC Conference предоставляются на уровне приложения (`SDKAppID`). Если у вас есть несколько приложений (`SDKAppID`), которым нужно использовать функции TRTC Conference, вам необходимо приобрести несколько пакетов TRTC Conference по месячной подписке.
- После истечения срока действия пакета TRTC Conference по месячной подписке, привязанного к вашему приложению (`SDKAppID`), **функции пакета станут недоступны**. Чтобы избежать прерывания работы вашего бизнеса, мы рекомендуем вам включить **автоматическое продление** для приобретенного пакета по месячной подписке в [**Консоли TRTC > Conference**](https://console.trtc.io/conference) > **Выберите приложение** > **Включить автоматическое продление**.
- **Вы можете подписаться только на один пакет TRTC Conference для одного приложения** (`SDKAppID`) **одновременно**. Например, для приложения **"test_trtc"**, если вы уже подписались на **Conference Lite**, до истечения срока действия пакета вы не сможете обновить его на **Conference Standard или Pro**. Если вам нужно обновить пакет, пожалуйста, [отправьте заявку](https://console.tencentcloud.com/workorder/category).
- Пакеты TRTC Conference необходимо приобрести перед их использованием для вычитания вашего использования. Период действия отсчитывается со дня покупки по тому же дню следующего месяца. Например, если вы приобрете пакет 1 марта 2023 г., период его действия будет с 1 марта 2023 г. по 1 апреля 2023 г.
- Конкретные инструкции по получению бесплатной пробной версии или приобретению официальной версии см. в документе [Активирование сервиса (TUIRoomKit)](https://www.tencentcloud.com/document/product/647/59832#ats).
- Вы можете обратиться к [Инструкциям по возврату средств за пакеты TRTC по месячной подписке](https://www.tencentcloud.com/document/product/647/58588) для ознакомления с правилами возврата для пакетов Conference по месячной подписке.

> **Примечание:** Ресурсы пакета: Ресурсы пакета могут вычитать использование функций TRTC Conference, включая длительность вызовов и MAU. Если вы используете пробную версию, сервис станет недоступным для вашего приложения после исчерпания пакета. Если вы используете Lite, Standard или Pro, после исчерпания пакета дополнительное использование будет оплачиваться по тарифам оплаты по факту. Бесплатные минуты: Каждая учетная запись Tencent Cloud получает 10 000 бесплатных минут в каждом цикле использования (цикл использования — один месяц) после покупки пакета TRTC Conference. Бесплатные минуты могут вычитать использование как функций TRTC Conference, так и базовых функций TRTC, таких как запись в облаке и микширование в облаке. Чтобы получить дополнительные сведения, обратитесь к документу [Бесплатные минуты](https://www.tencentcloud.com/document/product/647/42735). Минуты пакета: Минуты пакета могут вычитать длительность ваших конференций. Они действительны в течение одного месяца и истекают в конце каждого цикла использования. Базовые функции TRTC: В дополнение к функциям TRTC Conference вы можете также использовать базовые функции TRTC, за которые взимается дополнительная плата. Сведения о биллинге см. в документе [Биллинг записи в облаке и доставки записи](https://www.tencentcloud.com/document/product/647/45176) и [Биллинг микширования-транскодирования и трансляции на CDN](https://www.tencentcloud.com/document/product/647/47631).

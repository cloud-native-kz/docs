# Введение в продукт

## Обзор

Conference (TUIRoomKit) — это продукт, предназначенный для сценариев многопользовательского аудио- и видеообщения, таких как деловые встречи, вебинары и онлайн-образование. Путем интеграции этого продукта вы можете добавить в приложение функции управления комнатами, управления участниками, общего доступа к экрану и другие функции всего в три шага за один день, что позволит вам быстро запустить свой бизнес. Основные функции компонента показаны ниже:

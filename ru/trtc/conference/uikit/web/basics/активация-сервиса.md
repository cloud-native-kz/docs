# Активация сервиса

## Активация пробной версии сервиса

Для лучшего изучения функций TUIRoomKit мы предоставляем 14-дневную пробную версию для каждого SDKAppID (пробная версия не включает бесплатное время разговора). Каждый SDKAppID можно попробовать бесплатно два раза, с периодом действия 14 дней каждый раз. При этом общее количество попыток для всех SDKAppID под одной учётной записью составляет 10.

1. Войдите в [консоль Tencent RTC](https://console.trtc.io) и нажмите **Create Application**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/51e8e2d8c5ea11f0a7ca5254001c06ec.png)

2. Во всплывающем окне введите **application name**, выберите **Conference** и выберите подходящий **deployment region**. Затем нажмите **Create**. Это создаст приложение TRTC, привязанное к пробной версии TRTC Room.

> **Примечание:** Регион развёртывания по умолчанию для данных сервиса TRTC находится в Сингапуре, а хранилище для данных сервиса Chat находится в выбранном вами центре данных.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/efae06a5c5ea11f0a7ca5254001c06ec.png)

3. После создания приложения вы автоматически будете переведены на страницу сведений о продукте Conference. На этом этапе вы быстро создали приложение и успешно получили пробную версию TRTC Conference (TUIRoomKit). Вы можете просмотреть информацию на текущей [**странице сведений о продукте Conference**](https://console.trtc.io/conference) или в [**Мои приложения**](https://console.trtc.io/app) и обратитесь к [руководству интеграции](https://www.tencentcloud.com/document/product/647/54845) для интеграции. `SDKAppID` и `SDKSecretKey` будут использоваться в руководстве интеграции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/22259b94c5eb11f086d9525400e889b2.png)

## Покупка официальных версий

**Вам необходимо приобрести пакет Conference monthly package для использования TUIRoomKit**. Сведения о цене и сравнении функций пакета Conference monthly package см. в [объяснении выставления счёта за пакет Conference Monthly Package](https://www.tencentcloud.com/document/product/647/59409). **Для покупки выполните следующие шаги:**

1. Посетите [страницу покупки Conference](https://console.trtc.io/subscription/buy/conference), выберите **приложение (SDKAppID) и версию для покупки**, **и рекомендуется включить автоматическое возобновление, чтобы избежать влияния на использование бизнеса**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Subscribe now**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cd31d325c6a411f09e745254007c27c5.png)

2. Перейдите на страницу платежа, чтобы завершить платёж. После завершения покупки вы можете перейти на [**страницу сведений о продукте Conference**](https://console.trtc.io/conference) или в [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [руководство интеграции](https://www.tencentcloud.com/document/product/647/54845) для интеграции.

## Возобновление официальной версии

1. Посетите [**страницу сведений о продукте Conference**](https://console.trtc.io/conference), нажмите **Renewal / Upgrade**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dc50914dc6a411f0a93d52540044a08e.png)

2. Посетите [страницу покупки Conference](https://console.trtc.io/subscription/buy/conference), подтвердите приложение (SDKAppID) для покупки, выберите версию того же пакета приложения, **и рекомендуется включить автоматическое возобновление, чтобы избежать влияния на использование бизнеса**. После включения, когда **баланс учётной записи будет достаточным, он будет автоматически возобновляться ежемесячно по истечении срока действия**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Subscribe now**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/dc528d08c6a411f0b011525400bf7822.png)

3. Перейдите на страницу платежа, чтобы завершить платёж. После завершения покупки вы можете перейти на [**страницу сведений о продукте Conference**](https://console.trtc.io/conference) или в [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения. См. [руководство интеграции](https://www.tencentcloud.com/document/product/647/54845) для интеграции.

## Обновление на официальную версию

1. Посетите [**страницу сведений о продукте Conference**](https://console.trtc.io/conference), нажмите **Renewal / Upgrade**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f7eac4f6c6a411f0a4a55254001c06ec.png)

2. Посетите [страницу покупки Conference](https://console.trtc.io/subscription/buy/conference), подтвердите приложение (SDKAppID), которое требует обновления, выберите **версию выше, чем пакет приложения**. После подтверждения информации о покупке и согласия с соответствующим соглашением нажмите **Subscribe now**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f7e1112fc6a411f0b011525400bf7822.png)

3. Перейдите на страницу платежа, чтобы завершить платёж. После завершения покупки вы можете перейти на [**страницу сведений о продукте Conference**](https://console.trtc.io/conference) или в [**Мои приложения**](https://console.trtc.io/app) для просмотра информации о версии приложения.

## Автоматическое возобновление

Действия для включения автоматического возобновления в консоли следующие:

1. Посетите [**страницу сведений о продукте Conference**](https://console.trtc.io/conference).
2. Выберите **приложение, которое требует автоматического возобновления**.
3. В информации о продукте Conference нажмите кнопку **Auto-renewable** **Enable**. Появится диалоговое окно подтверждения. Нажмите **Enable**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0ccd8876c6a511f0a4a55254001c06ec.png)

# Пометить членов комнаты

## Обзор функции

Администраторы приложений могут помечать членов комнаты через этот API.

## Описание вызова API

### Примеры URL запроса

```
https://xxxxxx/v4/room_engine_http_srv/mark_room_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, которые изменяются при вызове этого API. Для получения дополнительной информации см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_srv/mark_room_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения дополнительной информации см. [Генерация UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Примеры пакетов запроса

**Основная форма**

```
{  "RoomId":"room-test",  "CommandType":1,  "MemberList":[      {          "Member_Account":"user2",          "Marks":[1000,1002,1003]      }  ]}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| CommandType | String | Обязательное | 1: Установить метки, 2: Удалить метки |
| MemberList | Array | Обязательное | Список учетных записей для установки, максимум 10 учетных записей одновременно |
| Marks | Array | Обязательное | Метка должна быть числом больше или равным 1000, и в каждой комнате можно установить максимум 10 различных меток, не более 3 меток за один раз |

### Примеры пакетов ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-ee0a1a5f2c70432f8273a2b279a5fa8f-O-Seq-57703"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха, FAIL для неудачи. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, остальное для неудачи. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется предоставить этот RequestId при поиске проблем. |

## Коды ошибок

Если не произойдет ошибка сети (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для общих кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Недопустимый параметр, проверьте корректность запроса на основе описания ошибки. |
| 100004 | Комната не существует, или она когда-то существовала, но теперь была удалена. |
| 100005 | Не члены комнаты |
| 100006 | Недостаточно прав для выполнения операции |
| 100009 | Количество меток превышает верхний предел |

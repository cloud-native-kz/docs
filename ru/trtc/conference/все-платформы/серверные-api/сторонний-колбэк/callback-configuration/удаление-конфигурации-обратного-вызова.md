# Удаление конфигурации обратного вызова

## Обзор функции

Администраторы приложения могут удалить конфигурацию обратного вызова через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_config/delete_callback?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого API. Дополнительные сведения см. в документе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_config/delete_callback | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Дополнительные сведения см. в документе [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в документе [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Примеры пакетов запросов

**Базовая форма**

Просто передайте пустое значение.

```
{}
```

### Описание полей пакета запроса

Отсутствует.

### Примеры пакетов ответов

**Базовая форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-1cc8828fd3d84795ac866ced43b15b5c-O-Seq-61309"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK — успех, FAIL — ошибка. |
| ErrorCode | Integer | Код ошибки. 0 — успех, другие значения — ошибка. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом. Необходимо предоставить этот RequestId при локализации проблем. |

## Коды ошибок

При отсутствии ошибки сети (например, ошибка 502) HTTP статус-код для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для распространенных кодов ошибок (60000–79999) см. документацию [Коды ошибок](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API представлены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Параметр недействителен, пожалуйста, проверьте корректность запроса в соответствии с описанием ошибки. |

# Управление учетными записями

Войдите в [консоль Chat](https://console.trtc.io/), перейдите к целевому приложению для доступа к странице управления приложением и выберите **Chat > Users** на левой боковой панели. Вы можете управлять учетными записями в соответствии с потребностями вашего бизнеса.

> **Примечание:** Live и Instant Messaging (Chat) используют одну и ту же систему учетных записей.

## Создание учетной записи

1. На странице **Account Management** нажмите **Create account**.
2. Настройте следующие параметры в появившемся диалоговом окне:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebe6df37256c11f0b0b1525400e889b2.png)

  - Account Type: выберите **General** или **Admin**. Роль администратора приложения имеет наивысший уровень прав управления. Она позволяет использовать REST API для выполнения операций, таких как создание или расформирование группы, отправка сообщений всем членам. Каждое приложение поддерживает до 10 администраторов.
  - Username: введите идентификатор пользователя. Это поле является обязательным.
  - Nickname: введите никнейм. Это поле является необязательным.
  - Profile Photo: введите URL фотографии профиля пользователя. Это поле является необязательным.
3. Нажмите **Confirm**.
4. После создания учетной записи вы сможете увидеть имя пользователя, никнейм, тип, фотографию профиля и время создания учетной записи в списке учетных записей.

## Удаление учетных записей

1. По умолчанию удаление учетной записи запрещено. Однако это ограничение может быть снято для разрешения удаления учетной записи. Для этого нажмите на **Click here to remove the restriction**. В появившемся диалоговом окне подтверждения, после ознакомления с соответствующими рисками, нажмите **Confirm** для снятия ограничения.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebda054e256c11f08caa5254005ef0f7.png)

2. На странице Account Management выберите одну или несколько учетных записей, которые вы хотите удалить из списка, и нажмите **Batch Delete**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebc8edbf256c11f0b0b1525400e889b2.png)

3. В появившемся диалоговом окне подтверждения, после понимания связанных рисков, нажмите **Confirm** для удаления учетной записи. Обратите внимание, что после удаления данные не могут быть восстановлены. Будьте осторожны при выполнении этой операции.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebdb894f256c11f09e67525400bf7822.png)

## Редактирование учетной записи

1. На странице **Account Management** найдите учетную запись для редактирования и нажмите **Edit** в столбце **Operation**.
2. Настройте следующие параметры в появившемся диалоговом окне:
  - Username: это поле не может быть отредактировано.
  - Nickname: вы можете редактировать никнейм пользователя. Это поле является необязательным.
  - Account Type: это поле не может быть отредактировано.
  - Gender: вы можете редактировать пол пользователя. Это поле является необязательным.
  - Birth Date: вы можете редактировать дату рождения пользователя. Это поле является необязательным.
  - Location: вы можете редактировать местоположение. Это поле является необязательным.
  - What's Up: вы можете редактировать статус. Это поле является необязательным.
  - Friend Request Verification: вы можете редактировать режим проверки запроса дружбы.
  - Language: вы можете редактировать язык. Это поле является необязательным.
  - Profile Photo: вы можете редактировать фотографию профиля. Это поле является необязательным.
  - Message settings: вы можете выбрать параметры сообщений. Это поле является необязательным.
  - Friending: вы можете указать, запрещает ли администратор пользователю инициировать запрос дружбы. Это поле является необязательным.
  - Level: вы можете редактировать уровень. Это поле является необязательным.
  - Role: вы можете редактировать роль. Это поле является необязательным.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebe43cdd256c11f08caa5254005ef0f7.png)

3. Нажмите **Confirm**.

## Экспорт учетных записей

1. Экспорт одной учетной записи: На странице **Account Management** найдите учетную запись для экспорта и нажмите **Export** в столбце **Operation**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebe6023b256c11f08caa5254005ef0f7.png)

2. Массовый экспорт: На странице **Account Management** выберите учетные записи для экспорта и нажмите **Batch Export** выше списка учетных записей.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebe44fbf256c11f08caa5254005ef0f7.png)

3. В появившемся диалоговом окне **Export successful** нажмите **Download**.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ebd8799b256c11f0b0b1525400e889b2.png)

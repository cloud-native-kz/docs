# Создание боя

## Описание функции

Администратор приложения может создавать бои через этот API.

## Описание вызова API

### Пример URL запроса:

```
https://xxxxxx/v4/live_engine_http_pk/create_battle?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Описание параметров запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого интерфейса. Подробнее о других параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметры | Описание |
| --- | --- |
| xxxxxx | Исключительное доменное имя, соответствующее стране/региону SDKAppID: Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_pk/create_battle | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Подробнее см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подписи, созданные учетной записью администратора приложения. Конкретные действия см. в [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883) |
| random | Введите случайное 32-битное целое число без знака, диапазон значений 0 - 4294967295 |
| contenttype | Формат запроса имеет фиксированное значение `JSON`. |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "FromRoomId":"pk-1",  //Caller's room ID in PK    "ToRoomIdList":["pk-2","pk-3"], //List of callee's room IDs in PK    "Duration":60000, //Duration of PK    "ExtensionInfo":"extension pk"}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| FromRoomId | String | Обязательно | ID комнаты инициатора боя |
| ToRoomIdList | String | Обязательно | Список ID комнат участников боя |
| Duration | Integer | Обязательно | Продолжительность боя |
| ExtensionInfo | String | Опционально | Поле расширения информации боя |

> **Примечание:** Перед инициированием боя убедитесь, что все участвующие комнаты находятся в подключенном состоянии. Этот API не проверяет статус подключения. При инициировании боя SDK уведомит владельцев комнат всех участвующих комнат.

### Пример текста пакета ответа:

**Базовая форма**

```
{  "ErrorCode": 0,   "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "requestId-123-456",  "Response":{    "BattleId": "battleid1",    "Result": [         {            "RoomId": "pk-2",             // 0 indicates initiate successfully             // 3 indicates that it is already in another PK.              // 5 System error             "ResultCode": 0,              "ResultMessage": "",         },         {            "RoomId": "pk-3",            "ResultCode": 0,             "ResultMessage": "",         }     ],  }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ErrorCode | Integer | Код ошибки. 0 указывает на успех; ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| ActionStatus | String | Результат обработки запроса. OK указывает на успешную обработку, FAIL указывает на ошибку. |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом, необходимый для устранения неполадок. |
| BattleId | String | ID этого боя |
| Result | Array | Ситуация боя участника |
| RoomId | String | ID комнаты |
| ResultCode | Integer | Результат: 0 указывает на успех, 3 указывает на участие в бою, 5 указывает на системную ошибку. |
| ResultMessage | String | Информация о результатах боя |

## Описание кодов ошибок

Если не происходит сетевая ошибка (например, ошибка 502), код возврата HTTP этого API всегда равен 200. Реальный код ошибки и сообщение об ошибке находятся в тексте пакета ответа. Обозначаются `ErrorCode` и `ErrorInfo`.

Коды ошибок общего характера (60000–79999) см. в документации [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API указаны ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Параметр недействителен. Проверьте правильность запроса согласно описанию ошибки. |
| 100003 | ID комнаты уже существует. Выберите другой ID комнаты. |
| 100412 | Все вызванные комнаты находятся в бою. Повторите попытку позже. |
| 100415 | Инициатор находится в бою. Повторите попытку позже. |

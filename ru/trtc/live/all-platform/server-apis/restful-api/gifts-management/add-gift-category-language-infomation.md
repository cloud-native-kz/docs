# Добавление информации о языке категории подарков

## Описание функции

Администратор приложения может использовать этот API для добавления информации о языке для designированной категории подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/set_gift_category_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице указаны только параметры, задействованные при изменении при вызове этого API, и их описание. Дополнительные сведения о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе, соответствующем выделенному доменному имени: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/set_gift_category_language | Запрос API. |
| sdkappid | SDKAppID, назначенный консолью чата при создании приложения. |
| identifier | Должна быть учетная запись администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для конкретных операций см. раздел [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное беззнаковое целое число в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Основная форма**

```
{    "CategoryId":"CId2",    "Language":"en",    "CategoryName": "name",    "CategoryDesc": "desc"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательно | ID категории подарков. |
| Language | String | Обязательно | Типы языков категории подарков, поддерживаемые в настоящее время: "en", "zh-Hans", "zh-Hant". |
| CategoryName | String | Обязательно при первоначальной установке, опционально при последующих установках. | Название категории подарков на текущем языке. |
| CategoryDesc | String | Опционально. | Описание категории подарков на текущем языке. |

> **Примечание:** Многократная установка информации о языке категории подарков будет изменять только поля в запросе. Поля, которые не находятся в запросе, не будут изменены. При использовании SDK для получения списка подарков и классификации подарков на указанном языке, если информация для этого языка установлена, вернуть соответствующую информацию напрямую. В противном случае вернуть информацию языка по умолчанию, установленную при добавлении категории подарков.

### Пример тела пакета ответа

**Основная форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9d167940547545dfa63d9151288d0e83-O-Seq-285390"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для определения проблемы. |

## Описание кодов ошибок

При отсутствии ошибок сети (например, 502) код HTTP, возвращаемый этим API, составляет 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo полезной нагрузки ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Параметры запроса недопустимы. Проверьте, правильный ли запрос в соответствии с описанием ошибки. |
| 100006 | Недостаточные разрешения на операцию. Требуется учетная запись администратора. |
| 100012 | Частые вызовы. Максимальная частота вызовов составляет 1/с. |
| 102001 | Нет информации об оплате. Приобретите пакет в консоли. |

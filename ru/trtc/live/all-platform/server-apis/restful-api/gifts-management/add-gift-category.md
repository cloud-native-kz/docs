# Добавить категорию подарков

## Описание функции

Администратор приложения может использовать этот API для добавления информации о классификации подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/add_gift_category?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице указаны только параметры, задействованные при изменении при вызове этого API, и его описание. Для получения дополнительной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Страна/регион, где находится SDKAppID: выделенное доменное имяКитай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/add_gift_category | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учётной записью администратора приложения. Для конкретных операций см. [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз/секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "CategoryId":"CId2",    "ExtensionInfo":"{\\"key\\": \\"value\\"}",    "DefaultName":"name1",    "DefaultDesc" : "desc1"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательно | ID категории подарков, максимум 20 байт, до 3 категорий подарков. |
| ExtensionInfo | String | Опционально. | Пользовательская информация расширения должна быть строкой в формате JSON, максимум 100 байт. |
| DefaultName | String | Обязательно | Имя подарка по умолчанию, максимум 20 байт. |
| DefaultDesc | String | Опционально. | Описание подарка по умолчанию, максимум 60 байт. |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9d167940547545dfa63d9151288d0e83-O-Seq-285390"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не выполнена. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый для каждого запроса. Для устранения неполадок запроса следует предоставить его RequestId. |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например, 502), HTTP код возврата этого API равен 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo полезной нагрузки ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточные разрешения операции. Требуются операции учётной записи администратора. |
| 100012 | Частые вызовы. Максимальная частота вызовов — 1/сек. |
| 102007 | Превышен лимит количества категорий подарков. |
| 102001 | Нет информации об оплате. Приобретите пакет в консоли. |

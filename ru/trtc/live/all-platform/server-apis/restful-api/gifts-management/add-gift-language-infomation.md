# Добавление информации о языке подарка

## Описание функции

Администратор приложения может использовать этот API для добавления информации о языке подарка.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/set_gift_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, участвующие в изменении, и их описания при вызове этого API. Для получения дополнительной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе, где расположено выделенное доменное имя. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/set_gift_language | Запрос API. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Для конкретных операций см. [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение варьируется от 0 до 4294967295. |
| contenttype | Формат запроса фиксирован как `json`. |

### Максимальная частота вызовов

1 раз/секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "GiftId":"GId1",    "Language":"en",    "Name": "name1",    "Desc": "desc1"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательно | ID подарка, максимум 20 байт, до 3 категорий подарков. |
| Language | String | Обязательно | Тип языка подарка, поддерживаемые: "en", "zh-Hans", "zh-Hant". |
| Name | String | Обязательно при первичной настройке, опционально при последующих настройках | Имя подарка по умолчанию, максимум 20 байт. |
| Desc | String | Опционально. | Описание подарка по умолчанию, максимум 20 байт. |

> **Примечание:** Множественная настройка информации о языке подарка будет только изменять поле в текущем запросе. Поля, не указанные в запросе, не будут изменены. При использовании SDK для получения списка подарков и категорий при указанном языке, если информация о языке установлена, возвращается соответствующая информация. В противном случае возвращается информация языка по умолчанию, установленная при добавлении подарка.

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9d167940547545dfa63d9151288d0e83-O-Seq-285390"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на отказ. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращается для каждого запроса. RequestId требуется для определения проблемы. |

## Описание кодов ошибок

Если не происходит сетевая ошибка (например, ошибка 502), код возврата HTTP для этого API составляет 200. Фактический код ошибки и информация об ошибке указываются ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неправильные параметры запроса. Проверьте, правильный ли запрос в соответствии с описанием ошибки. |
| 100006 | Недостаточно разрешений на операцию. Требуется учетная запись администратора для выполнения операции. |
| 100012 | Частые вызовы, максимальная частота вызовов 1/с. |
| 102001 | Отсутствует информация об оплате, приобретите пакет в консоли. |

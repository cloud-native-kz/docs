# Добавить подарок

## Описание функции

Администратор приложения может использовать этот API для добавления информации о подарке.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/add_gift?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого интерфейса. Для получения дополнительной информации о других параметрах см. раздел [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: Прочие регионы в Китае: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/add_gift | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунт администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, сгенерированная аккаунтом администратора приложения. Для конкретных операций см. раздел [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883) |
| random | 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса — фиксированное значение: `json` |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "GiftId" :  "GId1",                       "IconUrl" :"https://www.xxx.com/xxx1.png",    "ResourceUrl" : "https://www.xxx.com/xxx2.png",                        "Coins" : 10,    "Level" : 1,    "ExtensionInfo" : "{\\"key\\": \\"value\\"}",    "DefaultName" :"",    "DefaultDesc" : ""}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательно | ID подарка, максимум 20 байт. |
| IconUrl | String | Опционально | Миниатюра подарка, максимум 200 байт. |
| ResourceUrl | String | Опционально | URL ресурса подарка, например URL полноэкранного видеоэффекта mp4, ресурсов svg, максимум 200 байт. |
| Coins | Integer | Обязательно | Цена подарка в виртуальной валюте. |
| Level | Integer | Опционально | Уровень подарка, по умолчанию 0, если оставлено пусто. |
| ExtensionInfo | String | Опционально | Пользовательская информация расширения для подарков должна быть в формате JSON-строки, максимум 100 байт. |
| DefaultName | String | Обязательно | Имя подарка по умолчанию, максимум 20 байт. |
| DefaultDesc | String | Опционально | Описание подарка по умолчанию, максимум 20 байт. |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-fecb13a5827b425d9be4b9c79ac10f67-O-Seq-284018"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для определения проблемы. |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например 502), код возврата HTTP этого API составляет 200. Фактические код ошибки и информация об ошибке находятся в полях ErrorCode и ErrorInfo полезной нагрузки ответа.

Распространенные коды ошибок (60000–79999), см. документ [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Частные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Недопустимые параметры запроса. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав на выполнение операции. Требуется аккаунт администратора. |
| 100012 | Частота вызовов слишком высока. Максимальная частота вызовов добавления подарков составляет 1/с. |
| 102006 | Превышен лимит количества подарков. |
| 102001 | Отсутствует информация об оплате. Приобретите пакет в консоли. |

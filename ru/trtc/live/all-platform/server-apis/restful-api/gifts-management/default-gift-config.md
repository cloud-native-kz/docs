# Конфигурация подарков по умолчанию

## Описание функции

Администраторы приложения могут настроить информацию о подарках по умолчанию через этот интерфейс. До настройки в учетной записи не допускаются никакие другие конфигурации подарков.

### Описание конфигурации по умолчанию

Конфигурация по умолчанию включает 3 категории подарков и 15 подарков. После настройки она может быть отображена в демонстрации. См. [Запуск демонстрационного примера](https://www.tencentcloud.com/document/product/647/60453).

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/copy_default_gift_config?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого интерфейса. Для получения дополнительной информации о других параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Другие в Китае: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/copy_default_gift_config | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная аккаунтом администратора приложения. Конкретные операции см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

1 раз/10 секунд.

### Пример пакета запроса

**Основная форма**

```
{}
```

### Пример пакета ответа

**Основная форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-fecb13a5827b425d9be4b9c79ac10f67-O-Seq-284018"}
```

### Поля пакета запроса

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для выявления проблемы. |

## Описание кодов ошибок

Если не происходит ошибка сети (например, 502), код возврата HTTP этого API составляет 200. Фактический код ошибки и информация об ошибке находятся в полях ErrorCode и ErrorInfo полезной нагрузки ответа.

Общие коды ошибок (60000–79999), см. документ [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, попробуйте позже. |
| 100006 | Недостаточно прав для операции. Требуется аккаунт администратора. |
| 100012 | Частота вызовов слишком высока. Максимальная частота вызовов добавления подарков составляет 1 раз/10 сек. |
| 102001 | Нет информации об оплате. Приобретите пакет в консоли. |
| 102010 | Конфигурация подарков уже существует, необходимо полностью удалить конфигурацию перед настройкой подарков по умолчанию. |

# Удаление информации о языке категории подарков

## Описание функции

Администратор приложения может использовать этот API для удаления указанной информации о языке категории подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_gift_category_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице указаны только параметры, относящиеся к изменению, и их описание при вызове этого API. Для получения более подробной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе, соответствующем выделенному доменному имени: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_gift_category_language | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, сгенерированная учётной записью администратора приложения. Конкретные операции см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное беззнаковое целое число в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз/секунда.

### Пример пакета запроса

**Основная форма**

```
{    "CategoryId":"CId2",    "Language":"en"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательное | ID категории подарков. |
| Language | String | Обязательное | Типы языков категории подарков, поддерживаемые в настоящее время: "en", "zh-Hans", "zh-Hant". |

### Пример пакета ответа

**Основная форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-c6e582c5350545a09297b062c82ada9f-O-Seq-578739"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка прошла успешно; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на сбой. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для локализации проблемы. |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), код возврата HTTP для этого API составляет 200. Фактический код ошибки и информация об ошибке указываются в ErrorCode и ErrorInfo в полезной нагрузке ответа.

Распространённые коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Закрытые коды ошибок этого API выглядят следующим образом:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неправильные параметры запроса. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав на операцию. Требуется учётная запись администратора. |
| 100012 | Частота вызовов слишком высока, максимум 1 вызов/с. |
| 102001 | Нет информации об оплате. Приобретите пакет в консоли. |

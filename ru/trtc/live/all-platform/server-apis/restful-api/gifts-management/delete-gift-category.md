# Удаление категории подарка

## Описание функции

Администратор приложения может использовать данный API для удаления информации об указанной классификации подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_gift_category?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, задействованные при изменении, и их описания при вызове этого API. Для получения более подробной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя SDKAppID в его местоположении:Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_gift_category | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учетная запись администратора приложения. Для получения более подробной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69881#.E8.AE.BE.E7.BD.AE.2F.E5.8F.96.E6.B6.88.E7.AE.A1.E7.90.86.E5.91.98.E8.B4.A6.E5.8F.B7). |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Для конкретных операций см. [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака со значением от 0 до 4294967295. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз/сек.

### Пример пакета запроса

**Основная форма**

```
{    "CategoryId":"CId2"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательное | ID категории подарка, которая будет удалена. |

> **Примечание:** При удалении подарка, соответствующего CategoryId, также будет удалена связанная информация о языке категоризации подарков и отношения категоризации подарков.

### Пример пакета ответа

**Основная форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9d167940547545dfa63d9151288d0e83-O-Seq-285390"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: успешная обработка; FAIL: ошибка обработки. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый для каждого запроса. RequestId требуется для определения проблемы. |

## Описание кодов ошибок

При отсутствии сетевых ошибок (например, ошибка 502) HTTP код возврата этого API равен 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Частные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, попробуйте еще раз. |
| 100002 | Неправильные параметры запроса. Проверьте правильность запроса согласно описанию ошибки. |
| 100006 | Недостаточно прав для выполнения операции. Для выполнения операции требуется учетная запись администратора. |
| 100012 | Частые вызовы, максимальная частота вызовов 1/с. |
| 102001 | Отсутствует информация об оплате. Приобретите пакет в консоли. |

# Удаление информации о языке подарка

## Описание функции

Администратор приложения может использовать этот API для удаления указанной информации о языке подарка.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_gift_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице указаны только параметры, задействованные при изменении, и их описания при вызове этого API. Для получения дополнительной информации о параметрах см. раздел [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID расположен в выделенном доменном имени страны/региона: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_gift_language | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Для получения дополнительной информации см. раздел [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учётной записью администратора приложения. Для выполнения конкретных операций см. раздел [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Основной формат**

```
{    "GiftId":"GId1",    "Language":"en"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательно | ID подарка. |
| Language | String | Обязательно | Тип языка подарка. В настоящее время поддерживаются "en", "zh-Hans" и "zh-Hant". |

### Пример пакета ответа

**Основной формат**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9d167940547545dfa63d9151288d0e83-O-Seq-285390"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка выполнена успешно; FAIL: обработка не выполнена. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. Для устранения неполадок в запросе должен быть предоставлен его RequestId. |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), код возврата HTTP этого API равен 200. Фактический код ошибки и информация об ошибке указаны полями ErrorCode и ErrorInfo в полезной нагрузке ответа.

Распространённые коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Закрытые коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Недопустимые параметры запроса. Проверьте корректность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав на операцию. Необходимы операции с учётной записью администратора. |
| 100012 | Частые вызовы, максимальная частота вызовов 1/с. |
| 102001 | Отсутствует информация об оплате, приобретите пакет в консоли. |

# Удаление подарка

## Описание функции

Администратор приложения может использовать этот API для удаления информации о указанном подарке.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_gift?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, используемые при вызове этого API, и их описание. Для получения дополнительной информации о параметрах см. [введение REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в выделенном доменном имени страны/региона:Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_gift | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69881#.E8.AE.BE.E7.BD.AE.2F.E5.8F.96.E6.B6.88.E7.AE.A1.E7.90.86.E5.91.98.E8.B4.A6.E5.8F.B7). |
| usersig | Подпись, созданная учётной записью администратора приложения. Для получения конкретных инструкций см. [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса фиксирован как `json`. |

### Максимальная частота вызовов

1 раз/сек.

### Пример пакета запроса

**Базовая форма**

```
{    "GiftId" :  "GId1"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательно | ID подарка для удаления. |

> **Примечание:** При удалении подарка, соответствующего GiftId, связанная информация о языке подарка и отношение категории подарка также удаляются одновременно.

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-821589b588fb4df79e6a5fe8f0cb4519-O-Seq-289843"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка выполнена успешно; FAIL: обработка не выполнена. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. Для диагностики запроса необходимо указать его RequestId. |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), HTTP код возврата этого API составляет 200. Фактические коды ошибок и информация об ошибках указываются в полях ErrorCode и ErrorInfo в полезной нагрузке ответа.

Коды общих ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Коды частных ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Параметры запроса недопустимы. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав доступа. Требуются операции учётной записи администратора. |
| 100012 | Частые вызовы. Максимальная частота вызовов 1/сек. |
| 102001 | Отсутствует информация об оплате. Приобретите пакет в консоли. |

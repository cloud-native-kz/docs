# Получение информации о языке категории подарков

## Описание функции

Администратор приложения может использовать этот API для получения информации о языке указанной категории подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift_category_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице приведены только параметры, задействованные при вызове этого API, и их описание. Дополнительные сведения о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе с выделенным доменным именем: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift_category_language | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должен быть учётной записью администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, сгенерированная учётной записью администратора приложения. Для конкретных операций см. раздел [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса фиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "CategoryId":"CId2",    "Language":"en"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательно | ID категории подарков. |
| Language | String | Обязательно | Типы языков категории подарков, поддерживаемые в настоящее время: "en", "zh-Hans", "zh-Hant". |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-358da87c1fda4303914b8f4a37bfe9c0-O-Seq-541528",    "CategoryInfo": {        "CategoryName": "name",        "CategoryDesc": "desc"    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка выполнена успешно; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId необходим для определения проблемы. |
| CategoryName | String | Имя категории подарков. |
| CategoryDesc | String | Описание категории подарков. |

## Описание кодов ошибок

Если не происходит сетевая ошибка (например, ошибка 502), код возврата HTTP этого API составляет 200. Фактический код ошибки и информация об ошибке указаны ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API выглядят следующим образом:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, попробуйте снова. |
| 100002 | Параметры запроса недопустимы. Проверьте, правильно ли выполнен запрос в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав для операции. Требуются операции учётной записи администратора. |
| 100012 | Частые вызовы. Максимальная частота вызовов составляет 1/с. |
| 102001 | Отсутствует информация об оплате. Приобретите пакет в консоли. |

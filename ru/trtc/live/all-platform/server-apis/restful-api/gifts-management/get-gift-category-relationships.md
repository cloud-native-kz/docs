# Получение связей категорий подарков

## Описание функции

Администратор приложения может использовать этот API для получения связей подарков в указанной категории.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift_category_relations?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, задействованные при вызове этого API, и их описания. Более подробную информацию о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе, где расположено выделенное доменное имя. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift_category_relations | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должен быть аккаунтом администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, сгенерированная аккаунтом администратора приложения. Конкретные операции см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "CategoryId":"id1"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательное | ID категории подарка, который необходимо удалить. |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-8014991f0c2a4b7e86814cf8b52618f6-O-Seq-331270",    "Relation": {        "CategoryId": "CId2",        "GiftIdList": [            "GId1"        ]    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: успешная обработка; FAIL: ошибка обработки. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для локализации проблемы. |
| CategoryId | String | ID категории подарка запроса. |
| GiftIdList | Array | Список ID подарков в категории подарков. |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например, ошибка 502), код возврата HTTP для этого API равен 200. Фактический код ошибки и информация об ошибке указываются в полях ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Недопустимые параметры запроса. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав на операцию. Требуются операции аккаунта администратора. |
| 100012 | Частота вызовов слишком высока, максимальная частота вызовов — 1/с. |
| 102001 | Отсутствует информация об оплате, приобретите пакет в консоли. |

# Получить категорию подарка

## Описание функции

Администратор приложения может использовать этот API для получения информации о указанной классификации подарков.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift_category?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице приведены только параметры, связанные с модификацией, и их описания при вызове этого API. Дополнительную информацию о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Страна/регион, в котором находится SDKAppID: выделенное доменное имя. Китай: `console.tim.qq.com`. Сингапур: `adminapisgp.im.qcloud.com`. Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift_category | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Дополнительную информацию см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69881#.E8.AE.BE.E7.BD.AE.2F.E5.8F.96.E6.B6.88.E7.AE.A1.E7.90.86.E5.91.98.E8.B4.A6.E5.8F.B7). |
| usersig | Подпись, созданная учётной записью администратора приложения. Конкретные операции см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-разрядное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса фиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "CategoryId":"CId2"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| CategoryId | String | Обязательно | ID категории подарка, которую вы хотите запросить. |

### Пример полезной нагрузки ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-205381db557a47bc93d6faa14d17dfeb-O-Seq-316720",    "GiftCategoryInfo": {        "CategoryId": "CId2",        "ExtensionInfo": "{\\"key\\": \\"value\\"}",        "DefaultName": "name1",        "DefaultDesc": "desc1"    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый для каждого запроса. RequestId требуется для локализации проблемы. |
| CategoryId | String | ID категории подарка. |
| ExtensionInfo | String | Информация о пользовательском расширении. |
| DefaultName | String | Имя подарка по умолчанию. |
| DefaultDesc | String | Описание подарка по умолчанию. |

## Описание кодов ошибок

Если не происходит ошибка сети (например, ошибка 502), код возврата HTTP этого API составляет 200. Фактический код ошибки и сообщение об ошибке обозначены параметрами ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, попробуйте ещё раз. |
| 100002 | Параметры запроса недопустимы. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав на операцию. Требуется учётная запись администратора. |
| 100012 | Частые вызовы. Максимальная частота вызовов составляет 1/с. |
| 102001 | Отсутствует информация об оплате. Приобретите пакет в консоли. |

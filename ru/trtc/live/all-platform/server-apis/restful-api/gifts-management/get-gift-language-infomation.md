# Получение информации о языке подарка

## Описание функции

Администратор приложения может использовать этот API для получения информации о языке подарка.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift_language?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, участвующие в изменении при вызове этого API, и их описания. Для получения дополнительной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в стране/регионе, соответствующем выделенному доменному имени: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift_language | Запрашиваемый API. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Для получения дополнительной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учётной записью администратора приложения. Для конкретных операций см. [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение колеблется от 0 до 4294967295. |
| contenttype | Формат запроса фиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "GiftId":"GId1",    "Language":"en"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательное | ID подарка, максимум 20 байт, максимум 3 категории подарков. |
| Language | String | Обязательное | Типы языков подарков, поддерживаемые в настоящее время: "en", "zh-Hans", "zh-Hant". |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-27cbb1204fc04358a23e519410ae5373-O-Seq-541258",    "GiftInfo": {        "Name": "name1",        "Desc": "desc1"    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 означает успех, ненулевое значение означает ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId требуется для определения проблемы. |
| Name | String | Название подарка. |
| Desc | String | Описание подарка. |

## Описание кодов ошибок

Если сетевая ошибка (например, 502) не происходит, код возврата HTTP этого API составляет 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в полезной нагрузке ответа.

Распространённые коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, попробуйте снова. |
| 100002 | Параметры запроса неправомерны. Проверьте правильность запроса в соответствии с описанием ошибки. |
| 100006 | Недостаточно прав доступа. Требуется учётная запись администратора. |
| 100012 | Количество вызовов частое, максимальная частота вызовов составляет 1/с. |
| 102001 | Отсутствует информация об оплате. Купите пакет в консоли. |

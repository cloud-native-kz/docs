# Получить статистику подарков

## Описание функции

Администратор приложения может использовать этот API для получения статистики подарков конкретного пользователя.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift_count?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В приведённой ниже таблице перечислены только параметры, которые необходимо изменить при вызове этого API, и их описания. Дополнительную информацию о параметрах см. в [введении REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя SDKAppID:Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift_count | API создания комнаты прямой трансляции |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения, можно получить на карточке приложения в [Консоли](https://console.tencentcloud.com/im) Chat. |
| identifier | Должен быть учётной записью администратора приложения. Дополнительную информацию см. в документации [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учётной записью администратора приложения. Для получения конкретных инструкций см. [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение варьируется от 0 до 4294967295. |
| contenttype | Формат запроса, значение фиксировано как `json`. |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "RoomId": "room_id",    "From_Account": "user1",}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательный | ID комнаты |
| From_Account | String | Обязательный | Получатель подарка |

### Пример пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720",  "Response": {    "Sequence": 1,    "TotalCount": 1,    "TotalCoins": 1,    "GiftSenderCount": 1,  }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса.OK: обработка успешна.FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки.0: успех.Ненулевое значение: указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный идентификатор запроса, возвращаемый для каждого запроса. Для обнаружения проблемы должен быть предоставлен RequestId запроса. |
| TotalCount | Integer | Общее количество подарков |
| TotalCoins | Integer | Общая стоимость подарков |
| GiftSenderCount | Integer | Общее количество отправителей подарков |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например, 502), HTTP код возврата этого API равен 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в полезной нагрузке ответа.

Для кодов ошибок общего применения (60000–79999) см. документ [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027).

Частные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр. Проверьте корректность запроса согласно описанию ошибки. |
| 100003 | ID комнаты уже существует. Пожалуйста, выберите другой ID комнаты. |

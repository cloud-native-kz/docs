# Получить подарок

## Описание функции

Администратор приложения может использовать этот API для запроса информации о подарках.

## Вызовы API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_gift?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, относящиеся к изменениям при вызове этого API, и их описания. Дополнительные сведения о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID находится в выделенном доменном имени, соответствующем стране/региону: Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_gift | API запроса. |
| sdkappid | SDKAppID, присваиваемый консолью Chat при создании приложения. |
| identifier | Должен быть учетной записью администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Дополнительные сведения см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение варьируется от 0 до 4294967295. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

1 раз в секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "GiftId" :  "GId1"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| GiftId | String | Обязательно | ID подарка, который вы хотите запросить. |

### Пример пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-a984952a5dde4b488caba4f75e5437b8-O-Seq-287281",    "GiftInfo": {        "GiftId": "GId1",        "IconUrl": "https://www.xxx.com/xxx1.png",        "ResourceUrl": "https://www.xxx.com/xxx2.png",        "Coins": 10,        "Level": 1,        "ExtensionInfo": "{\\"key\\": \\"value\\"}",        "DefaultName": "name1",        "DefaultDesc": "desc1"    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка выполнена успешно; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 означает успех, ненулевое значение означает ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. Для устранения неполадок запроса следует предоставить его RequestId. |
| GiftId | String | ID подарка. |
| IconURL | String | Миниатюра подарка. |
| ResourceUrl | String | Ссылка на ресурс подарка, например ссылки на ресурсы полноэкранного специального эффекта mp4 и svg. |
| Coins | Integer | Цена подарка в виртуальной валюте. |
| Level | Integer | Уровень подарка. |
| ExtensionInfo | String | Информация о пользовательском расширении подарка. |
| DefaultName | String | Имя подарка по умолчанию. |
| DefaultDesc | String | Описание подарка по умолчанию. |

## Описание кодов ошибок

Если не происходит сетевая ошибка (например, 502), код возврата HTTP этого API составляет 200. Фактический код ошибки и информация об ошибке указаны в полях ErrorCode и ErrorInfo полезной нагрузки ответа.

Общие коды ошибок (60000–79999) см. в документе [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неправильные параметры запроса. Проверьте правильность запроса согласно описанию ошибки. |
| 100006 | Недостаточно прав на выполнение операции. Требуется учетная запись администратора. |
| 100012 | Частые вызовы, максимальная частота вызовов 1/сек. |
| 102001 | Отсутствует информация об оплате, приобретите пакет в консоли. |

# Руководство по настройке системы подарков

## Обзор системы подарков

Информация, связанная с подарками, состоит из подарка, категоризации подарков, отношения подарок-категория и информации на языках. Информация представлена следующим образом:

- **Подарок**: включает GiftId, название по умолчанию, описание по умолчанию, степень серьезности, цену в виртуальной валюте, иконку и информацию о пользовательском расширении. Среди них GiftId является уникальным идентификатором подарка в системе и объектом для последующих операций. Цена подарка в виртуальной валюте используется для "ценообразования" при отображении подарка. Соответствие между виртуальной валютой и фактической суммой платежа требует дополнительного определения со стороны бизнеса.
- **Категоризация подарков**: включает CategoryId, название по умолчанию, описание по умолчанию и информацию о пользовательском расширении. Как и GiftId, CategoryId является уникальным идентификатором категоризации подарков в системе и объектом для последующих операций.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/36136f47cd9c11f0afdc52540044a08e.png)

> **Примечание:** Информация о классификации подарков не включает отношение подарок-категория. Она включает только ID категории подарков, название категории, описание категории и расширенную информацию. Подробности см. в разделе [Добавить категорию подарков](https://www.tencentcloud.com/document/product/647/72848)

- **Отношение подарок-категория**: описание того, какие подарки находятся в категории

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/7e9f0096cd9c11f0afdc52540044a08e.png)

- **Информация на языках**: названия и описания подарков и категорий подарков поддерживают интернационализацию. Информация на языках используется для записи переводов описаний подарков на разные языки.

> **Примечание:** При использовании SDK для получения списка подарков, если подарок или категория подарков не имеет названия и описания на указанном языке, возвращается название и описание подарка по умолчанию, установленные при создании подарка.

## Процесс конфигурации системы подарков

Функция подарков в компоненте требует, чтобы соответствующее приложение (SDKAppID) было [пробной версией TUILiveKit или версией для крупномасштабной трансляции](https://www.tencentcloud.com/document/product/647/60033#ce1a5d1e-44c2-4801-afd5-183cd9e306b0) для использования. После активации добавьте информацию, связанную с подарками, в упорядоченном порядке. Ниже приведен порядок конфигурации подарков:

1. Добавить подарки и категории подарков

Сначала добавьте подарки и категории подарков. После добавления вы получите несколько GiftIDs и CategoryIds. Используйте эти идентификаторы для последующих операций. О методе добавления см. в разделах [Добавить подарок](https://www.tencentcloud.com/document/product/647/72845) и [Добавить категорию подарков](https://www.tencentcloud.com/document/product/647/72848).

2. Добавить отношения категорий подарков и подарков

Используйте GiftId и CategoryId, полученные на предыдущем шаге, чтобы установить соответствующее отношение. Инструкции см. в разделе [Добавить отношения категорий подарков](https://www.tencentcloud.com/document/product/647/72851).

3. Добавить информацию на языках для подарков и категорий подарков

Этот шаг не требуется. Если требуется интернационализация, вы можете его настроить. Инструкции см. в разделах [Добавить информацию на языках для подарков](https://www.tencentcloud.com/document/product/647/72854) и [Добавить информацию на языках для категорий подарков](https://www.tencentcloud.com/document/product/647/72857).

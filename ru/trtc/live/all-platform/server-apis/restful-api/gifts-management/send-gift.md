# Отправка подарка

## Описание функции

Администратор приложения может использовать этот API для отправки подарка.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/send_gift?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице приведены только параметры, задействованные в модификации, и их описания при вызове этого API. Более подробную информацию о параметрах см. в разделе [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | SDKAppID расположен в выделенном доменном имени для страны/региона. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/send_gift | API создания комнаты прямой трансляции |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения, можно получить из карточки приложения в [Консоли](https://console.tencentcloud.com/im) Chat. |
| identifier | Должна быть учетная запись администратора приложения. Более подробную информацию см. в документации [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Конкретные операции см. в разделе [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение находится в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, значение фиксировано как `json`. |

### Максимальная частота вызовов

200 раз/секунду.

### Пример пакета запроса

**Базовая форма**

```
{    "RoomId": "room_id",    "GiftId": "gift1",    "Count": 1,    "From_Account": "user1",    "ReceiverList_Account": ["user2"]}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| GiftId | String | Обязательное | ID подарка, должен быть уже настроен через API конфигурации подарков |
| From_Account | String | Обязательное | Отправитель подарка |
| ReceiverList_Account | Array | Обязательное | Получатель подарка |

### Пример тела пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна. FAIL: обработка не выполнена. |
| ErrorCode | Integer | Код ошибки. 0: указывает на успех. Ненулевое значение: указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. При решении проблем с запросом следует указать его RequestId. |

## Описание кодов ошибок

За исключением сетевых ошибок (таких как ошибка 502), код возврата HTTP этого API всегда равен 200. Фактический код ошибки и сообщение об ошибке указаны в полях ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000 по 79999) см. в документе [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр. Проверьте, правильно ли сформирован запрос, согласно описанию ошибки. |
| 100003 | ID комнаты уже существует. Пожалуйста, выберите другой ID комнаты. |
| 102001 | Функция подарков не включена. Пожалуйста, проверьте платный план. |
| 102002 | ID подарка не существует. |
| 102004 | Ошибка обратного вызова подарка. |
| 102008 | Переключатель подарков в текущей комнате не включен. |

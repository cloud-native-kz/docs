# Удаление робота из комнаты

## Описание функции

Администраторы приложения могут использовать этот API для удаления роботов в указанной комнате.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_robot?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице приведены только параметры, задействованные при вызове этого API, и их описания. Дополнительную информацию о параметрах см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметры | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID: Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_robot | API запроса |
| sdkappid | Назначенный SDKAppID, созданный для Live приложения |
| identifier | Должна быть учетная запись администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Сгенерированная подпись учетной записи администратора приложения. Конкретные операции см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0,4294967295]. |
| contenttype | Формат запроса имеет фиксированное значение `json`. |

### Максимальная частота вызовов

100 раз/сек.

### Пакет запроса

**Базовая форма**

```
{    "RoomId":"mix1",    "RobotList_Account":["brennanli"]    }
```

### Поля пакета запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| RoomId | String | Да | ID комнаты, максимум 48 байт |
| RobotList_Account | Array | Да | Список роботов, учетная запись бота, т. е. UserID пользователя, должна быть [импортированной учетной записью](https://www.tencentcloud.com/document/product/647/69887) |

### Тело пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK указывает на успешную обработку; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки, 0 указывает на успех, ненулевое значение указывает на ошибку |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса. Возвращается для каждого запроса. RequestId требуется для локализации проблемы |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), код возврата HTTP этого API — 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в теле пакета ответа.

Для общих кодов ошибок (60000–79999) см. документ [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Неверный параметр. Проверьте, правильный ли запрос в соответствии с описанием ошибки |
| 100006 | Этот тип комнаты должен быть типом Live комнаты |
| 100004 | Комната не существует или распущена |

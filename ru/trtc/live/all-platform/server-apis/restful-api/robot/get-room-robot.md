# Получить робота в комнате

## Описание функции

Администраторы приложения могут получить список роботов в комнате с помощью этого API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_robot?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, задействованные при изменении при вызове этого API, и их описания. Для получения более подробной информации о параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметры | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где расположен ваш SDKAppID: Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_robot | API запроса |
| sdkappid | SDKAppID, назначенный при создании приложения Live |
| identifier | Должен быть аккаунт администратора приложения. Для более подробной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Сгенерированная подпись аккаунта администратора приложения. Для конкретных операций см. [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883) |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса имеет фиксированное значение `json` |

### Максимальная частота вызовов

100 раз/секунду.

### Пакет запроса

**Базовая форма**

```
{    "RoomId":"mix1"}
```

### Поля пакета запроса

| Поле | Тип | Обязательно | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты, максимум 48 байт |

> **Примечание:** Если в комнате присутствуют роботы, комната не будет автоматически закрыта и требует ручного вызова интерфейса [dismiss](https://www.tencentcloud.com/document/product/647/63406).

### Тело пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720",   "Response": {        "RobotList_Account": [            "brennanli",            "brennanli2"        ]    }}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась |
| ErrorCode | Integer | Код ошибки. 0: успех; не 0: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса. Возвращается для каждого запроса. RequestId необходим для локализации проблемы |
| RobotList_Account | Array | Список роботов |
| Response | Object | Ответ на конкретные данные |

## Описание кодов ошибок

Если не происходит ошибка сети (например, ошибка 502), код возврата HTTP этого API равен 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в теле пакета ответа.

Для распространённых кодов ошибок (60000–79999) см. документ [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API указаны ниже:

| Код ошибки | Описание значения |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Неверный параметр. Проверьте правильность запроса согласно описанию ошибки |
| 100006 | Требуемый тип комнаты — тип Live комнаты |
| 100004 | Комната не существует или растворена |

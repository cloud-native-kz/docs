# Создание комнаты

## Обзор функции

Администратор приложения может создать комнату через этот интерфейс.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/create_room?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, измененные при вызове этого интерфейса. Для получения дополнительной информации о других параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где расположен SDKAppID: Другие регионы в Китае: `console.tim.qq.com`. Сингапур: `adminapisgp.im.qcloud.com`. Соединённые Штаты: `adminapiusa.im.qcloud.com`. |
| v4/live_engine_http_srv/create_room | API запроса. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учётной записью администратора приложения. Конкретные операции см. в разделе [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, фиксированное значение: `json`. |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакетов запроса

**Базовая форма**

```
{  "RoomInfo":{              "RoomId":"live-room",               "RoomType":"Live",               "RoomName":"live name",               "Owner_Account":"administrator",               "CoverURL":"cover url",               "IsMessageDisabled":true,               "Category":[1, 2, 3],               "IsPublicVisible": true,               "IsSeatEnabled": true,               "TakeSeatMode":"ApplyToTake",               "MaxSeatCount":8,               "ActivityStatus":1,              "IsUnlimitedRoomEnabled":true,              "SeatLayoutTemplateId":600              }}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты, максимум 48 байт. |
| RoomName | String | Опционально | Имя комнаты, по умолчанию использует ID комнаты, максимум 100 байт. |
| RoomType | String | Обязательно | Тип комнаты: Live (Прямая трансляция). |
| Owner_Account | String | Опционально | ID хоста (должна быть [импортированная](https://www.tencentcloud.com/document/product/647/69887) учётная запись), по умолчанию — это ID пользователя вызывающего API. |
| CoverURL | String | Опционально | Обложка комнаты. Максимум 200 байт. |
| IsMessageDisabled | Bool | Опционально | Отключить всё аудио, по умолчанию false. |
| Category | Array | Опционально | Индикатор типа комнаты, может использоваться приложениями как идентификатор типа комнаты, такой как Game, Music и т. д. Поддерживает до трёх. |
| IsPublicVisible | Bool | Опционально | Индикатор открытости комнаты. |
| IsSeatEnabled | Bool | Опционально | Поддержка позиций микрофона, по умолчанию не поддерживается. |
| TakeSeatMode | String | Опционально | При использовании функции позиции микрофона необходимо заполнить режим микрофона, FreeToTake (свободное включение микрофона), ApplyToTake (подать заявку на включение микрофона). |
| MaxSeatCount | Integer | Опционально | Максимальное количество позиций микрофона, при использовании функции позиции микрофона по умолчанию ограничено пакетом. |
| ActivityStatus | Integer | Опционально | Статус активности прямой трансляции: определение тега, определяемого пользователем. |
| IsUnlimitedRoomEnabled | Bool | Обязательно | Включить возможность микширования потока прямой трансляции для поддержки сценариев с высокой параллелью. Если установлено значение true, это означает использование SDK [TUILiveKit](https://www.tencentcloud.com/document/product/647/60034). |
| SeatLayoutTemplateId | Integer | Опционально | ID стиля макета комнаты. Пожалуйста, см. [стиль макета](https://www.tencentcloud.com/document/product/647/72225#b2c1399d-7641-4f9f-b7af-95ddb06b9dc3), [описание макета](https://www.tencentcloud.com/document/product/647/72223#d63921ff-c54b-4b69-9f99-3992293b4f46). Поддерживается с версии: RTCRoomEngine 3.3.0, LiveStreamCore 3.3.0 |
| Notice | String | Опционально | Объявление комнаты, максимум 100 байт. |

> **Примечание:** После завершения прямой трансляции пользователям необходимо вручную вызвать API [Уничтожить комнату](https://www.tencentcloud.com/document/product/647/63406) для завершения комнаты. [Механизм автоматического уничтожения комнаты](https://www.tencentcloud.com/document/product/647/63406#07069efb-059d-48f2-a507-2040c11cfaeb)

### Пример пакетов ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720",  "RoomId": "live-room"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса: OK указывает на успех; FAIL указывает на ошибку. |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется предоставить этот RequestId при выявлении проблем. |
| RoomId | String | ID комнаты. |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.

Распространённые коды ошибок (60000–79999) см. в документации [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Ошибка внутреннего сервера, пожалуйста, повторите попытку. |
| 100002 | Недействительный параметр, проверьте корректность запроса в соответствии с описанием ошибки. |
| 100003 | ID комнаты уже существует, выберите другой ID комнаты. |
| 100007 | Отсутствует платёжная информация, необходимо приобрести пакет из консоли. |
| 100010 | ID комнаты уже использован, оператор является владельцем, может использоваться напрямую. |
| 100011 | ID комнаты занят IM, вы можете изменить на другой ID комнаты. |
| 100012 | Создание комнат превышает ограничение по частоте; один и тот же ID комнаты может быть создан только один раз в секунду. |
| 100026 | Имя комнаты или другая информация содержат чувствительный контент. |

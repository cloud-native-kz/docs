# Удаление метаданных комнаты

## Обзор функции

Администратор приложения может удалять метаданные комнаты через этот интерфейс.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/del_room_metadata?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже перечислены только параметры, измененные при вызове этого интерфейса. Дополнительные сведения о других параметрах см. в разделе [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Others in China: `console.tim.qq.com`Singapore : `adminapisgp.im.qcloud.com`United States: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/del_room_metadata | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная аккаунтом администратора приложения. Подробные инструкции см. в разделе [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакетов запроса

**Базовая форма**

```
{  "RoomId": "live-room",  "Keys": ["key1", "key2"]}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| Keys | Array | Обязательное | Список ключей для указанного удаления метаданных комнаты |

### Пример пакетов ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом и требуемый при поиске проблем |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указаны в полезной нагрузке ответа параметрами ErrorCode и ErrorInfo. Общие коды ошибок (60000–79999) можно найти в документе [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027#). Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Недопустимый параметр, проверьте корректность запроса на основе описания ошибки |
| 100003 | ID комнаты уже существует, выберите другой ID комнаты |
| 100006 | Нет разрешения на операцию с метаданными комнаты |
| 100007 | Отсутствует информация об оплате, необходимо приобрести пакет из консоли |
| 100500 | Количество ключей для метаданных комнаты превышает верхний лимит в 10 |
| 100501 | Значение одного ключа для метаданных комнаты превышает верхний лимит в 2 КБ |
| 100502 | Общий размер значений всех ключей метаданных комнаты превышает верхний лимит в 16 КБ |
| 100503 | Ни один из указанных ключей не является допустимым при удалении метаданных комнаты |

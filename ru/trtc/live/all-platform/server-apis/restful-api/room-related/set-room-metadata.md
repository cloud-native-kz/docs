# Установка метаданных комнаты

## Обзор функции

Администратор приложения может устанавливать метаданные комнаты через этот интерфейс.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/set_room_metadata?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого интерфейса. Для получения дополнительной информации о других параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, в котором находится SDKAppID: Прочие в Китае: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/set_room_metadata | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Подробнее см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, сгенерированная аккаунтом администратора приложения. Для получения подробных инструкций см. [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-битное беззнаковое целое число в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызова

200 запросов/сек.

### Примеры пакетов запроса

**Базовая форма**

```
{  "RoomId": "live-room",  "Metadata": [ // Количество ключей ограничено 10, длина каждого ключа ограничена 50 байтами или менее. Отдельные значения могут быть размером до 2 КБ, в то время как общий размер всех значений не должен превышать 16 КБ.                {                    "Key":"key1",  //Если ключ существует, он будет обновлен; в противном случае он будет добавлен                    "Value":"value1"                },                {                    "Key":"key2",                    "Value":"value2"                }            ]}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |
| Metadata | Array | Обязательно | Список метаданных комнаты. Количество ключей ограничено 10, длина каждого ключа ограничена 50 байтами или менее. Отдельные значения могут быть размером до 2 КБ, в то время как общий размер всех значений не должен превышать 16 КБ. |

### Примеры пакетов ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом, необходимо предоставить этот RequestId при поиске неисправностей |

## Коды ошибок

Если не происходит сетевая ошибка (например, ошибка 502), код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указываются в полезной нагрузке ответа с помощью ErrorCode и ErrorInfo. Общие коды ошибок (60000–79999) можно найти в документе [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027#). Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку |
| 100002 | Недействительный параметр, пожалуйста, проверьте корректность запроса на основе описания ошибки |
| 100003 | ID комнаты уже существует, пожалуйста, выберите другой ID комнаты |
| 100006 | Нет разрешения на управление метаданными комнаты |
| 100007 | Нет информации об оплате, необходимо приобрести пакет в консоли |
| 100500 | Количество ключей для метаданных комнаты превышает верхний предел 10 |
| 100501 | Значение одного ключа для метаданных комнаты превышает верхний предел 2 КБ |
| 100502 | Общий размер значений для всех ключей метаданных комнаты превышает верхний предел 16 КБ |
| 100503 | При удалении метаданных комнаты ни один из указанных ключей не является действительным |

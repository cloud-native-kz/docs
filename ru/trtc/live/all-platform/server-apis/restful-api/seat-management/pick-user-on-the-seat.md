# Выбрать пользователя на месте

## Обзор функции

Администраторы приложения могут разрешить участникам занять место через этот API.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_mic/pick_user_on_seat?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого API. Для получения дополнительной информации обратитесь к [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/60703#).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированное доменное имя для страны/региона, где расположен SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_mic/pick_user_on_seat | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения дополнительной информации обратитесь к [Администратор приложения](https://www.tencentcloud.com/document/product/1047/33517#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения дополнительной информации обратитесь к [Создание UserSig](https://www.tencentcloud.com/document/product/1047/34385). |
| random | Введите случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение `json` |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Основная форма**

```
{  "RoomId":"room-test",  "Member_Account":"user2",  "Index":1}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |
| Index | Integer | Обязательно | Номер позиции места |
| Member_Account | String | Обязательно | ID пользователя места, должен быть членом комнаты |

### Пример пакета ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-7c1680be52734bdc8d6de398ab9505e7-O-Seq-57717"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса. OK для успеха, FAIL для отказа. |
| ErrorCode | Integer | Код ошибки. 0 для успеха, другие для отказа. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется предоставить этот RequestId при локализации проблем. |

## Коды ошибок

Если не возникает ошибка сети (например, ошибка 502), код состояния HTTP для этого интерфейса всегда будет 200. Фактические коды ошибок и сообщения передаются через ErrorCode и ErrorInfo в теле ответа.
Для часто встречающихся кодов ошибок (60000–79999) см. документацию [Код ошибки](https://www.tencentcloud.com/document/product/647/57276#).
Приватные коды ошибок для этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр, пожалуйста, проверьте корректность запроса на основе описания ошибки. |
| 100004 | Комната не существует, или она когда-то существовала, но теперь была уничтожена. |
| 100005 | Пользователь на месте не является членом комнаты. |
| 100200 | Позиция места заблокирована, вы можете попробовать переключиться на другую позицию. |
| 100202 | Уже в очереди на место. |
| 100203 | Уже на месте. |
| 100205 | Позиция места полностью занята. |
| 100210 | На позиции места уже есть пользователь. |
| 100211 | Эта комната не поддерживает подключение к месту. |

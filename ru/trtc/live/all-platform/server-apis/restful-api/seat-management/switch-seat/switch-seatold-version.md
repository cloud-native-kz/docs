# Переключение места (Старая версия)

## Описание функции

Администратор приложения может использовать этот API для перемещения пользователя на определенный свободный слот микрофона.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/room_engine_http_mic/switch_seat?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице указаны только параметры, требующие изменения при вызове этого API, и их описания. Более подробную информацию о параметрах см. в [введении в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя для страны/региона, где находится SDKAppID: Сингапур: `adminapisgp.im.qcloud.com` |
| v4/room_engine_http_mic/switch_seat | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Подробнее см. [app admin](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробнее см. [generate UserSig](https://www.tencentcloud.com/document/product/647/69883) |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295] |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Основная форма**

```
{  "RoomId":"room-test",  "ToIndex": 1,  "Member_Account": "user"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| ToIndex | Integer | Обязательное | ID места |
| Member_Account | String | Обязательное | ID пользователя |

### Пример пакета ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-7c1680be52734bdc8d6de398ab9505e7-O-Seq-57717"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 означает успех; ненулевое значение означает ошибку. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. RequestId необходим для поиска проблемы. |

## Описание кодов ошибок

Если не происходит сетевая ошибка (такая как 502), код возврата HTTP этого API составляет 200. Фактический код ошибки и информация об ошибке указаны в ErrorCode и ErrorInfo в полезной нагрузке ответа.

Общие коды ошибок (60000 до 79999), см. документ [Error Code](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку. |
| 100002 | Неверный параметр. Проверьте корректность запроса в соответствии с описанием ошибки. |
| 100004 | Комната не существует, или она могла существовать, но была закрыта. |

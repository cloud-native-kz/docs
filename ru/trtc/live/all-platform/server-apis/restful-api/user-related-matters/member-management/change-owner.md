# Изменение владельца комнаты

## Описание функции

Администратор приложения может использовать этот API для изменения владельца комнаты.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/change_room_owner?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице перечислены только параметры, используемые при модификации, и их описания при вызове этого API. Для получения подробной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Выделенное доменное имя, соответствующее стране/региону, где находится SDKAppID: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/change_room_owner | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть учетной записью администратора приложения. Дополнительные сведения см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin). |
| usersig | Подпись, созданная учетной записью администратора приложения. Для конкретных операций см. раздел [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака, значение в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

200 раз/сек.

### Пример пакета запроса

**Основной формат**

```
{    "RoomId": "test",    "NewOwner_Account": "user2"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| NewOwner_Account | String | Обязательное | новый владелец |

### Пример пакета ответа

**Основной формат**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не удалась. |
| ErrorCode | Integer | Код ошибки: 0 означает успех, ненулевое значение означает сбой. |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. Для устранения неполадок запроса следует предоставить его RequestId. |

## Описание кодов ошибок

За исключением сетевых ошибок (например, ошибки 502), код возврата HTTP этого API составляет 200. Фактические код ошибки и информация об ошибке находятся в теле пакета ответа.

ErrorCode и ErrorInfo.

Распространенные коды ошибок (60000–79999) см. в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неверный параметр. Проверьте запрос в соответствии с описанием ошибки. |
| 100004 | Комната не существует. Причина может быть в том, что комната не была создана или уже распущена. |

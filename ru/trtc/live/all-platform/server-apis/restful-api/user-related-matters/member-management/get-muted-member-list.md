# Получить список отключённых пользователей

## Описание функции

Администраторы приложения могут использовать этот API для отключения и включения членов.

## Описание вызова API

### Пример URL запроса:

```
https://xxxxxx/v4/live_engine_http_srv/get_muted_member_list?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже приведены только параметры, измененные при вызове этого интерфейса. Более подробную информацию о других параметрах см. в [REST API](https://www.tencentcloud.com/document/product/647/63398#).

| Параметры | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com`. Сингапур: `adminapisgp.im.qcloud.com`. США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_muted_member_list | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть учетной записью администратора приложения. Подробнее см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная учетной записью администратора приложения. Конкретные шаги см. в [Создание UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-разрядное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса — фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакетов запроса

**Базовая форма**

```
{     "RoomId": "test" }
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |

### Пример пакетов ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-9a9fe01d96164827b49dacb6b3cdadba-O-Seq-126318",    "Response": {        "MutedAccountList": [            {                "Member_Account": "user1",                "MutedUntil": 1719293107            },            {                "Member_Account": "brennanli",                "MutedUntil": 1719293107            }        ]    }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется указать этот RequestId при обращении за помощью |
| MutedAccountList | Array | Список отключённых пользователей |
| Member_Account | String | ID отключённого пользователя |
| MutedUntil | Integer | Время истечения отключения |

## Коды ошибок

Если не возникает сетевая ошибка (например, ошибка 502), код возврата HTTP для этого интерфейса всегда 200. Фактический код ошибки и сообщение об ошибке представлены в теле ответа в полях ErrorCode и ErrorInfo.

Общие коды ошибок (60000-79999) см. в документации [Код ошибки](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок для этого API приведены ниже

| Код ошибки | Описание |
| --- | --- |
| 100001 | Системная ошибка, повторите попытку |
| 100002 | Недействительный параметр, проверьте корректность запроса на основе описания ошибки |
| 100004 | Комната не существует, возможно, она никогда не была создана или уже была удалена |

# Одноканальная мультигостевая конференция

В этом документе описывается, как использовать `LiveCoreView` модуля `LiveStreamCore` для реализации подключения микрофона.

`LiveCoreView` поддерживает следующие возможности для подключения микрофона:

| [Запрос на подключение](https://www.tencentcloud.com/document/product/647/67472#f1743fa3-3864-4085-a62e-892c86e8ab13) | [Зритель прерывает соединение](https://www.tencentcloud.com/document/product/647/67472#b42d71d5-e940-469c-bdd8-6b23d2e685fd) | [Ведущий прерывает соединение](https://www.tencentcloud.com/document/product/647/67472#1f8f55ea-0af7-4297-86a8-d8a208d38a89) |
| --- | --- | --- |

## Предварительные требования

Перед использованием `LiveStreamCore` необходимо [интегрировать и выполнить вход](https://www.tencentcloud.com/document/product/647/67475#) в LiveStreamCore, чтобы обеспечить правильную работу последующих функций.

## Руководство по использованию

### Шаг 1: Добавление LiveCoreView в представление

Сначала необходимо импортировать модуль `LiveStreamCore`, затем создать объект представления `LiveCoreView` и добавить его в ваше представление.

iOS

Android

```
LiveStreamCore
```

```
public class AudienceConnectionActivity extends AppCompatActivity implements LiveCoreViewDefine.ConnectionObserver {    private LiveCoreView liveCoreView;    @Override    protected void onCreate(@Nullable Bundle savedInstanceState) {        super.onCreate(savedInstanceState);        liveCoreView = new LiveCoreView(this);        addContentView(liveCoreView,                new ViewGroup.LayoutParams(ViewGroup.LayoutParams.MATCH_PARENT, ViewGroup.LayoutParams.MATCH_PARENT));        liveCoreView.registerConnectionObserver(this);    }    @Override    protected void onDestroy() {        super.onDestroy();        liveCoreView.unregisterConnectionObserver(this);    }}
```

### Шаг 2: Подключение зрителя

#### Запрос на подключение

Когда вы хотите подключиться к кому-либо через микрофон, вызовите API `requestIntraRoomConnection` с тремя параметрами: userId человека для подключения, продолжительность истечения времени ожидания и включение ли камеры.

iOS

Android

```
let userId = "100012" // Замените это на userId пользователя, с которым вы хотите инициировать подключение микрофона. Если передать пустую строку, по умолчанию будет выполнено подключение к ведущему.let timeout = 30     // Замените это на продолжительность истечения времени ожидания для запроса на трансляцию в секундах. Если установить на 0, SDK не будет выполнять обнаружение истечения времени ожидания или запускать обратный вызов истечения времениlet openCamera = true // Когда вы хотите инициировать видеозвонок, установите openCamera на true. Для аудиозвонка установите на false.self.liveCoreView.requestIntraRoomConnection(userId: userId, timeOut: timeout, openCamera: true) {    // Запрос на подключение микрофона успешно отправлен} onError: { code, message in    // Ошибка при отправке запроса на подключение микрофона}
```

#### Зритель прерывает соединение

Если вы не хотите подключаться к ведущему, вызовите API `terminateIntraRoomConnection` для отключения.

iOS

Android

```
self.liveCoreView.terminateIntraRoomConnection()
```

```
liveCoreView.terminateIntraRoomConnection();
```

#### Ведущий прерывает соединение

Если вы хотите прервать соединение с участником аудитории, вызовите API `disconnectUser` и передайте соответствующий userId пользователя.

iOS

Android

```
let targetUserId = "100010"  // Замените это на userId участника аудитории, с которым вы хотите разорвать соединениеself.liveCoreView.disconnectUser(userId: targetUserId) {    // Соединение успешно разорвано} onError: { code, message in     // Ошибка при разрыве соединения}
```

```
String targetUserId = "100010";  // Замените это на userId участника аудитории, с которым вы хотите разорвать соединениеliveCoreView.disconnectUser(targetUserId, new TUIRoomDefine.ActionCallback() {    @Override    public void onSuccess() {        // Соединение успешно разорвано    }    @Override    public void onError(TUICommonDefine.Error error, String message) {        // Ошибка при разрыве соединения    }});
```

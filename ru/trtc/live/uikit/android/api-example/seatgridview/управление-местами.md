### Шаг 2: Управление местами

> **Примечание:** При использовании управления местами необходимо убедиться, что вы начали трансляцию или вошли в комнату.

#### Занять место

Поведение при запросе на выступление зависит от режима выступления.

Когда режим выступления в комнате — это «режим подачи заявки на выступление (`applyToTake`)», для запроса на выступление требуется одобрение владельца комнаты.

Когда режим выступления в комнате — это «режим свободного выступления (`freeToTake`)», для запроса на выступление не требуется одобрение владельца комнаты, и это можно сделать сразу.

Если вы хотите выступать, вызовите API `takeSeat` и передайте два параметра: индекс места и длительность тайм-аута.

Android

```

Если установлено значение 0, SDK не будет выполнять проверку тайм-аута или запускать обратные вызовы тайм-аута.self.seatGridView.takeSeat(index: index, timeout: timeout) { userInfo in    // Запрос на выступление принят} onRejected: { userInfo in    // Запрос на выступление отклонен} onCancelled: { userInfo in    // Запрос на выступление отменен} onTimeout: { userInfo in    // Истек тайм-аут запроса на выступление} onError: { userInfo, code, message in    // Ошибка запроса на выступление}

Когда вы являетесь хостом и кто-то запрашивает выступление, вы получите обратный вызов `onSeatRequestReceived`. Вы можете вызвать `responseRemoteRequest` для принятия/отклонения запроса на выступление.

iOS

Android

```
func onSeatRequestReceived(type: SGRequestType, userInfo: TUIUserInfo) {    if type == .applyToTakeSeat {        let agree = true // true означает        self.seatGridView.responseRemoteRequest(userId: userInfo.userId, agree: true) {            // Успешно согласились на выступление        } onError: { code, message in             // Ошибка при согласии на выступление        }    }}
```

```

#### Заблокировать место

Когда вы являетесь хостом и хотите заблокировать пустое место на позиции 5, чтобы помешать другим его занять, или отключить хоста на месте 6, вы можете вызвать API `lockSeat`, передав два параметра: индекс места, которое нужно заблокировать, и режим блокировки.

Структура режима блокировки (`TUISeatLockParams`) выглядит следующим образом:

| Имя свойства | Описание поля | Дополнительные примечания | Тип данных | Пример |
| --- | --- | --- | --- | --- |
| lockSeat |  Заблокировать позицию микрофона | Блокировка соответствующего места предотвращает попытки занять это место. | Логическое значение | true, когда место заблокировано. |
| lockVideo | Заблокировать камеру места. | Блокировка камеры соответствующего места остановит трансляцию видеопотока с этого места. | Логическое значение | false |
| lockAudio | Заблокировать микрофон места. | Блокировка камеры соответствующего места остановит трансляцию потока звука с этого места.  | Логическое значение | true, когда микрофон места заблокирован. |

iOS

Android

```
// Заблокировать место.let lockSeatIndex = 5let lockSeatParam = TUISeatLockParams()lockSeatParam.lockSeat = trueself.seatGridView.lockSeat(index: lockSeatIndex, lockMode: lockSeatParam) {   // Место успешно заблокировано} onError: { code, message in    // Ошибка при блокировке места}// Заблокировать микрофон места.let lockSeatAudioIndex = 6let lockSeatAudioParam = TUISeatLockParams()lockSeatAudioParam.lockAudio = trueself.seatGridView.lockSeat(index: lockSeatAudioIndex, lockMode: lockSeatAudioParam) {    // Микрофон места успешно заблокирован} onError: { code, message in     // Ошибка при блокировке микрофона места}
```

```
// Заблокировать место.int lockSeatIndex = 5;TUIRoomDefine.SeatLockParams lockSeatParam = new TUIRoomDefine.SeatLockParams();lockSeatParam.lockSeat = true;seatGridView.lockSeat(lockSeatIndex, lockSeatParam, new TUIRoomDefine.ActionCallback() {    @Override    public void onSuccess() {        // Место успешно заблокировано    }    @Override    public void onError(TUICommonDefine.Error error, String message) {       // Ошибка при блокировке места    }});// Заблокировать микрофон места.int lockSeatIndex = 6;TUIRoomDefine.SeatLockParams lockSeatAudioParam = new TUIRoomDefine.SeatLockParams();lockSeatAudioParam.lockAudio = true;seatGridView.lockSeat(lockSeatIndex, lockSeatAudioParam, new TUIRoomDefine.ActionCallback() {    @Override    public void onSuccess() {        // Микрофон места успешно заблокирован    }    @Override    public void onError(TUICommonDefine.Error error, String message) {        // Ошибка при блокировке микрофона места    }});
```

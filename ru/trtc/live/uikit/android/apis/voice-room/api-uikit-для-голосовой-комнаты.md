# API UIKit для голосовой комнаты

## Введение в API

VoiceRoomKit — это компонент голосовой чат-комнаты **с интерфейсом пользователя**, использующий VoiceRoomKit API, вы можете быстро реализовать голосовую чат-комнату через простые интерфейсы. Если вы хотите протестировать и отладить голосовую чат-комнату, прочитайте [Быстрое развертывание демо](https://www.tencentcloud.com/document/product/647/63378#). Если вы хотите интегрировать нашу функцию непосредственно в ваш проект, прочитайте [Быстрая интеграция (TUILiveKit)](https://www.tencentcloud.com/document/product/647/60335#).

## Детали API

### createInstance

Получить экземпляр объекта VoiceRoomKit.

```
static VoiceRoomKit createInstance(Context context)
```

**Параметр:**

| **Параметр** | **Тип** | **Примечание** | **Значение по умолчанию** | **Значение** |
| --- | --- | --- | --- | --- |
| context | Context | Обязательно | - | Объект контекста Android |

**Возвращаемое значение:** VoiceRoomKit

### createRoom

Создать комнату прямой трансляции в голосовой чат-комнате.

```
void createRoom(String roomId, VoiceRoomDefine.CreateRoomParams params);
```

**Параметр:**

| **Параметр** | **Тип** | **Примечание** | **Значение по умолчанию** | **Значение** |
| --- | --- | --- | --- | --- |
| roomId | String | Обязательно | - | ID комнаты прямой трансляции |
| params | [CreateRoomParams](https://www.tencentcloud.com/document/product/647/66794#dc90ee9d-a1c6-4933-b7b1-7899a76eae0e) | Обязательно | - | Параметры создания комнаты прямой трансляции |

**Возвращаемое значение:** void

### enterRoom

Войти в комнату прямой трансляции голосовой чат-комнаты, используя roomId.

```
void enterRoom(String roomId);
```

**Параметр:**

| **Параметр** | **Тип** | **Примечание** | **Значение по умолчанию** | **Значение** |
| --- | --- | --- | --- | --- |
| roomId | String | Обязательно | - | ID комнаты прямой трансляции |

**Возвращаемое значение:** void

## Введение в VoiceRoomDefine

VoiceRoomKit является классом модели данных UIKit для голосовых чат-комнат, включающим в основном следующие структуры данных:

### CreateRoomParams

Объект параметров при создании комнаты прямой трансляции голосовой чат-комнаты, включающий в основном следующие параметры конфигурации:

| **Параметр** | **Тип** | **Примечание** | **Значение по умолчанию** | **Значение** |
| --- | --- | --- | --- | --- |
| roomName | String |  | "" | Имя голосовой чат-комнаты |
| maxAnchorCount | int |  | 10 | Максимальное количество пользователей на сцене |
| seatMode | String | Обязательно |  | Режим включения микрофона делится на следующие два типа: FREE_TO_TAKE(1): Режим свободного присоединения к подиуму, где аудитория может свободно присоединиться к месту без подачи заявки. APPLY_TO_TAKE(2): Режим подачи заявки на подключение микрофона, где аудитория нуждается в одобрении хоста или администратора для присоединения к месту. |

# Запуск демонстрационного приложения

Это руководство проведет вас через настройку **TUILiveKit** Android Demo всего за 10 минут. Вы сможете испытать полнофункциональное видеотрансляцию и чат-комнату голосовых вызовов с полным пользовательским интерфейсом.

## Быстрое ознакомление

Вы можете непосредственно испытать все функции **Live**, используя следующие методы.

## Предварительные требования

### Включение сервиса

Обратитесь к разделу [Включение сервиса](https://www.tencentcloud.com/document/product/647/60033), чтобы получить пробную версию TUILiveKit. Затем перейдите в [консоль](https://console.trtc.io/app) для управления приложением и получите следующие данные:

- `SDKAppID`: идентификатор приложения (обязательный). Tencent Cloud использует `SDKAppId` для выставления счетов и дополнительной информации.
- `SDKSecretKey`: секретный ключ приложения, используется для инициализации конфигурационного файла с конфиденциальной информацией.

### Подготовка окружения

- Android 5.0 (уровень API 21) или выше
- Gradle 8.0 или выше
- Два устройства Android версии 5.0 или выше

## Инструкции

### Получение демонстрационного приложения

1. **Скачивание исходного кода**: скачайте исходный код [TUIKit_Android](https://github.com/Tencent-RTC/TUIKit_Android) с GitHub или выполните следующую команду в терминале:

```
  git clone https://github.com/Tencent-RTC/TUIKit_Android.git
```

2. **Откройте проект**: откройте проект TUILiveKit Android в `Android Studio`:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/02f11d34bef911f0a808525400bf7822.png)

### Настройка

1. **Конфигурирование `SDKAppID` и `SDKSecretKey`**: откройте файл `TUIKit_Android/application/debug/src/main/java/com/tencent/qcloud/tuikit/debug/GenerateTestUserSig.java` и введите ваши `SDKAppID` и `SDKSecretKey`:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/03127a95bef911f08863525400e889b2.png)

### **Сборка и запуск демонстрационного приложения**

1. В верхнем правом углу `Android Studio` выберите устройство, на которое вы хотите запустить Demo:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/02f82e15bef911f0a808525400bf7822.png)

2. Нажмите Run, чтобы развернуть `TUIKit_Android` на выбранное устройство.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/02f6f350bef911f0a6c652540044a08e.png)

> **Примечание:** для полного опыта трансляции войдите в Demo на двух устройствах с разными пользователями — один в качестве ведущего, другой в качестве зрителя.

> **Примечание:** не используйте простые UserID, такие как "1", "123" или "111". Используйте уникальные UserID при отладке.

## Часто задаваемые вопросы

### Demo сообщает об ошибке подписи или сбое входа?

Убедитесь, что **SDKAppID** и **SDKSecretKey** в файле `TUIKit_Android/application/debug/src/main/java/com/tencent/qcloud/tuikit/debug/GenerateTestUserSig.java` верны. Убедитесь, что они совпадают со значениями на странице [Управления приложениями консоли Tencent Cloud](https://console.trtc.io/app).

### Будет ли работа Demo затронута, если версия Gradle ниже 8.0?

Да. Если ваша версия `Gradle` ниже 8.0, вы можете столкнуться с ошибками при скачивании зависимостей или ошибками сборки. Обновите версию `Gradle` в Android Studio в соответствии с требованиями.

### После входа в комнату вы можете столкнуться с кодом ошибки "-3301" и сообщением "Сервисы недоступны в вашем регионе".

Если вы получили этот код ошибки, [свяжитесь с нами](https://trtc.io/contact) для получения поддержки.

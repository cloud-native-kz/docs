# Прослушивание аудитории

`TUILiveKit` Voice Chat Room предоставляет комплексный готовый к использованию интерфейс для сценариев чистого аудиопотока в прямом эфире. Это позволяет быстро реализовать основные функции, такие как прослушивание аудитории и взаимодействие с микрофоном, избавляя вас от необходимости самостоятельно управлять сложной логикой пользовательского интерфейса или управления местами.

## Обзор функций

- **Прослушивание трансляций**: Слушайте аудиопоток хоста в режиме реального времени с четкостью и низкой задержкой.
- **Со-гость**: Запросите присоединение к микрофону и взаимодействуйте с хостом через аудио.
- **Информация о прямом эфире**: Просмотрите объявления комнаты и список онлайн-зрителей.
- **Взаимодействие в эфире**: Используйте функции, такие как текстовые комментарии, подарки и лайки.

## Быстрая интеграция

### Интеграция компонента

Следуйте руководству [Подготовка](https://www.tencentcloud.com/document/product/647/60335) для интеграции `TUILiveKit` в ваш проект.

### Запуск Activity голосовой комнаты

Компонент `VoiceRoomActivity` предоставляет полный пользовательский интерфейс и бизнес-логику со стороны аудитории для голосовой комнаты. Просто запустите этот `Activity`, чтобы позволить пользователям присоединиться к комнате прямого эфира. Обычно, когда пользователь выбирает комнату из [Списка прямых трансляций](https://www.tencentcloud.com/document/product/647/74325), вы должны перейти к представлению аудитории. См. пример ниже:

```
import android.content.Intentimport android.os.Bundleimport androidx.appcompat.app.AppCompatActivityimport com.trtc.uikit.livekit.voiceroom.VoiceRoomActivityimport com.trtc.uikit.livekit.voiceroom.VoiceRoomDefine// YourLiveListActivity represents your live room list pageclass YourLiveListActivity : AppCompatActivity() {    // Handle "click live room" event    fun onJoinVoiceRoomClicked(roomId: String) {        // 1. Prepare the Intent and pass the required parameters        //    - roomId: The ID of the live room you want to join        //    - behavior: VoiceRoomDefine.RoomBehavior.JOIN means joining the room as an "audience"        val intent = Intent(this, VoiceRoomActivity::class.java).apply {            putExtra(VoiceRoomActivity.INTENT_KEY_ROOM_ID, roomId)            putExtra(VoiceRoomActivity.INTENT_KEY_ROOM_BEHAVIOR, JOIN.ordinal)        }        // 2. Navigate to the voice chat room page        startActivity(intent)    }}
```

- **Описание параметров Intent Extra:**

| **Параметр** | **Тип** | **Описание** |
| --- | --- | --- |
| `VoiceRoomActivity.INTENT_KEY_ROOM_ID` | `string` | Глобально уникальный ID комнаты прямого эфира. |
| `VoiceRoomActivity.INTENT_KEY_ROOM_BEHAVIOR` | `Int` | Поведение при входе в комнату:- `AUTO_CREATE`: Автоматически создать и войти в комнату прямого эфира.- `PREPARE_CREATE`: Сначала войти на страницу предпросмотра перед эфиром, затем создать и войти в комнату прямого эфира после того, как пользователь нажмет кнопку "Начать эфир".- `JOIN`: Присоединиться к комнате как зритель. |

## Настройка пользовательского интерфейса

`TUILiveKit` поддерживает настройку пользовательского интерфейса в соответствии с различными бизнес-потребностями. Вы можете легко изменить текст интерфейса и значки.

### Настройка текста (строковые ресурсы)

`TUILiveKit` использует стандартные **файлы ресурсов Android XML** для управления текстом, отображаемым в пользовательском интерфейсе. Вы можете напрямую изменить строки, требующие корректировки, через XML-файл:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6d9d2af2c07211f0a0935254007c27c5.png)

### Настройка значков (ресурсы Drawable)

`TUILiveKit` использует стандартную **папку ресурсов Android drawable** для управления ресурсами изображений пользовательского интерфейса. Вы можете быстро изменить пользовательские значки, заменив файлы ресурсов. При замене убедитесь, что новые имена файлов совпадают с исходными именами файлов.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6d94ebddc07211f0b4a7525400454e06.png)

## Часто задаваемые вопросы

### Нет звука у зрителя после присоединения в качестве со-гостя?

Убедитесь, что приложение имеет разрешение на доступ к микрофону. Перейдите в раздел `Информация о приложении > Разрешения > Микрофон` на вашем устройстве и убедитесь, что доступ к микрофону включен.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/839f1b75c07311f0b4a7525400454e06.png)

### Текстовые комментарии, отправленные зрителем, не видны другим в комнате?

Есть 3 причины, на которые вы можете обратить внимание:

- Проверьте подключение к сети, чтобы убедиться, что устройство зрителя находится в сети.
- Зритель был отключен хостом и не может отправлять текстовые комментарии.
- Текстовый комментарий содержит заблокированные ключевые слова. Убедитесь, что комментарий соответствует правилам комнаты.

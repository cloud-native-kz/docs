# Интеграция

Данная статья содержит подробное руководство по быстрому запуску Electron **Live Streaming Kit**. Всего за несколько шагов вы сможете быстро запустить мощное программное обеспечение для трансляции видео в высоком разрешении на рабочий стол. Оно не только поддерживает эффективное смешивание нескольких источников медиа, таких как камеры, экраны, окна и изображения, но также предоставляет гибкие функции компоновки, редактирования и композитинга изображений, придав вашему видеоконтенту более профессиональный вид.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3109b1e1984c11f09b0c525400bf7822.png)

## Шаг 1: Подготовка окружения

Перед запуском демонстрации необходимо обратиться к разделу [подготовка](https://www.tencentcloud.com/document/product/647/66938) и выполнить требуемую конфигурацию окружения, а также активировать соответствующие сервисы.

> **Примечание:** Данная демонстрация в настоящий момент поддерживает только операционные системы Windows 10+. Убедитесь, что ваша операционная система соответствует этим требованиям.

## Шаг 2: Загрузка демонстрации

1. Загрузите [исходный код проекта Streaming Assistant](https://github.com/Tencent-RTC/ultra-live-electron) с GitHub или выполните следующую команду непосредственно в командной строке:

```
git clone https://github.com/Tencent-RTC/ultra-live-electron.git
```

2. Установите зависимости

```
cd ultra-live-electronnpm install
```

## Шаг 3: Конфигурация демонстрации

Обратитесь к разделу [подготовка](https://www.tencentcloud.com/document/product/647/66938) на шаге 1. Откройте файл `ultra-live-electron/src/debug/basic-info-config.js` в демонстрации и введите **SDKAppID** и **SDKSecretKey**, полученные при активации сервиса на этапе подготовки:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/864d4151985511f0b95552540044a08e.png)

## Шаг 4: Начните вашу первую трансляцию

Запустите текущий проект с помощью следующей команды

```
npm run start
```

После запуска демонстрации вы можете обогатить содержимое вашей трансляции и начать свою первую трансляцию, выполнив следующие шаги.

1. **Добавьте источник видео**: Перед началом трансляции необходимо добавить источник видео для трансляции. Приложение в настоящий момент поддерживает: камеру, общий доступ к экрану, общий доступ к окну и изображение. Нажмите кнопку добавления слева на интерфейсе и выберите источник видео, который вы хотите добавить.
2. **Отрегулируйте компоновку**: Используйте мышь для перемещения и изменения размера источника видео. После выбора источника видео (появится желтая граница) щелкните правой кнопкой мыши, чтобы вызвать контекстное меню для поворота или регулировки уровня отображения.
3. **Начните трансляцию**: Нажмите кнопку "Начать трансляцию" на интерфейсе. Когда кнопка изменится на "Завершить трансляцию", трансляция успешно начнется.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/ca988fda985211f0ad595254007c27c5.png)

> **Примечание:** На этом этапе вы завершили весь процесс Live Streaming Kit и можете начать трансляцию в любой момент. Данное настольное программное обеспечение поддерживает только трансляцию. Для просмотра трансляций используйте мобильное приложение или веб-браузер. Вам потребуется запустить TUILiveKit для iOS, Android или веб-версии. Найдите и откройте созданную комнату трансляции в списке трансляций.

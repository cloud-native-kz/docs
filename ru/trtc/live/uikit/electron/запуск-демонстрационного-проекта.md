# Запуск демонстрационного проекта

- Данное руководство предоставляет подробное пошаговое описание для быстрого начала работы с демонстрацией **Electron Live Streaming Assistant**. Всего за несколько шагов вы можете запустить мощное настольное приложение для высокодефинного потокового видео.
- Демонстрация позволяет смешивать несколько источников медиаконтентаâвключая камеру, экран, окно и изображенияâи предлагает гибкие функции компоновки, редактирования и композитинга изображений для создания профессиональных потоковых трансляций.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/343ccca7cf1f11ef9d3a5254001c06ec.png)

## Быстрый старт

Вы можете напрямую использовать все функции **Live** следующими способами.

## Предварительные требования

### Включение сервиса

Обратитесь к разделу [Включение сервиса](https://www.tencentcloud.com/document/product/647/60033) для получения пробной лицензии TUILiveKit. В [Консоли](https://console.trtc.io/app) вы можете найти следующую информацию:

- `SDKAppID`: Требуемый идентификатор приложения. Используется Tencent Cloud для выставления счетов и аналитики.
- `SDKSecretKey`: Секретный ключ приложения. Используется для инициализации файла конфигурации ключевой информацией.

### Подготовка окружения

- **Node.js**: â¥ 18.19.1 (рекомендуется официальная LTS-версия)
- **Vue:** â¥ 3.4.21
- **Современный браузер:** браузер, поддерживающий [API WebRTC](https://web.sdk.qcloud.com/trtc/webrtc/demo/detect/index.html)
- **Устройства:** камера, микрофон и динамик
- **Операционная система:** в настоящее время поддерживаются только Windows 10 и более поздние версии.

## Инструкции

### Получение демонстрационного проекта

> **Примечание:** в настоящее время данная демонстрация поддерживает Windows 10 и более поздние версии. Убедитесь, что ваша операционная система соответствует этому требованию.

1. Скачайте [исходный код проекта Live Streaming Assistant](https://github.com/Tencent-RTC/ultra-live-electron) с GitHub или выполните следующую команду в терминале:

```
git clone https://github.com/Tencent-RTC/ultra-live-electron.git
```

2. Установите зависимости:

```
cd ultra-live-electronnpm install
```

### Настройка

Откройте файл `ultra-live-electron/src/debug/basic-info-config.js` и введите полученные `SDKAppID` и `SDKSecretKey` при включении сервиса.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/477ee320bfaa11f09a6b525400454e06.png)

### Сборка и запуск демонстрационного проекта

Запустите проект, выполнив:

```
npm run start
```

### Опробование основных функций

После запуска демонстрационного проекта выполните следующие шаги для обогащения трансляции и начала вещания:

1. **Добавление видеоисточника**: перед началом трансляции добавьте видеоисточник. Приложение в настоящее время поддерживает источники камеры, общего доступа к экрану, общего доступа к окну и изображения. Нажмите кнопку добавления на левой стороне интерфейса и выберите требуемый видеоисточник.
2. **Регулировка компоновки**: перетащите видеоисточники для их перемещения и изменения размера. После выбора видеоисточника (выделено желтой рамкой) щелкните правой кнопкой мыши, чтобы открыть контекстное меню для **поворота** или регулировки **порядка отображения**.
3. **Запуск потоковой трансляции:** нажмите кнопку "**Start Live**". Когда кнопка изменится на "**End Live**", ваша трансляция запущена.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/097eac10bfbc11f0b9945254005ef0f7.png)

> **Примечание:** не используйте простые `UserIDs`, такие как "1", "123" или "111". TRTC не поддерживает многоконечный вход для одного и того же `UserID`, а эти простые `UserIDs` вероятно будут использованы другими разработчиками при совместной разработке, что может привести к ошибкам входа. Используйте характерные `UserIDs` при отладке.

### **Следующие шаги**

Вы завершили процесс настройки Live Streaming Assistant и можете начать трансляцию в любой момент.

> **Примечание:** это настольное приложение поддерживает только трансляцию. Для просмотра живого потока используйте мобильное приложение или веб-браузер. Запустите `TUILiveKit` на iOS, Android или Web и найдите вашу комнату трансляции в списке прямых трансляций.

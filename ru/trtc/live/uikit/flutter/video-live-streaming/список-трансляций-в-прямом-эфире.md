# Список трансляций в прямом эфире

## Предпросмотр функции

Этот документ содержит подробное описание функции **страницы списка трансляций в прямом эфире** в **TUILiveKit**. Вы можете непосредственно интегрировать нашу готовую страницу списка трансляций в ваш существующий проект, следуя этому руководству. Эта страница поддерживает такие функции, как обновление при потягивании, загрузка дополнительного контента при прокрутке и присоединение к комнате прямого эфира по клику.

### Шаг 3: Добавление страницы списка трансляций

В точке входа, где вам нужен доступ к странице списка трансляций (определяется вашей бизнес-логикой), выполните следующую операцию для навигации на страницу списка трансляций или интеграции её в существующее дерево **Widget**:

Прямая навигация

Интеграция в дерево Widget

```
// navigate to live list pageNavigator.push(context, MaterialPageRoute(  builder: (context) {     return LiveListWidget();}));
```

## Часто задаваемые вопросы

### Что делать, если на странице не отображаются трансляции после интеграции функции списка трансляций?

Если вы видите пустую страницу, необходимо проверить, завершили ли вы [Полный вход в систему](https://www.tencentcloud.com/document/product/647/63255#6b283542-df2f-4f16-b3b4-1fb0065d497a). Для тестирования функции вы можете использовать два устройства: одно устройство для запуска трансляции, а другое устройство на странице списка для загрузки комнаты трансляции, которая перешла в режим прямого эфира.

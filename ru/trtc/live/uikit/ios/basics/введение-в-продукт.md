# Введение в продукт

## Обзор

Live — это компонент для интерактивного прямого вещания в реальном времени, включающий функции трансляции якоря, просмотра аудитории, чата и эмодзи, лайков и отправки подарков, управления аудиторией и управления несколькими гостями. Он идеально подходит для сценариев прямого вещания в сфере развлечений, электронной коммерции, образования и многого другого. Интегрировав Live, вы сможете быстро добавить все перечисленные выше функции прямого вещания в свое приложение всего за три шага, обеспечивая быстрое развертывание вашего сервиса. ## Функции

### Основные функции

|  | **Описание функции** |
| --- | --- |
| Высокое качество прямого вещания | Хост может предпросмотреть экран перед началом трансляции, с возможностью запуска и остановки потока |
| Комната голосового чата | Хост может установить фон и звуковые эффекты перед началом комнаты голосового чата, с возможностью открытия и закрытия комнаты |
| Просмотр прямого вещания | Поддерживает одновременный просмотр прямого эфира несколькими зрителями |
| Несколько гостей в одной комнате | Поддерживает нескольких гостей в одной комнате |
| Кросс-комнатное подключение нескольких гостей/PK | Поддерживает двухсторонее и многостороннее кросс-комнатное подключение/PK |
| Список аудитории | Компонент списка аудитории поддерживает отображение последних 100 зрителей, вошедших в прямую комнату, и общего количества зрителей, с вариантами упрощенного и подробного представления |
| Чат и эмодзи | Пользователи могут отправлять эмодзи и текстовые сообщения, которые отображаются в реальном времени в прямой комнате |
| Лайки | Поддерживает функцию лайков и отслеживание базовых данных о лайках |

### Продвинутые функции

|  | **Описание функции** |
| --- | --- |
| Управление членами | Поддерживает блокировку и отключение звука зрителей для хоста. |
| Предпросмотр прямого эфира | Зрители могут предпросмотреть содержимое прямой комнаты без входа в нее. |
| Лента прямого вещания | Зрители могут прокручивать вверх и вниз экран для просмотра различных прямых комнат, поддерживая форматы водопада как с одной, так и с двумя колонками. |
| Система подарков | Предоставляет комплексное решение для подарков, включая конфигурацию материалов подарков, отображение панели, воспроизведение спецэффектов и возможности аналитики данных для поддержки быстрой монетизации. |
| Количество роботов | Поддерживает добавление роботов в прямую комнату, которые могут присоединяться к комнате как фиктивные зрители, взаимодействовать через нескольких гостей и отправлять комментарии. |
| Просмотр прошлых сообщений | Пользователи, присоединившиеся к прямой комнате, могут просматривать сообщения, отправленные до их входа в комнату. |
| Подписка | Зрители могут подписаться на хоста и других зрителей в прямой комнате. |
| Поиск в комнате | Поддерживает поиск членов в прямой комнате. |
| Максимальное качество изображения | Значительно снижает битрейт при сохранении качества изображения или улучшает качество изображения при том же битрейте, оптимизируя впечатление от просмотра в условиях ограниченной полосы пропускания. |
| Помощник прямого вещания для ПК | Поддерживает захват многокамерных сцен и локальное аудио-видео микширование, подходит для профессионального вещания на ПК. |
| RTMP вход/ввод интернет-потока медиа | Отправляет локальное видео и аудио в прямую комнату через протокол RTMP, поддерживая сценарии такие как трансляция OBS хоста и прямое вещание виртуального хоста. |
| Оптимизация QUIC при слабой сети | Автоматически выбирает и переключает сетевые каналы или передачу по множественным каналам на основе сетевого состояния терминала, улучшая скорость передачи в условиях слабой сети. |
| Красота AR | С включенной по умолчанию красотой вы можете интегрировать [Beauty AR](https://trtc.io/document/60216) для разблокировки более широкого спектра продвинутых функций красоты. |
| Звуковые эффекты | Поддерживает установку громкости, изменения голоса и эффектов реверберации перед началом трансляции. |

#  Коды ошибок

## Общий код ошибки

| Код ошибки | Описание |
| --- | --- |
| 0 | Операция успешна. |
| -1 | Временно неклассифицированная общая ошибка. |
| -2 | `Request Rate` ограничен, попробуйте позже. |
| -3 | Повторите операцию. |
| -1000 | Не найден `SDKAppID`, подтвердите информацию приложения в [консоли TRTC](https://console.tencentcloud.com/trtc/allapp). |
| -1001 | При вызове API переданы незаконные параметры, проверьте корректность параметров. |
| -1002 | Не авторизован, вызовите API `Login`. |
| -1003 | Не удалось получить разрешение, несанкционированное разрешение на аудио/видео, проверьте, включена ли `Device Permission`. |
| -1004 | Эта функция требует дополнительного пакета. Активируйте соответствующий пакет в [консоли TRTC](https://console.tencentcloud.com/trtc/allapp). |
| -3301 | Доступ в сеть ограничен. Пожалуйста, [свяжитесь с нами](https://trtc.io/contact) для получения поддержки. |

| -2108 | Пользователь уже находится в другой комнате, один экземпляр roomengine поддерживает только пользователя, входящего в одну комнату, для входа в другую комнату, пожалуйста, покиньте комнату или используйте новый экземпляр roomengine |

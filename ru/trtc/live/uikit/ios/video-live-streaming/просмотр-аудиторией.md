# Просмотр аудиторией

**TUILiveKit** предоставляет страницу просмотра для аудитории с различными и удобными функциями трансляции и взаимодействия, позволяя быстро интегрировать её в ваше приложение для удовлетворения разнообразных потребностей аудитории.

## Обзор функций

- **Прямая трансляция:** Четкое и плавное просмотрение прямой трансляции хоста в реальном времени.
- **Интерактивное со-ведение:** Подайте заявку на подключение микрофона для взаимодействия с хостом через аудио и видео.
- **Информация о трансляции:** Просмотрите название комнаты трансляции, описание, список аудитории и т.д.
- **Интерактивное взаимодействие:** Отправляйте подарки (с анимационными эффектами и уведомлением хосту), ставьте лайки (с анимацией и статистикой в реальном времени) и взаимодействуйте через стену комментариев.

## Быстрый старт

### Шаг 1. Активируйте сервис

Обратитесь к документации [Активация сервиса](https://www.tencentcloud.com/document/product/647/60033), чтобы включить пробный период или официальный пакет.

### Шаг 2. Интеграция кода

Обратитесь к руководству [Подготовка](https://www.tencentcloud.com/document/product/647/72223) для интеграции SDK TUILiveKit.

### Шаг 3. Добавьте представление просмотра аудиторией

В контроллере представления аудитории `YourAudienceViewController` инициализируйте и добавьте `AudienceContainerView` для просмотра трансляции аудиторией:

Swift

```
YourAudienceViewController
```

### Шаг 4. Перейдите к представлению **Просмотр аудиторией**

Обычно в [списке прямых трансляций](https://www.tencentcloud.com/document/product/647/72226) используйте следующий пример кода, когда необходимо перейти на страницу просмотра:

Swift

```
YourAudienceViewController
```

## Настройте макет вашего пользовательского интерфейса

### Настройте область функций `AudienceContainerView`

Вызывая API `audienceView`, созданный на [Шаге 3](#c9d8cb5f-0da5-496d-b175-0c0d58cdfb02), вы можете настроить и отключить (скрыть) область операций или конкретные функции на странице просмотра:

Swift

```
YourAudienceViewController
```

| **API** | **Описание** |
| --- | --- |
| `disableHeaderLiveData(true)` | Скрыть информацию о трансляции в верхней секции операций |
| `disableHeaderFloatWin(true)` | Скрыть функцию плавающего окна в верхней секции операций |
| `disableHeaderVisitorCnt(true)` | Скрыть функцию списка аудитории в верхней секции операций |
| `disableFooterCoGuest(true)` | Скрыть функцию со-ведения в нижней секции операций |
| `disableSliding(true)` | Запретить переключение трансляций путем проведения пальцем |

### Настройка текста (локализация)

TUILiveKit использует формат [Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog)(`.xcstrings`), введенный в **Xcode 15**, для управления текстом, отображаемым в пользовательском интерфейсе. Вы можете изменять ресурсы строк, используя графический интерфейс Xcode:

> **Примечание:** [Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) (.xcstrings) — это формат локализации, введенный в Xcode 15. Он улучшает способ управления локализованными строками разработчиками, поддерживая структурированный формат для работы с множественными формами, вариантами для конкретных устройств и т.д. Этот формат становится рекомендуемым способом управления локализацией для приложений iOS и macOS.

## Часто задаваемые вопросы

### Почему экран видео черный, когда зритель выбирает видео со-ведение?

Перейдите в **Настройки телефона** > **Приложение >** **Камера**, проверьте, включено ли разрешение на доступ к камере, как показано на изображении ниже:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/6bcbba71991611f0961e52540099c741.png)

### Почему другие зрители в комнате трансляции не видят содержимое стены комментариев, отправленное зрителем?

- **Причина 1**: Сначала проверьте подключение к сети, чтобы убедиться, что сетевое соединение устройства зрителя нормально.
- **Причина 2**: Зритель был **отключен (заблокирован)** хостом и не может отправлять комментарии.
- **Причина 3**: Содержимое стены комментариев зрителя попадает под **блокировку ключевых слов**. Пожалуйста, убедитесь, что содержимое, отправляемое зрителем, соответствует правилам комнаты трансляции.

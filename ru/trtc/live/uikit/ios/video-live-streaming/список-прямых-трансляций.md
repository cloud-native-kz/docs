## Обзор функции

- **Одноколоночный каскадный поток**: вид по умолчанию, одновременно отображающий предпросмотр 1 прямой трансляции.

### Шаг 5. Оптимизация переходов между страницами для взаимодействия с `LiveListView`

Чтобы обеспечить более полный и плавный опыт при переходах между страницами, компонент `LiveListView` в сочетании с атрибутами системы маршрутизации страниц iOS предоставляет методы `refreshLiveList` и `onRouteToNextPage`. С помощью всего одной строки кода вы можете оптимизировать отображение списка прямых трансляций и воспроизведение видеопотока. Способ оптимизации описан далее:

Swift

```
// Реализуйте в контроллере представления списка прямых трансляций, например: YourLiveListViewControllerclass YourLiveListViewController: UIViewController {    override func viewDidAppear(_ animated: Bool) {        super.viewDidAppear(animated)        // refreshLiveList: вызовите этот метод в viewDidAppear, чтобы получить последний список каждый раз при возврате на текущую страницу        liveListView.refreshLiveList()    }    override func viewWillDisappear(_ animated: Bool) {        super.viewWillDisappear(animated)        // onRouteToNextPage: вызовите этот метод в viewWillDisappear, чтобы остановить воспроизведение видеопотока на текущей странице списка прямых трансляций при входе на другую страницу        liveListView.onRouteToNextPage()    }}
```

## Часто задаваемые вопросы

### Что делать, если на странице не отображаются прямые трансляции после интеграции функции списка прямых трансляций?

Если вы видите пустую страницу, вам необходимо проверить, завершили ли вы [полный вход](https://www.tencentcloud.com/document/product/647/72223#6ebcab01-c3d7-4fb4-a933-016cf1cf5193). Для тестирования этой функции вы можете использовать два устройства: на одном устройстве начать трансляцию, а на другом на странице списка прямых трансляций загружать список текущих прямых трансляций.

### В чем разница между одноколоночным и двухколоночным каскадными потоками?

Одноколоночный каскадный поток показывает предпросмотр одной прямой комнаты одновременно, в то время как двухколоночный каскадный поток показывает предпросмотр двух прямых комнат одновременно. Вы можете выбрать соответствующий макет в зависимости от требований вашего дизайна.

### Существует ли плата за предпросмотр прямых трансляций?

Да, продолжительность предпросмотра прямой комнаты учитывается в аудио/видео продолжительности аудитории, что является платной услугой. Для получения подробной информации о [тарификации](https://www.tencentcloud.com/document/product/647/66078) обратитесь к соответствующему содержимому в документации.

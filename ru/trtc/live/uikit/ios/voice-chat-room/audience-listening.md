# Прослушивание трансляции в аудитории

`TUILiveKit` Voice Chat Room предоставляет комплексный готовый к использованию интерфейс для сценариев чистой аудиотрансляции. Он позволяет быстро реализовать основные функции, такие как прослушивание аудитории и микрофонное взаимодействие, избегая необходимости самостоятельно управлять сложной логикой пользовательского интерфейса или управления местами.

## Обзор функций

- **Прослушивание трансляций**: Слушайте аудиопоток хоста в реальном времени с ясностью и низкой задержкой.
- **Соведущий**: Отправьте запрос на присоединение к микрофону и взаимодействуйте с хостом через аудио.
- **Информация о трансляции**: Просмотрите объявления комнаты и список онлайн-членов аудитории.
- **Живое взаимодействие**: Используйте функции, такие как текущие комментарии, подарки и лайки.

## Быстрая интеграция

### Интеграция компонента

Следуйте руководству [Подготовка](https://www.tencentcloud.com/document/product/647/60334) для интеграции `TUILiveKit` в ваш проект.

### Создание и представление контроллера представления голосовой комнаты

`TUIVoiceRoomViewController` предоставляет полный пользовательский интерфейс на стороне аудитории и логику бизнеса для сценария голосовой комнаты. Просто создайте экземпляр и представьте этот контроллер представления, чтобы позволить пользователям войти в прямую комнату. Обычно, когда пользователь нажимает на комнату в [Списке трансляций](https://www.tencentcloud.com/document/product/647/74326), перейдите на страницу просмотра аудитории. См. пример ниже:

```
import TUILiveKitimport UIKit// YourLiveListViewController represents your live list view controllerclass YourLiveListViewController: UIViewController {    // Handle "Join Voice Room" action    @objc func onJoinVoiceRoomClicked(roomId: String) {        // 1. Instantiate the voice room controller        //    - roomId: The ID of the live room to join        //    - behavior: .join indicates joining as an audience member        let voiceRoomVC = TUIVoiceRoomViewController(roomId: roomId, behavior: .join)        voiceRoomVC.modalPresentationStyle = .fullScreen        // 2. Present the voice room page        present(voiceRoomVC, animated: true)    }}
```

- **Параметры TUIVoiceRoomViewController:**

| Параметр | Тип | Описание |
| --- | --- | --- |
| `roomId` | `string` | Глобально уникальный идентификатор прямой комнаты. |
| `behavior` | `RoomBehavior` | Поведение входа в комнату:- `autoCreate`: Автоматически создать и присоединиться к прямой комнате.- `prepareCreate`: Войти на страницу предпросмотра перед трансляцией. Комната создается и присоединяется после того, как пользователь нажмет "Начать трансляцию".-`join`: Присоединиться к комнате как член аудитории. |

## Настройка пользовательского интерфейса

`TUILiveKit` поддерживает настройку пользовательского интерфейса в соответствии с различными бизнес-потребностями. Вы можете легко изменять текст интерфейса и значки.

### Настройка текста (локализация)

`TUILiveKit` использует формат [Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog)(`.xcstrings`), введенный в **Xcode 15**, для управления текстом, отображаемым в пользовательском интерфейсе. Вы можете изменять строковые ресурсы, используя графический интерфейс Xcode:

> **Примечание:**[Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) (.xcstrings) — это формат локализации, введенный в Xcode 15. Он улучшает управление локализованными строками разработчиками, поддерживая структурированный формат для работы с множественными числами, вариантами для конкретных устройств и т. д. Этот формат становится рекомендуемым способом управления локализацией для приложений iOS и macOS.

## Часто задаваемые вопросы

### Нет звука у члена аудитории после присоединения к микрофону?

Перейдите в `Параметры > Приложение > Микрофон` на вашем устройстве и убедитесь, что разрешения на использование микрофона включены.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/73e81142c07411f0a6dd5254005ef0f7.png)

### Текущие комментарии, отправленные членом аудитории, не видны другим в комнате?

Есть 3 причины, на которые вы можете обратить внимание:

- Проверьте подключение к сети, чтобы убедиться, что устройство члена аудитории находится в сети.
- Член аудитории был отключен хостом и не может отправлять текущие комментарии.
- Текущий комментарий содержит заблокированные ключевые слова. Подтвердите, что комментарий соответствует правилам комнаты.

# Трансляция хоста

`UILiveKit` Voice Chat Room — это готовое к использованию интерфейс, предоставляющий все основные функциональные возможности для запуска прямой трансляции. Это позволяет вам интегрировать поток трансляции хоста в вашего приложение несколькими простыми шагами.

## Обзор функций

- **Предварительный просмотр перед трансляцией**: позволяет хостам настраивать личные параметры, такие как имя комнаты и изображение обложки перед началом трансляции.
- **Управление местами**: поддерживает операции управления местами, включая занятие места, освобождение места, отключение звука и блокировку мест.
- **Взаимодействие со зрителями**: обеспечивает интерактивные функции, такие как прямой чат (комментарии), виртуальные подарки и лайки.
- **Управление прямой комнатой**: отображает список онлайн-пользователей и предоставляет элементы управления модерацией, такие как отключение звука или удаление пользователей из прямой комнаты.

| **Предварительный просмотр перед трансляцией** | **Управление местами** | **Взаимодействие со зрителями** | **Управление прямой комнатой** |

## Интеграция компонента

### Создание и представление контроллера представления Voice Room

Компонент `TUIVoiceRoomViewController` предоставляет полный пользовательский интерфейс на стороне хоста и бизнес-логику для сценария голосовой комнаты. Чтобы позволить хостам начать прямую трансляцию, просто создайте и представьте этот контроллер представления. Мы рекомендуем вызвать следующую логику в обработчике нажатия кнопки "Start Live" вашего приложения:

```
import TUILiveKitimport UIKit// YourViewController представляет контроллер представления, из которого вы начинаете прямую трансляциюclass YourViewController: UIViewController {    // Обработайте событие нажатия кнопки "Start Live"    @objc func onStartVoiceRoomClicked() {        // 1. Настройте параметры комнаты (RoomParams)        var params = RoomParams()        params.maxSeatCount = 10 // Максимальное количество мест        params.seatMode = .applyToTake // Режим мест        // 2. Создайте экземпляр контроллера голосовой комнаты        let roomId = "test_voice_room_id"        let voiceRoomVC = TUIVoiceRoomViewController(roomId: roomId, behavior: .prepareCreate, roomParams: params)        voiceRoomVC.modalPresentationStyle = .fullScreen        // 3. Представьте страницу голосовой комнаты        present(voiceRoomVC, animated: true)    }}
```

- **Справочник параметров TUIVoiceRoomViewController:**

| **Параметр** | **Тип** | **Описание** |
| --- | --- | --- |
| `roomId` | `string` | Глобально уникальный идентификатор прямой комнаты. |
|  `behavior` | `RoomBehavior` | Поведение входа в комнату:- autoCreate: автоматически создает и входит в прямую комнату.- prepareCreate: сначала переходит на страницу предварительного просмотра перед трансляцией, затем создает и входит в прямую комнату после нажатия пользователем кнопки "Start Live".- join: зритель присоединяется к комнате. |
| `roomParams` | `RoomParams?` | Параметры для хоста, чтобы начать прямую трансляцию. Подробнее см. таблицу ниже. |

- **Справочник параметров RoomParams:**

| **Параметр** | **Тип** | **Описание** |
| --- | --- | --- |
| `maxSeatCount` | `Int` | Максимальное количество мест в прямой комнате. |
| `seatMode` | `TUISeatMode` | Режим мест зрителей:- `applyToTake`: зритель должен подать заявку и получить одобрение от хоста, чтобы занять место.- `freeToTake`: зритель может свободно занять место без одобрения хоста. |

## Настройка пользовательского интерфейса

`TUILiveKit` предоставляет гибкую настройку пользовательского интерфейса для удовлетворения различных бизнес-требований. Вы можете выбрать различные шаблоны макетов и легко обновить текст и значки пользовательского интерфейса.

### Выбор шаблона макета прямой трансляции

Voice Room `TUILiveKit` предлагает **два стиля макета**. Выберите предпочитаемый стиль, используя опцию **"Layout"** на странице предварительного просмотра перед трансляцией хоста:

### Настройка текста (локализация)

`TUILiveKit` использует формат [Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) (`.xcstrings`), представленный в **Xcode 15**, для управления текстом, отображаемым в пользовательском интерфейсе. Вы можете изменить строковые ресурсы, используя графический интерфейс Xcode:

> **Примечание:** [Apple Strings Catalog](https://developer.apple.com/documentation/xcode/localizing-and-varying-text-with-a-string-catalog) (.xcstrings) — это формат локализации, представленный в Xcode 15. Он улучшает управление локализованными строками разработчиками, поддерживая структурированный формат для обработки множественных форм, вариантов для конкретных устройств и т. д. Этот формат становится рекомендуемым способом управления локализацией для приложений iOS и macOS.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cfb30877c06d11f08c0e52540044a08e.png)

TUILiveKit использует `TUILiveKit.xcassets` для управления графическими ресурсами пользовательского интерфейса. Вы можете быстро изменять пользовательские значки, используя графические инструменты Xcode.

## Часто задаваемые вопросы

### Нет звука после начала трансляции?

Перейдите в `Параметры вашего устройства > Приложение > Микрофон` и убедитесь, что разрешения микрофона включены.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2372bb5fc07211f09c26525400bf7822.png)

### Невозможно начать трансляцию при нажатии кнопки запуска с сообщением "Not Logged In"?

См. [Завершить вход](https://www.tencentcloud.com/document/product/647/60334#6ebcab01-c3d7-4fb4-a933-016cf1cf5193), чтобы убедиться, что функция входа правильно интегрирована.

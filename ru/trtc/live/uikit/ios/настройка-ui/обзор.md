# Обзор

## Интеграция отдельного компонента

При интеграции отдельного компонента мы предоставляем серию UI-компонентов для сценариев прямой трансляции, включая основные компоненты аудио и видео, основные компоненты голосового чата, компоненты информации о прямой трансляции, компоненты списка зрителей, компоненты сообщений, компоненты подарков и многое другое. На основе этих UI-компонентов вы можете быстро создавать свою собственную бизнес-логику прямой трансляции, как при сборке конструктора.

## Введение в компонент LiveCoreView

LiveCoreView — это базовый элемент управления, разработанный нами для UIKit видеотрансляции. Этот основной элемент управления предоставляет различные API, включая предпросмотр перед трансляцией, запуск видеотрансляции, остановку видеотрансляции, подключение к комнате прямой трансляции со зрителями и кроссрум-подключение с другими ведущими и т. д. Вам просто нужно вызвать несколько строк API из компонента LiveCoreView, чтобы достичь следующей прямой трансляции.

Подробно об интеграции компонента LiveCoreView см. в документации [LiveCoreView](https://www.tencentcloud.com/document/product/647/69844).

## Введение в компонент SeatGridView

**Основные компоненты управления комнаты голосового чата** (**SeatGridView**) легко позволяют включать или отключать комнату голосового чата и эффективно управлять операциями со слотами в комнате прямой трансляции, включая запрос на говорение, приглашение к говорению, перемещение слотов и удаление ораторов. С помощью **SeatGridView** вы можете быстро интегрировать модуль комнаты голосового чата в ваше приложение, значительно повысить пользовательский опыт и одновременно снизить затраты на разработку.

Подробно об интеграции компонента SeatGridView см. в документации [SeatGridView](https://www.tencentcloud.com/document/product/647/69845).

# Трансляция для ведущих

Это руководство помогает вам быстро интегрировать страницу начала трансляции ведущего и включить функции прямой трансляции хоста. С этой интеграцией хосты могут получить доступ к предпросмотру камеры, фильтрам красоты, звуковым эффектам, переключению камер, подключениям гостей аудитории, совместному размещению, отображению информации о прямой комнате, списку аудитории, анимациям подарков и прямым комментариям.

## Предпросмотр функций

Страница начала трансляции ведущего включает встроенное поведение и стили по умолчанию. Если эти значения по умолчанию полностью не соответствуют вашим требованиям, вы можете настроить пользовательский интерфейс в соответствии с вашими потребностями. Диаграмма ниже выделяет основные функции, доступные на странице начала трансляции ведущего.

## Быстрая интеграция

### Шаг 1: Активация услуги

Обратитесь к документу [Активация услуги](https://www.tencentcloud.com/document/product/647/60033), чтобы включить **пробную версию** или активировать пакет **Pro Edition**.

### Шаг 2: Интеграция кода

Завершите шаги [Подготовки](https://www.tencentcloud.com/document/product/647/77112), чтобы добавить TUILiveKit в ваш проект.

Убедитесь, что ваш проект включает `react-native-safe-area-context`. Если нет, установите его с помощью:

```
yarn add react-native-safe-area-context
```

### Шаг 3: Добавление страницы ведущего

Пакет `react-native-tuikit-live` предоставляет полный пользовательский интерфейс и бизнес-логику для трансляции со стороны хоста. Чтобы интегрировать страницу ведущего:

1. Установите точку входа в вашем приложении для запуска компонента `AnchorPage`.
2. Реализуйте логику навигации для переключения между страницами.

Вот простой пример навигации.

Компонент `AnchorPage` автоматически создаст комнату прямой трансляции, вам нужно только введите следующие параметры обратного вызова:

- `onBack`: Срабатывает при возврате на предыдущую страницу.
- `onEndLive`: Срабатывает при окончании прямой трансляции.

> **Примечание:** Этот пример демонстрирует простое переключение страниц с использованием `useState`. Для производственных приложений рекомендуется использовать библиотеки навигации, такие как React Navigation, для управления страницами. Чтобы понять, как интегрировать библиотеки навигации, обратитесь к [официальной документации React Navigation](https://reactnavigation.org/).

```
/** Simple navigation example - use useState to manage page transitions */ import React, { useState } from 'react';import { AnchorPage } from 'react-native-tuikit-live';import { View, Button, StyleSheet } from 'react-native';import { SafeAreaProvider } from 'react-native-safe-area-context';type PageType = 'home' | 'anchor' | 'liveEnd';function MyApp() {  const [currentPage, setCurrentPage] = useState<PageType>('home');  const [endedLiveID, setEndedLiveID] = useState<string>('');  // start live streaming  const handleStartLive = () => {    setCurrentPage('anchor');  };  // Return from anchor page  const handleBackFromAnchor = () => {    setCurrentPage('home');  };  // live streaming end  const handleEndLive = (liveID?: string) => {    setEndedLiveID(liveID || '');    setCurrentPage('liveEnd');  };  return (    <SafeAreaProvider>      {currentPage === 'home' && (        <View style={styles.container}>          <Button title="Start live streaming" onPress={handleStartLive} />        </View>      )}      {currentPage === 'anchor' && (        <AnchorPage          onBack={handleBackFromAnchor}          onEndLive={handleEndLive}        />      )}      {currentPage === 'liveEnd' && (        <View style={styles.container}>          <Button title="Back to homepage" onPress={() => setCurrentPage('home')} />        </View>      )}    </SafeAreaProvider>  );}const styles = StyleSheet.create({  container: {    flex: 1,    justifyContent: 'center',    alignItems: 'center',  },});export default MyApp;
```

После интеграции вызовите код для вывода AnchorPage.

## Настройка макета пользовательского интерфейса

TUILiveKit позволяет настроить как страницы начала трансляции, так и страницы прямой трансляции в соответствии с требованиями вашего бизнеса.

### Настройка значков

Все значки, используемые TUILiveKit, находятся в каталоге `tuikit-atomic-x/src/static/images`. В таблице ниже приведены некоторые примеры. Вы можете заменить эти значки по мере необходимости.

| **Путь значка** | **Описание** |
| --- | --- |
| /static/images/link-host.png | Значок «Совместное размещение» на нижней панели действий |
| /static/images/link-guest.png | Значок «Гость» на нижней панели действий |
| /static/images/live-more.png | Значок «Ещё» на нижней панели действий |
| /static/images/live-end.png | Значок «Завершить трансляцию» на верхней панели действий |

После обновления значков пересоберите и запустите приложение, чтобы применить изменения.

### Настройка текстов

TUILiveKit управляет всеми текстами пользовательского интерфейса в централизованном файле. Чтобы изменить любой текст, просто отредактируйте соответствующие json-файлы в каталоге `tuikit-atomic-x/src/locales/`.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a87e6a44eaeb11f0a6f452540097cba1.png)

- zh.json: Китайский текст
- en.json: Английский текст

После редактирования пересоберите и запустите приложение, чтобы увидеть обновленный текст пользовательского интерфейса.

## Часто задаваемые вопросы

### Почему экран видео внезапно чернеет, когда хост был в эфире некоторое время?

Проверьте, не вошли ли вы с одинаковым userID на другом устройстве. Убедитесь, что аккаунт, используемый как для аудитории, так и для хоста, не используется в других местах. Если аккаунт вошел на другом устройстве, это может привести к почернению экрана трансляции или просмотра.

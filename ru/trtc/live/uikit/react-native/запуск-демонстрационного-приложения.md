## Предварительные требования

**Активировать сервис**

Перейдите на страницу [Активировать сервис](https://www.tencentcloud.com/document/product/647/60033), чтобы получить пробную версию TUILiveKit. Затем на странице [Управление приложением](https://trtc.io/login?fro=gt&s_url=https%3A%2F%2Fconsole.trtc.io%2Fapp) соберите следующие данные:

- **SDKAppID**: идентификатор вашего приложения (обязательно). TRTC использует SDKAppId для выставления счетов и сбора статистики.
- **SDKSecretKey**: секретный ключ вашего приложения, необходимый для инициализации файлов конфигурации.

**Настройка окружения**

- **Node.js**: версия 20.0.0 или выше (рекомендуется LTS)
- **Yarn**: версия 4.11.0 или выше
- **Android Studio (только для Android)**: требуется для разработки на Android
- **Xcode 14+**: требуется для разработки на iOS (только macOS)
- **CocoaPods (только для iOS)**: CocoaPods установлен. Если нет, см. [Руководство по установке CocoaPods](https://guides.cocoapods.org/using/getting-started.html) или выполните:

`sudo gem install cocoapods` в терминале.

> **Советы:** При запуске `sudo gem install cocoapods` вас могут попросить пароль компьютера. Введите пароль администратора для продолжения.

- **Устройства**: два смартфона с рабочими камерами и микрофонами

## Инструкции

### Загрузить демонстрацию

1. Клонируйте исходный код демонстрации TUILiveKit с GitHub или выполните:

```
git clone https://github.com/Tencent-RTC/TUIKit_ReactNative.git
```

2. Установите зависимости проекта:

```
cd TUIKit_ReactNativeyarn install
```

3. Установите зависимости iOS Pod (пропустите этот шаг, если вы запускаете только на Android):

```
cd TUIKit_ReactNative/application/iospod install
```

### Настроить демонстрацию

1. Установите **SDKAppID** и **SDKSecretKey**: откройте `TUIKit_ReactNative/application/src/debug/UserSigGenerator.ts` и введите **SDKAppID** и **SDKSecretKey**, полученные при активации сервиса:

```
// Your SDKAppIDexport const sdkAppID = 0;// Your SDKSECRETKEYconst SDKSECRETKEY = "";
```

2. Установите подпись Apple Developer (пропустите этот шаг, если вы запускаете только на Android): в вашем проекте перейдите на **TARGETS** > **Signing & Capabilities**, включите **Automatically** **manage** signing и выберите учетную запись Apple Developer и Bundle Identifier:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/591263f2ea2b11f0a75a525400370dda.png)

### Сборка и запуск

Запуск на Android

Запуск на iOS

Чтобы запустить демонстрацию на Android:

```
cd TUIKit_ReactNative/applicationyarn android
```

Чтобы запустить демонстрацию на iOS:

```
cd TUIKit_ReactNative/applicationyarn ios
```

### Попробуйте прямую трансляцию и просмотр

1. **Войдите или зарегистрируйтесь с пользовательским ID.**

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/d998d214ea2b11f0ae695254001d6acc.png)

> **Примечание:** TUILiveKit не поддерживает вход на несколько устройств с одним UserID. Во время командной разработки простые UserID типа «1», «123» или «111» могут быть заняты, что вызывает ошибки входа. Используйте уникальные UserID для тестирования. Чтобы полностью ознакомиться с рабочим процессом прямой трансляции, войдите на двух устройствах с разными пользователями — один в качестве хоста, один в качестве зрителя.

2. **Начните прямую трансляцию в качестве хоста:**

Нажмите центральную кнопку внизу главной страницы, чтобы открыть страницу предпросмотра трансляции. Затем нажмите **Start Live**, чтобы начать трансляцию.

## Часто задаваемые вопросы

### Ошибка сигнатуры или ошибка входа в демонстрацию?

Убедитесь, что SDKAppID и SDKSecretKey, введенные в `TUIKit_ReactNative/application/src/debug/UserSigGenerator.ts`, совпадают с учетными данными на странице [Управление приложением консоли TRTC](https://trtc.io/login?fro=gt&s_url=https%3A%2F%2Fconsole.trtc.io%2Fapp).

### Как проверить проблемы с окружением React Native?

Чтобы проверить окружение React Native, выполните следующую команду:

```
npx react-native doctor
```

# Просмотр трансляции

Это руководство проходит вас через страницу зрителя в демонстрационном приложении TUILiveKit React, показывая, как интегрировать её в ваш проект и настроить стили, функции и макет.

## Отображение функций

Страница зрителя по умолчанию включает следующие функции: **отображение информации о прямой трансляции, видеопроигрыватель, панель подарков, список присутствующих зрителей, интерактивный чат и элементы управления воспроизведением**.

- **Воспроизведение видео**: высококачественная трансляция с адаптивным переключением разрешения.
- **Интерактивный чат**: обмен сообщениями в реальном времени с поддержкой текста и эмодзи.
- **Панель подарков**: отправка виртуальных подарков хосту.
- **Элементы управления воспроизведением**: пауза/возобновление, переключение разрешения, регулировка громкости, режим "картинка в картинке" и полноэкранный режим.

## Часто задаваемые вопросы

### Почему автоматическое воспроизведение не работает в браузерах?

Современные браузеры ограничивают автоматическое воспроизведение для улучшения пользовательского опыта. Медиа со звуком могут воспроизводиться только после взаимодействия с пользователем (например, нажатием или касанием). Это предотвращает воспроизведение звука веб-сайтами без явного согласия пользователя. Большинство браузеров позволяют автоматически воспроизводить беззвучное видео, но iOS Safari в режиме низкого энергопотребления и iOS WKWebView с пользовательскими ограничениями (например, встроенный браузер WeChat) также блокируют автоматическое воспроизведение беззвучного видео.

- **Поведение при отказе автоматического воспроизведения**

Когда индекс взаимодействия с медиа (MEI) сайта не достиг требуемого порога, попытка автоматического воспроизведения аудио не удастся. По умолчанию SDK отображает диалоговое окно, предлагающее взаимодействие пользователя (например, нажатие **OK**). После взаимодействия политика браузера соблюдается, и воспроизведение возобновляется нормально.

- **Решение: пользовательская обработка отказа автоматического воспроизведения**

Если вы хотите настроить метод взаимодействия при отказе автоматического воспроизведения (например, заменить пользовательский интерфейс диалога по умолчанию), вы можете это сделать, слушая обратный вызов `onAutoplayFailed`, выдаваемый SDK. Ниже приведен фрагмент кода из [Быстрая интеграция > Шаг 3](#шаг-3-интеграция-страницы-просмотра), который слушает событие и отображает пользовательское диалоговое окно для запроса пользователя.

```
import React, { useEffect, useCallback } from 'react';import TUIRoomEngine, { TUIRoomEvents } from "@tencentcloud/tuiroom-engine-js";import { useLiveListState, useRoomEngine } from 'tuikit-atomicx-react';import { MessageBox } from '@tencentcloud/uikit-base-component-react';import styles from './LivePlayerView.module.scss';const LivePlayer: React.FC<LivePlayerProps> = ({ className }) => {  const roomEngine = useRoomEngine();  const { currentLive, leaveLive, subscribeEvent, unsubscribeEvent } = useLiveListState();  // Обработчик события отказа автоматического воспроизведения  const handleAutoPlayFailed = useCallback(() => {    MessageBox.alert({      content: 'Content is ready, click the [Play] button to start playback',      confirmText: 'Play',      showClose: false,      modal: false,    });  }, []);  // Настройка слушателей событий  useEffect(() => {    // Слушайте событие отказа автоматического воспроизведения. Браузеры отключают воспроизведение звука по умолчанию. Когда автоматическое воспроизведение не удается, добавьте взаимодействие с пользовательским интерфейсом для запуска воспроизведения звука    if (roomEngine.instance) {      roomEngine.instance.on(TUIRoomEvents.onAutoPlayFailed, handleAutoPlayFailed);    } else {      TUIRoomEngine.once("ready", () => {        roomEngine.instance?.on(TUIRoomEvents.onAutoPlayFailed, handleAutoPlayFailed);      });    }    // ... остальной код опущен    return () => {      roomEngine.instance?.off(TUIRoomEvents.onAutoPlayFailed, handleAutoPlayFailed);      // ... остальной код опущен    };  }, [handleAutoPlayFailed, handleLiveEnded, handleKickedOutOfLive, roomEngine.instance, subscribeEvent, unsubscribeEvent]);  //... остальной код опущен}
```

# Запуск демо-версии

Это руководство поможет вам запустить полнофункциональную демо-версию TUILiveKit для прямой трансляции на вашем локальном компьютере менее чем за 10 минут.

После завершения настройки вы можете попробовать её в качестве зрителя: смотреть HD-трансляции, участвовать в чате с комментариями и отправлять подарки.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9f48d3d3115711f18bc5525400074c32.png)

## Предварительные требования

### Включение сервиса

Обратитесь к документу [Включение сервиса](https://www.tencentcloud.com/document/product/647/60033), чтобы получить пробную версию TUILiveKit. Затем перейдите в [Консоль](https://console.trtc.io/app) для управления приложениями и получите следующие данные:

- `SDKAppID`: идентификатор приложения (обязателен). TRTC использует `SDKAppId` для биллинга и деталей.
- `SDKSecretKey`: секретный ключ приложения, используется для инициализации конфигурационного файла с секретной информацией.

### Подготовка окружения

- **Современный браузер**: браузер, поддерживающий [WebRTC API](https://web.sdk.qcloud.com/trtc/webrtc/demo/detect/index.html).
- **Устройства**: камера, микрофон и динамики.
- **Node.js**: ≥ 18.19.1 (рекомендуется официальная версия LTS, проверьте вашу текущую версию командой `node -v`).

## Инструкции

### Получение демо

1. Скачайте исходный код [TUILiveKit React Demo](https://github.com/Tencent-RTC/TUIKit_React) с GitHub или выполните следующую команду в `командной строке`:

```
git clone https://github.com/Tencent-RTC/TUIKit_React.git
```

2. Перейдите в директорию демо-проекта и установите зависимости:

```
cd TUIKit_React/demos/live-vite-reactnpm install
```

### Конфигурация демо

В корневой директории проекта откройте файл `src/config/basic-info-config.js` и введите `SDKAppID` и `SDKSecretKey`, полученные при активации сервиса:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5d7c361312bb11f1a751525400380f7d.png)

### Сборка и запуск демо

```
npm run dev
```

После успешного запуска проекта браузер автоматически откроется и перейдёт на страницу списка трансляций. На этом этапе вы можете увидеть пустой список трансляций, так как в вашем приложении ещё никто не транслирует.

### Трансляция и просмотр

#### Начало трансляции

Текущий React-проект поддерживает только просмотр. Чтобы попробовать, сначала начните трансляцию, а затем откройте запущенное вами React-демо и смотрите трансляцию там.

- **Вариант 1** (рекомендуется)

Используйте наш [Веб-сайт для трансляции](https://web.sdk.qcloud.com/trtc/livekit/pusher/index.html) для начала трансляции.

- **Вариант 2**

Начните трансляцию из другого демо-проекта платформы — например, запустите проект Web Vue3 [Трансляция и просмотр](https://www.tencentcloud.com/document/product/647/66940).

#### Просмотр трансляции с использованием React Demo

После запуска трансляции обновите вкладку браузера, открытую командой `npm run dev` (обычно http://localhost:5173). Начатая вами трансляция появится в списке — нажмите на неё, чтобы открыть страницу просмотра.

> **Примечание:** Для плавного опыта используйте разные ID пользователей для трансляции и просмотра. Вход с одним и тем же ID в обоих местах может вызвать конфликты и исключить вас, и вы не сможете попробовать полный процесс.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f4e14ffa12bd11f1a92352540097cba1.png)

# Компонент Live Gift

В этом руководстве показано, как интегрировать компонент **LiveGift** из **TUILiveKit** в веб-приложение React для настольных браузеров.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/5d82e4a111f711f18b07525400ecee81.png)

## Основные возможности

| **Функция** | **Описание** |
| --- | --- |
| Список подарков | Отображает настроенный список подарков. Нажмите «Подробнее» для просмотра полного каталога. |
| Отправка подарков | Выберите подарок из списка и нажмите для отправки. Отправленные подарки появляются в виде сообщений в чате. |
| Воспроизведение подарков | Поддерживает форматы SVG 2D, SVG 3D и видео MP4. Трёхмерные подарки могут воспроизводиться в полноэкранном режиме. |

## Этапы интеграции

### Шаг 1: Предварительные требования

Перед быстрой интеграцией необходимо обратиться к разделу [Подготовка](https://www.tencentcloud.com/document/product/647/77813), чтобы выполнить соответствующую конфигурацию окружения и активировать необходимые сервисы.

> **Примечание:** Для использования системы подарков активируйте издание **Бесплатный пробный период** или **Pro**. Количество настраиваемых подарков зависит от выбранного пакета. Подробнее см. раздел **Система подарков** в документе [Описание функций и тарификация](https://www.tencentcloud.com/document/product/647/59407#658e2423-30d2-45e2-91b8-128b2730b072) и выберите пакет, соответствующий вашим требованиям.

Импортируйте и используйте **компонент live gift** в своем проекте. Вы можете скопировать приведённый ниже пример кода в свой проект, чтобы реализовать **список подарков, отправку подарков и воспроизведение трёхмерных подарков**.

GiftPanel.module.scss

## Дальнейшие действия

После завершения указанной выше интеграции пользовательского интерфейса вы реализовали возможность отображения подарков на стороне клиента. Для создания полнофункциональной коммерческой системы подарков необходимо также обратиться к документу [Интеграция серверной части системы подарков и расширенные функции](https://www.tencentcloud.com/document/product/647/69849) и завершить интеграцию следующей основной бизнес-логики:

- **Настраиваемая конфигурация подарков**: Загрузите пользовательские значки подарков и файлы анимации через API на стороне сервера и установите цены.
- **Обратный вызов при вычитании подарков**: Настройте URL обратного вызова для пересылки запросов подарков в ваш биллинг-бэкенд для проверки баланса и вычитания средств.
- **Интеграция PK Score**: В сценариях PK хостов преобразуйте стоимость подарков в PK баллы в реальном времени.
- **Аналитика**: Получайте доступ к записям отправки подарков, общей стоимости и другим операционным данным.
- **Обновление Animation SDK**: Если SVGA не соответствует вашим требованиям, интегрируйте продвинутый плеер, поддерживающий MP4/PAG и другие высокочастотные прозрачные анимации.

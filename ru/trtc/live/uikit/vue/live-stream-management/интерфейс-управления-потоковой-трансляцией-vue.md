# Интерфейс управления потоковой трансляцией (Vue)

## Обзор

Этот документ содержит подробное введение в **Страницу мониторинга** в TUILiveKit Demo. Вы можете напрямую интегрировать разработанную нами страницу просмотра в ваш существующий проект, как указано в этом документе, или настроить стиль веб-страницы, макет и элементы функций в соответствии с вашими потребностями.

## Обзор функций

| **Категория функций** | **Конкретные возможности** |
| --- | --- |
| **Мониторинг комнат с множественными потоками** | Поддерживает одновременное отображение 8+ потоков трансляции (фактическое количество настраивается). |
| **Воспроизведение с низкой задержкой** | Получайте поток трансляции в реальном времени с задержкой ≤3s. |
| **Независимое управление звуком** | Включайте/отключайте звук для любой комнаты потоковой трансляции отдельно, чтобы избежать помех. |
| **Просмотр деталей одним щелчком** | Щелкните на любом окне комнаты потоковой трансляции, чтобы быстро перейти на страницу деталей и просмотреть информацию об автор контента и ключевые детали. |
| **Одним щелчком закрыть комнату потоковой трансляции, нарушающую правила** | Работайте напрямую на странице деталей или панели мониторинга, чтобы быстро закрыть комнаты потоковой трансляции, нарушающие правила, и повысить эффективность обработки. |

## **Демонстрация функций**

Страница мониторинга предоставляет поведение и стиль по умолчанию. Однако, если поведение и стиль по умолчанию не полностью соответствуют вашим требованиям, вы также можете настроить пользовательский интерфейс. Он в основном включает **мониторинг комнат с множественными потоками на одном экране, воспроизведение с низкой задержкой, независимое управление звуком, одного щелчка для просмотра деталей и одного щелчка для закрытия комнат потоковой трансляции, нарушающих правила.**

### Шаг 1: Настройка окружения и активация сервиса

Перед выполнением быстрой интеграции вам необходимо обратиться к [подготовке](https://www.tencentcloud.com/document/product/647/73735) для выполнения конфигурации окружения и активации соответствующего сервиса.

```
git clone https://github.com/Tencent-RTC/TUIKit_Vue3.git
```

## Индивидуальная настройка

Вы успешно запустили Demo страницы мониторинга. Если вам требуется настройка пользовательского интерфейса, вы можете обратиться к следующим методам. Пользовательское содержимое включает, но не ограничивается `цветовой темой, шрифтом, скругленными углами, кнопками, значками, полями ввода, всплывающими окнами` в исходном коде, которые можно **добавлять, удалять или изменять**. Ниже приведены примеры настройки цветовой темы, кнопок и значков. Вы можете обратиться к подходу реализации и применить его к другим компонентам, чтобы удовлетворить ваши потребности в настройке пользовательского интерфейса.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/bd2809a1ae4611f096c2525400454e06.png)

### Цветовая тема

Например, на [Шаге 5](#step5) вы можете использовать пример кода для манипулирования значением темы для переключения цветовых тем.

```
<UIKitProvider theme="dark">  // Когда входное значение темы - dark, общая цветовая тема интерфейса черная  xxx                         // Когда входное значение темы - light, общая цветовая тема интерфейса белая-theme</UIKitProvider>
```

### Кнопка

Если вам нужно добавить или заменить кнопки для настройки пользовательского интерфейса, это можно получить следующим образом, используя в качестве примера три кнопки для операций со страницей в списке трансляций.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/a34a1e85ae2f11f096c2525400454e06.png)

Обратитесь к изображению выше, чтобы найти исходный код соответствующей кнопки в указанной позиции, и выполните операции настройки пользовательского интерфейса, такие как **добавление, удаление или замена** текущей кнопки.

### Значки

Если вам нужно выполнить настройку пользовательского интерфейса, такую как добавление или замена значков, вы можете реализовать это следующим образом, используя в качестве примера значок в списке трансляций. Следуйте приведенному ниже руководству, чтобы найти соответствующую позицию значка и выполнить операции настройки пользовательского интерфейса, такие как добавление, удаление или замена.

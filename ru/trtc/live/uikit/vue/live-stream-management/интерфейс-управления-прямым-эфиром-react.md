# Интерфейс управления прямым эфиром (React)

## Описание

Этот документ содержит подробное введение в **Мониторинговую веб-страницу** в TUILiveKit Demo. Вы можете интегрировать нашу предварительно разработанную страницу просмотра в свой существующий проект в соответствии с этим документом или настроить стиль веб-страницы, макет и элементы функций в соответствии с вашими потребностями.

## Обзор функций

| **Категория функций** | **Конкретные возможности** |
| --- | --- |
| **Мониторинг мультипотоковой комнаты прямого эфира** | Поддержка одновременного отображения 8+ прямых трансляций (фактическое количество настраивается). |
| **Воспроизведение с низкой задержкой** | Получение прямого потока в реальном времени с задержкой ≤3s. |
| **Независимое управление звуком** | Включение/отключение звука для любой комнаты прямой трансляции отдельно, чтобы избежать помех. |
| **Просмотр сведений одним щелчком** | Щелкните на окно любой комнаты прямой трансляции, чтобы быстро перейти на страницу сведений и просмотреть ключевую информацию, такую ​​как информация об авторе. |
| **Отключение комнат прямого эфира, нарушающих правила, одним щелчком** | Работайте непосредственно на странице сведений или панели мониторинга, чтобы быстро отключить комнаты прямого эфира, нарушающие правила, и повысить эффективность обработки. |

## **Демонстрация функций**

Страница мониторинга предоставляет поведение и стиль по умолчанию. Однако если поведение и стиль по умолчанию полностью не соответствуют вашим требованиям, вы также можете настроить пользовательский интерфейс. Это в основном включает **одновременный мониторинг мультипотоковых комнат прямого эфира, воспроизведение с низкой задержкой, независимое управление звуком, просмотр сведений одним щелчком и отключение комнат прямого эфира, нарушающих правила, одним щелчком.**

Перед выполнением быстрой интеграции необходимо обратиться к [подготовке](https://www.tencentcloud.com/document/product/647/73735), чтобы выполнить требуемую конфигурацию окружения и активировать соответствующий сервис.


## Таможенное оформление

Вы успешно запустили демонстрацию страницы мониторинга. Если вам необходимо настроить пользовательский интерфейс, см. следующие методы реализации. Пользовательское содержимое включает, но не ограничивается `цветовыми темами, шрифтами, скругленными углами, кнопками, значками, полями ввода, всплывающими окнами` в исходном коде, и вы можете **добавлять, удалять или изменять** их. Ниже приведены примеры настройки цветовых тем, кнопок и значков. Вы можете обратиться к этим подходам реализации и применить их к другим компонентам, чтобы удовлетворить ваши потребности в настройке пользовательского интерфейса.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/cfb63626ae4611f09710525400e889b2.png)

### Цветовая тема

Как показано в примере кода шага 5, вы можете использовать значение темы для переключения цветовых тем.

```
<UIKitProvider theme="dark">  // When theme is set to dark, the interface adopts a black color theme overall  xxx                         // When theme is set to light, the interface adopts a white-theme overall</UIKitProvider>
```

### Кнопка

Если вам необходимо выполнить настройку пользовательского интерфейса, такую ​​как добавление или замена кнопок, вы можете реализовать это следующим образом, используя в качестве примера три кнопки на странице списка прямого эфира. Обратитесь к рисунку ниже, вы можете найти исходный код в указанной позиции соответствующей кнопки и выполнить операции настройки пользовательского интерфейса, такие как **добавление, удаление, замена** в текущей части кнопок.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/15975d86ae2f11f09b75525400bf7822.png)

Если вам необходимо выполнить настройку пользовательского интерфейса, такую ​​как добавление или замена значков, вы можете реализовать это следующим образом, используя в качестве примера значок в списке прямого эфира. Следуйте приведенным ниже рекомендациям, чтобы найти соответствующее расположение значка и выполнить операции настройки пользовательского интерфейса, такие как добавление, удаление или замена.

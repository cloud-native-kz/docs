# Компонент Live Gift

В этом руководстве описано, как интегрировать компонент **LiveGift** из **TUILiveKit** в вашу веб-приложение React для настольных браузеров.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/662e7f9b11f711f18bc5525400074c32.png)

## Основные возможности

| **Функция** | **Описание** |
| --- | --- |
| Список подарков | Отображает сконфигурированный список подарков. Нажмите "Подробнее" для просмотра полного каталога. |
| Отправка подарков | Выберите подарок из списка и нажмите для отправки. Отправленные подарки появляются в виде сообщений в чате. |
| Воспроизведение подарков | Поддерживает форматы SVG 2D, SVG 3D и видео MP4. 3D-подарки могут воспроизводиться в полноэкранном режиме. |

## Шаги интеграции

### Шаг 1: Предварительные требования

Перед интеграцией компонента **LiveGift** необходимо обратиться к [Подготовке](https://www.tencentcloud.com/document/product/647/66938) для завершения интеграции TUILiveKit.

> **Примечание:** Для использования системы подарков активируйте либо **бесплатный пробный период**, либо издание **Pro**. Количество настраиваемых подарков зависит от выбранного пакета. Для получения подробной информации см. раздел **Gift System** в [описании функций и выставлении счетов](https://www.tencentcloud.com/document/product/647/59407#658e2423-30d2-45e2-91b8-128b2730b072) и выберите подходящий для вас пакет.

### Шаг 2: Установка зависимостей

npm

pnpm

yarn

```
npm install tuikit-atomicx-vue3 @tencentcloud/uikit-base-component-vue3 --save
```

```
pnpm add tuikit-atomicx-vue3 @tencentcloud/uikit-base-component-vue3
```

```
yarn add tuikit-atomicx-vue3 @tencentcloud/uikit-base-component-vue3
```

### Шаг 3: Интеграция компонента Live Gift

Импортируйте и используйте **компонент LiveGift** в своем проекте. Приведенный ниже пример демонстрирует, как реализовать функции **список подарков**, **отправка подарков** и **воспроизведение 3D-подарков**.

```
<template>    <div class="gift-panel">        <LiveGift />    </div></template><script setup lang="ts">import { LiveGift } from 'tuikit-atomicx-vue3';</script><style lang="scss" scoped>.gift-panel{  padding: 6px 0;  border-top: 1px solid var(--stroke-color-primary);  background-color: var(--bg-color-operate);  display: flex;  &.disabled {    pointer-events: none;    opacity: 0.5;    cursor: not-allowed;  }}</style>
```

## Следующие шаги

После завершения интеграции пользовательского интерфейса выше вы реализовали функциональность отображения подарков на клиентской стороне. Для создания полнофункциональной коммерческой системы подарков обратитесь к руководству [интеграция бэкенда системы подарков и расширенные функции](https://www.tencentcloud.com/document/product/647/69849) для реализации следующей основной бизнес-логики:

- **Пользовательская конфигурация подарков**: загрузите пользовательские значки подарков и файлы анимации через API на стороне сервера и установите цены.
- **Обратный вызов списания подарков**: настройте URL обратного вызова для перенаправления запросов подарков на ваш биллинговый бэкенд для проверки баланса и списания.
- **Интеграция PK Score**: в сценариях PK хостов конвертируйте значения подарков в PK Score в режиме реального времени.
- **Аналитика**: получите доступ к записям отправки подарков, общей стоимости и другим операционным данным.
- **Обновление SDK анимации**: если SVGA не соответствует вашим потребностям, интегрируйте расширенный плеер, поддерживающий MP4/PAG и другие высокочеткие прозрачные анимации.

# Запуск примера демонстрации

- Этот документ проведет вас через весь процесс работы с `TUILiveKit` Demo. Выполнив эти шаги, вы можете настроить и запустить Demo менее чем за 10 минут и испытать полнофункциональное приложение для онлайн-трансляции с полноценным интерфейсом.
- Вы также можете настроить стиль страницы, макет и функции в соответствии с вашими требованиями, следуя инструкциям по настройке в каждом разделе.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1a1ed63db3d411f0b96752540044a08e.png)

## Быстрый старт

Вы можете напрямую испытать все функции **Live**, используя следующие методы.

## Предварительные требования

### Активировать сервис

Обратитесь к разделу [Активировать сервис](https://www.tencentcloud.com/document/product/647/60033), чтобы получить пробную лицензию TUILiveKit. Затем получите следующую информацию из [Консоли](https://console.trtc.io/app) для управления приложением:

- `SDKAppID`: Идентификатор приложения (обязательно). Tencent Cloud использует `SDKAppId` для выставления счетов и деталей.
- `SDKSecretKey`: Секретный ключ приложения, используется для инициализации учетных данных конфигурационного файла.

### Подготовка окружения

- **Node.js**: ≥ 18.19.1 (рекомендуется официальная LTS версия)
- **Vue**: ≥ 3.4.21
- **Современный браузер**: Браузер, поддерживающий [WebRTC APIs](https://web.sdk.qcloud.com/trtc/webrtc/demo/detect/index.html)
- **Устройства**: Камера, микрофон и динамики

## Инструкции

### Получить Demo

1. Скачайте [TUILiveKit Demo](https://github.com/Tencent-RTC/TUILiveKit) с GitHub или выполните следующую команду в командной строке:

```
git clone https://github.com/Tencent-RTC/TUILiveKit.git
```

2. Установите зависимости

```
cd TUILiveKit/Web/web-vite-vue3npm install
```

### Настройка

Откройте файл `TUILiveKit/Web/web-vite-vue3/src/config/basic-info-config.js` в Demo и введите `SDKAppID` и `SDKSecretKey`, которые вы получили при активации сервиса:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/fa056245b3d011f0995e525400454e06.png)

### Сборка и запуск Demo

```
npm run dev
```

### Испытание основных функций

#### Потоковая трансляция хостом

- Чтобы начать трансляцию в качестве хоста, перейдите на страницу трансляции, установив URL на `http://localhost:5173/#/live-pusher`. Эта страница поддерживает как **ландшафтный**, так и **портретный** режимы трансляции.
- Используйте область добавления материалов слева для добавления источников медиа и нажмите кнопку **Connect with Audience** (Подключиться к аудитории) для принятия или отклонения запросов на подключение зрителей.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/f9efe903b3d011f0bd1d5254001c06ec.png)

#### Просмотр аудиторией

При первом запуске проекта введите `UserId` для регистрации вашей личности. При последующих запусках страница списка прямых трансляций отображается сразу, как показано ниже.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/65ca0d17b3d211f0bb7b525400bf7822.png)

Нажмите на обложку прямой трансляции, чтобы войти в комнату. В комнате трансляции вы можете **смотреть поток, следить за хостом, отправлять комментарии-пули, просматривать список аудитории** и многое другое.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/850e9549b3d211f09e195254007c27c5.png)

> **Примечание:** Не используйте простые UserID, такие как "1", "123" или "111". TRTC не поддерживает многоконечный вход для одного и того же UserID, и эти простые UserID, вероятно, используются другими во время совместной разработки, что может привести к ошибкам входа. Используйте отличительные UserID во время отладки. Несколько членов аудитории могут одновременно **отправлять комментарии-пули, следить за хостом, запрашивать подключение** и другое, чтобы смоделировать реальные сценарии взаимодействия нескольких пользователей.

### Руководство по настройке

Вы можете изменить предоставленный исходный код пользовательского интерфейса для настройки интерфейса `TUILiveKit` Demo по мере необходимости. Выполните следующие команды в каталоге `TUILiveKit/Web/web-vite-vue3`:

macOS

Windows

```
mkdir -p ./src/TUILiveKit && rsync -av --exclude={'node_modules','package.json','excluded-list.txt'} ./node_modules/@tencentcloud/livekit-web-vue3/src/* ./src/TUILiveKit
```

```
xcopy .\\node_modules\\@tencentcloud\\livekit-web-vue3\\src\\* .\\src\\TUILiveKit /i /e /exclude:.\\node_modules\\@tencentcloud\\livekit-web-vue3\\excluded-list.txt
```

> **Примечание:** После запуска скрипта экспорта:Исходный код `livekit-web-vue3` будет сгенерирован в указанном каталоге src. Затем вы должны вручную обновить проект `Web/web-vite-vue3`. Измените все ссылки на компоненты `livekit-web-vue3` с пути пакета npm на относительный путь локального исходного кода. Для ссылки на исходный код `tuikit-atomicx-vue3`, пожалуйста, посетите его [репозиторий GitHub](https://github.com/Tencent-RTC/TUIKit_Vue3/tree/main/packages/tuikit-atomicx-vue3).

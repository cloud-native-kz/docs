# Установка боевого счета

## Описание функции

Администратор приложения может использовать этот API для установки баллов комнат в Battle.

## Описание вызова API

### Пример URL запроса:

```
https://xxxxxx/v4/live_engine_http_pk/set_battle_score?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Описание параметров запроса

В таблице ниже перечислены только параметры, задействованные при изменении при вызове этого API, и их описания.

| Параметры | Описание |
| --- | --- |
| xxxxxx | Эксклюзивное доменное имя, соответствующее стране/региону SDKAppID: Китай: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_pk/set_battle_score | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Для получения дополнительной информации см. [App Admin](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подписи, созданные аккаунтом администратора приложения. Для конкретных операций см. [Generate UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Введите случайное 32-битное целое число без знака, диапазон значений 0 - 4294967295 |
| contenttype | Формат запроса имеет фиксированное значение `json` |

### Максимальная частота вызовов

200 раз/сек.

### Пример пакета запроса

**Базовая форма**

```
{              "BattleId":"HuYM036AMk2JJeSUgnv5H0KDJpc+HXfqTAtGuYWzRPs=",              "OpType":1,              "ScoreList":[                    {                        "RoomId":"pk-live1",                        "Score":100                    },                    {                        "RoomId":"pk-live2",                        "Score":100                    }              ]}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| BattleId | String | Обязательное | ID боевого сеанса |
| OpType | String | Обязательное | Тип установки: 1 для накопления, 2 для перезаписи |
| ScoreList | Array | Обязательное | Список установки баллов |
| RoomId | String | Обязательное | ID комнаты |
| Score | Integer | Обязательное | Балл, соответствующий комнате |

### Пример тела пакета ответа

**Базовая форма**

```
{    "ErrorCode": 0,    "ErrorInfo": "",    "ActionStatus": "OK",    "RequestId": "Id-e7625183568d4058bd48c9924d4b1ed7-O-Seq-125382",    "Response": {        "Result": [            {                "RoomId": "pk-3",                "ResultCode": 0            },            {                "RoomId": "pk-4",                "ResultCode": 0            }        ]    }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ErrorCode | Integer | Код ошибки, 0 указывает на успех, ненулевое значение указывает на ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| ActionStatus | String | Результат обработки запроса, OK указывает на успех, FAIL указывает на ошибку. |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом, необходим для устранения неполадок. |
| Result | Array | Результат операции |
| ResultCode | Integer | Результат, 0 указывает на успешную установку, 1 указывает на несуществующую комнату. |
| RoomId | String | ID комнаты |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), HTTP-код возврата для этого API всегда равен 200. Фактический код ошибки и сообщение об ошибке находятся в теле пакета ответа. Представлены `ErrorCode` и `ErrorInfo`.

Распространённые коды ошибок (60000–79999) см. в документе [Error Code](https://www.tencentcloud.com/document/product/647/60027#).

Приватные коды ошибок этого API следующие:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Параметр недействителен. Проверьте, что запрос правильный согласно описанию ошибки. |
| 100006 | Текущий PK ещё не начался или уже закончился, установка счета снова не допускается. |

# Импорт нескольких учетных записей

## Обзор функции

Этот API используется для пакетного импорта принадлежащих приложению учетных записей в систему Chat с созданием соответствующего внутреннего ID для учетной записи, что позволяет ей использовать услуги Chat.

> **Примечание:** Этот API поддерживает импорт до 100 учетных записей за раз. Повторный импорт одной и той же учетной записи создаст только один внутренний ID. При пакетном импорте учетных записей, если учетная запись уже существует, она будет обновлена с новыми предоставленными параметрами (прозвище пользователя, URL аватара и т. д.).

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/im_open_login_svc/multiaccount_import?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице описаны измененные параметры при вызове этого API. Для других параметров см. [Обзор RESTful API](https://intl.cloud.tencent.com/document/product/1047/34620).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, где находится ваш SDKAppID. Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` Сеул: `adminapikr.im.qcloud.com` Токио: `adminapijpn.im.qcloud.com` Франкфурт: `adminapiger.im.qcloud.com` Кремниевая долина: `adminapiusa.im.qcloud.com` |
| v4/im_open_login_svc/multiaccount_import | API запроса, который используется. |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Учетная запись администратора приложения. Для получения дополнительной информации см. раздел **App Admin** в [Аутентификации входа](https://intl.cloud.tencent.com/document/product/1047/33517). |
| usersig | Подпись, созданная учетной записью администратора приложения. Подробные сведения см. в разделе [Генерация UserSig](https://intl.cloud.tencent.com/document/product/1047/34385). |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295. |
| contenttype | Формат запроса, который должен быть всегда `json`. |

### Максимальная частота вызовов

100 раз/сек

### Пример пакета запроса для старой версии

```
{  "Accounts": [    "UserID_001",    "UserID_002",    "UserID_003"  ]}
```

### Поля пакета запроса для старой версии

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| Accounts | Array | Обязательное | Список имен пользователей, длина каждого имени пользователя не превышает 32 байта, за раз можно импортировать до 100 имен пользователей |

### Пример пакета запроса для новой версии

```
{  "AccountList": [    {      "UserID": "UserID_001",      "Nick": "Nick_001",      "FaceUrl": "http://www.qq.com"    },    {      "UserID": "UserID_002",      "Nick": "Nick_002",      "FaceUrl": "http://www.qq.com"    },    {      "UserID": "UserID_002",      "Nick": "Nick_001",      "FaceUrl": "http://www.qq.com"    }  ]}
```

### Поля пакета запроса для новой версии

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| AccountList | Array | Обязательное | Список импортируемых пользователей, за раз можно импортировать до 100 пользователей |
| UserID | String | Обязательное | Имя пользователя, строка длиной до 32 байт |
| Nick | String | Опциональное | Прозвище пользователя |
| FaceUrl | String | Опциональное | URL фотографии профиля пользователя |

### Пример пакета ответа

```
{    "ActionStatus": "OK",    "ErrorCode": 0,    "ErrorInfo": "",    "FailAccounts": [        "UserID_002",        "UserID_003"    ]}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат запроса. `OK`: успешно. `FAIL`: неудачно |
| ErrorCode | Integer | Код ошибки. `0`: успешно. Другие значения: неудачно |
| ErrorInfo | String | Сообщение об ошибке |
| FailAccounts | Array | Список неудачных учетных записей |

## Описание кодов ошибок

Если не возникает ошибка сети (например, ошибка 502), код состояния HTTP для этого API всегда равен 200. Фактический код ошибки и сообщение об ошибке указаны полями ErrorCode и ErrorInfo в теле ответа.

Общие коды ошибок (60000–79999) можно найти в документе [Код ошибки](https://www.tencentcloud.com/document/product/1047/34348).

Приватные коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 40006 | Внутренняя ошибка сервера при установке данных, повторите попытку позже. |
| 40601 | Длина Value в поле данных превышает 500 байт. |
| 70169 | Внутреннее истечение времени на сервере, повторите попытку позже |
| 70202 | Внутреннее истечение времени на сервере, повторите попытку позже |
| 70398 | Количество учетных записей превышает лимит. Для создания более 100 учетных записей обновите приложение на профессиональную версию. Для получения конкретного руководства см. [Руководство по покупке](https://www.tencentcloud.com/document/product/1047/36021). |
| 70399 | Для приложений, не являющихся пробной версией IM, повторный импорт запрещен в течение трех месяцев после удаления учетной записи. |
| 70402 | Неверные параметры. Пожалуйста, проверьте, заполнены ли обязательные поля или соответствуют ли поля требованиям протокола. |
| 70403 | Запрос не выполнен. Требуются разрешения администратора приложения. |
| 70500 | Внутренняя ошибка сервера, повторите попытку позже. |

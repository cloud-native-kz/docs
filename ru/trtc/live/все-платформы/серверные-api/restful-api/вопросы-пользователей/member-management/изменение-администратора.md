# Изменение администратора

## Обзор функции

Администратор приложения может назначать и отзывать администраторов комнат через этот интерфейс.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/modify_admin?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, которые изменяются при вызове этого интерфейса. Для получения более подробной информации о других параметрах обратитесь к [обзору RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Прочие регионы Китая: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/modify_admin | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунт администратора приложения. Для получения дополнительной информации обратитесь к [App Admin](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная аккаунтом администратора приложения. Для получения подробных инструкций обратитесь к [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса — фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакетов запроса

**Основная форма**

```
{    "RoomId": "test",     "Admin_Account": [            "user1",            "user2"            ],    "OpType":"add"}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |
| Admin_Account | Array | Обязательно | Измененный список администраторов, может поддерживать до 5 администраторов |
| OpType | String | Обязательно | Тип операции: add означает добавление администратора, delete означает удаление администратора |

### Пример пакетов ответа

**Основная форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса, возвращаемый с каждым запросом. Требуется предоставить этот RequestId при поиске проблем |

## Коды ошибок

Если не произойдет сетевая ошибка (например, ошибка 502), код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указаны в полезной нагрузке ответа через ErrorCode и ErrorInfo. Общие коды ошибок (60000–79999) можно найти в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027#). Приватные коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Неверный параметр. Проверьте запрос на правильность на основе описания ошибки |
| 100004 | Комната не существует, возможно, она никогда не была создана или уже была растворена |
| 100020 | Количество администраторов комнаты превышает лимит. Максимальный лимит — 5 |

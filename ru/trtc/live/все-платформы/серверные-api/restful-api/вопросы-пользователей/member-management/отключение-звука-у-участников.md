# Отключение звука у участников

## Описание функции

Администраторы приложения могут использовать этот API для отключения и включения звука у участников.

## Описание вызова API

### Пример URL запроса:

```
https://xxxxxx/v4/live_engine_http_srv/mute_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В таблице ниже указаны только параметры, изменяемые при вызове этого интерфейса. Для получения дополнительной информации о других параметрах обратитесь к [RESTful API](https://www.tencentcloud.com/document/product/647/63398#).

| Параметры | Описание |
| --- | --- |
| xxxxxx | Доменное имя, соответствующее стране/региону, в котором находится ваш SDKAppID: Китай: `console.tim.qq.com` Сингапур: `adminapisgp.im.qcloud.com` США: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/mute_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Дополнительную информацию см. в разделе [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная учетной записью администратора приложения. Для получения конкретных шагов обратитесь к [Генерация UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакета запроса

**Базовая форма**

```
{     "RoomId": "test",      "MemberList_Account": [            "user1",            "user2"            ],    "MuteTime": 3600  // В секундах, 0 означает включение звука}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательный | ID комнаты |
| MemberList_Account | Array | Обязательный | Список участников с отключенным звуком |
| MuteTime | integer | Обязательный | Длительность отключения звука, 0 означает включение звука |

### Пример пакета ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720",}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается при каждом запросе и требуется предоставить этот RequestId при поиске проблем |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например, ошибка 502), код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указаны в полезной нагрузке ответа параметрами ErrorCode и ErrorInfo. Распространенные коды ошибок (60000-79999) можно найти в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027#). Закрытые коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Недопустимый параметр, проверьте правильность запроса на основе описания ошибки |
| 100004 | Комната не существует, возможно, она никогда не была создана или уже была закрыта |

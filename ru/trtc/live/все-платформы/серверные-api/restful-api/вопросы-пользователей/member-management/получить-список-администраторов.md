# Получить список администраторов

## Обзор функции

Администратор приложения может получить список администраторов комнаты через этот интерфейс.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/get_admin_list?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

Таблица ниже содержит только параметры, изменяемые при вызове этого интерфейса. Для получения дополнительной информации о других параметрах, пожалуйста, обратитесь к [Обзору RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Прочие в Китае: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединённые Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/get_admin_list | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должен быть аккаунтом администратора приложения. Для получения дополнительной информации, пожалуйста, обратитесь к [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, созданная аккаунтом администратора приложения. Для получения подробных инструкций обратитесь к [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-битное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Примеры пакетов запросов

**Основная форма**

```
{    "RoomId": "test" }
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательно | ID комнаты |

### Примеры пакетов ответов

**Основная форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720",  "Response": { //Response Body        "Admin_Account": [            "user2",            "user1"        ]  }}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный ID запроса возвращается с каждым запросом и требуется указывать этот RequestId при поиске проблем |
| Admin_Account | Array | Список администраторов |

## Коды ошибок

Если не возникает сетевая ошибка (например, ошибка 502), код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указываются в полезной нагрузке ответа параметрами ErrorCode и ErrorInfo. Распространённые коды ошибок (60000–79999) можно найти в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027#). Коды ошибок, специфичные для этого API:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, пожалуйста, повторите попытку |
| 100002 | Неверный параметр, пожалуйста, проверьте корректность запроса на основе описания ошибки |
| 100004 | Комната не существует, возможно, потому что она никогда не была создана или уже была закрыта |

# Разбан членов комнаты

## Обзор функции

Администратор приложения может использовать этот интерфейс для разблокировки членов.

## Описание вызова API

### Пример URL запроса

```
https://xxxxxx/v4/live_engine_http_srv/unban_member?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

Таблица ниже содержит только параметры, измененные при вызове этого интерфейса. Для получения более подробной информации о других параметрах см. [Обзор RESTful API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| xxxxxx | Зарезервированный домен для страны/региона, где находится SDKAppID: Прочие в Китае: `console.tim.qq.com`Сингапур: `adminapisgp.im.qcloud.com`Соединенные Штаты: `adminapiusa.im.qcloud.com` |
| v4/live_engine_http_srv/unban_member | API запроса |
| sdkappid | SDKAppID, назначенный консолью Chat при создании приложения |
| identifier | Должна быть учетная запись администратора приложения. Для получения более подробной информации см. [Администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-admin) |
| usersig | Подпись, сгенерированная учетной записью администратора приложения. Для получения подробной информации о процедуре см. [Генерирование UserSig](https://www.tencentcloud.com/document/product/647/69883#) |
| random | Случайное 32-разрядное целое число без знака в диапазоне от 0 до 4294967295 |
| contenttype | Формат запроса, фиксированное значение: `json` |

### Максимальная частота вызовов

200 запросов/сек.

### Пример пакетов запроса

**Базовая форма**

```
{    "RoomId": "test",     "MemberList_Account": [            "user1",            "user2"            ]}
```

### Описание полей пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | Идентификатор комнаты |
| MemberList_Account | Array | Обязательное | Список разбанённых членов: до 20 одновременно. |

### Пример пакетов ответа

**Базовая форма**

```
{  "ActionStatus": "OK",  "ErrorInfo": "",  "ErrorCode": 0,  "RequestId": "Id-8c9858f01e954611ae2d4c1b1ed7d583-O-Seq-52720"}
```

### Описание полей пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат процесса запроса: OK указывает на успех; FAIL указывает на ошибку |
| ErrorCode | Integer | Код ошибки. 0: успех; другие значения: ошибка |
| ErrorInfo | String | Сообщение об ошибке |
| RequestId | String | Уникальный идентификатор запроса возвращается при каждом запросе и требуется предоставить этот RequestId при локализации проблем |

## Коды ошибок

При отсутствии сетевой ошибки (например, ошибки 502) код возврата HTTP для этого интерфейса всегда будет 200. Фактический код ошибки и информация об ошибке указаны в полезной нагрузке ответа полями ErrorCode и ErrorInfo. Распространённые коды ошибок (60000–79999) можно найти в документе [Код ошибки](https://www.tencentcloud.com/document/product/647/60027#). Приватные коды ошибок для этого API приведены ниже:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку |
| 100002 | Неверный параметр. Проверьте правильность запроса на основе описания ошибки |
| 100004 | Комната не существует. Возможно, она никогда не была создана или уже была распущена |

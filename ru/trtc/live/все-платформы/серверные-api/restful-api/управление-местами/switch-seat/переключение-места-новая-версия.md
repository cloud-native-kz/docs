# Переключение места (Новая версия)

## Описание функции

Администратор приложения может использовать этот API для перемещения определённого пользователя с микрофоном на другое место. Автоматическое разблокирование срабатывает при отключении звука, а также поддерживаются принудительное перемещение, обмен местами и другие операции.

> **Примечание:** Этот API предназначен для новой версии функции переключения микрофона. Для более ранней версии см. [Переключение места (Старая версия)](https://www.tencentcloud.com/document/product/647/72552). Ранняя версия поддерживает только перемещение на пустое место с микрофоном.

## Описание вызова API

### Пример URL запроса

```
https://console.tim.qq.com/v4/room_engine_http_mic/move_user_to_seat?sdkappid=88888888&identifier=admin&usersig=xxx&random=99999999&contenttype=json
```

### Параметры запроса

В следующей таблице приведены только параметры, задействованные при вызове этого API, и их описания. Для получения дополнительной информации о параметрах см. [Введение в REST API](https://www.tencentcloud.com/document/product/647/63398).

| Параметр | Описание |
| --- | --- |
| v4/room_engine_http_mic/move_user_to_seat | API запроса |
| sdkappid | SDKAppID, назначенный консолью IM при создании приложения. |
| identifier | Должна быть учётная запись администратора приложения. Дополнительные сведения см. в разделе [администратор приложения](https://www.tencentcloud.com/document/product/647/69882#app-.E7.AE.A1.E7.90.86.E5.91.98). |
| usersig | Подпись, созданная учётной записью администратора приложения. Конкретные операции см. в разделе [генерация UserSig](https://www.tencentcloud.com/document/product/647/69883). |
| random | Введите случайное 32-битное целое число без знака в диапазоне [0, 4294967295]. |
| contenttype | Формат запроса зафиксирован как `json`. |

### Максимальная частота вызовов

200 раз в секунду.

### Пример пакета запроса

**Основная форма**

```
{  "RoomId":"room-test",  "SeatIndex": 1,  "Member_Account": "user",  "Policy": 0 // Specify the seat move mode when occupied, 0 cancel, 1 force move and kick this user, 2 exchange seats}
```

### Поля пакета запроса

| Поле | Тип | Атрибут | Описание |
| --- | --- | --- | --- |
| RoomId | String | Обязательное | ID комнаты |
| SeatIndex | Integer | Обязательное | ID места. |
| Member_Account | String | Обязательное | ID пользователя. |
| Policy | Integer | Обязательное | Режим перемещения места. Когда место занято: 0 отмена, 1 принудительное перемещение и отключение этого пользователя, 2 обмен местами. |

### Пример полезной нагрузки ответа

**Основная форма**

```
{  "ErrorCode": 0,  "ErrorInfo": "",  "ActionStatus": "OK",  "RequestId": "Id-7c1680be52734bdc8d6de398ab9505e7-O-Seq-57717"}
```

### Поля пакета ответа

| Поле | Тип | Описание |
| --- | --- | --- |
| ActionStatus | String | Результат обработки запроса. OK: обработка успешна; FAIL: обработка не выполнена. |
| ErrorCode | Integer | Код ошибки: 0 означает успех, ненулевое значение означает ошибку. |
| ErrorInfo | String | Сообщение об ошибке. |
| RequestId | String | Уникальный ID запроса, возвращаемый для каждого запроса. Для устранения неполадок запроса следует предоставить его RequestId. |

## Описание кодов ошибок

Если не возникает сетевая ошибка (например, 502), HTTP код возврата этого API равен 200. Фактический код ошибки и информация об ошибке указаны в полях ErrorCode и ErrorInfo в полезной нагрузке ответа.

Распространённые коды ошибок (60000–79999) см. в документе [Коды ошибок](https://www.tencentcloud.com/document/product/647/60027).

Приватные коды ошибок этого API:

| Код ошибки | Описание |
| --- | --- |
| 100001 | Внутренняя ошибка сервера, повторите попытку. |
| 100002 | Неправильные параметры запроса. Проверьте, является ли запрос правильным в соответствии с описанием ошибки. |
| 100004 | Комната не существует или она могла существовать, но была закрыта. |
| 100005 | Пользователь с микрофоном не является членом комнаты |
| 100200 | Место заблокировано. Попробуйте другое место. |
| 100203 | Уже на микрофоне |
| 100206 | Пользователь не на микрофоне |
| 100210 | Пользователь на месте |

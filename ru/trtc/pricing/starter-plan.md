# План Starter

Чтобы помочь вам минимизировать первоначальные затраты на проект, мы предлагаем План Starter, разработанный специально для новых пользователей. Этот план позволяет вам получить доступ к официальным пакетам подписки различных наших продуктов по значительно сниженной цене ***в первый месяц***.

## Описание

### Кто может купить

План Starter доступен новым клиентам, которые ранее не приобретали официальные пакеты продуктов.

> **Условия и положения акции:** Эта акция доступна только клиентам, которые приобретают услуги Tencent Cloud (включая trtc.io) непосредственно у Tencent Cloud. Клиенты, приобретающие через посредников, не имеют права на участие.

### Период действия

Пакет действует с дня покупки до того же дня следующего месяца. Например, если пакет был куплен 1 марта 2024 г., период его действия будет с 1 марта 2024 г. по 1 апреля 2024 г.

### Как использовать

1. **RTC Engine, Call, Conference, Live, Chat**

Пакет вступает в силу немедленно после покупки и функции пакета напрямую привязываются к приложению, идентифицируемому по `SDKAppID`. Это зависит от того, какое приложение вы выбрали во время покупки. Если у вас есть несколько приложений, которым требуется использовать одну и ту же функцию продукта, вам необходимо приобрести несколько пакетов.

![Выберите SDKAppID ](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/9d404cb7ab3811f0a68e5254001c06ec.png)

Например, если ваше приложение 123456 приобрело **Conference**, и ваше другое приложение 222222 также хочет использовать функции Conference, вам потребуется приобрести еще один пакет **Conference** специально для приложения 222222.

2. **Beauty AR**

Вы можете приобрести План Starter для Beauty AR и получить лицензию. Каждый домен может быть привязан к лицензии Plan Starter только один раз. Период действия начинается со времени покупки, поэтому используйте его как можно скорее. Подробные инструкции по созданию и привязке лицензии см. в [руководстве пользователя Beauty AR](https://trtc.io/document/60213?platform=web&product=beautyar&menulabel=web).

### Ограничения на покупку и условия приемлемости

- Каждая учетная запись ограничена покупкой План Starter один раз за продукт. Например, если вы уже приобрели План Starter для Chat, вы все еще можете приобрести План Starter для Conference, но не можете снова приобрести План Starter для Chat.
- План Starter предназначен исключительно для новых пользователей продукта. Если вы ранее приобретали какой-либо официальный пакет для продукта, вы не имеете права на скидку План Starter для этого продукта.

### Правила продления и обновления

- **Автоматическое продление:** Только План Starter для Chat и Beauty AR поддерживают автоматическое продление по исходной цене. Например, если при покупке План Starter для Chat включено автоматическое продление, то в следующем месяце он автоматически обновится как Chat Pro с платой 699 USD.
- **Обновление в течение периода действия:**
  - **Правила обновления:** В течение периода действия План Starter для Call, Conference, Live и RTC Engine могут быть обновлены на любой официальный пакет. Например, План Starter для Live может быть обновлен на Live Lite, Live Standard или Live Pro в течение периода его действия. Обратите внимание, что после истечения срока действия План Starter больше не может быть обновлен, и вам потребуется приобрести любой официальный пакет.
  - **Расчет цены обновления:** Цена обновления рассчитывается по формуле:

**(Розничная цена обновленного плана - Розничная цена предыдущего плана) × Оставшиеся дни действия / Общее количество дней действия.**

Примечание: Розничная цена План Starter для RTC Engine составляет 49,5 USD/месяц. Розничная цена План Starter для Call, Conference и Live составляет 95 USD/месяц.

  - **Доступные минуты в обновленном плане:** Доступные минуты в обновленном плане рассчитываются по формуле:

**(Доступные минуты обновленного плана - Доступные минуты предыдущего плана) × Оставшиеся дни действия / Общее количество дней действия.**

**Обратите внимание,** План Starter для Chat и Beauty AR не поддерживают обновления в течение периода действия. Вы можете выбрать включение автоматического продления при покупке или вручную продлить ту же версию в течение периода действия.

### Поддержка оплаты по факту использования

- В течение периода действия пакета привилегии оплаты по факту использования автоматически активируются для вашей учетной записи. Любое использование, превышающее пакет, будет оплачиваться на основе ежедневного постоплаченного счета, обеспечивая непрерывность обслуживания. Для режима выставления счетов по факту использования ознакомьтесь [здесь](https://trtc.io/document/42734).
- После истечения срока действия пакета, если у вас нет действительных пакетов, привилегии оплаты по факту использования будут деактивированы. Для продолжения использования услуги мы рекомендуем подписаться на наш официальный ежемесячный пакет с автоматическим продлением.

## Часто задаваемые вопросы

### **Вопрос 1:** Почему мой заказ не прошел?

**Ответ:** Есть две возможные причины. Одна заключается в том, что вы можете не соответствовать требованиям для покупки акции, например не быть новым пользователем или уже приобретавшим План Starter для этого продукта. Другая причина — у вас может быть ожидающий платеж. Пожалуйста, проверьте [Консоль](https://console.tencentcloud.com/expense/deal) на предмет ожидающих платежей. Если платеж не завершится в течение 15 минут, заказ будет автоматически отменен, и вы можете разместить новый заказ.

### **Вопрос 2:** Что произойдет, когда я исчерпаю длительность пакета?

**Ответ:** Если длительность в пакете исчерпана и пакет еще не истек, превышенная длительность будет рассчитана и выставлена в соответствии с [ценами оплаты по факту использования](https://trtc.io/document/42734?product=pricing).

### Вопрос 3: Я уже приобрел План Starter для Conference. Могу ли я все еще купить План Starter для других продуктов?

**Ответ:** Да, если вы не приобретали официальный пакет для соответствующего продукта, вы можете повторно покупать План Starter для разных продуктов. Обратите внимание, что План Starter для каждого продукта можно приобрести только один раз.

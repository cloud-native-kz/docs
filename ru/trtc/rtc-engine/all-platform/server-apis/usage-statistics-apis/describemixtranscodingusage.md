# DescribeMixTranscodingUsage

## 1. Описание API

Доменное имя для запроса API: trtc.intl.tencentcloudapi.com.

Этот API используется для запроса информации об использовании сервиса On-Cloud MixTranscoding в TRTC.

Если запрашиваемый период составляет один день или меньше, возвращаемая статистика рассчитывается на основе пятиминутных интервалов. Если запрашиваемый период превышает один день, возвращаемая статистика рассчитывается на основе суточных интервалов.
Период, запрашиваемый в одном запросе, не может превышать 31 дня.
При запросе статистики текущего дня возвращаемые данные могут быть неточными из-за задержки в сборе данных.
Вы можете использовать этот API для запроса исторических данных об использовании или для выверки данных, но мы не рекомендуем использовать его для критически важной бизнес-логики.
Лимит частоты запросов для этого API составляет пять вызовов в секунду.

Максимум 5 запросов можно инициировать в секунду для этого API.

## 2. Входные параметры

Приведённый ниже список параметров запроса содержит только параметры запроса API и некоторые общие параметры. Полный список общих параметров см. в разделе [Общие параметры запроса](https://www.tencentcloud.com/document/api/647/34263).

| Название параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: DescribeMixTranscodingUsage. |
| Version | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: 2019-07-22. |
| Region | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Дополнительные сведения см. в разделе [список регионов](https://www.tencentcloud.com/document/api/647/34263#region-list), поддерживаемых продуктом. Этот API поддерживает только: ap-guangzhou, ap-singapore. |
| StartTime | Да | String | Дата начала в формате YYYY-MM-DD. |
| EndTime | Да | String | Дата окончания в формате YYYY-MM-DD. Период, запрашиваемый в одном запросе, не может превышать 31 дня. |
| SdkAppId | Нет | Integer | `SDKAppID` приложения TRTC, к которому принадлежит целевая комната. Если вы не указываете этот параметр, будет возвращена статистика использования всех приложений TRTC под текущей учётной записью. |

## 3. Выходные параметры

| Название параметра | Тип | Описание |
| --- | --- | --- |
| UsageKey | Array of String | Тип использования. Каждый элемент этого параметра соответствует элементу `UsageValue` в порядке их перечисления. |
| UsageList | Array of [TrtcUsage](https://www.tencentcloud.com/document/api/647/36760#TrtcUsage) | Данные об использовании в каждой единице времени. |
| RequestId | String | Уникальный идентификатор запроса, генерируемый сервером, который будет возвращен для каждого запроса (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId требуется для определения проблемы. |

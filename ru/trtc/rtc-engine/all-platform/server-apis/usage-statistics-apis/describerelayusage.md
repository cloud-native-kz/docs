# DescribeRelayUsage

## 1. Описание API

Доменное имя для запроса API: trtc.intl.tencentcloudapi.com.

Этот API используется для запроса использования вами услуги трансляции TRTC в CDN.

Если запрашиваемый период составляет один день или меньше, возвращаемая статистика основана на пятиминутных интервалах. Если запрашиваемый период больше одного дня, возвращаемая статистика основана на суточных интервалах.
Запрашиваемый период в одном запросе не может превышать 31 день.
Если вы запрашиваете статистику текущего дня, возвращаемая статистика может быть неточной из-за задержки в сборе данных.
Вы можете использовать этот API для запроса исторических данных об использовании или для согласования данных, но мы не рекомендуем использовать его для критически важной бизнес-логики.
Ограничение частоты вызовов для этого API составляет пять вызовов в секунду.

Максимум 5 запросов можно инициировать в секунду для этого API.

Мы рекомендуем вам использовать API Explorer

## 2. Входные параметры

В следующем списке параметров запроса приводятся только параметры запроса API и некоторые общие параметры. Полный список общих параметров см. в [Общих параметрах запроса](https://www.tencentcloud.com/document/api/647/34263).

| Имя параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: DescribeRelayUsage. |
| Version | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: 2019-07-22. |
| Region | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Дополнительную информацию см. в [списке регионов](https://www.tencentcloud.com/document/api/647/34263#region-list), поддерживаемых продуктом. Этот API поддерживает только: ap-guangzhou, ap-singapore. |
| StartTime | Да | String | Дата начала в формате YYYY-MM-DD. |
| EndTime | Да | String | Дата окончания в формате YYYY-MM-DD. Запрашиваемый период в одном запросе не может превышать 31 день. |
| SdkAppId | Нет | Integer | `SDKAppID` приложения TRTC, к которому принадлежит целевая комната. Если вы не укажете этот параметр, будет возвращена статистика использования всех приложений TRTC в рамках текущего аккаунта. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| UsageKey | Array of String | Тип использования. Каждый элемент этого параметра соответствует элементу `UsageValue` в порядке их перечисления. |
| UsageList | Array of [TrtcUsage](https://www.tencentcloud.com/document/api/647/36760#TrtcUsage) | Данные об использовании в каждой единице времени. |
| RequestId | String | Уникальный ID запроса, генерируемый сервером, будет возвращен для каждого запроса (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId требуется для определения проблемы. |

## 4. Пример

### Пример 1. Запрос использования приложения, `SDKAppID` которого равен `1400123456`, с 1 января 2022 г. по 2 января 2022 г.

#### Пример входных данных

```
POST / HTTP/1.1
Host: trtc.intl.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DescribeRelayUsage
<Common request parameters>

{
    "EndTime": "2022-01-01",
    "StartTime": "2022-01-02",
    "SdkAppId": 1400123456
}
```

#### Пример выходных данных

```json
{
    "Response": {
        "UsageKey": [
            "Bandwidth"
        ],
        "UsageList": [
            {
                "TimeKey": "2022-01-01 00:00:00",
                "UsageValue": [
                    10.11
                ]
            },
            {
                "TimeKey": "2022-01-02 00:00:00",
                "UsageValue": [
                    10.11
                ]
            }
        ],
        "RequestId": "xx"
    }
}
```

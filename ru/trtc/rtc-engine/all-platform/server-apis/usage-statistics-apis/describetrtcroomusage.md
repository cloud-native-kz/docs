# DescribeTrtcRoomUsage

## 1. Описание API

Доменное имя для запроса API: trtc.intl.tencentcloudapi.com.

Этот API используется для запроса данных об использовании, сгруппированных по комнатам.

Запрашиваемый период не может превышать 24 часа. Если период охватывает два разных дня, возвращаемые данные могут быть неточными из-за задержки при сборе данных. Вы можете сделать несколько вызовов для запроса использования в разные дни.
Вы можете использовать этот API для запроса своего исторического использования или для сверки данных, но мы не рекомендуем использовать его для критичной бизнес-логики.
Ограничение частоты запросов этого API составляет один вызов каждые 15 секунд.

Мы рекомендуем использовать API Explorer

Попробовать

API Explorer предоставляет различные возможности, включая онлайн-вызовы, аутентификацию подписи, генерацию кода SDK и быстрый поиск API. Это позволяет просматривать запрос, ответ и автоматически сгенерированные примеры.

## 2. Входные параметры

В следующем списке параметров запроса указаны только параметры запроса API и некоторые общие параметры. Полный список общих параметров см. в разделе [Общие параметры запроса](https://www.tencentcloud.com/document/api/647/34263).

| Имя параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: DescribeTrtcRoomUsage. |
| Version | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение, используемое для этого API: 2019-07-22. |
| Region | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Дополнительные сведения см. в [списке регионов](https://www.tencentcloud.com/document/api/647/34263#region-list), поддерживаемых продуктом. Этот API поддерживает только: ap-guangzhou, ap-singapore. |
| SdkAppid | Да | Integer | `SDKAppID` комнаты. |
| StartTime | Да | String | Время начала в формате `YYYY-MM-DD HH:MM` (точность до минуты). |
| EndTime | Да | String | Время окончания в формате `YYYY-MM-DD HH:MM`. Время начала и окончания не может отличаться более чем на 24 часа. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| Data | String | Данные об использовании, сгруппированные по комнатам, в формате CSV. |
| RequestId | String | Уникальный идентификатор запроса, генерируемый сервером и возвращаемый для каждого запроса (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId необходим для определения проблемы. |

## 4. Пример

### Пример 1. Запрос данных об использовании, сгруппированных по комнатам

Этот пример показывает, как запросить данные об использовании, сгруппированные по комнатам.

#### Пример входных данных

```
POST / HTTP/1.1
Host: trtc.intl.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DescribeTrtcRoomUsage
<Common request parameters>

{
    "StartTime": "2023-01-06 00:00",
    "EndTime": "2023-01-06 10:00",
    "SdkAppid": 1400017192
}
```

#### Пример выходных данных

```json
{
    "Response": {
        "RequestId": "68fccf1c-c2c7-466a-9f14-1db4bed6054b",
        "Data": "RoomId,Audio,SD,hd,FullHD,2K,4K
test_room,1,2,3,4,5,6
"
    }
}
```

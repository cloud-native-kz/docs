# DescribeTrtcUsage

## 1. Описание API

Доменное имя для запроса API: trtc.intl.tencentcloudapi.com.

Этот API используется для запроса продолжительности аудио/видео TRTC.

Если период запроса составляет один день или меньше, возвращаемая статистика рассчитывается на пятиминутной основе. Если период запроса превышает один день, возвращаемая статистика рассчитывается на суточной основе.
Период запроса на один запрос не может превышать 31 день.
Если вы запрашиваете статистику текущего дня, возвращаемая статистика может быть неточной из-за задержки сбора данных.
Этот API можно использовать для запроса исторического использования или согласования данных, но мы не рекомендуем использовать его для критически важной бизнес-логики.
Ограничение частоты запросов этого API составляет пять вызовов в секунду.

Максимум 5 запросов можно инициировать в секунду для этого API.

Рекомендуем использовать API Explorer

## 2. Входные параметры

Следующий список параметров запроса содержит только параметры запроса API и некоторые общие параметры. Полный список общих параметров см. в [Общих параметрах запроса](https://www.tencentcloud.com/document/api/647/34263).

| Имя параметра | Обязательный | Тип | Описание |
| --- | --- | --- | --- |
| Action | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение для этого API: DescribeTrtcUsage. |
| Version | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Значение для этого API: 2019-07-22. |
| Region | Да | String | [Общие параметры](https://www.tencentcloud.com/document/api/647/34263). Дополнительную информацию см. в [списке регионов](https://www.tencentcloud.com/document/api/647/34263#region-list), поддерживаемых продуктом. Этот API поддерживает только: ap-guangzhou, ap-singapore. |
| StartTime | Да | String | Дата начала в формате YYYY-MM-DD. |
| EndTime | Да | String | Дата окончания в формате YYYY-MM-DD. Период запроса на один запрос не может превышать 31 день. |
| SdkAppId | Нет | Integer | `SDKAppID` приложения TRTC, которому принадлежит целевая комната. Если вы не указываете этот параметр, будет возвращена статистика использования всех приложений TRTC в текущей учетной записи. |

## 3. Выходные параметры

| Имя параметра | Тип | Описание |
| --- | --- | --- |
| UsageKey | Array of String | Тип использования. Каждый элемент этого параметра соответствует элементу `UsageValue` в порядке их перечисления. |
| UsageList | Array of [TrtcUsage](https://www.tencentcloud.com/document/api/647/36760#TrtcUsage) | Данные об использовании в каждой единице времени. |
| RequestId | String | Уникальный ID запроса, созданный сервером, будет возвращен для каждого запроса (если запрос не достигнет сервера по другим причинам, запрос не получит RequestId). RequestId необходим для определения проблемы. |

## 4. Пример

### Пример 1. Запрос использования приложения с `SDKAppID` `1400123456` с 1 января 2022 г. по 2 января 2022 г.

#### Пример входных данных

```
POST / HTTP/1.1
Host: trtc.intl.tencentcloudapi.com
Content-Type: application/json
X-TC-Action: DescribeTrtcUsage
<Common request parameters>

{
    "EndTime": "2022-01-01",
    "StartTime": "2022-01-02",
    "SdkAppId": 1400123456
}
```

#### Пример выходных данных


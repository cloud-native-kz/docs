# Активировать сервис

В этой статье рассказывается, как [Активировать сервис Conversational AI](https://www.tencentcloud.com/document/product/647/69002#2d869be6-ae67-4c31-a16e-d0be3bbc7234) и [Продлить сервис Conversational AI](https://www.tencentcloud.com/document/product/647/69002#b5336e6c-7c68-4d85-9837-7746d3094414). Вы можете ознакомиться с необходимыми сервисами и расходами на основе [Тарификации сервисов Conversational AI](https://trtc.io/document/67833?product=pricing) и следовать руководству для покупки и продления официального сервиса Conversational AI.

## Активировать сервис Conversational AI

Сервис Conversational AI обеспечивает естественное и плавное взаимодействие в реальном времени между человеком и моделями больших языков. Для использования этой функции вам необходимо приобрести [RTC-Engine Monthly Packages](https://trtc.io/document/56025) (версия Starter plan/Light/Standard/Pro). Общие расходы состоят из трёх частей: плата за аудиовызовы, плата за сервис Conversational AI и плата за сервис преобразования речи в текст. Для получения подробной информации о тарификации см. [Тарификация сервисов Conversational AI.](https://trtc.io/document/67833?product=pricing) В этом разделе объясняется, как приобрести пакет RTC-Engine в месячный подписку, чтобы включить плату за сервис Conversational AI и сервис STT для приложения. Для платежей за аудиовызовы нет процесса активации; они будут автоматически включены с постоплатой при использовании.

1. Войдите в [консоль Tencent RTC](https://console.trtc.io/) и нажмите Create Application.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/e08c7c07085b11f09f385254001c06ec.png)

2. Во всплывающем окне введите имя приложения, выберите RTC Engine и выберите подходящий регион развёртывания. Затем нажмите Create. Это создаст приложение RTC engine, привязанное к пробной версии RTC Engine.

> **Примечание:** Регион развёртывания по умолчанию для данных сервиса видеоконференций в реальном времени находится в Сингапуре, а хранилище для данных сервиса мгновенных сообщений находится в выбранном вами центре данных.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/00a0a916211511f0b0b1525400e889b2.png)

3. Нажмите на Conversational AI, Start Configuring.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/3471407d211511f0a62e525400454e06.png)

4. Нажмите на [Package Subscription](https://console.trtc.io/subscription) в консоли Tencent RTC, перейдите на вкладку пакета подписки RTC Engine Monthly Packages и выберите подходящий пакет подписки.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/1bfa7aeb085c11f09cbb525400454e06.png)

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/218a059a085c11f0890f52540099c741.png)

5. Выберите соответствующее имя приложения и завершите платёж.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/2b797788085c11f0bc9a5254005ef0f7.png)

## Продлить сервис Conversational AI

Обратитесь к разделу «Покупка официального издания RTC Engine» в этой главе и приобретите заново пакет в месячную подписку того же издания, чтобы завершить продление.

Рекомендуется продлевать RTC Engine путём включения автоматического продления. При достаточном балансе счёта сервис будет автоматически продлеваться ежемесячно по истечении срока действия. Вы можете включить автоматическое продление при покупке или включить его в консоли после завершения покупки.

> **Примечание:** Пожалуйста, выберите SDKAppID приложения, которое вы хотите продлить.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/38cd4b78085c11f080b5525400bf7822.png)

При необходимости вы можете выбрать отправку тикета для обновления версии пакета.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/428fc2d9085c11f0a94152540044a08e.png)

## Автоматическое продление

Пошаговые инструкции по включению автоматического продления в консоли приведены ниже:

1. Откройте [консоль Tencent RTC > Application](https://console.trtc.io/app), найдите приложение и нажмите Manage, чтобы перейти на страницу с информацией о приложении.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/53358852085c11f0a3ce525400e889b2.png)

2. В информации о продукте RTC Engine нажмите Enable auto-renewal, появится окно подтверждения, нажмите Enable.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/629a6651085c11f0a3ce525400e889b2.png)

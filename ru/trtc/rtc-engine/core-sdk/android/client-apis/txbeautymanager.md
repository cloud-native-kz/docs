## setFilter

**setFilter**

| void setFilter | (Bitmap image) |
| --- | --- |

**Устанавливает цветовой фильтр.**

Цветовой фильтр — это изображение таблицы поиска цветов, содержащее отношения отображения цветов. Вы можете найти несколько предопределённых изображений фильтров в официальной демонстрации, которую мы предоставляем.

SDK выполняет вторичную обработку исходного видеоизображения, захватываемого камерой, в соответствии с отношениями отображения в таблице поиска для достижения ожидаемого эффекта фильтра.

| Param | DESC |
| --- | --- |
| image | Таблица поиска цветов, содержащая отношения отображения цветов. Изображение должно быть в формате PNG. |

## setFilterStrength

**setFilterStrength**

| void setFilterStrength | (float strength) |
| --- | --- |

**Устанавливает интенсивность цветового фильтра.**

Чем больше это значение, тем более очевидным становится эффект цветового фильтра и тем больше различие в цвете между видеоизображением, обработанным фильтром, и исходным видеоизображением.

Интенсивность по умолчанию — 0.5, и если её недостаточно, она может быть увеличена до значения выше 0.5. Максимальное значение — 1.

| Param | DESC |
| --- | --- |
| strength | Диапазон значений: [0, 1]. Чем больше значение, тем более очевидным становится эффект. Значение по умолчанию: 0.5 |

| Param | DESC |
| --- | --- |
| path | Путь видеофайла в формате MP4. Пустое значение указывает на отключение эффекта. |

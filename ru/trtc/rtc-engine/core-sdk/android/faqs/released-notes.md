# Заметки о выпуске

На этой странице представлена история версий SDK TRTC и All-In-One SDK. Пожалуйста, выберите подходящий SDK в соответствии с вашими функциональными потребностями.

#### TRTC SDK 13.1 Выпущен 29 января 2026 г.

#### Новые функции

- Все платформы: Поддержка возможностей AI для транскрипции и перевода в реальном времени.
- Android и iOS: Захват с камеры теперь поддерживает возможность стабилизации видео.
- Mac: Интерфейс пользовательской отправки C++ теперь поддерживает тип данных CVPixelBuffer.

#### Исправление ошибок

- Mac: Исправлена проблема общего доступа к экрану, когда совместное использование WPS с открытым PDF-файлом вызывало мерцание черного экрана.
- Mac: Исправлена проблема сбоя режима динамика при общем доступе на вторичном экране.
- Android: Исправлена проблема совместимости аппаратного декодирования для некоторых устройств с чипсетом MediaTek с малым разрешением.

#### Корректировка поведения интерфейса

- Все платформы: Добавлен интерфейс AITranscriberManager для AI транскрипции и перевода в реальном времени

#### TRTC SDK 13.0 Выпущен 18 декабря 2025 г.

#### Новые функции

- iOS и Mac: Переход на динамическую библиотеку для iOS/Mac SDK.
- OHOS: Поддержка возможности микширования системного звука.
- ITXLocalMediaTranscoding поддерживает системы macOS.

#### Улучшения

- Все платформы: Оптимизация модели защиты речи от AI шума для решения проблем потери начальных символов и символов в окружении с низким соотношением сигнал-шум.
- Все платформы: Выходная ширина и высота кодирования должны быть выровнены по 2 байтам по умолчанию.

#### Исправление ошибок

- Android: Проблема с беззвучным воспроизведением в фоне в течение длительного времени на моделях Xiaomi.
- Android: Исправлена проблема, при которой Bluetooth гарнитура становилась непригодной для использования после отключения функции вызова в системных параметрах.
- Mac: Исправлен сбой общего доступа к экрану в режиме Presenter.
- iOS: Исправлена случайная аномалия эффекта красоты при переключении между передним и задним планом с включенной предварительной обработкой.

#### TRTC SDK 12.5 Выпущен 12 мая 2025 г.

#### Улучшения

- Полная платформа: Сокращена частое создание больших блоков памяти и снижена частота сбоев на 50% в сценариях с низкой памятью.
- Android: Оптимизировано качество стереозвука.
- iOS: Улучшена скорость распознавания звуковой карты.

#### Исправление ошибок

- Android: Исправлены проблемы черного экрана, вызванные настройкой разрешения общего доступа к экрану между горизонтальной и вертикальной ориентацией на некоторых устройствах.
- Android: Исправлено аномальное отображение локального предварительного просмотра при установке горизонтальной ориентации через Activity.
- Windows: Исправлены проблемы черного экрана, вызванные сбоем преобразования формата пикселей D3D11.

#### Корректировка поведения интерфейса

1. Корректировка поведения интерфейса setLocalRenderMode.
- VIDEO_MIRROR_MODE_AUTO (0) (Режим автоматического отзеркаливания): Если используется передняя камера, отзеркаливание включено по умолчанию для локального и удаленного представлений. Если используется задняя камера, отзеркаливание отключено для локального и удаленного представлений (применяется только для мобильных устройств).
- VIDEO_MIRROR_MODE_ENABLED (1) (Принудительное включение отзеркаливания): Отзеркаливание принудительно включено для локального и удаленного представлений независимо от того, используется передняя или задняя камера.
- VIDEO_MIRROR_MODE_DISABLED (2) (Принудительное отключение отзеркаливания): Отзеркаливание принудительно отключено для локального и удаленного представлений независимо от того, используется передняя или задняя камера.
2. SetVideoEncoderMirror больше не рекомендуется использовать. При вызове это будет влиять только на удаленный видеопоток и больше не будет влиять на локальный предварительный просмотр.
3. В сценариях пользовательского захвата видео и общего доступа к экрану: интерфейсы setVideoEncoderMirror и setVideoEncoderRotation больше не будут применяться.

#### Исправление ошибок

- Android: Исправлена проблема, при которой некоторые устройства переключались в режим динамика при изменении наушников при активном микрофоне.

1. Корректировка логики сопоставления формата пользовательской предварительной обработки.
- Формат TRTCVideoPixelFormat_Texture_2D внутренне отображается на RGBA32.
- TRTCVideoBufferType_Texture поддерживает только комбинацию с TRTCVideoPixelFormat_Texture_2D.
2. iOS: Обновление системой активированных причин паузы/возобновления общего доступа к экрану.
- Событие паузы: Когда общий доступ к экрану активен и пользователь нажимает кнопку остановки записи экрана системы, обратный вызов onScreenCapturePaused код причины: -4.
- Событие возобновления: Когда общий доступ к экрану приостановлен и пользователь нажимает кнопку запуска записи экрана системы, обратный вызов onScreenCaptureResumed код причины: -4.

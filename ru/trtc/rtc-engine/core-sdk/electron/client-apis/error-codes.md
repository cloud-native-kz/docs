# Коды ошибок

## Коды ошибок

### Базовые коды ошибок

| Код | Значение | Описание |
| --- | --- | --- |
| ERR_NULL | 0 | Нет ошибки. |

### Коды ошибок входа в комнату

`TRTCCloud.enterRoom()` вызывает этот тип кода ошибки при сбое входа в комнату. Вы можете использовать функции обратного вызова `TRTCCloudDelegate.onEnterRoom()` и `TRTCCloudDelegate.OnError()` для захвата соответствующих уведомлений.

| Код | Значение | Описание |
| --- | --- | --- |
| ERR_ROOM_ENTER_FAIL | -3301 | Ошибка входа в комнату. |
| ERR_ENTER_ROOM_PARAM_NULL | -3316 | Пустые параметры входа в комнату. Проверьте, передаются ли допустимые параметры в API `TRTCCloud.enterRoom():` при его вызове. |
| ERR_SDK_APPID_INVALID | -3317 | Неверный `sdkAppId`. |
| ERR_ROOM_ID_INVALID | -3318 | Неверный `roomId`. |
| ERR_USER_ID_INVALID | -3319 | Неверный `userID`. |
| ERR_USER_SIG_INVALID | -3320 | Неверный `userSig`. |
| ERR_ROOM_REQUEST_ENTER_ROOM_TIMEOUT | -3308 | Истекло время ожидания запроса входа в комнату. Проверьте вашу сеть. |
| ERR_SERVER_INFO_PRIVILEGE_FLAG_ERROR | -100006 | Ошибка при проверке билета разрешений. Проверьте, верен ли `privateMapKey`. |
| ERR_SERVER_INFO_SERVICE_SUSPENDED | -100013 | Сервис недоступен. Проверьте наличие оставшихся минут в ваших пакетах и наличие задолженности по счету в Tencent Cloud. |
| ERR_SERVER_INFO_ECDH_GET_TINYID | -100018 | Ошибка проверки `userSig`. Проверьте, верен ли `userSig`. |

### Код ошибки выхода из комнаты

`TRTCCloud.exitRoom()` вызывает этот код ошибки при сбое выхода из комнаты. Вы можете использовать функцию обратного вызова `TRTCCloudDelegate.OnError()` для захвата соответствующих уведомлений.

| Код | Значение | Описание |
| --- | --- | --- |
| ERR_ROOM_REQUEST_QUIT_ROOM_TIMEOUT | -3325 | Истекло время ожидания запроса выхода из комнаты. |

| Код | Значение | Описание |
| --- | --- | --- |
| ERR_SCREEN_CAPTURE_START_FAIL | -1308 | Ошибка запуска записи экрана. Если эта ошибка возникает на мобильном устройстве, это может быть потому, что пользователи отказали в разрешении на запись экрана; если она возникает на Windows или macOS, проверьте, установлены ли параметры API записи экрана в соответствии с требованиями. |
| ERR_SCREEN_CAPTURE_UNSURPORT | -1309 | Ошибка записи экрана. Если вы используете Android, убедитесь, что его версия 5.0 или выше; если вы используете iOS, убедитесь, что его версия 11.0 или выше. |

| ERR_SCREEN_CAPTURE_STOPPED | -7001 | Запись экрана остановлена системой. |

### Коды ошибок кодирования и декодирования

Вы можете использовать функцию обратного вызова `TRTCCloudDelegate.OnError()` для захвата соответствующих уведомлений.

| Код | Значение | Описание |
| --- | --- | --- |
| ERR_VIDEO_ENCODE_FAIL | -1303 | Ошибка кодирования видеокадров. Эта ошибка может возникнуть, когда пользователь на iOS переходит на другое приложение, что может привести к освобождению аппаратного кодировщика системой. Когда пользователь возвращается, эта ошибка может быть выброшена до перезагрузки аппаратного кодировщика. |

| ERR_AUDIO_ENCODE_FAIL | -1304 | Ошибка кодирования аудиокадров. Эта ошибка может возникнуть, когда SDK не может обработать переданные вами пользовательские аудиоданные. |
| PUSH_ERR_UNSUPPORTED_SAMPLERATE | -1306 | Неподдерживаемая частота дискретизации звука. |

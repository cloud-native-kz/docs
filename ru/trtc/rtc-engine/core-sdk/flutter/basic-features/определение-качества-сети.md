# Определение качества сети

Этот документ в первую очередь описывает, как определить качество текущей сети.

## Проверка качества сети во время звонка

TRTC предоставляет событие обратного вызова **onNetworkQuality**, которое сообщает вам о текущем качестве сети каждые две секунды. Его параметры включают две части: localQuality и remoteQuality.

- **localQuality**: Представляет ваше текущее качество сети, разделенное на 6 уровней: Excellent, Good, Poor, Bad, VeryBad и Down.
- **remoteQuality**: Представляет качество сети удаленных пользователей. Это массив, каждый Element (XML) в массиве представляет качество сети удаленного пользователя.

| Качество | Описание |
| --- | --- |
| unknown | Не определено |
| excellent | Текущая сеть исключительно хорошего качества |
| good | Текущая сеть имеет довольно хорошее качество |
| poor | Текущая сеть среднего качества |
| bad | Текущее качество сети плохое, может вызвать заметные задержки и проблемы со связью |
| vBad | Текущие условия сети ужасны, TRTC едва может поддерживать соединение, но не может гарантировать качество связи |
| down | Текущая сеть не соответствует минимальным требованиям TRTC, препятствуя нормальному видеозвонку |

Все, что вам нужно сделать, это мониторить `onNetworkQuality` TRTC и выводить соответствующие подсказки на интерфейсе:

```
TRTCCloudListener(    onNetworkQuality: (localQuality, remoteQuality) {      switch(localQuality.quality) {        case TRTCQuality.unknown:          //TODO: Handle this case.          break;        case TRTCQuality.excellent:          // TODO:Handle this case          break;        case TRTCQuality.good:          // TODO: Handle this case.          break;        case TRTCQuality.poor:          // TODO: Handle this case.          break;        case TRTCQuality.bad:          // TODO: Handle this case.          break;        case TRTCQuality.vBad:          // TODO: Handle this case.          break;        case TRTCQuality.down:          // TODO: Handle this case.          break;      }            for (TRTCQualityInfo info in remoteQuality) {        // TODO: Handle this case.      }    });
```

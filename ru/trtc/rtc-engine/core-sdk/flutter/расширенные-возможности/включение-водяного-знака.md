# Включение водяного знака

В этом документе описывается, как добавить водяной знак на видеопоток.

## Этапы реализации

### Включение камеры

```
trtcCloud.startLocalPreview(isFrontCamera, viewId);
```

### Добавление водяного знака

Предположим, необходимо добавить изображение с путем хранения imagePath в качестве водяного знака на основной экран (TRTCVideoStreamType.big):

```
setWatermark(String imagePath, double x, double y, double width) {  _trtcCloud.setWatermark(imagePath, TRTCVideoStreamType.big, x, y, width);}
```

API требует, чтобы путь изображения был доступным путем на текущем устройстве.

- Android: /storage/emulated/0/Android/data/имя вашего пакета/files/watermark.png
- iOS: /Library/Caches/watermark.png

Позиция водяного знака задается параметром rect, представляющим собой четверку значений в формате (x, y, width, height).

- x: координата водяного знака, число с плавающей точкой с диапазоном значений 0 - 1.
- y: координата водяного знака, число с плавающей точкой с диапазоном значений 0 - 1.
- width: ширина водяного знака, число с плавающей точкой с диапазоном значений 0 - 1.
- height: не используется, SDK автоматически рассчитывает подходящую высоту на основе соотношения сторон изображения водяного знака.

Пример установки параметров:

Если разрешение кодирования текущего видео составляет 540 × 960, а параметр rect установлен на (0.1, 0.1, 0.2, 0.0),

То верхняя левая точка координаты водяного знака составляет (540 × 0.1, 960 × 0.1), то есть (54, 96). Ширина водяного знака составляет 540 × 0.2 = 108px, а высота будет автоматически рассчитана SDK на основе соотношения сторон изображения водяного знака.

> **Примечание:** Изображение водяного знака должно быть в формате **PNG с прозрачным фоном.** Водяной знак, добавленный через API setWatermark, невидим в локальном предпросмотре. Для просмотра эффекта водяного знака необходимо получить поток пользователя с водяным знаком с удаленной стороны.

### Получение видеопотока с водяным знаком

Получите видеопоток пользователя с водяным знаком на другом устройстве.

```
trtcCloud.startRemoteView("denny", TRTCVideoStreamType.big, viewId);
```

### Удаление водяного знака

Введите пустое значение, чтобы удалить водяной знак из видеопотока, публикуемого вами.

```
trtcCloud.setWatermark("", TRTCVideoStreamType.big, 0.3, 0.4, 0.4);
```

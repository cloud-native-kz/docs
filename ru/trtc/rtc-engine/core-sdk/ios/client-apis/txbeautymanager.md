## setFilter:

**setFilter:**

| - (void)setFilter: | (nullable TXImage *)image |
| --- | --- |

**Устанавливает цветовой фильтр.**

Цветовой фильтр — это изображение таблицы поиска цветов, содержащее отношения сопоставления цветов. Вы можете найти несколько предопределенных изображений фильтров в официальном примере, который мы предоставляем.

SDK выполняет вторичную обработку исходного видеоизображения, захваченного камерой, в соответствии с отношениями сопоставления в таблице поиска для достижения ожидаемого эффекта фильтра.

| Param | DESC |
| --- | --- |
| image | Таблица поиска цветов, содержащая отношения сопоставления цветов. Изображение должно быть в формате PNG. |

## setFilterStrength:

**setFilterStrength:**

| - (void)setFilterStrength: | (float)strength |
| --- | --- |

**Устанавливает интенсивность цветового фильтра.**

Чем больше это значение, тем более очевидным является эффект цветового фильтра и тем больше разница в цвете между видеоизображением, обработанным фильтром, и исходным видеоизображением.

Интенсивность по умолчанию составляет 0,5, и если этого недостаточно, значение можно увеличить выше 0,5. Максимальное значение — 1.

| Param | DESC |
| --- | --- |
| strength | Диапазон значений: [0, 1]. Чем больше значение, тем более очевидным является эффект. Значение по умолчанию: 0.5 |

Этот интерфейс доступен только в SDK корпоративной версии (старая версия снята с производства; если вам нужно использовать функцию расширенной красоты в новом SDK, обратитесь к [Tencent Beauty Effect SDK](https://www.tencentcloud.com/document/product/1143/53942)).

| Param | DESC |
| --- | --- |
| path | Путь видеофайла в формате MP4. Пустое значение означает отключение эффекта. |

**Return Desc:**

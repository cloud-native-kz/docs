# Интеграция

Данное руководство в основном описывает, как реализовать базовый аудио- и видеовызов с помощью Objective-C.

## Предварительные требования

- Xcode 13.0 или позднее.
- iPhone или iPad с iOS 9.0 или позднее.
- Действительная подпись разработчика для вашего проекта.

## Руководство по интеграции

### Шаг 1. Импорт TRTC SDK

1. Выполните следующую команду в окне терминала для установки **CocoaPods**. Если вы уже установили **CocoaPods**, пропустите этот шаг.

```
sudo gem install cocoapods
```

2. В окне терминала перейдите в **корневую директорию проекта** и выполните следующую команду для создания **Podfile** для вашего проекта.

```
pod init
```

3. Отредактируйте и сохраните **Podfile** следующим образом.

```
platform :ios, '8.0'# Измените 'App' на название вашего проектаtarget 'App' dopod 'TXLiteAVSDK_TRTC', :podspec => 'https://liteav.sdk.qcloud.com/pod/liteavsdkspec/TXLiteAVSDK_TRTC.podspec'end
```

4. В окне терминала выполните следующую команду для обновления локальных файлов библиотеки и загрузки **TRTC SDK**.

```
pod install
```

> **Примечание:** После выполнения `pod install` будет создан новый файл проекта **.xcworkspace**. Дважды щелкните файл **.xcworkspace**, чтобы открыть его.

### Шаг 2. Конфигурация проекта

1. После открытия файла **.xcworkspace** щелкните **Project Navigator** в левой части **панели навигации Xcode**, щелкните название вашего проекта и убедитесь, что вы выбрали правильный **TARGETS** в области редактирования.
2. На вкладке `Build Settings` найдите **User Script Sandboxing** и установите его значение на **No**, что позволит пользовательским скриптам получать доступ к более широкому диапазону системных ресурсов и файлов.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/50de66827ca311ef852f52540075b605.png)

3. На вкладке `Info.plist` добавьте **Privacy-Microphone Usage Description** и **Privacy-Microphone Usage Description**, и заполните целевые подсказки, используемые **микрофоном/камерой** для получения разрешения на использование микрофона и камеры.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/50e906b47ca311ef852f52540075b605.png)

4. На вкладке `Signing & Capabilities` добавьте **Background Modes** и отметьте **Audio, AirPlay and Picture in Picture**, чтобы разрешить приложению запускать функции аудио, AirPlay и картинка-в-картинке в фоне.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/50ecee707ca311ef80ff525400d5f8ef.png)

### Шаг 3. Создание экземпляра TRTC

1. Импортируйте **TRTC SDK** в файл `AppDelegate.h`.

```
@import TXLiteAVSDK_TRTC; // Импорт модуля TRTC SDK
```

2. Объявите свойство **TRTCCloud** и интерфейс **TRTCCloudDelegate** в файле `AppDelegate.h`.

```
@interface AppDelegate : UIResponder <UIApplicationDelegate, TRTCCloudDelegate> // Объявление интерфейса TRTCCloudDelegate@property (nonatomic, strong) TRTCCloud *trtcCloud; // Объявление свойства TRTCCloud
```

3. После входа в файл `AppDelegate.m` вызовите `sharedInstance` для создания экземпляра TRTC в методе `didFinishLaunchingWithOptions` и установите слушатель событий.

```
- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {    // Создание экземпляра TRTC и установка слушателя    _trtcCloud = [TRTCCloud sharedInstance];    _trtcCloud.delegate = self;        return YES;}// Слушание события 'onError'- (void)onError:(TXLiteAVError)errCode         errMsg:(nullable NSString *)errMsg        extInfo:(nullable NSDictionary *)extInfo{    // Обработка события 'onError'    // Рекомендуется отображать информацию OnError через 'toastTip:'    NSString *errCodeStr = [NSString stringWithFormat:@"%ld", (long)errCode];    NSString *extInfoStr = extInfo ? [extInfo description] : @"";    NSString *notification = [NSString stringWithFormat:@"%@, %@, %@", errCodeStr, errMsg, extInfoStr];    [self toastTip:notification];}// Реализация 'toastTip:'- (void)toastTip:(NSString *)tip {    UIAlertController *alert = [UIAlertController alertControllerWithTitle:@"Tip"                                                                   message:tip                                                            preferredStyle:UIAlertControllerStyleAlert];    UIAlertAction *okAction = [UIAlertAction actionWithTitle:@"Confirm"                                                       style:UIAlertActionStyleDefault                                                     handler:nil];    [alert addAction:okAction];    [self.window.rootViewController presentViewController:alert animated:YES completion:nil];        dispatch_after(dispatch_time(DISPATCH_TIME_NOW, (int64_t)(2.0 * NSEC_PER_SEC)), dispatch_get_main_queue(), ^{        [alert dismissViewControllerAnimated:YES completion:nil];    });}
```

### Шаг 4. Вход в комнату

Установите параметр входа `TRTCParams` и вызовите `enterRoom` для успешного входа в комнату, что обычно вызывается после нажатия кнопки **Начать вызов**.

| Параметр | Тип | Описание |
| --- | --- | --- |
| sdkAppId | number | sdkAppId приложения аудио и видео, которое вы создали в [консоли TRTC](https://trtc.io/login?s_url=https://console.trtc.io/). |
| userId | string | ID пользователя, указанный вами. |
| userSig | string | Подпись пользователя, см. [UserSig](https://trtc.io/document/35166?platform=ios&product=rtcengine&menulabel=sdk) ã |
| roomId | number | ID комнаты, указанный вами, обычно уникальный ID комнаты. |

Для получения более подробного описания параметров обратитесь к документации интерфейса [enterRoom](https://trtc.io/document/50754#011dce4d6afaa3bcd684bebb77829689) ã

```
#import "AppDelegate.h" // Импорт файла "AppDelegate.h"- (void)enterRoom {    // Измените следующие параметры на ваши собственные    TRTCParams *trtcParams = [[TRTCParams alloc] init];    trtcParams.sdkAppId = 1400000123;    trtcParams.userId = @"denny";    trtcParams.userSig = @"";    trtcParams.roomId = 123321;        // Для многосторонних видеовызовов рекомендуется `TRTC_APP_SCENE_LIVE`    AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];    [appDelegate.trtcCloud enterRoom:trtcParams appScene:TRTCAppSceneLIVE];}// Слушание события 'onEnterRoom'- (void)onEnterRoom:(NSInteger)result {    // Обработка события 'onEnterRoom'    // Рекомендуется отображать информацию OnError через 'toastTip:'}
```

### Шаг 5. Включение/отключение камеры

1. Объявите свойства **UIWindow** и **UIView** в файле `ViewController.h`.

```
@property (strong, nonatomic) UIWindow *window; // Объявление свойства UIWindow@property (nonatomic, strong) UIView *localCameraVideoView; // Объявление свойства UIView
```

2. Установите параметр рендеринга `setLocalRenderParams` для локального предпросмотра и вызовите `startLocalPreview` для локального предпросмотра, а также начните передачу после успешного вызова `enterRoom`.

```
- (void)startLocalPreview {    // Установка параметра рендеринга локального предпросмотра    TRTCRenderParams *trtcRenderParams = [[TRTCRenderParams alloc] init];    trtcRenderParams.fillMode   = TRTCVideoFillMode_Fill;    trtcRenderParams.mirrorType = TRTCVideoMirrorTypeAuto;        AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];    [appDelegate.trtcCloud setLocalRenderParams:trtcRenderParams];    // Локальный предпросмотр захватываемого контента    [appDelegate.trtcCloud startLocalPreview:YES view:self.localCameraVideoView];}
```

Вызовите `stopLocalPreview` для отключения предпросмотра камеры и остановки передачи локальной видеоинформации.

```
AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud stopLocalPreview];
```

3. Выполняйте другие расширенные функции устройства, такие как **«Переключение передней и задней камер»**, **«Установка режима фокуса»** и **«Вспышка»**.

```
// По умолчанию передняя камера включена и переключение на заднюю камеруAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];if([deviceManager isFrontCamera]) {    [deviceManager switchCamera:false];}// Переключение на переднюю камеруAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];[deviceManager switchCamera:true];
```

```
// Если поддерживается автоматическое распознавание лиц, включите функцию автофокуса AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];if([deviceManager isAutoFocusEnabled]) {    [deviceManager enableCameraAutoFocus:true];}// Отключение функции автофокусаAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];[deviceManager enableCameraAutoFocus:false];
```

```
// Вспышку можно включить при использовании задней камерыAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];if(![deviceManager isFrontCamera]) {    [deviceManager enableCameraTorch:true];}// Отключение вспышкиAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];TXDeviceManager * deviceManager = [appDelegate.trtcCloud getDeviceManager];[deviceManager enableCameraTorch:false];
```

4. Инициализируйте **localCameraVideoView** в файле `ViewController.m` и добавьте его в иерархию представлений текущего представления.

```
#import "AppDelegate.h" // Импорт файла "AppDelegate.h"- (void)viewDidLoad {    [super viewDidLoad];    // Выполните любые дополнительные настройки после загрузки представления.    // Инициализация localCameraVideoView    self.localCameraVideoView = [[UIView alloc] initWithFrame:self.view.bounds];    self.localCameraVideoView.backgroundColor = [UIColor blackColor];        [self viewDidAppear];}- (void)viewDidAppear {    AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];        // Добавление localCameraVideoView в иерархию представлений текущего представления    [self.view addSubview:self.localCameraVideoView];    // Начало локального предпросмотра    [appDelegate.trtcCloud startLocalPreview:YES view:self.localCameraVideoView];}@end
```

### Шаг 6. Включение/отключение микрофона

Вызовите `startLocalAudio` для включения захвата микрофона. **Выберите один из следующих параметров качества звука** в соответствии с вашими требованиями.

```
// Включение захвата микрофона и установка текущей сцены на: Режим голоса // Для высокой способности подавления шума, сильной и слабой стойкости к сети AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud startLocalAudio:TRTCAudioQualitySpeech];
```

```
// Включение захвата микрофона и установка текущей сцены на: Режим музыки // Для высокого качества захвата, низких потерь качества звука, рекомендуется использовать с профессиональными звуковыми картамиAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud startLocalAudio:TRTCAudioQualityMusic];
```

Вызовите `stopLocalAudio` для отключения захвата микрофона и остановки передачи локальной аудиоинформации.

```
AppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud stopLocalAudio];
```

### Шаг 7. Воспроизведение/остановка видеопотока

1. Слушайте [onUserVideoAvailable](https://trtc.io/document/50755#36daf607f51a906ea0b48b33fc628161) перед входом в комнату. Когда вы получите уведомление `onUserVideoAvailable(userId, true)`, это означает, что имеются видеокадры для воспроизведения из удаленного экрана.
2. Вызовите `startRemoteView` для воспроизведения видеоконтента, захватываемого удаленной стороной.

```
startRemoteView
```

Вызовите `stopRemoteView` для остановки удаленного видео.

```
// Остановка видеоAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud stopRemoteView:@"denny"]; // Остановить видео только denny[appDelegate.trtcCloud stopAllRemoteView]; // Остановить все видео
```

### Шаг 8. Воспроизведение/остановка аудиопотока

Вызовите `muteRemoteAudio` для отключения или включения звука удаленной стороны.

```
// ОтключениеAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud muteRemoteAudio:@"denny" mute:YES]; // Отключить только denny[appDelegate.trtcCloud muteAllRemoteAudio:YES]; // Отключить всех удаленных пользователей
```

```
// ВключениеAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud muteRemoteAudio:@"denny" mute:NO]; // Включить только denny[appDelegate.trtcCloud muteAllRemoteAudio:NO]; // Включить всех удаленных пользователей
```

### Шаг 9. Выход из комнаты

Вызовите `exitRoom` для выхода из текущей комнаты, и TRTC SDK уведомит вас об этом через событие обратного вызова `onExitRoom`.

```
// Выход из текущей комнатыAppDelegate *appDelegate = (AppDelegate *)[[UIApplication sharedApplication] delegate];[appDelegate.trtcCloud exitRoom];// Слушание обратного вызова onExitRoom для выяснения причины выхода- (void)onExitRoom:(NSInteger)reason {    if (reason == 0) {        NSLog(@"Выход из текущей комнаты путем вызова API 'exitRoom' SDK ...");    } else if (reason == 1) {        NSLog(@"Исключен из текущей комнаты сервером через REST API...");    } else if (reason == 2) {        NSLog(@"Текущая комната была растворена сервером через REST API...");    }}
```

# Коды ошибок

## Коды ошибок

### Базовые коды ошибок

| Code | Value | Description |
| --- | --- | --- |
| ERR_NULL | 0 | No error. |

### Коды ошибок при входе в комнату

`TRTCCloud.enterRoom()` выдаст этот тип кода ошибки, если вход в комнату не удался. Вы можете использовать функции обратного вызова `TRTCCloudDelegate.onEnterRoom()` и `TRTCCloudDelegate.OnError()` для захвата соответствующих уведомлений.

| Code | Value | Description |
| --- | --- | --- |
| ERR_ROOM_ENTER_FAIL | -3301 | Не удалось войти в комнату. |
| ERR_ENTER_ROOM_PARAM_NULL | -3316 | Пустые параметры входа в комнату. Проверьте, переданы ли корректные параметры в API `TRTCCloud.enterRoom():` при его вызове. |
| ERR_SDK_APPID_INVALID | -3317 | Недействительный `sdkAppId`. |
| ERR_ROOM_ID_INVALID | -3318 | Недействительный `roomId`. |
| ERR_USER_ID_INVALID | -3319 | Недействительный `userID`. |
| ERR_USER_SIG_INVALID | -3320 | Недействительный `userSig`. |
| ERR_ROOM_REQUEST_ENTER_ROOM_TIMEOUT | -3308 | Время ожидания запроса входа в комнату истекло. Проверьте вашу сеть. |
| ERR_SERVER_INFO_PRIVILEGE_FLAG_ERROR | -100006 | Не удалось проверить билет разрешения. Проверьте, правильно ли установлен `privateMapKey`. |
| ERR_SERVER_INFO_SERVICE_SUSPENDED | -100013 | Услуга недоступна. Проверьте, остались ли минуты в ваших пакетах и нет ли просроченного платежа в вашей учетной записи Tencent Cloud. |
| ERR_SERVER_INFO_ECDH_GET_TINYID | -100018 | Проверка `userSig` не удалась. Проверьте, правильно ли установлен `userSig`. |

### Коды ошибок при выходе из комнаты

`TRTCCloud.exitRoom()` выдает этот код ошибки, если выход из комнаты не удался. Вы можете использовать функцию обратного вызова `TRTCCloudDelegate.OnError()` для захвата соответствующих уведомлений.

| Code | Value | Description |
| --- | --- | --- |
| ERR_ROOM_REQUEST_QUIT_ROOM_TIMEOUT | -3325 | Время ожидания запроса выхода из комнаты истекло. |

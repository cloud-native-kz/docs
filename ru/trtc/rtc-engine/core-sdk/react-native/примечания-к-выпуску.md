# Примечания к выпуску

## Версия 3.2.0 @2025.05.16

### Обновление зависимостей

- **Android**: Обновление TRTC SDK до 12.5.0.17568
- **iOS**: Обновление TRTC SDK до 12.5.18359

## Версия 3.1.1 @2025.04.28

### **Исправленные ошибки**

- **iOS**: Удалено объявление, но метод не реализован, чтобы избежать сбоев в React Native версии выше 0.76.

### Функция

- Новый метод тестирования скорости сети ([startSpeedTest](https://www.tencentcloud.com/document/product/647/69704#.E5.A6.82.E4.BD.95.E6.B5.8B.E9.80.9F)).

## Версия 3.1.0 @2025.04.24

### Обновление зависимостей

- **Android**: Обновление TRTC SDK до 12.4.0.17372
- **iOS**: Обновление TRTC SDK до 12.4.17856

## Версия 3.0.2 @2015.01.20

### **Исправленные ошибки**

- **iOS:** Исправлена проблема, при которой удаление локального представления (TXVideoView.LocalView) не отключает камеру.

## Версия 3.0.1 @2025.01.10

### Функция

- Добавлен новый параметр frontCamera для локального представления (TXVideoView.LocalView), поддерживает установку включения фронтальной камеры при создании.

## Версия 3.0.0 @2025.01.08

### **Исправленные ошибки**

- **iOS&Android**: Переключение TRTC SDK на Professional.

## Версия 2.5.4 @2025.01.07

### **Исправленные ошибки**

- **iOS & Android:** Исправлена проблема, при которой API в TXDeviceManager, TXBeautyManager и TXAudioEffectManager недоступны.

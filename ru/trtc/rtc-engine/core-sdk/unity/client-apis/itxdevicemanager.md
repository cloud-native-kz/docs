## setCameraFocusPosition

**setCameraFocusPosition**

| int setCameraFocusPosition | (int x |
| --- | --- |
|  | int y) |

#### Регулировка фокуса (для мобильных ОС).

Этот API можно использовать для достижения следующего:

1. Пользователь может коснуться предпросмотра камеры.

2. На месте касания пользователя появится прямоугольник, указывающий точку, на которой камера будет фокусироваться.

3. Пользователь передает координаты точки в SDK, используя этот API, и SDK инструктирует камеру фокусироваться по мере необходимости.

| Параметр | Описание |
| --- | --- |
| position | Точка для фокусировки. Передайте координаты точки, на которой вы хотите сфокусироваться. |

> **Примечание**: перед использованием этого API необходимо сначала отключить автофокус с помощью [enableCameraAutoFocus](https://www.tencentcloud.com/document/product/647/72274#6786f243b9354cf0163d5e37a45125e0).

#### Описание возврата:

0: операция успешна; отрицательное число: операция не удалась.

## setAudioRoute

**setAudioRoute**

| int setAudioRoute | ([TXAudioRoute](https://www.tencentcloud.com/document/product/647/72274#39c508dc7c357c2feda25e56f2d729c1) route) |
| --- | --- |

#### Установка маршрута звука (для мобильных ОС).

Мобильный телефон имеет два устройства воспроизведения звука: приемник вверху и динамик внизу.

Если маршрут звука установлен на приемник, громкость относительно низкая, и звук можно услышать только при приближении телефона к уху. Этот режим имеет высокий уровень приватности и подходит для ответа на звонки.

Если маршрут звука установлен на динамик, громкость относительно высокая, и нет необходимости приближать телефон к уху. Этот режим позволяет использовать функцию "hands-free".

## TXSystemVolumeType(Deprecated)

**TXSystemVolumeType(Deprecated)**

#### Тип системной громкости (для мобильных ОС).

@deprecated Не рекомендуется использовать с версии 9.5.

Современные смартфоны обычно имеют два типа системной громкости: "громкость разговора" и "громкость медиа".

- Громкость разговора: тип громкости, разработанный специально для совершения и приема звонков. Он имеет встроенную функцию подавления эхо (AEC) и поддерживает захват звука через микрофон гарнитуры Bluetooth. Недостатком является то, что качество звука среднее.

Если вы используете кнопку громкости на боку телефона для уменьшения громкости, и она не может быть отрегулирована до нуля (т. е. не может быть полностью отключена), это означает, что ваш телефон в настоящее время находится в режиме громкости разговора.

- Громкость медиа: тип громкости, разработанный специально для музыкальных сцен на телефоне, не может использовать встроенную функцию AEC системы и не поддерживает захват звука через микрофон гарнитуры Bluetooth, но имеет лучший эффект воспроизведения музыки.

Если вы используете кнопку громкости на боку телефона для уменьшения громкости, и громкость телефона может быть отрегулирована до полного отключения, это означает, что ваш телефон в настоящее время находится в режиме громкости медиа.

SDK в настоящее время предоставляет три режима управления типами системной громкости: автоматический, режим громкости медиа и режим громкости разговора.

| Enum | Значение | Описание |
| --- | --- | --- |
| TXSystemVolumeTypeAuto | 0 | Автоматический |
| TXSystemVolumeTypeMedia | 1 | Громкость медиа |
| TXSystemVolumeTypeVOIP | 2 | Громкость разговора |

## TXAudioRoute

**TXAudioRoute**

#### Маршрут звука (маршрут, через который воспроизводится звук).

Маршрут звука — это маршрут (динамик или приемник), через который воспроизводится звук. Это применимо только к мобильным устройствам, таким как мобильные телефоны.

Мобильный телефон имеет два динамика: один вверху (приемник) и один внизу.

- Если маршрут звука установлен на приемник, громкость относительно низкая, и звук можно услышать только при приближении телефона к уху. Этот режим имеет высокий уровень приватности и подходит для ответа на звонки.
- Если маршрут звука установлен на динамик, громкость относительно высокая, и нет необходимости приближать телефон к уху. Этот режим позволяет использовать функцию "hands-free".

| Enum | Значение | Описание |
| --- | --- | --- |
| TXAudioRouteSpeakerphone | 0 | Громкая связь: нижний динамик используется для воспроизведения (hands-free). С относительно высокой громкостью используется для громкого воспроизведения музыки. |
| TXAudioRouteEarpiece | 1 | Наушник: верхний приемник используется для воспроизведения. С относительно низкой громкостью подходит для сценариев звонков, требующих приватности. |

## TXCameraCaptureMode

**TXCameraCaptureMode**

#### Параметры захвата камеры.

Это перечисление используется для установки параметров захвата камеры.

| Enum | Значение | Описание |
| --- | --- | --- |
| TXCameraResolutionStrategyAuto | 0 | Автоматическая регулировка параметров захвата камеры. | TXCameraResolutionStrategyHighQuality | 2 | Приоритет качества видеопредпросмотра. SDK выбирает более высокие параметры вывода камеры для улучшения качества видео предпросмотра. В этом случае потребуется больше CPU и памяти для предварительной обработки видео. |
| TXCameraCaptureManual | 3 | Позволяет пользователю установить ширину и высоту видео, захватываемого локальной камерой. |

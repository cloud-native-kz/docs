# Интеграция

В этой статье описано, как быстро интегрировать Unity RTC Engine и реализовать базовый аудио и видео вызов.

## Подготовка окружения

- Unity 2022.3.13f1.
- В настоящее время поддерживаются платформы Android, iOS, Windows, Mac.
- Unity должна включать модули Android Build Support, iOS Build Support, Windows Build Support и MacOs Build Support.
- Для разработки для iOS убедитесь, что ваш проект имеет действительную подпись разработчика.

## Руководство по интеграции

### Шаг 1. Импорт TRTC SDK

1. Откройте Unity и создайте собственную программу Unity.
2. Скачайте SDK Unity и распакуйте библиотеку SDK, содержащую следующие файлы:

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/0929d5935d3611f09fd0525400bf7822.png)

3. Скопируйте файлы **Plugins** и **TRTCSDK** из директории Assets распакованного **SDK** в директорию Assets файлов вашего проекта.

### Шаг 2. Конфигурация проекта

iOS / Mac / Android требуют добавления запросов разрешений камеры и микрофона в проект:

iOS/Mac

Android

1. Откройте интерфейс настроек Player в меню **Project Settings** проекта Unity.
2. Добавьте описания запросов соответствующих разрешений в **Camera Usage Description и Microphone Usage Description** платформ MAC/iOS соответственно, как показано на следующем рисунке.

![](https://cloudcache.intl.tencent-cloud.com/cms/backend-cms/b51f8faf5d3a11f094cd52540099c741.png)

1. Откройте файл `AndroidManifest.xml` в вашем проекте.
2. Добавьте следующие разрешения:

```
<uses-permission android:name="android.permission.INTERNET" /><uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" /><uses-permission android:name="android.permission.ACCESS_WIFI_STATE" /><uses-permission android:name="android.permission.RECORD_AUDIO" /><uses-permission android:name="android.permission.MODIFY_AUDIO_SETTINGS" /><uses-permission android:name="android.permission.BLUETOOTH" /><uses-permission android:name="android.permission.CAMERA" /><uses-feature android:name="android.hardware.camera.autofocus" />
```

## Шаг 3. Создание экземпляра TRTC

1. Импортируйте пространство имен TRTC.

```
using trtc;
```

2. Наследуйте **ITRTCCloudCallback** и переопределите методы **ITRTCCloudCallback** для объявления и определения событий **TRTCCloud**.

```
public class AppExample : MonoBehaviour,                          ITRTCCloudCallback {private ITRTCCloud mTRTCCloud;  // Declare the TRTCCloud member variable#region ITRTCCloudCallbackpublic:	public void onError(TXLiteAVError errCode, String errMsg, IntPtr arg) {} // Listen for the 'onError' event    public void onWarning(TXLiteAVWarning warningCode, String warningMsg, IntPtr arg) {} // Listen for the 'onWarning' event    public void onEnterRoom(int result) {} // Listen for the result of 'onEnterRoom' event    public void onExitRoom(int reason) {}  // Listen for the result of 'onExitRoom' event    //  to do other event......... #endregion}
```

3. Вызовите интерфейс для получения экземпляра TRTC.

```
mTRTCCloud = ITRTCCloud.getTRTCShareInstance();
```

4. Вызовите интерфейс `addCallback` для добавления слушателей событий **TRTCCloud**.

```
mTRTCCloud.addCallback(this);
```

### Шаг 4. Вход в комнату

1. Установите параметр входа `TRTCParams` и вызовите `enterRoom` для успешного входа в комнату.

| Параметр | Тип | Описание |
| --- | --- | --- |
| sdkAppId | UInt32 | sdkAppId приложения аудио и видео, которое вы создали в [TRTC Console](https://trtc.io/login?s_url=https://console.trtc.io/). |
| userId | String | ID пользователя, указанный вами. |
| userSig | String | Подпись пользователя, см. [UserSig](https://trtc.io/document/35166?platform=windows&product=rtcengine&menulabel=sdk). |
| roomId | UInt32 | ID комнаты, указанный вами, обычно уникальный ID комнаты. |

Для получения более подробного описания параметров обратитесь к документации интерфейса [enterRoom](https://trtc.io/document/50770#b6eb951dc67569848a415ba028f6746d).

```
TRTCParams trtcParams = new TRTCParams();trtcParams.sdkAppId = 1400xxxxx;trtcParams.roomId = 345;trtcParams.userId = "123";trtcParams.userSig = "";TRTCAppScene scene = TRTCAppScene.TRTCAppSceneLIVE;mTRTCCloud.enterRoom(ref trtcParams, scene);public void onEnterRoom(int result) {} // Listen for the result of 'onEnterRoom' event
```

### Шаг 5. Включение/отключение камеры

1. Создайте компонент `RawImage: videoView` в проекте Unity в качестве значения параметра view интерфейса `startLocalPreview`.
2. Перед вызовом интерфейса `startLocalPreview` для открытия предпросмотра камеры вы можете установить параметры рендеринга локального предпросмотра, вызвав интерфейс `setLocalRenderParams`.

```
// Set local preview rendering parametersprivate TRTCRenderParams renderParams = new TRTCRenderParams();renderParams.fillMode = TRTCVideoFillMode.TRTCVideoFillMode_Fit;renderParams.mirrorType = TRTCVideoMirrorType.TRTCVideoMirrorType_Disable;renderParams.rotation = TRTCVideoRotation.TRTCVideoRotation0;mTRTCCloud.setLocalRenderParams(renderParams);// Local preview of the content captured by the front cameramTRTCCloud.startLocalPreview(true, videoView);// Local preview of the content captured by the rear cameramTRTCCloud.startLocalPreview(false, videoView);
```

3. Вызовите `stopLocalPreview` для закрытия предпросмотра камеры и остановки отправки локальной видеоинформации.

```
mTRTCCloud.stopLocalPreview();
```

### Шаг 6. Включение/отключение микрофона

1. Вызовите `startLocalAudio` для включения захвата микрофона. **Выберите один из следующих параметров качества звука** `Quality` **в соответствии с вашими требованиями**.

```
// Enable microphone acquisition and set the current scene to: Voice mode // For high noise suppression capability, strong and weak network resistancemTRTCCloud.startLocalAudio(TRTCAudioQuality.TRTCAudioQualitySpeech);
```

```
// Enable microphone acquisition, and set the current scene to: Music mode // For high fidelity acquisition, low sound quality loss, recommended to use with professional sound cardsmTRTCCloud.startLocalAudio(TRTCAudioQuality.TRTCAudioQualityMusic);
```

2. Вызовите `stopLocalAudio` для отключения захвата микрофона и остановки отправки локальной аудиоинформации.

```
mTRTCCloud.stopLocalAudio()
```

### Шаг 7. Воспроизведение/остановка видеопотока

1. Перед входом в комнату слушайте событие `onUserVideoAvailable`. Когда вы получите уведомление `onUserVideoAvailable(userId, true)`, это означает, что доступны видеокадры для воспроизведения видеопотока этого пользователя.

```
public void onUserVideoAvailable(String userId, bool available) {}
```

2. Вызовите `startRemoteView/stopRemoteView` для воспроизведения или остановки удаленного видео.

```
mTRTCCloud.startRemoteView(âdennyâ, TRTCVideoStreamType.TRTCVideoStreamTypeBig, videoView);// videoView is the RawImage component of Unity
```

3. Вызовите `stopRemoteView` для остановки воспроизведения удаленного изображения.

```
mTRTCCloud.stopRemoteView(âdennyâ, TRTCVideoStreamType.TRTCVideoStreamTypeBig);  // Stop playing denny's videomTRTCCloud.stopAllRemoteView(true); // Mute all remote users
```

### Шаг 8. Воспроизведение/остановка аудиопотока

Вызовите `muteRemoteAudio` для отключения или включения удаленного звука.

```
// MutemTRTCCloud.muteRemoteAudio("denny", true); // Mute dennymTRTCCloud.muteAllRemoteAudio(true); // Mute all remote users
```

```
// UnmutemTRTCCloud.muteRemoteAudio("denny", false); // Unmute dennymTRTCCloud.muteAllRemoteAudio(false); // Unmute all remote users
```

### Шаг 9. Выход из комнаты

1. Вызовите `exitRoom` для выхода из текущей комнаты.

```
mTRTCCloud.exitRoom();
```

2. **TRTC SDK** уведомит вас о выходе из комнаты через событие обратного вызова `onExitRoom`.

```
public void onExitRoom(int reason) {}  // Listen for the result of 'onExitRoom' event
```

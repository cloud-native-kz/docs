# Установка профиля кодирования

Это руководство главным образом описывает:

1. Установку профиля кодирования видео.
2. Установку профиля кодирования аудио.
3. Установку профиля кодирования совместного использования экрана.

## Демонстрация

## Установка профиля кодирования видео

Вы можете установить профиль кодирования видео через параметр **profile** в [trtc.startLocalVideo()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#startLocalVideo) и [trtc.updateLocalVideo()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#updateLocalVideo).

Укажите предопределённый профиль, и каждый профиль соответствует набору рекомендуемых разрешения, частоты кадров и битрейта.

```
// Укажите профиль видео при запускеawait trtc.startLocalVideo({  option: { profile: '480p' }});// Динамически измените профиль видео во время вызоваawait trtc.updateLocalVideo({  option: { profile: '360p' }});
```

| Профиль видео | Разрешение (ширина x высота) | Частота кадров (fps) | Битрейт (кбит/с) | Примечание |
| --- | --- | --- | --- | --- |
| 120p | 160 x 120 | 15 | 200 |  |
| 120p_2 | 160 x 120 | 15 | 100 | Поддерживается только в v5.1.1+ |
| 180p | 320 x 180 | 15 | 350 |  |
| 180p_2 | 320 x 180 | 15 | 150 | Поддерживается только в v5.1.1+ |
| 240p | 320 x 240 | 15 | 400 |  |
| 240p_2 | 320 x 240 | 15 | 200 | Поддерживается только в v5.1.1+ |
| 360p | 640 x 360 | 15 | 800 |  |
| 360p_2 | 640 x 360 | 15 | 400 | Поддерживается только в v5.1.1+ |
| 480p | 640 x 480 | 15 | 900 |  |
| 480p_2 **(По умолчанию)** | 640 x 480 | 15 | 500 | Поддерживается только в v5.1.1+ |
| 720p | 1280 x 720 | 15 | 1500 |  |
| 1080p | 1920 x 1080 | 15 | 2000 |  |
| 1440p | 2560 x 1440 | 30 | 4860 |  |
| 4K | 3840 x 2160 | 30 | 9000 |  |

Укажите пользовательское разрешение, частоту кадров и битрейт:

```
// Укажите профиль видео при запускеawait trtc.startLocalVideo({  option: { profile: { width: 640, height: 480, frameRate: 15, bitrate: 900 /* kpbs */} }});// Динамически измените профиль видео во время вызоваawait trtc.updateLocalVideo({  option: { profile: { width: 640, height: 360, frameRate: 15, bitrate: 800 /* kpbs */} }});
```

> **Примечание:** Из-за ограничений устройства и браузера разрешение видео может не совпадать точно. В этом случае браузер автоматически приведёт разрешение в соответствие с указанным в профиле. Из-за производительности сети и процессора разрешение и частота кадров кодирования видео могут быть ниже ожидаемых. Для камеры по умолчанию приоритет отдаётся плавности, что снижает разрешение надлежащим образом и отдаёт приоритет кодированию частоты кадров при недостаточной производительности. Для совместного использования экрана по умолчанию приоритет отдаётся чёткости, что снижает частоту кадров надлежащим образом и отдаёт приоритет кодированию разрешения при недостаточной производительности. Вы можете изменить этот приоритет с помощью параметра **qosPreference** в [trtc.startLocalVideo()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#startLocalVideo) и [trtc.updateLocalVideo()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#updateLocalVideo). Смотрите [QOS_PREFERENCE_SMOOTH](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/module-TYPE.html#.QOS_PREFERENCE_SMOOTH) и [QOS_PREFERENCE_CLEAR](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/module-TYPE.html#.QOS_PREFERENCE_CLEAR).

## Установка профиля кодирования аудио

Вы можете установить профиль кодирования аудио через параметр **profile** в [trtc.startLocalAudio()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#startLocalAudio).

```
// Профиль по умолчаниюawait trtc.startLocalAudio({ option: { profile: TRTC.TYPE.AUDIO_PROFILE_STANDARD }});// Примечание: SDK не поддерживает динамическое изменение профиля аудио во время вызова.// Вы должны установить профиль аудио в trtc.startLocalAudioawait trtc.updateLocalAudio({ option: { profile: TRTC.TYPE.AUDIO_PROFILE_HIGH }}); // Не работает.
```

| Профиль аудио | Частота дискретизации | Канал | Битрейт (кбит/с) |
| --- | --- | --- | --- |
| TRTC.TYPE.AUDIO_PROFILE_STANDARD **(По умолчанию)** | 48000 | Моно | 40 |
| TRTC.TYPE.AUDIO_PROFILE_HIGH | 48000 | Моно | 128 |
| TRTC.TYPE.AUDIO_PROFILE_STANDARD_STEREO | 48000 | Стерео | 64 |
| TRTC.TYPE.AUDIO_PROFILE_HIGH_STEREO | 48000 | Стерео | 192 |

## Установка профиля кодирования совместного использования экрана

Вы можете установить профиль кодирования совместного использования экрана через параметр **profile** в [trtc.startScreenShare()](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/TRTC.html#startScreenShare).

```
// Профиль по умолчанию — '1080p'await trtc.startScreenShare({ option: { profile: '1080p_2' }});// Примечание: SDK не поддерживает динамическое изменение профиля совместного использования экрана во время вызова.// Укажите пользовательский профиль при запускеawait trtc.startLocalVideo({  option: { profile: { width: 1280, height: 720, frameRate: 15, bitrate: 1500 /* kpbs */} }});
```

| Профиль экрана | Разрешение (ширина x высота) | Частота кадров (fps) | Битрейт (кбит/с) |
| --- | --- | --- | --- |
| 480p | 640 x 480 | 5 | 900 |
| 480p_2 | 640 x 480 | 30 | 1000 |
| 720p | 1280 x 720 | 5 | 1200 |
| 720p_2 | 1280 x 720 | 30 | 3000 |
| 1080p | 1920 x 1080 | 5 | 1600 |
| 1080p_2 | 1920 x 1080 | 30 | 4000 |

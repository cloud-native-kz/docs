# Введение в продукт

Опираясь на более чем 20-летний опыт Tencent в области сетевых технологий и технологий аудио/видео, Tencent Real-Time Communication (TRTC) предлагает решения для группового аудио/видео общения и низколатентного интерактивного трансляции в реальном времени. С помощью TRTC вы можете быстро разработать экономически эффективные, низколатентные и высококачественные интерактивные аудио/видео сервисы.

- Групповое аудио/видео общение
 Построенное на сети Direct Connect облака Tencent Cloud, это решение обеспечивает глобальное подключение и предоставляет клиентские SDK и облачные API для мобильных и настольных платформ. Пользователи также могут легко получить доступ к сервисам TRTC на веб-страницах.
- Низколатентная интерактивная трансляция в реальном времени
 Это решение использует ведущие в отрасли сетевые технологии, технологии аудио/видео облака Tencent Cloud, а также высококачественные ресурсы узлов для создания сервисов интерактивной трансляции с минимальной задержкой и латентностью ниже 1 секунды, переводя трансляцию в эру CDN 2.0.

## Архитектура продукта

TRTC предлагает кроссплатформенные решения для аудио/видео общения и низколатентной интерактивной трансляции. С SDK для мини-программ, веб, Android, iOS, Electron, Windows, macOS и других платформ вы можете быстро интегрировать сервисы TRTC в ваши проекты и подключиться к серверной части TRTC. Вы также можете комбинировать TRTC с другими продуктами Tencent Cloud, такими как Instant Messaging (IM), Cloud Streaming Services (CSS) и Video on Demand (VOD), чтобы изучить больше вариантов использования.

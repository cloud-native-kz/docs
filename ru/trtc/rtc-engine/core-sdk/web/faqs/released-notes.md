# Примечания к выпускам

Номер версии имеет формат `major.minor.patch`.

- `major`: Номер основной версии. При значительной переделке версии это поле увеличивается. Как правило, API различных основных версий несовместимы друг с другом.
- `minor`: Номер минорной версии. API различных минорных версий совместимы друг с другом. При добавлении новых или оптимизированных API это поле увеличивается.
- `patch`: Номер патча. При улучшении функциональности или исправлении ошибок это поле увеличивается.

> **Примечание:** Пожалуйста, обновляйте до последней версии своевременно для стабильности обслуживания и лучшей поддержки в сети. Для предосторожностей при обновлении версии см. [Руководство по обновлению](https://web.sdk.qcloud.com/trtc/webrtc/v5/doc/en/tutorial-00-info-update-guideline.html).

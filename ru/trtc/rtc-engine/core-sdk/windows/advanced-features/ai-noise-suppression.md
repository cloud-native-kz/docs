# Подавление шума на основе ИИ

Подавление шума на основе ИИ разработано на основе алгоритмов ИИ от Tencent Tianlai Laboratory. Оно способно интеллектуально обнаруживать и устранять помехи шума, смешанные с передаваемым сигналом, тем самым значительно улучшая качество голоса, повышая четкость и улучшая опыт прослушивания пользователя.

Применение функций подавления шума на основе ИИ в RTC Engine позволяет пользователям испытать четкий и стабильный звук в различных окружениях, таких как офисы, интернет-кафе, торговые центры, открытые пространства и т.д.

## Активируйте подавление шума на основе ИИ

RTC-Engine теперь имеет подавление шума на основе ИИ включенным по умолчанию при использовании [TRTCAudioQualitySpeech](https://trtc.io/document/50770#37f11bf81ac7eef6af030790d31bc86d), поэтому пользователи могут наслаждаться высококачественным подавлением шума в своем приложении без дополнительных настроек или операций.

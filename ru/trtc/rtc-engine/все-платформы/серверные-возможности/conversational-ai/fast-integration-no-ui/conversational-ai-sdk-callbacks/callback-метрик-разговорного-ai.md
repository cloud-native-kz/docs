# Callback метрик разговорного AI

Разговорный AI Tencent RTC предоставляет набор rich callback метрик. Callback метрики отправляются через пользовательские сообщения Tencent RTC, что позволяет легко контролировать производительность разговора на клиенте. Метрики, такие как продолжительность и производительность вызовов LLM и TTS, передаются на терминалы в реальном времени через пользовательские сообщения Tencent RTC, обеспечивая мониторинг длительности и качества разговора AI.

Вы можете использовать функцию [Receive Custom Messages](https://www.tencentcloud.com/document/product/647/47866), предоставляемую SDK RTC Engine, для прослушивания callback в клиенте и получения данных метрик разговора AI. **Значение cmdID фиксировано на 1.**

| Поле | Тип | Описание |
| --- | --- | --- |
| type | Number | Тип сообщения. 10020 указывает на callback вызова сервиса AI. |
| sender | String | User ID отправителя, который является ID чатбота. |
| payload | Object | Payload сообщения, включающий детали метрик. |

Объект payload содержит следующие поля:

| Поле | Тип | Описание |
| --- | --- | --- |
| metric | String | Названия вызываемых метрик: asr_latencyllm_network_latencyllm_first_tokentts_network_latencytts_first_frame_latencytts_discontinuityinterruption |
| value | Number | Вызываемая метрика. |
| tag | Object | Тег, связанный с метрикой. |

Объект tag содержит следующие поля:

| Поле | Тип | Описание |
| --- | --- | --- |
| roundid | String | ID раунда разговора. |

Описание названий метрик:

| Код статуса | Описание |
| --- | --- |
| asr_latency | Задержка ASR (STT). Примечание: эта метрика включает время, установленное VadSilenceTime при запуске разговорного AI. |
| llm_network_latency | Задержка сети запроса LLM. |
| llm_first_token | Задержка первого токена LLM. Эта метрика включает задержку сети. |
| tts_network_latency | Задержка сети запроса TTS. |
| tts_first_frame_latency | Задержка первого кадра TTS. Эта метрика включает задержку сети. |
| tts_discontinuity | Количество случаев разрыва запроса TTS. Разрыв указывает на то, что результат не возвращается для следующего запроса после завершения текущего потокового запроса TTS, что обычно вызывается высокой задержкой сети запроса TTS. |
| interruption | Эта метрика указывает на то, что разговор прерван. |

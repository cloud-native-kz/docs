# Callback ошибок Conversational AI

Conversational AI платформы Tencent RTC предоставляет функцию callback ошибок. Callbacks ошибок отправляются через пользовательские сообщения Tencent RTC, что позволяет легко контролировать производительность разговора на клиенте. Например, когда вызов LLM/TTS не удается, данные передаются на терминал в реальном времени через пользовательские сообщения Tencent RTC для мониторинга качества и коэффициента успеха AI разговора.

Вы можете использовать функцию [Receive Custom Messages](https://www.tencentcloud.com/document/product/647/47866), предоставленную SDK RTC Engine, для прослушивания callbacks на клиенте и получения данных от callbacks ошибок AI разговора. **Значение cmdID зафиксировано на 1.**

| Поле | Тип | Описание |
| --- | --- | --- |
| type | Number | Тип сообщения. 10030 указывает на ошибку AI сервиса. |
| sender | String | ID пользователя отправителя, который является ID чат-бота. |
| payload | Object | Полезная нагрузка сообщения, включающая подробную информацию. |

Объект payload содержит следующие поля:

| Поле | Тип | Описание |
| --- | --- | --- |
| metric | String | Имена вызванных метрик следующие: asr_errorllm_errortts_error |
| tag | Object | Тег, связанный с метрикой. |

Объект tag содержит следующие поля:

| Поле | Тип | Описание |
| --- | --- | --- |
| roundid | String | ID раунда разговора. |
| code | Number | Код ошибки вызова. |
| message | String | Подробное описание сообщения об ошибке. |

Описание имен метрик:

| Код состояния | Описание |
| --- | --- |
| asr_error | Ошибка ASR (STT). |
| llm_error | Ошибка запроса LLM. |
| tts_error | Ошибка запроса TTS. |

Список кодов ошибок:

| Категория сервиса | Код ошибки | Описание ошибки |
| --- | --- | --- |
| ASR (STT) | 30100 | Истечение времени ожидания запроса. |
|  | 30102 | Внутренняя ошибка. |
| LLM | 30200 | Истечение времени ожидания запроса LLM. |
|  | 30201 | Ограничена частота запросов LLM. |
|  | 30202 | Ошибка ответа сервиса LLM. |
| TTS | 30300 | Истечение времени ожидания запроса TTS. |
|  | 30301 | Ограничена частота запросов TTS. |
|  | 30302 | Ошибка ответа сервиса TTS. |

> **Примечание:** Более конкретные коды ошибок, такие как ошибки сервисов LLM и TTS третьих сторон, будут включены в `tag.message` для быстрого определения проблемы и уведомления пользователей об ошибках.

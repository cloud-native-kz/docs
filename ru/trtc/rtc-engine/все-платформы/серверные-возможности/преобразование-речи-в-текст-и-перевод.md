# Преобразование речи в текст и перевод

## Варианты использования

Tencent Real-Time Communication (TRTC) поддерживает **преобразование речи в текст** и **перевод**. Это позволяет транскрибировать аудиопотоки указанных пользователей или всех пользователей в комнате в текст и переводить их на другие языки с помощью искусственного интеллекта, обеспечивая универсальные решения, такие как субтитры в реальном времени и перевод.

## Предварительные условия

- Войдите в [консоль TRTC](https://console.trtc.io/), активируйте сервис TRTC и [создайте приложение RTC-Engine](https://www.tencentcloud.com/document/product/647/39077).
- Приобретение [пакета RTC-Engine](https://console.trtc.io/subscription/buy/rtc?packType=starter) (версия Lite или выше) разблокирует функции преобразования речи в текст и перевода в реальном времени.

> **Примечание:** Функции преобразования речи в текст и перевода в реальном времени тарифицируются в зависимости от использования. Подробнее см. в [Прайс-лист](https://trtc.io/document/64149?product=pricing).

## Обзор возможностей

После запуска задачи служба TRTC AI запускает бота автоматического распознавания речи (ASR), который присоединяется к комнате TRTC и подписывается на потоки указанных пользователей или всех пользователей для распознавания речи. Результаты распознавания затем доставляются как клиенту, так и серверу в реальном времени через обратные вызовы.

## Руководство по интеграции

### Шаг 1: Получение результатов преобразования речи в текст

#### Способ 1: Получение текстовых сообщений через SDK клиента

Используя функцию [пользовательского сообщения](https://trtc.io/document/47866#) SDK TRTC, клиент может прослушивать события обратного вызова для получения результатов преобразования речи в текст и перевода в реальном времени. Формат сообщения обратного вызова клиента показан ниже (пример веб-клиента):

```
trtc.on(TRTC.EVENT.CUSTOM_MESSAGE, event => { // Получение пользовательских сообщений.
   // event.userId: userId робота ASR.
   // event.cmdId: ID сообщения, для транскрипций и субтитров зафиксировано на 1.
   // event.seq: Порядковый номер сообщения.
   // event.data: Тип ArrayBuffer. Содержимое транскрипций или субтитров см. в объяснении поля data ниже.
   const data = new TextDecoder().decode(event.data)
   // Объяснение поля data выглядит следующим образом.
   console.log(`received custom msg from ${event.userId}, message: ${ data }`)
})
```

#### Описание поля Data (Субтитры в реальном времени)

| Название поля | Тип | Значение |
| --- | --- | --- |
| type | Integer | 10000: При наличии субтитров в реальном времени и завершенного предложения будет доставлен тип сообщения. |
| sender | String | userId говорящего. |
| receiver | Array | Список userId получателей. Сообщение фактически транслируется в комнате. |
| payload.text | String | Распознанный текст, кодированный в Unicode. |
| payload.start_time | String | Время начала сообщения. Это абсолютное время после запуска задачи. |
| payload.end_time | String | Время окончания сообщения. Это абсолютное время после запуска задачи. |
| payload.end | Boolean | Если true, это указывает на завершенное предложение |

```
{  "type": 10000,  "sender": "user_a",  "payload": {     "text":"",     "start_time":"00:00:02",     "end_time":"00:00:05",     "end": true  }}
```

#### Сообщение перевода в реальном времени

```
{  "type": 10000,  "sender": "ai_951073",  "payload": {    "start_time_ms": 1760,    "end_time_ms": 5530,    "end": false,    "roundid": "e6330a3c-eed7-40bb-8229-9bbe733a313f", // Уникальный ID для каждого раунда разговора    "translation_text": "simultaneous interpretation of the meeting", // Переведённый текст    "translation_language": "en",   // Код языка перевода    "taskid": "x-dPLCz" // Уникальный идентификатор задачи транскрипции  }}
```

> **Примечание:** Пример обратного вызова: **Транскрипция:** Завершённое предложение транскрибируется и отправляется один раз. "How's the weather today?" **Субтитры и перевод:** Результаты субтитров или перевода отправляются поэтапно в сегментах, каждый следующий сегмент содержит предыдущее содержимое для достижения эффекта субтитров в реальном времени. "How" "How's the weather" "How's the weather today?" **Последовательность сообщений: Сообщение субтитров > Сообщение субтитров > .... > Сообщение субтитров (end = true)**

#### Способ 2: Получение через серверные обратные вызовы

Служба преобразования речи в текст также предоставляет серверные обратные вызовы событий, облегчая получение вашим сервисом сообщений разговоров в реальном времени. См. [Подробные события обратного вызова](https://www.tencentcloud.com/document/product/647/66149).

### Шаг 2: Инициирование задачи преобразования речи в текст и перевода

TRTC предоставляет следующие API Tencent Cloud для инициирования и управления задачами преобразования речи в текст и перевода:

- Начать задачу преобразования речи в текст: [StartAITranscription](https://trtc.io/document/64967)
- Запрос задачи преобразования речи в текст: [DescribeAITranscription](https://trtc.io/document/64968)
- Остановить задачу преобразования речи в текст: [StopAITranscription](https://trtc.io/document/64966)
- Начать задачу перевода в реальном времени: Настройте параметр [TranslationConfig](https://trtc.io/zh/document/36760#TranslationConfig) во входных данных [StartAITranscription](https://trtc.io/zh/document/64967?platform=web&product=rtcengine&menulabel=core%20sdk); Укажите целевые языки перевода в [TargetLanguages](https://trtc.io/zh/document/36760#TranslationConfig), например ["en", "ja"]. В настоящее время поддерживаются следующие целевые языки:

| **Код языка** | **Название языка** |
| --- | --- |
| "zh" | Китайский |
| "en" | Английский |
| "es" | Испанский |
| "pt" | Португальский |
| "fr" | Французский |
| "de" | Немецкий |
| "ru" | Русский |
| "ar" | Арабский |
| "ja" | Японский |
| "ko" | Корейский |
| "vi" | Вьетнамский |
| "ms" | Малайский |
| "id" | Индонезийский |
| "it" | Итальянский |
| "th" | Тайский |

> **Примечание:** Функции преобразования речи в текст и перевода в реальном времени поддерживают до **100** одновременных задач на SDKAppId. Чтобы увеличить это ограничение, отправьте запрос в службу поддержки. Функция перевода в реальном времени в настоящее время поддерживает 15 языков: китайский, английский, испанский, португальский, французский, немецкий, русский, арабский, японский, корейский, вьетнамский, малайский, индонезийский, итальянский и тайский. Если вам требуется поддержка дополнительных языков, пожалуйста, [свяжитесь с нами](https://trtc.io/zh/contact). Из-за различий в контексте и языке переводы, созданные искусственным интеллектом, предоставляются только в справочных целях и не должны рассматриваться как единственное профессиональное мнение или вывод.

# Узнайте о Tencent RTC

Опираясь на более чем 20-летний опыт Tencent в области сетевых технологий и аудиовидеотехнологий, Tencent RTC предлагает решения для многопользовательских голосовых и видеозвонков, а также низколатентного интерактивного потокового вещания. В Tencent RTC мы специализируемся на предоставлении глобальным предприятиям передовых решений для связи в реальном времени. Tencent RTC сосредоточена на доставке высококачественных, экономически эффективных и низколатентных решений, обеспечивая бесперебойную и эффективную связь для предприятий по всему миру.

# Каковы преимущества Tencent RTC?

- Ультравысокое качество: оснащённый тем же 3A-аудиомодулем, что и Tencent Meeting, Tencent RTC поддерживает нормальное качество голосовых вызовов даже при потере пакетов 80%, со средней сквозной задержкой менее 300 мс по всему миру.
- Низкий барьер входа и быстрая интеграция: предлагает богатый набор компонентов, специфичных для сценариев, и примеры кода на нескольких платформах для различных секторов, включая развлечения, образование и корпоративные услуги. Официальная поддержка предоставляется для кроссплатформенных фреймворков, таких как Flutter и Electron.
- Кроссплатформенная глобальная совместимость: предоставляет кроссплатформенные SDK для Android, iOS, Windows, Mac и Web с сетью транспортировки в реальном времени, охватывающей более 200 стран и регионов по всему миру.
- Высококачественное глобальное развертывание: обеспечивает всемирное ускоренное подключение с глобальными точками присутствия, включая Азиатско-Тихоокеанский регион, Северную Америку, Европу, Ближний Восток, Африку и Латинскую Америку.

# Какова модель выставления счётов Tencent RTC?

Позиции услуг Tencent RTC (включая Call, Conference, Live и RTC-engine), Chat и Beauty AR следуют комбинированной модели выставления счётов: предоплаченную ежемесячную подписку в качестве базы с оплатой по мере использования за сверхлимитное использование. Для получения дополнительной информации обратитесь к [Обзору выставления счётов](https://www.tencentcloud.com/document/product/647/64149).

**Ежемесячная подписка:** Tencent RTC представила планы предоплаченных пакетов, начиная с 7 марта 2023 г. (UTC+8), открывая различные услуги с добавленной стоимостью для различных приложений (SDKAppIDs). Планы пакетов для Call, Conference, Live, Chat, Beauty AR, RTC Engine и Monitoring Dashboard — это все ежемесячные пакеты подписки. Вы можете выбрать подходящий предоплаченный пакетный план для вашего сценария бизнеса.

**Оплата по мере использования:** Tencent RTC по умолчанию использует предоплаченное пакетное выставление счётов. Если использование услуги не может быть вычтено из пакета или превышает баланс предоплаченного пакета, оно будет выставлено на основе оплаты по мере использования в соответствии с фактическим использованием вашего бизнеса.

> **Примечание:** когда вы начинаете тестирование, мы рекомендуем начать с бесплатной пробной версии. Каждый из наших продуктов предлагает бесплатную пробную версию, которая помогает вам сделать лучший выбор на основе функций продукта и вашего бизнес-сценария. Когда вы тестируете с реальными пользователями или запускаете свой проект в производственной среде, мы рекомендуем приобрести план ежемесячной подписки. Мы разработали [план Starter](https://trtc.io/pricing/promotion) для новых пользователей при первой покупке. Это позволяет вам использовать наши продукты с ежемесячным пакетом по значительно сниженной цене. В период действия режим выставления счётов по мере использования будет включен для вашего аккаунта, позволяя вам масштабировать в любой момент. Даже если использование превышает лимит плана, обслуживание не будет прервано. Вы можете выбрать, какой продукт использовать, на основе вашего конкретного сценария. Если ваш бизнес охватывает несколько вариантов использования продукта, вы можете выбрать использование RTC Engine.
